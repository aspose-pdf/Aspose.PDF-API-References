## CosPdfString.IsHexadecimal 속성

이 인스턴스가 16진수인지 여부를 나타내는 값을 가져옵니다.

```csharp
public bool IsHexadecimal { get; }
```

### 속성 값

`true` 이 인스턴스가 16진수인 경우; 그렇지 않으면 `false`입니다.
