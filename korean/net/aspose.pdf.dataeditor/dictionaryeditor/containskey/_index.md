## DictionaryEditor.ContainsKey 메서드

[`DictionaryEditor`](../)에 지정된 키를 가진 요소가 포함되어 있는지 여부를 결정합니다.

```csharp
public bool ContainsKey(string key)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| key | String | [`DictionaryEditor`](../)에서 찾을 키입니다. |

### 반환 값

[`DictionaryEditor`](../)에 키가 있는 편집 가능한 요소가 포함되어 있으면 true; 그렇지 않으면 false입니다.
