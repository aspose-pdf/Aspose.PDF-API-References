
### 값들

| 이름 | 값 | 설명 |
| --- | --- | --- |
| 정의되지 않음 | `0` | 정의되지 않은 상태. |
| 표시됨 | `1` | 사용자가 주석을 표시했습니다. |

| 완료됨 | `6` | 변경 사항이 완료되었습니다. |
| 없음 | `7` | 사용자가 변경 사항에 대해 아무것도 표시하지 않았습니다. |
