## AppearanceDictionary 인덱서

외관 스트림을 가져오는 편리한 형식을 나타냅니다.

```csharp
public XForm this[string key] { get; set; }
```

| 매개변수 | 설명 |
| --- | --- |
| key | 외관 스트림에 대한 경로를 나타냅니다. 외관 사전이 하위 사전을 가지고 있다면, 경로는 2부분을 포함해야 합니다 ([`Keys`](../keys/)), 그렇지 않으면 경로는 1부분만 가집니다. |

### 반환 값

주어진 키에 해당하는 XForm 객체(외관 스트림).
