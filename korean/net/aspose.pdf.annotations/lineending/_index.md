## LineEnding 열거형

선을 그릴 때 사용할 선 끝 스타일을 열거합니다.

```csharp
public enum LineEnding
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 선 끝 없음. |
| Square | `1` | 주석의 내부 색상으로 채워진 정사각형, 있을 경우. |
| Circle | `2` | 주석의 내부 색상으로 채워진 원, 있을 경우. |
| Diamond | `3` | 주석의 내부 색상으로 채워진 다이아몬드 모양, 있을 경우. |
| OpenArrow | `4` | 예각에서 만나는 두 개의 짧은 선으로 열린 화살촉을 형성합니다. |
| ClosedArrow | `5` | OpenArrow 스타일과 같이 예각에서 만나는 두 개의 짧은 선이 세 번째 선으로 연결되어 주석의 내부 색상으로 채워진 삼각형 닫힌 화살촉을 형성합니다, 있을 경우. |
| Butt | `6` | 선 자체에 수직인 끝점에 있는 짧은 선. |
| ROpenArrow | `7` | OpenArrow의 반대 방향으로 두 개의 짧은 선. |
| RClosedArrow | `8` | ClosedArrow의 반대 방향으로 삼각형 닫힌 화살촉. |
| Slash | `9` | 선 자체에 수직에서 시계 방향으로 약 30도 위치한 끝점에 있는 짧은 선. |
