## Add(Annotation, bool) {#add_1}

주석을 컬렉션에 추가합니다. 페이지가 회전되면 주석 사각형이 그에 따라 재계산됩니다.

```csharp
public void Add(Annotation annotation, bool considerRotation)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| annotation | Annotation | 추가될 주석입니다. |
| considerRotation | Boolean | true이고 페이지가 회전된 경우 주석 위치가 페이지 회전에 따라 재계산됩니다. |
