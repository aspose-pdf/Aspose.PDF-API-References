## HighlightingMode 열거형

주석의 강조 모드, 마우스 버튼이 활성 영역 내에서 눌리거나 눌린 상태일 때 사용할 시각적 효과를 열거합니다.

```csharp
public enum HighlightingMode
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 강조 없음. |
| Invert | `1` | 주석 사각형의 내용을 반전합니다. |
| Outline | `2` | 주석의 테두리를 반전합니다. |
| Push | `3` | 주석의 눌림 상태를 표시합니다. 눌림 상태가 정의되지 않은 경우, 주석 사각형의 내용을 페이지 표면 아래로 눌린 것처럼 보이도록 오프셋합니다. |
| Toggle | `4` | Push와 동일합니다(선호됨). |
