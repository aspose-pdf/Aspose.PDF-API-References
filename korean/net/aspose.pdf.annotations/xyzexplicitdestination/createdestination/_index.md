## XYZExplicitDestination.CreateDestination 메서드

페이지 회전을 고려하여 지정된 위치에 목적지를 생성합니다.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| page | Page | 목적지 페이지. |
| left | Double | 페이지의 왼쪽 위치. |
| top | Double | 페이지의 위쪽 위치. |
| zoom | Double | 확대 비율 (기본값은 0). |
| considerRotation | Boolean | true인 경우 위치가 페이지 회전에 따라 재계산됩니다. |

### 반환 값

목적지 객체.
