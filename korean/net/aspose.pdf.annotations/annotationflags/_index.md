## AnnotationFlags 열거형

주석의 다양한 특성을 지정하는 플래그 집합입니다.

```csharp
[Flags]
public enum AnnotationFlags
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Default | `0` | 기본값입니다. |
| Invisible | `1` | 설정된 경우, 표준 주석 유형 중 하나에 속하지 않거나 주석 처리기가 없는 경우 주석을 표시하지 않습니다. 해제된 경우, 해당 주석 사전에서 지정된 외관 스트림을 사용하여 이러한 알 수 없는 주석을 표시합니다. |
| Hidden | `2` | 설정된 경우, 주석 유형이나 주석 처리기의 유무에 관계없이 주석을 표시하거나 인쇄하지 않으며 사용자와 상호작용하지 않도록 합니다. 화면 공간이 제한된 경우, 주석을 선택적으로 숨기고 표시하는 기능은 외관 스트림과 결합하여 온라인 도움말 시스템과 유사한 보조 팝업 정보를 표시하는 데 사용할 수 있습니다. |
| Print | `4` | 설정된 경우, 페이지가 인쇄될 때 주석을 인쇄합니다. 해제된 경우, 화면에 표시되더라도 주석을 절대 인쇄하지 않습니다. 예를 들어, 인쇄된 페이지에서 의미 있는 목적이 없는 대화형 푸시 버튼을 나타내는 주석에 유용할 수 있습니다. |
| NoZoom | `8` | 설정된 경우, 페이지의 확대/축소에 맞춰 주석의 외관을 조정하지 않습니다. 주석의 위치(주석 사각형의 왼쪽 상단 모서리로 정의됨)는 페이지 확대/축소에 관계없이 고정됩니다. |
| NoRotate | `10` | 설정된 경우, 페이지의 회전에 맞춰 주석의 외관을 회전하지 않습니다. 주석 사각형의 왼쪽 상단 모서리는 페이지 회전에 관계없이 고정된 위치에 유지됩니다. |
| NoView | `20` | 설정된 경우, 화면에 주석을 표시하지 않거나 사용자와 상호작용하지 않도록 합니다. 주석은 인쇄될 수 있지만(인쇄 플래그의 설정에 따라) 화면 표시 및 사용자 상호작용의 목적상 숨겨진 것으로 간주해야 합니다. |
| ReadOnly | `40` | 설정된 경우, 주석이 사용자와 상호작용하지 않도록 합니다. 주석은 표시되거나 인쇄될 수 있지만(표시 및 인쇄 플래그의 설정에 따라) 마우스 클릭에 반응하거나 마우스 움직임에 따라 외관을 변경하지 않아야 합니다. 이 플래그는 위젯 주석에 대해 무시되며, 그 기능은 관련 양식 필드의 ReadOnly 플래그에 포함됩니다. |
| Locked | `80` | 설정된 경우, 사용자가 주석을 삭제하거나 속성(위치 및 크기 포함)을 수정하지 못하도록 합니다. 그러나 이 플래그는 주석의 내용 변경(예: 양식 필드의 값)에는 제한을 두지 않습니다. |
| ToggleNoView | `100` | 설정된 경우, 특정 이벤트에 대해 NoView 플래그의 해석을 반전시킵니다. 일반적인 사용 예는 마우스 커서가 주석 위에 있을 때만 주석이 나타나는 것입니다. |
| LockedContents | `200` | 설정된 경우, 사용자가 주석의 내용을 수정하지 못하도록 합니다. 이 플래그는 주석 삭제 또는 위치 및 크기와 같은 다른 주석 속성 변경을 제한하지 않습니다. |
