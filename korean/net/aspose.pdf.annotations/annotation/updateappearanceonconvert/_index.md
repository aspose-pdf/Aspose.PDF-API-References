## Annotation.UpdateAppearanceOnConvert 속성

true인 경우 주석 모양이 PF 문서를 이미지로 변환하기 전에 업데이트됩니다. 이렇게 하면 필드를 올바르게 변환할 수 있지만 아마도 더 많은 시간이 필요할 것입니다.

```csharp
public static bool UpdateAppearanceOnConvert { get; set; }
```
