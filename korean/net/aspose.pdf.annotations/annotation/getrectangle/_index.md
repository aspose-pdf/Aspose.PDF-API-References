## Annotation.GetRectangle 메서드

페이지 회전을 고려하여 주석의 사각형을 반환합니다.

```csharp
public Rectangle GetRectangle(bool considerRotation)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| considerRotation | Boolean | true인 경우 페이지 회전이 고려됩니다. |

### 반환 값

사각형이 발견되면 true; 그렇지 않으면 false입니다.
