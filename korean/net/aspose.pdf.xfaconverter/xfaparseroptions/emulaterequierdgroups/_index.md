## XfaParserOptions.EmulateRequierdGroups 속성

이 속성이 true이면 필수 Xfa "제외 그룹"에 대해 추가적인 빨간 사각형이 그려집니다. 이 속성은 표준으로의 양식 변환 중 제외된 그룹의 유사체가 없기 때문에 도입되었습니다. 기본값은 false입니다.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
