## XfaParserOptions.Signed 속성

이 속성이 true이면 문서는 xfa 양식 스트림을 사용하여 변환됩니다(존재하는 경우). false이면 xfa 양식 스트림은 무시됩니다. 이 속성은 서명을 확인하는 데 사용되는 체크섬을 계산하는 방법이 명확하지 않기 때문에 도입되었습니다.

```csharp
public bool Signed { get; set; }
```
