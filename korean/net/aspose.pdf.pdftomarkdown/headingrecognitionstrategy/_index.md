## HeadingRecognitionStrategy 열거형

헤더 인식 전략의 유형을 나타냅니다.

```csharp
public enum HeadingRecognitionStrategy
```

### 값들

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Outlines | `0` | 윤곽선을 통해 헤더 인식 전략을 나타냅니다. |
| Heuristic | `1` | 휴리스틱 규칙과 글꼴 크기 통계를 통해 헤더 인식 전략을 나타냅니다. |
| Auto | `2` | 자동 헤더 인식 전략 선택을 제공합니다. 이것이 기본 옵션입니다. 문서에 북마크가 포함되어 있으면 Outlines 전략이 선택되고, 그렇지 않으면 Heuristic이 선택됩니다. |
| None | `3` | 헤더를 인식하지 않습니다. 이 옵션은 복잡하게 형식이 지정된 문서에서 유용할 수 있습니다. |
