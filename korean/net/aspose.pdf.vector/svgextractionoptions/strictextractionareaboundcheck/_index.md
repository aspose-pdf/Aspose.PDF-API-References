## SvgExtractionOptions.StrictExtractionAreaBoundCheck 속성

[`ExtractionAreaBound`](../extractionareabound/)에서 하위 경로가 지정된 사각형 내에 있는지 엄격하게 확인하는 옵션을 가져오고 설정합니다. false로 설정하면 [`ExtractionAreaBound`](../extractionareabound/)에 완전히 포함되지 않은 하위 경로가 추출됩니다. 기본값은 `True`입니다.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
