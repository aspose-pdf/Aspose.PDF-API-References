## SvgExtractionOptions.MinStrokeWidth 속성

결과 SVG에서 사용될 최소 스트로크 너비를 가져오거나 설정합니다. PDF가 더 얇은 스트로크 너비를 사용하면 이 너비로 대체됩니다. 기본값은 0.5입니다.

```csharp
public double MinStrokeWidth { get; set; }
```

## 비고

값은 변환된 PDF 페이지의 변환된 사용자 공간 단위로 표현됩니다. 기본적으로 1 사용자 공간 단위는 1/72 인치(0.35 mm)지만, 이는 PDF 문서에 의해 재정의될 수 있습니다. 변환은 생성된 SVG에서 실제 최소 너비에 영향을 줄 수 있습니다.
