**Aspose.Pdf.Vector**는 그래픽 작업을 위한 루트 네임스페이스입니다.

## 클래스

| 클래스 | 설명 |
| --- | --- |
| [GraphicElement](./graphicelement/) | 페이지의 그래픽 객체에 대한 기본 클래스를 나타냅니다. |
| [GraphicElementCollection](./graphicelementcollection/) | [`GraphicElement`](../aspose.pdf.vector/graphicelement/) 컬렉션을 나타냅니다. |
| [GraphicsAbsorber](./graphicsabsorber/) | 그래픽 요소의 흡수기 객체를 나타냅니다. 그래픽 검색을 수행하고 [`Elements`](../aspose.pdf.vector/graphicsabsorber/elements/) 컬렉션을 통해 검색 결과에 대한 액세스를 제공합니다. |
| [GraphicState](./graphicstate/) | 현재 [`GraphicElement`](../aspose.pdf.vector/graphicelement/)의 그래픽 상태를 나타냅니다. |
| [SubPath](./subpath/) | 페이지의 벡터 그래픽 객체를 나타냅니다. 기본적으로 벡터 그래픽 객체는 두 그룹의 SubPaths로 표현됩니다. 그 중 하나는 선과 곡선의 집합으로 표현됩니다. 다른 하나는 사각형으로 표현되며 때때로 혼동될 수 있습니다. 일반적으로 색상이 있는 직사각형 영역이지만, 이 직사각형은 페이지의 시작 부분에 배치되어 페이지의 전체 공간을 흰색으로 정의합니다. 따라서 SubPath를 얻지만 시각적으로는 페이지의 텍스트만 볼 수 있습니다. |
| [SubPathGroup](./subpathgroup/) | 그래픽 요소 컨테이너 그룹을 위한 클래스를 나타냅니다. 클래스 객체는 그룹 크기를 고려하기 위한 경계 상자를 가지고 있습니다. |
| [SvgExtractionOptions](./svgextractionoptions/) | PDF 문서 페이지에서 벡터 그래픽을 추출하기 위한 옵션 클래스를 나타냅니다. |
| [SvgExtractor](./svgextractor/) | 페이지에서 SVG 이미지를 추출하기 위한 클래스를 나타냅니다. |
| [XFormPlacement](./xformplacement/) | XForm 배치를 나타냅니다. XForm이 페이지에 1회 이상 표시되는 경우, 이 XForm과 관련된 모든 XformPlacements는 공통 그래픽 요소를 가지지만 서로 다른 그래픽 상태를 가집니다. |
