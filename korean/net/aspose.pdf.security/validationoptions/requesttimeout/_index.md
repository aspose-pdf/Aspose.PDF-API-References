## ValidationOptions.RequestTimeout 속성

검증 프로세스 중 네트워크 관련 작업에 대한 타임아웃 기간(밀리초)을 가져오거나 설정합니다. RequestTimeout 속성은 취소 상태 또는 OCSP 서버와 같은 온라인 리소스에 접근할 때 시스템이 네트워크 응답을 기다려야 하는 최대 시간을 정의합니다.

```csharp
public int RequestTimeout { get; set; }
```
