## ValidationResult.Status 속성

인증서의 검증 프로세스 상태를 가져옵니다.

```csharp
public ValidationStatus Status { get; }
```

## 비고

Status 속성은 인증서 검증의 결과를 나타냅니다. 가능한 값은 [`ValidationStatus`](../../validationstatus/) 열거형에 정의되어 있으며, Valid, Invalid 또는 Undefined와 같은 값이 포함됩니다. 이는 인증서가 검증 검사를 통과했는지 여부에 대한 통찰력을 제공합니다.
