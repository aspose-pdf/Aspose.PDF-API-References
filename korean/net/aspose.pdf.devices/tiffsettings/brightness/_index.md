## TiffSettings.Brightness 속성

흑백 색상의 변환 값 경계를 가져오거나 설정합니다. 이 매개변수는 EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle 또는 ColorDepth.Format1bpp == 1과 함께 적용할 수 있습니다.

```csharp
public float Brightness { get; set; }
```

### 속성 값

밝기 값은 0에서 1 사이여야 합니다. 기본값은 0.33f입니다.
