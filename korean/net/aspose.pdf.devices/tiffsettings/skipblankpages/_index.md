## TiffSettings.SkipBlankPages 속성

빈 페이지를 건너뛸지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool SkipBlankPages { get; set; }
```

### 속성 값

빈 페이지를 건너뛰어야 하면 `true`; 그렇지 않으면 `false`입니다.

## 비고

기본값은 false입니다.
