**Aspose.Pdf.Devices** 네임스페이스는 문서를 이미지 또는 일반 텍스트로 표현하는 데 사용되는 클래스를 제공합니다. 따라서 문서는 텍스트 또는 다양한 그래픽 장치로 전송될 수 있으며, 이는 우리가 텍스트 또는 그래픽 표현을 얻고자 함을 의미합니다.

## 클래스

| 클래스 | 설명 |
| --- | --- |
| [BmpDevice](./bmpdevice/) | pdf 문서 페이지를 bmp로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [Device](./device/) | 모든 유형의 장치에 대한 추상 클래스입니다. 장치는 pdf 문서를 일부 형식으로 표현하는 데 사용됩니다. 예를 들어, 문서 페이지는 이미지 또는 텍스트로 표현될 수 있습니다. |
| [DicomDevice](./dicomdevice/) | pdf 문서 페이지를 Dicom 형식으로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [DocumentDevice](./documentdevice/) | 전체 pdf 문서를 처리하는 데 사용되는 모든 장치에 대한 추상 클래스입니다. |
| [EmfDevice](./emfdevice/) | pdf 문서 페이지를 emf로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [GifDevice](./gifdevice/) | pdf 문서 페이지를 gif로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [ImageDevice](./imagedevice/) | 이미지 장치에 대한 추상 클래스입니다. |
| [JpegDevice](./jpegdevice/) | pdf 문서 페이지를 jpeg로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [Margins](./margins/) | 이 클래스는 이미지의 여백을 나타냅니다. |
| [PageDevice](./pagedevice/) | 특정 페이지의 pdf 문서를 처리하는 데 사용되는 모든 장치에 대한 추상 클래스입니다. |
| [PngDevice](./pngdevice/) | pdf 문서 페이지를 png로 저장하는 데 도움이 되는 이미지 장치를 나타냅니다. |
| [Resolution](./resolution/) | 이미지 해상도를 보유하는 클래스를 나타냅니다. |
| [TextDevice](./textdevice/) | pdf 문서 페이지를 텍스트로 변환하는 클래스를 나타냅니다. |
| [ThumbnailDevice](./thumbnaildevice/) | pdf 문서 페이지를 썸네일 이미지로 저장하는 이미지 장치를 나타냅니다. |
| [TiffDevice](./tiffdevice/) | 이 클래스는 pdf 문서를 페이지별로 하나의 tiff 이미지로 저장하는 데 도움이 됩니다. |
| [TiffSettings](./tiffsettings/) | 이 클래스는 pdf를 Tiff로 가져오는 설정을 나타냅니다. |
