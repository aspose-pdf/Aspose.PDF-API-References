## BoundsCheckableList&lt;T&gt;.IndexOf 메서드

지정된 객체를 검색하고 전체 System.Collections.Generic.List 내에서 첫 번째 발생의 0 기반 인덱스를 반환합니다.

```csharp
public int IndexOf(T item)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| item | T | System.Collections.Generic.List에서 찾을 객체입니다. 참조 유형의 경우 값은 null일 수 있습니다. |

### 반환 값

찾은 경우 전체 System.Collections.Generic.List 내에서 *item*의 첫 번째 발생의 0 기반 인덱스; 그렇지 않으면 –1입니다.
