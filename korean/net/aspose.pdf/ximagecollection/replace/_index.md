## Replace(int, Stream) {#replace}


| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | [1..images count] 범위 내에서 교체될 컬렉션 항목의 인덱스입니다. |

```csharp
public void Replace(int index, Stream stream, int quality, bool isBlackAndWhite)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | [1..images count] 범위 내에서 교체될 컬렉션 항목의 인덱스입니다. |
