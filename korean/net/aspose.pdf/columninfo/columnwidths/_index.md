## ColumnInfo.ColumnWidths 속성

열의 너비를 포함하는 문자열을 가져오거나 설정합니다. 각 열의 값은 공백으로 구분되어야 합니다. 기본 단위는 포인트이지만 cm, 인치 및 사용 가능한 너비의 백분율도 지원됩니다. 예: "120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
