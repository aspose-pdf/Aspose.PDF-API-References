## Artifact.ArtifactType 열거형


### 값들

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Pagination | `0` | 페이지 매김 아티팩트. 실행 헤드 및 페이지 번호와 같은 부수적인 페이지 기능. |
| Layout | `1` | 레이아웃 아티팩트. 각주 규칙이나 배경 화면과 같은 순수하게 장식적인 타이포그래피 또는 디자인 요소. |
| Page | `2` | 페이지 아티팩트. 문서 자체와는 무관한 제작 보조 도구, 예를 들어 절단 표시 및 색상 막대. |
| Background | `3` | 배경 아티팩트. 이미지, 패턴 또는 색상 블록. |
| Undefined | `4` | 아티팩트 유형이 정의되지 않았거나 알 수 없음. |
