## TeXLoadOptions.RequiredInputDirectory 속성

TeX가 요구하는 입력 디렉토리를 가져오거나 설정합니다. 필수 입력은 주 .tex 파일에 포함된 파일로, 예를 들어 내장 지원이 없는 패키지입니다.

```csharp
public ITeXInputDirectory RequiredInputDirectory { get; set; }
```
