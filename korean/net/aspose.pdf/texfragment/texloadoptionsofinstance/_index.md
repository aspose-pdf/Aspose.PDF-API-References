## TeXFragment.TeXLoadOptionsOfInstance 속성

이 클래스의 인스턴스에 LaTeX를 로드(및 렌더링)하는 데 사용될 TeXLoadOptions를 가져오거나 설정합니다. 특정 인스턴스에 대해 LaTeX를 가져오기 위한 특정 설정을 사용해야 할 때 사용하십시오. 예를 들어, 특정 인스턴스가 가져온 LaTeX에 대해 특정 BasePath를 사용해야 하거나 외부 리소스의 특정 로더를 사용해야 할 때입니다. 매개변수가 기본값(null)인 경우 표준 LaTeX 로딩 옵션이 사용됩니다.

```csharp
public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
```
