## Stamp.Rotate 속성

[`Rotation`](../../rotation/) 값을 기준으로 스탬프 콘텐츠의 회전을 설정하거나 가져옵니다. 참고. 이 속성은 90도(0, 90, 180, 270도)의 배수인 각도를 설정하기 위한 것입니다. 임의의 각도를 설정하려면 RotateAngle 속성을 사용하십시오. ArbitraryAngle로 설정된 각도가 90의 배수가 아닌 경우 Rotate 속성은 Rotation.None을 반환합니다.

```csharp
public Rotation Rotate { get; set; }
```
