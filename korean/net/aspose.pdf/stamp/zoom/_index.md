## Stamp.Zoom 속성

스탬프의 확대 비율. 스탬프를 크기 조정할 수 있습니다. ZoomX 및 ZoomY 속성 쌍을 사용하면 각 축에 대해 별도로 확대 비율을 설정할 수 있습니다. 이 속성을 설정하면 ZoomX 및 ZoomY 속성이 모두 변경됩니다. ZoomX와 ZoomY가 다르면 Zoom 속성은 ZoomX 값을 반환합니다.

```csharp
public double Zoom { get; set; }
```
