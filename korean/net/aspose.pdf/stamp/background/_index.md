## Stamp.Background 속성

콘텐츠가 배경으로 스탬프되었음을 나타내는 bool 값을 설정하거나 가져옵니다. 값이 true이면 스탬프 콘텐츠가 하단에 배치됩니다. 기본적으로 값은 false이며 스탬프 콘텐츠가 상단에 배치됩니다.

```csharp
public bool Background { get; set; }
```
