## PaginationArtifact.EndPage 속성

아티팩트의 끝 페이지 번호를 가져오거나 설정합니다. 값은 0 이상이어야 합니다. 0 미만의 값이 설정되면 0으로 조정됩니다. 기본값 0은 끝 페이지 경계가 없음을 의미합니다.

```csharp
public int EndPage { get; set; }
```
