## PageLayout 열거형

페이지 레이아웃을 설명합니다.

```csharp
public enum PageLayout
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| SinglePage | `0` | 단일 페이지. |
| OneColumn | `1` | 페이지를 한 열로 표시합니다. |
| TwoColumnLeft | `2` | 페이지를 두 열로 표시하며, 홀수 페이지는 왼쪽에 있습니다. |
| TwoColumnRight | `3` | 페이지를 두 열로 표시하며, 홀수 페이지는 오른쪽에 있습니다. |
| TwoPageLeft | `4` | 페이지를 두 개씩 표시하며, 홀수 페이지는 왼쪽에 있습니다. |
| TwoPageRight | `5` | 페이지를 두 개씩 표시하며, 홀수 페이지는 오른쪽에 있습니다. |
| Default | `6` | 기본 레이아웃. |
