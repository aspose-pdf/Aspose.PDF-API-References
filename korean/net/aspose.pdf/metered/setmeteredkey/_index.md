## Metered.SetMeteredKey 메서드

미터링된 공개 및 비공개 키를 설정합니다. 미터링된 라이센스를 구매하면 애플리케이션을 시작할 때 이 API를 호출해야 하며, 일반적으로 이것으로 충분합니다. 그러나 소비 데이터 업로드에 항상 실패하고 24시간을 초과하면 라이센스가 평가 상태로 설정됩니다. 이러한 경우를 피하기 위해 라이센스 상태를 정기적으로 확인해야 하며, 평가 상태인 경우 이 API를 다시 호출하십시오.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| publicKey | 문자열 | 공개 키 |
| privateKey | 문자열 | 비공개 키 |
