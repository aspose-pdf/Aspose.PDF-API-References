### 값들

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Auto | `0` | 서명 알고리즘의 재량에 따라 해싱 알고리즘을 자동으로 설정합니다. EDCSA의 경우 기본값은 키 크기에 따라 결정됩니다. 분리되지 않은 PKCS7의 기본값은 Sha1입니다. |

| Sha384 | `3` | SHA-384. 보안 해시 알고리즘 2 |
| Sha512 | `4` | SHA-512. 보안 해시 알고리즘 2 |
| Sha3_256 | `5` | SHA3-256. 보안 해시 알고리즘 3 |
