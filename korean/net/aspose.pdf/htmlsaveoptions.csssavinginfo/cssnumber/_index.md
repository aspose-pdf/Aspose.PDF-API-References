## HtmlSaveOptions.CssSavingInfo.CssNumber 필드

변환기에 의해 설정됩니다. 변환 중 여러 CSS 파일이 생성됩니다. 이 속성은 변환 중 저장된 CSS 파일의 순서를 보여줍니다. CSS 콘텐츠를 처리하거나 저장할 위치를 결정하기 위해 사용자 정의 코드의 논리에서 사용할 수 있습니다.

```csharp
public int CssNumber;
```
