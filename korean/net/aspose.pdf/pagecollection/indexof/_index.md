## PageCollection.IndexOf 메서드

지정된 페이지의 인덱스를 반환합니다.

```csharp
public int IndexOf(Page entity)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| entity | Page | 페이지 객체. 페이지 번호는 1부터 시작합니다. |

### 반환 값

컬렉션에서 페이지의 인덱스.

## 비고

페이지 번호는 1부터 시작합니다. 컬렉션에 페이지가 포함되어 있지 않은 경우 0을 반환합니다.
