## PasswordType 열거형

이 열거형은 비밀번호로 보호된 PDF 문서에 사용되는 알려진 비밀번호 유형을 나타냅니다.

```csharp
public enum PasswordType
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | PDF 문서는 비밀번호로 보호되지 않습니다. |
| User | `1` | PDF 문서는 문서 열기 비밀번호(제한된 접근)를 사용하여 열렸습니다. |
| Owner | `2` | PDF 문서는 권한 변경 비밀번호(전체 접근)를 사용하여 열렸습니다. |
| Inaccessible | `3` | PDF 문서는 비밀번호로 보호되지만 사용자 및 소유자 비밀번호가 모두 비어 있지 않으며 비밀번호가 정의되지 않았거나 제공된 비밀번호가 잘못되었습니다. 따라서 비밀번호 유형을 추론할 수 없습니다. |
