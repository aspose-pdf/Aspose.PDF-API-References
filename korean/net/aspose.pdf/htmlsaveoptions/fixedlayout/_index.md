## HtmlSaveOptions.FixedLayout 속성

HTML이 고정 레이아웃으로 생성되는지를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool FixedLayout { get; set; }
```

### 속성 값

`true`이면 [고정 레이아웃]; 그렇지 않으면 `false`입니다.
