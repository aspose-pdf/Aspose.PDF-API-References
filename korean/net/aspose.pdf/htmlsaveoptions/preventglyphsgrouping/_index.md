## HtmlSaveOptions.PreventGlyphsGrouping 속성

이 속성은 텍스트 글리프가 단어와 문자열로 그룹화되지 않는 모드를 활성화합니다. 이 모드는 페이지에서 글리프의 위치를 최대한 정확하게 유지할 수 있게 하며, 음악 기호나 서로 별도로 배치해야 하는 글리프가 있는 문서를 변환하는 데 사용할 수 있습니다. 이 매개변수는 FixedLayout 속성의 값이 true일 때만 문서에 적용됩니다.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
