## HtmlSaveOptions.FontSources 속성

미리 저장된 글꼴의 글꼴 소스입니다.

```csharp
public FontSourceCollection FontSources { get; }
```

## 비고

글꼴은 캐시 목적으로 미리 저장된 후 Html 변환 프로세스에 전달될 수 있습니다. 예를 들어, 문서 분할 시나리오와 단일 글꼴 세트로 여러 스레드에서 문서 페이지를 처리하는 데 유용할 수 있습니다.
