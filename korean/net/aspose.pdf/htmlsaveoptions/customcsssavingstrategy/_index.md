## HtmlSaveOptions.CustomCssSavingStrategy 필드

이 필드는 Pdf를 Html로 변환할 때 생성된 HTML 문서 전체 또는 여러 HTML 페이지가 생성된 경우 해당 페이지와 관련된 CSS 저장을 처리하기 위해 사용해야 하는 저장 전략을 포함할 수 있습니다. CSS 파일을 특정 방식으로 처리하려면 관련 메서드를 생성하고 그로부터 생성된 델리게이트를 이 속성에 할당하십시오.

```csharp
public CssSavingStrategy CustomCssSavingStrategy;
```
