## HtmlSaveOptions.RenderTextAsImage 속성

RenderTextAsImage 속성이 true로 설정되면 소스의 텍스트가 HTML에서 이미지로 변환됩니다. 텍스트를 선택할 수 없게 하거나 HTML 텍스트가 제대로 렌더링되지 않을 때 유용할 수 있습니다.

```csharp
public bool RenderTextAsImage { get; set; }
```
