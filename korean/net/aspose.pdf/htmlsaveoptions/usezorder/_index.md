## HtmlSaveOptions.UseZOrder 속성

UseZOrder 속성이 true로 설정되면 그래픽과 텍스트가 원본 PDF 문서의 Z 순서에 따라 결과 HTML 문서에 추가됩니다. 이 속성이 false인 경우 모든 그래픽이 단일 레이어로 배치되어 겹친 객체에 대해 불필요한 효과를 초래할 수 있습니다.

```csharp
public bool UseZOrder { get; set; }
```
