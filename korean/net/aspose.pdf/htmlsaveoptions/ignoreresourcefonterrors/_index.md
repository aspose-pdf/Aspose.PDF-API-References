## HtmlSaveOptions.IgnoreResourceFontErrors 속성

글꼴의 부재와 관련된 오류를 무시할 것인지의 여부를 가져오거나 설정합니다. true - 글꼴의 부재로 인한 오류가 무시됨을 의미합니다. 잘못된 리소스를 참조하는 텍스트 세그먼트는 처리 중에 건너뛰어집니다. 기본값은 false입니다.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
