## HtmlSaveOptions.SaveFullFont 속성

전체 글꼴이 저장됨을 나타내며, True Type Fonts만 지원합니다. 기본적으로 SaveFullFont = false이며, 변환기는 문서의 텍스트를 표시하는 데 필요한 초기 글꼴의 하위 집합을 저장합니다.

```csharp
public bool SaveFullFont { get; set; }
```
