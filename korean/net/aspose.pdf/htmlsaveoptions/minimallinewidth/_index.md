## HtmlSaveOptions.MinimalLineWidth 속성

이 속성은 그래픽 경로 선의 최소 너비를 설정합니다. 선의 두께가 1px보다 작으면 Adobe Acrobat이 이 값으로 반올림합니다. 따라서 이 속성은 HTML 브라우저에서 이러한 동작을 에뮬레이트하는 데 사용할 수 있습니다.

```csharp
public float MinimalLineWidth { get; set; }
```
