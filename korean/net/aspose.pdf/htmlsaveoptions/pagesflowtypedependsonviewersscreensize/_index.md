## HtmlSaveOptions.PagesFlowTypeDependsOnViewersScreenSize 필드

속성 'SplitOnPages=false'인 경우 모든 입력 PDF 페이지를 나타내는 전체 HTML이 하나의 큰 결과 HTML 파일로 들어갑니다. 이 플래그는 결과 HTML이 생성되는 방식이 결과 HTML에서 PDF 페이지를 나타내는 영역의 흐름이 뷰어의 화면 해상도에 따라 달라지도록 정의합니다. 뷰어 측의 화면 너비가 수평 방향으로 2개 이상의 페이지를 나란히 배치할 수 있을 만큼 충분히 크다고 가정합니다. 이 플래그가 true로 설정되면 이 기회가 사용되며 (가능한 한 많은 페이지가 수평 방향으로 나란히 표시되고 다음 수평 페이지 그룹이 첫 번째 아래에 표시됩니다). 그렇지 않으면 페이지는 다음 페이지가 항상 이전 페이지 아래로 흐르는 방식으로 흐릅니다.

```csharp
public bool PagesFlowTypeDependsOnViewersScreenSize;
```
