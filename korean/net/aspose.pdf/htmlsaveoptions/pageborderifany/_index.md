## HtmlSaveOptions.PageBorderIfAny 필드

이 속성은 결과 HTML 문서에서 소스 PDF 페이지를 나타내는 영역 주위에 경계를 그리는 데 사용되는 설정 집합을 나타냅니다. 본질적으로 이는 PDF 페이지 자체에서 참조된 페이지 경계가 아닌 페이지의 종이 가장자리를 표시하는 것과 관련이 있습니다.

```csharp
public BorderInfo PageBorderIfAny;
```
