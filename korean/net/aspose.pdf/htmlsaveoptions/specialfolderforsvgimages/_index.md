## HtmlSaveOptions.SpecialFolderForSvgImages 필드

문서를 HTML로 저장하는 동안 SVG 이미지가 발견되면 저장해야 하는 디렉토리의 경로를 가져오거나 설정합니다. 매개변수가 비어 있거나 null인 경우 SVG 파일(있는 경우)은 다른 이미지 파일(출력 파일 근처)과 함께 저장되거나 이미지용 특별 폴더(특별 폴더가 SpecialImagesFolderIfAny 옵션에 지정된 경우)에 저장됩니다. CustomImageSavingStrategy 속성이 관련 이미지 파일을 처리하는 데 성공적으로 사용된 경우에는 아무런 영향을 미치지 않습니다.

```csharp
public string SpecialFolderForSvgImages;
```
