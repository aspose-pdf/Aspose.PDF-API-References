## HtmlSaveOptions.ConvertMarkedContentToLayers 속성

ConvertMarkedContentToLayers 속성이 true로 설정되면 PDF 마크된 콘텐츠(레이어) 내의 모든 요소가 "data-pdflayer" 속성이 있는 HTML div에 배치됩니다. 이 레이어 이름은 PDF 마크된 콘텐츠의 선택적 속성에서 추출됩니다. 이 속성이 false(기본값)인 경우 PDF 마크된 콘텐츠에서 레이어가 생성되지 않습니다.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
