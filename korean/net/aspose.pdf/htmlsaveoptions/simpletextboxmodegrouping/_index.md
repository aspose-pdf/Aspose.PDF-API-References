## HtmlSaveOptions.SimpleTextboxModeGrouping 속성

이 속성은 글리프와 단어를 문자열로 순차적으로 그룹화하는 것을 지정합니다. 예를 들어 태그와 단어는 변환된 HTML에서 다른 순서를 가지며, 이를 일치시키고자 할 때 사용됩니다. 이 매개변수는 FixedLayout 속성의 값이 true일 때만 문서에 적용됩니다.

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
