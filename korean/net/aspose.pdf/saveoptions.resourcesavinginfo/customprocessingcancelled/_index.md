## SaveOptions.ResourceSavingInfo.CustomProcessingCancelled 필드

이 플래그는 어떤 이유로 제안된 파일이 사용자 정의 코드가 아닌 변환기 코드 자체로 표준 변환기 방식으로 처리되어야 하는 경우 사용자 정의 코드에서 "true"로 설정해야 합니다. 따라서 true로 설정하면 사용자 정의 코드가 참조된 파일을 처리하지 않았으며 변환기가 이를 자체적으로 처리해야 함을 의미합니다(저장할 위치와 참조 파일의 이름 지정 두 가지 모두에 대해).

```csharp
public bool CustomProcessingCancelled;
```
