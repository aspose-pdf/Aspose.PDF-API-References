```csharp
public enum FileEncoding
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 파일이 압축되지 않음. |
| Zip | `1` | 파일이 ZIP 알고리즘으로 압축됨. |
