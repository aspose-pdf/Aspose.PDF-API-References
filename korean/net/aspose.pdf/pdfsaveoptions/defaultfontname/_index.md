## PdfSaveOptions.DefaultFontName 속성

컴퓨터에 없는 글꼴에 대해 기본적으로 사용되는 글꼴 이름. PDF로 저장된 PDF 문서에 문서 자체와 장치에서 사용할 수 없는 글꼴이 포함되어 있을 때, API는 장치에서 `DefaultFontName`이 발견되면 이 글꼴을 기본 글꼴로 교체합니다.

```csharp
public string DefaultFontName { get; set; }
```
