## ProgressEventType 열거형

이 열거형은 변환 중 발생할 수 있는 가능한 진행 이벤트 유형을 설명합니다.

```csharp
public enum ProgressEventType
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| TotalProgress | `0` | 발생한 이벤트가 변환의 전체 진행 상황에 대해 알림을 의미합니다. |
| SourcePageAnalysed | `1` | 발생한 이벤트가 변환 전에 페이지 중 하나의 분석이 끝났음을 알림을 의미합니다. |
| ResultPageCreated | `2` | 발생한 이벤트가 물리적 내보내기 전에 하나의 결과 페이지가 생성되었음을 알림을 의미합니다. |
| ResultPageSaved | `3` | 발생한 이벤트가 하나의 결과 페이지 저장이 완료되었음을 알림을 의미합니다. |
