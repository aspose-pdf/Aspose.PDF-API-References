## LineBreakStyle 열거형

파일에 대한 가능한 줄 바꿈 스타일을 나타냅니다.

```csharp
public enum LineBreakStyle
```

### 값들

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Windows | `0` | Windows 문자열 줄 바꿈 스타일(CRLF)을 선택합니다. |
| Unix | `1` | Unix 문자열 줄 바꿈 스타일(LF)을 선택합니다. OSX에도 사용됩니다. |
| Auto | `2` | 운영 체제에 따라 문자열 줄 바꿈 유형을 자동으로 선택합니다. |
