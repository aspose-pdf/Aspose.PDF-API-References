## PageLabelCollection.GetLabel 메서드

페이지 인덱스로 페이지 레이블을 가져옵니다 (페이지 인덱스는 0부터 시작합니다).

```csharp
public PageLabel GetLabel(int pageIndex)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| pageIndex | Int32 | 페이지의 인덱스. |

### 반환 값

지정된 페이지 인덱스에 대한 페이지 레이블 또는 페이지 레이블이 존재하지 않으면 null입니다.
