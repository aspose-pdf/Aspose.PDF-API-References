## Image.IsBlackWhite 속성

이미지가 흑백으로 강제 설정되었는지를 나타내는 bool 값을 가져오거나 설정합니다. CCITT 하위 형식의 TIFF 이미지가 사용되는 경우 이 속성은 true로 설정되어야 합니다.

```csharp
public bool IsBlackWhite { get; set; }
```
