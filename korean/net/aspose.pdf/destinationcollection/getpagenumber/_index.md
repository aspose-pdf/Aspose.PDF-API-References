## DestinationCollection.GetPageNumber 메서드

이름으로 목적지의 페이지 번호를 반환합니다.

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| destinameName | String | 목적지의 이름입니다. |
| useCache | Boolean | 캐시된 컬렉션 버리를 사용할지 여부를 결정합니다. |

### 반환 값

목적지가 발견되면 페이지 번호; 그렇지 않으면 -1입니다.
