## ExportFieldsToJsonOptions.WriteIndented 속성

Json 출력이 들여쓰기가 되어야 하는지를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool WriteIndented { get; set; }
```

### 속성 값

`true`이면 Json 출력이 들여쓰기가 되어야 하고, 그렇지 않으면 `false`입니다.
