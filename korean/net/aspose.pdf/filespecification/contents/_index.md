## FileSpecification.Contents 속성

내용 파일을 가져오거나 설정합니다. 이 속성은 메모리에 로드된 데이터를 반환하며, 대용량 데이터의 경우 메모리 부족 예외를 발생시킬 수 있습니다. 메모리 사용량을 줄이려면 StreamContents를 사용하십시오.

```csharp
public Stream Contents { get; set; }
```
