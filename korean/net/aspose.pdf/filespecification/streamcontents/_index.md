## FileSpecification.StreamContents 속성

파일의 내용을 스트림으로 가져옵니다. 내용은 메모리에 로드되지 않아 메모리 사용량을 줄일 수 있습니다. 그러나 이 스트림은 위치 지정 및 Length 속성을 지원하지 않습니다. 이러한 기능이 필요하면 대신 Contents 속성을 사용하십시오.

```csharp
public Stream StreamContents { get; }
```
