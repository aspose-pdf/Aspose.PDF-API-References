## OutputIntents.Contains 메서드

컬렉션에 특정 출력 의도가 포함되어 있는지 여부를 결정합니다.

```csharp
public bool Contains(OutputIntent item)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| item | OutputIntent | 컬렉션에서 찾을 출력 의도입니다. |

### 반환 값

`item`이 컬렉션에서 발견되면 `true`; 그렇지 않으면 `false`입니다.
