## PageNumber.GetPageNumberString 메서드

현재 설정에 따라 페이지 번호를 나타내는 형식화된 문자열을 반환합니다.

```csharp
public string GetPageNumberString(int pageNumber, int count)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| pageNumber | Int32 | 현재 페이지 번호. |
| count | Int32 | 총 페이지 수. |

### 반환 값

형식화된 페이지 번호 문자열.
