## FloatingBox.IsNeedRepeating 속성

단락이 다음 페이지에 반복되어야 하는지를 나타내는 bool 값을 가져오거나 설정합니다. 기본값은 false입니다. 이 속성은 단락 자체와 그 참조 단락 ID가 참조하는 객체가 모두 RepeatingRows에 포함될 때만 유효합니다.

```csharp
public bool IsNeedRepeating { get; set; }
```
