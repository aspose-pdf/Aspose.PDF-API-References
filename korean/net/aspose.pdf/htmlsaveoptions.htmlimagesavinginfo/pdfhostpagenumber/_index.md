## HtmlSaveOptions.HtmlImageSavingInfo.PdfHostPageNumber 필드

저장된 이미지가 원본 PDF 문서의 어떤 페이지에 해당하는지를 사용자 정의 코드에 알려줍니다. 원본 문서의 모든 페이지가 저장되지 않을 수 있으므로 이 값은 원본 PDF의 호스트 페이지 번호에 대해 알려줍니다. 원본 페이지 번호가 어떤 이유로 인해 알려지지 않은 경우 항상 '1'을 반환합니다.

```csharp
public int PdfHostPageNumber;
```
