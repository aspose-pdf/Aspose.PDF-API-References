## ExportFieldsOptions.ExportPasswordValue 속성

비밀번호 값을 내보낼지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool ExportPasswordValue { get; set; }
```

### 속성 값

`true` 비밀번호 값을 내보내야 하는 경우; 그렇지 않으면 `false`입니다.
