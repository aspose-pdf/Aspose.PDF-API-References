## PdfFormatConversionOptions.ExcludeFontsStrategy 속성

불필요한 글꼴을 제외하고 문서 파일 크기를 줄이는 전략. 이 매개변수는 플래그 [`OptimizeFileSize`](../optimizefilesize/)가 true로 설정된 경우에만 의미가 있습니다. 기본적으로 SubsetFonts와 RemoveDuplicatedFonts 전략의 조합이 사용됩니다.

```csharp
public RemoveFontsStrategy ExcludeFontsStrategy { get; set; }
```
