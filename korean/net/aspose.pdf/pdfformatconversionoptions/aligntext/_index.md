## PdfFormatConversionOptions.AlignText 속성

이 플래그는 변환된 문서의 텍스트 정렬을 제어합니다. 기본적으로 문서 변환은 텍스트 정렬에 영향을 미치지 않으며 텍스트를 그대로 둡니다. 그러나 경우에 따라 글꼴 대체로 인해 변환된 문서에서 텍스트가 겹치거나 여분의 공백이 발생할 수 있습니다. 이 플래그가 설정되면 특별한 정렬 작업이 수행됩니다. 이 플래그는 겹친 텍스트나 여분의 텍스트 공백 문제가 있는 문서에 대해서만 설정해야 하며, 이 플래그를 사용하면 성능이 저하되고 경우에 따라 텍스트 내용이 손상될 수 있습니다.

```csharp
public bool AlignText { get; set; }
```
