## PdfFormatConversionOptions.OptimizeFileSize 속성

PDF/A 문서를 파일 크기를 줄여서 얻기 위한 특별 변환 모드를 활성화/비활성화하는 플래그를 가져오거나 설정합니다. 현재 이 플래그는 PDF 문서에서 사용되는 글꼴의 최적화에 영향을 미치며, 미래에는 이 플래그가 그래픽과 같은 다른 데이터 구조의 최적화를 전환하는 데 사용될 수도 있습니다. 이 플래그와 모드를 설정하면 파일 크기를 상당히 줄일 수 있지만 동시에 변환 성능이 크게 감소할 수 있습니다.

```csharp
public bool OptimizeFileSize { get; set; }
```
