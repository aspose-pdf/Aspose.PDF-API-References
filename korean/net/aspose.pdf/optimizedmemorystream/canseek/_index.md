## OptimizedMemoryStream.CanSeek 속성

파생 클래스에서 재정의될 때 현재 스트림이 탐색을 지원하는지 여부를 나타내는 값을 가져옵니다.

```csharp
public override bool CanSeek { get; }
```

### 반환 값

스트림이 탐색을 지원하면 true; 그렇지 않으면 false입니다.
