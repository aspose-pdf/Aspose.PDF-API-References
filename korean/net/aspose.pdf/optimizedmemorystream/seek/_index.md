## OptimizedMemoryStream.Seek 메서드

파생 클래스에서 재정의될 때 현재 스트림 내의 위치를 설정합니다.

```csharp
public override long Seek(long offset, SeekOrigin origin)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| offset | Int64 | *origin* 매개변수에 대한 바이트 오프셋입니다. |
| origin | SeekOrigin | 새 위치를 얻기 위해 사용되는 기준점을 나타내는 SeekOrigin 유형의 값입니다. |

### 반환 값

현재 스트림 내의 새 위치입니다.
