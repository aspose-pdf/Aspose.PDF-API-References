## OptimizedMemoryStream.Write 메서드

파생 클래스에서 재정의될 때 현재 스트림에 바이트 시퀀스를 쓰고 작성된 바이트 수만큼 이 스트림 내에서 현재 위치를 이동합니다.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| buffer | Byte[] | 바이트 배열. 이 메서드는 *buffer*에서 현재 스트림으로 *count* 바이트를 복사합니다. |
| offset | Int32 | 현재 스트림으로 바이트 복사를 시작할 *buffer*의 0 기반 바이트 오프셋입니다. |
| count | Int32 | 현재 스트림에 작성될 바이트 수입니다. |
