## SaveOptions.MarginPartStyle.IsAuto 속성

이 인스턴스가 자동인지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool IsAuto { get; set; }
```

### 속성 값

`true` 이 인스턴스가 자동인 경우; 그렇지 않으면 `false`입니다.
