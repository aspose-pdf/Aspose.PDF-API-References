## SvgSaveOptions.CompressOutputToZipArchive 필드

출력이 하나의 zip 아카이브로 생성될지 여부를 지정합니다. 다중 페이지 소스 문서의 svg 파일 이름 규칙을 보려면 'TreatTargetFileNameAsDirectory' 옵션에 대한 주석을 참조하십시오. 이 규칙은 압축된 출력 파일 세트에도 적용됩니다.

```csharp
public bool CompressOutputToZipArchive;
```
