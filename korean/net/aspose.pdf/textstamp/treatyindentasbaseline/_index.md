## TextStamp.TreatYIndentAsBaseLine 속성

텍스트 배치를 위한 좌표 원점을 정의합니다. TreatYIndentAsBaseLine = true(그림이 true일 때 기본값)인 경우 YIndent 값은 텍스트 기준선으로 처리됩니다. TreatYIndentAsBaseLine = false(그림이 false일 때 기본값)인 경우 YIndent 값은 텍스트의 하단(하강선)으로 처리됩니다.

```csharp
public bool TreatYIndentAsBaseLine { get; set; }
```
