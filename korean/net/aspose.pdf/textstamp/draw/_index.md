## TextStamp.Draw 속성

이 속성은 페이지에 스탬프가 어떻게 그려지는지를 결정합니다. Draw = true이면 스탬프가 그래픽 연산자로 그려지고, draw = false이면 스탬프가 텍스트로 그려집니다.

```csharp
public bool Draw { get; set; }
```
