## UnifiedSaveOptions.ExtractOcrSublayerOnly 속성

이 속성은 OCR 서브 레이어가 있는 PDF 문서에서 이미지 또는 텍스트를 추출하는 기능을 활성화합니다.

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### 속성 값

`true` 결과 문서에서 텍스트가 추출됩니다; 그렇지 않으면 `false`입니다.
