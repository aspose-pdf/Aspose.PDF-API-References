## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace 속성

일부 글꼴은 일부 텍스트 기호에 대한 유니코드 정보를 제공하지 않습니다. 이 정보의 부족으로 인해 "텍스트를 유니코드로 매핑할 수 없습니다"라는 오류가 발생합니다. 이 플래그를 사용하여 유니코드 "공간"(코드 32)에서 비연결 기호를 매핑합니다.

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
