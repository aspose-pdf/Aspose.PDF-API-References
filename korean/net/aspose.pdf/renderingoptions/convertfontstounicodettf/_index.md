## RenderingOptions.ConvertFontsToUnicodeTTF 속성

모든 글꼴이 TTF 유니코드 버전으로 변환됨을 나타냅니다. 이는 호환성 이유와 글꼴 사용 최적화를 위해 유용합니다. 모든 새로운 TTF 글꼴은 원본 글꼴의 모든 기호를 포함하지 않고, 텍스트에서 사용되는 기호만 포함합니다.

```csharp
public bool ConvertFontsToUnicodeTTF { get; set; }
```
