## RenderingOptions.UseFontHinting 속성

이 플래그를 사용하면 글꼴 힌팅 메커니즘이 활성화됩니다. 글꼴 힌팅은 윤곽 글꼴의 표시를 조정하기 위해 수학적 지침을 사용하는 것입니다. 경우에 따라 이 플래그를 활성화하면 텍스트 가독성 문제를 해결할 수 있습니다. 현재 이 플래그의 사용은 소스 문서에서 이러한 글꼴이 사용되는 경우에만 TTF 글꼴에 효과를 줄 수 있습니다.

```csharp
public bool UseFontHinting { get; set; }
```
