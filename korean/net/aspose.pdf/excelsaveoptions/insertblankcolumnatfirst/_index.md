## ExcelSaveOptions.InsertBlankColumnAtFirst 속성

워크시트의 첫 번째 열로 빈 열을 삽입해야 하는 경우 true로 설정합니다. 기본값은 false이며, 이는 빈 열이 삽입되지 않음을 의미합니다.

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
