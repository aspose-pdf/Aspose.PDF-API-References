## ExcelSaveOptions.MinimizeTheNumberOfWorksheets 속성

결과 워크북에서 워크시트 수를 최소화해야 하는 경우 true로 설정합니다. 기본값은 false이며, 이는 각 PDF 페이지를 별도의 워크시트로 저장함을 의미합니다.

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
