## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber 필드

변환기에 의해 설정됩니다. SplitToPages 속성이 설정되면 변환 중에 여러 HTML 파일(변환된 페이지당 하나의 HTML 파일)이 생성됩니다. 이 속성은 저장된 HTML 페이지 파일의 순서를 포함합니다. 이 속성은 사용자 정의 코드의 논리에서 HTML 페이지를 처리하거나 저장할 위치를 결정하는 데 사용할 수 있으며, 페이지 분할이 꺼져 있으면 이 값은 항상 '1'을 포함합니다. 왜냐하면 이 경우 전체 원본 문서에 대해 하나의 큰 HTML 페이지만 생성되기 때문입니다.

```csharp
public int HtmlHostPageNumber;
```
