## HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber 필드

변환기에 의해 설정됩니다. SplitToPages 속성이 설정되면 변환 중에 여러 HTML 파일(변환된 페이지당 하나의 HTML 파일)이 생성됩니다. 이 속성은 사용자 정의 코드에 저장된 HTML 마크업이 원본 PDF의 어떤 페이지에서 생성되었는지를 알려줍니다. 원본 페이지 번호가 어떤 이유로 알 수 없거나 SplitOnPages=false인 경우, 이 속성은 항상 '0'을 포함하여 변환기가 제공된 HTML 마크업 파일에 대한 정확한 원본 PDF의 페이지 번호를 제공할 수 없음을 나타냅니다.

```csharp
public int PdfHostPageNumber;
```
