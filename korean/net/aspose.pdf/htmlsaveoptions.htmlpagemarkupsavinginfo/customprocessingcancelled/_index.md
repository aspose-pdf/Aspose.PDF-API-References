## HtmlSaveOptions.HtmlPageMarkupSavingInfo.CustomProcessingCancelled 필드

필요할 때 사용자 정의 코드에서 설정해야 합니다. 어떤 이유로 제공된 html-markup가 사용자 정의 코드가 아닌 변환기 코드 자체로 표준 변환기 방식으로 처리되어야 하는 경우 이 플래그는 사용자 정의 코드에서 "true"로 설정해야 합니다. 따라서 사용자 정의 코드에서 이 플래그를 설정하는 것은 사용자 정의 코드가 참조된 파일을 처리하지 않았으며 변환기가 이를 자체적으로 처리해야 함을 의미합니다.

```csharp
public bool CustomProcessingCancelled;
```
