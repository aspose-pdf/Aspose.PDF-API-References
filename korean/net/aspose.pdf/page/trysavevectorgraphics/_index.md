## Page.TrySaveVectorGraphics 메서드

페이지에 벡터 그래픽이 있는 경우 저장을 시도합니다. 저장 형식은 SVG입니다.

```csharp
public bool TrySaveVectorGraphics(string pathToSave)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| pathToSave | 문자열 | 출력 파일 |

### 반환 값

페이지에 경로 구성 연산자가 포함되어 있으면 True; 그렇지 않으면 False입니다.
