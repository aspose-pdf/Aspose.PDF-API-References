## Page.UserUnit 속성

UserUnit 값을 가져오거나 설정합니다. 기본 사용자 공간 단위의 크기를 1 / 72 인치의 배수로 나타내는 양수입니다. 기본값은 1입니다. 이 항목을 페이지에서 지우려면 0 또는 음수 값을 설정하십시오.

```csharp
public double UserUnit { get; set; }
```
