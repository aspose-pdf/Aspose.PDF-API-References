## Page.AddGraphics 메서드

페이지에 그래픽을 추가합니다. [`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/) 메서드를 사용하여 요소를 하나씩 추가하는 것보다 빠릅니다.

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| elements | GraphicElementCollection | 그래픽 컬렉션. |
| rectangle | Rectangle | 요소가 사각형 영역 내에 있을 경우 페이지에 추가됩니다. 사각형이 null이면 모든 그래픽 요소가 추가됩니다. |
