## Page.IsAddParagraphsAfterLast 속성

페이지의 마지막 단락 뒤에 단락 추가 여부를 가져오거나 설정합니다.

```csharp
public bool IsAddParagraphsAfterLast { get; set; }
```

### 속성 값

값은 페이지의 마지막 단락 뒤에 단락이 추가될지 여부를 나타냅니다. 값이 true이면 페이지의 마지막 단락 뒤에 단락이 추가됩니다.
