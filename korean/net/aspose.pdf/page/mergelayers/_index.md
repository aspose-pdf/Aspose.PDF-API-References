```csharp
public void MergeLayers(string newLayerName, string newOptionalContentGroupId)
```

| 매개변수 | 유형 | 설명 |

| newLayerName | String | 병합 후 새 레이어의 이름입니다. |
| newOptionalContentGroupId | String | 병합된 레이어의 선택적 콘텐츠 그룹 ID입니다. |
