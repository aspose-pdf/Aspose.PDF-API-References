## Page.IsBlank 메서드

페이지가 비어 있는지 여부를 나타내는 플래그를 가져옵니다.

```csharp
public bool IsBlank(double fillThresholdFactor)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| fillThresholdFactor | Double | 감지의 민감도를 관리하는 채우기 임계값입니다. 0.01 이상이어야 합니다. |

### 반환 값

True - 페이지가 비어 있으면; 그렇지 않으면 false입니다.
