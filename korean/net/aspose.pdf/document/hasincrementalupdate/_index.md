## Document.HasIncrementalUpdate 메서드

현재 PDF 문서가 증분 업데이트로 저장되었는지 확인합니다.

```csharp
public bool HasIncrementalUpdate()
```

### 반환 값

PDF 문서에 증분 업데이트가 있는 경우 `true`; 그렇지 않으면 `false`입니다.
