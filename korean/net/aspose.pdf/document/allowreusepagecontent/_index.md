## Document.AllowReusePageContent 속성

문서 크기를 최적화하기 위해 페이지 내용을 병합할 수 있습니다. 사용되는 경우 서로 다르지만 중복된 페이지가 동일한 콘텐츠 객체를 참조할 수 있습니다. 이 모드는 다른 페이지가 변경될 때 페이지 내용이 변경되는 등의 부작용을 일으킬 수 있습니다.

```csharp
public bool AllowReusePageContent { get; set; }
```
