## Document.IgnoreCorruptedObjects 속성

소스 파일에서 오류를 무시하는 플래그를 가져오거나 설정합니다. 소스 문서의 페이지가 대상 문서로 복사될 때, 이 플래그가 false인 경우 소스 파일의 일부 객체가 손상되면 복사 프로세스가 예외로 중단됩니다. 예: dest.Pages.Add(src.Pages); 이 플래그가 true로 설정되면 손상된 객체는 빈 값으로 대체됩니다. 기본값: true.

```csharp
public bool IgnoreCorruptedObjects { get; set; }
```
