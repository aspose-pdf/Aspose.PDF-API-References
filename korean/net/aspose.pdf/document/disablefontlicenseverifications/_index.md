## Document.DisableFontLicenseVerifications 속성

이 글꼴의 라이선스에 의해 이러한 작업이 금지된 경우 글꼴과 관련된 많은 작업을 실행할 수 없습니다. 예를 들어, 라이선스 규칙이 이 글꼴의 포함을 비활성화하면 일부 글꼴은 PDF 문서에 포함될 수 없습니다. 이 플래그는 현재 PDF 문서의 모든 글꼴에 대한 라이선스 제한을 비활성화하는 데 사용됩니다. 이 플래그를 사용할 때는 주의하십시오. 이 플래그가 설정되면 이 플래그를 설정한 사람이 가능한 라이선스/법률 위반에 대한 모든 책임을 지는 것을 의미합니다. 따라서 그는 자신의 위험을 감수합니다. 저작권법을 위반하지 않는다고 확신할 때만 이 플래그를 사용하는 것이 강력히 권장됩니다. 기본값은 false입니다.

```csharp
public bool DisableFontLicenseVerifications { get; set; }
```
