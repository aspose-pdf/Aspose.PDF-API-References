## Document.EmbedStandardFonts 속성

문서가 IsEmbedded 플래그가 true로 설정된 모든 표준 Type1 글꼴을 포함해야 함을 선언하는 속성입니다. 모든 PDF 글꼴은 IsEmbedded 플래그를 true로 설정하여 문서에 포함될 수 있지만, PDF 표준 Type1 글꼴은 이 규칙의 예외입니다. 표준 Type1 글꼴 포함은 많은 시간이 소요되므로, 이러한 글꼴을 포함하려면 지정된 글꼴에 대해 IsEmbedded 플래그를 true로 설정하는 것뿐만 아니라 문서 수준에서 추가 플래그인 EmbedStandardFonts = true를 설정해야 합니다. 이 속성은 모든 글꼴에 대해 한 번만 설정할 수 있습니다. 기본값은 false입니다.

```csharp
public bool EmbedStandardFonts { get; set; }
```
