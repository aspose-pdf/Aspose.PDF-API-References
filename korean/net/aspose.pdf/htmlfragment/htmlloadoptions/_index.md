## HtmlFragment.HtmlLoadOptions 속성

이 클래스의 인스턴스에 HTML을 로드(및 렌더링)하는 데 사용될 HtmlLoadOptions를 가져오거나 설정합니다. 이 인스턴스에 대해 HTML을 가져오기 위해 특정 설정을 사용해야 할 때 사용하십시오. 예를 들어, 이 인스턴스가 가져온 HTML에 대해 특정 BasePath를 사용해야 하거나 외부 리소스의 특정 로더를 사용해야 할 때입니다. 매개변수가 기본값(null)인 경우 표준 HTML 로딩 옵션이 사용됩니다.

```csharp
public HtmlLoadOptions HtmlLoadOptions { get; set; }
```
