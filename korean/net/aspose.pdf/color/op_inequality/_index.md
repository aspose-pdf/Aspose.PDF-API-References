## 색상 불일치 연산자

두 개의 색상이 같지 않으면 true를 반환합니다.

```csharp
public static bool operator !=(Color x, Color y)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| x | Color | 첫 번째 Color 객체. |
| y | Color | 두 번째 Color 객체. |

### 반환 값

Color 객체가 같지 않은 경우 true입니다.
