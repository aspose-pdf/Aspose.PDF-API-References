## SaveOptions.CacheGlyphs 속성

aps 페이지를 준비하는 동안 글꼴 글리프가 캐시될지 여부를 나타내는 부울 값을 가져오거나 설정합니다. PDF를 다른 형식으로 변환할 때 성능을 향상시키지만 메모리 소비를 증가시킵니다.

```csharp
public bool CacheGlyphs { get; set; }
```
