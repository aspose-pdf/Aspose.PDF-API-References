## SaveOptions.WarningHandler 속성

생성된 경고를 처리하기 위한 콜백. WarningHandler는 Continue 또는 Abort를 지정하는 ReturnAction 열거형 항목을 반환합니다. Continue는 기본 동작이며 Save 작업이 계속되지만 사용자가 Abort를 반환할 수도 있으며, 이 경우 Save 작업은 중단되어야 합니다.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
