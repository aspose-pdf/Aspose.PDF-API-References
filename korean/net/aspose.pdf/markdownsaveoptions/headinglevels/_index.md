## MarkdownSaveOptions.HeadingLevels 속성

FontSize 인식 헤더 전략에서 사용할 예상 헤딩 수준을 정의합니다. 이 속성 값이 설정되면, 문서에 북마크가 포함되어 있더라도 설정된 !:PdfToMarkdown.HeadingRecognitionStrategy.Auto 전략 대신 헤더 인식 !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic 전략이 선택됩니다.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
