## MarkdownSaveOptions.UseImageHtmlTag 속성

텍스트의 왼쪽과 오른쪽에 이미지를 삽입하기 위해 img 태그 사용 허용을 가져오고 설정합니다. 이 경우, 마크다운 뷰어에서 텍스트는 이미지 주위로 감쌀 것입니다.

```csharp
public bool UseImageHtmlTag { get; set; }
```
