## MarkdownSaveOptions.ResourcesDirectoryName 속성

문서 리소스(예: 이미지)를 저장할 디렉토리 이름을 가져오고 설정합니다. 값이 지정되지 않으면 이미지는 마크다운 파일 자체와 동일한 디렉토리에 기록됩니다. 이것은 경로가 아니라 이름일 뿐입니다! 이 디렉토리는 저장된 마크다운 파일이 있는 디렉토리에서 자동으로 생성됩니다.

```csharp
public string ResourcesDirectoryName { get; set; }
```
