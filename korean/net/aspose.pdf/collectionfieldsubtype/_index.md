| Desc | `5` | 필드 데이터는 파일 사양의 UF 항목에 의해 식별된 포함된 파일 스트림의 파일 이름이어야 하며, 존재하지 않을 경우 파일 사양의 F 항목에 의해 식별됩니다. |

| Size | `8` | 필드 데이터는 포함된 파일 매개변수 사전의 Size 항목에 의해 식별된 포함된 파일의 크기여야 합니다. |
