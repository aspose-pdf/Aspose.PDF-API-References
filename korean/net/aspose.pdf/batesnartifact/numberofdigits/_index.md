## BatesNArtifact.NumberOfDigits 속성

Bates 번호 매기기를 위한 숫자의 개수를 가져오거나 설정합니다. 값은 3에서 15 사이여야 합니다. 3보다 작은 값이 설정되면 3으로 조정됩니다. 15보다 큰 값이 설정되면 15로 조정됩니다. 기본값은 6입니다.

```csharp
public int NumberOfDigits { get; set; }
```
