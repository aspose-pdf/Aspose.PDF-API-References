## CollectionSchema.HasName 메서드

지정된 이름이 스키마에 존재하는지 여부를 결정합니다.

```csharp
public bool HasName(string name)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| name | 문자열 | 확인할 이름입니다. |

### 반환 값

지정된 이름이 스키마에 존재하면 `true`; 그렇지 않으면 `false`입니다.
