## Metadata.TryGetValue 메서드

사전에서 키를 찾으려고 시도하고 발견되면 값을 검색합니다.

```csharp
public bool TryGetValue(string key, out XmpValue value)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| key | String | 사전에서 검색할 키. |
| value | XmpValue& | 검색된 값. |

### 반환 값

키가 발견되면 true입니다.
