## HtmlSaveOptions.HtmlMarkupGenerationModes 열거형

때때로 생성된 HTML에 대한 특정 요구 사항이 존재합니다. 이 열거형은 PDF를 HTML로 변환하는 동안 이러한 특정 요구 사항에 맞게 사용할 수 있는 HTML 준비 모드를 정의합니다.

```csharp
public enum HtmlMarkupGenerationModes
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| WriteAllHtml | `0` | 특정 요구 사항이 없는 기본 모드입니다. 특별한 추가 처리 없이 모든 HTML 부분을 포함하는 출력이 생성됩니다. |
| WriteOnlyBodyContent | `1` | HTML의 본문 외부에 있는 모든 HTML 콘텐츠가 제거되며, 즉 \c \ \c \ 태그 내부에 있는 콘텐츠만 남게 됩니다. |
