## PdfANonSpecificationFlags 클래스

이 클래스는 소스 PDF 문서가 PDF 사양에 해당하지 않는 경우 PDF/A 변환을 제어하는 플래그를 보유합니다. 이 클래스의 플래그를 사용하면 성능이 저하되지만, 소스 PDF 문서를 일반적인 방법으로 PDF/A 형식으로 변환할 수 없을 때 필요합니다. 기본적으로 모든 플래그는 false로 설정됩니다.

