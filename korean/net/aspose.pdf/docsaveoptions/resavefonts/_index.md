## DocSaveOptions.ReSaveFonts 속성

글꼴을 다시 저장하는 절차를 가져오거나 설정합니다. true로 설정하면 이전 글꼴 속성의 영향을 피하기 위해 모든 페이지에서 글꼴을 다시 로드하고 새로 생성된 글꼴을 처음부터 로드합니다. 성능을 개선하려면 이 옵션을 false로 설정하십시오. 기본값은 true입니다;

```csharp
public bool ReSaveFonts { get; set; }
```
