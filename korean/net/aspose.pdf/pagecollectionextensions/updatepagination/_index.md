## PageCollectionExtensions.UpdatePagination 메서드

모든 페이지의 헤더 및 푸터 페이지 번호와 날짜를 업데이트합니다. 문서에 특수 설정 데이터가 있는 최소한 하나의 페이지 매김 아티팩트가 있는 경우에 작동합니다. 컬렉션의 모든 페이지는 해당 설정에 따라 소스 아티팩트로 업데이트됩니다.

```csharp
public static void UpdatePagination(this PageCollection pageCollection)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| pageCollection | PageCollection | 페이지 컬렉션. |
