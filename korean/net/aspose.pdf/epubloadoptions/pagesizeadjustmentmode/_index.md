## EpubLoadOptions.PageSizeAdjustmentMode 필드

주의! 이 기능은 구현되었지만 샘플 문서에서 발견된 OSHARED 레이어의 차단 문제로 인해 아직 공개 API에 포함되지 않았습니다. 변환 중 페이지 크기 사용 모드를 나타냅니다. 형식(HTML, EPUB 등)은 일반적으로 플로트 디자인을 가지고 있어 필요한 페이지 크기에 맞출 수 있습니다. 그러나 때때로 콘텐츠는 지정된 수평 위치나 크기를 가지고 있어 필요한 페이지 크기에 콘텐츠를 넣을 수 없습니다. 이러한 경우, 콘텐츠의 크기가 결과 PDF 문서의 초기 페이지 크기에 맞지 않을 때 무엇을 해야 하는지를 정의할 수 있습니다.

```csharp
public PageSizeAdjustmentModes PageSizeAdjustmentMode;
```
