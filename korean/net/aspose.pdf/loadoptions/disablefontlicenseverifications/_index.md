## LoadOptions.DisableFontLicenseVerifications 속성

파일을 로드하는 동안 모든 글꼴에 대한 라이선스 제한을 비활성화하는 플래그를 가져오거나 설정합니다. `true`일 때, 이 글꼴의 라이선스에 의해 금지된 글꼴로 작업을 실행할 수 있도록 허용합니다. 예를 들어, 이 글꼴에 대한 라이선스 규칙이 포함을 비활성화하더라도 PDF 문서에 글꼴을 포함할 수 있도록 허용합니다. 기본값은 `false`입니다.

```csharp
public bool DisableFontLicenseVerifications { get; set; }
```

## 비고

이 플래그를 사용할 때 주의하십시오. 이 플래그가 설정되면 이 플래그를 설정한 사람이 가능한 라이선스/법 위반에 대한 모든 책임을 지게 됩니다. 따라서 스스로 위험을 감수하는 것입니다. 저작권법을 위반하지 않는다고 확신할 수 있을 때만 이 플래그를 사용하는 것이 강력히 권장됩니다.
