## CollectionItem.IsEmpty 속성

컬렉션 항목이 비어 있는지 여부를 나타내는 값을 가져옵니다.

```csharp
public bool IsEmpty { get; }
```

### 반환 값

컬렉션 항목이 비어 있으면 true; 그렇지 않으면 false입니다.

## 비고

이 속성은 컬렉션 항목이 문자열 값, 이중 값, 정수 값 및 날짜 값을 포함하지 않을 경우 true를 반환합니다. 이러한 값 유형 중 하나라도 컬렉션 항목에 존재하면 이 속성은 false를 반환합니다.
