## Rectangle.ContainsLine 메서드

사각형이 두 점으로 표현된 선을 포함하는지 여부를 결정합니다.

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| x1 | Double | 선의 시작 점의 X 좌표. |
| y1 | Double | 선의 시작 점의 Y 좌표. |
| x2 | Double | 선의 끝 점의 X 좌표. |
| y2 | Double | 선의 끝 점의 Y 좌표. |

### 반환 값

사각형이 선을 포함하면 `true`; 그렇지 않으면 `false`입니다.
