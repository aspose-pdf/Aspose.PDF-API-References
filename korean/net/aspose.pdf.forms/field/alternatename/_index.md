## Field.AlternateName 속성

필드의 대체 이름을 가져오거나 설정합니다(사용자 인터페이스에서 필드가 식별될 때 실제 필드 이름 대신 사용될 대체 필드 이름). 대체 이름은 Adobe Acrobat에서 필드 툴팁으로 사용됩니다.

```csharp
public string AlternateName { get; set; }
```
