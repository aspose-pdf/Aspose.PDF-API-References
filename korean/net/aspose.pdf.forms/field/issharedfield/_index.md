## Field.IsSharedField 속성

생성기 지원을 위한 속성. 필드가 머리글 또는 바닥글에 추가될 때 사용됩니다. true인 경우 이 필드는 한 번 생성되며 문서의 모든 페이지에서 그 모습이 보입니다. false인 경우 각 문서 페이지에 대해 분리된 필드가 생성됩니다.

```csharp
public bool IsSharedField { get; set; }
```
