**Aspose.Pdf.Forms** 네임스페이스는 양식(표준, 정적, 동적) 및 텍스트 상자, 목록 상자, 라디오 버튼 등과 같은 다양한 유형의 필드를 설명하는 클래스를 포함하고 있습니다.

## 클래스

| 클래스 | 설명 |
| --- | --- |
| [BarcodeField](./barcodefield/) | 바코드 필드를 나타내는 클래스입니다. |
| [ButtonField](./buttonfield/) | 푸시 버튼 필드를 나타내는 클래스입니다. |
| [CheckboxField](./checkboxfield/) | 체크박스 필드를 나타내는 클래스입니다. |
| [ChoiceField](./choicefield/) | 선택 필드의 기본 클래스를 나타냅니다. |
| [ComboBoxField](./comboboxfield/) | 양식의 콤보박스 필드를 나타내는 클래스입니다. |
| [DateField](./datefield/) | 달력 보기가 있는 날짜 필드입니다. |
| [DocMDPSignature](./docmdpsignature/) | 문서 MDP(수정 감지 및 방지) 서명 유형의 클래스를 나타냅니다. |
| [ExternalSignature](./externalsignature/) | X509Certificate2를 사용하여 분리된 PKCS#7 서명을 생성합니다. USB 스마트 카드 및 내보낼 수 없는 개인 키가 있는 토큰을 지원합니다. |
| [Field](./field/) | 아크로 폼 필드의 기본 클래스입니다. |
| [FileSelectBoxField](./fileselectboxfield/) | 파일 선택 상자 요소에 대한 필드입니다. |
| [Form](./form/) | 양식 객체를 나타내는 클래스입니다. |
| [IconFit](./iconfit/) | 위젯 주석의 아이콘이 주석 사각형 내에서 어떻게 표시될지를 설명합니다. |
| [ListBoxField](./listboxfield/) | 목록 상자 필드를 나타내는 클래스입니다. |
| [NumberField](./numberfield/) | 지정된 유효 문자로 구성된 텍스트 필드입니다. |
| [Option](./option/) | 선택 필드의 옵션을 나타내는 클래스입니다. |
| [OptionCollection](./optioncollection/) | 선택 필드의 옵션 컬렉션을 나타내는 클래스입니다. |
| [PasswordBoxField](./passwordboxfield/) | 비밀번호 입력을 위한 텍스트 필드를 설명하는 클래스입니다. |
| [PKCS1](./pkcs1/) | PKCS#1 표준에 관한 서명 객체를 나타냅니다. 서명에는 RSA 암호화 알고리즘과 SHA-1 다이제스트 방법이 사용됩니다. |
| [PKCS7](./pkcs7/) | 인터넷 RFC 2315의 PKCS#7 사양에 부합하는 PKCS#7 객체를 나타냅니다. 문서의 바이트 범위에 대한 `SHA1 다이제스트`가 PKCS#7 SignedData 필드에 캡슐화됩니다. |
| [PKCS7Detached](./pkcs7detached/) | 인터넷 RFC 2315의 PKCS#7 사양에 부합하는 PKCS#7 객체를 나타냅니다. 문서의 바이트 범위에 대한 원본 서명 메시지 다이제스트가 일반 PKCS#7 SignedData 필드로 통합됩니다. PKCS#7 SignedData 필드에는 데이터가 캡슐화되지 않습니다. |
| [RadioButtonField](./radiobuttonfield/) | 라디오 버튼 필드를 나타내는 클래스입니다. |
| [RadioButtonOptionField](./radiobuttonoptionfield/) | 라디오 버튼 필드의 항목을 나타내는 클래스입니다. |
| [RichTextBoxField](./richtextboxfield/) | 리치 텍스트 편집기 구성 요소를 설명하는 클래스입니다. |
| [Signature](./signature/) | PDF 문서에서 서명 객체를 나타내는 추상 클래스입니다. 서명은 서명 객체의 값을 가진 필드이며, 마지막 필드는 문서 유효성을 검증하는 데 사용되는 데이터를 포함합니다. |
| [SignatureCustomAppearance](./signaturecustomappearance/) | 서명 사용자 정의 외관 객체를 나타내는 추상 클래스입니다. |
| [SignatureField](./signaturefield/) | 서명 양식 필드를 나타냅니다. |
| [SignHash](./signhash/) | 문서 해시를 사용자 정의 서명하기 위한 대리자입니다. |
| [TextBoxField](./textboxfield/) | 텍스트 상자 필드를 나타내는 클래스입니다. |
| [XFA](./xfa/) | XML 양식 아키텍처(XFA)에 관한 XML 양식을 나타냅니다. |
## 열거형

| 열거형 | 설명 |
| --- | --- |
| [BoxStyle](./boxstyle/) | 체크박스에서 체크를 그리기 위한 스타일을 나타냅니다. |
| [DocMDPAccessPermissions](./docmdpaccesspermissions/) | 이 문서에 부여된 접근 권한입니다. 유효한 값은 다음과 같습니다: 1 - 문서에 대한 변경이 허용되지 않으며, 문서에 대한 모든 변경은 서명을 무효화합니다. 2 - 허용된 변경은 양식 작성, 페이지 템플릿 인스턴스화 및 서명이며, 다른 변경은 서명을 무효화합니다. 3 - 허용된 변경은 2와 동일하며, 주석 생성, 삭제 및 수정도 포함됩니다. 다른 변경은 서명을 무효화합니다. |
