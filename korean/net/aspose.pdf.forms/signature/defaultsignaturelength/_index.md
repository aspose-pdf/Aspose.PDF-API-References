## Signature.DefaultSignatureLength 속성

서명 데이터의 기본 길이를 바이트 단위로 가져오거나 설정합니다.

```csharp
public int DefaultSignatureLength { get; set; }
```

## 비고

이는 바이트 단위로 서명의 길이에 대한 추정입니다. [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) 매개변수가 설정된 경우 [`CustomSignHash`](../customsignhash/)를 통해 서명하는 데 사용됩니다. 기본값은 3000입니다.
