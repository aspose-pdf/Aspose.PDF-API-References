## Form.SignaturesAppendOnly 속성

설정하면 문서에 서명이 포함되어 있으며, 파일이 이전 내용을 변경하는 방식으로 저장(작성)될 경우 무효화될 수 있습니다. 이는 점진적 업데이트와는 반대입니다.

```csharp
public bool SignaturesAppendOnly { get; set; }
```
