## Form.RemovePermission 속성

이 속성이 true이면 "Perms" 사전이 동적 문서를 표준으로 변환한 후 pdf 문서에서 제거됩니다. "Perms" 사전은 Adobe Acrobat 리더에서 필수 필드 선택 표시를 방해하는 규칙을 포함할 수 있습니다. 기본값은 false입니다.

```csharp
public bool RemovePermission { get; set; }
```
