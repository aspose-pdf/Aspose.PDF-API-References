## Form.IgnoreNeedsRendering 속성

이 속성이 true이면 NeedsRendering 키의 값은 XFA 양식을 표준 양식으로 변환하는 동안 무시됩니다. 기본값은 false입니다.

```csharp
public bool IgnoreNeedsRendering { get; set; }
```
