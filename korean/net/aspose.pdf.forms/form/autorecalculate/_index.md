## Form.AutoRecalculate 속성

설정하면 모든 양식 필드는 필드가 변경될 때 재계산됩니다. 기본값은 true입니다. 계산된 필드가 많은 양식을 작성할 때 성능을 높이기 위해 false로 설정하십시오.

```csharp
public bool AutoRecalculate { get; set; }
```
