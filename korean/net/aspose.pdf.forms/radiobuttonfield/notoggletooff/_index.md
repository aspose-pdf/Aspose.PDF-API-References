## RadioButtonField.NoToggleToOff 속성

라디오 버튼이 선택된 값이 없도록 허용하는 플래그를 가져오거나 설정합니다. `true`인 경우 항상 정확히 하나의 라디오 버튼이 선택되어야 하며, 현재 선택된 버튼을 선택하는 것은 아무런 효과가 없습니다. `false`인 경우 선택된 버튼을 클릭하면 선택 해제되어 선택된 버튼이 없게 됩니다.

```csharp
public bool NoToggleToOff { get; set; }
```

## 비고

일부 PDF 리더(Adobe Acrobat 포함)는 플래그의 `false` 상태를 무시할 수 있습니다.
