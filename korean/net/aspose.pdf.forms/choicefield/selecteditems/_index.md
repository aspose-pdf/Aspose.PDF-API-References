## ChoiceField.SelectedItems 속성

선택된 항목의 배열을 가져오거나 설정합니다. 다중 선택 목록의 경우 배열에는 둘 이상의 항목이 포함됩니다. 단일 선택 목록의 경우 단일 항목이 포함됩니다.

```csharp
public virtual int[] SelectedItems { get; set; }
```
