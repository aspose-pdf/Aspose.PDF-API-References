## MCRElement.ImageSrc 속성

방금 생성된 [`IllustrationElement`](../../illustrationelement/)에 대해. 다른 경우에는 null입니다.

```csharp
public string ImageSrc { get; }
```

### 속성 값

마크된 콘텐츠 참조 요소에 대한 이미지 소스입니다.
