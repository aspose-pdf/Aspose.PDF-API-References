## StructureTypeStandard.Warichu 필드

(Warichu; PDF 1.5) 더 작은 텍스트 크기로 주석 또는 주석을 달아 포함된 텍스트 줄의 높이 내에서 두 개의 더 작은 줄로 형식화되고 참조하는 기본 텍스트 뒤에 (인라인으로) 배치됩니다. Warichu 요소는 WT 및 WP 요소를 포함할 수도 있습니다.

(Warichu) 전체 warichu 어셈블리를 감싸는 래퍼입니다. WP, WT 및 WP로 구성된 세 요소 그룹을 포함할 수 있습니다. Warichu 요소(및 해당 콘텐츠 요소)는 일본 산업 표준(JIS) X 4051-1995에 설명된 warichu 줄 바꿈 규칙에 따라 여러 줄에 걸쳐 래핑될 수 있습니다.

```csharp
public static readonly StructureTypeStandard Warichu;
```
