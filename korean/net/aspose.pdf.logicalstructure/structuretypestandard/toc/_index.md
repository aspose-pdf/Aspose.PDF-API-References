## StructureTypeStandard.TOC 필드

(목차) 목차 항목 항목(구조 유형 TOCI) 및/또는 기타 중첩된 목차 항목(TOC)으로 구성된 목록입니다.

TOCI 항목만 포함하는 TOC 항목은 평면 계층 구조를 나타냅니다. 다른 중첩된 TOC 항목(및 가능하면 TOCI 항목)을 포함하는 TOC 항목은 더 복잡한 계층 구조를 나타냅니다. 이상적으로, 최상위 TOC 항목의 계층 구조는 문서 본문의 구조를 반영합니다.

```csharp
public static readonly StructureTypeStandard TOC;
```
