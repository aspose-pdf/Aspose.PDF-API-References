## StructureTypeStandard.RP 필드

(루비 구두점) 루비 주석 텍스트를 둘러싼 구두점. 루비 주석이 루비 스타일로 제대로 형식화될 수 없을 때만 사용되며, 대신 일반 주석으로 형식화되거나 와리치로 형식화될 때 사용됩니다. 일반적으로 단일 왼쪽 또는 오른쪽 괄호 또는 유사한 괄호 문자를 포함하는 텍스트를 포함합니다.

```csharp
public static readonly StructureTypeStandard RP;
```
