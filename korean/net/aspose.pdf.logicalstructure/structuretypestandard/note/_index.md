## StructureTypeStandard.Note 필드

(노트) 문서 본문에서 참조되는 각주 또는 미주와 같은 설명 텍스트 항목입니다. 자식으로 레이블(구조 유형 Lbl)을 가질 수 있습니다. 노트는 이를 참조하는 본문 텍스트의 구조 요소의 자식으로 포함되거나, 다른 곳(예: 미주 섹션)에 포함되어 참조(구조 유형 Reference)를 통해 접근할 수 있습니다.

태그가 있는 PDF는 페이지 콘텐츠 순서에서 각주의 배치를 규정하지 않습니다. 각주는 작성자의 재량에 따라 인라인 또는 페이지 끝에 있을 수 있습니다.

```csharp
public static readonly StructureTypeStandard Note;
```
