## ElementList 인덱서

주어진 인덱스에서 요소를 가져옵니다.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| 매개변수 | 설명 |
| --- | --- |
| index | 요소 목록의 인덱스입니다. |

### 반환 값

컬렉션에서 지정된 인덱스의 요소입니다. 인덱스가 목록의 요소 수보다 크거나 같으면 null을 반환합니다.
