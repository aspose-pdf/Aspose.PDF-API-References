## ITaggedContent.SetLanguage 메서드

PDF 문서의 자연어를 설정합니다.

문서의 모든 텍스트에 대한 자연어를 지정하는 언어 식별자로, 구조 요소 또는 표시된 콘텐츠에 대한 언어 사양으로 재정의되지 않는 한 적용됩니다.

```csharp
public void SetLanguage(string lang)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| lang | 문자열 | 언어 식별자는 언어가 알려지지 않음을 나타내기 위해 빈 텍스트 문자열이거나 RFC 3066에 정의된 언어 태그여야 합니다. |
