## OptimizationOptions.RemoveUnusedObjects 속성

이 플래그가 true로 설정되면 모든 문서 객체가 확인되고 사용되지 않는 객체(즉, 참조가 없는 객체)가 문서에서 제거됩니다.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
