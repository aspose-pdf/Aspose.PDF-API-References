## OptimizationOptions.LinkDuplcateStreams 속성

이 플래그가 true로 설정되면 리소스 스트림이 분석됩니다. 중복 스트림이 발견되면(즉, 스트림 내용이 동일한 경우) 이러한 스트림은 하나의 객체로 저장됩니다. 이는 동일한 문서가 여러 번 연결된 경우와 같이 일부 경우에 문서 크기를 줄이는 데 도움이 됩니다.

```csharp
public bool LinkDuplcateStreams { get; set; }
```
