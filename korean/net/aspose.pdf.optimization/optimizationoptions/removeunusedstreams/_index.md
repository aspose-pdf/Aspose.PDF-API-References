## OptimizationOptions.RemoveUnusedStreams 속성

이 플래그가 true로 설정되면 모든 리소스의 사용 여부가 확인됩니다. 리소스가 사용되지 않으면 리소스가 제거됩니다. 이는 문서에서 페이지가 추출될 때와 같이 문서 크기를 줄일 수 있습니다.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
