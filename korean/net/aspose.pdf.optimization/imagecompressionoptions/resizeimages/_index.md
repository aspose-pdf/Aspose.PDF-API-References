## ImageCompressionOptions.ResizeImages 속성

이 플래그가 true로 설정되고 CompressImages가 true이면 이미지 해상도가 지정된 MaxResolution 매개변수보다 클 경우 이미지가 크기 조정됩니다.

```csharp
public bool ResizeImages { get; set; }
```
