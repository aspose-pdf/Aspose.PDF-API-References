## SideBySideComparisonOptions.AdditionalChangeMarks 속성

추가 변경 마커가 표시되는지 여부를 결정하는 속성을 가져오고 설정합니다. 설정하면 현재 페이지에 없지만 다른 페이지에 있는 변경 마크가 표시됩니다. 변경이 단어 사이에 위치하는 경우, 마크가 공백 문자에 대해 정확하게 위치하지 않을 수 있습니다. 기본값은 `false`입니다.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
