## ComparisonOptions.ExcludeAreas2 속성

제외 영역을 가져오고 설정합니다. 비교 방법에서 두 번째 페이지 또는 문서에 사용됩니다. 이 옵션은 [`ExcludeTables`](../excludetables/)와 함께 설정할 수 있습니다. 이 옵션은 [`ExtractionArea`](../extractionarea/) 옵션과 함께 설정할 수 없습니다.

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
