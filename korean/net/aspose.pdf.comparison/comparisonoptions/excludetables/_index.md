## ComparisonOptions.ExcludeTables 속성

비교에서 테이블이 제외되는지 여부를 결정하는 옵션을 가져오고 설정합니다. 이 옵션은 [`ExtractionArea`](../extractionarea/) 옵션과 함께 설정할 수 없습니다. 기본값은 `false`입니다.

```csharp
public bool ExcludeTables { get; set; }
```
