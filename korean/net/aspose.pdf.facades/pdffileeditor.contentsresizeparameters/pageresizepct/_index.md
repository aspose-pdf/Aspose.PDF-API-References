## PdfFileEditor.ContentsResizeParameters.PageResizePct 메서드

페이지 크기 조정을 위한 크기 매개변수를 생성합니다. 새로운 크기는 백분율로 지정됩니다.

```csharp
public static ContentsResizeParameters PageResizePct(double widthPct, double heightPct)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| widthPct | Double | 새로운 페이지 너비(백분율). |
| heightPct | Double | 새로운 페이지 높이(백분율). |

### 반환 값

새로운 크기 매개변수.
