## PdfFileEditor.ContentsResizeParameters.MarginsPercent 메서드

크기 조정 매개변수를 생성합니다. 여백은 초기 페이지 크기의 백분율로 지정됩니다.

```csharp
public static ContentsResizeParameters MarginsPercent(double left, double right, double top, 
    double bottom)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| left | Double | 왼쪽 여백 (페이지 너비의 백분율). |
| right | Double | 오른쪽 여백 (페이지 높이의 백분율). |
| top | Double | 위쪽 여백 (페이지 높이의 백분율). |
| bottom | Double | 아래쪽 여백 (페이지 높이의 백분율). |

### 반환 값

새로운 크기 조정 매개변수를 반환합니다.
