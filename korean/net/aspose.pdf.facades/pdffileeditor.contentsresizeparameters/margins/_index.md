## PdfFileEditor.ContentsResizeParameters.Margins 메서드

지정된 여백 값으로 크기 조정 매개변수를 생성합니다. 내용 크기는 자동으로 계산됩니다.

```csharp
public static ContentsResizeParameters Margins(double left, double right, double top, double bottom)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| left | Double | 왼쪽 여백. |
| right | Double | 오른쪽 여백. |
| top | Double | 위쪽 여백. |
| bottom | Double | 아래쪽 여백. |

### 반환 값

생성된 크기 조정 매개변수.
