## PdfFileEditor.ContentsResizeParameters.ContentSizePercent 메서드

```csharp
public static ContentsResizeParameters ContentSizePercent(double width, double height)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| width | Double | 백분율로 새로운 콘텐츠 너비. |
| height | Double | 백분율로 새로운 콘텐츠 높이. |

### 반환 값

새로운 크기 조정 매개변수.
