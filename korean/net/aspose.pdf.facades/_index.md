**Aspose.Pdf.Facades** 네임스페이스는 원래 Aspose.Pdf.Kit에서 온 클래스들을 제공합니다. 이 클래스들은 문서를 조작하는 데 사용되며, 문서의 내부 구조에 접근하지 않고도 연결, 스탬핑, 서명, 주석 달기 등의 작업을 수행합니다.

## 클래스

| 클래스 | 설명 |
| --- | --- |
| [AutoFiller](./autofiller/) | 데이터베이스 또는 다른 데이터 소스에서 데이터를 수신하고, 이를 템플릿 PDF의 설계된 필드에 채운 후, 새로운 PDF 파일 또는 스트림을 생성하는 클래스를 나타냅니다. 두 가지 템플릿 파일 입력 모드가 있습니다: 스트림으로 입력하거나 PDF 파일로 입력합니다. 네 가지 출력 모드가 있습니다: 하나의 병합된 스트림, 하나의 병합된 파일, 여러 개의 작은 스트림, 여러 개의 작은 파일. System.Data.DataTable에 포함된 리터럴 데이터를 수신할 수 있습니다. |
| [BDCProperties](./bdcproperties/) | BDC 연산자 속성입니다. |
| [Bookmark](./bookmark/) | 북마크를 나타냅니다. |
| [Bookmarks](./bookmarks/) | [`Bookmark`](../aspose.pdf.facades/bookmark/) 객체의 컬렉션을 나타냅니다. |
| [DocumentPrivilege](./documentprivilege/) | PDF 파일에 접근하기 위한 권한을 나타냅니다. [`PdfFileSecurity`](../aspose.pdf.facades/pdffilesecurity/)를 참조하십시오. 이 클래스를 사용하는 방법은 4가지입니다: 1. 미리 정의된 권한을 직접 사용합니다. 2. 미리 정의된 권한을 기반으로 특정 권한을 변경합니다. 3. 미리 정의된 권한을 기반으로 특정 Adobe Professional 권한 조합을 변경합니다. 4. 방법 2와 방법 3을 혼합합니다. |

| [FormDataConverter](./formdataconverter/) | 한 형식에서 다른 형식으로 데이터를 변환하는 클래스를 나타냅니다. fdf/xml/pdf/xfdf의 데이터를 OLEDB/OdbcDB로 변환할 수 있습니다. 또한 OLEDB/OdbcDB의 데이터를 fdf/xml/xfdf의 데이터로 변환할 수 있습니다. "하드 이름" 태그로 fdf를 xml로 변환할 수 있습니다. |
| [FormEditor](./formeditor/) | 양식을 편집하는 클래스입니다 (필드 추가/삭제 등). |

| [PdfFileSanitization](./pdffilesanitization/) | 정화 및 복구 API를 나타냅니다. 다른 방법으로 문서를 생성/열 수 없는 경우 사용하십시오. |
| [PdfFileSecurity](./pdffilesecurity/) | 소유자 또는 사용자 비밀번호로 PDF 파일을 암호화하거나 복호화하고, 보안 설정 및 비밀번호를 변경하는 클래스를 나타냅니다. |
