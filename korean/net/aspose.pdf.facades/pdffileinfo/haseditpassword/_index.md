## PdfFileInfo.HasEditPassword 속성

권한 또는 문서 보안 속성을 수정하는 데 비밀번호가 필요한 경우 true를 반환합니다. 이 속성은 [`PdfFileInfo`](../) 생성자에 유효한 비밀번호가 제공된 경우에만 읽을 수 있습니다. PasswordType이 Inaccessible(즉, 잘못된 비밀번호가 제공된 경우)인 경우 이 속성을 읽으면 [`InvalidPasswordException`](../../../aspose.pdf/invalidpasswordexception/)이 발생합니다.

```csharp
public bool HasEditPassword { get; }
```
