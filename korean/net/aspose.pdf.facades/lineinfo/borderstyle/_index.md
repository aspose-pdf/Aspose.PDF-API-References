## LineInfo.BorderStyle 속성

선의 테두리 스타일을 가져오거나 설정합니다. 0은 실선을 나타내고, 1은 점선을 나타내며, 2는 베벨을 나타내고, 3은 삽입을 나타내고, 4는 밑줄을 나타냅니다.

```csharp
public int BorderStyle { get; set; }
```
