## PdfExtractor.IsBidi 속성

텍스트에 히브리어 또는 아랍어 기호가 있을 때 true입니다. 이 경우는 문자열 함수가 동작을 변경하고 숫자 및 기타 비텍스트 문자를 제외하고 텍스트를 오른쪽에서 왼쪽으로 처리하기 시작하므로 특별히 고려해야 합니다.

```csharp
public bool IsBidi { get; }
```
