## PdfExtractor.Resolution 속성

추출된 이미지의 해상도를 설정하거나 가져옵니다. 기본값은 150입니다. 해상도 값이 더 높은 이미지는 더 선명합니다. 그러나 해상도 값을 증가시키면 이미지를 추출하는 데 필요한 시간과 메모리가 증가합니다. 일반적으로 선명한 이미지를 얻으려면 해상도를 150 또는 300으로 설정하는 것으로 충분합니다.

```csharp
public int Resolution { get; set; }
```
