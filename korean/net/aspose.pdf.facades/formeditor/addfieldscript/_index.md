## FormEditor.AddFieldScript 메서드

PushButton 필드에 JavaScript를 추가합니다. 이전 이벤트가 존재하는 경우 새 이벤트가 그 뒤에 추가됩니다.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| fieldName | String | 완전한 필드 이름입니다. |
| script | String | PushButton 필드에 추가/배치할 Java 스크립트입니다. |

### 반환 값

스크립트가 성공적으로 추가된 경우 True입니다.
