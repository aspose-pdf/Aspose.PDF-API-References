## FormEditor.SetFieldScript 메서드

PushButton 필드에 대한 JavaScript를 설정합니다. 이전 JavaScript가 존재하는 경우 새 것으로 대체됩니다.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| fieldName | String | 완전한 필드 이름입니다. |
| script | String | PushButton 필드에 추가/배치할 Java 스크립트입니다. |

### 반환 값

필드 스크립트가 성공적으로 설정되면 true입니다.
