## DocumentPrivilege.CopyAllowLevel 속성

문서의 권한 복사 수준을 가져오고 설정합니다. Adobe Professional의 권한 설정과 같습니다. 0: 없음. 1: 시각 장애인을 위한 화면 읽기 장치에 대한 텍스트 접근을 활성화합니다. 2: 텍스트, 이미지 및 기타 콘텐츠 복사를 활성화합니다.

```csharp
public int CopyAllowLevel { get; set; }
```

## 비고

속성의 값이 -1인 경우 수준이 정의되지 않습니다.
