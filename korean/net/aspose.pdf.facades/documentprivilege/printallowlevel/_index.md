## DocumentPrivilege.PrintAllowLevel 속성

문서 권한의 인쇄 수준을 가져오고 설정합니다. Adobe Professional의 Printing Allowed 설정과 같습니다. 0: 없음. 1: 낮은 해상도 (150 dpi). 2: 높은 해상도.

```csharp
public int PrintAllowLevel { get; set; }
```

## 비고

속성의 값이 -1이면 수준이 정의되지 않습니다.
