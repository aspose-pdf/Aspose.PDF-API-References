## AutoFiller.BasicFileName 속성

많은 작은 파일이 생성될 경우 기본 파일 이름을 가져오거나 설정합니다. 생성된 파일은 "BasicFileName0", "BasicFileName1",...와 같은 형식이 됩니다. 다른 속성 [`GeneratingPath`](../generatingpath/)GeneratingPath와 함께 작동합니다.

```csharp
public string BasicFileName { get; set; }
```
