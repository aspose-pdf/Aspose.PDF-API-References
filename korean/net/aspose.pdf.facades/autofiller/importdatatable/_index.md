## AutoFiller.ImportDataTable 메서드

DataTable 유형의 데이터를 가져옵니다. 데이터 테이블의 모든 열 이름은 대소문자를 구분하여 템플릿 PDF의 하나의 필드 이름과 같아야 합니다.

```csharp
public void ImportDataTable(DataTable dataTable)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| dataTable | DataTable | 데이터베이스 또는 기타 데이터 소스에서 모든 데이터를 포함하는 데이터 테이블입니다. |
