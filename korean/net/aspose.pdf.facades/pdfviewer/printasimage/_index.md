## PdfViewer.PrintAsImage 속성

PdfViewer가 이미지를 인쇄하는 모드를 설정하거나 가져옵니다.

```csharp
public bool PrintAsImage { get; set; }
```

## 비고

true인 경우 항상 이미지를 인쇄합니다(인쇄되는 이미지를 생성). false인 경우 모든 기능이 지원되는 경우 장치에 직접 인쇄합니다. 문서에 지원되지 않는 기능이 포함된 경우 시스템이 자동으로 이미지를 인쇄하도록 결정할 수 있습니다. 기본 값은 false입니다.
