## PdfFileEditor.KeepFieldsUnique 속성

true인 경우 양식이 연결될 때 필드 이름이 고유하게 만들어집니다. 필드 이름에 접미사가 추가되며, 접미사 템플릿은 UniqueSuffix 속성에서 지정할 수 있습니다.

```csharp
public bool KeepFieldsUnique { get; set; }
```
