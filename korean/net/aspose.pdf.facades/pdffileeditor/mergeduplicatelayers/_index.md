## PdfFileEditor.MergeDuplicateLayers 속성

이 속성이 true인 경우 동일한 이름을 가진 연결된 문서의 선택적 내용이 결과 문서의 하나의 레이어로 병합됩니다. 그렇지 않으면 동일한 이름을 가진 레이어는 결과 문서에서 서로 다른 레이어로 저장됩니다.

```csharp
public bool MergeDuplicateLayers { get; set; }
```
