## PdfToDocOptions.ConversionMode 속성

PDF 문서가 워드 프로세싱 문서로 변환되는 방식을 제어할 수 있습니다.

```csharp
public ConversionMode ConversionMode { get; set; }
```

## 비고

결과 문서가 더 이상 많이 수정되지 않을 경우 TextBox 모드를 사용하십시오. 수정할 것이 많지 않을 때 텍스트 박스는 수정하기 쉽습니다.

출력 문서가 추가 편집이 필요할 경우 Flow 모드를 사용하십시오. Flow 모드의 단락 및 텍스트 줄은 텍스트 수정이 용이하지만, 지원되지 않는 형식 객체는 TextBox 모드보다 더 나쁘게 보일 것입니다.
