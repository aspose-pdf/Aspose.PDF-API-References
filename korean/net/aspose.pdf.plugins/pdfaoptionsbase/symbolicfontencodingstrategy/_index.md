## PdfAOptionsBase.SymbolicFontEncodingStrategy 속성

PDF/A 형식으로 변환할 때 기호 글꼴 인코딩 전략을 가져오거나 설정합니다.

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### 속성 값

기호 글꼴 인코딩 전략.

## 비고

이 속성을 사용하면 원본 문서의 TrueType 기호 글꼴에 여러 CMap 서브테이블이 포함된 경우 결과 문서에 복사될 CMap 서브테이블을 제어할 수 있습니다.
