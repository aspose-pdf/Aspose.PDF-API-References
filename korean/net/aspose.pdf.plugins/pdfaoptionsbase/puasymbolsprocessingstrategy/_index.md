## PdfAOptionsBase.PuaSymbolsProcessingStrategy 속성

PDF 문서에서 개인 사용 영역 (PUA) 기호를 처리하기 위한 전략을 가져오거나 설정합니다.

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### 속성 값

PUA 기호를 처리하기 위한 전략입니다. 기본값은 Level B 적합성 문서의 경우 None이며, Level A 적합성 문서의 경우 SurroundPuaTextWithEmptyActualText입니다.
