## PdfAOptionsBase.OptimizeFileSize 속성

PDF/A 변환 과정에서 파일 크기를 줄이려고 시도할지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool OptimizeFileSize { get; set; }
```

### 속성 값

`true`는 파일 크기를 가능한 한 많이 줄여야 함을 나타내고, 그렇지 않으면 `false`입니다.

## 비고

`true`로 설정하면 변환 과정에서 결과 파일 크기를 최소화하려고 시도합니다. 이는 변환 과정의 성능에 영향을 미칠 수 있습니다.
