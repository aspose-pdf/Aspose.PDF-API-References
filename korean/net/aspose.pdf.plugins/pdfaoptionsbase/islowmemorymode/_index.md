## PdfAOptionsBase.IsLowMemoryMode 속성

PDF/A 변환 과정에서 저메모리 모드가 활성화되어 있는지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool IsLowMemoryMode { get; set; }
```

### 속성 값

`true`는 저메모리 모드가 활성화되어 있음을 의미합니다; 그렇지 않으면 `false`입니다.
