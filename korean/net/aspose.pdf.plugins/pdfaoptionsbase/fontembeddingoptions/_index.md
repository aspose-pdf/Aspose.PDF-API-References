## PdfAOptionsBase.FontEmbeddingOptions 속성

문서에 포함할 수 없는 글꼴을 처리하는 옵션을 가져옵니다.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### 속성 값

글꼴 포함 옵션.

## 비고

PDF/A 표준은 모든 글꼴이 문서에 포함되어야 한다고 요구합니다. 이 속성은 일부 글꼴을 포함할 수 없는 경우를 처리하기 위한 옵션을 제공합니다. 이는 대상 PC에 글꼴이 없기 때문입니다.
