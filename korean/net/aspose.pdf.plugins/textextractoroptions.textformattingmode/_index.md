## TextExtractorOptions.TextFormattingMode 열거형

PDF 문서를 텍스트로 변환할 때 사용할 수 있는 다양한 모드를 정의합니다. [`TextExtractorOptions`](../textextractoroptions/) 클래스를 참조하십시오.

```csharp
public enum TextFormattingMode
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Pure | `0` | 몇 가지 형식화 루틴을 사용하여 PDF 텍스트 콘텐츠를 나타냅니다. 여기에는 상대 위치를 고려하고 텍스트를 페이지 너비에 맞추기 위해 추가 공백을 추가하는 것이 포함됩니다. |
| Raw | `1` | 형식 없이 PDF 콘텐츠를 있는 그대로 나타냅니다. |
| Plain | `2` | 텍스트 조각의 상대적 위치를 고려하되 추가 공백을 추가하지 않고 PDF 텍스트 콘텐츠를 나타냅니다. |
