## EncryptionOptions 생성자

기본 옵션으로 [`EncryptionOptions`](../) 객체의 새 인스턴스를 초기화합니다.

```csharp
public EncryptionOptions(string ownerPassword, string userPassword, 
    DocumentPrivilege documentPrivilege, CryptoAlgorithm cryptoAlgorithm = CryptoAlgorithm.AESx256)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| ownerPassword | 문자열 | 소유자 비밀번호. |
| userPassword | 문자열 | 사용자 비밀번호. |
| documentPrivilege | DocumentPrivilege | 문서 권한. |
| cryptoAlgorithm | CryptoAlgorithm | 암호화 알고리즘. |
