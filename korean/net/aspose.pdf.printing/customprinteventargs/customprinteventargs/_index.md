## CustomPrintEventArgs 생성자

주어진 프린터 및 페이지 설정으로 [`CustomPrintEventArgs`](../)를 초기화합니다.

```csharp
public CustomPrintEventArgs(string fileName, PrinterSettings printerSettings, 
    PageSettings pageSettings)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| fileName | String | 인쇄 중인 파일의 이름입니다. |
| printerSettings | PrinterSettings | 문서가 인쇄될 프린터에 대한 정보입니다. |
| pageSettings | PageSettings | 문서의 각 페이지에 적용되어야 하는 설정입니다. |
