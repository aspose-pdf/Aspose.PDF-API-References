## ParagraphAbsorberOptions.SectionUnbreakingVerticalOverride 속성

'제로 채움 수준'의 최대 길이를 가져오거나 설정합니다. 이는 실제 수직 섹션 구분으로 간주되지 않으며(페이지의 너비/높이에 대한 비율로). 기본값은 0.005입니다.

```csharp
public double SectionUnbreakingVerticalOverride { get; set; }
```
