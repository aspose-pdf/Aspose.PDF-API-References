## TextExtractionOptions.ScaleFactor 속성

순수 모드에서 추출하는 동안 글꼴 크기를 조정하는 데 적용될 계수를 가져오거나 설정합니다. 값이 낮을수록 추출된 텍스트에 더 많은 공백이 생깁니다. 기본값은 1 - 스케일링 없음; 값을 0으로 설정하면 알고리즘이 자동으로 스케일링을 선택할 수 있습니다.

```csharp
public double ScaleFactor { get; set; }
```
