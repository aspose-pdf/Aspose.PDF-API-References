## TextEditOptions.ClippingPathsProcessingMode 열거형

클리핑 경로 처리 모드

```csharp
public enum ClippingPathsProcessingMode
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| KeepIntact | `0` | 원본 페이지 레이아웃의 클리핑 경로를 유지합니다. (기본값) |
| Expand | `1` | 편집된 텍스트가 더 많은 공간을 요구하는 경우 원본 클리핑 경로가 확장됩니다. |
| Remove | `2` | 편집된 텍스트가 더 많은 공간을 요구하는 경우 원본 클리핑 경로가 제거됩니다. 주의: 클리핑 경로가 서로 상호작용할 수 있으므로 제거 시 페이지 레이아웃에 예기치 않은 결과를 초래할 수 있습니다. |
