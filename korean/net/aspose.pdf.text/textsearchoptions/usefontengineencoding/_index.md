## TextSearchOptions.UseFontEngineEncoding 속성

텍스트가 글꼴 엔진 인코딩을 사용하여 검색될 것임을 나타내거나 설정합니다. true - 글꼴 엔진 인코딩이 사용됨을 의미합니다(문서의 불완전한 인코딩으로 인해 텍스트 검색이 실패하는 경우 시도하십시오). false - 문서 글꼴 인코딩이 사용됨을 의미합니다(기본값).

```csharp
public bool UseFontEngineEncoding { get; set; }
```
