## TextSearchOptions.SearchInAnnotations 속성

주석에서 텍스트를 검색할 수 있는 값을 가져오거나 설정합니다. true - 텍스트가 주석에서 검색됩니다. false - 주석의 텍스트는 TextFragmentAbsorber에 의해 구문 분석되지 않습니다.

```csharp
public bool SearchInAnnotations { get; set; }
```
