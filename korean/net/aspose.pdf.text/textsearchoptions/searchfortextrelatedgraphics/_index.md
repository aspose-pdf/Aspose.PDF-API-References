## TextSearchOptions.SearchForTextRelatedGraphics 속성

텍스트 검색 중 텍스트 관련 그래픽(강조, 배경 등)을 검색할 수 있는 값을 가져오거나 설정합니다. true - 텍스트 관련 그래픽 검색이 수행됩니다(기본값). false - 원본 문서에 존재할 수 있는 그래픽 요소는 무시됩니다. 성능 문제나 강조, 배경 또는 클리핑을 처리할 필요가 없는 경우 이 값을 설정하십시오.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
