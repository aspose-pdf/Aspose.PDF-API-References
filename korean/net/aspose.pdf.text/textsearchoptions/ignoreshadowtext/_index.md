## TextSearchOptions.IgnoreShadowText 속성

일반 텍스트의 그림자를 나타내는 텍스트 조각이 검색 중에 무시될 것인지에 대한 표시를 가져오거나 설정합니다. true - 그림자 텍스트가 발견되지 않음을 의미합니다(가까운 위치에서 텍스트 검색이 중복된 조각을 반환하는 경우 시도해 보세요). false - 그림자 텍스트가 일반 텍스트와 함께 발견됨을 의미합니다(기본값).

```csharp
public bool IgnoreShadowText { get; set; }
```
