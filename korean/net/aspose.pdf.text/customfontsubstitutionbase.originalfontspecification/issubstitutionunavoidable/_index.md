## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable 속성

대체가 불가피하다는 것을 나타내는 값을 가져옵니다.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

## 비고

원본 글꼴이 없거나 원본 글꼴을 특정 작업의 맥락에서 사용할 수 없는 경우 대체가 요청된 경우 true를 반환합니다. 사용자가 플래그를 무시하고 글꼴을 대체하지 않으면 기본 글꼴 대체 절차가 수행됩니다. 그러나 사용자가 표준 글꼴 대체 절차를 변경하고 시스템에 더 나은 글꼴을 설정할 수 있는 기회를 제공합니다. 원본 글꼴이 존재하고 유효하지만 사용자가 대체할 수 있도록 허용된 경우 false를 반환합니다.
