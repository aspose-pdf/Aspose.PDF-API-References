## AbsorbedRow.CompareTo 메서드

현재 AbsorbedRow 객체를 다른 AbsorbedRow 객체와 비교하고 현재 객체가 다른 객체보다 앞서는지, 뒤지는지 또는 정렬 순서에서 같은 위치에 있는지를 나타내는 정수를 반환합니다.

```csharp
public int CompareTo(AbsorbedRow other)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| other | AbsorbedRow | 이 객체와 비교할 AbsorbedRow 객체입니다. |

### 반환 값

비교되는 객체의 상대적인 순서를 나타내는 값입니다. 반환 값은 다음과 같은 의미를 가집니다: -1: 이 객체는 정렬 순서에서 다른 객체보다 앞섭니다. 0: 이 객체는 정렬 순서에서 다른 객체와 같은 위치에 있습니다. 1: 이 객체는 정렬 순서에서 다른 객체 뒤에 있습니다.
