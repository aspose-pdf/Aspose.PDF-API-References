## MarkupSection.Fragments 속성

섹션 내에 있는 비어 있지 않은 [`TextFragment`](../../textfragment/) 객체의 모음입니다.

```csharp
public List<TextFragment> Fragments { get; }
```

## 비고

[`TextFragment`](../../textfragment/) 객체는 검색 발생 텍스트, 텍스트 속성에 대한 접근을 제공하며, 텍스트를 편집하고 텍스트 상태(글꼴, 글꼴 크기, 색상 등)를 변경할 수 있습니다.
