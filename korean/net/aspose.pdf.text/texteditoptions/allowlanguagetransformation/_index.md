## TextEditOptions.AllowLanguageTransformation 속성

텍스트를 추가하거나 편집할 때 언어 변환 사용을 허용하는 값을 가져오거나 설정합니다. true - 필요할 경우 언어 변환이 적용됩니다(기본값). false - 언어 변환이 적용되지 않습니다.

```csharp
public bool AllowLanguageTransformation { get; set; }
```
