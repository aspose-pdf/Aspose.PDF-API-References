## SystemFontsSubstitution.DefaultFont 속성

기본 대체 글꼴을 가져오거나 설정합니다. 다른 유효한 대체 글꼴이 발견되지 않았지만 초기 글꼴이 대상 대체 범주 ([`FontCategories`](../fontcategories/))에 속할 때 사용되는 글꼴입니다.

```csharp
public Font DefaultFont { get; set; }
```
