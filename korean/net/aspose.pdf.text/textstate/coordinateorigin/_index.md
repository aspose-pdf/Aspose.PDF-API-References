## TextState.CoordinateOrigin 속성

텍스트 CoordinateOrigin을 가져오거나 설정합니다. CoordinateOrigin이 Descender인 경우 텍스트 Y 좌표는 글꼴의 가장 낮은 지점에 해당합니다. CoordinateOrigin이 BaseLine인 경우 텍스트 Y 좌표는 글꼴의 기준선에 해당합니다. 기본값은 Descender입니다. 글꼴의 Descent 값이 너무 크면 텍스트가 다른 글꼴보다 높게 렌더링될 수 있습니다. 이 경우, CoordinateOrigin BaseLine을 선택하여 더 나은 텍스트 렌더링을 할 수 있습니다.

```csharp
public virtual CoordinateOrigin CoordinateOrigin { get; set; }
```
