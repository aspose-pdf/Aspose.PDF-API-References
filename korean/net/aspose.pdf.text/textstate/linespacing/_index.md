## TextState.LineSpacing 속성

텍스트의 줄 간격을 가져오거나 설정합니다.

```csharp
public virtual float LineSpacing { get; set; }
```

## 비고

값은 문서 내에서 텍스트 특성으로 보존되지 않음을 유의하십시오. LineSpacing 속성의 getter는 해당 객체에 대해 LineSpacing setter로 명시적으로 설정된 경우에 작동합니다. 이 속성은 현재 생성/수정 프로세스의 맥락에서 런타임에 의해 사용됩니다.
