## TextReplaceOptions.AdjustmentNewLineSpacing 속성

텍스트의 새 줄을 만들기 위해 교체 조정이 강제될 경우 사용되는 줄 간격의 값을 가져오거나 설정합니다. 예상되는 값은 교체된 텍스트의 글꼴 크기의 배수입니다. 기본값은 1.2입니다.

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
