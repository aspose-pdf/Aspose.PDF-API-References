## AppendLine(string, float) {#appendline_6}

텍스트 줄을 추가합니다.

```csharp
public void AppendLine(string line, float lineSpacing)
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| line | String | 새 줄의 텍스트입니다. |
| lineSpacing | Single | 추가 간격(0.0은 기본값이며 기본 텍스트 줄 높이에 해당합니다). 간격 값은 특정 줄의 기본 줄 간격에 추가되므로, 12pt 글꼴로 그려진 텍스트 줄 뒤에 빈 행을 얻으려면 12.0을 지정할 수 있습니다. |
