## TextReplaceOptions.Scope 열거형

텍스트 교체 작업이 적용되는 범위는 기본적으로 REPLACE_FIRST입니다. 이 구식 옵션은 호환성을 위해 유지되었습니다. PdfContentEditor에 영향을 미치며 TextFragmentAbsorber에는 영향을 미치지 않습니다.

```csharp
public enum Scope
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| REPLACE_FIRST | `0` | 영향을 받는 각 페이지에서 텍스트의 첫 번째 발생만 교체합니다. |
| REPLACE_ALL | `1` | 영향을 받는 모든 페이지에서 모든 텍스트 발생을 교체합니다. |
