## ThreadResponse.ToolResources 속성

이 스레드에서 도우미의 도구에 제공되는 리소스 집합을 가져오거나 설정합니다. 리소스는 도구의 유형에 따라 다릅니다. 예를 들어, code_interpreter 도구는 파일 ID 목록이 필요하고, file_search 도구는 벡터 저장소 ID 목록이 필요합니다.

```csharp
public ToolResources ToolResources { get; set; }
```
