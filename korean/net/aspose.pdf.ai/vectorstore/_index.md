## VectorStore class

file_ids로 벡터 저장소를 생성하고 이를 이 스레드에 연결하는 도우미입니다. 스레드에 연결할 수 있는 벡터 저장소는 최대 1개입니다.

```csharp
public class VectorStore
```

## Properties

| Name | Description |
| --- | --- |
| [FileIds](../../aspose.pdf.ai/vectorstore/fileids/) { get; set; } | 벡터 저장소에 추가할 파일 ID 목록을 가져오거나 설정합니다. 벡터 저장소에는 최대 10000개의 파일이 있을 수 있습니다. |
| [Metadata](../../aspose.pdf.ai/vectorstore/metadata/) { get; set; } | 벡터 저장소에 첨부할 수 있는 16개의 키-값 쌍 집합을 가져오거나 설정합니다. 이는 구조화된 형식으로 벡터 저장소에 대한 추가 정보를 저장하는 데 유용할 수 있습니다. 키는 최대 64자 길이일 수 있으며 값은 최대 512자 길이일 수 있습니다. |
