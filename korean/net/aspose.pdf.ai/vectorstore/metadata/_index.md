## VectorStore.Metadata 속성

벡터 저장소에 첨부할 수 있는 16개의 키-값 쌍 집합을 가져오거나 설정합니다. 이는 구조화된 형식으로 벡터 저장소에 대한 추가 정보를 저장하는 데 유용할 수 있습니다. 키는 최대 64자 길이일 수 있으며 값은 최대 512자 길이일 수 있습니다.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
