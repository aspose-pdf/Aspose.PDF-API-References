## BaseListQueryParameters.After 속성

페이지 매김에 사용할 커서를 가져오거나 설정합니다. after는 목록에서의 위치를 정의하는 객체 ID입니다. 예를 들어, 목록 요청을 하고 obj_foo로 끝나는 100개의 객체를 받으면, 이후 호출에서 after=obj_foo를 포함하여 목록의 다음 페이지를 가져올 수 있습니다.

```csharp
public string After { get; set; }
```
