## MessageContentRequest.CreateImageUrlContent 메서드

메시지에 대한 이미지 URL 콘텐츠를 생성합니다.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| 매개변수 | 유형 | 설명 |
| --- | --- | --- |
| imageUrl | 문자열 | 이미지의 URL입니다. |
| detail | 문자열 | 이미지 URL의 세부정보입니다. 사용자가 지정한 경우 이미지의 세부정보 수준을 지정합니다. "low"는 더 적은 토큰을 사용하며, "high"를 사용하여 고해상도로 선택할 수 있습니다. 기본값은 "auto"입니다. |

### 반환 값

ImageUrl 콘텐츠가 포함된 MessageContentRequest의 새 인스턴스입니다.
