## AssistantCreateRequest.TopP 속성

온도를 사용한 샘플링의 대안인 핵 샘플링을 가져오거나 설정합니다. 여기서 모델은 top_p 확률 질량을 가진 토큰의 결과를 고려합니다. 따라서 0.1은 상위 10% 확률 질량을 구성하는 토큰만 고려됨을 의미합니다. 일반적으로 이 값이나 온도를 변경하는 것을 권장하지만 둘 다 변경하지는 않는 것이 좋습니다.

```csharp
public double? TopP { get; set; }
```
