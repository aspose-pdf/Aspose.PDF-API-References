## RunCreateRequest.TopP 속성

온도를 사용한 샘플링의 대안인 핵심 샘플링을 가져오거나 설정하며, 모델이 top_p 확률 질량을 가진 토큰의 결과를 고려합니다. 따라서 0.1은 상위 10% 확률 질량을 구성하는 토큰만 고려됨을 의미합니다. 일반적으로 이 또는 온도를 변경하는 것을 권장하지만 둘 다는 권장하지 않습니다.

```csharp
public double? TopP { get; set; }
```
