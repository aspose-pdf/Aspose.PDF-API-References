## Enumeración HeadingRecognitionStrategy

Representa tipos de estrategias de reconocimiento de encabezados.

```csharp
public enum HeadingRecognitionStrategy
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Outlines | `0` | Representa la estrategia de reconocimiento de encabezados mediante contornos. |
| Heuristic | `1` | Representa la estrategia de reconocimiento de encabezados mediante reglas heurísticas y estadísticas de tamaño de fuente. |
| Auto | `2` | Proporciona una selección automática de estrategia de reconocimiento de encabezados. Esta es la opción predeterminada. Si el documento contiene marcadores, se seleccionará la estrategia Outlines, de lo contrario Heuristic |
| None | `3` | No reconocer encabezados. Esta opción puede ser útil en documentos con formato complejo. |
