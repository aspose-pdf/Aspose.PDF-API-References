## Espacios de nombres

| Espacio de nombres | Descripción |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | El **Aspose.Pdf** es un espacio de nombres raíz para todas las clases de la biblioteca Aspose.Pdf que están directamente en él como **Document** o indirectamente a través de varios subespacios de nombres. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | El espacio de nombres **Aspose.Pdf.AI** proporciona clases para funcionalidades de IA, incluidos clientes de API y asistentes inteligentes. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | El espacio de nombres **Aspose.Pdf.Annotations** proporciona clases para trabajar con varios tipos de acciones, destinos y otras características del documento que tradicionalmente se llaman interactivas, proporcionando medios para que el usuario pueda intercomunicarse con él. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | El espacio de nombres **Aspose.Pdf.Comparison** proporciona clases que se utilizan para la comparación de documentos. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | El **Aspose.Pdf.DataEditor** contiene herramientas para editar datos dentro de un documento. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | El espacio de nombres **Aspose.Pdf.Devices** proporciona clases que se utilizan para representar un documento como imagen(s) o texto plano. Así, el documento puede ser enviado a dispositivos textuales o gráficos, lo que significa que queremos obtener su representación textual o gráfica. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | El espacio de nombres **Aspose.Pdf.Drawing** proporciona clases que describen primitivas gráficas como arco, línea, círculo, etc. Los objetos correspondientes pueden ser utilizados para crear nuevos documentos. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | El espacio de nombres **Aspose.Pdf.Facades** proporciona clases que originalmente provienen de Aspose.Pdf.Kit. Estas clases se utilizan para manipular documentos realizando operaciones como concatenar, estampar, firmar, anotar, etc., pero a un nivel alto sin acceso a la estructura interna de un documento. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | El espacio de nombres **Aspose.Pdf.Forms** tiene clases que describen formularios (estándar, estáticos, dinámicos) y varios tipos de campos como cuadro de texto, cuadro de lista, botón de opción, etc. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | El **Aspose.Pdf.Tagged.LogicalStructure** es un espacio de nombres para clases de Estructura Lógica PDF. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | El espacio de nombres **Aspose.Pdf.Multithreading** proporciona clases que están destinadas a ser utilizadas en escenarios multihilo. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | El **Aspose.Pdf.Operators** es un espacio de nombres para implementaciones de Operadores. Estas clases describen operadores utilizados en los contenidos de las páginas PDF. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | El **Aspose.Pdf.Optimization** es un espacio de nombres para clases que gestionan el proceso de optimización de documentos. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | El **Aspose.Pdf.Plugins** es un espacio de nombres raíz para todas las clases de Aspose.Pdf.Plugins que están directamente en él como **PdfOrganizer** o indirectamente a través de varios subespacios de nombres. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | El espacio de nombres **Aspose.Pdf.Printing** proporciona clases y funcionalidades para manejar operaciones de impresión de PDF, incluidos ajustes de impresora, ajustes de página y extensiones para gestionar configuraciones relacionadas con la impresión. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | El **Aspose.Pdf.Sanitization** es un espacio de nombres para operaciones de saneamiento. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | El espacio de nombres **Aspose.Pdf.Security** contiene clases utilizadas para cifrado y firma digital. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | El espacio de nombres **Aspose.Pdf.Structure** proporciona clases que ayudan a trabajar con la estructura lógica de un documento que presenta la organización del documento en capítulos y secciones o la identificación de elementos especiales como figuras, tablas y notas al pie. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | El **Aspose.Pdf.Tagged** es un espacio de nombres para clases que soportan documentos PDF etiquetados. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | El espacio de nombres **Aspose.Pdf.Text** proporciona clases que permiten extraer texto, agregar texto, manipular texto existente de un documento. También contiene clases que permiten extraer, reemplazar, sustituir fuentes de un documento. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | El **Aspose.Pdf.Vector** es un espacio de nombres raíz para operaciones gráficas. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | El espacio de nombres **Aspose.Pdf.XfaConverter** proporciona clases que permiten convertir XFA. |
