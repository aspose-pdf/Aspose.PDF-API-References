## Propiedad AnnotationElement.AlternateDescriptions

Obtiene o establece las Descripciones Alternativas para la anotación. Texto que se mostrará para la anotación o, si este tipo de anotación no muestra texto, una descripción alternativa del contenido de la anotación en forma legible por humanos.

```csharp
public string AlternateDescriptions { get; set; }
```
