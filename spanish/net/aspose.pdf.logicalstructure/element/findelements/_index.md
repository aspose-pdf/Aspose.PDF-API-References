## Método Element.FindElements&lt;T&gt;

Encontrar elementos de un tipo dado

```csharp
public List<T> FindElements<T>(bool recursiveSearch = false)
    where T : Element
```

| Parámetro | Descripción |
| --- | --- |
| T | Tipo de elemento de estructura para buscar |
| recursiveSearch | (Opcional) Búsqueda recursiva (por defecto falso, buscar solo desde los hijos directos) |

### Valor de retorno

Lista de elementos encontrados
