## Campo StructureTypeStandard.TOCI

(Elemento de tabla de contenidos) Un miembro individual de una tabla de contenidos. Los hijos de esta entrada pueden ser cualquiera de los siguientes tipos de estructura:

**Lbl**

Una etiqueta

**Reference**

Una referencia al título y al número de página

**NonStruct**

Elementos no estructurados para envolver un artefacto líder

**P**

Texto descriptivo

**TOC**

Elementos de tabla de contenido para tablas de contenido jerárquicas, como se describe para la entrada TOC

```csharp
public static readonly StructureTypeStandard TOCI;
```
