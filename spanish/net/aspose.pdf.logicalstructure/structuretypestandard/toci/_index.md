## StructureTypeStandard.TOCI field

(Elemento de tabla de contenido) Un miembro individual de una tabla de contenido. Los hijos de esta entrada pueden ser cualquiera de los siguientes tipos de estructuras:

**Lbl**

Una etiqueta

**Referencia**

Una referencia al título y el número de página.

**no estructura**

Elementos no estructurados para envolver un artefacto líder

**PAGS**

Texto descriptivo

**TOC**

Tabla de elementos de contenido para tablas de contenido jerárquicas, como se describe para la entrada TOC
