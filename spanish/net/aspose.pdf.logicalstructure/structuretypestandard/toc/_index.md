## Campo StructureTypeStandard.TOC

(Tabla de contenido) Una lista compuesta por entradas de elementos de tabla de contenido (tipo estructura TOCI) y/o otras entradas de tabla de contenido anidadas (TOC).

Una entrada TOC que incluye solo entradas TOCI representa una jerarquía plana. Una entrada TOC que incluye otras entradas TOC anidadas (y posiblemente entradas TOCI) representa una jerarquía más compleja. Idealmente, la jerarquía de una entrada TOC de nivel superior refleja la estructura del cuerpo principal del documento.

```csharp
public static readonly StructureTypeStandard TOC;
```
