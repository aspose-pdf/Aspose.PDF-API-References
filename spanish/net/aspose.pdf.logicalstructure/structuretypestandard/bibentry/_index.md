## Campo StructureTypeStandard.BibEntry

(Entrada de bibliografía) Una referencia que identifica la fuente externa de algún contenido citado. Puede contener una etiqueta (tipo de estructura Lbl) como hijo.

Aunque una entrada de bibliografía probablemente incluya partes componentes que identifiquen al autor del contenido citado, la obra, el editor, etc., no se definen tipos de estructura estándar a este nivel de detalle.

```csharp
public static readonly StructureTypeStandard BibEntry;
```
