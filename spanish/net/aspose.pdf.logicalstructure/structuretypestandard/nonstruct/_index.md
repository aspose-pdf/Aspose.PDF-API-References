## Campo StructureTypeStandard.NonStruct

(Elemento no estructural) Un elemento de agrupamiento que no tiene un significado estructural inherente; sirve únicamente para fines de agrupamiento. Este tipo de elemento difiere de una división (tipo de estructura Div) en que no debe ser interpretado ni exportado a otros formatos de documento; sin embargo, sus descendientes serán procesados normalmente.

```csharp
public static readonly StructureTypeStandard NonStruct;
```
