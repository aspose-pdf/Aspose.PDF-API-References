## Campo StructureTypeStandard.THead

(Grupo de fila de encabezado de tabla; PDF 1.5) Un grupo de filas que constituyen el encabezado de una tabla. Si la tabla se divide en varias páginas, estas filas pueden ser redibujadas en la parte superior de cada fragmento de tabla (aunque solo hay un elemento THead).

```csharp
public static readonly StructureTypeStandard THead;
```
