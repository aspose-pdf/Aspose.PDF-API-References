## StructureTypeStandard.TBody field

(Grupo de filas del cuerpo de la tabla; PDF 1.5) Un grupo de filas que constituyen la parte principal del cuerpo de una tabla. Si la tabla está dividida en varias páginas, el área del cuerpo puede dividirse en un límite de fila. Una tabla puede tener varios elementos TBody para permitir el dibujo de un borde o fondo para un conjunto de filas.
