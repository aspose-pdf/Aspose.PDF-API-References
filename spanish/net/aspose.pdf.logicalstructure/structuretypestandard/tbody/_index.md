## Campo StructureTypeStandard.TBody

(Grupo de filas del cuerpo de la tabla; PDF 1.5) Un grupo de filas que constituyen la parte principal del cuerpo de una tabla. Si la tabla se divide en varias páginas, el área del cuerpo puede separarse en un límite de fila. Una tabla puede tener múltiples elementos TBody para permitir el dibujo de un borde o fondo para un conjunto de filas.

```csharp
public static readonly StructureTypeStandard TBody;
```
