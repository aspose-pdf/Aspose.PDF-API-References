## Indexador de ElementList

Obtiene un elemento en el índice dado.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Parámetro | Descripción |
| --- | --- |
| index | El índice en la lista de elementos. |

### Valor de Retorno

El Elemento con el índice especificado en la colección. Si el índice es mayor o igual al número de elementos en la lista, esto devuelve null.
