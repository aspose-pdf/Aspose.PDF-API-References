## Método ElementList.Item

Recupera un elemento en el índice dado.

```csharp
public abstract Element Item(int index)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| index | Int32 | El índice en la lista de elementos. |

### Valor de Retorno

El Elemento con el índice especificado en la colección. Si *index* es mayor o igual al número de elementos en la lista, esto devuelve null.
