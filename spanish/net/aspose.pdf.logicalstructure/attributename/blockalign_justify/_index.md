## AttributeName.BlockAlign_Justify field

Atributo BlockAlign: Justificar: elementos secundarios alineados con los bordes anteriores y posteriores del rectángulo de contenido de la celda de la tabla. El primer niño se colocará como se describe para Antes y el último niño como se describe para Después, con el mismo espacio entre los niños. Si solo hay un hijo, se alineará solo con el borde anterior, como para Antes.
