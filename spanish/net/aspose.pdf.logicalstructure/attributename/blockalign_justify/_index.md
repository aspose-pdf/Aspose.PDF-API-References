## Campo AttributeName.BlockAlign_Justify

Atributo BlockAlign: Justify - Hijos alineados con los bordes antes y después del rectángulo de contenido de la celda de la tabla. El primer hijo se colocará como se describe para Antes y el último hijo como se describe para Después, con un espaciado igual entre los hijos. Si solo hay un hijo, se alineará solo con el borde anterior, como para Antes.

```csharp
public static readonly AttributeName BlockAlign_Justify;
```
