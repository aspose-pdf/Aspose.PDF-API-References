## Campo AttributeName.TextAlign_Justify

Atributo TextAlign: Justify - Alineado con los bordes de inicio y fin, con un espaciado interno dentro de cada línea expandido, si es necesario, para lograr tal alineación. La última (o única) línea debe estar alineada solo con el borde de inicio.

```csharp
public static readonly AttributeName TextAlign_Justify;
```
