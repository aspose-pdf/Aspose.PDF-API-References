## Propiedad SetDash.Phase

Fase de guiones. Antes de comenzar a trazar un camino, el arreglo de guiones debe ser cíclico, sumando las longitudes de los guiones y los espacios. Cuando la longitud acumulada sea igual al valor especificado por la fase de guiones, comenzará el trazado del camino y el arreglo de guiones se utilizará cíclicamente a partir de ese momento.

```csharp
public int Phase { get; set; }
```
