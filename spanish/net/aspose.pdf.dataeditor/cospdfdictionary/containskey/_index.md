## Método CosPdfDictionary.ContainsKey

Determina si el [`CosPdfDictionary`](../) contiene un elemento con la clave especificada.

```csharp
public bool ContainsKey(string key)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| key | String | La clave a localizar en el [`CosPdfDictionary`](../). |

### Valor de Retorno

true si el [`CosPdfDictionary`](../) contiene un elemento editable con la clave; de lo contrario, false.
