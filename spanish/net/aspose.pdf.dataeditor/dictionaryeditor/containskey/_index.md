## Método DictionaryEditor.ContainsKey

Determina si el [`DictionaryEditor`](../) contiene un elemento con la clave especificada.

```csharp
public bool ContainsKey(string key)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| key | String | La clave a localizar en el [`DictionaryEditor`](../). |

### Valor de Retorno

true si el [`DictionaryEditor`](../) contiene un elemento editable con la clave; de lo contrario, false.
