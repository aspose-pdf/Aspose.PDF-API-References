## Método DictionaryEditor.Contains

Determina si el [`DictionaryEditor`](../) contiene un valor específico.

```csharp
public bool Contains(KeyValuePair<string, ICosPdfPrimitive> item)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| item | KeyValuePair`2 | El objeto deseado. |

### Valor de Retorno

true si el item se encuentra en el [`DictionaryEditor`](../); de lo contrario, false.
