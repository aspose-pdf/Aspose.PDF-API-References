## Enumeración ImageCompressionVersion

Describe versiones del algoritmo de compresión de imágenes.

```csharp
public enum ImageCompressionVersion
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Standard | `0` | Algoritmo estándar. Valor predeterminado. |
| Fast | `2` | Algoritmo mejorado más rápido que el estándar, pero aplicable no en todos los casos. |
| Mixed | `3` | Utiliza el algoritmo rápido cuando sea posible y el estándar para otros casos. Puede ser más lento que "Fast" pero puede producir mejor compresión. |
