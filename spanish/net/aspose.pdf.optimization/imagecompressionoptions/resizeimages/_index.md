## Propiedad ImageCompressionOptions.ResizeImages

Si esta bandera se establece en verdadero y CompressImages es verdadero, las imágenes se redimensionarán si la resolución de la imagen es mayor que el parámetro MaxResolution especificado.

```csharp
public bool ResizeImages { get; set; }
```
