## Propiedad ImageCompressionOptions.CompressImages

Si esta bandera se establece en verdadero, las imágenes se comprimirán en el documento. El nivel de compresión se especifica con la propiedad ImageQuality.

```csharp
public bool CompressImages { get; set; }
```
