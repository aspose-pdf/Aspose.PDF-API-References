## Propiedad OptimizationOptions.RemoveUnusedStreams

Si esta bandera se establece en verdadero, se verifica el uso de cada recurso. Si el recurso nunca se utiliza, entonces se elimina. Esto puede disminuir el tamaño del documento, por ejemplo, cuando se extraen páginas del documento.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
