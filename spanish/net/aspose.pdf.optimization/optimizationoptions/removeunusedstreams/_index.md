## OptimizationOptions.RemoveUnusedStreams property

Si este indicador se establece en verdadero, se comprueba el uso de cada recurso. Si nunca se usa el recurso, entonces se eliminan los recursos. Esto puede disminuir el tamaño del documento, por ejemplo, cuando se extrajeron páginas del documento.

public bool RemoveUnusedStreams { get; set; }
