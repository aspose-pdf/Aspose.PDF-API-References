## Propiedad OptimizationOptions.LinkDuplcateStreams

Si esta bandera se establece en verdadero, se analizarán los flujos de recursos. Si se encuentran flujos duplicados (es decir, si el contenido del flujo es igual), entonces estos flujos se almacenarán como un solo objeto. Esto permite reducir el tamaño del documento en algunos casos (por ejemplo, cuando el mismo documento se ha concatenado múltiples veces).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
