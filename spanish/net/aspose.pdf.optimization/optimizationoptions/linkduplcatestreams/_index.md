## OptimizationOptions.LinkDuplcateStreams property

Si este indicador se establece en verdadero, se analizarán los flujos de recursos. Si se encuentran secuencias duplicadas (es decir, si el contenido de la secuencia es igual), estas secuencias se almacenarán como un solo objeto. Esto permite disminuir el tamaño del documento en algunos casos (por ejemplo, cuando el mismo documento se concatenó varias veces).

public bool LinkDuplcateStreams { get; set; }
