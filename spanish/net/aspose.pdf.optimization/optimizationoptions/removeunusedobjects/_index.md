## Propiedad OptimizationOptions.RemoveUnusedObjects

Si esta bandera se establece en verdadero, todos los objetos del documento serán verificados y los objetos no utilizados (es decir, los objetos que no tienen ninguna referencia) se eliminarán del documento.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
