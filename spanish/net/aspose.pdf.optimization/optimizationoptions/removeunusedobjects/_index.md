## OptimizationOptions.RemoveUnusedObjects property

Si este indicador se establece en verdadero, todos los objetos del documento se verificarán y los objetos no utilizados (es decir, los objetos que no tienen ninguna referencia) se eliminarán del documento.

public bool RemoveUnusedObjects { get; set; }
