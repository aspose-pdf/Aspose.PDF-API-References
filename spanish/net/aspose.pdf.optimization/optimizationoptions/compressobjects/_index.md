## Propiedad OptimizationOptions.CompressObjects

Si esta bandera se establece en `true`, los objetos Pdf se empaquetarán en flujos de objetos y se comprimirán para reducir el tamaño del archivo pdf.

```csharp
public bool CompressObjects { get; set; }
```
