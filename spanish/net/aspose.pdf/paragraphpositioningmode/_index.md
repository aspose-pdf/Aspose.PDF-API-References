## Enumeración ParagraphPositioningMode

Especifica la variante para determinar la ubicación del elemento en la página.

```csharp
public enum ParagraphPositioningMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `0` | La ubicación se determina por los elementos colocados previamente. La adición de un elemento se tiene en cuenta al determinar la ubicación de los elementos posteriores. |
| Absolute | `1` | La ubicación se especifica mediante los valores Left y Top, no depende de elementos anteriores y no afecta la ubicación de los posteriores. |
