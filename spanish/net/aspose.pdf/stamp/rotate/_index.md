## Propiedad Stamp.Rotate

Establece o obtiene la rotación del contenido del sello de acuerdo con los valores de [`Rotation`](../../rotation/). Nota. Esta propiedad es para establecer ángulos que son múltiplos de 90 grados (0, 90, 180, 270 grados). Para establecer un ángulo arbitrario, use la propiedad RotateAngle. Si el ángulo establecido por ArbitraryAngle no es múltiplo de 90, entonces la propiedad Rotate devuelve Rotation.None.

```csharp
public Rotation Rotate { get; set; }
```
