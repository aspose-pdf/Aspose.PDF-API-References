## Campo LoadOptions.ResourceLoadingResult.MIMETypeIfKnown

A veces, el conocimiento sobre el tipo MIME del recurso cargado es útil para el convertidor. Puedes proporcionar el tipo MIME (si se conoce después de la carga) en este parámetro. Por favor, deja el parámetro igual a nulo cuando el tipo MIME es desconocido o no es necesario proporcionarlo.

```csharp
public string MIMETypeIfKnown;
```
