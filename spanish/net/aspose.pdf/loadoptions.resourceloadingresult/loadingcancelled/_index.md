## LoadOptions.ResourceLoadingResult.LoadingCancelled field

A veces, por alguna razón, la carga no debería ocurrir código personalizado. En tal caso, por favor, establezca esta bandera como True. En tal caso, el convertidor intentará usar el cargador de recursos predeterminado interno para obtener ese resultado (como se comporta en situaciones cuando no se suministra una estrategia personalizada).

```csharp
public bool LoadingCancelled;
```
