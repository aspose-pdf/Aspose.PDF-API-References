## Método Matrix.Skew

Crea una matriz para el ángulo de rotación dado.

```csharp
public static Matrix Skew(double alpha, double beta)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| alpha | Doble | Ángulo de sesgo x en radianes. |
| beta | Doble | Ángulo de sesgo y en radianes. |

### Valor de Retorno

Matriz de transformación.

## Ejemplos

```csharp
Matrix m = Matrix.Skew(Math.PI / 2, Math.PI / 2);
```
