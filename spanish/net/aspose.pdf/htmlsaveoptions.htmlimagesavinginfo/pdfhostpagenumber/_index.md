## Campo HtmlSaveOptions.HtmlImageSavingInfo.PdfHostPageNumber

Indica al código personalizado a qué página del documento PDF original pertenece la imagen guardada. Dado que es posible que no se guarden todas las páginas del documento original, este valor nos indica el número de página anfitriona en el PDF original. Si el número de página original por alguna razón es desconocido, siempre devuelve '1'

```csharp
public int PdfHostPageNumber;
```
