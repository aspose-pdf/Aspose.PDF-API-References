## Campo HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber

Indica al código personalizado a qué página del conjunto generado de archivos de página HTML pertenece la imagen guardada. Si la división en páginas está desactivada, este valor siempre contiene '1', ya que en tal caso solo se genera una página HTML.

```csharp
public int HtmlHostPageNumber;
```
