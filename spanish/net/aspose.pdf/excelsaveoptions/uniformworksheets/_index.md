## Propiedad ExcelSaveOptions.UniformWorksheets

Establezca verdadero para usar una división uniforme de columnas a través del documento. El valor predeterminado es falso; significa que la división de columnas será independiente para cada página.

```csharp
public bool UniformWorksheets { get; set; }
```
