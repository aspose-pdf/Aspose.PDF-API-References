## Propiedad PdfFormatConversionOptions.ExcludeFontsStrategy

Estrategia(s) para excluir fuentes superfluas y reducir el tamaño del archivo del documento. Este parámetro tiene sentido solo cuando el indicador [`OptimizeFileSize`](../optimizefilesize/) está configurado en verdadero. Por defecto, se utiliza la combinación de estrategias SubsetFonts y RemoveDuplicatedFonts.

```csharp
public RemoveFontsStrategy ExcludeFontsStrategy { get; set; }
```
