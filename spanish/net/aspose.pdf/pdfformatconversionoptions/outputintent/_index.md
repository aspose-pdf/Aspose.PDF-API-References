## Propiedad PdfFormatConversionOptions.OutputIntent

Obtiene o establece el [`OutputIntent`](../../outputintent/) para la conversión de formato PDF.

```csharp
public OutputIntent OutputIntent { get; set; }
```

## Observaciones

El `OutputIntent` especifica el dispositivo o condición de salida previsto para el cual se está preparando el documento PDF. Se utiliza para garantizar que los colores en el documento se representen correctamente en el dispositivo de destino.
