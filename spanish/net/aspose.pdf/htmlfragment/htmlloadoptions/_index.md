## HtmlFragment.HtmlLoadOptions property

Obtiene o establece HtmlLoadOptions que se usarán para cargar (y renderizar) HTML en esta instancia de class. Úselo cuando sea necesario use una configuración específica para importar HTML para esta o aquella instancia (por ejemplo, cuando esta o aquella instancia debería use BasePath específico para HTML importado o debe usar un cargador específico de recursos externos) Si el parámetro es predeterminado (nulo), entonces se usarán las opciones de carga de HTML estándar.

```csharp
public HtmlLoadOptions HtmlLoadOptions { get; set; }
```
