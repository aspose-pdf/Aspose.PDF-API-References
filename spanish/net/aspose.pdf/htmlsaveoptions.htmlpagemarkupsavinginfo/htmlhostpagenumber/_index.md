## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber field

Establecido por convertidor. Si se establece la propiedad SplitToPages, se crean varios archivos HTML (un archivo HTML por página convertida) durante la conversión. Esta propiedad contiene el ordinal del archivo de la página HTML guardada. La propiedad se puede usar en la lógica del código personalizado para decidir cómo procesar o dónde guardar la página HTML y Si la división en páginas está desactivada, este valor siempre contiene '1' ya que en tal caso, solo se genera una gran página HTML para todo el documento de origen.

public int HtmlHostPageNumber;
