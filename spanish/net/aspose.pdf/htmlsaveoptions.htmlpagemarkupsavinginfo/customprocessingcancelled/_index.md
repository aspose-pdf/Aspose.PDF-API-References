## HtmlSaveOptions.HtmlPageMarkupSavingInfo.CustomProcessingCancelled field

Debe configurarse en código personalizado cuando sea necesario. Este indicador debe configurarse en "verdadero" en código personalizado si, por alguna razón, el marcado html proporcionado por debe procesarse no con código personalizado sino con el propio código del convertidor de forma estándar para el convertidor. Por lo tanto, configurar si este indicador en el código personalizado significa que el código personalizado no procesó el archivo al que se hace referencia y el convertidor debe manejarlo por sí mismo

public bool CustomProcessingCancelled;
