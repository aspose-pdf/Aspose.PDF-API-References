## Campo HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber

Establecido por el convertidor. Si la propiedad SplitToPages está establecida, entonces se crean varios archivos HTML (un archivo HTML por página convertida) durante la conversión. Esta propiedad indica al código personalizado de qué página del PDF original se creó el HTML guardado. Si el número de página original por alguna razón es desconocido o SplitOnPages=false, entonces esta propiedad siempre contiene '0', lo que indica que el convertidor no puede proporcionar el número de página exacto del PDF original para el archivo de HTML guardado.

```csharp
public int PdfHostPageNumber;
```
