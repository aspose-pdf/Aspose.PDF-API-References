## HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber field

Establecido por el convertidor. Si se establece la propiedad SplitToPages, se crean varios archivos HTML (un archivo HTML por página convertida) durante la conversión creada . Esta propiedad le dice al código personalizado desde qué página del PDF original se creó el marcado HTML guardado . Si el número de página original por alguna razón es desconocido o SplitOnPages=false, entonces esta propiedad siempre contiene '0' que indica que el convertidor no puede proporcionar el número de página del PDF original exacto para el archivo de marcado HTML provisto.

public int PdfHostPageNumber;
