## Propiedad SaveOptions.WarningHandler

Callback para manejar cualquier advertencia generada. El WarningHandler devuelve un elemento de enumeración ReturnAction que especifica ya sea Continuar o Abort. Continuar es la acción predeterminada y la operación de Guardar continúa, sin embargo, el usuario también puede devolver Abort, en cuyo caso la operación de Guardar debe cesar.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
