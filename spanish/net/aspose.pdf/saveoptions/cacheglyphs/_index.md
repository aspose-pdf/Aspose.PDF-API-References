## Propiedad SaveOptions.CacheGlyphs

Obtiene o establece un valor booleano que indica si los glifos de fuente se almacenarán en caché mientras se preparan las páginas aps. Mejora el rendimiento de la conversión de pdf a otros formatos, pero aumenta el consumo de memoria.

```csharp
public bool CacheGlyphs { get; set; }
```
