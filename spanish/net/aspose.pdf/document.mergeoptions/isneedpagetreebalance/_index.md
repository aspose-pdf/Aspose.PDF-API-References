## Propiedad Document.MergeOptions.IsNeedPageTreeBalance

Obtiene y establece el requisito para el balanceo del árbol de páginas. Todo el árbol de páginas en el documento resultante será reequilibrado. Crea un árbol de páginas balanceado para acelerar el acceso a las páginas.

```csharp
public bool IsNeedPageTreeBalance { get; set; }
```
