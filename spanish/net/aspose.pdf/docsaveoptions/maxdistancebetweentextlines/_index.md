## Propiedad DocSaveOptions.MaxDistanceBetweenTextLines

Este parámetro se utiliza para agrupar líneas de texto en párrafos. Determina cuán separadas pueden estar dos líneas de texto relativas. Especificado en cientos de por ciento de la altura de las líneas de texto.

```csharp
public float MaxDistanceBetweenTextLines { get; set; }
```
