## ImageDeleteAction enumeration

Acción que se realizó con el objeto de imagen cuando la imagen se eliminó de la colección. Si se elimina el objeto de imagen

```csharp
public enum ImageDeleteAction
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| KeepContents | `0` | La imagen se eliminará de la colección. Si el contenido de la página contiene referencias a la imagen, no se eliminarán. El documento puede dejar de ser válido. |
| None | `1` | La imagen se eliminará de la colección y del contenido de la página, pero el objeto de imagen no se eliminará. El tamaño del archivo no se reducirá. |
| ForceDelete | `2` | La imagen se eliminará de la colección y el objeto de imagen se eliminará del documento. Si existen otras referencias sobre el mismo objeto, es posible que el documento esté dañado. |
| Check | `3` | La imagen se eliminará de la colección y el objeto de imagen se eliminará solo si no hay otras referencias a la imagen de otras páginas. Esto puede requerir más tiempo en comparación con la opción ForceDelete. |
