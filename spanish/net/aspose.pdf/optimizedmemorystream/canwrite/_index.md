## Propiedad OptimizedMemoryStream.CanWrite

Cuando se anula en una clase derivada, obtiene un valor que indica si el flujo actual admite escritura.

```csharp
public override bool CanWrite { get; }
```

### Valor de Retorno

true si el flujo admite escritura; de lo contrario, false.
