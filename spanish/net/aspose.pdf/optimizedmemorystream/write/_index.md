## Método OptimizedMemoryStream.Write

Cuando se anula en una clase derivada, escribe una secuencia de bytes en el flujo actual y avanza la posición actual dentro de este flujo por el número de bytes escritos.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| buffer | Byte[] | Un arreglo de bytes. Este método copia *count* bytes de *buffer* al flujo actual. |
| offset | Int32 | El desplazamiento de bytes basado en cero en *buffer* en el que comenzar a copiar bytes al flujo actual. |
| count | Int32 | El número de bytes que se escribirán en el flujo actual. |
