## Método OptimizedMemoryStream.Seek

Cuando se anula en una clase derivada, establece la posición dentro del flujo actual.

```csharp
public override long Seek(long offset, SeekOrigin origin)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| offset | Int64 | Un desplazamiento de bytes relativo al parámetro *origin*. |
| origin | SeekOrigin | Un valor de tipo SeekOrigin que indica el punto de referencia utilizado para obtener la nueva posición. |

### Valor de Retorno

La nueva posición dentro del flujo actual.
