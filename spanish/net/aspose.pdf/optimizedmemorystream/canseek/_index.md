## Propiedad OptimizedMemoryStream.CanSeek

Cuando se anula en una clase derivada, obtiene un valor que indica si el flujo actual admite búsqueda.

```csharp
public override bool CanSeek { get; }
```

### Valor de Retorno

true si el flujo admite búsqueda; de lo contrario, false.
