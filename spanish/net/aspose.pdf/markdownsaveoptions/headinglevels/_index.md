## Propiedad MarkdownSaveOptions.HeadingLevels

Define los niveles de encabezado esperados para usar en la estrategia de reconocimiento de encabezados de FontSize. Si se establece este valor de propiedad, entonces se seleccionará la estrategia de reconocimiento de encabezados !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic cuando se establezcan las estrategias !:PdfToMarkdown.HeadingRecognitionStrategy.Auto, incluso si el documento contiene marcadores.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
