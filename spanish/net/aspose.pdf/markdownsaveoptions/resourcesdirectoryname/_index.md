## Propiedad MarkdownSaveOptions.ResourcesDirectoryName

Obtiene y establece el nombre del directorio para guardar recursos del documento como imágenes. Si el valor no se especifica, entonces las imágenes se escribirán en el mismo directorio que el archivo markdown. ¡Esto no es una ruta, solo es un nombre! Este directorio se creará automáticamente en el directorio con el archivo markdown guardado.

```csharp
public string ResourcesDirectoryName { get; set; }
```
