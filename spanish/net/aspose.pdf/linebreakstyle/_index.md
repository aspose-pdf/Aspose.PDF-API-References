## Enumeración LineBreakStyle

Representa los posibles estilos de salto de línea para un archivo.

```csharp
public enum LineBreakStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Windows | `0` | Selecciona el estilo de ajuste de cadena de Windows (CRLF). |
| Unix | `1` | Selecciona el estilo de ajuste de cadena de Unix (LF). También se utiliza para OSX. |
| Auto | `2` | Selecciona automáticamente el tipo de ajuste de cadena según el sistema operativo. |
