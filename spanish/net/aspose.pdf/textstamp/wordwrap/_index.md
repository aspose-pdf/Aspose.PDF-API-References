## Propiedad TextStamp.WordWrap

Define el ajuste de palabras. Si esta propiedad se establece en verdadero y se especifica un valor de Width, el texto se dividirá en varias líneas para ajustarse al ancho especificado. Valor predeterminado: falso.

public bool WordWrap { get; set; }
