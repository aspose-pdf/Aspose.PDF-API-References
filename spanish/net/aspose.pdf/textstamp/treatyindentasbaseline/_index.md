## TextStamp.TreatYIndentAsBaseLine property

Define el origen de las coordenadas para colocar el texto. Si TreatYIndentAsBaseLine = true (predeterminado cuando Draw = true), el valor de YIndent se tratará como línea base de texto. Si TreatYIndentAsBaseLine = false (predeterminado cuando Draw = false), el valor de YIndent se tratará como inferior ( línea de descenso) de texto.

public bool TreatYIndentAsBaseLine { get; set; }
