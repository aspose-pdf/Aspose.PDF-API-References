## Propiedad TextStamp.TreatYIndentAsBaseLine

Define el origen de coordenadas para colocar texto. Si TreatYIndentAsBaseLine = true (por defecto cuando Draw = true) el valor de YIndent se tratará como la línea base del texto. Si TreatYIndentAsBaseLine = false (por defecto cuando Draw = false) el valor de YIndent se tratará como la parte inferior (línea de descenso) del texto.

public bool TreatYIndentAsBaseLine { get; set; }
