## Propiedad TextStamp.AutoAdjustFontSizeToFitStampRectangle

Si está habilitada, el tamaño de la fuente se ajustará automáticamente para encajar en el rectángulo del sello de tamaño: [`Width`](../width/) y [`Height`](../height/). El ancho y alto predeterminados se derivan del rectángulo de la página.

```csharp
public bool AutoAdjustFontSizeToFitStampRectangle { get; set; }
```
