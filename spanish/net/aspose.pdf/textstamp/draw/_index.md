## Propiedad TextStamp.Draw

Esta propiedad determina cómo se dibuja el sello en la página. Si Draw = true, el sello se dibuja como operadores gráficos y si draw = false, entonces el sello se dibuja como texto.

```csharp
public bool Draw { get; set; }
```
