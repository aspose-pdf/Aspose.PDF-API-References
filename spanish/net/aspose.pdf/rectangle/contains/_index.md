## Método Rectangle.Contains

Determina si un punto dado está dentro del rectángulo.

```csharp
public bool Contains(Point point, bool inclusive = false)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| point | Point | Punto a verificar. |
| inclusive | Boolean | inclusivo: el punto puede estar en el borde |

### Valor de Retorno

Verdadero si el punto está dentro del rectángulo.
