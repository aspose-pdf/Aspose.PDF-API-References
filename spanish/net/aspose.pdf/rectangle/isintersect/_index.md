## Método Rectangle.IsIntersect

Determina si este rectángulo intersecta con otro rectángulo.

```csharp
public bool IsIntersect(Rectangle otherRect)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| otherRect | Rectangle | La intersección se probará con el rectángulo especificado. |

### Valor de Retorno

Verdadero si este rectángulo intersecta con el rectángulo especificado. De lo contrario, falso.
