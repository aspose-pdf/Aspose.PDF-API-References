## Propiedad ColumnInfo.ColumnWidths

Obtiene o establece una cadena que contiene el ancho de las columnas. El valor de cada columna debe estar separado por un espacio en blanco. La unidad predeterminada es punto, pero también se admiten cm, pulgada y porcentaje del ancho disponible. Por ejemplo, "120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
