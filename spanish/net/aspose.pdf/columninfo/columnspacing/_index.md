## Propiedad ColumnInfo.ColumnSpacing

Obtiene o establece una cadena que contiene el espaciado entre columnas. El valor de cada espaciado debe estar separado por un espacio en blanco. La unidad predeterminada es punto, pero también se admiten cm y pulgadas. Por ejemplo, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Observaciones

Si esta propiedad no se establece, se utilizará el valor predeterminado 0 para cada espaciado.
