## ColumnInfo.ColumnSpacing property

Obtiene o establece una cadena que contiene el espacio entre columnas. El valor de cada espacio debe estar separado por espacios en blanco. La unidad predeterminada es punto, pero también se admiten cm y pulgadas. Por ejemplo, "120 2,5 cm 1,5 pulgadas".

```csharp
public string ColumnSpacing { get; set; }
```

### Observaciones

Si no se establece esta propiedad, se utilizará el valor predeterminado 0 para cada espaciado.
