## Propiedades

| Nombre | Descripción |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | Obtiene una bandera que indica si el procesador PDF interactivo debe proporcionar soporte para editar el valor del campo. Valor predeterminado: false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | Obtiene el tipo de un valor de campo en una colección de esquema. Este campo describe el tipo de valor correspondiente a [`Subtype`](./subtype/). |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | Obtiene el nombre de campo textual que se presentará al usuario por el procesador PDF interactivo |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | Obtiene el orden relativo del nombre del campo en la interfaz de usuario. Los campos deben ser ordenados por el procesador PDF interactivo en orden ascendente. |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | Obtiene el subtipo de un valor de campo en una colección de esquema. El subtipo de campo de colección o campo relacionado con archivos que este diccionario describe. Esta entrada identifica el tipo de datos que se almacenará en el campo. |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | Obtiene la visibilidad inicial del campo en la interfaz de usuario. Valor predeterminado: true. |
