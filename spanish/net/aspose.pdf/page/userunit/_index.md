## Propiedad Page.UserUnit

Obtiene o establece el valor de UserUnit. Un número positivo que da el tamaño de las unidades de espacio de usuario predeterminadas, en múltiplos de 1 / 72 pulgadas. El valor predeterminado es 1. Por favor, establezca un valor cero o negativo para borrar esta entrada en la página.

```csharp
public double UserUnit { get; set; }
```
