## Page.Rect property

Obtiene o establece el rectángulo de la página. Se devuelve el cuadro de recorte de página si se especifica; de lo contrario, se devuelve el cuadro de medios de página. Tenga en cuenta que esta propiedad no tiene en cuenta la rotación de páginas. Para obtener un rectángulo de página teniendo en cuenta la rotación, utilice ActualRect.

```csharp
public Rectangle Rect { get; set; }
```

### Ejemplos

El ejemplo demuestra cómo obtener el rectángulo de página:

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
Rectangle pageRect = page.Rect;
```
