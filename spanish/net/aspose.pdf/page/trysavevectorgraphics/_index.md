## Método Page.TrySaveVectorGraphics

Intenta guardar gráficos vectoriales si están presentes en la página. El formato de guardado es SVG.

```csharp
public bool TrySaveVectorGraphics(string pathToSave)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| pathToSave | String | Archivo de salida |

### Valor de Retorno

Verdadero si la página contiene operadores de construcción de rutas; de lo contrario, Falso.
