## Propiedad ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace

Algunas fuentes no proporcionan información sobre unicodes para algunos símbolos de texto. Esta falta de información provoca un error "El texto no se puede mapear a Unicode". Utilice esta bandera para mapear símbolos no vinculados en "espacio" unicode (código 32).

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
