## Enumeración DigestHashAlgorithm

Representa el tipo de algoritmo que mapea datos a un "hash"

```csharp
public enum DigestHashAlgorithm
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Auto | `0` | Configuración automática del algoritmo de hash a discreción del algoritmo de firma. Para EDCSA, el valor predeterminado se determina por el tamaño de la clave. El valor predeterminado para un PKCS7 no separado es Sha1. |
| Sha1 | `1` | SHA-1. Algoritmo de Hash Seguro 1. Es un valor predeterminado para un PKCS7 no separado. |
| Sha256 | `2` | SHA-256. Algoritmo de Hash Seguro 2. Es un valor predeterminado para un PKCS7 separado. |
| Sha384 | `3` | SHA-384. Algoritmo de Hash Seguro 2 |
| Sha512 | `4` | SHA-512. Algoritmo de Hash Seguro 2 |
| Sha3_256 | `5` | SHA3-256. Algoritmo de Hash Seguro 3 |
| Sha3_384 | `6` | SHA3-384. Algoritmo de Hash Seguro 3 |
| Sha3_512 | `7` | SHA3-512. Algoritmo de Hash Seguro 3 |
