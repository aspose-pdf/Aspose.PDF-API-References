## Observaciones

Utilice la clase ComHelper para cargar un documento desde un archivo o stream en un objeto Documento en una aplicación COM. La clase Documento proporciona un constructor predeterminado para crear un nuevo documento y también proporciona constructores sobrecargados para cargar un documento desde un archivo o stream. Si está utilizando Aspose.Words desde una aplicación .NET, puede utilizar todos los constructores de Documento directamente, pero si está utilizando Aspose.Pdf desde una aplicación COM, solo está disponible el constructor predeterminado de Documento.
