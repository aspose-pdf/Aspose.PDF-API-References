### Observaciones

Utilice la clase ComHelper para cargar un documento desde un archivo o secuencia en un objeto Documento en una aplicación COM. La clase Documento proporciona un constructor predeterminado para crear un nuevo documento y también proporciona constructores sobrecargados para cargar un documento desde un archivo o secuencia . Si usa Aspose.Words desde una aplicación .NET, puede usar todos los constructores de documentos directamente, pero si usa Aspose.Pdf desde una aplicación COM, solo está disponible el constructor de documentos predeterminado.
