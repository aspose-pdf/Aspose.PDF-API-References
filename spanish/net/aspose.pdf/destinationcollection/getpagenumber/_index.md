## Método DestinationCollection.GetPageNumber

Devuelve el número de página del destino por el nombre.

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| destinameName | String | El nombre del destino. |
| useCache | Boolean | Determina si se utiliza o no la versión en caché de la colección. |

### Valor de Retorno

El número de página si se encontró el destino; de lo contrario, -1.
