## Método DestinationCollection.GetExplicitDestination

Devuelve el destino explícito por el nombre.

```csharp
public ExplicitDestination GetExplicitDestination(string destinameName, bool useCache)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| destinameName | String | El nombre del destino. |
| useCache | Boolean | Determina si se utiliza o no la versión en caché de la colección. |

### Valor de Retorno

El objeto ExplicitDestination para el destino encontrado; de lo contrario, null.
