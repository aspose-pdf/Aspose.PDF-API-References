## Propiedad CollectionItem.IsEmpty

Obtiene un valor que indica si el elemento de la colección está vacío.

```csharp
public bool IsEmpty { get; }
```

### Valor de Retorno

true si el elemento de la colección está vacío; de lo contrario, false.

## Observaciones

Esta propiedad devuelve true si el elemento de la colección no contiene ningún valor, incluidos valores de cadena, valores dobles, valores enteros y valores de fecha. Si alguno de estos tipos de valor está presente en el elemento de la colección, esta propiedad devuelve false.
