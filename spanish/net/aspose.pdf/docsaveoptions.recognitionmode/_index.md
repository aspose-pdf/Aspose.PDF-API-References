## DocSaveOptions.RecognitionMode enumeration

Permite controlar cómo se convierte un documento PDF en un documento de procesamiento de texto.

```csharp
public enum RecognitionMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Textbox | `0` | Este modo es rápido y bueno para conservar al máximo el aspecto original del archivo PDF, pero la capacidad de edición del documento resultante podría ser limitada. |
| Flow | `1` | Modo de reconocimiento completo, el motor realiza agrupaciones y análisis multinivel para restaurar la intención del autor del documento original y producir un documento editable al máximo. La desventaja es que el documento de salida puede verse diferente del archivo PDF original. |
| EnhancedFlow | `2` | Una versión alfa temprana de un nuevo modo de flujo que admite el reconocimiento de tablas. |

### Observaciones

Utilizar elTextbox mode cuando el documento resultante no va a ser editado mucho más. Los cuadros de texto son fáciles de modificar cuando no hay mucho que hacer.

Utilizar elFlowmodo cuando el documento de salida necesita más edición. Los párrafos y las líneas de texto en el modo de flujo permiten una fácil modificación del texto, pero los objetos de formato no admitidos se verán peor que en el modo de flujo.Textbox modo.
