## Propiedad XImage.Name

Obtiene o establece el nombre de la imagen. Tenga en cuenta que si cambia el nombre de la imagen que tiene referencias en el contenido de la página, el documento puede volverse incorrecto. Utilice el método XImage.Rename en este caso.

```csharp
public string Name { get; set; }
```
