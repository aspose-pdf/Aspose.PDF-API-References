## Propiedad HtmlSaveOptions.SplitIntoPages

Obtiene o establece el indicador que indica si cada página del documento fuente se convertirá en su propio documento HTML de destino, es decir, si el HTML resultante se dividirá en varias páginas HTML.

```csharp
public bool SplitIntoPages { get; set; }
```
