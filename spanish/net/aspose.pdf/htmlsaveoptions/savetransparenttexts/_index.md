## HtmlSaveOptions.SaveTransparentTexts field

Pdf puede contener textos transparentes que se pueden seleccionar en el portapapeles (generalmente sucede cuando el documento contiene imágenes y textos OCRed extraídos de él). Esta configuración le dice al convertidor si necesitamos guardar dichos textos como transparent textos seleccionables en el resultado HTML

```csharp
public bool SaveTransparentTexts;
```
