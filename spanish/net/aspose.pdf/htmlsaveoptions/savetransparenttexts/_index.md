## Campo HtmlSaveOptions.SaveTransparentTexts

Pdf puede contener textos transparentes que se pueden seleccionar para el portapapeles (generalmente sucede cuando el documento contiene imágenes y textos extraídos mediante OCR). Esta configuración indica al convertidor si necesitamos guardar dichos textos como textos seleccionables transparentes en el HTML resultante.

```csharp
public bool SaveTransparentTexts;
```
