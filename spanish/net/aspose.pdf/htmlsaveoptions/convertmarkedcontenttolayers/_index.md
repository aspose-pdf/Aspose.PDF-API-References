## Propiedad HtmlSaveOptions.ConvertMarkedContentToLayers

Si el atributo ConvertMarkedContentToLayers se establece en verdadero, entonces todos los elementos dentro de un contenido marcado de PDF (capa) se colocarán en un div HTML con el atributo "data-pdflayer" que especifica un nombre de capa. Este nombre de capa se extraerá de las propiedades opcionales del contenido marcado de PDF. Si este atributo es falso (por defecto), entonces no se crearán capas a partir del contenido marcado de PDF.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
