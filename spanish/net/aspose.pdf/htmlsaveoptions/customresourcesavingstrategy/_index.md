## HtmlSaveOptions.CustomResourceSavingStrategy field

Este campo puede contener una estrategia de guardado que se debe usar (si está presente) durante la conversión para el manejo personalizado de archivos de recursos referenciados creados (como imágenes y fuentes) relacionados con nodos de HTML guardado. Esa estrategia debe procesar recursos y devolver una cadena que representa la URL deseada del recurso guardado en HTML generado.

public ResourceSavingStrategy CustomResourceSavingStrategy;
