## Propiedad HtmlSaveOptions.SaveFullFont

Indica que se guardará la fuente completa, solo admite fuentes True Type. Por defecto, SaveFullFont = falso y el convertidor guarda el subconjunto de la fuente inicial necesario para mostrar el texto del documento.

```csharp
public bool SaveFullFont { get; set; }
```
