## HtmlSaveOptions.SaveFullFont field

Indica que se guardará la fuente completa, solo admite fuentes True Type. De manera predeterminada, SaveFullFont = false y el convertidor guarda el subconjunto de la fuente inicial necesaria para mostrar el texto del documento.
