## HtmlSaveOptions.UseZOrder property

Si el atributo UseZORder se establece en verdadero, los gráficos y el texto se agregan al documento HTML resultante según el orden Z en el documento PDF original. Si este atributo es falso, todos los gráficos se ponen como una sola capa, lo que puede causar algunos efectos innecesarios para los objetos superpuestos.

public bool UseZOrder { get; set; }
