## Propiedad HtmlSaveOptions.UseZOrder

Si el atributo UseZOrder se establece en verdadero, los gráficos y el texto se agregan al documento HTML resultante de acuerdo con el orden Z en el documento PDF original. Si este atributo es falso, todos los gráficos se colocan como una sola capa, lo que puede causar algunos efectos innecesarios para los objetos superpuestos.

```csharp
public bool UseZOrder { get; set; }
```
