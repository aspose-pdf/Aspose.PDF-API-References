## HtmlSaveOptions.SpecialFolderForSvgImages field

Obtiene o establece la ruta al directorio en el que se deben guardar solo las imágenes SVG si se encuentran al guardar el documento como HTML. Si el parámetro está vacío o es nulo , los archivos SVG (si los hay) se guardarán junto con otros archivos de imagen (cerca del archivo de salida) o en una carpeta especial para imágenes (si se especificó en la opción SpecialImagesFolderIfAny). No afecta nada si la propiedad CustomImageSavingStrategy se usó correctamente para procesar el archivo de imagen relevante.

public string SpecialFolderForSvgImages;
