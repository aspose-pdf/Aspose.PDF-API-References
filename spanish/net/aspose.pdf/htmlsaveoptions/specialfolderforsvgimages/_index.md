## HtmlSaveOptions.SpecialFolderForSvgImages field

Obtiene o establece la ruta al directorio al que deben guardarse solo las imágenes SVG si se encuentran durante el guardado del documento como HTML. Si el parámetro está vacío o es nulo, entonces los archivos SVG (si los hay) se guardarán junto con otros archivos de imagen (cerca del archivo de salida) o en una carpeta especial para imágenes (si se especifica en la opción SpecialImagesFolderIfAny). No afecta nada si la propiedad CustomImageSavingStrategy se utilizó con éxito para procesar el archivo de imagen relevante.

```csharp
public string SpecialFolderForSvgImages;
```
