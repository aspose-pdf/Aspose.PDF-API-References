## Propiedad HtmlSaveOptions.DefaultFontName

Especifica el nombre de una fuente instalada que se utiliza para sustituir cualquier fuente de documento que no esté incrustada y no esté instalada en el sistema. Si es nulo, se utiliza la fuente de sustitución predeterminada.

```csharp
public string DefaultFontName { get; set; }
```

### Valor de la Propiedad

Nombre de la fuente
