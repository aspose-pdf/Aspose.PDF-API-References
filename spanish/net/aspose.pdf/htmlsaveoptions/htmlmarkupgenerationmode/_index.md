## Campo HtmlSaveOptions.HtmlMarkupGenerationMode

A veces, existen requisitos específicos para la generación de marcado HTML. Este parámetro define los modos de preparación de HTML que se pueden utilizar durante la conversión de PDF a HTML para cumplir con dichos requisitos específicos.

```csharp
public HtmlMarkupGenerationModes HtmlMarkupGenerationMode;
```
