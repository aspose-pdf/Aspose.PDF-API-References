## Propiedad HtmlSaveOptions.MinimalLineWidth

Este atributo establece el ancho mínimo de la línea del camino gráfico. Si el grosor de la línea es menor a 1px, Adobe Acrobat lo redondea a este valor. Por lo tanto, este atributo se puede utilizar para emular este comportamiento en navegadores HTML.

```csharp
public float MinimalLineWidth { get; set; }
```
