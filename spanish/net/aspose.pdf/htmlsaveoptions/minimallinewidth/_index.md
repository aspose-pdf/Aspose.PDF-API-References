## HtmlSaveOptions.MinimalLineWidth property

Este atributo establece el ancho mínimo de la línea de la ruta del gráfico. Si el grosor de la línea es inferior a 1 px, Adobe Acrobat lo redondea a este valor. Entonces, este atributo puede usarse para emular este comportamiento para navegadores HTML.

```csharp
public float MinimalLineWidth { get; set; }
```
