## Propiedad HtmlSaveOptions.RenderTextAsImage

Si el atributo RenderTextAsImage se establece en verdadero, el texto de la fuente se convierte en una imagen en HTML. Puede ser útil para hacer que el texto no sea seleccionable o si el texto HTML no se renderiza correctamente.

```csharp
public bool RenderTextAsImage { get; set; }
```
