## Propiedad HtmlSaveOptions.FontSources

Fuentes de fuentes preguardadas.

```csharp
public FontSourceCollection FontSources { get; }
```

## Comentarios

Las fuentes pueden ser guardadas preliminarmente con el propósito de caché y luego pasadas al proceso de conversión a Html. Por ejemplo, puede ser útil en un escenario de división de documentos y procesamiento de páginas de documentos en múltiples hilos con un solo conjunto de fuentes.
