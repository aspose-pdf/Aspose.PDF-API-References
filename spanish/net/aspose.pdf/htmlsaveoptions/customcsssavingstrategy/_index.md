## Campo HtmlSaveOptions.CustomCssSavingStrategy

Este campo puede contener la estrategia de guardado que debe usarse (si está presente) durante la conversión de Pdf a Html para el manejo del guardado de los CSS relacionados con el documento HTML creado en su totalidad o con sus páginas (si se generan varias páginas HTML). Si desea manejar el archivo CSS de alguna manera específica, simplemente cree el método relevante y asigne el delegado creado a partir de él a esta propiedad.

```csharp
public CssSavingStrategy CustomCssSavingStrategy;
```
