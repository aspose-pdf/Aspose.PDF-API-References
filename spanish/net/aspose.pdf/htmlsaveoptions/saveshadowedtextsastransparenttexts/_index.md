## HtmlSaveOptions.SaveShadowedTextsAsTransparentTexts field

Pdf puede contener textos sombreados por otros elementos (por ejemplo, por imágenes), pero se puede seleccionar en el portapapeles en Acrobat Reader (generalmente sucede cuando el documento contiene imágenes y textos OCR extraídos de él). Esta configuración le dice al convertidor si necesita guardar dichos textos como transparent textos seleccionables en el resultado HTML para imitar el comportamiento de Acrobat Reader (de lo contrario, dichos textos generalmente se guardan como ocultos, no disponibles para copiar al portapapeles)

```csharp
public bool SaveShadowedTextsAsTransparentTexts;
```
