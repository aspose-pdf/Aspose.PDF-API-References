## SaveOptions.ResourceSavingInfo.CustomProcessingCancelled field

este indicador debe establecerse en "verdadero" en el código personalizado si, por alguna razón, el archivo propuesto debe procesarse no con el código personalizado sino con el propio código del convertidor de forma estándar para el convertidor. Por lo tanto, su configuración establecida en verdadero significa que el código personalizado no procesó el archivo de referencia y el convertidor debe manejarlo por sí mismo (en ambos sentidos, para guardar en algún lugar y para nombrar en el archivo de referencia).

```csharp
public bool CustomProcessingCancelled;
```
