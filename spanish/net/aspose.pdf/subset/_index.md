## Enumeración de Subconjunto

Representa el subconjunto de páginas al que puede aplicarse un artefacto de paginación.

```csharp
public enum Subset
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Todos | `0` | Indica que el artefacto se aplica a todas las páginas. |
| Pares | `1` | Indica que el artefacto se aplica solo a las páginas con números pares. |
| Impares | `2` | Indica que el artefacto se aplica solo a las páginas con números impares. |
