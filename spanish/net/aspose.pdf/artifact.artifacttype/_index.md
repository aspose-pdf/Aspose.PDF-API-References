## Enumeración Artifact.ArtifactType

Enumeración de posibles tipos de artefactos.

```csharp
public enum ArtifactType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Pagination | `0` | Artefactos de paginación. Características auxiliares de la página como encabezados y folios (números de página). |
| Layout | `1` | Artefactos de diseño. Elementos tipográficos o de diseño puramente cosméticos como reglas de notas al pie o pantallas de fondo. |
| Page | `2` | Artefactos de página. Ayudas de producción ajenas al documento en sí, como marcas de corte y barras de color. |
| Background | `3` | Artefactos de fondo. Imágenes, patrones o bloques de color. |
| Undefined | `4` | El tipo de artefacto no está definido o es desconocido. |
