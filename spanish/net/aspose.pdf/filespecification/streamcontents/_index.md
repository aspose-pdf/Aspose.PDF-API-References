## Propiedad FileSpecification.StreamContents

Obtiene el contenido del archivo como un flujo. El contenido no se carga en la memoria, lo que permite reducir el uso de memoria. Pero este flujo no admite posicionamiento ni la propiedad Length. Si necesita estas características, utilice la propiedad Contents en su lugar.

```csharp
public Stream StreamContents { get; }
```
