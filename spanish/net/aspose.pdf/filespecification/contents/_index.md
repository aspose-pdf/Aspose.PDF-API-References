## Propiedad FileSpecification.Contents

Obtiene o establece el archivo de contenido. Esta propiedad devuelve datos cargados en memoria que pueden causar una excepción de falta de memoria para datos grandes. Para disminuir el uso de memoria, utilice StreamContents.

```csharp
public Stream Contents { get; set; }
```
