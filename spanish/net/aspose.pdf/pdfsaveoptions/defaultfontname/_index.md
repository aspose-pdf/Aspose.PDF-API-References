## Propiedad PdfSaveOptions.DefaultFontName

Nombre de fuente utilizado por defecto para fuentes que están ausentes en la computadora. Cuando el documento PDF que se guarda en PDF contiene fuentes que no están disponibles en el documento en sí y en el dispositivo, la API reemplaza estas fuentes con la fuente predeterminada (si se encuentra una fuente con `DefaultFontName` en el dispositivo).

```csharp
public string DefaultFontName { get; set; }
```
