## Campo HtmlSaveOptions.CssSavingInfo.CssNumber

Establecido por el convertidor. Durante la conversión se crean varios archivos CSS. Esta propiedad muestra el ordinal del archivo CSS guardado durante la conversión. Se puede utilizar en la lógica del código personalizado para decidir cómo procesar o dónde guardar el contenido CSS

```csharp
public int CssNumber;
```
