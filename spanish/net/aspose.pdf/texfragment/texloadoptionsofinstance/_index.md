## TeXFragment.TeXLoadOptionsOfInstance property

Obtiene o establece TeXLoadOptions que se usarán para cargar (y renderizar) LaTeX en esta instancia de clase. Úselo cuando sea necesario use una configuración específica para importar LaTeX para esta o aquella instancia (por ejemplo, cuando esta o aquella instancia debería use BasePath específico para LaTeX importado o debe usar un cargador específico de recursos externos) Si el parámetro es predeterminado (nulo), entonces se usarán las opciones estándar de carga de LaTeX.

```csharp
public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
```
