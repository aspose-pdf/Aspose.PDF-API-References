## Propiedad Image.IsBlackWhite

Obtiene o establece un valor bool que indica si la imagen está forzada a ser en blanco y negro. Si se utiliza una imagen TIFF de subformato CCITT, esta propiedad debe establecerse en verdadero.

```csharp
public bool IsBlackWhite { get; set; }
```
