## LoadOptions.WarningHandler property

Devolución de llamada para manejar cualquier advertencia generada. WarningHandler devuelve el elemento de enumeración ReturnAction especificando Continuar o Anular. Continuar es la acción predeterminada y la operación de carga continúa; sin embargo, el usuario también puede devolver Anular, en cuyo caso la operación de carga debe cesar.
