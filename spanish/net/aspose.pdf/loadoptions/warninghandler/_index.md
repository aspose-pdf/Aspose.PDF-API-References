## Propiedad LoadOptions.WarningHandler

Callback para manejar cualquier advertencia generada. El WarningHandler devuelve un elemento de enumeración ReturnAction que especifica Continuar o Abort. Continuar es la acción predeterminada y la operación de carga continúa, sin embargo, el usuario también puede devolver Abort, en cuyo caso la operación de carga debe cesar.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
