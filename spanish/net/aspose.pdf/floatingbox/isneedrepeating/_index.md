## Propiedad FloatingBox.IsNeedRepeating

Obtiene o establece un valor booleano que indica si el párrafo necesita ser repetido en la siguiente página. El valor predeterminado es falso. El atributo solo es válido cuando el párrafo en sí y el objeto al que se refiere su ReferenceParagraphID están incluidos en RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
