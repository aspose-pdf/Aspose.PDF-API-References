## HtmlSaveOptions.RasterImagesSavingModes enumeration

El PDF convertido puede contener imágenes de trama (.png, *.jpeg, etc.) Esta enumeración define métodos de cómo se pueden manejar las imágenes de trama durante la conversión de PDF a HTML

```csharp
public enum RasterImagesSavingModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| AsPngImagesEmbeddedIntoSvg | `0` | para cada archivo ráster distinto se generará una imagen SVG envolvente, y la imagen ráster se incrustará como cadenas codificadas en Base64 en esa imagen SVG |
| AsExternalPngFilesReferencedViaSvg | `1` | distintas imágenes rasterizadas se separarán como archivos PNG, pero se hará referencia a ellas a través de imágenes SVG envolventes, es decir, se generará un archivo PNG y un SVG para cada imagen rasterizada, y cada uno de estos SVG contendrá enlaces a archivos PNG relevantes |
| AsEmbeddedPartsOfPngPageBackground | `2` | Se generará un archivo de fondo PNG grande para cada página de resultados. Las imágenes ráster se incrustarán en ese archivo y se representarán como regiones de esa imagen. No se generarán archivos PNG externos para cada imagen, solo un archivo PNG por página estará presente en el conjunto de resultados de conversión de archivos. |
