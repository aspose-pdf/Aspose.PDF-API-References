## Método OutputIntents.Add

Agrega una intención de salida a la colección.

```csharp
public void Add(OutputIntent item)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| item | OutputIntent | La intención de salida que se va a agregar a la colección. |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentNullException | El *item* es nulo. |
| InvalidOperationException | El documento que contiene la colección no tiene un catálogo para acceder a las OutputIntents. |
