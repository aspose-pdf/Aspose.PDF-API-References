## Método DocumentInfo.IsPredefinedKey

Determina si la clave es predefinida (Título, Autor, etc.), no personalizada.

```csharp
public static bool IsPredefinedKey(string key)
```

| Parámetro | Tipo | Descripción |

| key | String | Clave seleccionada |

Verdadero en caso de que la clave sea predefinida.
