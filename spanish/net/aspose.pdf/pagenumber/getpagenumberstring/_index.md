## Método PageNumber.GetPageNumberString

Devuelve una cadena formateada que representa el número de página según la configuración actual.

```csharp
public string GetPageNumberString(int pageNumber, int count)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| pageNumber | Int32 | El número de página actual. |
| count | Int32 | El número total de páginas. |

### Valor de retorno

Una cadena de número de página formateada.
