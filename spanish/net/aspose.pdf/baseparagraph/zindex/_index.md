## Propiedad BaseParagraph.ZIndex

Obtiene o establece un valor int que indica el Z-order del gráfico. Un gráfico con un ZIndex mayor se colocará sobre el gráfico con un ZIndex menor. ZIndex puede ser negativo. Un gráfico con ZIndex negativo se colocará detrás del texto en la página.

```csharp
public int ZIndex { get; set; }
```
