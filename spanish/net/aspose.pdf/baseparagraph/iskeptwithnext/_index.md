## Propiedad BaseParagraph.IsKeptWithNext

Obtiene o establece un valor bool que indica si el párrafo actual permanece en la misma página junto con el siguiente párrafo. El valor predeterminado es falso. (para generación de pdf)

public bool IsKeptWithNext { get; set; }
