## Enumeración ExtractImageMode

Define diferentes modos que se pueden utilizar al extraer imágenes de documentos.

```csharp
public enum ExtractImageMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DefinedInResources | `0` | Define el modo de extracción de imágenes en el que se extraen todas las imágenes definidas en los recursos para una página particular. |
| ActuallyUsed | `1` | Define el modo de extracción de imágenes en el que solo se extraen aquellas imágenes que se muestran realmente en una página. |
