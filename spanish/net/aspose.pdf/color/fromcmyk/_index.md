## Método Color.FromCmyk

Obtiene un objeto Color pdf válido a partir de componentes de color CMYK.

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | Doble | El componente de color Cian (valor 0 - 1). |
| m | Doble | El componente de color Magenta (valor 0 - 1). |
| y | Doble | El componente de color Amarillo (valor 0 - 1). |
| k | Doble | El componente de color Clave (valor 0 - 1). |

### Valor de Retorno

Objeto Color con cada valor de componente en el rango [0..1].
