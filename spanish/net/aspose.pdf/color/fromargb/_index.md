## FromArgb(int, int, int) {#fromargb}

Obtiene un objeto Color pdf válido a partir de componentes de color RGB.

```csharp
public static Color FromArgb(int r, int g, int b)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| r | Int32 | El componente de color Rojo (valor 0 - 255). |
| g | Int32 | El componente de color Verde (valor 0 - 255). |
| b | Int32 | El componente de color Azul (valor 0 - 255). |

### Valor de Retorno

Objeto Color con cada valor de componente en el rango [0..255].
