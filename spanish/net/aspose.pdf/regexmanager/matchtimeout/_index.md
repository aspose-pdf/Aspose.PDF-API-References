## Propiedad RegexManager.MatchTimeout

Obtiene o establece el tiempo de espera para las operaciones de Regex en toda la biblioteca. El valor predeterminado es 1000 ms.

```csharp
public static TimeSpan MatchTimeout { get; set; }
```

### Valor de la Propiedad

Un TimeSpan que representa la duración del tiempo de espera predeterminado.
