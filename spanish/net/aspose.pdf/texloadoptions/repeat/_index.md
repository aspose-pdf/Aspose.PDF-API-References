## Propiedad TeXLoadOptions.Repeat

Obtiene/establece la bandera que indica si es necesario ejecutar el trabajo de TeX dos veces en caso de que, por ejemplo, haya referencias en el/los archivo(s) TeX de entrada. En general, este comportamiento es útil cuando el motor recopila algunos datos a lo largo del proceso de composición y los almacena en un archivo auxiliar, todo en la primera ejecución. Y en la segunda ejecución, el motor de alguna manera utiliza esos datos.

```csharp
public bool Repeat { get; set; }
```
