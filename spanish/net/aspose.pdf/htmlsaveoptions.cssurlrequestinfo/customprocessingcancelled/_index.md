## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled field

Debe ser establecido por código personalizado si no puede o no debe definir la URL que se utilizará en el HTML generado para hacer referencia a ese CSS. Si es 'true', entonces el archivo CSS se guardará de manera estándar en un lugar estándar.

```csharp
public bool CustomProcessingCancelled;
```
