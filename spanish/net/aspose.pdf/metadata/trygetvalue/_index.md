## Método Metadata.TryGetValue

Intenta encontrar la clave en el diccionario y recupera el valor si se encuentra.

```csharp
public bool TryGetValue(string key, out XmpValue value)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| key | String | Clave para buscar en el diccionario. |
| value | XmpValue& | Valor recuperado. |

### Valor de Retorno

true si se encontró la clave.
