## Add() {#add}

Agrega una página vacía. Si el documento ya contiene páginas de diferentes tamaños, se seleccionará el tamaño de la página que más frecuentemente ocurre. En caso de que solo haya dos páginas diferentes, se utilizará el tamaño de la primera página.

```csharp
public Page Add()
```

### Valor de Retorno

Página agregada.
