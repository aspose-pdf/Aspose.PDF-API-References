## Método Document.HasIncrementalUpdate

Verifica si el documento PDF actual ha sido guardado con actualizaciones incrementales.

```csharp
public bool HasIncrementalUpdate()
```

### Valor de Retorno

`true` si el documento PDF tiene actualizaciones incrementales; de lo contrario, `false`.
