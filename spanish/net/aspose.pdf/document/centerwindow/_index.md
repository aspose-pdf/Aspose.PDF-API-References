## Propiedad Document.CenterWindow

Obtiene o establece un indicador que especifica si la posición de la ventana del documento estará centrada en la pantalla.

```csharp
public bool CenterWindow { get; set; }
```

El ejemplo demuestra cómo obtener el indicador CenterWindow:

