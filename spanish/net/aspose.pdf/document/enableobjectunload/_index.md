## Propiedad Document.EnableObjectUnload

Obtiene o establece una bandera que permite que el documento se descargue parcialmente de la memoria. Esto permite disminuir el uso de memoria, pero puede tener un efecto negativo en el rendimiento.

```csharp
public bool EnableObjectUnload { get; set; }
```
