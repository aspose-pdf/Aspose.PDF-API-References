## Propiedad Document.OptimizeSize

Obtiene o establece la bandera de optimización. Cuando se agregan páginas al documento, los flujos de recursos iguales en el archivo resultante se fusionan en un objeto PDF si esta bandera está establecida. Esto permite reducir el tamaño del archivo resultante, pero puede causar una ejecución más lenta y mayores requisitos de memoria. Valor predeterminado: falso.

```csharp
public bool OptimizeSize { get; set; }
```
