## Propiedad Document.EmbedStandardFonts

Propiedad que declara que el documento debe incrustar todas las fuentes estándar Type1 que tienen la bandera IsEmbedded establecida en verdadero. Todas las fuentes PDF se pueden incrustar en el documento simplemente configurando la bandera IsEmbedded en verdadero, pero las fuentes estándar Type1 de PDF son una excepción a esta regla. La incrustación de fuentes estándar Type1 requiere mucho tiempo, por lo que para incrustar estas fuentes no solo es necesario establecer la bandera IsEmbedded en verdadero para la fuente especificada, sino también establecer una bandera adicional a nivel de documento - EmbedStandardFonts = true; Esta propiedad solo se puede establecer una vez para todas las fuentes. Por defecto falso.

```csharp
public bool EmbedStandardFonts { get; set; }
```
