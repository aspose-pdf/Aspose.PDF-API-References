## Propiedad Document.Metadata

Metadatos del documento. (Un documento PDF puede incluir información general, como el título del documento, el autor y las fechas de creación y modificación. Dicha información global sobre el documento (en contraposición a su contenido o estructura) se llama metadatos y está destinada a ayudar en la catalogación y búsqueda de documentos en bases de datos externas.)

public Metadata Metadata { get; }
