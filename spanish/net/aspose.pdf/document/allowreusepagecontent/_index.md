## Propiedad Document.AllowReusePageContent

Permite fusionar el contenido de las páginas para optimizar el tamaño del documento. Si se utiliza, entonces páginas diferentes pero duplicadas pueden hacer referencia al mismo objeto de contenido. Tenga en cuenta que este modo puede causar efectos secundarios, como cambiar el contenido de la página cuando se cambia otra página.

```csharp
public bool AllowReusePageContent { get; set; }
```
