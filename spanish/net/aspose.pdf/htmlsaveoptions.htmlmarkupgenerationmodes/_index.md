## Enumeración HtmlSaveOptions.HtmlMarkupGenerationModes

A veces hay requisitos específicos para el HTML creado. Este enum define los modos de preparación de HTML que se pueden utilizar durante la conversión de PDF a HTML para cumplir con dichos requisitos específicos.

```csharp
public enum HtmlMarkupGenerationModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| WriteAllHtml | `0` | Modo predeterminado cuando no hay requisitos específicos. Se generará una salida que contendrá todas las partes del HTML sin ningún procesamiento adicional especial. |
| WriteOnlyBodyContent | `1` | Se eliminará todo el contenido HTML que esté fuera del cuerpo del HTML, es decir, solo se dejará el contenido que esté dentro de las etiquetas \c \ \c \ |
