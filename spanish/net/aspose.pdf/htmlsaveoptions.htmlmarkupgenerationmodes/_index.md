## HtmlSaveOptions.HtmlMarkupGenerationModes enumeration

A veces existen requisitos específicos para el HTML creado. Esta enumeración define los modos de preparación de HTML que se pueden usar durante la conversión de PDF a HTML para cumplir con dichos requisitos específicos.

```csharp
public enum HtmlMarkupGenerationModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| WriteAllHtml | `0` | Modo predeterminado: no hay ningún requisito específico. Se generará una salida que contendrá todas las partes de HTML sin ningún procesamiento adicional especial. |
| WriteOnlyBodyContent | `1` | se eliminará todo el contenido HTML que está fuera del cuerpo de HTML, es decir, solo quedará el contenido que está dentro etiquetas |
