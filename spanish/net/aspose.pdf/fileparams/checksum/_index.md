## Propiedad FileParams.CheckSum

Una cadena de 16 bytes que es la suma de verificación de los bytes del archivo incrustado descomprimido. La suma de verificación se calcula aplicando el algoritmo estándar de resumen de mensaje MD5 a los bytes del flujo del archivo incrustado.

```csharp
public string CheckSum { get; }
```
