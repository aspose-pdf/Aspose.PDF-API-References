## Método XmpField.Equals

Indica si esta instancia y un objeto especificado son iguales.

```csharp
public override bool Equals(object obj)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| obj | Object | Otro objeto con el que comparar. |

### Valor de Retorno

true si obj y esta instancia son del mismo tipo y representan el mismo valor; de lo contrario, false.
