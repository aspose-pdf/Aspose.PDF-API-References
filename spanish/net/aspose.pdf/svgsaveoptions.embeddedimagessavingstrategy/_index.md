## SvgSaveOptions.EmbeddedImagesSavingStrategy delegate

A una propiedad de este tipo Puede asignar un delegado creado a partir de un método personalizado que implementa el procesamiento de guardado externo de la imagen que se extrajo de un SVG creado a partir de PDF y debe guardarse como recurso externo durante la conversión de PDF a HTML. En tal caso, el procesamiento (como el guardado propio en la transmisión o en el disco) se puede hacer en ese código personalizado y ese código personalizado debe devolver la ruta (o cualquier otra cadena sin comillas) que luego se incorporará al SVG generado en lugar de la supuesta ruta original a ese recurso de imagen. En tal caso, se deben realizar todas las acciones necesarias para guardar la imagen en el código del método proporcionado, porque no se usará el guardado del resultado en el código del convertidor. Si el procesamiento de este o aquel archivo por algún motivo debe realizarse mediante el propio código del convertidor, no en el código personalizado, establezca en el indicador de código personalizado 'CustomProcessingCancelled' de la variable del parámetro 'imageSavingInfo'. Le indica al convertidor que todos los pasos necesarios para el procesamiento de ese recurso debe realizarse en el convertidor como si no hubiera ningún código personalizado externo . representa información sobre la imagen guardada que se puede usar en código personalizadodebe devolver una cadena que represente la URL de la imagen que se colocará en SVG
