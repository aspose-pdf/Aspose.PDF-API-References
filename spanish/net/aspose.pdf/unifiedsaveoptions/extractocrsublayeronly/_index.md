## Propiedad UnifiedSaveOptions.ExtractOcrSublayerOnly

Este atributo activó la funcionalidad para extraer imagen o texto para documentos PDF con subcapa OCR.

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### Valor de la Propiedad

`true` el texto será extraído en el documento resultante; de lo contrario, `false`.
