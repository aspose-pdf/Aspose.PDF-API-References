## Propiedad RenderingOptions.IgnoreResourceFontErrors

Obtiene o establece la indicación de que se ignorarán los errores relacionados con la ausencia de fuente. true - significa que se ignorarán los errores de ausencia de fuente. Los segmentos de texto que se refieren a recursos incorrectos se omitirán durante el procesamiento. false por defecto

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
