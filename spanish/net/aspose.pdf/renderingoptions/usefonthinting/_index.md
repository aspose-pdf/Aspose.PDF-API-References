## RenderingOptions.UseFontHinting property

El uso de esta bandera activa el mecanismo de sugerencias de fuente. La sugerencia de fuente es el uso de instrucciones matemáticas para ajustar la visualización de una fuente de contorno. En algunos casos, activar esta bandera puede resolver problemas con la legibilidad del texto. En el momento actual, el uso de este indicador podría tener efecto solo para las fuentes TTF, si estas fuentes se usan en el documento de origen.

public bool UseFontHinting { get; set; }
