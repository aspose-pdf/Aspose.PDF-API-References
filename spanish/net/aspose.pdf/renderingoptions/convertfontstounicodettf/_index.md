## Propiedad RenderingOptions.ConvertFontsToUnicodeTTF

Indica que todas las fuentes se convertirán a versiones TTF unicode. Eso es útil por razones de compatibilidad y para optimizar el uso de fuentes, ya que cada nueva fuente TTF no tendrá todos los símbolos de la fuente original, sino solo los símbolos que se utilizan en el texto.

```csharp
public bool ConvertFontsToUnicodeTTF { get; set; }
```
