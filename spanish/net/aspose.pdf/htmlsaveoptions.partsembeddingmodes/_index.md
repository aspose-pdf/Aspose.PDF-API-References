## HtmlSaveOptions.PartsEmbeddingModes enumeration

Esta enumeración enumera posibles modos de incrustación de archivos a los que se hace referencia en HTML Permite controlar si los archivos a los que se hace referencia (HTML, fuentes, imágenes, CSS) se incrustarán en el archivo HTML principal o se generarán como entidades binarias separadas

```csharp
public enum PartsEmbeddingModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| EmbedAllIntoHtml | `0` | Obliga a incrustar todos los archivos a los que se hace referencia (Css, Imágenes, Fuentes) en el marcado HTML generado (es decir, en el mismo HTML) Este enfoque genera un archivo HTML, pero el tamaño total de salida se vuelve más grande (porque se usa la codificación Base64 de binarios) y no todos los navegadores (especialmente los heredados) procesan correctamente los archivos binarios incrustados en HTML. Pero permite obtener HTML que contiene el resultado completo, sin ningún archivo adicional. |
| EmbedCssOnly | `1` | Obliga a separar todos los archivos a los que se hace referencia, excepto CSS (imágenes y fuentes) Es decir, CSS se incrustará en el resultado HTML, y todos los demás archivos a los que se hace referencia (imágenes y fuentes) se procesarán como partes externas Genera HTML que es adecuado para un conjunto amplio de navegadores |
| NoEmbedding | `2` | Obliga a separar los archivos referenciados (Css, Imágenes, Fuentes) Este enfoque genera un conjunto de archivos, pero el tamaño total de la salida se vuelve más pequeño (porque no se usa la codificación Base64 de binarios) También este enfoque genera HTML que es adecuado para amplio conjunto de navegadores |
