## TextFragmentAbsorber.TextSearchOptions property

Obtiene o establece las opciones de búsqueda. Las opciones habilitan la búsqueda usando expresiones regulares.

```csharp
public TextSearchOptions TextSearchOptions { get; set; }
```

### Ejemplos

El ejemplo demuestra cómo realizar una búsqueda de texto usando una expresión regular.

```csharp
// Abrir documento
Document doc = new Document(@"D:\Tests\input.pdf");

// Crear objeto TextFragmentAbsorber
TextFragmentAbsorber absorber = new TextFragmentAbsorber();

// hacer que el absorbedor busque todas las palabras que comienzan con 'h' y terminan con 'o' usando una expresión regular.
absorber.Phrase = @"h\w*?o";
absorber.TextSearchOptions = new TextSearchOptions(true);

// debemos encontrar la palabra "hola" y reemplazarla con "Hola"
doc.Pages[1].Accept(absorber);
absorber.TextFragments[1].Text = "Hi"; 

// Guardar documento
doc.Save(@"D:\Tests\output.pdf"); 
```
