## Método FontRepository.LoadFonts

Carga fuentes instaladas en el sistema y fuentes Pdf estándar. Este método fue diseñado para acelerar el proceso de carga de fuentes. Por defecto, las fuentes se cargan en la primera solicitud de cualquier fuente. El uso de este método carga las fuentes del sistema y las fuentes Pdf estándar inmediatamente antes de que se abra cualquier documento Pdf.

```csharp
public static void LoadFonts()
```
