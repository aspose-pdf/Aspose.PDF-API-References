## Font.DecodedFontName property

A veces, las fuentes PDF (generalmente fuentes chinas/japonesas/coreanas) pueden tener un nombre de fuente específico. Este nombre es el valor de la propiedad de fuente PDF "BaseFont" y, a veces, esta propiedad se puede representar en forma hexadecimal. Si lee este nombre directamente, podría representarse en forma no legible. Para obtener una forma legible, es necesario decodificar el nombre de la fuente mediante reglas específicas para esta fuente. Esta propiedad devuelve el nombre de la fuente decodificada, así que utilícela en los casos en que se encuentre con con un nombre no legible.[`FontName`](../fontname) . Si propiedad[`FontName`](../fontname) tiene forma legible esta propiedad será la misma que [`FontName`](../fontname) , por lo que puede usar esta propiedad en cualquier caso cuando necesite obtener el nombre de la fuente en una forma legible.

public string DecodedFontName { get; }
