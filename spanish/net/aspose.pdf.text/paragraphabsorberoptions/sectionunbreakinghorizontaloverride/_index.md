## Propiedad ParagraphAbsorberOptions.SectionUnbreakingHorizontalOverride

Obtiene o establece la longitud máxima de la zona con 'nivel de llenado cero' que no se tratará como un verdadero salto de sección horizontal (como una fracción del ancho / alto de una página). El valor predeterminado es 0.005.

```csharp
public double SectionUnbreakingHorizontalOverride { get; set; }
```
