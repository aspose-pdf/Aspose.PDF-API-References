## Enumeración TextFormattingOptions.WordWrapMode

Define estrategias de ajuste de palabras

```csharp
public enum WordWrapMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| NoWrap | `0` | No se realiza ajuste |
| DiscretionaryHyphenation | `1` | Se realiza separación de sílabas discrecional. Permite romper palabras en el medio. |
| ByWords | `2` | El ajuste de palabras solo envuelve palabras completas. Si la palabra completa no se puede ajustar, intenta usar separación de sílabas discrecional |
| Undefined | `3` | WordWrapMode no establecido. Se utilizarán las estrategias de ajuste de la estructura de nivel superior (celda de tabla, párrafo, etc.). |
