## TextExtractionOptions.ScaleFactor property

Obtiene o establece el factor que se aplicará para escalar el tamaño de la fuente durante la extracción en modo puro. Establecer un valor menor genera más espacios en el texto extraído. El valor predeterminado es 1: sin escala; Establecer el valor en cero permite elegir el algoritmo escalando automáticamente.

public double ScaleFactor { get; set; }
