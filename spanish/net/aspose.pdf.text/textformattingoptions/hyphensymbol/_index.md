## Propiedad TextFormattingOptions.HyphenSymbol

Obtiene o establece el símbolo de guion que se utiliza en el proceso de separación de sílabas.

```csharp
public string HyphenSymbol { get; set; }
```

## Observaciones

Para eliminar el dibujo del guion (con el procedimiento de ajuste aún en su lugar), establezca una cadena vacía string.Empty para HyphenSymbol.
