## Propiedad TextAbsorber.Errors

Lista de [`TextExtractionError`](../../textextractionerror/) objetos. Contiene información sobre los errores encontrados durante la extracción de texto. La búsqueda de errores se realizará solo si TextSearchOptions.LogTextExtractionErrors = true; y puede disminuir el rendimiento.

```csharp
public List<TextExtractionError> Errors { get; }
```
