### Observaciones

Realiza la extracción de texto y proporciona acceso al texto extraído a través de[`Text`](../text) objeto.

### Ejemplos

El ejemplo muestra cómo extraer texto de todas las páginas del documento PDF.

```csharp
// abrir documento
Document doc = new Document(inFile);

// crea un objeto TextAbsorber para extraer texto
TextAbsorber absorber = new TextAbsorber();

// aceptar el absorbedor para todas las páginas del documento
doc.Pages.Accept(absorber);

// obtener el texto extraído
string extractedText = absorber.Text;

```
