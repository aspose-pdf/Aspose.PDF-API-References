## Propiedad TextAbsorber.TextSearchOptions

Obtiene o establece opciones de búsqueda de texto.

```csharp
public virtual TextSearchOptions TextSearchOptions { get; set; }
```

## Observaciones

Permite definir un rectángulo que delimita el texto extraído. Por defecto, el rectángulo está vacío. Eso significa que los límites de la página solo definen la región de extracción de texto.
