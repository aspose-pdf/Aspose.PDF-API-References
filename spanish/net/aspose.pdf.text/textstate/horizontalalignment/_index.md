## Propiedad TextState.HorizontalAlignment

Obtiene o establece la alineación horizontal para el texto.

```csharp
public virtual HorizontalAlignment HorizontalAlignment { get; set; }
```

## Observaciones

HorizontalAlignment.None es igual a HorizontalAlignment.Left. Tenga en cuenta que la propiedad TextState.HorizontalAlignment funciona solo en escenarios de generación de nuevos documentos.
