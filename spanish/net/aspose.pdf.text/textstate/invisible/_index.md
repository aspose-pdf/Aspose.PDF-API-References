## Propiedad TextState.Invisible

Obtiene o establece la invisibilidad del texto. Esto refleja básicamente el estado de [`RenderingMode`](../renderingmode/), excepto en algunos casos especiales (como el recorte).

```csharp
public virtual bool Invisible { get; set; }
```
