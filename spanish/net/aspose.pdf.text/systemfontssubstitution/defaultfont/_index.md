## Propiedad SystemFontsSubstitution.DefaultFont

Obtiene o establece la fuente de sustitución predeterminada. La fuente se utiliza cuando no se encontró ninguna otra sustitución válida, pero la fuente inicial pertenece a la categoría de sustitución objetivo ([`FontCategories`](../fontcategories/)).

```csharp
public Font DefaultFont { get; set; }
```
