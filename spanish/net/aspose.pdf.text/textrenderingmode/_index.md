## TextRenderingMode enumeration

El modo de representación de texto, Tmode, determina si mostrar texto hará que los contornos de los glifos se tracen, rellenen, se utilicen como un límite de recorte o alguna combinación de los tres.

```csharp
public enum TextRenderingMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| FillText | `0` | Texto de relleno. |
| StrokeText | `1` | Trazar texto. |
| FillThenStrokeText | `2` | Rellenar, luego trazar texto. |
| Invisible | `3` | Ni relleno ni trazo de texto (invisible). |
| FillTextAndAddPathToClipping | `4` | Rellene el texto y agréguelo a la ruta para recortarlo (consulte 9.3.6, "Modo de representación de texto"). |
| StrokeTextAndAddPathToClipping | `5` | Trazar texto y agregar a la ruta para recortar. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Rellene, luego delinee el texto y agréguelo a la ruta para recortarlo. |
| AddPathToClipping | `7` | Agregar texto a la ruta para recortar. |
