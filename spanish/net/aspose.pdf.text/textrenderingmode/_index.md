## Enumeración TextRenderingMode

El modo de renderizado de texto, Tmode, determina si mostrar texto hará que los contornos de los glifos se dibujen, se rellenen, se utilicen como un límite de recorte o alguna combinación de los tres.

```csharp
public enum TextRenderingMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| FillText | `0` | Rellenar texto. |
| StrokeText | `1` | Dibujar texto. |
| FillThenStrokeText | `2` | Rellenar, luego dibujar texto. |
| Invisible | `3` | Ni rellenar ni dibujar texto (invisible). |
| FillTextAndAddPathToClipping | `4` | Rellenar texto y agregar al camino para recorte (ver 9.3.6, "Modo de Renderizado de Texto,"). |
| StrokeTextAndAddPathToClipping | `5` | Dibujar texto y agregar al camino para recorte. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Rellenar, luego dibujar texto y agregar al camino para recorte. |
| AddPathToClipping | `7` | Agregar texto al camino para recorte. |
