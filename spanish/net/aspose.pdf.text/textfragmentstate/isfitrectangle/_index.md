## Método TextFragmentState.IsFitRectangle

Verifica si la cadena de entrada podría colocarse dentro del rectángulo definido.

```csharp
public bool IsFitRectangle(string str, Rectangle rect)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| str | String | Cadena a verificar. |
| rect | Rectangle | Rectángulo a verificar. |

### Valor de Retorno

Verdadero si la cadena se ajusta al rectángulo; de lo contrario, falso.
