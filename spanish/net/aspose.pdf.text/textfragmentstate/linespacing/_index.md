## Propiedad TextFragmentState.LineSpacing

Obtiene o establece el espaciado de línea del texto.

```csharp
public override float LineSpacing { get; set; }
```

## Observaciones

Tenga en cuenta que el valor no se conserva como una característica del texto dentro del documento. El getter de la propiedad LineSpacing funciona para un objeto en caso de que se haya establecido explícitamente anteriormente con el setter de LineSpacing para esos objetos. La propiedad es utilizada por el tiempo de ejecución en el contexto del proceso actual de generación/modificación.
