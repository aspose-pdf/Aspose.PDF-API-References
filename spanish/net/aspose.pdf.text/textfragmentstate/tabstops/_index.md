## Propiedad TextFragmentState.TabStops

Obtiene los tabuladores para el texto.

```csharp
public TabStops TabStops { get; }
```

## Observaciones

Tenga en cuenta que la propiedad Tabstops funciona solo en escenarios de generación de nuevos documentos. Los tabuladores pueden ser añadidos durante la inicialización de [`TextFragment`](../../textfragment/). Los tabuladores deben ser construidos antes del texto.
