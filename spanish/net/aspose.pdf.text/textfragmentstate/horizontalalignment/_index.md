## Propiedad TextFragmentState.HorizontalAlignment

Obtiene o establece la alineación horizontal para el texto.

```csharp
public override HorizontalAlignment HorizontalAlignment { get; set; }
```

## Observaciones

HorizontalAlignment.None es igual a HorizontalAlignment.Left. Tenga en cuenta que la propiedad TextFragmentState.VerticalAlignment funciona solo en escenarios de generación de nuevos documentos.
