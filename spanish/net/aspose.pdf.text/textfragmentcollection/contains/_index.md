## Método TextFragmentCollection.Contains

Determina si la colección contiene un valor específico.

```csharp
public bool Contains(TextFragment item)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| item | TextFragment | El objeto a localizar en la colección |

### Valor de Retorno

true si el item se encuentra en la colección; de lo contrario, false.
