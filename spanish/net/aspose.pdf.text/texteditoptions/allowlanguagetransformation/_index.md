## Propiedad TextEditOptions.AllowLanguageTransformation

Obtiene o establece un valor que permite el uso de transformación de lenguaje durante la adición o edición de texto. true - se aplicará la transformación de lenguaje si es necesario (valor predeterminado). false - NO se aplicará la transformación de lenguaje.

```csharp
public bool AllowLanguageTransformation { get; set; }
```
