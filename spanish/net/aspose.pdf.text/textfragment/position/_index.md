## TextFragment.Position property

Obtiene o establece la posición del texto para el texto, representado con[`TextFragment`](../../textfragment) objeto.

```csharp
public Position Position { get; set; }
```

### Ejemplos

El ejemplo demuestra cómo ver la ubicación de un texto, representado por[`TextFragment`](../../textfragment) objeto.

```csharp
// Abrir documento
Document doc = new Document(@"D:\Tests\input.pdf");

// Crear objeto TextFragmentAbsorber para encontrar todas las apariciones de texto "hola mundo"
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");

// Aceptar el absorbedor para la primera página
doc.Pages[1].Accept(absorber);

// Ver texto e información de ubicación de la primera aparición de texto
TextFragment firstOccurrence = absorber.TextFragments[1];

Console.Out.WriteLine(string.Format("fragment text: {0}", firstOccurrence.Text));
Console.Out.WriteLine(string.Format("fragment X indent: {0}", firstOccurrence.Position.XIndent));
Console.Out.WriteLine(string.Format("fragment Y indent: {0}", firstOccurrence.Position.YIndent));

```
