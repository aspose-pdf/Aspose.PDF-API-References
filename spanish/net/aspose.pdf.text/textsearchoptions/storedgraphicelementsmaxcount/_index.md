## TextSearchOptions.StoredGraphicElementsMaxCount property

Obtiene o establece el valor que limita la búsqueda de gráficos relacionados con el texto (subrayado, fondo, etc.) en una página para el número de elementos especificado. El valor predeterminado es 250. Establezca un valor menor en el caso de problemas de rendimiento, pruebe un valor mayor en caso de que no se encontraran algunos elementos gráficos.

public int StoredGraphicElementsMaxCount { get; set; }
