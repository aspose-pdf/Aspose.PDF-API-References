## Propiedad TextSearchOptions.StoredGraphicElementsMaxCount

Obtiene o establece un valor que limita la búsqueda de gráficos relacionados con el texto (subrayado, fondo, etc.) en una página para el número especificado de elementos. El valor predeterminado es 250. Establezca un valor menor en caso de problemas de rendimiento, intente un valor mayor en caso de que no se encuentren algunos elementos gráficos.

```csharp
public int StoredGraphicElementsMaxCount { get; set; }
```
