## Propiedad TextSearchOptions.IgnoreShadowText

Obtiene o establece la indicación de que los fragmentos de texto que representan la sombra del texto normal serán ignorados durante la búsqueda. true - significa que el texto sombra no será encontrado (prueba esto si la búsqueda de texto devuelve fragmentos duplicados en posiciones cercanas) false - significa que el texto sombra será encontrado así como el texto normal (valor predeterminado)

```csharp
public bool IgnoreShadowText { get; set; }
```
