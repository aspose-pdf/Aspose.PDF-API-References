## TextSearchOptions.IgnoreShadowText property

Obtiene o establece una indicación de que los fragmentos de texto que representan la sombra del texto normal se ignorarán durante la búsqueda. verdadero: significa que no se encontrará el texto de la sombra (pruebe esto si la búsqueda de texto arroja fragmentos duplicados en las posiciones cercanas) falso: significa que la sombra se encontrará texto así como texto normal (valor predeterminado)

public bool IgnoreShadowText { get; set; }
