## TextSearchOptions.SearchForTextRelatedGraphics property

Obtiene o establece el valor que permite buscar gráficos relacionados con el texto (subrayado, fondo, etc.) durante la búsqueda de texto. verdadero: se realizará la búsqueda de gráficos relacionados con el texto (valor predeterminado). falso: elementos gráficos que pueden estar presentes en el documento de origen será ignorado. Configure esto en caso de problemas de rendimiento o si no necesita manejar el subrayado, el fondo o el recorte.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
