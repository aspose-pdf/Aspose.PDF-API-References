## Propiedad TextSearchOptions.SearchForTextRelatedGraphics

Obtiene o establece un valor que permite buscar gráficos relacionados con el texto (subrayado, fondo, etc.) durante la búsqueda de texto. true - se realizará la búsqueda de gráficos relacionados con el texto (valor predeterminado). false - se ignorarán los elementos gráficos que puedan estar presentes en el documento fuente. Establezca esto en caso de problemas de rendimiento o si no es necesario manejar el subrayado, el fondo o el recorte.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
