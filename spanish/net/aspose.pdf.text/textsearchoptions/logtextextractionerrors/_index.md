## Propiedad TextSearchOptions.LogTextExtractionErrors

Obtiene o establece la indicación de que los errores de extracción (decodificación) de texto se registrarán en el absorbedor de texto (fragmento). true - significa que los errores de extracción (decodificación) de texto se registrarán. Puede disminuir el rendimiento. false (por defecto) - no hay registro de errores.

```csharp
public bool LogTextExtractionErrors { get; set; }
```
