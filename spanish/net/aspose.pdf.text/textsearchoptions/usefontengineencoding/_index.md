## Propiedad TextSearchOptions.UseFontEngineEncoding

Obtiene o establece la indicación de que el texto se buscará utilizando la codificación del motor de fuentes. true - significa que se utilizará la codificación del motor de fuentes (prueba esto si la búsqueda de texto falla debido a una codificación imperfecta en el documento) false - significa que se utilizará la codificación de fuentes del documento (valor predeterminado)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
