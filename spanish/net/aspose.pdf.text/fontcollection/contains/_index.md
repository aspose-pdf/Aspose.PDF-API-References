## Contains(string) {#contains_1}

Verifica si la fuente existe en la colección de fuentes.

```csharp
public bool Contains(string name)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| name | String | Nombre de la fuente. |

### Valor de Retorno

Verdadero en caso de que la colección contenga la fuente con el nombre especificado.

## Contains(Font) {#contains}

Determina si la colección contiene un valor específico.

```csharp
public bool Contains(Font item)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| item | Font | El objeto a localizar en la colección |

### Valor de Retorno

verdadero si el elemento se encuentra en la colección; de lo contrario, falso.
