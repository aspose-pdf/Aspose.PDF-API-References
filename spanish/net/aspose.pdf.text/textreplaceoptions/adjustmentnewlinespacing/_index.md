## TextReplaceOptions.AdjustmentNewLineSpacing property

Obtiene o establece el valor del espacio entre líneas que se usa si se fuerza el ajuste de reemplazo para crear una nueva línea de texto. El valor esperado es el multiplicador del tamaño de fuente del texto reemplazado. El valor predeterminado es 1.2.
