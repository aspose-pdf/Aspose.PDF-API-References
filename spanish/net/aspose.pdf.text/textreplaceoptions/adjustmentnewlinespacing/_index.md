## Propiedad TextReplaceOptions.AdjustmentNewLineSpacing

Obtiene o establece el valor del espaciado de líneas que se utiliza si el ajuste de reemplazo se ve obligado a crear una nueva línea de texto. El valor esperado es un multiplicador del tamaño de fuente del texto reemplazado. El valor predeterminado es 1.2.

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
