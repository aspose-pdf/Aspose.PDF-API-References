## Propiedad MarkupParagraph.ContinuationPageNumbers

Lista de números de página en los que se continúa el párrafo. Coincidirá con la página donde comenzó el párrafo si se continúa en la siguiente columna en la misma página.

public List<int> ContinuationPageNumbers { get; }
