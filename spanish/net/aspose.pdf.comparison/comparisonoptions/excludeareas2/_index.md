## Propiedad ComparisonOptions.ExcludeAreas2

Obtenga y establezca las áreas excluidas. Utilizado para la segunda página o documento en el método de comparación. Esta opción se puede establecer junto con [`ExcludeTables`](../excludetables/). Esta opción no se puede establecer junto con la opción [`ExtractionArea`](../extractionarea/).

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
