## Propiedad ComparisonOptions.ExtractionArea

Obtenga y establezca el área rectangular en la que se comparará el texto de las páginas. Esta opción no se puede establecer junto con las opciones [`ExcludeTables`](../excludetables/), [`ExcludeAreas1`](../excludeareas1/) y [`ExcludeAreas2`](../excludeareas2/).

```csharp
public Rectangle ExtractionArea { get; set; }
```
