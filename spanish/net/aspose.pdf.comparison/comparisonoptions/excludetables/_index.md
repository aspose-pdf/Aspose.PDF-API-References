## Propiedad ComparisonOptions.ExcludeTables

Obtenga y establezca la opción que determina si las tablas están excluidas de la comparación. Esta opción no se puede establecer junto con la opción [`ExtractionArea`](../extractionarea/). El valor predeterminado es `false`.

```csharp
public bool ExcludeTables { get; set; }
```
