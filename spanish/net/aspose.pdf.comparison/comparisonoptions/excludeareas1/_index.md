## Propiedad ComparisonOptions.ExcludeAreas1

Obtenga y establezca las áreas excluidas. Utilizado para la primera página o documento en el método de comparación. Esta opción se puede establecer junto con [`ExcludeTables`](../excludetables/). Esta opción no se puede establecer junto con la opción [`ExtractionArea`](../extractionarea/).

public Rectangle[] ExcludeAreas1 { get; set; }
