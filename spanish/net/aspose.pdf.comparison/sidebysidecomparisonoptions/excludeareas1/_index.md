## Propiedad SideBySideComparisonOptions.ExcludeAreas1

Obtener y establecer las áreas excluidas. Utilizado para la primera página o documento en el método de comparación. Esta opción se puede establecer junto con [`ExcludeTables`](../excludetables/). Esta opción no se puede establecer junto con la opción [`ComparisonArea1`](../comparisonarea1/).

public Rectangle[] ExcludeAreas1 { get; set; }
