## Propiedad SideBySideComparisonOptions.ExcludeAreas2

Obtener y establecer las áreas excluidas. Utilizado para la segunda página o documento en el método de comparación. Esta opción se puede establecer junto con [`ExcludeTables`](../excludetables/). Esta opción no se puede establecer junto con la opción [`ComparisonArea2`](../comparisonarea2/).

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
