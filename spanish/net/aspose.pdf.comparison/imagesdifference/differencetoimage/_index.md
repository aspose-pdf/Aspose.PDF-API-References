## Método ImagesDifference.DifferenceToImage

Convierte el array de diferencias en una imagen de mapa de bits utilizando los colores especificados.

```csharp
public Bitmap DifferenceToImage(Color color, Color backgroundColor)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| color | Color | El color para diferencias no nulas. |
| backgroundColor | Color | El color de fondo para diferencias nulas. |

### Valor de Retorno

Una imagen de mapa de bits que representa el array de diferencias.
