## Campo PaperSizes.A4Extra

Papel A4 extra (236 mm por 322 mm).

```csharp
public static readonly PaperSize A4Extra;
```

## Observaciones

Este valor es específico para el controlador PostScript y se utiliza únicamente por impresoras Linotronic para ayudar a ahorrar papel.
