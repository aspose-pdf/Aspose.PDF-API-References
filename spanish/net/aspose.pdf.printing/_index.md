El **Aspose.Pdf.Printing** espacio de nombres proporciona clases y funcionalidades para manejar operaciones de impresión de PDF, incluyendo configuraciones de impresora, configuraciones de página y extensiones para gestionar configuraciones relacionadas con la impresión.

## Clases

| Clase | Descripción |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Proporciona datos para el evento [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Especifica configuraciones que se aplican a una sola página impresa. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Representa métodos de extensión para [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Especifica el tamaño de una hoja de papel. |
| [PaperSizeExtensions](./papersizeextensions/) | Representa métodos de extensión para [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Representa los tamaños de papel estándar. |
| [PaperSource](./papersource/) | Especifica la bandeja de papel de la cual la impresora obtiene papel. |
| [PaperSourceExtensions](./papersourceextensions/) | Representa métodos de extensión para [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Proporciona un conjunto de instancias predefinidas de [`PaperSource`](../aspose.pdf.printing/papersource/) que representan fuentes de papel comunes. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Proporciona datos para el evento [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Representa la resolución soportada por una impresora. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Representa métodos de extensión para [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Especifica información sobre cómo se imprime un documento, incluyendo la impresora que lo imprime. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Representa métodos de extensión para [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Proporciona datos para los eventos [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) y [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) de la clase [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
