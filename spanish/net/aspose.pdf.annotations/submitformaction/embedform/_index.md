## Campo SubmitFormAction.EmbedForm

Si se establece, la entrada F del FDF enviado será una especificación de archivo que contiene un flujo de archivo incrustado que representa el archivo PDF desde el cual se está enviando el FDF.

```csharp
public const int EmbedForm;
```
