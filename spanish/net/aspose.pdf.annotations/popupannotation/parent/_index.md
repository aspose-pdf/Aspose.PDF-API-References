## Propiedad PopupAnnotation.Parent

Obtiene o establece la anotación padre con la que se asociará esta anotación emergente. Si esta entrada está presente, las entradas de Contenidos, M, C y T de la anotación padre anularán las de la anotación emergente misma.

```csharp
public Annotation Parent { get; set; }
```
