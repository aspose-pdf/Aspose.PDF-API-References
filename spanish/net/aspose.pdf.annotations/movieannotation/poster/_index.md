## Propiedad MovieAnnotation.Poster

Obtiene o establece una bandera o flujo que especifica si y cómo se debe mostrar una imagen de cartel que represente la película. Si es verdadero, la imagen del cartel se recuperará del archivo de la película; si es falso, no se mostrará ningún cartel.

```csharp
public bool Poster { get; set; }
```
