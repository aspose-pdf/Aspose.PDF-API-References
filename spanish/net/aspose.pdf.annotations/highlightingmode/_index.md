## HighlightingMode enumeration

Enumera el modo de resaltado de la anotación, el efecto visual que se utilizará cuando se presione o mantenga presionado el botón del mouse dentro de su área activa.

```csharp
public enum HighlightingMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Sin resaltar. |
| Invert | `1` | Invierte el contenido del rectángulo de anotación. |
| Outline | `2` | Invierte el borde de la anotación. |
| Push | `3` | Muestra la apariencia hacia abajo de la anotación, si la hay. Si no se define una apariencia hacia abajo, desplace el contenido del rectángulo de anotación para que aparezca como si estuviera siendo empujado debajo de la superficie de la página. |
| Toggle | `4` | Igual que Push (que es el preferido). |
