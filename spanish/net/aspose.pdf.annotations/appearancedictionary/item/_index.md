## Indexador AppearanceDictionary

Representa una forma conveniente de obtener flujos de apariencia.

```csharp
public XForm this[string key] { get; set; }
```

| Parámetro | Descripción |
| --- | --- |
| key | Representa la ruta al flujo de apariencia. Si el diccionario de apariencia tiene subdiccionarios, entonces la ruta debe contener 2 partes ([`Keys`](../keys/)), de lo contrario, la ruta tiene solo una parte. |

### Valor de Retorno

Objeto XForm (flujo de apariencia) que corresponde a la clave dada.
