## Propiedad AppearanceDictionary.Keys

Obtiene las claves del diccionario. Si el diccionario de apariencia tiene subdiccionarios, entonces `Keys` contiene valores (N&#x7C;R&#x7C;D).state, donde N - apariencia normal, R - apariencia de rollover, D - apariencia de presionado y state - el nombre del estado (por ejemplo, On, Off para casillas de verificación).

public ICollection<string> Keys { get; }
