## Enumeración PrinterMarksKind

Especifica los tipos de marcas de impresora que se agregarán a un documento.

```csharp
[Flags]
public enum PrinterMarksKind
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Ninguno | `0` | Especifica que no se agregarán marcas de impresora. |
| MarcasDeRecorte | `1` | Especifica que se agregarán marcas de recorte. |
| MarcasDeSangrado | `2` | Especifica que se agregarán marcas de sangrado. |
| MarcasDeRegistro | `4` | Especifica que se agregarán marcas de registro. |
| BarrasDeColor | `8` | Especifica que se agregarán barras de color. |
| InformaciónDePágina | `10` | Especifica que se agregará información de página. |
| Todo | `1F` | Especifica que se agregarán todos los tipos de marcas de impresora. |

## Observaciones

Esta enumeración tiene un atributo FlagsAttribute que permite una combinación bit a bit de sus valores de miembro.
