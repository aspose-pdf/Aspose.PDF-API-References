## Indexador PDF3DViewArray

Obtiene o establece el [`PDF3DView`](../../pdf3dview/) para ver el array en el índice especificado.

```csharp
public PDF3DView this[int index] { get; set; }
```

| Parámetro | Descripción |
| --- | --- |
| index | El índice. |

### Valor de Retorno

PDF3DView.

### Excepciones

| excepción | condición |
| --- | --- |
| IndexOutOfRangeException | Índice no válido: el índice debe estar en el rango [1..n] donde n es igual al conteo de vistas. |
