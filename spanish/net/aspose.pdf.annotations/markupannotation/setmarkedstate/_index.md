## Método MarkupAnnotation.SetMarkedState

Establece el estado Marcado y No Marcado para la anotación. Tenga en cuenta que el estado se almacena en otra anotación de texto que tiene claves state y statemodel.

```csharp
public void SetMarkedState(bool marked)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| marked | Boolean | Verdadero si establece el estado Marcado, y falso si establece el estado No Marcado. |
