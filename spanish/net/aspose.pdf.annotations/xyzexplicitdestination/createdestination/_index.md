## Método XYZExplicitDestination.CreateDestination

Crear destino en la ubicación especificada de la página considerando la rotación de la página si es necesario.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| page | Page | Página de destino. |
| left | Double | Posición izquierda en la página. |
| top | Double | Posición superior en la página. |
| zoom | Double | Factor de zoom (0 para predeterminado). |
| considerRotation | Boolean | Si es verdadero, la posición se recalculará de acuerdo con la rotación de la página. |

### Valor de Retorno

Objeto de destino.
