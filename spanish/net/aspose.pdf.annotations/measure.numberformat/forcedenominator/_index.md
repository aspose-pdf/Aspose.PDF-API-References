## Propiedad Measure.NumberFormat.ForceDenominator

Si FractionDisplayment es ShowAsFraction, este valor determina si la fracción puede o no ser reducida. Si el valor es verdadero, la fracción no puede ser reducida.

public bool ForceDenominator { get; set; }
