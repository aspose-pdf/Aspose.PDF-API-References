## Propiedad Annotation.UpdateAppearanceOnConvert

Si es verdadero, la apariencia de la anotación se actualizará antes de convertir el documento PF en imagen. Esto permite convertir campos correctamente, pero probablemente demande más tiempo.

public static bool UpdateAppearanceOnConvert { get; set; }
