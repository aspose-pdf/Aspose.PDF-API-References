## Propiedad SvgExtractionOptions.StrictExtractionAreaBoundCheck

Obtiene y establece una opción para definir estrictamente si los subcampos están dentro del rectángulo especificado en [`ExtractionAreaBound`](../extractionareabound/). Si se establece en falso, entonces los subcampos que no están completamente incluidos en [`ExtractionAreaBound`](../extractionareabound/) serán extraídos. El valor predeterminado es `True`.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
