## Propiedad SvgExtractionOptions.MinStrokeWidth

Obtiene o establece el ancho de trazo mínimo que se utilizará en el SVG resultante. Si el PDF utiliza un ancho de trazo más delgado, se reemplazará con este ancho. El valor predeterminado es 0.5.

```csharp
public double MinStrokeWidth { get; set; }
```

## Observaciones

El valor se expresa en unidades de espacio de usuario transformadas de la página PDF convertida. Por defecto, 1 unidad de espacio de usuario es 1/72 de pulgada (0.35 mm), pero esto puede ser anulado por el documento PDF. Las transformaciones pueden afectar el ancho mínimo real en el SVG generado.
