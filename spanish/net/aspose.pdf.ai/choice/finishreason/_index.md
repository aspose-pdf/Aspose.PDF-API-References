## Propiedad Choice.FinishReason

Obtiene o establece la razón por la cual el modelo dejó de generar tokens. Esto se detendrá si el modelo alcanza un punto de parada natural o una secuencia de parada proporcionada, longitud si se alcanzó el número máximo de tokens especificado en la solicitud.

```csharp
public string FinishReason { get; set; }
```
