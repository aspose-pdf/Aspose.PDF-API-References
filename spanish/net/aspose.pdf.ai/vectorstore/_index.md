## Clase VectorStore

Un ayudante para crear un almacén de vectores con file_ids y adjuntarlo a este hilo. Puede haber un máximo de 1 almacén de vectores adjunto al hilo.

```csharp
public class VectorStore
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [FileIds](../../aspose.pdf.ai/vectorstore/fileids/) { get; set; } | Obtiene o establece una lista de IDs de archivos para agregar al almacén de vectores. Puede haber un máximo de 10000 archivos en un almacén de vectores. |
| [Metadata](../../aspose.pdf.ai/vectorstore/metadata/) { get; set; } | Obtiene o establece un conjunto de 16 pares clave-valor que se pueden adjuntar a un almacén de vectores. Esto puede ser útil para almacenar información adicional sobre el almacén de vectores en un formato estructurado. Las claves pueden tener un máximo de 64 caracteres de longitud y los valores pueden tener un máximo de 512 caracteres de longitud. |
