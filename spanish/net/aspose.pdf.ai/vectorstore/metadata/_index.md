## Propiedad VectorStore.Metadata

Obtiene o establece un conjunto de 16 pares clave-valor que se pueden adjuntar a un almacén de vectores. Esto puede ser útil para almacenar información adicional sobre el almacén de vectores en un formato estructurado. Las claves pueden tener un máximo de 64 caracteres y los valores pueden tener un máximo de 512 caracteres.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
