## Propiedad RunCreateRequest.MaxCompletionTokens

Obtiene o establece el número máximo de tokens de finalización que se pueden usar durante el transcurso de la ejecución. La ejecución hará un esfuerzo por usar solo el número de tokens de finalización especificado, a lo largo de múltiples turnos de la ejecución. Si la ejecución excede el número de tokens de finalización especificados, la ejecución terminará con el estado incompleto. Consulte incomplete_details para más información.

```csharp
public int? MaxCompletionTokens { get; set; }
```
