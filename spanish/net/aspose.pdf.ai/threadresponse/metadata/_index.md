## Propiedad ThreadResponse.Metadata

Obtiene o establece un conjunto de 16 pares clave-valor que se pueden adjuntar a un objeto. Esto puede ser útil para almacenar información adicional sobre el objeto en un formato estructurado. Las claves pueden tener un máximo de 64 caracteres de longitud y los valores pueden tener un máximo de 512 caracteres de longitud.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
