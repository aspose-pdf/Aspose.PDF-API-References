## Propiedad AssistantResponse.ToolResources

Obtiene o establece un conjunto de recursos que son utilizados por las herramientas del asistente. Los recursos son específicos del tipo de herramienta. Por ejemplo, la herramienta code_interpreter requiere una lista de IDs de archivos, mientras que la herramienta file_search requiere una lista de IDs de almacenes de vectores.

```csharp
public ToolResources ToolResources { get; set; }
```
