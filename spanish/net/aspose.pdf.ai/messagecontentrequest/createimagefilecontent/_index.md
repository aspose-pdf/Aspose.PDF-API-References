## Método MessageContentRequest.CreateImageFileContent

Crea un contenido de archivo de imagen para un mensaje.

```csharp
public static MessageContentRequest CreateImageFileContent(string fileId, string detail = "auto")
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fileId | String | La ID del archivo de imagen. |
| detail | String | El detalle del archivo de imagen. Especifica el nivel de detalle de la imagen si lo especifica el usuario. "low" utiliza menos tokens, puede optar por alta resolución usando "high". El valor predeterminado es "auto". |

### Valor de Retorno

Una nueva instancia de MessageContentRequest con contenido de ImageFile.
