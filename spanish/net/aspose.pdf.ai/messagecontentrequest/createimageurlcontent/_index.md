## Método MessageContentRequest.CreateImageUrlContent

Crea un contenido de URL de imagen para un mensaje.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| imageUrl | String | La URL de la imagen. |
| detail | String | El detalle de la URL de la imagen. Especifica el nivel de detalle de la imagen si lo especifica el usuario. "low" utiliza menos tokens, puedes optar por alta resolución usando "high". El valor predeterminado es "auto". |

### Valor de Retorno

Una nueva instancia de MessageContentRequest con contenido de ImageUrl.
