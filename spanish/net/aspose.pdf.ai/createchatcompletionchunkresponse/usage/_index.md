## Propiedad CreateChatCompletionChunkResponse.Usage

Obtiene o establece un campo opcional que solo estará presente cuando establezca stream_options: {"include_usage": true} en su solicitud. Cuando está presente, contiene un valor nulo, excepto para el último fragmento que contiene las estadísticas de uso de tokens para toda la solicitud.

```csharp
public Usage Usage { get; set; }
```
