## Propiedad CompletionCreateRequest.Seed

Obtiene o establece el valor de Seed. Esta función está en Beta. Si se especifica, nuestro sistema hará un esfuerzo por muestrear de manera determinista, de modo que las solicitudes repetidas con la misma semilla y parámetros deberían devolver el mismo resultado. No se garantiza la determinación y debe consultar el parámetro de respuesta system_fingerprint para monitorear cambios en el backend.

```csharp
public int? Seed { get; set; }
```
