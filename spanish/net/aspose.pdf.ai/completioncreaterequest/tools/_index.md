## Propiedad CompletionCreateRequest.Tools

Obtiene o establece una lista de herramientas que el modelo puede llamar. Actualmente, solo se admiten funciones como herramienta. Utilice esto para proporcionar una lista de funciones para las que el modelo puede generar entradas JSON. Se admiten un máximo de 128 funciones.

```csharp
public List<Tool> Tools { get; set; }
```
