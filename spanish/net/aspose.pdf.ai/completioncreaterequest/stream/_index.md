## Propiedad CompletionCreateRequest.Stream

Obtiene o establece si se debe usar streaming. Si se establece, se enviarán deltas de mensajes parciales, como en ChatGPT. Los tokens se enviarán como eventos enviados por el servidor solo de datos a medida que estén disponibles, con el stream terminado por un mensaje de datos: [DONE].

```csharp
public bool? Stream { get; set; }
```
