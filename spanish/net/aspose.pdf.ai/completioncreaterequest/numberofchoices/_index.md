## Propiedad CompletionCreateRequest.NumberOfChoices

Obtiene o establece cuántas opciones de finalización de chat generar para cada mensaje de entrada. Tenga en cuenta que se le cobrará según el número de tokens generados en todas las opciones. Mantenga n como 1 para minimizar costos.

public int? NumberOfChoices { get; set; }
