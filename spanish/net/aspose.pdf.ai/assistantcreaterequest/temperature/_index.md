## Propiedad AssistantCreateRequest.Temperature

Obtiene o establece la temperatura de muestreo a utilizar, entre 0 y 2. Valores más altos como 0.8 harán que la salida sea más aleatoria, mientras que valores más bajos como 0.2 la harán más enfocada y determinista.

```csharp
public double? Temperature { get; set; }
```
