## Propiedad AssistantCreateRequest.Tools

Obtiene o establece una lista de herramientas habilitadas en el asistente. Puede haber un máximo de 128 herramientas por asistente. Las herramientas pueden ser de tipos code_interpreter, file_search o function.

public List<Tool> Tools { get; set; }
