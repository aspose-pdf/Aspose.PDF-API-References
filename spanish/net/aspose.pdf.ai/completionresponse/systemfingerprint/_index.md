## Propiedad CompletionResponse.SystemFingerprint

Obtiene o establece la huella digital que representa la configuración del backend con la que se ejecuta el modelo. Se puede utilizar junto con el parámetro de solicitud seed para entender cuándo se han realizado cambios en el backend que podrían afectar el determinismo.

```csharp
public string SystemFingerprint { get; set; }
```
