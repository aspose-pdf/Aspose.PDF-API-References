## Propiedad OpenAIImageDescriptionCopilotOptions.ImageDetail

Obtiene o establece el nivel de detalle de la imagen si lo especifica el usuario. "low" utiliza menos tokens, puedes optar por alta resolución usando "high". Si no se establece, por defecto es "auto".

public string ImageDetail { get; set; }
