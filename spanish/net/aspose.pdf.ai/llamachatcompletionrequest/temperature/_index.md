## Propiedad LlamaChatCompletionRequest.Temperature

Establece o obtiene la temperatura de muestreo a utilizar, entre 0 y 2. Valores más altos como 0.8 harán que la salida sea más aleatoria, mientras que valores más bajos como 0.2 la harán más enfocada y determinista. El valor predeterminado es 1.

```csharp
public double? Temperature { get; set; }
```
