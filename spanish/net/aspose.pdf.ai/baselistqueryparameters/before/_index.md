## Propiedad BaseListQueryParameters.Before

Obtiene o establece un cursor para su uso en la paginación. before es un ID de objeto que define su lugar en la lista. Por ejemplo, si realiza una solicitud de lista y recibe 100 objetos, que terminan con obj_foo, su llamada subsiguiente puede incluir before=obj_foo para obtener la página anterior de la lista.

```csharp
public string Before { get; set; }
```
