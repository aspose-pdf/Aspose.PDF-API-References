## Propiedad RunThreadCreateRequest.MaxPromptTokens

Obtiene o establece el número máximo de tokens de aviso que se pueden usar durante el transcurso de la ejecución. La ejecución hará un esfuerzo por usar solo el número de tokens de aviso especificados, a lo largo de múltiples turnos de la ejecución. Si la ejecución excede el número de tokens de aviso especificados, la ejecución terminará con estado incompleto. Consulte incomplete_details para más información.

```csharp
public int? MaxPromptTokens { get; set; }
```
