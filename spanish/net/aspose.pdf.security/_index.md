El **Aspose.Pdf.Security** espacio de nombres contiene clases utilizadas para la encriptación y la firma digital.

## Clases

| Clase | Descripción |
| --- | --- |
| [DsaAlgorithmInfo](./dsaalgorithminfo/) | Representa una clase para la información sobre el algoritmo de firma DSA. |
| [EcdsaAlgorithmInfo](./ecdsaalgorithminfo/) | Representa una clase para la información sobre el algoritmo de firma ECDSA. |
| [KeyedSignatureAlgorithmInfo](./keyedsignaturealgorithminfo/) | Representa una clase para la información sobre un algoritmo de firma con clave. |
| [RsaAlgorithmInfo](./rsaalgorithminfo/) | Representa una clase para la información sobre el algoritmo de firma RSA. |
| [SignatureAlgorithmInfo](./signaturealgorithminfo/) | Representa una clase para la información sobre un algoritmo de firma, incluyendo su tipo, estándar criptográfico y algoritmo de hash de resumen. |
| [SignatureLengthMismatchException](./signaturelengthmismatchexception/) | Representa errores que ocurren durante la firma de PDF. Ocurre si [`SignHash`](../aspose.pdf.forms/signhash/) se utiliza para firmar un documento y la longitud real de la firma es mayor que la especificada en la opción [`DefaultSignatureLength`](../aspose.pdf.forms/signature/defaultsignaturelength/). |
| [TimestampAlgorithmInfo](./timestampalgorithminfo/) | Representa una clase para la información sobre el algoritmo de firma de marca de tiempo. |
| [UnknownSignatureAlgorithmInfo](./unknownsignaturealgorithminfo/) | Representa una clase para la información sobre el algoritmo de firma desconocido. |
| [ValidationOptions](./validationoptions/) | Representa opciones para validar una firma digital en un documento PDF. |
| [ValidationResult](./validationresult/) | Representa el resultado de un proceso de validación para un certificado. |
