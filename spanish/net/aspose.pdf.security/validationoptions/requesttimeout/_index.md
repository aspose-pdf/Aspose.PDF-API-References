## Propiedad ValidationOptions.RequestTimeout

Obtiene o establece la duración del tiempo de espera, en milisegundos, para operaciones relacionadas con la red durante el proceso de validación. La propiedad RequestTimeout define el tiempo máximo que el sistema debe esperar por una respuesta de red al acceder a recursos en línea, como el estado de revocación o servidores OCSP.

```csharp
public int RequestTimeout { get; set; }
```
