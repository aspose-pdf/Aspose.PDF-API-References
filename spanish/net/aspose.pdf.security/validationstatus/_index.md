## Enumeración ValidationStatus

Representa el estado de validación de una validación de certificado.

```csharp
public enum ValidationStatus
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Válido | `0` | Indica que el certificado ha sido validado con éxito. |
| Inválido | `1` | Indica que la validación del certificado falló. |
| Indefinido | `2` | Indica que el proceso de validación fue inconcluso o no se realizó. |

## Observaciones

Esta enumeración define los posibles resultados de la validación de certificados: - Válido: Indica que el certificado ha sido validado con éxito. - Inválido: Indica que la validación del certificado falló. - Indefinido: Indica que el proceso de validación fue inconcluso o no se realizó.
