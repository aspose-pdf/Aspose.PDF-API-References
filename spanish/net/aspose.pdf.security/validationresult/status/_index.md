## Propiedad ValidationResult.Status

Obtiene el estado del proceso de validación de un certificado.

```csharp
public ValidationStatus Status { get; }
```

## Observaciones

La propiedad Status indica el resultado de la validación del certificado. Los valores posibles están definidos en la enumeración [`ValidationStatus`](../../validationstatus/), como Válido, Inválido o Indefinido. Proporciona una visión sobre si el certificado pasó las verificaciones de validación o no.
