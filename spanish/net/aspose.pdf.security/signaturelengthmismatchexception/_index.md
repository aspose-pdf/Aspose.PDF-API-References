## Clase SignatureLengthMismatchException

Representa errores que ocurren durante la firma de PDF. Ocurre si se utiliza [`SignHash`](../../aspose.pdf.forms/signhash/) para firmar un documento y la longitud real de la firma es mayor que la especificada en la opción [`DefaultSignatureLength`](../../aspose.pdf.forms/signature/defaultsignaturelength/).

```csharp
public class SignatureLengthMismatchException : PdfException
```
