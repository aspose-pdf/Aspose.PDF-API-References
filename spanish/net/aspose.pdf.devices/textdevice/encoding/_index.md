## TextDevice.Encoding property

Obtiene o establece la codificación del texto extraído.

```csharp
public Encoding Encoding { get; set; }
```

### Ejemplos

El ejemplo demuestra cómo representar texto extraído en codificación UTF-8.

```csharp
Document doc = new Document(inFile);
string extractedText;

// crear dispositivo de texto
TextDevice device = new TextDevice(Encoding.UTF8);

// convertir la página y guardar el texto en la secuencia
device.Process(doc.Pages[1], outFile);

// usa el texto extraído
extractedText = File.ReadAllText(outFile, Encoding.UTF8);
```
