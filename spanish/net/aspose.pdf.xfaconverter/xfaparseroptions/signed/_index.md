## XfaParserOptions.Signed property

Si esta propiedad es verdadera, el documento se convertirá con el flujo de formulario xfa (si existe). Si es falso, se ignorará el flujo de formulario xfa. Esta propiedad se introdujo porque no está claro cómo calcular la suma de verificación que se utiliza para comprobar la firma.
