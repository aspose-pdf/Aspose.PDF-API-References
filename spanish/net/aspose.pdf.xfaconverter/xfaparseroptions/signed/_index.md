## Propiedad XfaParserOptions.Signed

Si esta propiedad es verdadera, entonces el documento se convertirá utilizando el flujo de formulario xfa (si existe). Si es falsa, entonces el flujo de formulario xfa será ignorado. Esta propiedad se introdujo porque no está claro cómo calcular la suma de verificación que se utiliza para verificar la firma.

```csharp
public bool Signed { get; set; }
```
