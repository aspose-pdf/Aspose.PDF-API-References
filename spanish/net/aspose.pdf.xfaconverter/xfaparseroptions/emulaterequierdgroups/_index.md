## Propiedad XfaParserOptions.EmulateRequierdGroups

Si esta propiedad es verdadera, se dibujarán rectángulos rojos adicionales para los "grupos excluidos" requeridos de Xfa. Esta propiedad se introdujo debido a la ausencia de análogos de grupos excluidos durante la conversión de la representación Xfa de formularios a estándar. Es falsa por defecto.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
