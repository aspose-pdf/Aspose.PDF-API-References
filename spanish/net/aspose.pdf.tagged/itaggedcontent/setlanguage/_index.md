## ITaggedContent.SetLanguage method

Establece el lenguaje natural para el documento pdf.

Un identificador de idioma que especificará el idioma natural de todo el texto del documento, excepto cuando lo anulen las especificaciones de idioma para elementos de estructura o contenido marcado.

```csharp
public void SetLanguage(string lang)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| lang | String | Un identificador de idioma será la cadena de texto vacía, para indicar que el idioma es desconocido, o una etiqueta de idioma como se define en RFC 3066, Etiquetas para la identificación de idiomas. |
