## Método ITaggedContent.SetLanguage

Establece el idioma natural para el documento pdf.

Un identificador de idioma que debe especificar el idioma natural para todo el texto en el documento, excepto donde se anule por especificaciones de idioma para elementos de estructura o contenido marcado.

```csharp
public void SetLanguage(string lang)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| lang | String | Un identificador de idioma debe ser la cadena de texto vacía, para indicar que el idioma es desconocido, o una Language-Tag como se define en RFC 3066, Tags for the Identification of Languages. |
