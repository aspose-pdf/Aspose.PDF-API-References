## Propiedad Form.SignaturesAppendOnly

Si se establece, el documento contiene firmas que pueden ser invalidadas si el archivo se guarda (escribe) de una manera que altera su contenido anterior, a diferencia de una actualización incremental.

```csharp
public bool SignaturesAppendOnly { get; set; }
```
