## Propiedad Form.EmulateRequierdGroups

Si esta propiedad es verdadera, se dibujarán rectángulos de límite rojos adicionales para los contenedores de elementos exclGroup requeridos de Xfa. Esta propiedad se introdujo debido a la ausencia de análogos para el exclGroup durante la conversión de la representación Xfa de formularios a estándar. Es falsa por defecto.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
