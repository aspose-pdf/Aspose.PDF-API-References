## Propiedad ChoiceField.SelectedItems

Obtiene o establece un array de elementos seleccionados. Para una lista de selección múltiple, el array contiene más de un elemento. Para una lista de selección única, contiene un solo elemento.

```csharp
public virtual int[] SelectedItems { get; set; }
```
