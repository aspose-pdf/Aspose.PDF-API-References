## Propiedad Field.IsSharedField

Propiedad para soporte de Generator. Se utiliza cuando el campo se agrega al encabezado o pie de página. Si es verdadero, este campo se creará una vez y su apariencia será visible en todas las páginas del documento. Si es falso, se creará un campo separado para cada página del documento.

```csharp
public bool IsSharedField { get; set; }
```
