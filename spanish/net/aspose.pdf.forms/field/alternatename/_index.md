## Propiedad Field.AlternateName

Obtiene o establece el nombre alternativo del campo (Un nombre alternativo del campo que se utilizará en lugar del nombre real del campo donde sea que el campo se identifique en la interfaz de usuario). El nombre alternativo se utiliza como tooltip del campo en Adobe Acrobat.

```csharp
public string AlternateName { get; set; }
```
