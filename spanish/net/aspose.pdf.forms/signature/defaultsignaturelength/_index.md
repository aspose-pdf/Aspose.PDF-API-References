## Propiedad Signature.DefaultSignatureLength

Obtiene o establece la longitud predeterminada para los datos de la firma en bytes.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Observaciones

Esta es una estimación de la longitud de la firma en bytes. Se utiliza para firmar a través de [`CustomSignHash`](../customsignhash/) si el parámetro [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) está establecido. El valor predeterminado es 3000.
