## Signature.ShowProperties property

Obligar a mostrar/ocultar las propiedades de la firma. En caso de que ShowProperties sea verdadero, el campo de la firma tiene un formato de apariencia predefinido (cadenas para representar): --------------------- ------------- Firmado digitalmente por {asunto del certificado} Fecha: {firma.Fecha} Motivo: {firma.Razón} Ubicación: { firma.Ubicación} --------------------------------------------x000d_ donde {X} es un marcador de posición para el valor X. También la firma puede tener una imagen, en este caso, las cadenas enumeradas se colocan sobre la imagen. ShowProperties es verdadero de forma predeterminada.

```csharp
public bool ShowProperties { get; set; }
```
