## Propiedad Form.FlattenSettings.UpdateAppearances

Si se establece, todas las apariencias de los campos se regenerarán antes de aplanar. Esta opción puede ayudar si el campo se aplanó incorrectamente. Esta opción puede disminuir el rendimiento. Por defecto se establece en falso.

```csharp
public bool UpdateAppearances { get; set; }
```
