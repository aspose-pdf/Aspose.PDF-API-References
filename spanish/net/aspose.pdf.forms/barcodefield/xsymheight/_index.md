## Propiedad BarcodeField.XSymHeight

Obtiene la distancia vertical entre dos módulos de código de barras, medida en píxeles. La relación XSymHeight/XSymWidth debe ser un valor entero. Para PDF417, el rango de relación aceptable es de 1 a 4. Para QRCode y DataMatrix, esta relación siempre debe ser 1

```csharp
public int XSymHeight { get; }
```
