## Enumeración de Simbología

Una Simbología (de Código de Barras) define los detalles técnicos de un tipo particular de código de barras: el ancho de las barras, conjunto de caracteres, método de codificación, especificaciones de suma de verificación, etc.

```csharp
public enum Symbology
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PDF417 | `0` | Una simbología 2D de múltiples filas y longitud variable con alta capacidad de datos y capacidad de corrección de errores. |
| QRCode | `1` | El código QR (abreviatura de Respuesta Rápida) es un código de barras de matriz específico (código 2D), legible por lectores de códigos de barras QR dedicados y teléfonos con cámara. |
| DataMatrix | `2` | Una simbología de matriz 2D que consiste en módulos cuadrados en blanco y negro dispuestos en un patrón cuadrado o rectangular. |
