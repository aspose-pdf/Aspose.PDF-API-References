## Indexador XFA

Obtiene o establece el valor del nodo de datos según *ruta*.

```csharp
public string this[string path] { get; set; }
```

| Parámetro | Descripción |
| --- | --- |
| ruta | Ruta del nodo de datos, por ejemplo, form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Asegúrese de incluir índices incluso si los datos contienen solo ocurrencias únicas de cada nodo, es decir, escriba node1[0].node2[0]... en lugar de node1.node2... |

### Valor de Retorno

Valor del nodo de datos.
