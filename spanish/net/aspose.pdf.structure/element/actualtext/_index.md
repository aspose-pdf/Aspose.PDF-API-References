## Propiedad Element.ActualText

(Opcional; PDF 1.4) Texto que es un reemplazo exacto para el elemento de estructura y sus hijos. Este texto de reemplazo (que debe aplicarse a la menor cantidad de contenido posible) es útil al extraer el contenido del documento en apoyo a la accesibilidad para usuarios con discapacidades o para otros propósitos.

```csharp
public virtual string ActualText { get; set; }
```
