## Propiedad PdfToXlsOptions.MinimizeTheNumberOfWorksheets

Establezca verdadero si necesita minimizar el número de hojas de cálculo en el libro de trabajo resultante. El valor predeterminado es falso; significa guardar cada página PDF como una hoja de cálculo separada.

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
