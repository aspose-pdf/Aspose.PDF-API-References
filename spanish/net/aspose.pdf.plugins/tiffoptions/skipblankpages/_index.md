## Propiedad TiffOptions.SkipBlankPages

Obtiene o establece un valor que indica si se deben omitir las páginas en blanco.

```csharp
public bool SkipBlankPages { get; set; }
```

### Valor de la Propiedad

`true` si se necesita omitir las páginas en blanco; de lo contrario, `false`.

## Comentarios

El valor predeterminado es falso
