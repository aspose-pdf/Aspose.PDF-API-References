## Propiedad PdfAOptionsBase.UnicodeProcessingRules

Obtiene o establece las reglas para procesar las tablas CMap ToUnicode y no vinculadas a símbolos Unicode durante el proceso de conversión PDF/A.

```csharp
public ToUnicodeProcessingRules UnicodeProcessingRules { get; set; }
```

### Valor de la Propiedad

Una instancia de [`ToUnicodeProcessingRules`](../../../aspose.pdf/tounicodeprocessingrules/) que define las reglas para manejar los casos problemáticos con caracteres Unicode.
