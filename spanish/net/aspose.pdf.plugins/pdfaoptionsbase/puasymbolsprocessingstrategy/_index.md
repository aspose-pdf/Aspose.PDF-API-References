## Propiedad PdfAOptionsBase.PuaSymbolsProcessingStrategy

Obtiene o establece la estrategia para procesar símbolos del Área de Uso Privado (PUA) en el documento PDF.

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### Valor de la Propiedad

La estrategia para procesar símbolos PUA. El valor predeterminado es Ninguno para los documentos de conformidad de Nivel B, y SurroundPuaTextWithEmptyActualText para los de conformidad de Nivel A.
