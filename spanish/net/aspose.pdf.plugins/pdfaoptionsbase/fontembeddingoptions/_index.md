## Propiedad PdfAOptionsBase.FontEmbeddingOptions

Obtiene las opciones para procesar fuentes que no se pueden incrustar en el documento.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Valor de la Propiedad

Las opciones de incrustación de fuentes.

## Observaciones

El estándar PDF/A requiere que todas las fuentes deben estar incrustadas en el documento. Esta propiedad proporciona opciones para manejar casos en los que no es posible incrustar algunas fuentes porque están ausentes en la PC de destino.
