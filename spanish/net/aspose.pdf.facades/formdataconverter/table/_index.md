## Propiedad FormDataConverter.Table

Obtiene o establece el contenedor de datos intermedio, una DataTable. Debe definirse antes de convertir datos de un formato a otro. Las Columnas y el TableName de la DataTable deben definirse. El TableName es el nombre de la Tabla en la base de datos. El ColumnName de cada columna es el nombre de campo calificado del pdf. El Caption de cada columna es el nombre de la columna de la tabla en la base de datos. Si el nombre del campo es el mismo que el nombre de la columna de la tabla, el Caption no necesita ser especificado.

public DataTable Table { get; set; }
