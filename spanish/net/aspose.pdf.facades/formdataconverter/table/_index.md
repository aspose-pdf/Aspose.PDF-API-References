## FormDataConverter.Table property

Obtiene o establece el contenedor de datos central, un DataTable. Debe definirse antes de convertir datos de un formato a otro formato. Deben definirse las Columnas y TableName de DataTable. TableName es el nombre de la tabla en el base de datos. ColumnName de cada columna es el nombre de campo calificado del pdf. Caption de cada columna es el nombre de columna de la tabla en la base de datos. Si el nombre del campo es el mismo que el nombre de la columna de la tabla, no es necesario especificar el Título.
