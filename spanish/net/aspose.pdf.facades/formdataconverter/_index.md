## FormDataConverter class

Representa una clase para convertir datos de un formato a otro formato. Puede convertir los datos en fdf/xml/pdf/xfdf a OLEDB/OdbcDB. También puede convertir los datos en OLEDB/OdbcDB a los datos en fdf/xml/xfdf. Puede convertir el fdf al xml con la etiqueta "hard-named".
