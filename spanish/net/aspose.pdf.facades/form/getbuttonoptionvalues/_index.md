## Form.GetButtonOptionValues method

Obtiene los campos de opción del botón de opción y los valores relacionados según el nombre del campo. Este método tiene significado para los grupos de botones de opción.

```csharp
public Dictionary<string, string> GetButtonOptionValues(string fieldName)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| fieldName | String | Nombre del campo |

### Valor_devuelto

Tabla hash de valores de opción con clave por nombre de elemento de formulario

### Ejemplos

```csharp
Form form = new Form("PdfForm.pdf");
Hashtable values = form.GetButtonOptionValues("Color");
Console.WriteLine(values["White"].ToString());
Console.WriteLine(values["Black"].ToString());
```
