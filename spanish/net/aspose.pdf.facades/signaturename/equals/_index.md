## Método SignatureName.Equals

Determina si esta instancia y un objeto especificado son iguales.

```csharp
public override bool Equals(object obj)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| obj | Object | El objeto a comparar con la instancia actual. |

### Valor de Retorno

True si el objeto especificado es igual a la instancia actual; de lo contrario, false.
