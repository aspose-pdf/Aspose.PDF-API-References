## Método Stamp.SetImageSize

Establece el tamaño del sello de imagen. La imagen se escalará de acuerdo con los valores especificados.

```csharp
public void SetImageSize(float width, float height)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| width | Single | Ancho de la imagen. |
| height | Single | Altura de la imagen. |
