## Stamp.Pages property

Obtiene o establece una matriz con el número de páginas que se verán afectadas por el sello. Si Páginas = nulo, todas las páginas del documento se ven afectadas.

```csharp
public int[] Pages { get; set; }
```

### Ejemplos

```csharp
PdfFileStamp fileStamp = new PdfFileStamp("input.pdf", "output.pdf");
Stamp stamp = new Aspose.Pdf.Facades.Stamp();
stamp.BindLogo(new FormattedText(text));
// poner sello solo en la 1ra, 4ta y 6ta página.
stamp.Pages = new int[] { 1, 4, 6 };
fileStamp.AddStamp(stamp);
fileStamp.Close();
```
