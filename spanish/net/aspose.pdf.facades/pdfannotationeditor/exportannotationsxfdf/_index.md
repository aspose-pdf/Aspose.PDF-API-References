
| Parámetro | Escribe | Descripción |

| start | Int32 | Página de inicio desde la que se exportarán las anotaciones del documento. |
| end | Int32 | Página final a la que se exportarán las anotaciones del documento. |
| annotTypes | String[] | Es necesario exportar la matriz de tipos de anotaciones. |
