## Propiedad DocumentPrivilege.PrintAllowLevel

Obtiene y establece el nivel de impresión del privilegio del documento. Al igual que la configuración de Impresión Permitida de Adobe Professional. 0: Ninguno. 1: Baja Resolución (150 dpi). 2: Alta Resolución.

```csharp
public int PrintAllowLevel { get; set; }
```

## Observaciones

Si la propiedad tiene un valor de -1, entonces el nivel es indefinido.
