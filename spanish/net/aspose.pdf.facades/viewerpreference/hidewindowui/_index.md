## Campo ViewerPreference.HideWindowUI

Una bandera que especifica si se deben ocultar los elementos de la interfaz de usuario en la ventana del documento (como las barras de desplazamiento y los controles de navegación), dejando solo el contenido del documento mostrado.

```csharp
public const int HideWindowUI;
```
