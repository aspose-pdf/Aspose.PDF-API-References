## Propiedad PdfFileStamp.OptimizeSize

Obtiene o establece la bandera de optimización. Los flujos de recursos iguales en el archivo resultante se combinan en un objeto PDF si esta bandera está establecida. Esto permite disminuir el tamaño del archivo resultante, pero puede causar una ejecución más lenta y mayores requisitos de memoria. Valor predeterminado: falso.

```csharp
public bool OptimizeSize { get; set; }
```
