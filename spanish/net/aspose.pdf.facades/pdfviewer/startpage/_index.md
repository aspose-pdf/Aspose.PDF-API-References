## Evento PdfViewer.StartPage

Ocurre antes de que una página comience a imprimirse.

```csharp
public event EventHandler<StartEndPageEventArgs> StartPage;
```

## Observaciones

Este evento se activa para cada página que está a punto de ser impresa. Se puede utilizar para realizar acciones personalizadas antes de que comience la impresión de la página.
