## PdfViewer.AutoResize property

Obtiene o establece un valor booleano que indica si el archivo se imprimirá con tamaño optimizado.  Si es falso, imprima la página sin escala de página. Si es verdadero, imprima la página con escala para ajustarse al área imprimible.
