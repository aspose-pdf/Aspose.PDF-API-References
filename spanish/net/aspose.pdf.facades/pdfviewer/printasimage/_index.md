## PdfViewer.PrintAsImage property

Establece u obtiene un modo para que PdfViewer imprima como imagen.

```csharp
public bool PrintAsImage { get; set; }
```

### Observaciones

Si es verdadero, imprime siempre como imagen (genera una imagen que se imprime) Si es falso, imprime directamente en el dispositivo si se admiten todas las funciones. En caso de que el documento contenga funciones no admitidas, el sistema puede decidir automáticamente imprimir como imagen. El valor predeterminado es falso.
