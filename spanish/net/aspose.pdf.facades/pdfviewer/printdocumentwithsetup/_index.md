## PdfViewer.PrintDocumentWithSetup method

Imprime el documento PDF con un cuadro de diálogo de configuración. Elija una impresora usando el diálogo.

```csharp
public void PrintDocumentWithSetup()
```

### Ejemplos

```csharp
[C#]
dfViewer viewer = new PdfViewer();
iewer.OpenPdfFile(@"d:\test.pdf");
iewer.AutoResize = true;         //imprime el archivo con el tamaño ajustado
iewer.AutoRotate = true;         //imprime el archivo con rotación ajustada
iewer.PrintPageDialog=false;//no producir el cuadro de diálogo de número de página al imprimir
iewer.PrintDocumentWithSetup();
iewer.ClosePdfFile();
```
