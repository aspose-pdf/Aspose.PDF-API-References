## Evento PdfViewer.EndPage

Ocurre cuando la impresión de una página termina en el PdfViewer.

```csharp
public event EventHandler<StartEndPageEventArgs> EndPage;
```

## Observaciones

Este evento se activa después de que una página ha sido procesada en el PdfViewer. Se puede utilizar para realizar acciones o actualizaciones después de la impresión de una página.
