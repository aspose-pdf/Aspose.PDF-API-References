## Propiedad PdfFileEditor.OptimizeSize

Obtiene o establece el indicador de optimización. Los flujos de recursos iguales en el archivo resultante se fusionan en un objeto PDF si se establece este indicador. Esto permite reducir el tamaño del archivo resultante, pero puede causar una ejecución más lenta y mayores requisitos de memoria. Valor predeterminado: falso.

```csharp
public bool OptimizeSize { get; set; }
```
