## Propiedad PdfFileEditor.PreserveUserRights

Si es verdadero, los derechos de usuario del primer documento se aplican al documento concatenado. Los derechos de usuario de todos los demás documentos se ignoran.

```csharp
public bool PreserveUserRights { get; set; }
```
