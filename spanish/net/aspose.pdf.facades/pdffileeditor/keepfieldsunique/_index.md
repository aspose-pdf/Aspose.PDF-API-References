## Propiedad PdfFileEditor.KeepFieldsUnique

Si es verdadero, los nombres de los campos se harán únicos cuando se concatenen formularios. Se agregarán sufijos a los nombres de los campos, el template de sufijo puede ser especificado en la propiedad UniqueSuffix.

```csharp
public bool KeepFieldsUnique { get; set; }
```
