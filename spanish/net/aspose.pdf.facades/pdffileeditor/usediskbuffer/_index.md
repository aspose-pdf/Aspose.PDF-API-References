## Propiedad PdfFileEditor.UseDiskBuffer

Si se utiliza esta opción, el documento de destino se guardará en el disco periódicamente y la concatenación posterior se aplicará como actualizaciones incrementales.

public bool UseDiskBuffer { get; set; }
