## PdfFileEditor.UniqueSuffix property

Formato del sufijo que se agrega al nombre del campo para que sea único cuando se concatenan formularios. Esta cadena debe contener %NUM% subcadena que se reemplazará con números. Por ejemplo, si UniqueSuffix = "ABC%NUM%", entonces para los nombres del campo "fieldName" serán: fieldNameABC1, fieldNameABC2, fieldNameABC3 etc.

```csharp
public string UniqueSuffix { get; set; }
```

### Ejemplos

```csharp
PdfFileEditor ed = new PdfFileEditor();
ed.UniqueSuffix = "_%NUM%";
```
