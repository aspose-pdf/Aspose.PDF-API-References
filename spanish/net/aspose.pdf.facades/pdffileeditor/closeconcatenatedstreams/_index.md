## Propiedad PdfFileEditor.CloseConcatenatedStreams

Si se establece en verdadero, los flujos se cierran después de la operación.

```csharp
public bool CloseConcatenatedStreams { get; set; }
```

## Ejemplos

```csharp
PdfFileEditor pfe = new PdfFileEditor();
pfe.CloseConcatenatedStreams = true;
```
