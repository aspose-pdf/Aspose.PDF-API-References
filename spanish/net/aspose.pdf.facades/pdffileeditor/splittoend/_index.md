```csharp
public bool SplitToEnd(Stream inputStream, int location, Stream outputStream)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| inputStream | Stream | Fuente Pdf archivo Stream. |
| location | Int32 | La posición de división. |
| outputStream | Stream | Archivo PDF de salida Stream. |

### Valor_devuelto

Verdadero para el éxito, o falso.

### Observaciones

Los flujos NO se cierran después de esta operación a menos que se especifique CloseConcatedStreams.

### Ejemplos

```csharp
PdfFileEditor pfe = new PdfFileEditor();
Stream sourceStream = new FileStream("file1.pdf", FileMode.Open, FileAccess.Read);
Stream outStream = new FileStream("out.pdf", FileMode.Create, FileAccess.Write);
pfe.SplitToEnd(sourceStream, 5, outStream);
```
