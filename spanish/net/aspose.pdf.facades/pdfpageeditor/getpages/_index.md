## Método PdfPageEditor.GetPages

Devuelve el número total de páginas.

```csharp
public int GetPages()
```

### Valor de Retorno

Número de páginas.

## Ejemplos

El siguiente ejemplo demuestra el uso del método GetPages():

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
Console.WriteLine("Document has: " + editor.GetPages());
```
