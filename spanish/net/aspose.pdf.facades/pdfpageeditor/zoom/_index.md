## PdfPageEditor.Zoom property

Obtiene o establece el coeficiente de zoom. El valor 1.0 corresponde al 100%. El valor predeterminado es 1.0.  El siguiente ejemplo muestra cómo cambiar el zoom de las páginas del documento.

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
editor.Zoom = 0.5f;
```
