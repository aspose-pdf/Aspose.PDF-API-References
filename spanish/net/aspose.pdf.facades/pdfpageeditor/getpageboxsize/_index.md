## PdfPageEditor.GetPageBoxSize method

Devuelve el tamaño del cuadro especificado en el documento.

```csharp
public Rectangle GetPageBoxSize(int page, string pageBoxName)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| page | Int32 | Índice de páginas. Las páginas del documento están numeradas a partir del 1. |
| pageBoxName | String | Nombre del tipo de caja. Los valores válidos son: "arte", "sangrado", "recortar", "medios", "recortar". |

### Valor_devuelto

Rectángulo que contiene el cuadro solicitado.

### Ejemplos

El siguiente ejemplo muestra cómo obtener el cuadro de medios de la primera página:

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
System.Drawing.Rectangle rect = editor.GetBoxSize(1, "media");
```
