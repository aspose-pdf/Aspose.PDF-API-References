## PdfExtractor.ExtractTextMode property

Establece el modo para extraer el resultado del texto.

```csharp
public int ExtractTextMode { get; set; }
```

### El valor de la propiedad

0 es el modo de texto puro y 1 es el modo de pedido sin formato. El valor predeterminado es 0.

### Ejemplos

El ejemplo demuestra la`ExtractTextMode` uso de propiedad en el escenario de extracción de texto.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractTextMode = 1;
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```
