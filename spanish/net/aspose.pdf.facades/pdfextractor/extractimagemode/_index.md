## Propiedad PdfExtractor.ExtractImageMode

Establece el modo para el proceso de extracción de imágenes.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## Observaciones

El valor predeterminado es ExtractImageMode.DefinedInResources que extrae todas las imágenes definidas en los recursos. Para extraer imágenes que se muestran realmente, se debe usar el modo ExtractImageMode.ActuallyUsed.
