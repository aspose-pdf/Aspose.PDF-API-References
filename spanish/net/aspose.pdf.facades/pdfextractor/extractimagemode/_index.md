## PdfExtractor.ExtractImageMode property

Establece el modo para el proceso de extracción de imágenes.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

### Observaciones

El valor predeterminado es ExtractImageMode.DefinedInResources que extrae todas las imágenes definidas en resources. Para extraer las imágenes mostradas, se debe usar el modo ExtractImageMode.ActuallyUsed.
