## Propiedad PdfExtractor.IsBidi

Es verdadero cuando el texto tiene símbolos hebreos o árabes. Este caso debe ser considerado especialmente porque las funciones de cadena cambian su comportamiento y comienzan a procesar el texto de derecha a izquierda (excepto números y otros caracteres no textuales).

```csharp
public bool IsBidi { get; }
```
