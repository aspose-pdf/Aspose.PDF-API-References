## GetNextPageText(Stream) {#getnextpagetext}

Guarda el texto de una página para transmitir.

```csharp
public void GetNextPageText(Stream outputStream)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| outputStream | Stream | La secuencia para guardar el texto. |

### Ejemplos

El ejemplo demuestra la[`GetNextPageText`](../getnextpagetext) uso del método en el escenario de extracción de texto.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(TestPath + @"Aspose.Pdf.Kit.Pdf");
extractor.ExtractText(Encoding.Unicode);
String prefix = TestPath + @"Aspose.Pdf.Kit";
String suffix = ".txt";
int pageCount = 1;
while (extractor.HasNextPageText())
{
    FileStream fs = new FileStream(prefix + pageCount + suffix, FileMode.Create);
    extractor.GetNextPageText(prefix + pageCount + suffix);
    fs.Close();
    pageCount++;
}
```
