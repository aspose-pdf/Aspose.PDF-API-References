## Método FormEditor.SetFieldScript

Establecer JavaScript para un campo PushButton. Si existía un JavaScript antiguo, será reemplazado por el nuevo.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fieldName | String | El nombre del campo completamente calificado. |
| script | String | El Java script que se añadirá/colocará en un campo de botón pulsador. |

### Valor de Retorno

true si el script del campo se estableció correctamente.
