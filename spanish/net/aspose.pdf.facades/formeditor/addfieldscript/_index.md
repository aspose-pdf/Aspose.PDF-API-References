## Método FormEditor.AddFieldScript

Agregar JavaScript para un campo PushButton. Si existe un evento antiguo, se agrega un nuevo evento después de él.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fieldName | String | El nombre de campo completamente calificado. |
| script | String | El script de Java que se va a agregar/colocar en un campo de botón pulsador. |

### Valor de Retorno

Verdadero en caso de que el script se haya agregado con éxito.
