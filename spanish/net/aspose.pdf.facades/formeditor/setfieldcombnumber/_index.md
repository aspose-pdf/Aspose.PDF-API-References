## FormEditor.SetFieldCombNumber method

Establece el número de peines para un campo de texto regular de una sola línea (el campo se se divide automáticamente en tantas posiciones o peines igualmente espaciados como el valor del parámetro combNumber).

```csharp
public bool SetFieldCombNumber(string fieldName, int combNumber)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| fieldName | String | El nombre de campo calificado. |
| combNumber | Int32 | El número de peines para dividir el campo. |

### Valor_devuelto

Si tiene éxito, devuelve verdadero; de lo contrario, falso.

### Ejemplos

```csharp
FormEditor formEditor = new FormEditor("PdfWithAcroForm.pdf", "FormEditor_SetFieldComb.pdf"));
formEditor.SetFieldCombNumber("textCombField", 5);
```
