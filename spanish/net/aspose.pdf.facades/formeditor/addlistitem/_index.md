## AddListItem(string, string[]) {#addlistitem_1}

Agregar un nuevo elemento con Valor de exportación al campo de cuadro de lista existente, solo para el campo de cuadro combinado de AcroForm.

```csharp
public void AddListItem(string fieldName, string[] exportName)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| fieldName | String | Nombre del campo al que se agregarán los elementos. |
| exportName | String[] | Una matriz de cadenas que denota un nuevo elemento de lista con Valor de exportación, es decir (Etiqueta de elemento, Valor de exportación). |

### Ejemplos

```csharp
FormEditor fe = new FormEditor("PdfForm.pdf", "FormEditor_AddListItem2.pdf");
fe.AddListItem("listboxField", new string[] { "4", "Item4(Added)" });
```
