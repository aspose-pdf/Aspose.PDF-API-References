## Propiedad Bookmark.Action

Obtiene o establece la acción vinculada con el marcador. Si se presenta PageNumber, la acción no se puede especificar. El tipo de acción incluye: "GoTo", "GoToR", "Launch", "Named".

```csharp
public string Action { get; set; }
```
