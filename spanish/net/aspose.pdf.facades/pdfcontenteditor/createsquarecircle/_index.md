## PdfContentEditor.CreateSquareCircle method

Crea una anotación de círculo cuadrado.

```csharp
public void CreateSquareCircle(Rectangle rect, string contents, Color clr, bool square, int page, 
    int borderWidth)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| rect | Rectangle | El rectángulo de anotación que define la ubicación de la anotación en la página. |
| contents | String | El contenido de la anotación. |
| clr | Color | El color del cuadrado o círculo. |
| square | Boolean | Verdadero (cuadrado), falso (señor). |
| page | Int32 | El número de página original donde se creará la anotación. |
| borderWidth | Int32 | El ancho del borde del cuadrado o círculo. |

### Ejemplos

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateSquareCircle(new System.Drawing.Rectangle(0, 0, 100, 100),
    "Welcome to Aspose", System.Drawing.Color.Red, false, 1, 5);
editor.Save("example_out.pdf");
```
