## Método PdfContentEditor.CreateSound

Crea anotaciones de sonido.

```csharp
public void CreateSound(Rectangle rect, string filePath, string name, int page, string rate)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| rect | Rectangle | El rectángulo de anotación que define la ubicación de la anotación en la página. |
| filePath | String | La ruta del archivo de sonido. |
| name | String | El nombre de un ícono que se utilizará para mostrar la anotación, incluye: Speaker y Mic. |
| page | Int32 | La página en la que se crea la anotación de sonido. |
| rate | String | La tasa de muestreo, en muestras por segundo. |
