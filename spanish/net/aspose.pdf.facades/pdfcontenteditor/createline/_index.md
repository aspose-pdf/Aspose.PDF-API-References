## PdfContentEditor.CreateLine method

Crea anotación de línea.

```csharp
public void CreateLine(Rectangle rect, string contents, float x1, float y1, float x2, float y2, 
    int page, int border, Color clr, string borderStyle, int[] dashArray, string[] LEArray)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| rect | Rectangle | El rectángulo de anotación que define la ubicación de la anotación en la página. |
| contents | String | El contenido de la anotación. |
| x1 | Single | La coordenada horizontal inicial de la línea. |
| y1 | Single | La coordenada vertical inicial de la línea. |
| x2 | Single | La coordenada horizontal final de la línea. |
| y2 | Single | La coordenada vertical final de la línea. |
| page | Int32 | El número de página original donde se creará la anotación. |
| border | Int32 | El ancho del borde en puntos. Si este valor es 0, no se dibuja ningún borde. El valor predeterminado es 1. |
| clr | Color | El color de la línea. |
| borderStyle | String | El estilo de borde que especifica el ancho y el patrón de guiones que se utilizará para dibujar la línea. Este valor puede ser: "S" (Sólido), "D" (Discontinua), "B" (Bisel), "I" (Recuadro) , "U" (subrayado). |
| dashArray | Int32[] | Una matriz de guiones que define un patrón de guiones y espacios que se usará para dibujar un borde discontinuo. Si se usa, borderSyle debe establecerse en "D". |
| LEArray | String[] | Una matriz de dos valores que especifican respectivamente el estilo inicial y final de la línea de dibujo. Los valores pueden ser: "Cuadrado", "Círculo", "Diamante", "Flecha abierta", "Flecha cerrada", "Ninguno", "Extremo" , "ROpenArrow", "RClosedArrow", "barra". |

### Ejemplos

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateLine(new System.Drawing.Rectangle(0, 0, 100, 100), "Welcome to Aspose", 0, 0, 100, 100,
    1, 1, System.Drawing.Color.Red, "D", new int[] {2, 3}, new string[] {"OpenArrow", "ClosedArrow"});
editor.Save("example_out.pdf");
```
