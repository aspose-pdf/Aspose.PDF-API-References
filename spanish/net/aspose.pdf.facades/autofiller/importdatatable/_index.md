## Método AutoFiller.ImportDataTable

Importa datos del tipo DataTable. El nombre de cada columna del dataTable debe ser el mismo que el de un campo del pdf de plantilla, respetando mayúsculas y minúsculas.

```csharp
public void ImportDataTable(DataTable dataTable)
```

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| dataTable | DataTable | El dataTable que contiene todos los datos de la base de datos u otra fuente de datos. |
