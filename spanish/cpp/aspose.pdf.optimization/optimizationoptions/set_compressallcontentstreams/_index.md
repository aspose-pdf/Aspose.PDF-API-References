## OptimizationOptions::set_CompressAllContentStreams method


Si se establece en **true**

, todos los flujos de contenido de página no comprimidos se comprimirán usando el filtro FlateDecode durante [Document::OptimizeResources()](../../../aspose.pdf/document/optimizeresources/). El valor predeterminado es **false**

para preservar la compatibilidad hacia atrás.
