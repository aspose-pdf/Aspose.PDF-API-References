## OptimizationOptions::set_LinkDuplicateStreams method


Si esta bandera se establece en verdadero, los flujos de recursos se analizarán. Si se encuentran flujos duplicados (p. ej., si el contenido del flujo es igual), entonces esos flujos se almacenarán como un solo objeto. Esto permite reducir el tamaño del documento en algunos casos (por ejemplo, cuando el mismo documento se concatenó varias veces).
