## ImageCompressionVersion enum


Describe versiones del algoritmo de compresión de imágenes.

```cpp
enum class ImageCompressionVersion
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Standard | 0 | Algoritmo estándar. Valor predeterminado. |
| Fast | 2 | Algoritmo mejorado más rápido que el estándar pero no aplicable a todos los casos. |
| Mixed | 3 | Utilice el algoritmo rápido cuando sea posible y el estándar para otros casos. Puede ser más lento que "Fast" pero puede producir una mejor compresión. |
