## XfaParserOptions::get_Signed method


Si esta propiedad es verdadera, el documento se convertirá utilizando el flujo de formulario xfa (si existe). Si es falsa, el flujo de formulario xfa se ignorará. Esta propiedad se introdujo porque no está claro cómo calcular la suma de verificación utilizada para comprobar la firma.
