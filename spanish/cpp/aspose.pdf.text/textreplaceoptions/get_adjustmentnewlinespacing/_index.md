## TextReplaceOptions::get_AdjustmentNewLineSpacing method


Obtiene el valor del interlineado que se usa si el ajuste de reemplazo se fuerza a crear una nueva línea de texto. El valor esperado es un multiplicador del tamaño de fuente del texto reemplazado. El valor predeterminado es 1.2.
