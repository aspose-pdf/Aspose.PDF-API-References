## FontSizeAdjustment enum


Especifica una política sobre cómo debe ajustarse el tamaño de fuente del texto para encajar dentro de un área contenedora.

```cpp
enum class FontSizeAdjustment
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Ninguno | 0 | El tamaño de fuente no se cambia. |
| ShrinkToFit | 1 | El tamaño de fuente se reduce si el texto es demasiado grande para encajar en los límites. El tamaño de fuente nunca se aumenta si el texto es más pequeño que los límites. |
| ScaleToFill | 2 | El tamaño de fuente se ajusta (tanto reduciéndose como aumentándose) para que el texto llene los límites del rectángulo tanto como sea posible. |
