## TextExtractionOptions::set_ScaleFactor method


Establece el factor que se aplicará para escalar el tamaño de la fuente durante la extracción en modo puro. Un valor menor genera más espacios en el texto extraído. El valor predeterminado es 1 - sin escalado; establecer el valor a cero permite que el algoritmo elija el escalado automáticamente.
