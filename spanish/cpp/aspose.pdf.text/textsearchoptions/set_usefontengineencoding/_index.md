## TextSearchOptions::set_UseFontEngineEncoding method


Establece la indicación de que el texto se buscará utilizando la codificación del motor de fuentes. true - indica que se usará la codificación del motor de fuentes (pruebe esto si la búsqueda de texto falla debido a una codificación imperfecta en el documento). false - indica que se usará la codificación de fuentes del documento (valor por defecto).
