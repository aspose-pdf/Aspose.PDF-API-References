## TextSearchOptions::get_SearchForTextRelatedGraphics method


Obtiene el valor que permite buscar gráficos relacionados con el texto (subrayado, fondo, etc.) durante la búsqueda de texto. true - la búsqueda de gráficos relacionados con el texto se realizará (valor por defecto). false - los elementos gráficos que puedan estar presentes en el documento fuente serán ignorados. Configure esto en caso de problemas de rendimiento o si no necesita manejar subrayado, fondo o recorte.
