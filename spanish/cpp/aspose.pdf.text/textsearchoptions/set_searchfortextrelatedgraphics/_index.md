## TextSearchOptions::set_SearchForTextRelatedGraphics method


Establece el valor que permite buscar gráficos relacionados con el texto (subrayado, fondo, etc.) durante la búsqueda de texto. true - la búsqueda de gráficos relacionados con el texto se realizará (valor por defecto). false - los elementos gráficos que puedan estar presentes en el documento fuente se ignorarán. Configure esto en caso de problemas de rendimiento o si no es necesario manejar subrayado, fondo o recorte.
