## TextSearchOptions::set_StoredGraphicElementsMaxCount method


Establece el valor que limita la búsqueda de gráficos relacionados con el texto (subrayado, fondo, etc.) en una página al número especificado de elementos. El valor por defecto es 250. Use un valor menor en caso de problemas de rendimiento, pruebe un valor mayor si algunos elementos gráficos no se encuentran.
