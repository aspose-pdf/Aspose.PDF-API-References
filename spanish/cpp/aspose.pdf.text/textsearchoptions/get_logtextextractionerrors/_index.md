## TextSearchOptions::get_LogTextExtractionErrors method


Obtiene la indicación de que los errores de extracción de texto (decodificación) se registrarán en el absorbente de texto (fragmento). true - indica que los errores de extracción de texto (decodificación) se registrarán. Puede disminuir el rendimiento. false (por defecto) - no se registran errores.
