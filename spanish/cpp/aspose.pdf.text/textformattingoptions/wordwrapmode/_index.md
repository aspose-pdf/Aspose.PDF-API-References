## WordWrapMode enum


Define estrategias de ajuste de palabras.

```cpp
enum class WordWrapMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| NoWrap | 0 | No se realiza ajuste. |
| DiscretionaryHyphenation | 1 | Se realiza la guionización discrecional. Permite dividir palabras en el medio. |
| PorPalabras | 2 | El ajuste de texto solo envuelve palabras completas. Si la palabra completa no puede ajustarse, se intenta usar la guionización discrecional. |
| Undefined | 3 | [WordWrapMode](./) no está configurado. Se usarán las estrategias de ajuste de la estructura de nivel superior (celda de tabla, párrafo, etc.). |
