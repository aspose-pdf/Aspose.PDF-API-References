## TextParagraph::AppendLine(const System::SharedPtr\<TextFragment\>\&, const System::SharedPtr\<TextState\>\&, float) method


Añade una línea de texto con parámetros de estado de texto.

```cpp
void Aspose::Pdf::Text::TextParagraph::AppendLine(const System::SharedPtr<TextFragment> &line, const System::SharedPtr<TextState> &textState, float lineSpacing)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| línea | const System::SharedPtr\<TextFragment\>\& | El texto de la nueva línea. |
| textState | const System::SharedPtr\<TextState\>\& | [Text](../../) estado de la nueva línea. |
| lineSpacing | float | Espaciado adicional (0.0 es el valor predeterminado y corresponde a la altura de línea de texto por defecto). El valor de espaciado se suma al espaciado de línea predeterminado para la línea en cuestión, por lo que puede especificar 12.0 para obtener una fila vacía después de una línea de texto dibujada con fuente de 12pt. |
