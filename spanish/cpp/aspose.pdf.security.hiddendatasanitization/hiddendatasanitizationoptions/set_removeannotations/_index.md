## HiddenDataSanitizationOptions::set_RemoveAnnotations method


Establece un valor que indica si se deben eliminar las anotaciones del documento. Cuando está habilitado, todas las anotaciones presentes en el documento se eliminarán durante el proceso de sanitización. Se aplicarán anotaciones de redacción.
