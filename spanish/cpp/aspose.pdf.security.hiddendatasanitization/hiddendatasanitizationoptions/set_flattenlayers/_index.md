## HiddenDataSanitizationOptions::set_FlattenLayers method


Establece la opción para aplanar las capas en el documento PDF. Cuando está habilitado, todas las capas del documento se fusionan en una sola capa, eliminando su estructura separada. Esta opción es útil para sanitizar documentos simplificando su contenido y asegurando que no haya datos ocultos dentro de las capas.
