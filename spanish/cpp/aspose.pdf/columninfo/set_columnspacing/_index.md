## ColumnInfo::set_ColumnSpacing method


Establece una cadena que contiene el espaciado entre columnas. El valor de cada espaciado debe estar separado por un espacio. La unidad predeterminada es punto, pero también se admiten cm y pulgada. Por ejemplo, "120 2.5cm 1.5inch".

```cpp
void Aspose::Pdf::ColumnInfo::set_ColumnSpacing(const System::String &value)
```

## Observaciones


Si esta propiedad no se establece, se utilizará el valor predeterminado 0 para cada espaciado.
