## ColumnInfo::get_ColumnSpacing method


Obtiene una cadena que contiene el espaciado entre columnas. El valor de cada espaciado debe estar separado por un espacio. La unidad predeterminada es punto, pero también se admiten cm y pulgada. Por ejemplo, "120 2.5cm 1.5inch".

```cpp
System::String Aspose::Pdf::ColumnInfo::get_ColumnSpacing() const
```

## Observaciones


Si esta propiedad no se establece, se utilizará el valor predeterminado 0 para cada espaciado.
