## PageLayout enum


Describe el diseño de página.


### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| SinglePage | 0 | Página única. |
| OneColumn | 1 | Mostrar las páginas en una columna. |
| TwoColumnLeft | 2 | Muestra las páginas en dos columnas, con las páginas impares a la izquierda. |
| TwoColumnRight | 3 | Mostrar las páginas en dos columnas, con las páginas impares a la derecha. |
| TwoPageLeft | 4 | Mostrar las páginas de dos en dos, con las páginas impares a la izquierda. |
| TwoPageRight | 5 | Mostrar las páginas de dos en dos, con las páginas impares a la derecha. |
| Predeterminado | 6 | Diseño predeterminado. |
