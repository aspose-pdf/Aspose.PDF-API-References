## HtmlSaveOptions::set_UseZOrder method


Si el atributo UseZORder se establece en true, los gráficos y el texto se añaden al documento HTML resultante de acuerdo con el orden Z del documento PDF original. Si este atributo es false, todos los gráficos se colocan en una sola capa, lo que puede causar efectos innecesarios en objetos superpuestos.
