## HtmlSaveOptions::set_PreventGlyphsGrouping method


Este atributo activa el modo en el que los glifos de texto no se agruparán en palabras y cadenas. Este modo permite mantener la máxima precisión al posicionar los glifos en la página y puede usarse para convertir documentos con notas musicales o glifos que deben colocarse por separado. Este parámetro se aplicará al documento solo cuando el valor del atributo FixedLayout sea true.
