## HtmlMarkupGenerationModes enum


A veces existen requisitos específicos para el HTML creado. Este enum define los modos de preparación de HTML que pueden usarse durante la conversión de PDF a HTML para cumplir con dichos requisitos específicos.

```cpp
enum class HtmlMarkupGenerationModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| WriteAllHtml | 0 | Modo predeterminado cuando no hay requisitos específicos. Se generará una salida que contendrá todas las partes del HTML sin ningún procesamiento adicional especial. |
| WriteOnlyBodyContent | 1 | se eliminará todo el contenido HTML que esté fuera del cuerpo del HTML, es decir, solo quedará el contenido que esté dentro de las etiquetas **<body>****</body>**. |
