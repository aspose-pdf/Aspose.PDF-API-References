## ParagraphPositioningMode enum


Especifica la variante para determinar la ubicación del elemento en la página.

```cpp
enum class ParagraphPositioningMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Predeterminado | 0 | La ubicación se determina por los elementos colocados previamente. Añadir un elemento se tiene en cuenta al determinar la ubicación de los elementos posteriores. |
| Absolute | 1 | La ubicación se especifica mediante los valores [Left](../left/) y Top, no depende de los elementos previos y no afecta la ubicación de los posteriores. |
