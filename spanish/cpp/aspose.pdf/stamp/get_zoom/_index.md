## Stamp::get_Zoom method


Factor de zoom del sello. Permite escalar el sello. Tenga en cuenta que el par de propiedades ZoomX y ZoomY permite establecer el factor de zoom para cada eje por separado. La configuración de esta propiedad cambia tanto las propiedades ZoomX como ZoomY. Si ZoomX y ZoomY son diferentes, la propiedad Zoom devuelve el valor de ZoomX.

double Aspose::Pdf::Stamp::get_Zoom() const
