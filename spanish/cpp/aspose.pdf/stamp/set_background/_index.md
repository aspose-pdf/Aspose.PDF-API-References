## Stamp::set_Background method


Establece o obtiene un valor booleano que indica que el contenido está sellado como fondo. Si el valor es verdadero, el contenido del sello se coloca en la parte inferior. Por defecto, el valor es falso, el contenido del sello se coloca en la parte superior.
