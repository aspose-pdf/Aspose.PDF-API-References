## FontSubsetStrategy enum


Enumera estrategias para el subconjunto de fuentes.

```cpp
enum class FontSubsetStrategy : uint8_t
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| SubsetEmbeddedFontsOnly | 0 | Declara que solo se subsegmentarán las fuentes que ya están incrustadas en el documento. |
| SubsetAllFonts | 1 | Declara que se subsegmentarán todas las fuentes, tanto las no incrustadas como las incrustadas. |
