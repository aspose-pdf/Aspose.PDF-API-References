## XImage::get_ImageMask method


Obtiene una bandera que indica si la imagen debe tratarse como una máscara de imagen (ver 8.9.6, "Masked Images"). Si esta bandera es verdadera, el valor de BitsPerComponent será 1 y no se especificarán Mask y [ColorSpace](../../colorspace/); las áreas sin máscara se pintarán usando el color de trazo no relleno actual. Valor predeterminado: false.

```cpp
bool Aspose::Pdf::XImage::get_ImageMask()
```

## Observaciones


True indica que la imagen es una máscara de imagen.
