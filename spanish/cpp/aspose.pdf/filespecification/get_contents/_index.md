## FileSpecification::get_Contents method


Obtiene el archivo de contenidos. Esta propiedad devuelve los datos cargados en memoria, lo que puede causar una excepción de falta de memoria para datos grandes. Para reducir el uso de memoria, por favor use StreamContents.
