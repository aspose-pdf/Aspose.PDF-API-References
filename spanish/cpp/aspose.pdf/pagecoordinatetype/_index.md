## PageCoordinateType enum


Describe el tipo de coordenada de página.

```cpp
enum class PageCoordinateType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| MediaBox | 0 | El MediaBox se usa para especificar el ancho y la altura de la página. Para el usuario promedio, esto probablemente equivale al tamaño real de la página. El MediaBox es el cuadro de página más grande en un PDF. Los otros cuadros de página pueden ser del mismo tamaño que el MediaBox, pero no pueden ser más grandes. |
| CropBox | 1 | El CropBox define la región a la que se recortarán los contenidos de la página. Acrobat usa este tamaño para la visualización en pantalla e impresión. |
