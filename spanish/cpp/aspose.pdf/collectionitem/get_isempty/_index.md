## CollectionItem::get_IsEmpty method


Obtiene un valor que indica si el elemento de la colección está vacío.

```cpp
bool Aspose::Pdf::CollectionItem::get_IsEmpty()
```


### ReturnValue

true si el elemento de colección está vacío; de lo contrario, false.
## Observaciones


Esta propiedad devuelve true si el elemento de colección no contiene ningún valor, incluidos valores de cadena, valores dobles, valores enteros y valores de fecha. Si alguno de estos tipos de valor está presente en el elemento de colección, esta propiedad devuelve false.
