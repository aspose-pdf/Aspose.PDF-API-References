## AutoTaggingSettings::set_HeadingRecognitionStrategy method


Establece la estrategia utilizada para reconocer encabezados en el documento durante el autoetiquetado.

```cpp
void Aspose::Pdf::AutoTaggingSettings::set_HeadingRecognitionStrategy(Aspose::Pdf::HeadingRecognitionStrategy value)
```

## Observaciones


La propiedad [HeadingRecognitionStrategy](../../headingrecognitionstrategy/) determina cómo se identifican los encabezados en el documento. Las estrategias disponibles incluyen reconocer encabezados basándose en esquemas, análisis heurístico o detección automática. Establecer esta propiedad a [HeadingRecognitionStrategy::None](../../headingrecognitionstrategy/) desactiva el reconocimiento de encabezados.
