## AutoTaggingSettings::set_HeadingLevels method


Establece los niveles de encabezado utilizados para determinar la estructura de los encabezados en un documento PDF.

## Observaciones


La propiedad [HeadingLevels](../../headinglevels/) permite configurar el mapeo de tamaños de fuente a niveles de encabezado. Esto se utiliza durante el proceso de autoetiquetado para identificar y asignar los niveles de encabezado apropiados según el tamaño de fuente de los elementos de texto en el documento.
