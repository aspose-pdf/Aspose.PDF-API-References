## ExtractImageMode enum


Define los diferentes modos que pueden usarse al extraer imágenes de documentos.

```cpp
enum class ExtractImageMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DefinedInResources | 0 | Define el modo de extracción de imágenes en el que se extraen todas las imágenes definidas en los recursos para una página determinada. |
| ActuallyUsed | 1 | Define el modo de extracción de imágenes en el que solo se extraen aquellas imágenes que se muestran realmente en una página. |
