## HtmlFragment::set_HtmlLoadOptions method


Establece [HtmlLoadOptions](../../htmlloadoptions/) que se utilizarán para cargar (y renderizar) HTML en esta instancia de la clase. Por favor, úselo cuando sea necesario usar una configuración específica para la importación de HTML para esta u otra instancia (p. ej., cuando esta u otra instancia deba usar una BasePath específica para HTML importado o deba usar un cargador específico de recursos externos). Si el parámetro es predeterminado (null), entonces se utilizarán las opciones estándar de carga de HTML.
