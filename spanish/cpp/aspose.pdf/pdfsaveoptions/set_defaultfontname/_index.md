## PdfSaveOptions::set_DefaultFontName method


Nombre de fuente usado por defecto para fuentes que están ausentes en el ordenador. Cuando el documento PDF que se guarda contiene fuentes que no están disponibles en el propio documento ni en el dispositivo, la API reemplaza esas fuentes con la fuente predeterminada (si se encuentra una fuente con [DefaultFontName](../) en el dispositivo).
