## HtmlPageLayoutOption enum


Especifica banderas que, junto con otras opciones, determinan los tamaños y disposiciones de las páginas.

```cpp
enum class HtmlPageLayoutOption
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | 0 | Valor predeterminado que indica que el [HtmlPageLayoutOption](./) no afectará los tamaños y disposiciones de las páginas. |
| FitToWidestContentWidth | n/a | Indica que el ancho de la página se determina a partir del tamaño del contenido mismo, no del ancho de página especificado. |
| ScaleToPageWidth | 256 | Indica que el contenido del documento se escalará para ajustarse a la página. |
