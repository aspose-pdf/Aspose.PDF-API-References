## HeadingRecognitionStrategy enum


Representa tipos de estrategias de reconocimiento de encabezados.

```cpp
enum class HeadingRecognitionStrategy
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Outlines | 0 | Representa la estrategia de reconocimiento de encabezados mediante outlines. |
| Heuristic | 1 | Representa la estrategia de reconocimiento de encabezados mediante reglas heurísticas y estadísticas de tamaño de fuente. |
| Auto | 2 | Proporciona una selección automática de la estrategia de reconocimiento de encabezados. Esta es la opción predeterminada. Si el documento contiene marcadores, se seleccionará la estrategia [Outlines](../outlines/), de lo contrario [Heuristic](./) |
| Ninguno | 3 | No reconocer encabezados. Esta opción puede ser útil en documentos con formato complejo. |
