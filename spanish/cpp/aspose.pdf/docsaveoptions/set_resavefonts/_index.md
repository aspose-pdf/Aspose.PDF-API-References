## DocSaveOptions::set_ReSaveFonts method


Establece el procedimiento para volver a guardar fuentes. Si se establece en true, recargamos las fuentes en cada página para evitar la influencia de las propiedades de fuentes anteriores y cargamos la fuente recién creada desde cero. Establece esta opción en false si deseas mejorar el rendimiento. El valor predeterminado es true;.
