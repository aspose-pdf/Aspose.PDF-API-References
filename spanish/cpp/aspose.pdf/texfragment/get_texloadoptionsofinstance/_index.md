## TeXFragment::get_TeXLoadOptionsOfInstance method


Obtiene [TeXLoadOptions](../../texloadoptions/) que se utilizarán para cargar (y renderizar) LaTeX en esta instancia de la clase. Úselo cuando sea necesario aplicar una configuración específica para la importación de LaTeX en esta u otra instancia (p. ej., cuando esta u otra instancia deba usar una BasePath específica para LaTeX importado o deba usar un cargador específico de recursos externos). Si el parámetro es el predeterminado (null), se utilizarán las opciones estándar de carga de LaTeX.

```cpp
System::SharedPtr<TeXLoadOptions> Aspose::Pdf::TeXFragment::get_TeXLoadOptionsOfInstance() const
```
