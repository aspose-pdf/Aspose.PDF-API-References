## CollectionField class


Representa una clase de campo de esquema de colección de documentos.

```cpp
class CollectionField : public System::Object
```

## Métodos

| Método | Descripción |
| --- | --- |
| [get_E](./get_e/)() const | Obtiene una bandera que indica si el procesador interactivo de PDF debe proporcionar soporte para editar el valor del campo. Valor predeterminado: false. |
| [get_FiledType](./get_filedtype/)() const | Obtiene el tipo de un valor de campo en una colección de esquema. Este campo describe el tipo de valor correspondiente a [Subtype](../). |
| [get_N](./get_n/)() const | Obtiene el nombre textual del campo que debe presentarse al usuario por el procesador interactivo de PDF. |
| [get_O](./get_o/)() const | Obtiene el orden relativo del nombre del campo en la interfaz de usuario. Los campos deben ordenarse por el procesador interactivo de PDF en orden ascendente. |
| [get_Subtype](./get_subtype/)() const | Obtiene el subtipo de un valor de campo en una colección de esquema. El subtipo del campo de colección o del campo relacionado con archivos que describe este diccionario. Esta entrada identifica el tipo de datos que debe almacenarse en el campo. |
| [get_V](./get_v/)() const | Obtiene la visibilidad inicial del campo en la interfaz de usuario. Valor predeterminado: true. |
