## ConversionEngines enum


Enumera los motores de conversión que pueden usarse para la conversión.

```cpp
enum class ConversionEngines
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| LegacyEngine | 0 | El motor de conversión heredado es un motor de conversión que se utilizó durante un tiempo y se usa por defecto. |
| NewEngine | 1 | El nuevo motor de conversión está actualmente en estado de pruebas A. Lo abrimos en la API pública para obtener comentarios. Actualmente se recomienda usarlo solo cuando el motor de conversión antiguo no pueda convertir el documento por alguna razón. |
