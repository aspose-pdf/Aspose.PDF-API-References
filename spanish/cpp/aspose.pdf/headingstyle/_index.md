## HeadingStyle enum


Define los estilos de serialización disponibles para encabezados. Para la especificación vea CommonMark - ATX headings, respectivamente CommonMark - Setext headings.

```cpp
enum class HeadingStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Atx | 0 | Prefija los encabezados con '#'. |
| Setext | 1 | Subraye los encabezados de nivel 1 y 2 con "======" y "------" respectivamente. |
