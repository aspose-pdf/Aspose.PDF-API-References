## Document::set_OptimizeSize method


Establece una bandera de optimización. Cuando se añaden páginas al documento, los flujos de recursos iguales en el archivo resultante se combinan en un solo objeto PDF si esta bandera está activada. Esto permite reducir el tamaño del archivo resultante pero puede causar una ejecución más lenta y mayores requisitos de memoria. Valor predeterminado: false.
