## Subset enum


Representa el subconjunto de páginas al que puede aplicarse un artefacto de paginación.

```cpp
enum class Subset
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Todo | 0 | Indica que el artefacto se aplica a todas las páginas. |
| Even | 1 | Indica que el artefacto se aplica solo a páginas pares. |
| Odd | 2 | Indica que el artefacto se aplica solo a páginas impares. |
