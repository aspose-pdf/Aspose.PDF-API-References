## ColorDepth enum


Se utiliza para especificar el valor del parámetro pasado a un dispositivo de imagen Tiff.

```cpp
enum class ColorDepth
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Predeterminado | 0 | Profundidad de color predeterminada. Se utiliza una profundidad de 32 bits por defecto. |
| Format24bpp | 1 | Profundidad de 24 bits RGB. |
| Format8bpp | 2 | 8 bits por píxel. Igual a [PixelFormat::Format8bppIndexed](../) |
| Format4bpp | 3 | 4 bits por píxel. Igual a [PixelFormat::Format4bppIndexed](../) |
| Format1bpp | 4 | 1 bit por píxel. Igual a [PixelFormat::Format32bppRgb](../) |
