## TiffSettings::set_Brightness method


Obtiene o establece un límite de valor de la transformación de colores en blanco y negro. Este parámetro puede aplicarse con EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle o [ColorDepth.Format1bpp](../../colordepth/) == 1.

```cpp
void Aspose::Pdf::Devices::TiffSettings::set_Brightness(float value)
```

## Observaciones


El valor de brillo debe estar en el rango de 0 a 1. Por defecto, el valor es igual a 0.33f.
