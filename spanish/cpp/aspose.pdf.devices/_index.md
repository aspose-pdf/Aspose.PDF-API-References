El **[Aspose.Pdf.Devices](./)** espacio de nombres proporciona clases que se utilizan para representar el documento como imagen(es) o texto plano. Así, el documento puede enviarse a dispositivos textuales o gráficos diversos, lo que significa que queremos obtener su representación textual o gráfica.

## Clases

| Clase | Descripción |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Representa un dispositivo de imagen que ayuda a guardar las páginas del documento PDF en bmp. |
| [Device](./device/) | Clase abstracta para todo tipo de dispositivos. [Device](./device/) se utiliza para representar un documento PDF en algún formato. Por ejemplo, la página del documento puede representarse como imagen o texto. |
| [DicomDevice](./dicomdevice/) | Representa un dispositivo de imagen que ayuda a guardar las páginas del documento PDF en formato Dicom. |
| [DocumentDevice](./documentdevice/) | Clase abstracta para todos los dispositivos que se utiliza para procesar todo el documento PDF. |
| [EmfDevice](./emfdevice/) | Representa un dispositivo de imagen que ayuda a guardar las páginas del documento PDF en emf. |
| [GifDevice](./gifdevice/) | Representa un dispositivo de imagen que ayuda a guardar las páginas del documento PDF en gif. |
| [ImageDevice](./imagedevice/) | Una clase abstracta para dispositivos de imagen. |
| [JpegDevice](./jpegdevice/) | Representa un dispositivo de imagen que ayuda a guardar páginas de documentos pdf en jpeg. |
| [Margins](./margins/) | Esta clase representa los márgenes de una imagen. |
| [PageDevice](./pagedevice/) | Clase abstracta para todos los dispositivos que se utilizan para procesar una página específica del documento pdf. |
| [PngDevice](./pngdevice/) | Representa un dispositivo de imagen que ayuda a guardar páginas de documentos pdf en png. |
| [Resolution](./resolution/) | Representa una clase para almacenar la resolución de la imagen. |
| [TextDevice](./textdevice/) | Representa una clase para convertir páginas de documentos pdf en texto. |
| [ThumbnailDevice](./thumbnaildevice/) | Representa un dispositivo de imagen que guarda páginas de documentos pdf en una imagen en miniatura. |
| [TiffDevice](./tiffdevice/) | Esta clase ayuda a guardar el documento pdf página por página en una única imagen tiff. |
| [TiffSettings](./tiffsettings/) | Esta clase representa la configuración para importar pdf a Tiff. |
