## String::CompareOrdinal(const String\&, const String\&) method


Less-equal-greater-compara dos cadenas usando el modo ordinal.

```cpp
static int System::String::CompareOrdinal(const String &strA, const String &strB)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| strA | const String\& | Primera cadena para comparar. |
| strB | const String\& | Segunda cadena para comparar. |

### ReturnValue

Valor negativo si la primera subcadena es menor que la segunda, cero si coinciden, valor positivo en caso contrario.
