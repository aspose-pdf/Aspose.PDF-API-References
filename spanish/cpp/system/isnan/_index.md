## System::IsNaN method


Determina si el valor especificado es un valor Not-A-Number.

```cpp
template<typename T> bool System::IsNaN(const T &value)
```


| Parámetro | Descripción |
| --- | --- |
| T | El tipo del valor verificado por la función |

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | const T\& | El valor a verificar |

### ReturnValue

Verdadero si **value** es un valor NaN, de lo contrario - falso
