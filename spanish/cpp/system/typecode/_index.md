## TypeCode enum


Representa el tipo de un objeto.

```cpp
enum class TypeCode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Empty | 0 | Una referencia nula. |
| Object | 1 | Un tipo general que representa cualquier tipo de referencia o valor que no esté representado explícitamente por otro [TypeCode](./). |
| DBNull | 2 | Un valor nulo de base de datos (columna). |
| Boolean | 3 | Un tipo simple que representa valores [Boolean](../boolean/) de verdadero o falso. |
| Char | 4 | Un tipo integral que representa enteros sin signo de 16 bits con valores entre 0 y 65535. |
| SByte | 5 | Un tipo integral que representa enteros con signo de 8 bits con valores entre -128 y 127. |
| Byte | 6 | Un tipo integral que representa enteros sin signo de 8 bits con valores entre 0 y 255. |
| Int16 | 7 | Un tipo integral que representa enteros con signo de 16 bits con valores entre -32768 y 32767. |
| UInt16 | 8 | Un tipo integral que representa enteros sin signo de 16 bits con valores entre 0 y 65535. |
| Int32 | 9 | Un tipo integral que representa enteros con signo de 32 bits con valores entre -2147483648 y 2147483647. |
| UInt32 | 10 | Un tipo integral que representa enteros sin signo de 32 bits con valores entre 0 y 4294967295. |
| Int64 | 11 | Un tipo integral que representa enteros con signo de 64 bits con valores entre -9223372036854775808 y 9223372036854775807. |
| UInt64 | 12 | Un tipo integral que representa enteros sin signo de 64 bits con valores entre 0 y 18446744073709551615. |
| Single | 13 | Un tipo de punto flotante que representa valores que van aproximadamente de 1.5 x 10 -45 a 3.4 x 10 38 con una precisión de 7 dígitos. |
| Double | 14 | Un tipo de punto flotante que representa valores que van aproximadamente de 5.0 x 10 -324 a 1.7 x 10 308 con una precisión de 15-16 dígitos. |
| Decimal | 15 | Un tipo simple que representa valores que van de 1.0 x 10 -28 a aproximadamente 7.9 x 10 28 con 28-29 dígitos significativos. |
| DateTime | 16 | Un tipo que representa un valor de fecha y hora. |
| String | 18 | Un tipo de clase sellada que representa cadenas de caracteres Unicode. |
