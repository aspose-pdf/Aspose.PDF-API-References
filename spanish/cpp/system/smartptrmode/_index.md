## SmartPtrMode enum


[SmartPtr](../smartptr/) pointer type: weak or shared. Defines whether pointer is being counted when it is being decided whether to delete object or not.

```cpp
enum class SmartPtrMode : char
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Shared | 0 | Modo compartido: el puntero participa en el recuento de referencias. |
| Weak | 1 | Modo débil: el puntero no participa en el recuento de referencias. |
