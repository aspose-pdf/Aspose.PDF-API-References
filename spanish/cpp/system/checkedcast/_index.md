## System::CheckedCast method


Determina si el valor especificado está dentro del rango de valores del tipo **TTo** y, de ser así, lo convierte al tipo **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parámetro | Descripción |
| --- | --- |
| TTo | El tipo al que se debe convertir el valor especificado |
| TFrom | El tipo del valor especificado |

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | TFrom | El valor a convertir |

### ReturnValue

El valor del tipo **TTo** equivalente a **value**
