## DateTime::SpecifyKind method


Construye un nuevo objeto [DateTime](../) que representa el mismo número de ticks que el objeto [DateTime](../) especificado y representa hora local, hora UTC o ninguna, según lo especificado por el argumento **kind**.

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| value | DateTime | El objeto [DateTime](../) del cual copiar el número de ticks |
| tipo | DateTimeKind | Especifica si el nuevo objeto debe representar hora local, hora UTC o ninguna. |

### ReturnValue

Un nuevo objeto [DateTime](../) que representa el mismo número de ticks que **value** y el valor [DateTimeKind](../../datetimekind/) especificado por **kind**.
