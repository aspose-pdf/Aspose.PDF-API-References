## DateTime::CompareTo method


Compara dos valores de fecha y hora representados por el objeto actual y la instancia especificada de la clase [DateTime](../) y devuelve el valor que indica las posiciones relativas de los valores en la línea de tiempo.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| value | DateTime | Una instancia de la clase [DateTime](../) con la que comparar el objeto actual. |

### ReturnValue

Un valor menor que 0 si el objeto actual representa un valor que es anterior al representado por **value**; 0 si los valores representados por ambos objetos son iguales; un valor mayor que 0 si el valor representado por el objeto actual es posterior al representado por **value**.
