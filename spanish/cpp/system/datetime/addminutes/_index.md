

| Parámetro | Tipo | Descripción |

| valor | double | El número de minutos a añadir; el valor puede ser positivo o negativo |

### ReturnValue

Una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora que es la suma del valor representado por el objeto actual y **value** minutos
