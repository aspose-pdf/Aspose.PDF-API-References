## DateTime::IsDaylightSavingTime method


Determina si el valor de fecha y hora representado por el objeto actual se encuentra dentro del rango de horario de verano para la zona horaria actual.

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### ReturnValue

True si el valor de fecha y hora especificado por el objeto actual se encuentra dentro del rango de horario de verano, de lo contrario - false
