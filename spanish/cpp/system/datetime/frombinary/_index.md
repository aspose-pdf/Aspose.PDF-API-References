## DateTime::FromBinary method


Deserializa el valor de fecha y hora a partir del entero sin signo de 64 bits especificado y establece la nueva instancia de la clase [DateTime](../) con ese valor.

```cpp
static DateTime System::DateTime::FromBinary(int64_t value)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | int64_t | El entero de 64 bits del cual deserializar el valor de fecha y hora. |

### ReturnValue

Una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora deserializado del entero sin signo de 64 bits especificado.
