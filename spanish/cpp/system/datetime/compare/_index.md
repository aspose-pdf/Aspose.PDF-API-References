## DateTime::Compare method


Compara dos valores representados por las instancias especificadas de la clase [DateTime](../) y devuelve el valor que indica las posiciones relativas de los valores en la línea de tiempo.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| t1 | DateTime | El primer comparando |
| t2 | DateTime | El segundo comparando |

### ReturnValue

Un valor menor que 0 si **t1** es anterior a **t2**; 0 si **t1** es igual a **t2**; un valor mayor que 0 si **t1** es posterior a **t2**
