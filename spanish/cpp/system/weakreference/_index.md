## WeakReference class


Representa una referencia débil, que referencia a un objeto mientras aún permite que ese objeto sea eliminado.

```cpp
template<typename...>class WeakReference
```


| Parámetro | Descripción |
| --- | --- |
| Args | Argumentos de plantilla de una referencia débil que declaran un tipo de objeto referenciado, el número de argumentos puede ser cero o uno. |
