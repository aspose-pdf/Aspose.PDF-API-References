## Uri::IsHexEncoding method


Determina si un carácter en la cadena especificada en la posición especificada está codificado en hexadecimal.

```cpp
static bool System::Uri::IsHexEncoding(const String &pattern, int32_t index)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| patrón | const String\& | La cadena a comprobar |
| índice | int32_t | Posición basada en cero del carácter a probar |

### ReturnValue

True si el carácter especificado está codificado en hexadecimal, de lo contrario - false
