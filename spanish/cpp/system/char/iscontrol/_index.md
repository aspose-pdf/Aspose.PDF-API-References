## Char::IsControl(char_t) method


Determina si el carácter especificado está clasificado como un carácter de control Unicode.

```cpp
static bool System::Char::IsControl(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un carácter de control Unicode, de lo contrario - false
