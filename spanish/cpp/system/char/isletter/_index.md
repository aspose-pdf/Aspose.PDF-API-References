## Char::IsLetter(char_t) method


Determina si el carácter especificado está clasificado como una letra Unicode.

```cpp
static bool System::Char::IsLetter(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

Verdadero si el carácter especificado es una letra Unicode, de lo contrario - falso
