## Char::IsWhiteSpace(char_t) method


Determina si el carácter especificado está clasificado como un carácter de espacio en blanco.

```cpp
static bool System::Char::IsWhiteSpace(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un carácter de espacio en blanco, de lo contrario - false
