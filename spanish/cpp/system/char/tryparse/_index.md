## Char::TryParse method


Intenta convertir una cadena que consiste en un solo carácter a un carácter UTF-16. La función tiene éxito solo cuando la cadena de entrada no es nula y tiene una longitud de exactamente un carácter.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) para convertir |
| resultado | char_t\& | La variable de salida que contendrá el resultado de la conversión si la conversión tiene éxito |

### ReturnValue

True si la conversión tuvo éxito, de lo contrario - false
