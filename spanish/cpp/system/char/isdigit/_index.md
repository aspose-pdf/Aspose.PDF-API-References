## Char::IsDigit(char_t) method


Determina si el carácter especificado está clasificado como un dígito decimal.

```cpp
static bool System::Char::IsDigit(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un dígito decimal, de lo contrario - false
