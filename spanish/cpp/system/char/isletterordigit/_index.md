## Char::IsLetterOrDigit(char_t) method


Determina si el carácter especificado está clasificado como letra Unicode o como dígito decimal.

```cpp
static bool System::Char::IsLetterOrDigit(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

Verdadero si el carácter especificado es una letra Unicode o un dígito decimal, de lo contrario - falso
