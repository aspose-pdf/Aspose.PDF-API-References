## Char::IsSurrogate(char_t) method


Determina si el carácter especificado es una unidad de código sustituto UTF-16.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | Un carácter |

### ReturnValue

Verdadero si el carácter especificado es una unidad de código sustituto UTF-16, de lo contrario - falso
