## Char::IsUpper(char_t) method


Determina si el carácter especificado está clasificado como una letra mayúscula.

```cpp
static bool System::Char::IsUpper(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es una letra mayúscula, de lo contrario - false
