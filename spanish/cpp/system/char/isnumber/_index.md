## Char::IsNumber(char_t) method


Determina si el carácter especificado está clasificado como un número.

```cpp
static bool System::Char::IsNumber(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

Verdadero si el carácter especificado es un número, de lo contrario - falso
