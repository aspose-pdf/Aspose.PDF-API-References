## Char::IsLowSurrogate(char_t) method


Determina si el carácter especificado es un surrogado bajo.

```cpp
static bool System::Char::IsLowSurrogate(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un surrogado bajo, de lo contrario - false
