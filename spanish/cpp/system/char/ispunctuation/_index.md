## Char::IsPunctuation(char_t) method


Determina si el carácter especificado está clasificado como un carácter de puntuación.

```cpp
static bool System::Char::IsPunctuation(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un carácter de puntuación, de lo contrario - false
