## Char::IsSymbol(char_t) method


Determina si el carácter especificado está clasificado como un carácter símbolo.

```cpp
static bool System::Char::IsSymbol(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un carácter símbolo, de lo contrario - false
