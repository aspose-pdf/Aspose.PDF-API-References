## Char::IsLower(char_t) method


Determina si el carácter especificado está clasificado como una letra minúscula.

```cpp
static bool System::Char::IsLower(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es una letra minúscula, de lo contrario - false
