## Boolean::Parse method


Convierte la cadena especificada a un valor del tipo bool.

```cpp
static bool System::Boolean::Parse(const String &value)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | const String\& | La cadena a convertir. |

### ReturnValue

La variable bool donde se coloca el resultado de la conversión; el resultado es true si la cadena especificada es igual a \"True\" y false si la cadena especificada es igual a \"False\".
