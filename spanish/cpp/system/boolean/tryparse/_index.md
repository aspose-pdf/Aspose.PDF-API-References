## Boolean::TryParse method


Convierte la cadena especificada a un valor del tipo bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | const String\& | La cadena a convertir. |
| resultado | bool\& | La referencia a una variable bool donde se coloca el resultado de la conversión; el resultado es true si la cadena especificada es igual a "True" y false si la cadena especificada es igual a "False". |

### ReturnValue

True si la cadena especificada es igual a "True" o "False", de lo contrario - false.
