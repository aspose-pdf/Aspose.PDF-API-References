## Array::crend method


Devuelve un iterador inverso al elemento que sigue al último elemento del contenedor invertido. Corresponde al elemento que precede al primer elemento del contenedor no invertido. Este elemento actúa como un marcador de posición; intentar acceder a él produce un comportamiento indefinido.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### ReturnValue

Un iterador que apunta al elemento teórico calificado como const que precede al primer elemento del contenedor.
