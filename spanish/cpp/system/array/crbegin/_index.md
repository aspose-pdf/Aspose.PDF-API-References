## Array::crbegin method


Devuelve un iterador inverso al primer elemento del contenedor invertido. Corresponde al último elemento del contenedor no invertido. Si el contenedor está vacío, el iterador devuelto es igual a [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

Un iterador que apunta al último elemento calificado como const del contenedor.
