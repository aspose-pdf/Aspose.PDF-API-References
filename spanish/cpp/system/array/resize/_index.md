## Array::Resize method


Cambia el tamaño del arreglo especificado al valor especificado o crea un nuevo arreglo con el tamaño especificado.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) para redimensionar. Si **arr** es un puntero nulo, se creará el nuevo array. |
| new_size | int | El nuevo tamaño del array, o el tamaño del nuevo array de **arr** es nulo |
