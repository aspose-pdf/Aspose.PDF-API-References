## Array::cend method


Devuelve un iterador al elemento que sigue al último elemento del contenedor. Este elemento actúa como un marcador de posición; intentar acceder a él produce un comportamiento indefinido.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### ReturnValue

Un iterador que apunta al elemento teóricamente const-qualificado situado después del último elemento del contenedor.
