## Array::cbegin method


Devuelve un iterador al primer elemento calificado como const del contenedor. Si el contenedor está vacío, el iterador devuelto será igual a [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Un iterador que apunta al primer elemento calificado como const del contenedor.
