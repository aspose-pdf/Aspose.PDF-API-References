## Object::Is method


Comprueba si el objeto representa una instancia del tipo descrito por targetType. Análogo al operador C# 'is'.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| targetType | const TypeInfo\& | estructura [TypeInfo](../../typeinfo/) que describe el tipo contra el que probar el objeto actual. |

### ReturnValue

Verdadero si el objeto es del tipo etiquetado o de su subclase, falso en caso contrario.
