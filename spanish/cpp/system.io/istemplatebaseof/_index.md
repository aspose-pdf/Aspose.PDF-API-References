## IsTemplateBaseOf typedef


Representa el equivalente de std::is_base_of<Base, Derived> que determina la herencia de una clase plantilla Base no instanciada a partir de una clase plantilla Derived instanciada. Fallará con herencia múltiple o herencia no pública de Base.

```cpp
using System::Details::IsTemplateBaseOf =  typename IsTemplateBaseOfImpl<Base, Derived>::type
```
