## Directory::Delete method


Elimina el archivo o directorio especificado. No lanza excepciones.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| ruta | const String\& | La ruta al directorio o archivo que se va a eliminar |
| recursivo | bool | Si **path** especifica un directorio no vacío, entonces **recursive** indica si todo el contenido del directorio debe eliminarse de forma recursiva; si el directorio especificado por **path** no está vacío y **recursive** es 'false', la operación falla. |
