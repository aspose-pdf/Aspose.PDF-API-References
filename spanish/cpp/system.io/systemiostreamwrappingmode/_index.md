## SystemIOStreamWrappingMode enum


Especifica el modo de operaciones de E/S que los wrappers realizarán sobre flujos similares a [System::IO::Stream](../stream/)-like.

```cpp
enum class SystemIOStreamWrappingMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Binario | 0 | Un modo que permite que las operaciones de entrada codifiquen los bytes del flujo en datos char_type y decodifiquen los datos char_type en bytes del flujo para operaciones de salida. |
| Conversión | 1 | Un modo que permite que las operaciones de entrada conviertan los bytes del flujo del tipo uint8_t al tipo char_type y viceversa para operaciones de salida. |
