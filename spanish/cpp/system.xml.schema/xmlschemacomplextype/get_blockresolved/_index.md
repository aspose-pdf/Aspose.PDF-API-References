## XmlSchemaComplexType::get_BlockResolved method


Devuelve el valor después de que el tipo haya sido compilado al conjunto de información posterior a la validación del esquema (infoset). Este valor indica cómo se aplica el tipo cuando se utiliza **xsi:type** en el documento de instancia.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

El valor del infoset posterior a la validación del esquema. El valor predeterminado es **BlockDefault** en el elemento **schema**.
