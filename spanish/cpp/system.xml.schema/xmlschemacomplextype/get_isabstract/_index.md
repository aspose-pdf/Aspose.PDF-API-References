## XmlSchemaComplexType::get_IsAbstract method


Devuelve la información que determina si el elemento **complexType** puede usarse en el documento de instancia.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Si **true**, un elemento no puede usar directamente este elemento **complexType** y debe usar un tipo complejo que derive de este elemento **complexType**. El valor predeterminado es **false**. Opcional.
