## XmlSchemaDerivationMethod enum

### Valores

| Nombre | Valor | Descripción |

| Empty | 0 | Sobrescribe el método de derivación predeterminado para permitir cualquier derivación. |
| Sustitución | 1 | Se refiere a derivaciones por **Substitution**. |
| Extensión | 2 | Se refiere a derivaciones por **Extension**. |
| Restricción | 4 | Se refiere a derivaciones por **Restriction**. |
| List | 8 | Se refiere a derivaciones por **List**. |
| Unión | 16 | Se refiere a derivaciones por **Union**. |
| Todo | 255 | **#all**. Se refiere a todos los métodos de derivación. |
| Ninguno | 256 | Acepta el método de derivación predeterminado. |
