## XmlSchemaFractionDigitsFacet class


Especifica una restricción en el número de dígitos que pueden ingresarse para el valor fraccionario de un elemento simpleType. El valor de fractionDigits debe ser un entero positivo. Representa la faceta **fractionDigits** del World Wide [Web](../../system.web/) Consortium (W3C).

class XmlSchemaFractionDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
