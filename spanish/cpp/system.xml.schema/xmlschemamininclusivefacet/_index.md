## XmlSchemaMinInclusiveFacet class


Representa el elemento **minInclusive** del XML [Schema](../) según lo especificado por el World Wide [Web](../../system.web/) Consortium (W3C). Esta clase puede usarse para especificar una restricción sobre el valor mínimo de un elemento simpleType. El valor del elemento debe ser mayor o igual al valor del elemento **minInclusive**.

```cpp
class XmlSchemaMinInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
