## XmlSchemaComplexContent::get_IsMixed method


Devuelve información que determina si el tipo tiene un modelo de contenido mixto.

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```


### ReturnValue

Si este valor es **true**, se permite que datos de carácter aparezcan entre los elementos hijos del tipo complejo (modelo de contenido mixto). El valor predeterminado es **false**. Opcional.
