## XmlSchemaEnumerationFacet class


Representa la faceta **enumeration** de XML [Schema](../) según lo especificado por el Consorcio World Wide [Web](../../system.web/). Esta clase especifica una lista de valores válidos para un elemento simpleType. La declaración está contenida dentro de una declaración **restriction**.

```cpp
class XmlSchemaEnumerationFacet : public System::Xml::Schema::XmlSchemaFacet
```
