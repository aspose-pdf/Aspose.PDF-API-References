## XmlSchemaTotalDigitsFacet class


Representa la faceta **totalDigits** de XML [Schema](../) según lo especificado por el Consorcio World Wide [Web](../../system.web/). Esta clase puede usarse para especificar una restricción sobre la cantidad de dígitos que pueden ingresarse para el valor de un elemento **simpleType**. Ese valor de **totalDigits** debe ser un entero positivo.

```cpp
class XmlSchemaTotalDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
