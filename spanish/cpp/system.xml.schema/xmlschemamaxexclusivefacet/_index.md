## XmlSchemaMaxExclusiveFacet class


Representa el elemento **maxExclusive** de XML [Schema](../) según lo especificado por el Consorcio World Wide [Web](../../system.web/). Esta clase puede usarse para especificar una restricción sobre el valor máximo de un elemento **simpleType**. El valor del elemento debe ser menor que el valor del elemento **maxExclusive**.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
