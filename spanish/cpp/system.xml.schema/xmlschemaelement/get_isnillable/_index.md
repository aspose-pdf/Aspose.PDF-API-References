## XmlSchemaElement::get_IsNillable method


Devuelve información que indica si **xsi:nil** puede aparecer en los datos de instancia. Indica si se puede asignar un valor nulo explícito al elemento.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Si nillable es **true**, esto permite que una instancia del elemento tenga el atributo **nil** establecido en **true**. El atributo **nil** está definido como parte del espacio de nombres XML [Schema](../../) para instancias. El valor predeterminado es **false**. Opcional.
