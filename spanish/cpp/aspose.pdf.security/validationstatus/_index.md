## ValidationStatus enum


Representa el estado de validación de una validación de certificado.

```cpp
enum class ValidationStatus
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Valid | 0 | Indica que el certificado ha sido validado con éxito. |
| Invalid | 1 | Indica que la validación del certificado falló. |
| Undefined | 2 | Indica que el proceso de validación fue inconcluso o no se realizó. |

## Observaciones


Esta enumeración define los posibles resultados de la validación de certificados:* Valid: Indica que el certificado ha sido validado con éxito.
* Invalid: Indicates that the certificate validation failed.
* Undefined: Indicates that the validation process was inconclusive or not performed.
