## SignatureAlgorithmType enum


Enumera los tipos de algoritmos de firma utilizados para firmas digitales.

```cpp
enum class SignatureAlgorithmType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Ecdsa | 0 | El Algoritmo de Firma Digital de Curva Elíptica (ECDSA) utilizado para firmas digitales. |
| Rsa | 1 | El algoritmo Rivest–Shamir–Adleman (RSA) utilizado para firmas digitales. |
| Dsa | 2 | El algoritmo Digital Signature Algorithm (DSA) utilizado para firmas digitales. |
| Timestamp | 3 | El Timestamp utilizado como algoritmo de firma para garantizar las firmas digitales. |
| Desconocido | 4 | Indica que el tipo de algoritmo de firma es desconocido. |
