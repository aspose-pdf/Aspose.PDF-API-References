## ValidationOptions::set_RequestTimeout method


Establece la duración del tiempo de espera, en milisegundos, para operaciones de red durante el proceso de validación. La propiedad RequestTimeout define el tiempo máximo que el sistema debe esperar una respuesta de red al acceder a recursos en línea, como el estado de revocación o servidores OCSP.
