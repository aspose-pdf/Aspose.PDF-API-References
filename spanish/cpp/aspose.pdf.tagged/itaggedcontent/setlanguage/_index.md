## ITaggedContent::SetLanguage method




```cpp
virtual void Aspose::Pdf::Tagged::ITaggedContent::SetLanguage(System::String lang)=0
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| lang | System::String | Un identificador de idioma debe ser la cadena de texto vacía, para indicar que el idioma es desconocido, o una etiqueta de idioma (Language-Tag) según lo definido en RFC 3066, Etiquetas para la identificación de idiomas. |
## Observaciones


Establece el idioma natural para el documento pdf.

Un identificador de idioma que especificará el lenguaje natural para todo el texto del documento, excepto cuando sea sobrescrito por especificaciones de idioma para elementos de estructura o contenido marcado.
