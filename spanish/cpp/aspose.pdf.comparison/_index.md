La selección de cambios implica añadir caracteres de espacio en blanco alrededor del formato, de lo contrario el visor markdown no mostrará correctamente el texto. Los saltos de línea eliminados se indican con el signo - de párrafo. |
| [OutputTextStyle](./outputtextstyle/) | Representa una clase de conjunto de estilos para marcar cambios de texto. |

| [SideBySideComparisonOptions](./sidebysidecomparisonoptions/) | Representa una clase de opciones para comparar documentos con salida lado a lado. |
| [SideBySideDocsComparisonResult](./sidebysidedocscomparisonresult/) | Representa la clase del resultado de una operación de comparación lado a lado realizada en dos documentos. |
| [SideBySidePagesComparisonResult](./sidebysidepagescomparisonresult/) | Representa la clase del resultado de una operación de comparación lado a lado realizada en dos páginas. |
