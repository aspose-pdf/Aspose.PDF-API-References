## TraceLevel enum


Especifica qué mensajes se deben generar para las clases [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) y System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Desactivado | 0 | No genera mensajes de rastreo ni de depuración. |
| Error | 1 | Genera mensajes de manejo de errores. |
| Warning | 2 | Genera mensajes de advertencia y de manejo de errores. |
| Información | 3 | Genera mensajes informativos, advertencias y de manejo de errores. |
| Detallado | 4 | Genera todos los mensajes de depuración y rastreo. |
