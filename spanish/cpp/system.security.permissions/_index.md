## Clases

| Clase | Descripción |
| --- | --- |
| [SecurityPermission](./securitypermission/) | Clase que describe el permiso de seguridad. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../system/makeobject/). Nunca cree una instancia de este tipo en la pila o usando el operador new, ya que resultará en errores de tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../system/smartptr/) y use este puntero para pasarlo a funciones como argumento. |
