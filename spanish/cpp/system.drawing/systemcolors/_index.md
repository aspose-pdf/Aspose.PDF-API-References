## SystemColors class


Clase que proporciona un conjunto de objetos [Color](../color/) precreados que representan los colores de los elementos de visualización de [Windows](../../system.windows/). Este es un tipo estático sin servicios de instancia. Nunca debe crear instancias de él por ningún medio.

```cpp
class SystemColors
```
