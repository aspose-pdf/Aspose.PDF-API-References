## SystemFonts class


Proporciona un conjunto de objetos [Font](../font/) precreados que representan fuentes usadas para mostrar texto en elementos de visualización de [Windows](../../system.windows/). Este es un tipo estático sin servicios de instancia. Nunca debe crear instancias de él por ningún medio.

```cpp
class SystemFonts : public System::Object
```
