## Size::Add method


Devuelve un nuevo objeto [Size](../) que es la suma del objeto [Size](../) especificado, es decir, cuyo valor de ancho es igual a la suma de los valores de ancho de los objetos especificados y cuyo valor de altura es igual a la suma de los valores de altura de los objetos especificados.

```cpp
static Size System::Drawing::Size::Add(const Size &size1, const Size &size2)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| size1 | const Size\& | El primer operando |
| size2 | const Size\& | El segundo operando |

### ReturnValue

El objeto [Size](../) que es la suma de **size1** y **size2**
