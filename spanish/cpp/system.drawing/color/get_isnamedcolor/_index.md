## Color::get_IsNamedColor method


Devuelve un valor que determina si la estructura [Color](../) representa un color con nombre o un miembro de la enumeración [KnownColor](../../knowncolor/).

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### ReturnValue

True si el [Color](../) se creó usando el método FromName(String) o el método [FromKnownColor(KnownColor)](../fromknowncolor/); de lo contrario, devuelve false.
