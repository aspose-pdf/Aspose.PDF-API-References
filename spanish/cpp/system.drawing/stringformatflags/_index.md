## StringFormatFlags enum


Especifica la información de visualización y diseño para las cadenas de texto.

```cpp
enum class StringFormatFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Dirección de texto de derecha a izquierda. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) está alineado verticalmente. |
| FitBlackBox | 4 | Las partes de los caracteres pueden sobresalir del cuadro de diseño. |
| DisplayFormatControl | 32 | Los caracteres de control se muestran. |
| NoFontFallback | 1024 | [Font](../font/) fallback está deshabilitado. |
| MeasureTrailingSpaces | 2048 | Inserta el espacio final al final de cada línea. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) wrapping está deshabilitado. |
| LineLimit | 8192 | Las líneas completas se disponen en el rectángulo de formato. |
| NoClip | 16384 | Se permite que se muestren las partes sobresalientes de los glifos. |
