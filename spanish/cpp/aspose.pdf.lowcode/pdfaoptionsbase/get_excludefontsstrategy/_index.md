## PdfAOptionsBase::get_ExcludeFontsStrategy method


Obtiene la estrategia para eliminar fuentes y minimizar el tamaño del archivo de salida durante el proceso de conversión PDF/A.

```cpp
PdfFormatConversionOptions::RemoveFontsStrategy Aspose::Pdf::LowCode::PdfAOptionsBase::get_ExcludeFontsStrategy()
```

## Observaciones


La estrategia para eliminar fuentes. Esto puede ser uno de los valores de la enumeración [PdfFormatConversionOptions::RemoveFontsStrategy](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/). El valor predeterminado es la combinación de [PdfFormatConversionOptions::RemoveFontsStrategy::SubsetFonts](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/) y [PdfFormatConversionOptions::RemoveFontsStrategy::RemoveDuplicatedFonts](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/).

Esta propiedad le permite controlar cómo se manejan las fuentes durante el proceso de conversión. Puede elegir eliminar fuentes duplicadas, eliminar fuentes similares con diferentes anchos, o crear subconjuntos de fuentes.
