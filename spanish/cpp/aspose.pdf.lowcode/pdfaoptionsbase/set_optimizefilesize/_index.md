## PdfAOptionsBase::set_OptimizeFileSize method


Establece un valor que indica si se debe intentar reducir el tamaño del archivo durante el proceso de conversión a PDF/A.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_OptimizeFileSize(bool value)
```

## Observaciones


**true**

si el tamaño del archivo debe reducirse tanto como sea posible; de lo contrario, **false**

.

Cuando se establece en **true**

, el proceso de conversión intentará minimizar el tamaño del archivo resultante. Esto podría afectar el rendimiento del proceso de conversión.
