## CompareInfo::Compare(const String\&, const String\&, CompareOptions) const method


Compara cadenas. Solo se admiten los modos Ordinal y OrdinalIgnoreCase.

```cpp
virtual int System::Globalization::CompareInfo::Compare(const String &a, const String &b, CompareOptions options) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| a | const String\& | Cadena LHS. |
| b | const String\& | Cadena RHS. |
| options | CompareOptions | [String](../../../system/string/) tipo de comparación. |

### ReturnValue

Valor negativo si la cadena LHS precede a la RHS, cero si coinciden, valor positivo en caso contrario.
