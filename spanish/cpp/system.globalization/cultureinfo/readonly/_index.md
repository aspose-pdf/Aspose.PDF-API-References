## CultureInfo::ReadOnly method


Obtiene una versión de solo lectura de la cultura.

```cpp
static CultureInfoPtr System::Globalization::CultureInfo::ReadOnly(const CultureInfoPtr &culture_info)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| culture_info | const CultureInfoPtr\& | Cultura para envolver como solo lectura. |

### ReturnValue

Si **culture_info** es de solo lectura, devuelve el mismo objeto; de lo contrario, crea una copia de solo lectura.
