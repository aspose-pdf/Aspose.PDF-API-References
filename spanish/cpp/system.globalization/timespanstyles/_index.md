## TimeSpanStyles enum


Especifica las opciones de formato que personalizan el análisis de cadenas para los métodos que convierten una representación de cadena de un intervalo de tiempo en un objeto Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Ninguno | 0 | El intervalo de tiempo representado por la cadena inptut se interpreta como un intervalo de tiempo negativo solo si está presente un signo negativo. |
| AssumeNegative | 1 | El intervalo de tiempo representado por la cadena inptut se interpreta siempre como un intervalo de tiempo negativo. |
