## SubPath class


Representa un objeto de gráficos vectoriales en la página. Básicamente, los objetos de gráficos vectoriales se representan mediante dos grupos de SubPaths. Uno de ellos se representa mediante un conjunto de líneas y curvas. Los demás se presentan como rectángulos y a veces pueden confundirse. Normalmente es un área rectangular que tiene color, pero con frecuencia este rectángulo se coloca al inicio de la página y define todo el espacio de la página en blanco. Así que obtienes el [SubPath](./), pero visualmente solo ves el texto en la página.

```cpp
class SubPath : public Aspose::Pdf::Vector::GraphicElement
```
