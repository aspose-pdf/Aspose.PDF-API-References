## BaseDictionary::begin method


Devuelve un iterador al KVPair-wrapper para el elemento clave-valor del contenedor. Implementado al estilo C# - el iterador debe devolver el objeto KVPair con la interfaz get_Key() y get_Value(). Si el contenedor está vacío, el iterador devuelto será igual a [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```


### ReturnValue

Un iterador que apunta al primer elemento de la colección.
