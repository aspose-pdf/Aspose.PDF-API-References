## MemoryMarshal class


Proporciona una implementación de marshaling de memoria. Solo para compatibilidad con el código traducido, ya que no se admite código administrado en el lado C++. Este es un tipo estático sin servicios de instancia. Nunca debe crear instancias de él por ningún medio.

```cpp

```
