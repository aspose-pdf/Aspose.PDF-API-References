## DashCap enum


Especifica el tipo de cap utilizado en ambos extremos de una raya en una línea punteada.

```cpp
enum class DashCap
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Plano | 0 | Un extremo cuadrado que cuadratea ambos extremos de una raya. |
| Round | 2 | Un extremo circular que redondea ambos extremos de una raya. |
| Triángulo | 3 | Un extremo triangular que apunta en ambos extremos de una raya. |
