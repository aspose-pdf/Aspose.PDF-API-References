## WrapMode enum


Especifica cómo se repite una textura o un degradado cuando es más pequeña que el área a rellenar.

```cpp
enum class WrapMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Tile | 0 | Repite el degradado o la textura. |
| TileFlipX | 1 | Invierte la textura o el degradado horizontalmente y luego repite la textura o el degradado. |
| TileFlipY | 2 | Invierte la textura o el degradado verticalmente y luego repite la textura o el degradado. |
| TileFlipXY | 3 | Invierte la textura o el degradado horizontal y verticalmente y luego repite la textura o el degradado. |
| Clamp | 4 | Fija la textura o el degradado al límite del objeto. |
