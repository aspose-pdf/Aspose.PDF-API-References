## LineCap enum


Indica los estilos de extremo disponibles con los que un objeto [Pen](../../system.drawing/pen/) puede terminar una línea.

```cpp
enum class LineCap
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Plano | 0 | Extremo de línea plano. |
| Square | 1 | Extremo de línea cuadrado. |
| Round | 2 | Extremo de línea redondo. |
| Triángulo | 3 | Extremo de línea triangular. |
| NoAnchor | 16 | Sin ancla. |
| SquareAnchor | 17 | Extremo de línea con ancla cuadrada. |
| RoundAnchor | 18 | Extremo de línea con ancla redonda. |
| DiamondAnchor | 19 | Extremo de línea con ancla de diamante. |
| ArrowAnchor | 20 | Extremo de línea con ancla de flecha. |
| AnchorMask | 240 | Una máscara utilizada para comprobar si un extremo de línea es un extremo de ancla. |
| Personalizado | 255 | Extremo personalizado. |
