## GraphicsPath::AddArc(const Rectangle\&, float, float) method


Agrega el arco elíptico especificado a la ruta representada por el objeto actual.

```cpp
void System::Drawing::Drawing2D::GraphicsPath::AddArc(const Rectangle &rect, float startAngle, float sweepAngle)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| rect | const Rectangle\& | El rectángulo que delimita la elipse de la cual se dibuja el arco |
| startAngle | float | Especifica el ángulo inicial del arco en grados, medido en sentido horario desde el eje X |
| sweepAngle | float | Especifica el ángulo entre el ángulo inicial y el final del arco |
