## SoapParameterStyle enum


Enumera los formatos de parámetros en un mensaje SOAP.

```cpp
enum class SoapParameterStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | 0 | Si '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' no se aplica a la clase, entonces el valor predeterminado es 'Wrapped'. |
| Bare | 1 | Los parámetros se colocan en elementos XML que siguen al elemento 'Body'. |
| Wrapped | 2 | Los parámetros están encapsulados dentro de un único elemento XML que sigue al elemento 'Body'. |
