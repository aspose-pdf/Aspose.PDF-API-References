## XmlReader::ReadString method


Cuando se sobrescribe en una clase derivada, lee el contenido de un elemento o nodo de texto como una cadena. Sin embargo, se recomienda usar el método [XmlReader::ReadElementContentAsString](../readelementcontentasstring/) en su lugar, porque ofrece una forma más directa de manejar esta operación.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```


### ReturnValue

El contenido del elemento o una cadena vacía.
