## XmlReader::ReadOuterXml method


Cuando se sobrescribe en una clase derivada, lee el contenido, incluido el marcado, que representa este nodo y todos sus hijos.

```cpp
virtual String System::Xml::XmlReader::ReadOuterXml()
```


### ReturnValue

Si el lector está posicionado en un nodo de elemento o de atributo, este método devuelve todo el contenido XML, incluido el marcado, del nodo actual y todos sus hijos; de lo contrario, devuelve una cadena vacía.
