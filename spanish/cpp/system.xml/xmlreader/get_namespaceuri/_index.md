## XmlReader::get_NamespaceURI method


Cuando se sobrescribe en una clase derivada, obtiene el URI del espacio de nombres (según lo definido en la especificación de espacios de nombres de W3C) del nodo en el que está posicionado el lector.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```


### ReturnValue

El URI de espacio de nombres del nodo actual; de lo contrario, una cadena vacía.
