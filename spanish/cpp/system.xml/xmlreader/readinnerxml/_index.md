## XmlReader::ReadInnerXml method


Cuando se sobrescribe en una clase derivada, lee todo el contenido, incluido el marcado, como una cadena.

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```


### ReturnValue

Todo el contenido XML, incluido el marcado, en el nodo actual. Si el nodo actual no tiene hijos, se devuelve una cadena vacía. Si el nodo actual no es ni un elemento ni un atributo, se devuelve una cadena vacía.
