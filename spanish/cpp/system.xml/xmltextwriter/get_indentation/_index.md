## XmlTextWriter::get_Indentation method


Devuelve cuántos IndentChars escribir para cada nivel en la jerarquía cuando [XmlTextWriter::set_Formatting](../set_formatting/) está configurado a [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```


### ReturnValue

Número de **IndentChars** para cada nivel. El valor predeterminado es 2.
