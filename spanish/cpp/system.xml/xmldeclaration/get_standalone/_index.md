## XmlDeclaration::get_Standalone method


Devuelve el valor del atributo standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Los valores válidos son **yes** si todas las declaraciones de entidad requeridas por el documento XML están contenidas dentro del documento o **no** si se requiere una definición de tipo de documento externa (DTD). Si el atributo standalone no está presente en la declaración XML, este método devuelve [String::Empty](../../../system/string/empty/).
