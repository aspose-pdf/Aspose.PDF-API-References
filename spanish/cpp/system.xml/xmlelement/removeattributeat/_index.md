## XmlElement::RemoveAttributeAt method


Elimina el nodo de atributo con el índice especificado del elemento. (Si el atributo eliminado tiene un valor predeterminado, se reemplaza inmediatamente).

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlElement::RemoveAttributeAt(int32_t i)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| i | int32_t | El índice del nodo a eliminar. El primer nodo tiene índice 0. |

### ReturnValue

El nodo de atributo eliminado o **nullptr** si no hay nodo en el índice dado.
