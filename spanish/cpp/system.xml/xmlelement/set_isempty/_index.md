## XmlElement::set_IsEmpty method


Establece el formato de etiqueta del elemento.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| valor | bool | El valor a establecer. Al establecer este valor, si se establece en **true**, los hijos del elemento se eliminan y el elemento se serializa en el formato de etiqueta corta. Si se establece en **false**, el valor se cambia (independientemente de si el elemento tiene contenido o no); si el elemento está vacío, se serializa en el formato largo. |
