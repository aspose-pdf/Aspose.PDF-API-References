Devuelve un objeto de tipo de esquema.

```cpp
SharedPtr<Object> System::Xml::XmlValidatingReader::get_SchemaType()
```

### ReturnValue

XmlSchemaDatatype, XmlSchemaSimpleType o XmlSchemaComplexType dependiendo de si el valor del nodo es un tipo de lenguaje de definición (XSD) XML [Schema](../../../system.xml.schema/) incorporado o un simpleType o complexType definido por el usuario; **nullptr** si el nodo actual no tiene tipo de esquema.
