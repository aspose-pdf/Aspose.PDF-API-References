## XmlReaderSettings::set_MaxCharactersInDocument method


Establece un valor que indica el número máximo permitido de caracteres en un documento XML. Un valor cero (0) significa que no hay límites en el tamaño del documento XML. Un valor distinto de cero especifica el tamaño máximo, en caracteres.

```cpp
void System::Xml::XmlReaderSettings::set_MaxCharactersInDocument(int64_t value)
```
