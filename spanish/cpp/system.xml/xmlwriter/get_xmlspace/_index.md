## XmlWriter::get_XmlSpace method


Cuando se sobrescribe en una clase derivada, obtiene un [XmlSpace](../../xmlspace/) que representa el alcance actual de **xml:space**.

virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()

### ReturnValue

Un [XmlSpace](../../xmlspace/) que representa el alcance actual de **xml:space**.

|||
|-|-|
| Valor | Significado |
| Ninguno | Este es el valor predeterminado si no existe ningún ámbito xml:space. |
| Predeterminado | El ámbito actual es xml:space=\"default\". |
| Preservar | El ámbito actual es xml:space=\"preserve\". |
