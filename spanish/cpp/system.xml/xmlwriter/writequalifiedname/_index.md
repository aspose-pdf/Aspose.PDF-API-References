## XmlWriter::WriteQualifiedName method


Cuando se sobrescribe en una clase derivada, escribe el nombre calificado con espacio de nombres. Este método busca el prefijo que está en alcance para el espacio de nombres dado.



| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| localName | const String\& | El nombre local a escribir. |
| ns | const String\& | El URI del espacio de nombres para el nombre. |
