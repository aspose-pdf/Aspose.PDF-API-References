## XmlWriterSettings::get_CheckCharacters method


Devuelve un valor que indica si el escritor XML debe comprobar que todos los caracteres del documento cumplen con la sección "2.2 Characters" de la [Recomendación XML 1.0](https://www.w3.org/TR/REC-xml/#charsets) de la W3C.

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```


### ReturnValue

**true** to do character checking; otherwise, **false**. The default is **true**.
