## XmlWriterSettings::get_NewLineChars method


Devuelve la cadena de caracteres a usar para los saltos de línea.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

La cadena de caracteres a usar para los saltos de línea. Esto puede establecerse a cualquier valor de cadena. Sin embargo, para garantizar XML válido, debe especificar solo caracteres de espacio en blanco válidos, como caracteres de espacio, tabulaciones, retornos de carro o saltos de línea. El valor predeterminado es **\\r\\n** (retorno de carro, nueva línea).
