## XmlOutputMethod enum


Especifica el método utilizado para serializar la salida del [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Xml | 0 | Serializa según las reglas XML 1.0. |
| Html | 1 | Serializa según las reglas HTML especificadas por XSLT. |
| Text | 2 | Serializa solo bloques de texto. |
| AutoDetect | 3 | Utiliza las reglas XSLT para elegir entre los métodos de salida [XmlOutputMethod::Xml](./) y [XmlOutputMethod::Html](./) en tiempo de ejecución. |
