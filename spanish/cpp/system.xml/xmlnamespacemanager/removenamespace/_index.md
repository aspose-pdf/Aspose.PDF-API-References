## XmlNamespaceManager::RemoveNamespace method


Elimina el espacio de nombres dado para el prefijo especificado.

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| prefijo | String | El prefijo del espacio de nombres. |
| uri | String | El espacio de nombres a eliminar para el prefijo dado. El espacio de nombres eliminado proviene del alcance de espacio de nombres actual. Los espacios de nombres fuera del alcance actual se ignoran. |
