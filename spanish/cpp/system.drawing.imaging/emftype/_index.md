## EmfType enum


Especifica los tipos de los registros que se colocan en un archivo EMF.

```cpp
enum class EmfType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| EmfOnly | n/a | Metarchivo mejorado de [Windows](../../system.windows/). Contiene comandos GDI. Los metarchivos de este tipo se denominan archivo EMF. |
| EmfPlusOnly | n/a | Metarchivo mejorado de [Windows](../../system.windows/) plus. Contiene comandos GDI+. Los metarchivos de este tipo se denominan archivo EMF+. |
| EmfPlusDual | n/a | Metarchivo mejorado dual de [Windows](../../system.windows/). Contiene comandos equivalentes GDI y GDI+. Los metarchivos de este tipo se denominan archivo EMF+. |
