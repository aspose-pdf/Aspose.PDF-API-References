## ImageLockMode enum


Especifica las propiedades de la región de una imagen que está bloqueada.

```cpp
enum class ImageLockMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| ReadOnly | 1 | La región está bloqueada solo para lectura. |
| WriteOnly | 2 | La región está bloqueada solo para escritura. |
| ReadWrite | n/a | La región está bloqueada tanto para lectura como para escritura. |
| UserInputBuffer | 4 | La imagen reside en un búfer de entrada del usuario, cuyo acceso está controlado por el usuario. |
