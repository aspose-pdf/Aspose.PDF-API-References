## TextAlign_Justify field


Alineación de texto del atributo: Justificar - Alineado con ambos bordes, inicial y final, con el espaciado interno dentro de cada línea expandido, si es necesario, para lograr dicha alineación. La última (o única) línea se alineará solo con el borde inicial.
