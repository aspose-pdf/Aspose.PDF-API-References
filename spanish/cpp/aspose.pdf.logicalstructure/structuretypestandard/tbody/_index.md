## TBody field


([Table](../../../aspose.pdf/table/) grupo de filas del cuerpo de la tabla; PDF 1.5) Un conjunto de filas que constituyen la parte principal del cuerpo de una tabla. Si la tabla se divide en varias páginas, el área del cuerpo puede separarse en el límite de una fila. Una tabla puede tener múltiples elementos TBody para permitir el dibujo de un borde o fondo para un conjunto de filas.
