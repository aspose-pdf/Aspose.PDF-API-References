## Observaciones


(Formula) Una fórmula matemática.

Este tipo de estructura es útil solo para identificar un elemento de contenido completo como una fórmula. No se definen tipos de estructura estándar para identificar componentes individuales dentro de la fórmula. Desde el punto de vista del formato, la fórmula deberá tratarse de manera similar a una figura (tipo de estructura Figure).
