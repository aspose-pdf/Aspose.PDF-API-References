## Table field


([Tabla](../../../aspose.pdf/table/)) Un diseño bidimensional de celdas de datos rectangulares, que posiblemente tenga una subestructura compleja. Contiene una o más filas de tabla (tipo de estructura TR) como hijos; o un encabezado de tabla opcional (tipo de estructura THead) seguido de una o más elementos del cuerpo de tabla (tipo de estructura TBody) y un pie de tabla opcional (tipo de estructura TFoot). Además, una tabla puede tener un título (tipo de estructura Caption) como su primer o último hijo.
