## XmlKnownDtds enum


La enumeración [Resolvers::XmlKnownDtds](./) es utilizada por el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) y define qué DTDs bien conocidos reconoce el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/).

```cpp
enum class XmlKnownDtds
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | 0 | Especifica que el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) no reconocerá ninguno de los DTDs predefinidos. |
| Xhtml10 | 1 | Especifica que el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconocerá DTDs y entidades que están definidas en XHTML 1.0. |
| Rss091 | 2 | Especifica que el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconocerá DTDs y entidades que están definidas en RSS 0.91. |
| All | 65535 | Especifica que el [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconocerá todos los DTDs actualmente soportados. Este es el comportamiento predeterminado. |
