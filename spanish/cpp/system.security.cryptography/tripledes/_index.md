## TripleDES class


Clase base del algoritmo Triple [Data](../../system.data/) Encryption Standard. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/) function. Nunca cree una instancia de este tipo en la pila o usando el operador new, ya que resultará en errores de tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use este puntero para pasarlo a funciones como argumento.

```cpp
class TripleDES : public System::Security::Cryptography::SymmetricAlgorithm
```
