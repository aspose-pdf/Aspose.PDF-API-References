## SHA384 class


Calcula el hash [SHA384](./) para los datos de entrada. En C++ [SHA384](./) no es una clase abstracta. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/). Nunca cree una instancia de este tipo en la pila o usando el operador new, ya que resultará en errores de tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use este puntero para pasarla a funciones como argumento.

```cpp
class SHA384 : public System::Security::Cryptography::HashAlgorithm
```
