## AnnotationState enum


La enumeración de estados a los que se puede establecer la anotación original.

```cpp
enum class AnnotationState
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Undefined | 0 | Estado no definido. |
| Marcado | 1 | La anotación ha sido marcada por el usuario. |
| Desmarcado | 2 | La anotación no ha sido marcada por el usuario. |
| Accepted | 3 | El usuario está de acuerdo con el cambio. |
| Rechazado | 4 | El usuario no está de acuerdo con el cambio. |
| Cancelado | 5 | El cambio ha sido cancelado. |
| Completado | 6 | El cambio ha sido completado. |
| Ninguno | 7 | El usuario no ha indicado nada sobre el cambio. |
