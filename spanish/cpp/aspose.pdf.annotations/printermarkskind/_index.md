## PrinterMarksKind enum


Especifica los tipos de marcas de impresora que se añadirán a un documento.

```cpp
enum class PrinterMarksKind
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Ninguno | 0 | Especifica que no se deben agregar marcas de la impresora. |
| TrimMarks | n/a | Especifica que se deben agregar marcas de recorte. |
| BleedMarks | n/a | Especifica que se deben agregar marcas de sangrado. |
| RegistrationMarks | n/a | Especifica que se deben agregar marcas de registro. |
| ColorBars | n/a | Especifica que se deben agregar barras de color. |
| PageInformation | n/a | Especifica que se debe agregar información de página. |
| Todo | n/a | Especifica que se deben agregar todos los tipos de marcas de impresora. |

## Observaciones


Esta enumeración tiene un atributo [FlagsAttribute](../) que permite una combinación bit a bit de los valores de sus miembros.
