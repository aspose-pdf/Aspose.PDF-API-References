## HostInformation class


Representa información sobre un host. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/) function. Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que resultará en errores de tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) pointer y use este puntero para pasarlo a funciones como argumento.

```cpp
class HostInformation : public System::Object
```
