## PdfExtractor::get_ExtractImageMode method


Establece el modo para el proceso de extracción de imágenes.

```cpp
Aspose::Pdf::ExtractImageMode Aspose::Pdf::Facades::PdfExtractor::get_ExtractImageMode() const
```

## Observaciones


El valor predeterminado es [ExtractImageMode.DefinedInResources](../../../aspose.pdf/extractimagemode/) que extrae todas las imágenes definidas en los recursos.

Para extraer las imágenes realmente mostradas, se debe usar el modo [ExtractImageMode.ActuallyUsed](../../../aspose.pdf/extractimagemode/).
