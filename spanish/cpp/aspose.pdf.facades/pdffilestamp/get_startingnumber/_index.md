## PdfFileStamp::get_StartingNumber method


Obtiene el número inicial para la primera página en el archivo de entrada. Las páginas siguientes se numerarán a partir de este valor. Por ejemplo, si StartingNumber se establece en 100, las páginas del documento tendrán los números 100, 101, 102...
