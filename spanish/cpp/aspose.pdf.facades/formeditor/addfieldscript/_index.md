## FormEditor::AddFieldScript method


Agregar JavaScript a un campo PushButton. Si existe un evento antiguo, el nuevo evento se agrega después de él.

```cpp
bool Aspose::Pdf::Facades::FormEditor::AddFieldScript(const System::String &fieldName, const System::String &script)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fieldName | const System::String\& | El nombre de campo totalmente calificado. |
| script | const System::String\& | El script Java que se añadirá/colocará en un campo de botón push. |

### ReturnValue

Verdadero en caso de que el script se haya añadido con éxito.
