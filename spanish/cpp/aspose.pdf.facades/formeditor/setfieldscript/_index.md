## FormEditor::SetFieldScript method


Establece JavaScript para un campo PushButton. Si existía JavaScript anterior, será reemplazado por el nuevo.

```cpp
bool Aspose::Pdf::Facades::FormEditor::SetFieldScript(const System::String &fieldName, const System::String &script)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fieldName | const System::String\& | El nombre de campo totalmente calificado. |
| script | const System::String\& | El script Java que se añadirá/colocará en un campo de botón push. |

### ReturnValue

true si el script del campo se estableció correctamente.
