## FormEditor::SetFieldCombNumber method


Establece el número de divisiones (combs) para un campo de texto de una sola línea regular (el campo se divide automáticamente en tantas posiciones equidistantes, o combs, como indica el parámetro combNumber).

```cpp
bool Aspose::Pdf::Facades::FormEditor::SetFieldCombNumber(const System::String &fieldName, int32_t combNumber)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| fieldName | const System::String\& | El nombre de campo calificado. |
| combNumber | int32_t | El número de combs en los que dividir el campo. |

### ReturnValue

Si tiene éxito, devuelve true; de lo contrario, false.
