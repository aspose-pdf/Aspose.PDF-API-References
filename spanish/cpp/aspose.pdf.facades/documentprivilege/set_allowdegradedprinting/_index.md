## DocumentPrivilege::set_AllowDegradedPrinting method


Establece el permiso que permite la impresión degradada o no. true permite y false está prohibido.

```cpp
void Aspose::Pdf::Facades::DocumentPrivilege::set_AllowDegradedPrinting(bool value)
```

## Observaciones


Cuando se establece, la impresión se limitará a una representación de bajo nivel de la apariencia, posiblemente de calidad degradada.
