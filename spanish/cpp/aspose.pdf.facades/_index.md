El espacio de nombres **[Aspose.Pdf.Facades](./)** proporciona clases que originalmente provienen de Aspose.Pdf.Kit. Estas clases se utilizan para manipular documentos realizando operaciones como concatenar, estampar, firmar, anotar, etc., pero a un nivel alto sin acceso a la estructura interna de un documento.

Tiene dos modos de entrada de archivo de plantilla: entrada como flujo o como archivo PDF. Tiene cuatro tipos de modos de salida: un flujo combinado, un archivo combinado, muchos flujos pequeños, muchos archivos pequeños. Puede recibir datos literales contenidos en una [System.Data.DataTable](../system.data/datatable/). |
