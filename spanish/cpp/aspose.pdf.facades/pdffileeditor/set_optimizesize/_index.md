## PdfFileEditor::set_OptimizeSize method


Establece la bandera de optimización. Los flujos de recursos iguales en el archivo resultante se fusionan en un solo objeto PDF si esta bandera está activada. Esto permite reducir el tamaño del archivo resultante pero puede causar una ejecución más lenta y mayores requisitos de memoria. Valor predeterminado: false.
