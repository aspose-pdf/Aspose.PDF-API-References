## PdfFileEditor::get_CorruptedItems method


Matriz de problemas encontrados cuando se realizó la concatenación. Por cada documento corrupto pasado a la función [Concatenate()](../concatenate/) se crea una nueva entrada [CorruptedItem](../corrupteditem/). Esta propiedad solo puede usarse cuando CorruptedFileAction es ConcatenateIgnoringCorrupted.
