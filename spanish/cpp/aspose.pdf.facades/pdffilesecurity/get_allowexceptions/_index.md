## PdfFileSecurity::get_AllowExceptions method


Si este valor se establece en true, se lanzará una excepción al fallar la operación. De lo contrario, el método devuelve false en caso de error y la última excepción puede verificarse con la propiedad LastException.

```cpp
bool Aspose::Pdf::Facades::PdfFileSecurity::get_AllowExceptions()
```


## Deprecated
