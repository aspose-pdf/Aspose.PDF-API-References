## Symbology enum


Una (código de barras) [Symbology](./) define los detalles técnicos de un tipo particular de código de barras: el ancho de las barras, el conjunto de caracteres, el método de codificación, las especificaciones de suma de verificación, etc.

```cpp
enum class Symbology
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PDF417 | 0 | Una simbología 2D de varias filas y longitud variable con alta capacidad de datos y capacidad de corrección de errores. |
| QRCode | 1 | El código QR (abreviatura de Quick Response) es un código de barras matricial específico (código 2D), legible por lectores de códigos QR dedicados y teléfonos con cámara. |
| DataMatrix | 2 | Una simbología matricial 2D que consiste en módulos cuadrados en blanco y negro dispuestos en un patrón cuadrado o rectangular. |
