## PKCS7 class


Representa el objeto PKCS#7 que se ajusta a la especificación PKCS#7 en el RFC 2315 de Internet, PKCS #7: Sintaxis de Mensaje Criptográfico, Versión 1.5. El **SHA1 digest** del rango de bytes del documento está encapsulado en el campo SignedData de PKCS#7.

class PKCS7 : public Aspose::Pdf::Forms::Signature
