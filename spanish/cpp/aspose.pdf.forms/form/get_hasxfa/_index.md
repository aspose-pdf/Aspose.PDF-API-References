## Form::get_HasXfa method


Obtiene un valor que indica si el documento contiene el formulario [XFA](../../xfa/). Esta propiedad se introdujo para determinar si se debe usar [IgnoreNeedsRendering](../) para eliminar el formulario [XFA](../../xfa/) en casos donde el formulario [XFA](../../xfa/) está presente y [NeedsRendering](../) es falso.
