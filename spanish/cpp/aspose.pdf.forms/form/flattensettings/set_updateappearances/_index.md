## FlattenSettings::set_UpdateAppearances method


Si está activado, todas las apariencias de los campos se regenerarán antes del aplanado. Esta opción puede ayudar si un campo se aplanó incorrectamente. Esta opción puede disminuir el rendimiento. Por defecto está desactivada.
