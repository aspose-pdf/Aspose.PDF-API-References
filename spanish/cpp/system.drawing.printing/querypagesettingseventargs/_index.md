## QueryPageSettingsEventArgs class


Proporciona datos para el evento QueryPageSettings. Los objetos de esta clase solo deben asignarse mediante la función [System::MakeObject()](../../system/makeobject/). Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que provocará errores en tiempo de ejecución y/o fallos de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use ese puntero para pasarlo a funciones como argumento.

class QueryPageSettingsEventArgs : private System::Drawing::Printing::PrintEventArgs
