## Timer::set_AutoReset method


Establece el temporizador en modo de reinicio automático o lo desactiva.

```cpp
void System::Timers::Timer::set_AutoReset(bool autoreset)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| autoreset | bool | true para hacer que el temporizador se invoque en un bucle, false para detenerlo después de la primera invocación. |
