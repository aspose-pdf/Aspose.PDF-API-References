## SHA384 class


Beräknar [SHA384](./)-hash för indata. I C++ är [SHA384](./) inte en abstrakt klass. Objekt av denna klass bör endast allokeras med [System::MakeObject()](../../system/makeobject/)-funktionen. Skapa aldrig en instans av denna typ på stacken eller med operator new, eftersom det kan leda till körfel och/eller påståendefel. Omslut alltid denna klass i en [System::SmartPtr](../../system/smartptr/)-pekare och använd denna pekare för att skicka den till funktioner som argument.

```cpp
class SHA384 : public System::Security::Cryptography::HashAlgorithm
```
