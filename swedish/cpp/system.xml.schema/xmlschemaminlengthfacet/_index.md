## XmlSchemaMinLengthFacet class


Representerar **minLength**-elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på minsta längden för datavärdet i ett **simpleType**-element. Längden måste vara större än värdet för **minLength**-elementet.
