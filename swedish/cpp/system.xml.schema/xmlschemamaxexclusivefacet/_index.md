## XmlSchemaMaxExclusiveFacet class


Representerar **maxExclusive**-elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på det maximala värdet för ett **simpleType**-element. Elementvärdet måste vara mindre än värdet för **maxExclusive**-elementet.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
