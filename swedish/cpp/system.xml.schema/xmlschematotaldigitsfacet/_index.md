## XmlSchemaTotalDigitsFacet class


Representerar **totalDigits**-facetet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på antalet siffror som kan anges för värdet av ett **simpleType**-element. Det värdet för **totalDigits** måste vara ett positivt heltal.

```cpp
class XmlSchemaTotalDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
