## XmlSchemaMaxLengthFacet class


Representerar **maxLength**-elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på den maximala längden av datavärdet för ett **simpleType**-element. Längden måste vara mindre än värdet för **maxLength**-elementet.

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
