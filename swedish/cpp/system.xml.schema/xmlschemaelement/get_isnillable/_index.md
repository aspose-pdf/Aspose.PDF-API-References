## XmlSchemaElement::get_IsNillable method


Returnerar information som visar om **xsi:nil** kan förekomma i instansdata. Anger om ett explicit nil‑värde kan tilldelas elementet.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Om nillable är **true**, möjliggör detta att en instans av elementet får **nil**‑attributet satt till **true**. **nil**‑attributet är definierat som en del av XML‑[Schema](../../)‑namnutrymmet för instanser. Standardvärdet är **false**. Valfritt.
