## XmlSchemaMaxInclusiveFacet class


Representerar **maxInclusive**-elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på det maximala värdet för ett **simpleType**-element. Elementets värde måste vara mindre än eller lika med värdet av **maxInclusive**-elementet.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
