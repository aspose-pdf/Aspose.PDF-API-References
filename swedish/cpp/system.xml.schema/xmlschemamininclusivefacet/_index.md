## XmlSchemaMinInclusiveFacet class


Representerar **minInclusive**‑elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en begränsning på det minsta värdet för ett simpleType‑element. Elementvärdet måste vara större än eller lika med värdet för **minInclusive**‑elementet.
