## XmlReaderSettings::set_MaxCharactersInDocument method


Ställer in ett värde som anger det maximalt tillåtna antalet tecken i ett XML‑dokument. Ett nollvärde (0) betyder att det inte finns några begränsningar för XML‑dokumentets storlek. Ett icke‑nollvärde anger den maximala storleken i tecken.

void System::Xml::XmlReaderSettings::set_MaxCharactersInDocument(int64_t value)
