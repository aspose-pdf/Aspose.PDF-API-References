## XmlReaderSettings::get_MaxCharactersInDocument method


Returnerar ett värde som anger det maximalt tillåtna antalet tecken i ett XML-dokument. Ett nollvärde (0) betyder att det inte finns några begränsningar för XML-dokumentets storlek. Ett icke-nollvärde specificerar den maximala storleken i tecken.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

Det maximalt tillåtna antalet tecken i ett XML‑dokument. Standardvärdet är 0.
