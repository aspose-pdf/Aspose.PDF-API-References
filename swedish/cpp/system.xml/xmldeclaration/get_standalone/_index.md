## XmlDeclaration::get_Standalone method


Returnerar värdet för attributet standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Giltiga värden är **yes** om alla entitetsdeklarationer som krävs av XML-dokumentet finns i dokumentet eller **no** om en extern dokumenttypdefinition (DTD) krävs. Om ett standalone-attribut inte finns i XML-deklarationen returnerar denna metod [String::Empty](../../../system/string/empty/).
