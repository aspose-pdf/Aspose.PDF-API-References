## NamespaceHandling enum


Anger om du ska ta bort dubblettnamnrymdsdeklarationer i [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Standard | 0 | Anger att dubblettnamnrymdsdeklarationer inte kommer att tas bort. |
| OmitDuplicates | 1 | Anger att dubblettnamnrymdsdeklarationer kommer att tas bort. För att dubblettnamnrymden ska tas bort måste prefixet och namnrymden matcha. |
