## XmlTextWriter::get_Indentation method


Returnerar hur många IndentChars som ska skrivas för varje nivå i hierarkin när [XmlTextWriter::set_Formatting](../set_formatting/) är satt till [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```


### ReturnValue

Antal **IndentChars** för varje nivå. Standardvärdet är 2.
