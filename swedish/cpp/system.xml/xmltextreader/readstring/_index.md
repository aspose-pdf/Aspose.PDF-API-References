### ReturnValue

Innehållet i elementet eller textnoden. Detta kan vara en tom sträng om läsaren är placerad på något annat än ett element eller en textnod, eller om det inte finns mer textinnehåll att returnera i det aktuella sammanhanget. **Obs:** Textnoden kan vara antingen ett element eller en attributtextnod.
