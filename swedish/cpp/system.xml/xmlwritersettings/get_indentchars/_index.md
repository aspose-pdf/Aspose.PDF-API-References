## XmlWriterSettings::get_IndentChars method


Returnerar teckensträngen som ska användas vid indentering. Denna inställning används när värdet [XmlWriterSettings::set_Indent](../set_indent/) är satt till **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Teckensträngen som ska användas vid indentering. Den kan sättas till vilket strängvärde som helst. För att säkerställa giltig XML bör du dock endast ange giltiga blankstegstecken, såsom mellanslag, tabulatorer, vagnretur eller radmatning. Standardvärdet är två mellanslag.
