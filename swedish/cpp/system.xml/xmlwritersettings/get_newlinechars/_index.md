## XmlWriterSettings::get_NewLineChars method


Returnerar teckensträngen som ska användas för radbrytningar.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

Teckensträngen som ska användas för radbrytningar. Den kan sättas till vilket strängvärde som helst. Men för att säkerställa giltig XML bör du endast ange giltiga blankstegstecken, såsom mellanslag, tabbar, vagnretur eller radmatning. Standardvärdet är **\r\n** (vagnretur, ny rad).
