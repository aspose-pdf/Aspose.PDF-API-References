## XmlWriterSettings::get_NamespaceHandling method


Returnerar ett värde som indikerar om [XmlWriter](../../xmlwriter/) ska ta bort dubblettnamnrymdsdeklarationer när XML-innehåll skrivs. Standardbeteendet är att skrivaren skriver ut alla namnrymdsdeklarationer som finns i skrivarens namnrymdslösare.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

Den [NamespaceHandling](../../namespacehandling/)-enumerationen som används för att ange om dubblettnamnrymdsdeklarationer ska tas bort i [XmlWriter](../../xmlwriter/).
