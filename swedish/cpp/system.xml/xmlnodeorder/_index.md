## XmlNodeOrder enum


Beskriver dokumentordningen för en nod jämfört med en andra nod.

```cpp
enum class XmlNodeOrder
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Före | 0 | Den aktuella noden för denna navigator är före den aktuella noden för den angivna navigatorn. |
| Efter | 1 | Den aktuella noden för denna navigator är efter den aktuella noden för den angivna navigatorn. |
| Samma | 2 | De två navigatorerna är placerade på samma nod. |
| Okänd | 3 | Nodpositionerna kan inte bestämmas i dokumentordning i förhållande till varandra. Detta kan inträffa om de två noderna finns i olika träd. |
