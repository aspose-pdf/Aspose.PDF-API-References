## XmlKnownDtds enum


Den [Resolvers::XmlKnownDtds](./)‑uppräkningen används av [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) och definierar vilka välkända DTD‑er som [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) känner igen.

```cpp
enum class XmlKnownDtds
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | 0 | Anger att [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) inte kommer att känna igen någon av de fördefinierade DTD‑erna. |
| Xhtml10 | 1 | Anger att [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) kommer att känna igen DTD‑er och enheter som är definierade i XHTML 1.0. |
| Rss091 | 2 | Anger att [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) kommer att känna igen DTD‑er och enheter som är definierade i RSS 0.91. |
| All | 65535 | Anger att [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) kommer att känna igen alla för närvarande stödda DTD‑er. Detta är standardbeteendet. |
