## AbsorbedTable::CompareTo method


Jämför det aktuella [AbsorbedTable](../)-objektet med ett annat [AbsorbedTable](../)-objekt och returnerar ett heltal som indikerar om det aktuella objektet föregår, följer eller hamnar på samma position i sorteringsordningen som det andra objektet.

```cpp
int32_t Aspose::Pdf::Text::AbsorbedTable::CompareTo(System::SharedPtr<AbsorbedTable> other) override
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| other | System::SharedPtr\<AbsorbedTable\> | Ett [AbsorbedTable](../)-objekt att jämföra med detta objekt. |

### ReturnValue

Ett värde som indikerar den relativa ordningen för de objekt som jämförs. Returvärdet har följande betydelser: -1: Detta objekt föregår det andra objektet i sorteringsordningen. 0: Detta objekt ligger på samma position i sorteringsordningen som det andra objektet. 1: Detta objekt följer det andra objektet i sorteringsordningen.
