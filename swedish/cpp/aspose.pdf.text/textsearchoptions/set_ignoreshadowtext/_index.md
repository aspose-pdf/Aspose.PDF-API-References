## TextSearchOptions::set_IgnoreShadowText method


Ställer in indikation på att textfragment som representerar skugga av normal text kommer att ignoreras under sökning. true - betyder att skuggtext inte kommer att hittas (prova detta om textsökning returnerar duplicerade fragment på nära positioner) false - betyder att skuggtext kommer att hittas liksom normal text (standardvärde).
