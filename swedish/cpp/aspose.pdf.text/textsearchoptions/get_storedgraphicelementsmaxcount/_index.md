## TextSearchOptions::get_StoredGraphicElementsMaxCount method


Hämtar värde som begränsar sökning efter textrelaterad grafik (understrykning, bakgrund etc.) på en sida till det angivna antalet element. Standardvärdet är 250. Sätt ett lägre värde vid prestandaproblem, prova ett högre värde om vissa grafiska element inte hittades.
