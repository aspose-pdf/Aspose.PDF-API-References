## TextSearchOptions::set_UseFontEngineEncoding method


Ställer in indikation på att text kommer att sökas med teckensnittsmotorns kodning. true - betyder att teckensnittsmotorns kodning kommer att användas (prova detta om textsökning misslyckas på grund av bristfällig kodning i dokumentet) false - betyder att dokumentets teckensnittskodning kommer att användas (standardvärde).
