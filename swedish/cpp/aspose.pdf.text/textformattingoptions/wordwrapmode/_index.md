## WordWrapMode enum


Definierar strategier för radbrytning.

```cpp
enum class WordWrapMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| NoWrap | 0 | Ingen radbrytning utförs. |
| DiscretionaryHyphenation | 1 | Diskretionär avstavning utförs. Tillåter att ord bryts i mitten. |
| ByWords | 2 | Radbrytning omsluter endast hela ord. Om hela ordet inte kan brytas, försöker man använda diskretionär avstavning. |
| Undefined | 3 | [WordWrapMode](./) är inte inställd. Omslagsstrategier för överordnad struktur (tabellcell, stycke osv.) kommer att användas. |
