## FontSizeAdjustment enum


Anger en policy för hur teckensnittsstorleken på text ska justeras för att passa inom ett omgivande område.

```cpp
enum class FontSizeAdjustment
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Ingen | 0 | Teckenstorleken ändras inte. |
| ShrinkToFit | 1 | Teckenstorleken minskas om texten är för stor för att passa inom gränserna. Teckenstorleken ökas aldrig om texten är mindre än gränserna. |
| ScaleToFill | 2 | Teckenstorleken justeras (både minskning och ökning) för att få texten att fylla rektangelns gränser så mycket som möjligt. |
