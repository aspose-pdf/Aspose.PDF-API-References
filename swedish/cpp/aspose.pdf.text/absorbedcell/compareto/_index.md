## AbsorbedCell::CompareTo method


Jämför det aktuella [AbsorbedCell](../)‑objektet med ett annat [AbsorbedCell](../)‑objekt och returnerar ett heltal som indikerar om det aktuella objektet föregår, följer eller ligger på samma position i sorteringsordningen som det andra objektet.

```cpp
int32_t Aspose::Pdf::Text::AbsorbedCell::CompareTo(System::SharedPtr<AbsorbedCell> other) override
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| other | System::SharedPtr\<AbsorbedCell\> | Ett [AbsorbedCell](../)‑objekt att jämföra med detta objekt. |

### ReturnValue

Ett värde som indikerar den relativa ordningen för de objekt som jämförs. Returvärdet har följande betydelser: -1: Detta objekt föregår det andra objektet i sorteringsordningen. 0: Detta objekt ligger på samma position i sorteringsordningen som det andra objektet. 1: Detta objekt följer det andra objektet i sorteringsordningen.
