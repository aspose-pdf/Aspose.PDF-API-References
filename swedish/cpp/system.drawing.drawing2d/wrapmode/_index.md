## WrapMode enum


Anger hur en textur eller en gradient tileas när den är mindre än det område som fylls.

```cpp
enum class WrapMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Tile | 0 | Tilear gradienten eller texturen. |
| TileFlipX | 1 | Vänder texturen eller gradienten horisontellt och sedan upprepar texturen eller gradienten. |
| TileFlipY | 2 | Vänder texturen eller gradienten vertikalt och sedan upprepar den. |
| TileFlipXY | 3 | Vänder texturen eller gradienten horisontellt och vertikalt och sedan upprepar den. |
| Clamp | 4 | Fäster texturen eller gradienten vid objektets gräns. |
