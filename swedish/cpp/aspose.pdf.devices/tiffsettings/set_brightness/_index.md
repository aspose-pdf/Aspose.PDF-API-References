## TiffSettings::set_Brightness method


Hämtar eller anger ett värdegräns för färgtransformeringen mellan vitt och svart. Denna parameter kan tillämpas med EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle eller [ColorDepth.Format1bpp](../../colordepth/) == 1.

```cpp
void Aspose::Pdf::Devices::TiffSettings::set_Brightness(float value)
```

## Anmärkningar


Värdet för ljusstyrka bör ligga i intervallet från 0 till 1. Som standard är värdet lika med 0.33f.
