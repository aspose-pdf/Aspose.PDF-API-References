## PrinterMarksKind enum


Specificerar typerna av skrivarens markeringar som ska läggas till i ett dokument.

```cpp
enum class PrinterMarksKind
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Ingen | 0 | Specificerar att inga printer's markeringar ska läggas till. |
| TrimMarks | n/a | Specificerar att trimmarkeringar ska läggas till. |
| BleedMarks | n/a | Specificerar att bleed-markeringar ska läggas till. |
| RegistrationMarks | n/a | Specificerar att registreringsmarkeringar ska läggas till. |
| ColorBars | n/a | Anger att färgfält ska läggas till. |
| PageInformation | n/a | Anger att sidinformation ska läggas till. |
| Alla | n/a | Anger att alla typer av skrivarens markeringar ska läggas till. |

## Anmärkningar


Denna uppräkning har ett [FlagsAttribute](../)-attribut som möjliggör en bitvis kombination av dess medlemsvärden.
