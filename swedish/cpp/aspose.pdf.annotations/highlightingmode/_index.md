## HighlightingMode enum


Enumererar annoteringens markeringsläge, den visuella effekten som ska användas när musknappen trycks ned eller hålls nedtryckt inom dess aktiva område.

```cpp
enum class HighlightingMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Ingen | 0 | Ingen markering. |
| Invert | 1 | Invertera innehållet i annoteringsrektangeln. |
| Kontur | 2 | Invertera annoteringens kant. |
| Tryck | 3 | Visa annoteringens nedre utseende, om något finns. Om inget nedre utseende är definierat, förskjut innehållet i annoteringsrektangeln så att det ser ut som om det trycks ner under sidans yta. |
| Växla | 4 | Samma som Tryck (vilket föredras). |
