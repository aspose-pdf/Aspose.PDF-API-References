Den **[Aspose.Pdf.Structure](./)** namnrymden tillhandahåller klasser som hjälper till att arbeta med en logisk struktur av ett dokument som presenterar dokumentets organisation i kapitel och sektioner eller identifieringen av speciella element såsom figurer, tabeller och fotnoter.

| Klass | Beskrivning |
| --- | --- |
| [Element](./element/) | Klass som representerar baselementet i den logiska strukturen. |
| [ElementCollection](./elementcollection/) | [Collection](../aspose.pdf/collection/) av baslogiska strukturelement. |
| [FigureElement](./figureelement/) | Klass som representerar en figur i den logiska strukturen. |
| [RootElement](./rootelement/) | Rotstrukturselement. |
| [StructElement](./structelement/) | Allmänt strukturselement. |
| [TextElement](./textelement/) | Allmänt textelement i dokumentets logiska struktur. |
