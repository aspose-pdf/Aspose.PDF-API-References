## PdfFileInfo::get_HasEditPassword method


Returnerar true om ett lösenord krävs för att ändra behörigheter eller dokumentets säkerhetsegenskap. Observera att denna egenskap endast kan läsas om ett giltigt lösenord har angetts i [PdfFileInfo](../)‑konstruktorn. Om [PasswordType](../../../aspose.pdf/passwordtype/) är Inaccessible (betyder att ett ogiltigt lösenord angavs) kommer läsning av denna egenskap att misslyckas med [InvalidPasswordException](../../../aspose.pdf/invalidpasswordexception/).
