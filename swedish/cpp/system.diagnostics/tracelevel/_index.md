## TraceLevel enum


Anger vilka meddelanden som ska skrivas ut för klasserna [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) och System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Av | 0 | Skriv ut inga spårnings- och felsökningsmeddelanden. |
| Error | 1 | Skriv ut felhanteringsmeddelanden. |
| Warning | 2 | Skriv ut varnings- och felhanteringsmeddelanden. |
| Info | 3 | Skriv ut informationsmeddelanden, varningar och felhanteringsmeddelanden. |
| Utförlig | 4 | Skriv ut alla felsöknings- och spårningsmeddelanden. |
