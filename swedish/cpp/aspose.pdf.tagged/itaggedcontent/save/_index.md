## ITaggedContent::Save method


Sparar det aktuella tillståndet för det taggade innehållet till det associerade PDF-dokumentet.

```cpp
virtual void Aspose::Pdf::Tagged::ITaggedContent::Save()=0
```

## Anmärkningar


Denna metod säkerställer att alla element i det taggade innehållet uppdateras och sparas korrekt i PDF-dokumentet. Den utför nödvändiga operationer såsom att uppdatera MCID för MCR‑element, sätta BDC‑operatorer och säkerställa efterlevnad av PDF/UA‑standarder.
