## TypeInfo::GetCustomAttribute method


Söker efter det anpassade attributet som har den angivna typen och som är tillämpat på typen som representeras av det aktuella objektet.

```cpp
ObjectPtr System::TypeInfo::GetCustomAttribute(const TypeInfo &attributeType) const
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Den konstanta referensen till [TypeInfo](../)-objektet som representerar typen av attributet att söka efter |

### ReturnValue

En pekare till ett objekt som representerar det hittade attributet, eller null-pekare om inget attribut hittades som matchade sökkriterierna
