## DateTime::Compare method


Jämför två värden som representeras av de angivna instanserna av klassen [DateTime](../) och returnerar värdet som indikerar värdens relativa positioner på tidslinjen.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| t1 | DateTime | Den första jämförelsetermen |
| t2 | DateTime | Den andra jämförelsetermen |

### ReturnValue

Ett värde som är mindre än 0 om **t1** är tidigare än **t2**; 0 om **t1** är samma som **t2**; ett värde som är större än 0 om **t1** är senare än **t2**
