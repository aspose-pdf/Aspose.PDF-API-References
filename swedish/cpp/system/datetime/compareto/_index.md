## DateTime::CompareTo method


Jämför två datum- och tidsvärden som representeras av det aktuella objektet och den angivna instansen av [DateTime](../)-klassen och returnerar ett värde som indikerar värdenas relativa positioner på tidslinjen.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| value | DateTime | En instans av [DateTime](../)-klassen för att jämföra med det aktuella objektet. |

### ReturnValue

Ett värde som är mindre än 0 om det aktuella objektet representerar ett värde som är tidigare än det som representeras av **value**; 0 om värdena som representeras av båda objekten är samma; ett värde som är större än 0 om värdet som representeras av det aktuella objektet är senare än det som representeras av **value**.
