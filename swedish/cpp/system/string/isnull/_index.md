## String::IsNull method


Kontrollerar om strängen anses vara null. [String](../) är null endast om den konstrueras via [String()](../string/) konstruktor, flyttas, kopieras eller tilldelas från en null-sträng eller om [reset()](../reset/) metoden har anropats.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true om strängen anses vara null, false annars.
