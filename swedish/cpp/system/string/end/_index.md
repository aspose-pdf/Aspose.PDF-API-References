## String::end method


Returnerar pekare till slutet av den faktiska strängbufferten. Allokerar aldrig om något. Garanterar inte att bufferten är null-terminerad.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Pekare till det första elementet efter den faktiska strängbufferten, nullterminerad eller ej.
