## SmartPtr::get_shared_count method


Hämtar antalet delade pekare som finns till det refererade objektet, inklusive den nuvarande. Påstår att den aktuella pekaren är i delat läge.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Antal delade pekare som finns till det refererade objektet, om några. Om pekaren är null returneras 0.
