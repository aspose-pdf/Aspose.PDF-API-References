## Object::Is method


Kontrollera om objektet representerar en instans av den typ som beskrivs av targetType. Analog till C# 'is'-operatorn.

```cpp
virtual bool System::Object::Is(const TypeInfo &targetType) const
```


| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| targetType | const TypeInfo\& | [TypeInfo](../../typeinfo/) struktur som beskriver typen att testa det aktuella objektet mot. |

### ReturnValue

Sant om objektet är av den märkta typen eller dess underklass, annars falskt.
