## PKCS7Detached class


Representerar PKCS#7-objektet som följer PKCS#7-specifikationen i Internet RFC 2315, PKCS #7: Cryptographic Message Syntax, Version 1.5. Den ursprungliga signerade meddelandedigesten över dokumentets byteintervall införlivas som det normala PKCS#7 SignedData-fältet. Ingen data ska kapslas in i PKCS#7 SignedData-fältet.

class PKCS7Detached : public Aspose::Pdf::Forms::Signature
