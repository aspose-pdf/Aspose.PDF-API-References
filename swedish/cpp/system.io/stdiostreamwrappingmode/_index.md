## STDIOStreamWrappingMode enum


Anger läget för I/O‑operationer som omslag kommer att utföra på strömmar liknande std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Binär | 0 | Ett läge som tillåter inmatningsoperationer att avkoda strömdata av typen char_type till byte, och att koda byte till char_type‑data för utmatningsoperationer. |
| Konvertering | 1 | Ett läge som tillåter inmatningsoperationer att konvertera strömdata från typen char_type till uint8_t och vice versa för utmatningsoperationer. |
