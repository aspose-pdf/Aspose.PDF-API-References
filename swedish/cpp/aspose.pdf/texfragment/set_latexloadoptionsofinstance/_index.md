## TeXFragment::set_LatexLoadOptionsOfInstance method


Anger [TeXLoadOptions](../../texloadoptions/) som kommer att användas för inläsning (och rendering) av LaTeX i detta klassinstans. Använd den när det är nödvändigt att använda en specifik inställning för import av LaTeX för denna eller den andra instansen (t.ex. när denna eller den andra instansen ska använda en specifik BasePath för importerad LaTeX eller en specifik laddare för externa resurser). Om parametern är standard (null) kommer standardalternativen för LaTeX‑inläsning att användas.

```cpp
void Aspose::Pdf::TeXFragment::set_LatexLoadOptionsOfInstance(const System::SharedPtr<TeXLoadOptions> &value)
```
