## TextStamp::set_TreatYIndentAsBaseLine method


Definierar koordinatursprunget för placering av text. Om TreatYIndentAsBaseLine = true (standard när Draw = true) kommer YIndent‑värdet att behandlas som textens baslinje. Om TreatYIndentAsBaseLine = false (standard när Draw = false) kommer YIndent‑värdet att behandlas som botten (nedstigningslinje) av texten.
