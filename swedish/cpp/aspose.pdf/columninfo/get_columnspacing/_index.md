## ColumnInfo::get_ColumnSpacing method


Hämtar en sträng som innehåller avståndet mellan kolumner. Värdet för varje avstånd ska separeras med blanksteg. Standardenheten är punkt, men cm och tum stöds också. Till exempel, "120 2.5cm 1.5inch".

```cpp
System::String Aspose::Pdf::ColumnInfo::get_ColumnSpacing() const
```

## Anmärkningar


Om den här egenskapen inte är angiven används standardvärdet 0 för varje avstånd.
