## HtmlPageLayoutOption enum


Anger flaggor som tillsammans med andra alternativ bestämmer sidors storlekar och layouter.

```cpp
enum class HtmlPageLayoutOption
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | 0 | Standardvärde som indikerar att [HtmlPageLayoutOption](./) inte kommer att påverka sidornas storlekar och layouter. |
| FitToWidestContentWidth | n/a | Indikerar att sidans bredd bestäms av innehållets storlek själv, inte av den angivna sidbredden. |
| ScaleToPageWidth | 256 | Indikerar att dokumentets innehåll kommer att skalas för att passa sidan. |
