## CollectionSort::get_S method


Hämtar namn eller namn på fält som den interaktiva PDF-processorn ska använda för att sortera objekten i samlingen. Om värdet är ett namn identifierar det ett fält som beskrivs i den överordnade samlingsordboken. Om värdet är en array ska varje element i arrayen vara ett namn som identifierar ett fält som beskrivs i den överordnade samlingsordboken. Arrayformen ska användas för att låta ytterligare fält bidra till sorteringen, där varje extra fält används för att bryta lika resultat. Mer specifikt, om flera samlingsobjektsordböcker har samma värde för det första fältet som anges i arrayen, ska värdena för efterföljande fält som anges i arrayen användas för sortering, tills en unik ordning har fastställts eller tills de namngivna fälten är uttömda.
