## Page::get_Rect method


Hämtar rektangeln för sidan. Vid hämtning: sidans beskärningsruta returneras om den är angiven, annars returneras sidans mediabox. Vid inställning: sidans mediabox sätts alltid. Observera att denna egenskap inte tar hänsyn till sidrotation. För att hämta sidrektangeln med hänsyn till rotation, använd ActualRect.
