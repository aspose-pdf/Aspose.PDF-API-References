## Page::set_Rect method


Ställer in rektangeln för sidan. Vid hämtning: returneras sidans beskärningsruta om den är specificerad, annars returneras sidans mediabox. Vid inställning: sidans mediabox sätts alltid. Observera att denna egenskap inte tar hänsyn till sidrotation. För att få sidrektangeln med hänsyn till rotation, använd ActualRect.
