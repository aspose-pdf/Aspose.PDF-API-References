## ParagraphPositioningMode enum


Anger variant för att bestämma elementets placering på sidan.

```cpp
enum class ParagraphPositioningMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Standard | 0 | Placeringen bestäms av de tidigare placerade elementen. När ett element läggs till tas det i beaktande vid bestämning av placeringen för efterföljande element. |
| Absolute | 1 | Placeringen anges av [Left](../left/)‑ och Top‑värdena, beror inte på tidigare element och påverkar inte placeringen av efterföljande. |
