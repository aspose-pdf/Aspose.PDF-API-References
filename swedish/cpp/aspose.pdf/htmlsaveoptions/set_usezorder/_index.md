## HtmlSaveOptions::set_UseZOrder method


Om attributet UseZORder är satt till true läggs grafik och text till det resulterande HTML‑dokumentet enligt Z‑ordning i den ursprungliga PDF‑filen. Om detta attribut är false placeras all grafik som ett enda lager, vilket kan orsaka onödiga effekter för överlappande objekt.
