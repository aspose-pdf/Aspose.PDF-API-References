## HtmlSaveOptions::get_SaveFullFont method


Indikerar att hela teckensnittet kommer att sparas, stöder endast True Type‑teckensnitt. Som standard är SaveFullFont = false och konverteraren sparar den delmängd av det ursprungliga teckensnittet som behövs för att visa dokumentets text.
