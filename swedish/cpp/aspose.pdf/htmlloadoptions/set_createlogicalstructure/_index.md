## HtmlLoadOptions::set_CreateLogicalStructure method


Ställer in ett värde som indikerar om en logisk struktur ska skapas i det resulterande PDF‑dokumentet.

```cpp
void Aspose::Pdf::HtmlLoadOptions::set_CreateLogicalStructure(bool value)
```

## Anmärkningar


När den är inställd på **true**

, den logiska strukturen för dokumentet skapas, vilket kan förbättra tillgängligheten och göra dokumentet mer lämpligt för skärmläsare och annan hjälpmedelsteknik.
