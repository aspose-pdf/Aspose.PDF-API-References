## DocSaveOptions::set_ConvertType3Fonts method


Ställer in konvertering för Type3-teckensnitt. I Type 3-teckensnitt ska glyfer definieras av strömmar av grafikoperatorer. Detta innebär att i DOC/DOCX-utdata ser vi bilder istället för text. Ställ in denna flagga till true för att konvertera Type3-teckensnitt till TTF och få text i den resulterande filen.
