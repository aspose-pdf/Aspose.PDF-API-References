## DocSaveOptions::get_ReSaveFonts method


Hämtar proceduren för att spara om teckensnitt. Om den är satt till true laddar vi om teckensnitt på varje sida för att undvika påverkan från tidigare teckensnittsegenskaper och laddar det nyss skapade teckensnittet från början. Ställ in detta alternativ till false om du vill förbättra prestandan. Standardvärdet är true;.
