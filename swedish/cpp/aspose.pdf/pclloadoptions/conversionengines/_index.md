## ConversionEngines enum


Enumererar konverteringsmotorer som kan användas för konvertering.

```cpp
enum class ConversionEngines
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| LegacyEngine | 0 | Legacy‑konverteringsmotor är en konverteringsmotor som användes ett tag och används som standard. |
| NewEngine | 1 | New‑konverteringsmotor är för närvarande i A‑testningsläge. Vi öppnar den i det offentliga API‑et för att få lite återkoppling. Det rekommenderas för närvarande att använda den endast när den gamla konverteringsmotorn inte kan konvertera dokument av vissa skäl. |
