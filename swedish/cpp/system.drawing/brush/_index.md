## Brush class


En basklass för klasser som representerar fyllningsmaterial som används för att fylla insidan av grafiska former. Objekt av denna klass bör endast allokeras med [System::MakeObject()](../../system/makeobject/) funktionen. Skapa aldrig en instans av denna typ på stacken eller med operator new, eftersom det kan leda till körfel och/eller påståendefel. Omslut alltid denna klass i en [System::SmartPtr](../../system/smartptr/) pekare och använd denna pekare för att skicka den till funktioner som argument.

```cpp
class Brush : public System::ICloneable,
              public System::IDisposable
```
