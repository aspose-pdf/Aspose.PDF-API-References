## StringDigitSubstitute enum


Anger hur siffror i en sträng ersätts enligt den lokala inställningen eller språket.

```cpp
enum class StringDigitSubstitute
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Användare | 0 | Användardefinierad ersättning. |
| Ingen | 1 | Subtitution är inaktiverad. |
| Nationell | 2 | Ersätt siffror som motsvarar det officiella nationella språket för användarens lokala inställning. |
| Traditionell | 3 | Ersätt siffror som motsvarar användarens inhemska skriftsystem eller språk, vilket kanske inte är detsamma som det officiella nationella språket för användarens lokala inställning. |
