## CancellationToken::get_CanBeCanceled method


Hämtar om denna token kan vara i avbrutet tillstånd.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true om denna token kan vara i avbrutet tillstånd; annars false.
## Anmärkningar



Token som skapas från [CancellationTokenSource](../../cancellationtokensource/) kommer att returnera true, medan None-token alltid returnerar false.
