## TimeSpanStyles enum


Anger formateringsalternativen som anpassar strängparsning för metoder som konverterar en strängrepresentation av ett tidsintervall till ett Timespan-objekt.

```cpp
enum class TimeSpanStyles : int32_t
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Ingen | 0 | Tidsintervallet som representeras av inmatningssträngen tolkas som ett negativt tidsintervall endast om ett negativt tecken finns. |
| AssumeNegative | 1 | Tidsintervallet som representeras av inmatningssträngen tolkas alltid som ett negativt tidsintervall. |
