## TextFormattingMode enum


Definierar olika lägen som kan användas vid konvertering av ett PDF-dokument till text. Se [TextExtractorOptions](../)-klassen.

```cpp
enum class TextFormattingMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Pure | 0 | Representerar PDF-textinnehåll med några formateringsrutiner. Detta inkluderar att ta hänsyn till relativa positioner och lägga till extra mellanslag för att justera texten till sidans bredd. |
| Raw | 1 | Representerar PDF-innehåll som det är, d.v.s. utan formatering. |
| Plain | 2 | Representerar PDF-textinnehåll som tar hänsyn till den relativa placeringen av textfragment men utan att lägga till extra mellanslag. |
