## PdfAOptionsBase::set_OptimizeFileSize method


Ställer in ett värde som indikerar om filstorleken ska försöka minskas under PDF/A-konverteringsprocessen.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_OptimizeFileSize(bool value)
```

## Anmärkningar


**true**

om filstorleken ska minskas så mycket som möjligt; annars, **false**

.

När den är inställd på **true**

, konverteringsprocessen kommer att försöka minimera den resulterande filstorleken. Detta kan påverka konverteringsprocessens prestanda.
