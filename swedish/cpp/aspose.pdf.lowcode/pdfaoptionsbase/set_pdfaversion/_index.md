## PdfAOptionsBase::set_PdfAVersion method


Ställer in versionen av PDF/A-standarden som ska användas för validering eller konvertering.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_PdfAVersion(PdfAStandardVersion value)
```

## Anmärkningar


Versionen av PDF/A‑standarden. Detta kan vara ett av värdena i uppräkningen [PdfAStandardVersion](../../pdfastandardversion/).

PDF/A‑standardversionen används för att bestämma efterlevnadsnivån för PDF/A‑validering och -konvertering. Om versionen är inställd på [PdfAStandardVersion::Auto](../../pdfastandardversion/) kommer systemet automatiskt att bestämma den lämpliga PDF/A‑standardversionen för validering baserat på dokumentets metadata. För PDF/A‑konverteringsprocessen är standardvärdet för [PdfAStandardVersion::Auto](../../pdfastandardversion/) PDF/A‑1b‑standardversionen.
