## PdfAOptionsBase::get_OptimizeFileSize method


Hämtar ett värde som indikerar om filstorleken ska försöka minskas under PDF/A‑konverteringsprocessen.

```cpp
bool Aspose::Pdf::LowCode::PdfAOptionsBase::get_OptimizeFileSize()
```

## Anmärkningar


**true**

om filstorleken ska minskas så mycket som möjligt; annars, **false**

.

När den är inställd på **true**

, konverteringsprocessen kommer att försöka minimera den resulterande filstorleken. Detta kan påverka konverteringsprocessens prestanda.
