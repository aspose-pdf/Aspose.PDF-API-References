## PdfAOptionsBase::set_ExcludeFontsStrategy method


Ställer in strategin för att ta bort teckensnitt för att minimera utdatafilens storlek under PDF/A-konverteringsprocessen.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_ExcludeFontsStrategy(PdfFormatConversionOptions::RemoveFontsStrategy value)
```

## Anmärkningar


Strategin för att ta bort typsnitt. Detta kan vara ett av värdena i uppräkningen [PdfFormatConversionOptions::RemoveFontsStrategy](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/). Standardvärdet är kombinationen av [PdfFormatConversionOptions::RemoveFontsStrategy::SubsetFonts](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/) och [PdfFormatConversionOptions::RemoveFontsStrategy::RemoveDuplicatedFonts](../../../aspose.pdf/pdfformatconversionoptions/removefontsstrategy/).

Denna egenskap låter dig kontrollera hur typsnitt hanteras under konverteringsprocessen. Du kan välja att ta bort duplicerade typsnitt, ta bort liknande typsnitt med olika bredd, eller delmängda typsnitt.
