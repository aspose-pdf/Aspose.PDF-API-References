## StructureTypeStandard.Note-fält

(Not) Ett objekt av förklarande text, såsom en fotnot eller en slutnot, som hänvisas till från dokumentets brödtext. Det kan ha en etikett (strukturtyp Lbl) som ett barn. Noten kan inkluderas som ett barn av strukturens element i brödtexten som hänvisar till den, eller så kan den inkluderas på annat håll (såsom i en slutnotavsnitt) och nås genom en referens (strukturtyp Reference).

Taggad PDF föreskriver inte placeringen av fotnoter i sidinnehållets ordning. De kan vara antingen inline eller i slutet av sidan, efter författarens eget omdöme.

```csharp
public static readonly StructureTypeStandard Note;
```
