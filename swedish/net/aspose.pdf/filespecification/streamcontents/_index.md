## FileSpecification.StreamContents-egenskap

Hämtar innehållet i filen som en ström. Innehållet laddas inte in i minnet vilket minskar minnesanvändningen. Men denna ström stöder inte positionering och Length-egenskapen. Om du behöver dessa funktioner, använd istället Contents-egenskapen.

```csharp
public Stream StreamContents { get; }
```
