## BatesNArtifact.NumberOfDigits-egenskap

Hämtar eller ställer in antalet siffror för Bates-nummerering. Värdet måste vara mellan 3 och 15 inklusive. Om ett värde mindre än 3 anges, justeras det till 3. Om ett värde större än 15 anges, justeras det till 15. Standardvärdet är 6.

```csharp
public int NumberOfDigits { get; set; }
```
