## OptimizedMemoryStream.Write metod

När den åsidosätts i en härledd klass, skriver den en sekvens av byte till den aktuella strömmen och avancerar den aktuella positionen inom denna ström med antalet skrivna byte.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| buffer | Byte[] | En array av byte. Denna metod kopierar *count* byte från *buffer* till den aktuella strömmen. |
| offset | Int32 | Den nollbaserade byte-offseten i *buffer* där kopieringen av byte till den aktuella strömmen ska börja. |
| count | Int32 | Antalet byte som ska skrivas till den aktuella strömmen. |
