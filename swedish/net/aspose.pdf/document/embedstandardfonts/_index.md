## Document.EmbedStandardFonts property

Egenskapen som deklarerar att dokumentet måste bädda in alla standardtypsnitt som har flaggan IsEmbedded satt till true. Alla PDF-teckensnitt kan bäddas in i dokument helt enkelt via inställningen av flaggan IsEmbedded in true, men PDF-standard Type1-teckensnitt är ett undantag från denna regel. Standard Type1-fontinbäddning kräver mycket tid, så för att bädda in dessa typsnitt är det nödvändigt inte bara ange flagga IsEmbedded in true för specificerat teckensnitt men ställ också in en tilläggsflagga på dokumentets nivå - EmbedStandardFonts = true; Den här egenskapen kan endast ställas in en gång för alla teckensnitt. Som standard false.

public bool EmbedStandardFonts { get; set; }
