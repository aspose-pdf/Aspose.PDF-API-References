## Document.AllowReusePageContent-egenskap

Möjliggör sammanslagning av sidinnehåll för att optimera dokumentstorlek. Om den används kan olika men duplicerade sidor referera till samma innehållsobjekt. Observera att detta läge kan orsaka bieffekter som att sidinnehållet ändras när en annan sida ändras.

```csharp
public bool AllowReusePageContent { get; set; }
```
