## HtmlSaveOptions.HtmlImageSavingInfo.ParentType-fält

Den sparade bilden kan hänvisa till HTML själv eller kan extraheras från SVG inbäddad i HTML. Denna egenskap kan berätta för anpassad kod vilken typ av förälder den bearbetade bilden har. Den sätts av konverteraren och kan användas i anpassad kod för att avgöra vad som ska göras med den bilden (t.ex. kan anpassad kod avgöra var bilden ska sparas eller hur den måste refereras i förälderns innehåll).

public ImageParentTypes ParentType;
