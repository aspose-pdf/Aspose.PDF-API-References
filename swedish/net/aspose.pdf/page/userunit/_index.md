## Page.UserUnit-egenskap

Hämtar eller ställer in UserUnit-värdet. Ett positivt tal som anger storleken på standard användarutrymmesenheter, i multiplar av 1 / 72 tum. Standardvärdet är 1. Vänligen ställ in noll eller negativt värde för att rensa denna post på sidan.

```csharp
public double UserUnit { get; set; }
```
