## Page.Rect property

Hämtar eller ställer in sidans rektangel. Sidbeskärningsruta returneras om det anges, annars returneras sidmediabox. Observera att den här egenskapen inte tar hänsyn till sidrotation. För att få sidrektangel med tanke på rotation använd ActualRect.

```csharp
public Rectangle Rect { get; set; }
```

### Exempel

Exempel visar hur man får sidrektangel:

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
Rectangle pageRect = page.Rect;
```
