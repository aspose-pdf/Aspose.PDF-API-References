## HtmlSaveOptions.UseZOrder property

Om attributet UseZORder är inställt på sant, läggs grafik och text till i det resulterande HTML-dokumentet i enlighet med Z-ordningen i det ursprungliga PDF-dokumentet. Om detta attribut är falskt är all grafik put som ett lager vilket kan orsaka onödiga effekter för överlappande objekt.

public bool UseZOrder { get; set; }
