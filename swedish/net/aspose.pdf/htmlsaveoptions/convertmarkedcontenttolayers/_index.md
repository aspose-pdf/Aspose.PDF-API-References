## HtmlSaveOptions.ConvertMarkedContentToLayers property

Om attributet ConvertMarkedContentToLayers är inställt på sant kommer alla element i ett PDF marked innehåll (lager) att läggas in i en HTML div med "data-pdflayer" attribut som anger ett lagernamn. Detta lagernamn kommer att extraheras från valfria egenskaper för PDF markerat innehåll. Om detta attribut är falskt (som standard) kommer inga lager att skapas från PDF-markerat innehåll.

public bool ConvertMarkedContentToLayers { get; set; }
