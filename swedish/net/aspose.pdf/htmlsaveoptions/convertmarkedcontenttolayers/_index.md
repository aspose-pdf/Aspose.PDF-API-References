## HtmlSaveOptions.ConvertMarkedContentToLayers-egenskap

Om attributet ConvertMarkedContentToLayers är inställt på true kommer alla element inuti ett PDF-märkt innehåll (lager) att placeras i en HTML-div med "data-pdflayer"-attributet som specificerar ett lager namn. Detta lager namn kommer att extraheras från valfria egenskaper för PDF-märkt innehåll. Om detta attribut är false (som standard) kommer inga lager att skapas från PDF-märkt innehåll.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
