## HtmlSaveOptions.MinimalLineWidth property

Det här attributet anger minimal bredd på grafisk väglinje. Om tjockleken på linjen är mindre än 1px avrundar Adobe Acrobat den till detta värde. Så detta attribut kan användas för att emulera detta beteende för HTML-webbläsare.

public float MinimalLineWidth { get; set; }
