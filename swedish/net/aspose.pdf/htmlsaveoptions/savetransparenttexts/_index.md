## HtmlSaveOptions.SaveTransparentTexts-fält

Pdf kan innehålla transparenta texter som kan väljas till urklipp (vanligtvis händer det när dokumentet innehåller bilder och OCR:ade texter som extraherats från det). Denna inställning talar om för konverteraren om vi behöver spara sådana texter som transparenta valbara texter i den resulterande HTML:en

```csharp
public bool SaveTransparentTexts;
```
