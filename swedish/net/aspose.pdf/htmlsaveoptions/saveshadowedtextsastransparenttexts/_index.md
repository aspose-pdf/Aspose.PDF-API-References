## HtmlSaveOptions.SaveShadowedTextsAsTransparentTexts-fält

Pdf kan innehålla texter som skuggas av andra element (t.ex. av bilder) men kan väljas till urklipp i Acrobat Reader (vanligtvis händer det när dokumentet innehåller bilder och OCR:ade texter som extraherats från det). Denna inställning talar om för konverteraren huruvida vi behöver spara sådana texter som transparenta valbara texter i den resulterande HTML:en för att efterlikna beteendet hos Acrobat Reader (annars sparas sådana texter vanligtvis som dolda, inte tillgängliga för kopiering till urklipp).

```csharp
public bool SaveShadowedTextsAsTransparentTexts;
```
