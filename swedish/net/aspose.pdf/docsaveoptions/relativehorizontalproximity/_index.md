## DocSaveOptions.RelativeHorizontalProximity-egenskap

I Pdf kan ord inre representeras med operatorer som skriver ut ord genom att oberoende skriva ut deras bokstäver eller stavelser. Så, för att upptäcka ord behöver vi ibland upptäcka grupper av oberoende tecken som faktiskt är ord. Denna inställning definierar bredden på utrymmet mellan textelement (bokstäver, stavelser) som måste behandlas som avstånd mellan ord under igenkänning av ord i käll-PDF. (Närvaron av tomt utrymme, åtminstone med denna bredd mellan bokstäver, betyder att textuella element tillhör olika ord). Det är normerat till teckenstorlek - 1.0 betyder 100% av den antagna ordets teckenstorlek. UPPMÄRKSAMHET! Det används endast i fall där käll-PDF innehåller specifika sällan använda typsnitt för vilka optimalt värde inte kan beräknas från typsnittet. Så, i den stora majoriteten av fallen ändrar denna parameter ingenting i resultatdokumentet.

```csharp
public float RelativeHorizontalProximity { get; set; }
```
