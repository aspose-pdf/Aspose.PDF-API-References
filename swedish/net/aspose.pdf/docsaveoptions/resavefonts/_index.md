## DocSaveOptions.ReSaveFonts-egenskap

Hämtar eller ställer in proceduren för att spara om typsnitt. Om det är inställt på sant, laddar vi om typsnitten på varje sida för att undvika påverkan av tidigare typsnittegenskaper och laddar det nyss skapade typsnittet från grunden. Ställ in det här alternativet på falskt om du vill förbättra prestanda. Standardvärdet är sant;

```csharp
public bool ReSaveFonts { get; set; }
```
