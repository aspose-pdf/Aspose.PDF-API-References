## HtmlSaveOptions.HtmlPageMarkupSavingInfo class

Om SplitToPages-egenskapen för HtmlSaveOptions, så skapas flera HTML-filer (en HTML-fil per konverterad sida) under konvertering av PDF till HTML. Den här klassen representerar uppsättning data som relaterade till anpassad lagring av en HTML-sidas markering under konvertering av PDF till HTML

Egenskapen kan användas i logiken för anpassad code för att bestämma hur HTML-sidan ska bearbetas eller var HTML-sidan ska sparas och Om delning på sidor avstängd innehåller detta värde alltid '1' eftersom i ett sådant fall genereras endast en stor HTML-sida för hela källdokumentet. |

Den här egenskapen talar om för anpassad kod från vilken sida i den ursprungliga PDF-filen som skapades sparad HTML-uppmärkning . Om det ursprungliga sidnumret av någon anledning är okänt eller SplitOnPages=false, så innehåller den här egenskapen alltid '0' som signalerar att omvandlaren inte kan tillhandahålla exakt original-PDFs sidnummer för den medföljande HTML-markeringsfilen. |
