## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber field

Ställs in av converter. Om SplitToPages-egenskapen ställs in, skapas flera HTML-filer (en HTML-fil per konverterad sida) under konverteringen som skapas. Den här egenskapen innehåller ordinal för sparad HTML-sidas fil. Egenskapen kan användas i logiken för anpassad code för att bestämma hur HTML-sidan ska bearbetas eller var HTML-sidan ska sparas och Om delning på sidor avstängd innehåller detta värde alltid '1' eftersom i ett sådant fall genereras endast en stor HTML-sida för hela källdokumentet.

public int HtmlHostPageNumber;
