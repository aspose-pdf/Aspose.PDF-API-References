## FloatingBox.IsNeedRepeating-egenskap

Hämtar eller ställer in ett bool-värde som indikerar om stycket behöver upprepas på nästa sida. Standardvärdet är false. Attributet är endast giltigt när stycket självt och objektet som dess ReferenceParagraphID hänvisar till båda ingår i RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
