## PdfFormatConversionOptions.OptimizeFileSize-egenskap

Hämtar eller ställer in en flagga som aktiverar/deaktiverar ett speciellt konverteringsläge för att få ett PDF/A-dokument med minskad filstorlek. Nu påverkar denna flagga optimeringen av teckensnitt som används i PDF-dokumentet, möjligtvis, i framtiden, kommer denna flagga också att användas för att aktivera optimering för andra datakonstruktioner, såsom grafik. Inställningen av denna flagga och läge kan avsevärt minska filstorleken men samtidigt kan det avsevärt minska prestandan för konverteringen.

```csharp
public bool OptimizeFileSize { get; set; }
```
