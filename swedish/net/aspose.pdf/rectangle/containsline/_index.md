## Rectangle.ContainsLine metod

Bestämmer om rektangeln innehåller en linje representerad av två punkter.

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x1 | Double | X-koordinaten för startpunkten av linjen. |
| y1 | Double | Y-koordinaten för startpunkten av linjen. |
| x2 | Double | X-koordinaten för slutpunkten av linjen. |
| y2 | Double | Y-koordinaten för slutpunkten av linjen. |

### Returvärde

`true` om rektangeln innehåller linjen; annars, `false`.
