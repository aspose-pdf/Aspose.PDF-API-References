## Stamp.Background-egenskap

Sätter eller hämtar ett bool-värde som indikerar att innehållet är stämplat som bakgrund. Om värdet är sant läggs stämpelinnehållet längst ner. Som standard är värdet falskt, stämpelinnehållet läggs längst upp.

```csharp
public bool Background { get; set; }
```
