## HtmlFragment.HtmlLoadOptions property

Hämtar eller ställer in HtmlLoadOptions som kommer att användas för att ladda (och rendera) av HTML i den här instansen av klassen. Använd den när det är nödvändigt använd specifik inställning för import av HTML för den eller den instansen (till exempel när den här eller den instansen ska använd specifik BasePath för importerad HTML eller bör använda specifik laddare av externa resurser) Om parametern är standard (null), kommer standardalternativ för HTML-laddning att användas.

```csharp
public HtmlLoadOptions HtmlLoadOptions { get; set; }
```
