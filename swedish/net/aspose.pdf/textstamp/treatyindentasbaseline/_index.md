## TextStamp.TreatYIndentAsBaseLine property

Definierar koordinatursprung för att placera text. Om TreatYIndentAsBaseLine = true (standard när Draw = true) YIndent-värde kommer att behandlas som textbasrad. Om TreatYIndentAsBaseLine = false (standardvärde när Draw = true) kommer YIndent-värdet att behandlas som textbaslinje. Om TreatYIndentAsBaseLine = false (standardvärde när Draw = true) nedstigningslinje) av text.
