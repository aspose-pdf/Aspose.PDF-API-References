## TeXFragment.TeXLoadOptionsOfInstance property

Hämtar eller ställer in TeXLoadOptions som kommer att användas för att ladda (och rendera) av LaTeX i denna instans av klassen. Använd den när det är nödvändigt använd specifik inställning för import av LaTeX för den eller den instansen (exempelvis när den här eller den instansen ska använd specifik BasePath för importerad LaTeX eller bör använda specifik laddare av externa resurser) Om parametern är standard (null), kommer standard LaTeX-laddningsalternativ att användas.

public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
