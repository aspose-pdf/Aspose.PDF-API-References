## RenderingOptions.UseFontHinting property

Användning av denna flagga aktiverar teckensnittsantydningsmekanism. Teckensnittstips är användningen av matematiska instruktioner för att justera visningen av ett konturteckensnitt. I vissa fall kan det lösa problem med textens läsbarhet att aktivera denna flagga. För närvarande kan användning av denna flagga endast ge effekt för TTF-teckensnitt, om dessa typsnitt används i källdokumentet.

public bool UseFontHinting { get; set; }
