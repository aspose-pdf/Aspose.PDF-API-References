## RenderingOptions.ConvertFontsToUnicodeTTF-egenskap

Indikerar att alla teckensnitt kommer att konverteras till TTF unicode-versioner. Det är användbart av kompatibilitetsskäl och för att optimera användningen av teckensnitt, eftersom varje nytt TTF-teckensnitt inte kommer att ha alla symboler från källteckensnittet, utan endast symboler som används i text.

```csharp
public bool ConvertFontsToUnicodeTTF { get; set; }
```
