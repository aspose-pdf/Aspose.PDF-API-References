## XslFoLoadOptions.ParsingErrorsHandlingTypes enumeration

Käll XSLFO-dokument kan innehålla formateringsfel. Denna uppräkning räknar upp möjliga strategier för att hantera sådana formateringsfel

```csharp
public enum ParsingErrorsHandlingTypes
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| TryIgnore | `0` | I det här fallet kommer omvandlaren att instrueras att försöka fortsätta med konvertering och ignorera hittade formateringsfel. I det här fallet kan framgång inte garanteras, allvarliga problem kan uppstå senare i omvandlaren, och i sugfall kommer att kastas undantag med lista över funna_x00 fel. |
| ThrowExceptionImmediately | `1` | I det här fallet kommer konverteringen att stoppas omedelbart och undantaget kommer att kastas omedelbart efter detektion av första formateringsfelet |
| InvokeCustomHandler | `2` | Detta är den mest smidiga metoden - anpassad kod måste tillhandahålla (i egenskapen WarningCallback) special -hanterare som kommer att anropas när formateringsfel upptäcks. Den hanteraren kan t.ex. logga eller räkna fel etc och kommer att ge beslut om huruvida behandlingen kan fortsätta för det här eller det felet. |
