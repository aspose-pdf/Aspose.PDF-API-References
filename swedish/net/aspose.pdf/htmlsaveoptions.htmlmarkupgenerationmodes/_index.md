## HtmlSaveOptions.HtmlMarkupGenerationModes enumeration

Ibland finns det specifika krav på den skapade HTML:en. Denna enum definierar HTML-förberedelselägen som kan användas under konvertering av PDF till HTML för att matcha sådana specifika krav.

```csharp
public enum HtmlMarkupGenerationModes
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| WriteAllHtml | `0` | Standardläge när inga specifika krav är närvarande. Utdata som genereras kommer att innehålla alla delar av HTML utan någon speciell ytterligare bearbetning. |
| WriteOnlyBodyContent | `1` | All HTML-innehåll som ligger utanför HTML:s kropp kommer att tas bort, dvs. endast innehåll som ligger inom \c \ \c \ taggarna kommer att lämnas kvar |
