## ColumnInfo.ColumnSpacing property

Hämtar eller ställer in en sträng som innehåller avståndet mellan kolumner. Värdet för varje mellanrum ska separeras med tomt. Standardenheten är punkt, men cm och tum stöds också. Till exempel,"120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

### Anmärkningar

Om den här egenskapen inte är inställd kommer standardvärdet 0 att användas för varje mellanrum.
