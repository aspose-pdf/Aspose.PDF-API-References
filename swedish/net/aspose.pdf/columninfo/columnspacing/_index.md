## ColumnInfo.ColumnSpacing-egenskap

Hämtar eller ställer in en sträng som innehåller avståndet mellan kolumner. Värdet för varje avstånd bör separeras med blanksteg. Standardenheten är punkt, men cm och tum stöds också. Till exempel, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Kommentarer

Om denna egenskap inte är inställd kommer standardvärdet 0 att användas för varje avstånd.
