## HtmlSaveOptions.FontSavingModes uppräkning

Uppräkning av lägen som kan användas för att spara typsnitt som refereras i sparad PDF.

```csharp
public enum FontSavingModes
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| AlwaysSaveAsWOFF | `0` | Alla refererade typsnitt kommer att sparas och refereras som WOFF-typsnitt. |
| AlwaysSaveAsTTF | `1` | Alla refererade typsnitt kommer att sparas och refereras som TTF-typsnitt. |
| AlwaysSaveAsEOT | `2` | Alla refererade typsnitt kommer att sparas och refereras som EOT-typsnitt. |
| SaveInAllFormats | `3` | Alla refererade typsnitt kommer att sparas (och refereras i CSS) som 3 oberoende filer: EOT, TTH, WOFF. Det ökar storleken på utdata men gör utdata lämplig för den överväldigande majoriteten av webbläsare. |
| DontSave | `4` | Alla refererade typsnitt kommer inte att sparas. |
