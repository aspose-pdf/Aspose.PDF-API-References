## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace property

Vissa teckensnitt ger inte information om unicodes för vissa textsymboler. Denna brist på information kallar ett fel "Text kan inte mappas till Unicode". Använd denna flagga för att mappa icke-länkade symboler på unicode "mellanslag" (kod 32).
