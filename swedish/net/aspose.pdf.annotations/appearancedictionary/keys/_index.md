## AppearanceDictionary.Keys-egenskap

Hämtar nycklarna i ordboken. Om utseendeordboken har underordböcker, då innehåller `Keys` (N&#x7C;R&#x7C;D).state-värden, där N - normalt utseende, R - rollover-utseende, D - nedtryckt utseende och state - namnet på tillståndet (t.ex. On, Off för kryssrutor).

public ICollection<string> Keys { get; }
