## HighlightingMode uppräkning

Uppräkning av annoteringens markeringsläge, den visuella effekten som ska användas när musknappen trycks ned eller hålls nere inom dess aktiva område.

```csharp
public enum HighlightingMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | `0` | Ingen markeringsning. |
| Invert | `1` | Invertera innehållet i annoteringens rektangel. |
| Outline | `2` | Invertera annoteringens kant. |
| Push | `3` | Visa annoteringens nedtryckta utseende, om något. Om inget nedtryckt utseende är definierat, förskjut innehållet i annoteringens rektangel så att det ser ut som om det trycktes under ytan på sidan. |
| Toggle | `4` | Samma som Push (vilket föredras). |
