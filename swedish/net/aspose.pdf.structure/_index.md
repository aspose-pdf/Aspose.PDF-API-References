Namnrymden **Aspose.Pdf.Struktur** tillhandahåller klasser som hjälper till att arbeta med en logisk struktur av ett dokument som presenterar organisationen av dokumentet i kapitel och sektioner eller identifieringen av speciella element såsom figurer, tabeller och fotnoter.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [Element](./element/) | Klass som representerar baselementet i den logiska strukturen. |
| [ElementCollection](./elementcollection/) | Samling av baslogiska strukturelement. |
| [FigureElement](./figureelement/) | Klass som representerar logisk strukturfigur. |
| [RootElement](./rootelement/) | Rotstrukturelement. |
| [StructElement](./structelement/) | Allmänt strukturelement. |
| [TextElement](./textelement/) | Allmänt textelement i dokumentets logiska struktur. |
