## Element.ActualText-egenskap

(Valfri; PDF 1.4) Text som är en exakt ersättning för struktur elementet och dess barn. Denna ersättningstext (som bör tillämpas på så liten del av innehållet som möjligt) är användbar när man extraherar dokumentets innehåll för att stödja tillgänglighet för användare med funktionsnedsättningar eller för andra ändamål.

```csharp
public virtual string ActualText { get; set; }
```
