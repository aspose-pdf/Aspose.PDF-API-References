## BaseListQueryParameters.Before-egenskap

Hämtar eller ställer in en kurs för användning i paginering. before är ett objekt-ID som definierar din plats i listan. Till exempel, om du gör en listförfrågan och får 100 objekt som slutar med obj_foo, kan din efterföljande anrop inkludera before=obj_foo för att hämta den föregående sidan av listan.

```csharp
public string Before { get; set; }
```
