## CompletionCreateRequest.ResponseFormat-egenskap

Hämtar eller ställer in ett objekt som specificerar det format som modellen måste producera. Kompatibel med GPT-4 Turbo och alla GPT-3.5 Turbo-modeller som är nyare än gpt-3.5-turbo-1106. Att ställa in till { "type": "json_object" } aktiverar JSON-läge, vilket garanterar att meddelandet som modellen genererar är giltig JSON.
