## MessageContentRequest.CreateImageUrlContent metod

Skapar en bild-URL-innehåll för ett meddelande.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| imageUrl | Sträng | URL:en till bilden. |
| detail | Sträng | Detaljen av bild-URL:en. Anger detaljnivån för bilden om den anges av användaren. "low" använder färre tokens, du kan välja hög upplösning med "high". Standard är "auto". |
