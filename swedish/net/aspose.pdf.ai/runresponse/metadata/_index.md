## RunResponse.Metadata-egenskap

Hämtar eller ställer in en uppsättning av 16 nyckel-värdepar som kan kopplas till ett objekt. Detta kan vara användbart för att lagra ytterligare information om objektet i ett strukturerat format. Nycklar kan vara maximalt 64 tecken långa och värden kan vara maximalt 512 tecken långa.

public Dictionary<string, string> Metadata { get; set; }
