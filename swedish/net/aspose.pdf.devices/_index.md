Den **Aspose.Pdf.Devices**namnutrymme tillhandahåller klasser som används för att representera dokument som bild(er) eller en vanlig text. Så dokument kan skickas vidare till text eller olika grafiska enheter vilket betyder att vi vill få det textuell eller grafisk representation.

| [DocumentDevice](./documentdevice) | Abstrakt klass för alla enheter som används för att bearbeta hela pdf-dokumentet. |
| [EmfDevice](./emfdevice) | Representerar bildenhet som hjälper till att spara pdf-dokumentsidor i emf. |
| [GifDevice](./gifdevice) | Representerar bildenhet som hjälper till att spara pdf-dokumentsidor till gif. |
| [ImageDevice](./imagedevice) | En abstrakt klass för bildenheter. |
| [JpegDevice](./jpegdevice) | Representerar bildenhet som hjälper till att spara pdf-dokumentsidor i jpeg. |
| [Margins](./margins) | Den här klassen representerar marginaler på en bild. |
| [PageDevice](./pagedevice) | Abstrakt klass för alla enheter som används för att bearbeta en viss sida i pdf-dokumentet. |

| [TiffDevice](./tiffdevice) | Den här klassen hjälper till att spara pdf-dokument sida för sida i en tiff-bild. |
| [TiffSettings](./tiffsettings) | Den här klassen representerar inställningar för att importera pdf till Tiff. |
