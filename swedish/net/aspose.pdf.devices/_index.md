Det **Aspose.Pdf.Devices** namnområdet tillhandahåller klasser som används för att representera dokument som bild(er) eller ren text. Så dokumentet kan skickas till text- eller olika grafiska enheter vilket innebär att vi vill få det i textuell eller grafisk representation.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i bmp. |
| [Device](./device/) | Abstrakt klass för alla typer av enheter. Enheten används för att representera pdf-dokument i något format. Till exempel kan dokumentets sida representeras som bild eller text. |
| [DicomDevice](./dicomdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i Dicom-format. |
| [DocumentDevice](./documentdevice/) | Abstrakt klass för alla enheter som används för att bearbeta hela pdf-dokumentet. |
| [EmfDevice](./emfdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i emf. |
| [GifDevice](./gifdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i gif. |
| [ImageDevice](./imagedevice/) | En abstrakt klass för bildeenheter. |
| [JpegDevice](./jpegdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i jpeg. |
| [Margins](./margins/) | Denna klass representerar marginaler av en bild. |
| [PageDevice](./pagedevice/) | Abstrakt klass för alla enheter som används för att bearbeta viss sida av pdf-dokumentet. |
| [PngDevice](./pngdevice/) | Representerar bildenhet som hjälper till att spara pdf-dokument sidor i png. |
| [Resolution](./resolution/) | Representerar klass för att hålla bildupplösning. |
| [TextDevice](./textdevice/) | Representerar klass för att konvertera pdf-dokument sidor till text. |
| [ThumbnailDevice](./thumbnaildevice/) | Representerar bildenhet som sparar pdf-dokument sidor i miniatyrbild. |
| [TiffDevice](./tiffdevice/) | Denna klass hjälper till att spara pdf-dokument sida för sida i en tiff-bild. |
| [TiffSettings](./tiffsettings/) | Denna klass representerar inställningar för att importera pdf till Tiff. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [ColorDepth](./colordepth/) | Används för att specificera parametervärdet som skickas till en Tiff-bildenhet. |
| [CompressionType](./compressiontype/) | Används för att specificera parametervärdet som skickas till en Tiff-bildenhet. |
| [FormPresentationMode](./formpresentationmode/) | Används för att specificera formulärpresentationläget vid utskrift eller konvertering av pdf-dokument till bild. |
| [ShapeType](./shapetype/) | Denna enum representerar formtyp för de extraherade bilderna. |
