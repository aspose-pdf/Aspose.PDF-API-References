## TextExtractorOptions.TextFormattingMode enumeration

Definierar olika lägen som kan användas vid konvertering av ett PDF-dokument till text. Se [`TextExtractorOptions`](../textextractoroptions/) klass.

```csharp
public enum TextFormattingMode
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Pure | `0` | Representerar PDF-textinnehåll med några formateringsrutiner. Detta inkluderar att ta hänsyn till relativa positioner och lägga till extra utrymmen för att justera texten till sidans bredd. |
| Raw | `1` | Representerar PDF-innehåll som det är, dvs. utan formatering. |
| Plain | `2` | Representerar PDF-textinnehåll med hänsyn till den relativa positioneringen av textfragment men utan att lägga till extra utrymmen. |
