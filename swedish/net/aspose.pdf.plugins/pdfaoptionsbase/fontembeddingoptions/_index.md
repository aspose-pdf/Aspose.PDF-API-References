## PdfAOptionsBase.FontEmbeddingOptions egenskap

Hämtar alternativen för att bearbeta typsnitt som inte kan bäddas in i dokumentet.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Egenskapsvärde

Alternativen för typsnittsbäddning.

## Kommentarer

PDF/A-standarden kräver att alla typsnitt måste bäddas in i dokumentet. Denna egenskap tillhandahåller alternativ för att hantera fall när det inte är möjligt att bädda in vissa typsnitt eftersom de saknas på den destination PC.
