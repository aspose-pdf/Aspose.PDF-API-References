## PdfFileInfo.PasswordType-egenskap

Returnerar typen av lösenord som angavs för att skapa PdfFileInfo-instansen. Se möjliga värden i `PasswordType`. Observera att pdf-dokumentet kan öppnas med både användar- (eller öppet) lösenord och ägar- (eller behörigheter, redigera) lösenord.

```csharp
public PasswordType PasswordType { get; }
```
