## PdfFileInfo.HasEditPassword-egenskap

Returnerar sant om lösenord krävs för att ändra behörigheter eller dokumentets säkerhetsegenskap. Observera att denna egenskap endast kan läsas om ett giltigt lösenord angavs i [`PdfFileInfo`](../) konstruktorn. Om PasswordType är Inaccessible (betyder att ett ogiltigt lösenord angavs) kommer läsning av denna egenskap att misslyckas med [`InvalidPasswordException`](../../../aspose.pdf/invalidpasswordexception/).

```csharp
public bool HasEditPassword { get; }
```
