## PdfViewer.AutoResize-egenskap

Hämtar eller ställer in ett bool-värde som indikerar om filen ska skrivas ut med optimerad storlek. Om false skriv ut sidan utan sidskalning. Om true skriv ut sidan med skalning för att passa det utskrivbara området.

```csharp
public bool AutoResize { get; set; }
```
