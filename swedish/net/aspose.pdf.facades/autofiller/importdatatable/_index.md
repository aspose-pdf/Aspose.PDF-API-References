## AutoFiller.ImportDataTable metod

Importerar data av typen DataTable. Varje kolumnnamn i dataTable måste vara detsamma som ett fältnamn i den mallade pdf:en, med skillnad på stora och små bokstäver.

```csharp
public void ImportDataTable(DataTable dataTable)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| dataTable | DataTable | Den dataTable som innehåller all data från databasen eller annan datakälla. |
