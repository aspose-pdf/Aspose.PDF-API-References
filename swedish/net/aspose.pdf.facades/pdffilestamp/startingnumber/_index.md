## PdfFileStamp.StartingNumber property

Hämtar eller ställer in startnummer för första sidan i inmatningsfilen. Nästa sidor kommer att numreras från detta värde. Till exempel om StartingNumber är inställt på 100 kommer dokumentsidorna att ha nummer 100, 101, 102...

```csharp
public int StartingNumber { get; set; }
```

### Exempel

```csharp
PdfFileStamp fileStamp = new PdfFileStamp("input.pdf", "output.pdf");
fileStamp.StartingNumber = 100;
fileStamp.AddPageNumber("Page #");
fileStamp.Close();
```
