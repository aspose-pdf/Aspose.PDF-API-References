## PdfFileStamp.OptimizeSize-egenskap

Hämtar eller ställer in optimeringsflagga. Lika resursströmmar i den resulterande filen slås samman till ett PDF-objekt om denna flagga är inställd. Detta gör att den resulterande filstorleken kan minskas men kan orsaka långsammare körning och större minneskrav. Standardvärde: false.

public bool OptimizeSize { get; set; }
