## PdfPageEditor.GetPageSize method

Returnerar sidstorleken för den angivna sidan.

```csharp
public PageSize GetPageSize(int page)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| page | Int32 | Sidindex. Dokumentsidorna är numrerade från 1. |

### Returvärde

Resultatet är en instans av PageSize. Använd egenskaperna Width och Height för det returnerade objektet för att få sidbredd och höjd.

### Exempel

Följande exempel visar hur man använder GetPageSize-metoden:

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
PageSize size = editor.GetPageSize(1);
Console.WriteLine("Size of 1st page : " + size.Width + " x " + size.Height);
```
