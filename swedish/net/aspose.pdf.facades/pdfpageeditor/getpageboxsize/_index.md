## PdfPageEditor.GetPageBoxSize method

Returnerar storleken på angiven ruta i dokumentet.

```csharp
public Rectangle GetPageBoxSize(int page, string pageBoxName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| page | Int32 | Sidindex. Dokumentsidorna är numrerade från 1. |
| pageBoxName | String | Boxtyps namn. Giltiga värden är: "art", "bleed", "crop", "media", "trim". |

### Returvärde

### Exempel

Följande exempel visar hur man får mediabox på första sidan:

```csharp

editor.BindPdf("sample.pdf");
System.Drawing.Rectangle rect = editor.GetBoxSize(1, "media");
```
