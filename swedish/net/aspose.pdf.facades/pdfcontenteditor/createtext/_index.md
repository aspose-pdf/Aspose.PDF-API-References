## PdfContentEditor.CreateText method

Skapar textkommentarer i PDF-dokument

```csharp
public void CreateText(Rectangle rect, string title, string contents, bool open, string icon, 
    int page)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| rect | Rectangle | Anteckningsrektangeln som definierar platsen för anteckningen på sidan. |
| title | String | Anteckningens titel. |
| contents | String | Innehållet i anteckningen. |
| open | Boolean | En flagga som anger om anteckningen initialt ska visas öppen. |
| icon | String | Namnet på en ikon kommer att användas för att visa anteckningen. Detta värde kan vara: "Kommentar", "Key", "Note", "Hjälp", "NewParagraph", "Paragraph", "Insert" |
| page | Int32 | Antalet originalsida där textanteckningen kommer att skapas. |

### Exempel

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateText(new System.Drawing.Rectangle(0, 0, 100, 100),
    "Welcome to Aspose", "You are welcome to Aspose!", true, "Key", 1);
editor.Save("example_out.pdf");
```
