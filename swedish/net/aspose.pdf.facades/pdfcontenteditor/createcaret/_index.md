## PdfContentEditor.CreateCaret method

Skapar fältanteckning.

```csharp
public void CreateCaret(int page, Rectangle annotRect, Rectangle caretRect, string symbol, 
    string annotContents, Color color)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| page | Int32 | Antalet originalsida där anteckningen kommer att skapas. |
| annotRect | Rectangle | Anteckningsrektangeln som definierar platsen för anteckningen på sidan. |
| caretRect | Rectangle | De faktiska gränserna för den underliggande careten. |
| symbol | String | En symbol kommer att förknippas med rutan. Värdet kan vara: "P" (stycke), "Inget". |
| annotContents | String | Innehållet i anteckningen. |
| color | Color | Färgen på anteckningen. |

### Exempel

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateCaret(1,
    new System.Drawing.Rectangle(50, 50, 100, 100),
    new System.Drawing.Rectangle(60, 60, 70, 70),
    "None", "Welcome to Aspose", System.Drawing.Color.Red);
editor.Save("example_out.pdf");
```
