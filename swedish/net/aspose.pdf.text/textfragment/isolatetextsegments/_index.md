## TextFragment.IsolateTextSegments metod

Hämtar [`TextSegment`](../../textsegment/)(s) som representerar angiven del av [`TextFragment`](../) text.

```csharp
public TextSegmentCollection IsolateTextSegments(int startIndex, int length)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| startIndex | Int32 | Position i texten från vilken nya [`TextSegment`](../../textsegment/)(s) kommer att börja. |
| length | Int32 | Längden på texten som kommer att isoleras i [`TextSegment`](../../textsegment/)(s). |

### Returvärde

[`TextSegmentCollection`](../../textsegmentcollection/) som innehåller textsegment som representerar textundersträng som börjar vid en angiven position och har en angiven längd.
