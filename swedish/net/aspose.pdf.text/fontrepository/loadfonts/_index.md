## FontRepository.LoadFonts metod

Laddar systeminstallerade typsnitt och standard Pdf-typsnitt. Denna metod är utformad för att snabba upp typsnittsprocessen. Som standard laddas typsnitt vid första begäran om något typsnitt. Användning av denna metod laddar system- och standard Pdf-typsnitt omedelbart innan något Pdf-dokument öppnas.

public static void LoadFonts()
