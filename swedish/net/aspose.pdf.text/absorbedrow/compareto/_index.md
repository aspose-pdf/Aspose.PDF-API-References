## AbsorbedRow.CompareTo metod

Jämför det aktuella AbsorbedRow-objektet med ett annat AbsorbedRow-objekt och returnerar ett heltal som indikerar om det aktuella objektet föregår, följer eller förekommer på samma position i sorteringsordningen som det andra objektet.

```csharp
public int CompareTo(AbsorbedRow other)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| other | AbsorbedRow | Ett AbsorbedRow-objekt att jämföra med detta objekt. |

### Returvärde

Ett värde som indikerar den relativa ordningen av de objekt som jämförs. Returvärdet har följande betydelser: -1: Detta objekt föregår det andra objektet i sorteringsordningen. 0: Detta objekt förekommer på samma position i sorteringsordningen som det andra objektet. 1: Detta objekt följer det andra objektet i sorteringsordningen.
