### Exempel

Exemplet visar hur man utför söktext flera gånger och utför textersättningar.

```csharp
// Öppna dokument
Document doc = new Document(@"D:\Tests\input.pdf");

// Skapa TextFragmentAbsorber-objekt för att hitta alla "hej" textförekomster
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello");

doc.Pages[1].Accept(absorber);
absorber.TextFragments[1].Text = "Hi";

// sök ett annat ord och ersätt det
absorber.Phrase = "world";

doc.Pages[1].Accept(absorber);
absorber.TextFragments[1].Text = "John";

// Spara dokument
doc.Save(@"D:\Tests\output.pdf");  
```
