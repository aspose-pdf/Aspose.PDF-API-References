## TextFragmentAbsorber.TextSearchOptions property

Hämtar eller ställer in sökalternativ. Alternativen möjliggör sökning med reguljära uttryck.

```csharp
public TextSearchOptions TextSearchOptions { get; set; }
```

### Exempel

Exemplet visar hur man utför söktext med reguljärt uttryck.

```csharp
// Öppna dokument
Document doc = new Document(@"D:\Tests\input.pdf");

// Skapa TextFragmentAbsorber-objekt
TextFragmentAbsorber absorber = new TextFragmentAbsorber();

// gör absorberaren att söka efter alla ord som börjar "h" och slutar "o" med hjälp av reguljära uttryck.
absorber.Phrase = @"h\w*?o";
absorber.TextSearchOptions = new TextSearchOptions(true);

// vi borde hitta ordet "hej" och ersätta det med "Hej"
doc.Pages[1].Accept(absorber);
absorber.TextFragments[1].Text = "Hi"; 

// Spara dokument
doc.Save(@"D:\Tests\output.pdf"); 
```
