## TextSearchOptions.SearchInAnnotations-egenskap

Hämtar eller ställer in värde som tillåter sökning efter text i Anmärkningar. true - text kommer att sökas i Anmärkningar. false - text i Anmärkningar kommer inte att tolkas av TextFragmentAbsorber.

```csharp
public bool SearchInAnnotations { get; set; }
```
