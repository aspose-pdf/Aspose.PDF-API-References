## TextSearchOptions.LogTextExtractionErrors-egenskap

Hämtar eller ställer in indikation att textutvinnings (avkodnings) fel kommer att loggas i text (fragment) absorbern. true - betyder att textutvinnings (avkodnings) fel kommer att loggas. Det kan minska prestanda. false (standard) - ingen felinloggning.

public bool LogTextExtractionErrors { get; set; }
