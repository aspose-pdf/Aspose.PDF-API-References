## TextSearchOptions.IgnoreShadowText-egenskap

Hämtar eller ställer in indikation att textfragment som representerar skuggan av normal text kommer att ignoreras under sökningen. true - betyder att skuggan text inte kommer att hittas (försök detta om textsökningen returnerar duplicerade fragment på nära positioner) false - betyder att skuggan text kommer att hittas liksom normal text (standardvärde)

```csharp
public bool IgnoreShadowText { get; set; }
```
