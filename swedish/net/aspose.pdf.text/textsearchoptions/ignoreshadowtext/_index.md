## TextSearchOptions.IgnoreShadowText property

Hämtar eller ställer in en indikation på att textfragment som representerar skugga av normal text kommer att ignoreras under sökning. true - betyder att skuggtext inte kommer att hittas (prova detta om textsökning returnerar dubblerade fragment på de nära positionerna) false - betyder att skuggan text kommer att hittas såväl som normal text (standardvärde)
