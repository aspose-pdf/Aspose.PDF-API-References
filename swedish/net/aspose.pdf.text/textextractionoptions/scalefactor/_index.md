## TextExtractionOptions.ScaleFactor-egenskap

Hämtar eller ställer in faktor som kommer att tillämpas för att skala teckenstorlek under extraktion i rent läge. Inställning av lägre värde leder till fler mellanrum i den extraherade texten. Standardvärde är 1 - ingen skalning; Att ställa in värdet till noll tillåter algoritmen att välja skalning automatiskt.

```csharp
public double ScaleFactor { get; set; }
```
