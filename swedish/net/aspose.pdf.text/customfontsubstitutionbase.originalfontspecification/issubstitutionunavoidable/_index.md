Får ett värde som indikerar att substitutionen är oundviklig.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

### Anmärkningar

Returnerar sant om ersättning begärdes på grund av frånvaro av det ursprungliga teckensnittet eller om det ursprungliga teckensnittet inte kan användas i samband med någon uppgift. Om användaren ignorerar flaggan och inte ersätter teckensnittet - standardproceduren för teckensnittsersättning utförs . Men det ger möjlighet för användaren att alternera standardtypsnittsersättningsprocedur och ställa in bättre teckensnitt till systemet. Returnerar falskt om det ursprungliga teckensnittet finns, giltigt, men det är tillåtet för användaren att ersätta det.
