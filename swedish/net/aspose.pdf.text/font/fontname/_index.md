### Exempel

Exemplet visar hur man söker efter text på första sidan och visar teckensnittsnamnet för en första textförekomst.

```csharp
// Öppna dokument
Document doc = new Document(@"D:\Tests\input.pdf");

// Skapa TextFragmentAbsorber-objekt för att hitta alla "hej världen" textförekomster
```
