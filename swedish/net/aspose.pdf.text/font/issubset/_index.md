## Font.IsSubset property

Hämtar eller ställer in ett värde som anger om teckensnittet är en delmängd.

```csharp
public bool IsSubset { get; set; }
```

### Exempel

Exemplet visar hur man söker efter text på första sidan och får värdet som anger om teckensnittet är en delmängd.

```csharp
// Öppna dokument
Document doc = new Document(@"D:\Tests\input.pdf");

// Skapa TextFragmentAbsorber-objekt för att hitta alla "hej världen" textförekomster
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");

// Acceptera absorbenten för första sidan
doc.Pages[1].Accept(absorber);

// Visa teckensnittets IsSubset-värde för första textförekomst
if(absorber.TextFragments[1].TextState.Font.IsSubset)
   Console.Out.WriteLine("the font is a subset");
```
