## TextAbsorber.ExtractionOptions property

Hämtar eller ställer in alternativ för textextraktion.

```csharp
public virtual TextExtractionOptions ExtractionOptions { get; set; }
```

### Anmärkningar

Tillåter att definiera textformateringsläge[`TextExtractionOptions`](../../textextractionoptions) under extraktion. Standardläget ärPure

### Exempel

Exemplet visar hur man ställer in rent textformateringsläge och utför textextrahering.

```csharp
// öppna dokument
Document doc = new Document(inFile);

// skapa TextAbsorber-objekt för att extrahera text med formatering
TextAbsorber absorber = new TextAbsorber();

// ställ in rent textformateringsläge
absorber.ExtractionOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure);

// acceptera absorbenten för alla dokumentsidor
doc.Pages.Accept(absorber);

// få den extraherade texten
string extractedText = absorber.Text;
```
