## OptimizationOptions.RemoveUnusedStreams-egenskap

Om denna flagga är inställd på true kontrolleras varje resurs för dess användning. Om resursen aldrig används tas den bort. Detta kan minska dokumentstorleken, till exempel när sidor har extraherats från dokumentet.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
