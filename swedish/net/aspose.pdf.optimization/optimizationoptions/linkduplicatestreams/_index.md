## OptimizationOptions.LinkDuplicateStreams-egenskap

Om denna flagga är inställd på true kommer resursströmmar att analyseras. Om dubblettströmmar hittas (det vill säga om strömarnas innehåll är lika), kommer dessa strömmar att lagras som ett objekt. Detta möjliggör att minska dokumentstorleken i vissa fall (till exempel när samma dokument har sammanfogats flera gånger).

```csharp
public bool LinkDuplicateStreams { get; set; }
```
