## OptimizationOptions.RemoveUnusedObjects-egenskap

Om denna flagga är inställd på true kommer alla dokumentobjekt att kontrolleras och oanvända objekt (dvs. objekt som inte har någon referens) tas bort från dokumentet.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
