## OptimizationOptions.LinkDuplcateStreams-egenskap

Om denna flagga är inställd på true kommer resursströmmar att analyseras. Om dubblettströmmar hittas (d.v.s. om strömarnas innehåll är lika), kommer dessa strömmar att lagras som ett objekt. Detta gör att dokumentstorleken kan minskas i vissa fall (till exempel när samma dokument har sammanfogats flera gånger).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
