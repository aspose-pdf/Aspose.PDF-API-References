Det **Aspose.Pdf.Security** namnområdet innehåller klasser som används för kryptering och digital signering.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [DsaAlgorithmInfo](./dsaalgorithminfo/) | Representerar en klass för information om DSA-signaturalgoritmen. |
| [EcdsaAlgorithmInfo](./ecdsaalgorithminfo/) | Representerar en klass för information om ECDSA-signaturalgoritmen. |
| [KeyedSignatureAlgorithmInfo](./keyedsignaturealgorithminfo/) | Representerar en klass för information om en nyckelbaserad signaturalgoritm. |
| [RsaAlgorithmInfo](./rsaalgorithminfo/) | Representerar en klass för information om RSA-signaturalgoritmen. |
| [SignatureAlgorithmInfo](./signaturealgorithminfo/) | Representerar en klass för information om en signaturalgoritm, inklusive dess typ, kryptografiska standard och hash-algoritm. |
| [SignatureLengthMismatchException](./signaturelengthmismatchexception/) | Representerar fel som uppstår under PDF-signering. Uppstår om [`SignHash`](../aspose.pdf.forms/signhash/) används för att signera ett dokument och den faktiska längden på signaturen är större än den som anges i alternativet [`DefaultSignatureLength`](../aspose.pdf.forms/signature/defaultsignaturelength/). |
| [TimestampAlgorithmInfo](./timestampalgorithminfo/) | Representerar en klass för information om tidsstämpelns signaturalgoritm. |
| [UnknownSignatureAlgorithmInfo](./unknownsignaturealgorithminfo/) | Representerar en klass för information om okända signaturalgoritmer. |
| [ValidationOptions](./validationoptions/) | Representerar alternativ för att validera en digital signatur i ett PDF-dokument. |
| [ValidationResult](./validationresult/) | Representerar resultatet av en valideringsprocess för ett certifikat. |
