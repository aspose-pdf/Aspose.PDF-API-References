## SideBySideComparisonOptions.AdditionalChangeMarks-egenskap

Hämta och ställ in egenskapen som avgör om ytterligare ändringsmarkörer visas. Om den är inställd visas ändringsmarkeringar som inte finns på den aktuella sidan men som finns på en annan sida. Om ändringen ligger mellan ord kan markeringen kanske inte vara exakt placerad i förhållande till mellanslagstecknet. Standardvärdet är `false`.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
