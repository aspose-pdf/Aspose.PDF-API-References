Det **Aspose.Pdf.Printing** namnområdet tillhandahåller klasser och funktioner för att hantera PDF-utskriftsoperationer, inklusive skrivarinställningar, sidinställningar och tillägg för att hantera utskriftsrelaterade konfigurationer.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Tillhandahåller data för [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/) händelsen. |
| [PageSettings](./pagesettings/) | Anger inställningar som gäller för en enda, utskriven sida. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Representerar tilläggsmetoder för [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Anger storleken på ett papper. |
| [PaperSizeExtensions](./papersizeextensions/) | Representerar tilläggsmetoder för [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Representerar standardpappersstorlekar. |
| [PaperSource](./papersource/) | Anger pappersfacket från vilket skrivaren hämtar papper. |
| [PaperSourceExtensions](./papersourceextensions/) | Representerar tilläggsmetoder för [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Tillhandahåller en uppsättning fördefinierade [`PaperSource`](../aspose.pdf.printing/papersource/) instanser som representerar vanliga papperskällor. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Tillhandahåller data för [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/) händelsen. |
| [PrinterResolution](./printerresolution/) | Representerar den upplösning som stöds av en skrivare. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Representerar tilläggsmetoder för [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Anger information om hur ett dokument skrivs ut, inklusive skrivaren som skriver ut det. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Representerar tilläggsmetoder för [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Tillhandahåller data för [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) och [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) händelserna i [`PdfViewer`](../aspose.pdf.facades/pdfviewer/) klassen. |
