## RadioButtonField.NoToggleToOff-egenskap

Hämtar eller ställer in flaggan som tillåter radioknappen att inte ha något valt värde. Om `true`, ska exakt en radioknapp alltid väljas; att välja den för närvarande valda knappen har ingen effekt. Om `false`, avmarkerar ett klick på den valda knappen den, vilket lämnar ingen knapp vald.

```csharp
public bool NoToggleToOff { get; set; }
```

## Kommentarer

Vissa PDF-läsare (inklusive Adobe Acrobat) kan ignorera `false`-tillståndet för flaggan.
