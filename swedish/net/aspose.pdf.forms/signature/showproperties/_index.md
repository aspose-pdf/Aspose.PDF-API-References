## Signature.ShowProperties egenskap

Tvinga att visa/dölja signatur egenskaper. Om ShowProperties är sant har signaturfältet ett fördefinierat format av utseende (strängar för att representera): ------------------------------------------- Digitally signed by {certificate subject} Datum: {signature.Date} Orsak: {signature.Reason} Plats: {signature.Location} ------------------------------------------- där {X} är en platshållare för X-värdet. Signaturen kan också ha en bild, i detta fall placeras de listade strängarna över bilden. ShowProperties är sant som standard.

```csharp
public bool ShowProperties { get; set; }
```
