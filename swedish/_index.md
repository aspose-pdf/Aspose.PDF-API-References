## Aspose.PDF för .NET

{{% alert color="primary" %}} 

![Aspose.PDF för .NET-produktlogotypen](../aspose_pdf-for-net.png)

Aspose.PDF är en .NET-komponent byggd för att göra det möjligt för utvecklare att skapa PDF-dokument, vare sig de är enkla eller komplexa, i farten programmatiskt. Aspose.PDF för .NET tillåter utvecklare att infoga tabeller, grafer, bilder, hyperlänkar, anpassade typsnitt - och mer - i PDF-dokument. Dessutom är det också möjligt att komprimera PDF-dokument. Aspose.PDF för .NET tillhandahåller utmärkta säkerhetsfunktioner för att utveckla säkra PDF-dokument. Och den mest distinkta egenskapen hos Aspose.PDF för .NET är att den stöder skapandet av PDF-dokument genom både ett API och från XML-mallar.

## Aspose.PDF för C++

{{% alert color="primary" %}} 

![Aspose.PDF för C++ produktlogotyp](../aspose_pdf-for-cpp.png)

Aspose.PDF för C++ är ett inbyggt C++-bibliotek som gör det möjligt för utvecklare att skapa, läsa och manipulera PDF-dokument utan att använda Adobe Acrobat. Aspose.PDF för C++ låter utvecklare skapa formulär, lägga till/redigera text, manipulera PDF-sidor, lägga till kommentarer, hantera anpassade typsnitt och mycket mer. Här i dokumentationsdelen av Aspose.PDF för C++ kan du utforska alla funktioner i API:t tillsammans med de grundläggande exemplen för att demonstrera implementeringen av dessa funktioner i din C++-applikation.
