### BaseURL {#BaseURL}
```
public static final int BaseURL
```


xmp:BaseURL 属性。文档内容中相对 URL 的基本 URL。如果此文档包含 Internet 链接，并且这些链接是相对的，则 /// 它们是相对于此基本 URL 的。此属性为工具解释嵌入式相对 URL 提供了一种标准方式。 /// Web 创作工具应该根据他们对 URL 将被解释的位置的概念来设置值
