这**com.aspose.pdf**是 Aspose.PDF for Java 库的所有类的根包，这些类直接在其中**Document**或间接通过几个子包。

1) 几乎所有类型的注释文档的类都可以有。

2) 描述表单（标准、静态、动态）和各种类型的字段（如文本框、列表框、单选按钮等）的类。

3) 这些类有助于处理文档的逻辑结构，将文档组织成章节或部分，或识别特殊元素，如图形、表格和脚注。

4) 允许提取文本、添加文本、操作文档的现有文本的类。它还包含允许提取、替换、替换文档字体的类。

5) 用于处理文档的各种类型的操作、目的地和其他功能的类，传统上称为交互式提供意味着用户可以与之交互。

6) 允许交替文本提取和操作功能行为的类。包 com.aspose.pdf.text 有额外的类。

7) DOM 的额外支持类。

8) PDF/A 转换选项的类。
