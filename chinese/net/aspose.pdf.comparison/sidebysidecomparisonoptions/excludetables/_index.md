## SideBySideComparisonOptions.ExcludeTables 属性

获取和设置决定是否排除表格进行比较的选项。此选项不能与 [`ComparisonArea1`](../comparisonarea1/) 和 [`ComparisonArea2`](../comparisonarea2/) 一起设置。默认值为 `false`。

```csharp
public bool ExcludeTables { get; set; }
```
