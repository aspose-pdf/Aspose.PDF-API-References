## SideBySideComparisonOptions.AdditionalChangeMarks 属性

获取和设置决定是否显示额外更改标记的属性。如果设置，则显示不在当前页面但存在于其他页面的更改标记。如果更改位于单词之间，标记可能不会相对于空格字符精确定位。默认值为 `false`。

```csharp
public bool AdditionalChangeMarks { get; set; }
```
