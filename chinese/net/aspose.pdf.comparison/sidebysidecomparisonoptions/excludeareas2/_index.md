## SideBySideComparisonOptions.ExcludeAreas2 属性

获取和设置排除区域。用于比较方法中第二个页面或文档。该选项可以与 [`ExcludeTables`](../excludetables/) 一起设置。该选项不能与 [`ComparisonArea2`](../comparisonarea2/) 选项同时设置。

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
