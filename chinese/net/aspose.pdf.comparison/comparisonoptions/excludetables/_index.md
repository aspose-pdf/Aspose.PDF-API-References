## ComparisonOptions.ExcludeTables 属性

获取和设置决定是否排除表格进行比较的选项。此选项不能与 [`ExtractionArea`](../extractionarea/) 选项一起设置。默认值为 `false`。

```csharp
public bool ExcludeTables { get; set; }
```
