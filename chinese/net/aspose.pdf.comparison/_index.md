## 类

| 类 | 描述 |
| --- | --- |
| [ComparisonOptions](./comparisonoptions/) | 表示 PDF 文档比较选项类。 |
| [DiffOperation](./diffoperation/) | 表示差异操作的类。 |
| [DocumentComparisonStatistics](./documentcomparisonstatistics/) | 表示文档比较统计类。 |
| [GraphicalPdfComparer](./graphicalpdfcomparer/) | 表示用于图形比较 PDF 文档的类。应使用此类查找小的变化，主要是图形性质的变化。要比较文本内容的变化，请使用其他 PDF 比较类。 |
| [HtmlDiffOutputGenerator](./htmldiffoutputgenerator/) | 表示生成文本差异的 HTML 表示的类。删除的换行符由段落标记表示。 |
| [ImagesDifference](./imagesdifference/) | 表示比较两个 PDF 页面结果的类。 |
| [JsonDiffOutputGenerator](./jsondiffoutputgenerator/) | 表示以 JSON 格式显示比较 PDF 文档或页面结果的类。 |
| [MarkdownDiffOutputGenerator](./markdowndiffoutputgenerator/) | 表示生成文本差异的 Markdown 表示的类。由于 Markdown 语法，无法显示空白字符的变化。选择变化时会在格式周围添加空白字符，否则 Markdown 查看器将无法正确显示文本。删除的换行符由 - 段落标记表示。 |
| [OutputTextStyle](./outputtextstyle/) | 表示标记文本变化的样式集类。 |
| [PdfOutputGenerator](./pdfoutputgenerator/) | 表示生成文本差异的 PDF 表示的类。 |
| [SideBySideComparisonOptions](./sidebysidecomparisonoptions/) | 表示用于并排输出比较文档的选项类。 |
| [SideBySidePdfComparer](./sidebysidepdfcomparer/) |  |
| [TextItemComparisonStatistics](./textitemcomparisonstatistics/) | 表示文本比较统计类。 |
| [TextPdfComparer](./textpdfcomparer/) | 表示比较两个 PDF 页面或 PDF 文档的类。 |
| [TextStyle](./textstyle/) | 表示文本样式类。 |
