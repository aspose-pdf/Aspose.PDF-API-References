## ElementList 索引器

获取给定索引处的元素。

```csharp
public virtual Element ElementOf[int index] { get; }
```

| 参数 | 描述 |
| --- | --- |
| index | 元素列表中的索引。 |

### 返回值

集合中具有指定索引的元素。如果索引大于或等于列表中的元素数量，则返回 null。
