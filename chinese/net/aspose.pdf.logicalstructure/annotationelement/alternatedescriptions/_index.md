## AnnotationElement.AlternateDescriptions 属性

获取或设置注释的备用描述。将为注释显示的文本，或者，如果此类型的注释不显示文本，则为注释内容的可读形式的备用描述。

```csharp
public string AlternateDescriptions { get; set; }
```
