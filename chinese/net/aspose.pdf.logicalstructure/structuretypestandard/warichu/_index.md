## StructureTypeStandard.Warichu 字段

(Warichu; PDF 1.5) 在包含文本行的高度内以较小的文本大小格式化的注释或注解，并放置在其所指的基础文本后面（行内）分为两行。Warichu 元素还可以包含 WT 和 WP 元素。

(Warichu) 包裹整个 warichu 组件的包装器。它可以包含由 WP、WT 和 WP 组成的三元素组。Warichu 元素（及其内容元素）可以根据日本工业标准 (JIS) X 4051-1995 中描述的 warichu 换行规则跨多行包装。

```csharp
public static readonly StructureTypeStandard Warichu;
```
