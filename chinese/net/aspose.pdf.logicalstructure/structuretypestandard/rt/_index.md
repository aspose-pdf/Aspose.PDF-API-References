## StructureTypeStandard.RT 字段

（Ruby 注释文本）应放置在 ruby 基础文本旁边的小号文本。它可以包含文本、其他内联元素或两者的混合。它可以具有 RubyAlign 和 RubyPosition 属性。

```csharp
public static readonly StructureTypeStandard RT;
```
