## StructureTypeStandard.Ruby 字段

（Ruby；PDF 1.5）旁注（注释）以较小的文本大小书写，并放置在其所指的基础文本旁边。Ruby 元素还可以包含 RB、RT 和 RP 元素。

（Ruby）整个 ruby 组件的包装器。它应包含一个 RB 元素，后跟一个 RT 元素或一个由 RP、RT 和 RP 组成的三元素组。Ruby 元素及其内容元素不得跨多行断开。

```csharp
public static readonly StructureTypeStandard Ruby;
```
