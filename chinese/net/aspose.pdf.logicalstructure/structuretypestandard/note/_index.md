## StructureTypeStandard.Note 字段

（注释）是指在文档主体中引用的解释性文本项，例如脚注或尾注。它可以作为子项具有标签（结构类型 Lbl）。该注释可以作为引用它的主体文本中的结构元素的子项包含，或者可以包含在其他地方（例如在尾注部分）并通过引用（结构类型 Reference）进行访问。

标记 PDF 并未规定脚注在页面内容顺序中的位置。它们可以是内联的或位于页面末尾，由符合要求的作者自行决定。

```csharp
public static readonly StructureTypeStandard Note;
```
