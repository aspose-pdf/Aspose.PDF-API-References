## StructureTypeStandard.Link 字段

(链接) ILSE 内容的一部分与相应的链接注释或注释之间的关联。它的子项应该是一个或多个内容项或子 ILSE，以及一个或多个对象引用，标识相关的链接注释。

```csharp
public static readonly StructureTypeStandard Link;
```
