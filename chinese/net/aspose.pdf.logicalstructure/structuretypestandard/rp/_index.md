## StructureTypeStandard.RP 字段

（Ruby 标点符号）包围 ruby 注释文本的标点符号。仅在 ruby 注释无法以 ruby 风格正确格式化时使用，而是以普通注释格式化，或当它被格式化为 warichu 时使用。它包含文本（通常是一个左括号或右括号或类似的括号字符）。

```csharp
public static readonly StructureTypeStandard RP;
```
