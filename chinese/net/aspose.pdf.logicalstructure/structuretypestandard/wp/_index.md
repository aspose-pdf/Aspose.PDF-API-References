## StructureTypeStandard.WP 字段

(Warichu 标点符号) 包围 WT 文本的标点符号。它包含文本（通常是一个单独的左括号或右括号或类似的括号字符）。根据 JIS X 4051-1995，包围 warichu 的括号可以根据格式化程序的自由裁量权转换为一个宽度（名义上为 1/4 EM）的空格。

```csharp
public static readonly StructureTypeStandard WP;
```
