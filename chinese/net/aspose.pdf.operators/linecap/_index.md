## LineCap 枚举

线帽样式应指定在开放子路径（和虚线，如果有）在描边时使用的形状。

```csharp
public enum LineCap
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| ButtCap | `0` | 平头帽。描边将在路径的端点处被平整切断。路径的末端不会有突起。 |
| RoundCap | `1` | 圆头帽。将在端点周围绘制一个直径等于线宽的半圆弧，并将其填充。 |
| SquareCap | `2` | 突出的方头帽。描边将在路径的端点处延续，延续的距离等于线宽的一半，并将被平整切断。 |
