## XfaParserOptions.EmulateRequierdGroups 属性

如果此属性为真，则将为所需的 Xfa "排除组" 绘制额外的红色矩形。引入此属性是因为在将 Xfa 表单表示转换为标准时，缺少排除组的类似物。默认值为假。

```csharp
public bool EmulateRequierdGroups { get; set; }
```
