## OptimizationOptions.RemoveUnusedObjects 属性

如果此标志设置为 true，则将检查所有文档对象，并从文档中删除未使用的对象（即没有任何引用的对象）。

```csharp
public bool RemoveUnusedObjects { get; set; }
```
