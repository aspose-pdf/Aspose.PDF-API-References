## OptimizationOptions.RemoveUnusedStreams 属性

如果此标志设置为 true，则检查每个资源的使用情况。如果资源从未使用，则将其移除。这可能会减少文档大小，例如当页面从文档中提取时。

```csharp
public bool RemoveUnusedStreams { get; set; }
```
