## ImageCompressionOptions.ResizeImages 属性

如果此标志设置为 true 且 CompressImages 为 true，则如果图像分辨率大于指定的 MaxResolution 参数，图像将被调整大小。

```csharp
public bool ResizeImages { get; set; }
```
