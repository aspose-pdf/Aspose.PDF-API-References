## MarkupAnnotation.SetMarkedState 方法

为注释设置标记和未标记状态。请注意，状态存储在其他具有 state 和 statemodel 键的文本注释中。

```csharp
public void SetMarkedState(bool marked)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| marked | 布尔值 | 如果设置标记状态则为真，如果设置未标记状态则为假。 |
