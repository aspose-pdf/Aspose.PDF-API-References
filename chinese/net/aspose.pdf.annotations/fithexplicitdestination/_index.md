## FitHExplicitDestination class

表示显式目标，该目标显示页面，垂直坐标 top 位于窗口的顶部边缘，页面内容放大到刚好适合窗口的整个宽度。top 的空值指定该参数的当前值保持不变。

public sealed class FitHExplicitDestination : ExplicitDestination
