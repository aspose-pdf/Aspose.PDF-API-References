## PrinterMarksKind 枚举

指定要添加到文档中的打印机标记类型。

```csharp
[Flags]
public enum PrinterMarksKind
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 指定不添加打印机标记。 |
| TrimMarks | `1` | 指定添加裁剪标记。 |
| BleedMarks | `2` | 指定添加出血标记。 |
| RegistrationMarks | `4` | 指定添加注册标记。 |
| ColorBars | `8` | 指定添加色条。 |
| PageInformation | `10` | 指定添加页面信息。 |
| All | `1F` | 指定添加所有类型的打印机标记。 |

## 备注

此枚举具有 FlagsAttribute 属性，允许其成员值的按位组合。
