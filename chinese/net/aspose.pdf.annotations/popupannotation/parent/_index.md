## PopupAnnotation.Parent 属性

获取或设置与此弹出注释关联的父注释。如果此条目存在，父注释的 Contents、M、C 和 T 条目将覆盖弹出注释本身的条目。

```csharp
public Annotation Parent { get; set; }
```
