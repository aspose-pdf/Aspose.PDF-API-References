## HighlightingMode 枚举

枚举注释的高亮模式，在其活动区域内按下或按住鼠标按钮时使用的视觉效果。

```csharp
public enum HighlightingMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 无高亮。 |
| Invert | `1` | 反转注释矩形的内容。 |
| Outline | `2` | 反转注释的边框。 |
| Push | `3` | 显示注释的下按外观（如果有）。如果没有定义下按外观，则偏移注释矩形的内容，使其看起来像是被推入页面表面以下。 |
| Toggle | `4` | 与 Push 相同（首选）。 |
