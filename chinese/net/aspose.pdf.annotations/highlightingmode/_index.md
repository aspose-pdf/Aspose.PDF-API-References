## HighlightingMode enumeration

枚举注解的高亮模式，即在其活动区域内按下或按住鼠标按钮时使用的视觉效果。

```csharp
public enum HighlightingMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 没有高亮显示。 |
| Invert | `1` | 反转注释矩形的内容。 |
| Outline | `2` | 反转注释的边框。 |
