## XYZExplicitDestination.CreateDestination 方法

创建目标到指定位置的页面，考虑页面旋转（如有必要）。

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| page | Page | 目标页面。 |
| left | Double | 页面上的左侧位置。 |
| top | Double | 页面上的顶部位置。 |
| zoom | Double | 缩放因子（0 为默认）。 |
| considerRotation | Boolean | 如果为 true，位置将根据页面旋转重新计算。 |

### 返回值

目标对象。
