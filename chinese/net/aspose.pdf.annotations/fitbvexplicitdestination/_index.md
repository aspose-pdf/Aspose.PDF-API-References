## FitBVExplicitDestination class

表示显式目标，该目标显示页面的水平坐标左侧定位在窗口的左边缘，并且页面的内容放大到足以使其边界框的整个高度适合窗口内。左侧的空值指定该参数的当前值应保持不变。

public sealed class FitBVExplicitDestination : ExplicitDestination
