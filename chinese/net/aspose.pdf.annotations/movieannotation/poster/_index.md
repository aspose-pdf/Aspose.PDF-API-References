## MovieAnnotation.Poster 属性

获取或设置一个标志或流，指定是否以及如何显示代表电影的海报图像。如果为 true，则海报图像将从电影文件中检索；如果为 false，则不显示海报。

```csharp
public bool Poster { get; set; }
```
