## FitExplicitDestination class

表示明确的目标，显示页面及其内容的放大程度刚好足以使整个页面在窗口中水平和垂直方向上都适合。如果所需的水平和垂直放大因子不同，则使用两个中的较小者，将页面在另一个维度中居中。

```csharp
public sealed class FitExplicitDestination : ExplicitDestination
```
