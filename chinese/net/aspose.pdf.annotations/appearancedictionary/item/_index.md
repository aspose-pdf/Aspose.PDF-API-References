## AppearanceDictionary 索引器

表示获取外观流的便捷形式。

```csharp
public XForm this[string key] { get; set; }
```

| 参数 | 描述 |
| --- | --- |
| key | 表示外观流的路径。如果外观字典有子字典，则路径必须包含 2 部分 ([`Keys`](../keys/))，否则路径只有一部分。 |

### 返回值

与给定键对应的 XForm 对象（外观流）。
