## Measure.NumberFormat.ForceDenominator 属性

如果 FractionDisplayment 为 ShowAsFraction，则此值决定分数是否可以被简化。如果值为 true，则分数可能无法被简化。

```csharp
public bool ForceDenominator { get; set; }
```
