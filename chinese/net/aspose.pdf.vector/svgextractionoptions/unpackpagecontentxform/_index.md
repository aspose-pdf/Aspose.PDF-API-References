## SvgExtractionOptions.UnpackPageContentXForm 属性

获取和设置一个标志，该标志确定页面上找到的 XFrom 是否应该被解包。XFrom 元素可以出现在不同的 SVG 文件中。只有通过页面内容的 Do 语句渲染的 XForms 才会被解包。嵌套的 XForms 不会被解包。

```csharp
public bool UnpackPageContentXForm { get; set; }
```
