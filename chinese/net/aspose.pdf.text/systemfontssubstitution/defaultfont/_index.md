## SystemFontsSubstitution.DefaultFont 属性

获取或设置默认替代字体。当没有找到其他有效的替代字体，但初始字体属于目标替代类别时使用该字体 ([`FontCategories`](../fontcategories/))。

```csharp
public Font DefaultFont { get; set; }
```
