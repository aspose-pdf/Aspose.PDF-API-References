## TextFragmentAbsorber.HasErrors 属性

值指示在文本提取过程中是否发现错误。仅当 TextSearchOptions.LogTextExtractionErrors = true 时才会执行错误搜索；这可能会降低性能。

```csharp
public bool HasErrors { get; }
```
