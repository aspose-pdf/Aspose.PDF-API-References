## AppendLine(string, float) {#appendline_6}

附加文本行。

```csharp
public void AppendLine(string line, float lineSpacing)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| line | String | 新行的文本。 |
| lineSpacing | Single | 额外间距（0.0 为默认值，对应于默认文本行高度）。该间距值会添加到特定行的默认行间距，因此您可以指定 12.0 以在用 12pt 字体绘制的文本行后获得空行。 |
