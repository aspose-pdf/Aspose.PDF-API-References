## ParagraphAbsorberOptions.SectionUnbreakingVerticalOverride 属性

获取或设置不会被视为实际垂直节段断裂的“零填充水平”区域的最大长度（作为页面宽度/高度的一个分数）。默认值为 0.005。

```csharp
public double SectionUnbreakingVerticalOverride { get; set; }
```
