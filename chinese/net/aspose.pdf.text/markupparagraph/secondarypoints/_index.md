## MarkupParagraph.SecondaryPoints 属性

次要多边形的点描述段落的延续。如果段落在下一列或页面中继续，则不会为 null。起始点是段落的左下角。下一个点按逆时针顺序排列。

```csharp
public List<Point[]> SecondaryPoints { get; }
```
