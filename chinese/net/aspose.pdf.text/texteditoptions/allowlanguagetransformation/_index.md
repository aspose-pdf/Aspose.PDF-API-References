## TextEditOptions.AllowLanguageTransformation 属性

获取或设置允许在添加或编辑文本时使用语言转换的值。 true - 如果需要，将应用语言转换（默认值）。 false - 不会应用语言转换。

```csharp
public bool AllowLanguageTransformation { get; set; }
```
