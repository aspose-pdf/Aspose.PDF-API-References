## Font.IsAccessible property

获取系统中是否存在（安装）字体。

```csharp
public bool IsAccessible { get; }
```

### 评论

某些操作不适用于系统中找不到的字体。

### 例子

该示例演示如何在第一页上搜索文本并获取指示字体是否为安装在系统中。
