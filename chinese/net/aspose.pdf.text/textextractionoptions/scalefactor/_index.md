## TextExtractionOptions.ScaleFactor 属性

获取或设置在纯模式下提取时应用于缩放字体大小的因子。设置较小的值会导致提取的文本中出现更多空格。默认值为 1 - 不缩放；将值设置为零允许算法自动选择缩放。

```csharp
public double ScaleFactor { get; set; }
```
