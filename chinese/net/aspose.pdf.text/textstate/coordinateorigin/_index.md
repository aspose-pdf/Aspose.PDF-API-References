## TextState.CoordinateOrigin 属性

获取或设置文本的 CoordinateOrigin。如果 CoordinateOrigin 是 Descender，则文本的 Y 坐标对应于字体的最低点。如果 CoordinateOrigin 是 BaseLine，则文本的 Y 坐标对应于字体的基线。默认值是 Descender。如果字体的 Descent 值过大，文本可能会比其他字体渲染得更高。在这种情况下，可以选择 CoordinateOrigin BaseLine 以获得更好的文本渲染。

```csharp
public virtual CoordinateOrigin CoordinateOrigin { get; set; }
```
