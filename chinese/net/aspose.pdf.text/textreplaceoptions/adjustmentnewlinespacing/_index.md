## TextReplaceOptions.AdjustmentNewLineSpacing 属性

获取或设置在强制替换调整以创建新文本行时使用的行间距值。期望的值是被替换文本的字体大小的倍数。默认值为 1.2。

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
