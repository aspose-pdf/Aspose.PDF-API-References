## TextAbsorber.Errors 属性

[`TextExtractionError`](../../textextractionerror/) 对象的列表。它包含在文本提取过程中发现的错误的信息。仅在 TextSearchOptions.LogTextExtractionErrors = true 时才会执行错误搜索；这可能会降低性能。

```csharp
public List<TextExtractionError> Errors { get; }
```
