## TextAbsorber.TextSearchOptions 属性

获取或设置文本搜索选项。

```csharp
public virtual TextSearchOptions TextSearchOptions { get; set; }
```

## 备注

允许定义一个矩形，该矩形限定提取的文本。默认情况下，矩形为空。这意味着页面边界仅定义文本提取区域。
