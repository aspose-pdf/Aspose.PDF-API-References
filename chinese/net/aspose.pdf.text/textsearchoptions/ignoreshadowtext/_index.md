## TextSearchOptions.IgnoreShadowText 属性

获取或设置指示在搜索过程中将忽略表示正常文本阴影的文本片段的标志。 true - 表示将不会找到阴影文本（如果文本搜索返回接近位置的重复片段，请尝试此选项）false - 表示将找到阴影文本以及正常文本（默认值）

```csharp
public bool IgnoreShadowText { get; set; }
```
