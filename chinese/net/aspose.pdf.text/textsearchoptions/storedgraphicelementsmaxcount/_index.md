## TextSearchOptions.StoredGraphicElementsMaxCount 属性

获取或设置限制在指定数量的元素上搜索与文本相关的图形（下划线、背景等）的值。默认值为 250。在性能问题的情况下设置较小的值，在某些图形元素未找到的情况下尝试较大的值。

```csharp
public int StoredGraphicElementsMaxCount { get; set; }
```
