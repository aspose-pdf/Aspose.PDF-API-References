## TextSearchOptions.LogTextExtractionErrors 属性

获取或设置指示文本提取（解码）错误将在文本（片段）吸收器中记录。 true - 表示文本提取（解码）错误将被记录。这可能会降低性能。 false（默认） - 不记录错误。

```csharp
public bool LogTextExtractionErrors { get; set; }
```
