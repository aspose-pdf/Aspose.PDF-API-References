## TextSearchOptions.SearchInAnnotations 属性

获取或设置允许在注释中搜索文本的值。 true - 文本将在注释中被搜索。 false - 注释中的文本将不会被 TextFragmentAbsorber 解析。

```csharp
public bool SearchInAnnotations { get; set; }
```
