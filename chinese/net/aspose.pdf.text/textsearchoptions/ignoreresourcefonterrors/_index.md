## TextSearchOptions.IgnoreResourceFontErrors 属性

获取或设置指示文本（片段）吸收器将忽略与缺少字体相关的错误。 true - 表示将忽略缺少字体的错误。处理过程中将跳过引用不正确资源的文本段。 false（默认） - 缺少字体错误将通过抛出异常终止处理。

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
