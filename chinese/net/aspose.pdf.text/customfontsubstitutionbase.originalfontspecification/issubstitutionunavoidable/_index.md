## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable property

获取指示替换不可避免的值。

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

### 评论

如果由于缺少原始字体而请求替换，或者在某些任务的上下文中无法使用原始字体，则返回 true。 如果用户忽略标志并且不替换字体 - 执行默认字体替换过程. 但它为用户提供了替代标准字体替换过程并为系统设置更好字体的机会。 如果存在原始字体，则返回 false，有效，但允许用户替换它。
