## FontRepository.LoadFonts 方法

加载系统安装的字体和标准 Pdf 字体。此方法旨在加快字体加载过程。默认情况下，字体在首次请求任何字体时加载。使用此方法会在打开任何 Pdf 文档之前立即加载系统和标准 Pdf 字体。

```csharp
public static void LoadFonts()
```
