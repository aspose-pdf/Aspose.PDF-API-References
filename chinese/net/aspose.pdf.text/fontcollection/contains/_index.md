## Contains(Font) {#contains}

确定集合是否包含特定值。

```csharp
public bool Contains(Font item)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| item | Font | 要在集合中查找的对象 |

### 返回值

如果在集合中找到 item，则返回 true；否则返回 false。
