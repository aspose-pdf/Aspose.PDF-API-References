## EmphasisStyle 枚举

定义了可用于强调和强强调的可用序列化样式。有关规范，请参见 CommonMark - 强调和强强调。

```csharp
public enum EmphasisStyle
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Asterisk | `0` | 使用 '*' 和 '**' 来表示（强烈）强调的文本。 |
| Underscore | `1` | 使用 '_' 和 '__' 来表示（强烈）强调的文本。 |
