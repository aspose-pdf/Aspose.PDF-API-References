## RunCreateRequest.TopP 属性

获取或设置一种替代温度采样的方法，称为核采样，其中模型考虑具有 top_p 概率质量的令牌结果。因此，0.1 意味着仅考虑组成前 10% 概率质量的令牌。我们通常建议更改此值或温度，但不建议同时更改两者。

```csharp
public double? TopP { get; set; }
```
