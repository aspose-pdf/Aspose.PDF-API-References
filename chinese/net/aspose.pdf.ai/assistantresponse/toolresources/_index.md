## AssistantResponse.ToolResources 属性

获取或设置助手工具使用的一组资源。这些资源特定于工具的类型。例如，code_interpreter 工具需要文件 ID 列表，而 file_search 工具需要向量存储 ID 列表。

public ToolResources ToolResources { get; set; }
