## ThreadResponse.ToolResources 属性

获取或设置在此线程中提供给助手工具的一组资源。这些资源特定于工具的类型。例如，code_interpreter 工具需要文件 ID 列表，而 file_search 工具需要向量存储 ID 列表。

```csharp
public ToolResources ToolResources { get; set; }
```
