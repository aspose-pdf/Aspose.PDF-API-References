## LlamaChatCompletionRequest.Temperature 属性

设置或获取使用的采样温度，范围在 0 到 2 之间。较高的值如 0.8 会使输出更随机，而较低的值如 0.2 会使其更集中和确定。默认值为 1。

```csharp
public double? Temperature { get; set; }
```
