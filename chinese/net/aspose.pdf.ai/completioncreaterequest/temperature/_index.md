## CompletionCreateRequest.Temperature 属性

获取或设置要使用的采样温度，范围在 0 到 2 之间。较高的值如 0.8 会使输出更随机，而较低的值如 0.2 会使其更集中和确定。

```csharp
public double? Temperature { get; set; }
```
