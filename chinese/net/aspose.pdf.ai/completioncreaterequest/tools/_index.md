## CompletionCreateRequest.Tools 属性

获取或设置模型可能调用的工具列表。目前，仅支持函数作为工具。使用此功能提供模型可能生成 JSON 输入的函数列表。最多支持 128 个函数。

```csharp
public List<Tool> Tools { get; set; }
```
