## CompletionCreateRequest.FrequencyPenalty 属性

获取或设置一个介于 -2.0 和 2.0 之间的数字。正值根据新令牌在文本中的现有频率对其进行惩罚，从而降低模型逐字重复相同行的可能性。

```csharp
public double? FrequencyPenalty { get; set; }
```
