## OpenAIImageDescriptionCopilotOptions.ImageDetail 属性

如果用户指定，则获取或设置图像的详细级别。“low”使用更少的令牌，您可以选择使用“high”来获得高分辨率。如果未设置，则默认为“auto”。

public string ImageDetail { get; set; }
