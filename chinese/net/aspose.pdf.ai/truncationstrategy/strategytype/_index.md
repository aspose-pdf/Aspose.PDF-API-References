## TruncationStrategy.StrategyType 属性

获取或设置用于线程的截断策略。默认值为 auto。如果设置为 last_messages，线程将被截断为线程中最近的 n 条消息。当设置为 auto 时，线程中间的消息将被丢弃，以适应模型的上下文长度 max_prompt_tokens。

```csharp
public string StrategyType { get; set; }
```
