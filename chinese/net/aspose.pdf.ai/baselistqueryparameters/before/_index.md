## BaseListQueryParameters.Before 属性

获取或设置用于分页的游标。before 是一个对象 ID，定义您在列表中的位置。例如，如果您发出列表请求并收到 100 个对象，以 obj_foo 结尾，则您的后续调用可以包括 before=obj_foo 以获取列表的上一页。

```csharp
public string Before { get; set; }
```
