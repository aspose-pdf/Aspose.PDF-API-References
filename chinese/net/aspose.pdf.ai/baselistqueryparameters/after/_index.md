## BaseListQueryParameters.After 属性

获取或设置用于分页的游标。after 是一个对象 ID，定义您在列表中的位置。例如，如果您发出列表请求并接收到 100 个对象，最后一个是 obj_foo，您的后续调用可以包括 after=obj_foo 以获取列表的下一页。

```csharp
public string After { get; set; }
```
