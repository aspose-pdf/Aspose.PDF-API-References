## ValidationOptions.RequestTimeout 属性

获取或设置验证过程中与网络相关操作的超时时间（以毫秒为单位）。RequestTimeout 属性定义系统在访问在线资源（如撤销状态或 OCSP 服务器）时应等待网络响应的最长时间。

```csharp
public int RequestTimeout { get; set; }
```
