## SignatureLengthMismatchException 类

表示在 PDF 签名过程中发生的错误。如果使用 [`SignHash`](../../aspose.pdf.forms/signhash/) 签署文档，并且签名的实际长度大于 [`DefaultSignatureLength`](../../aspose.pdf.forms/signature/defaultsignaturelength/) 选项中指定的长度，则会发生此错误。

```csharp
public class SignatureLengthMismatchException : PdfException
```
