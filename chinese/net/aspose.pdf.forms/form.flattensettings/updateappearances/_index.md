## Form.FlattenSettings.UpdateAppearances 属性

如果设置，所有字段外观将在扁平化之前重新生成。如果字段被错误地扁平化，此选项可能会有所帮助。此选项可能会降低性能。默认设置为 false。

```csharp
public bool UpdateAppearances { get; set; }
```
