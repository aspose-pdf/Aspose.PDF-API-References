## Form.RemovePermission 属性

如果此属性为真，则 "Perms" 字典将在将动态文档转换为标准后从 pdf 文档中删除。 "Perms" 字典可以包含干扰在 Adobe Acrobat 阅读器中显示必填字段选择的规则。默认情况下为假。

```csharp
public bool RemovePermission { get; set; }
```
