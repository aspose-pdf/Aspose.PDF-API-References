## Form.AutoRecalculate 属性

如果设置为真，当任何字段更改时，所有表单字段将被重新计算。默认值为真。为了在填写大量计算字段的表单时提高性能，请设置为假。

```csharp
public bool AutoRecalculate { get; set; }
```
