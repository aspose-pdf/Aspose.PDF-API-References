## Field.AlternateName 属性

获取或设置字段的备用名称（备用字段名称应在用户界面中标识字段的地方替代实际字段名称）。备用名称在 Adobe Acrobat 中用作字段工具提示。

```csharp
public string AlternateName { get; set; }
```
