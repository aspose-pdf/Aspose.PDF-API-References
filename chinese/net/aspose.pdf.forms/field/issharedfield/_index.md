## Field.IsSharedField 属性

用于生成器支持。当字段添加到页眉或页脚时使用。如果为 true，则该字段将创建一次，并且其外观将在文档的所有页面上可见。如果为 false，则每个文档页面将创建一个单独的字段。

```csharp
public bool IsSharedField { get; set; }
```
