## BarcodeField.ECC 属性

获取一个整数值，表示错误修正系数。对于 PDF417，应在 0 到 8 之间。对于 QRCode，应在 0 到 3 之间（0 为 'L'，1 为 'M'，2 为 'Q'，3 为 'H'）。

```csharp
public int ECC { get; }
```
