## PdfFileSecurity.DecryptFile method

通过所有者密码解密加密的 Pdf 文档。 如果文档没有所有者密码，则允许使用用户密码。 如果进程失败则抛出异常。

```csharp
public bool DecryptFile(string ownerPassword)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| ownerPassword | String | 所有者密码。 |

### 返回值

成功为真。
