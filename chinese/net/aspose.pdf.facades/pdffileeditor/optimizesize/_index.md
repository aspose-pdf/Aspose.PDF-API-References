## PdfFileEditor.OptimizeSize 属性

获取或设置优化标志。如果设置了此标志，结果文件中的相等资源流将合并为一个 PDF 对象。这可以减少结果文件的大小，但可能导致执行速度变慢和更大的内存需求。默认值：false。

```csharp
public bool OptimizeSize { get; set; }
```
