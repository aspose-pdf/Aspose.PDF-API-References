## AddMarginsPct(string, string, int[], double, double, double, double) {#addmarginspct_1}

调整页面内容的大小并添加指定的边距。 边距以初始页面大小的百分比指定。

```csharp
public bool AddMarginsPct(string source, string destination, int[] pages, double leftMargin, 
    double rightMargin, double topMargin, double bottomMargin)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| source | String | 源文档的路径。 |
| destination | String | 保存结果文档的路径。 |
| pages | Int32[] | 页面索引数组。如果为 null，则将处理所有文档页面。 |
| leftMargin | Double | 以初始页面大小百分比表示的左边距。 |
| rightMargin | Double | 以初始页面大小百分比表示的右边距。 |
| topMargin | Double | 初始页面大小百分比的上边距。 |
| bottomMargin | Double | 以初始页面大小百分比表示的底部边距。 |

### 返回值

如果调整大小成功则为真

### 例子

```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
fileEditor.AddMarginsPct("input.pdf", "output.pdf", 
      //处理第1、2、3页
    new int[] { 1, 2, 3}, 
    //左边距是页面宽度的 15% 
    15, 
      //右边距是页面宽度的10%
    10, 
      //上边距是页面宽度的20%
    20, 
      //下边距是页面宽度的5%
    5);
```
