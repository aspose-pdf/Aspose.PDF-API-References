## PdfFileEditor.MergeDuplicateLayers 属性

如果此属性为真，则具有相同名称的连接文档的可选内容将合并为结果文档中的一个图层。否则，具有相同名称的图层将在结果文档中保存为不同的图层。

```csharp
public bool MergeDuplicateLayers { get; set; }
```
