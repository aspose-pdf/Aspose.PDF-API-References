## PdfFileEditor.ConvertTo 属性

设置 PDF 文件格式。结果文件将以指定的文件格式保存。如果未指定此属性，则文件将以默认 PDF 格式保存，而不进行转换。

public PdfFormat ConvertTo { set; }
