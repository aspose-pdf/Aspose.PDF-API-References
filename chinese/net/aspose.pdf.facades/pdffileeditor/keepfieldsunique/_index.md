## PdfFileEditor.KeepFieldsUnique 属性

如果为真，则在合并表单时字段名称将变得唯一。将向字段名称添加后缀，后缀模板可以在 UniqueSuffix 属性中指定。

```csharp
public bool KeepFieldsUnique { get; set; }
```
