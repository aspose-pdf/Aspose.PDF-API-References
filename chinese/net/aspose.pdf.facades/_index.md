**Aspose.Pdf.Facades** 命名空间提供的类最初来自 Aspose.Pdf.Kit。这些类用于操作文档，执行诸如连接、盖章、签名、注释等操作，但在高层次上，不需要访问文档的内部结构。

## 类

| 类 | 描述 |
| --- | --- |
| [AutoFiller](./autofiller/) | 表示一个类，用于从数据库或其他数据源接收数据，将其填充到设计的模板 PDF 的字段中，最后生成新的 PDF 文件或流。它有两种模板文件输入模式：作为流或 PDF 文件输入。它有四种输出模式：一个合并流，一个合并文件，多个小流，多个小文件。它可以接收包含在 System.Data.DataTable 中的文字数据。 |
| [BDCProperties](./bdcproperties/) | BDC 操作员属性。 |
| [Bookmark](./bookmark/) | 表示一个书签。 |
| [Bookmarks](./bookmarks/) | 表示 [`Bookmark`](../aspose.pdf.facades/bookmark/) 对象的集合。 |
| [DocumentPrivilege](./documentprivilege/) | 表示访问 PDF 文件的权限。参考 [`PdfFileSecurity`](../aspose.pdf.facades/pdffilesecurity/)。使用此类有 4 种方式：1. 直接使用预定义权限。2. 基于预定义权限并更改某些特定权限。3. 基于预定义权限并更改某些特定的 Adobe Professional 权限组合。4. 混合方式 2 和方式 3。 |
| [Facade](./facade/) | 基础外观类。 |

| [FormEditor](./formeditor/) | 用于编辑表单（添加/删除字段等）的类。 |

| [PdfConverter](./pdfconverter/) | 表示一个类，用于将 PDF 文件的每一页转换为图像，目前支持 BMP、JPEG、PNG 和 TIFF。支持的 PDF 内容：图片、表单、评论。 |
| [PdfExtractor](./pdfextractor/) | 用于从 PDF 文档中提取图像和文本的类。 |
| [PdfFileEditor](./pdffileeditor/) | 实现与 PDF 文件的操作：连接、拆分、提取页面、制作小册子等。 |
| [PdfFileInfo](./pdffileinfo/) | 表示一个类，用于访问 PDF 文档的元信息。 |
| [PdfFileMend](./pdffilemend/) | 表示一个类，用于在现有 PDF 文档的页面上添加文本和图像。 |
| [PdfFileSanitization](./pdffilesanitization/) | 表示清理和恢复 API。如果您无法以任何其他方式创建/打开文档，请使用它。 |
| [PdfFileSecurity](./pdffilesecurity/) | 表示使用所有者或用户密码加密或解密 PDF 文件，改变安全设置和密码。 |
| [PdfFileSignature](./pdffilesignature/) | 表示一个类，用于使用证书对 PDF 文件进行签名。 |
| [PdfFileStamp](./pdffilestamp/) | 用于向 PDF 文件添加印章（水印或背景）的类。 |
| [PdfJavaScriptStripper](./pdfjavascriptstripper/) | 用于移除所有 JavaScript 代码的类。 |

| [SaveableFacade](./saveablefacade/) | 所有可保存外观的基类。 |
| [SignatureName](./signaturename/) | 表示一个类，用于签名名称。 |
| [Stamp](./stamp/) | 表示印章的类。 |
| [StampInfo](./stampinfo/) | 表示印章信息的类。 |

| [ViewerPreference](./viewerpreference/) | 描述查看器偏好（页面模式、非全屏页面模式、页面布局）。 |
