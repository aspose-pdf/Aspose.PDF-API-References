## PdfExtractor.IsBidi 属性

当文本包含希伯来或阿拉伯符号时为真。此情况必须特别考虑，因为字符串函数会改变其行为，并开始从右到左处理文本（数字和其他非文本字符除外）。

public bool IsBidi { get; }
