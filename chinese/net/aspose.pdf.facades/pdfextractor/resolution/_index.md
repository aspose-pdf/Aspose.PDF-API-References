## PdfExtractor.Resolution 属性

设置或获取提取图像的分辨率。默认值为 150。具有更高分辨率值的图像更清晰。然而，增加分辨率值会导致提取图像所需的时间和内存增加。通常，要获得清晰的图像，将分辨率设置为 150 或 300 就足够了。

```csharp
public int Resolution { get; set; }
```
