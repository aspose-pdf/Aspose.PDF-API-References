## PdfExtractor.ExtractImageMode 属性

设置提取图像过程的模式。

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## 备注

默认值是 ExtractImageMode.DefinedInResources，它提取所有在资源中定义的图像。要提取实际显示的图像，应使用 ExtractImageMode.ActuallyUsed 模式。
