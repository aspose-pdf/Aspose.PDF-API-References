## PositioningMode 枚举

定义定位模式。可能的值包括遗留（向后兼容）和当前（更新的文本位置计算方法）

```csharp
public enum PositioningMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Legacy | `0` | 遗留文本定位 |
| ModernLineSpacing | `1` | 更新的行间距，垂直位置计算遵循旧规则（即文本相对于指定矩形的左下角定位） |
| Current | `2` | 更新的行间距，垂直位置计算基于左上角而不是左下角。 |
