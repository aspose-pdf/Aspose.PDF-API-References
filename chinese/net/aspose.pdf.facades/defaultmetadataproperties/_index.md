## DefaultMetadataProperties 枚举

```csharp
public enum DefaultMetadataProperties
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Advisory | `0` | xmp:Advisory 属性。一个无序数组，指定在创作应用程序外编辑的属性。每个项目应包含一个单一的 /// 命名空间和由一个 ASCII 空格分隔的 XPath |
| BaseURL | `1` | xmp:BaseURL 属性。文档内容中相对 URL 的基本 URL。如果该文档包含 Internet 链接，并且这些链接是相对的，/// 它们是相对于此基本 URL 的。此属性提供了一种标准方式，使嵌入的相对 URL 能够被工具解释。/// Web 创作工具应根据它们对 URL 将被解释的位置的理解来设置该值 |
| CreateDate | `2` | xmp:CreateDate 属性。资源最初创建的日期和时间。 |
| CreatorTool | `3` | xmp:CreatorTool 属性。用于创建资源的第一个已知工具的名称。 |
| Identifier | `4` | xmp:Identifier 属性。一个无序数组的文本字符串，明确标识给定上下文中的资源 |
| MetadataDate | `5` | xmp:MetadataDate 属性。此资源的任何元数据最后更改的日期和时间 |
| ModifyDate | `6` | xmp:ModifyDate 属性。资源最后修改的日期和时间。 |
| Nickname | `7` | xmp:Nickname 属性。资源的一个简短非正式名称。 |
| Thumbnails | `8` | xmp:Thumbnails 属性。文件的替代缩略图图像数组，其特征可能在大小或图像编码等方面有所不同。 |
