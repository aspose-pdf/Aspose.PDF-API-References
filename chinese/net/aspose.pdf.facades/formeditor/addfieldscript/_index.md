## FormEditor.AddFieldScript 方法

为 PushButton 字段添加 JavaScript。如果存在旧事件，则新事件将添加在其后。

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| fieldName | 字符串 | 完全限定的字段名称。 |
| script | 字符串 | 要添加/放入 PushButton 字段的 Java 脚本。 |

### 返回值

如果脚本成功添加，则返回 True。
