## FormEditor.SetFieldAlignmentV method

设置文本字段的垂直对齐方式。

```csharp
public bool SetFieldAlignmentV(string fieldName, int alignment)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| fieldName | String | 限定字段名。 |
| alignment | Int32 | 对齐方式定义，包括FormFieldFacade.AlignTop、 FormFieldFacade.AlignMiddle和FormFieldFacade.AlignRight。 |

### 返回值

如果找到字段并成功填充对齐，则为 true。

### 例子

