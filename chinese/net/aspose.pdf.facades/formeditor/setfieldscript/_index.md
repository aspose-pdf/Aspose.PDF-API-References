## FormEditor.SetFieldScript 方法

为 PushButton 字段设置 JavaScript。如果存在旧的 JavaScript，将被新的替换。

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| fieldName | 字符串 | 完全限定的字段名称。 |
| script | 字符串 | 要添加/放入 PushButton 字段的 Java 脚本。 |

### 返回值

如果字段脚本成功设置，则返回 true。
