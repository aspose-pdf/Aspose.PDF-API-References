## DocumentPrivilege.ChangeAllowLevel 属性

获取和设置文档权限的更改级别。就像 Adobe Professional 的允许更改设置。0: 无。1: 插入、删除和旋转页面。2: 填写表单字段和签署现有签名字段。3: 评论、填写表单字段和签署现有签名字段。4: 除提取页面外的任何操作。

```csharp
public int ChangeAllowLevel { get; set; }
```

## 备注

如果属性的值为 -1，则级别未定义。
