## DocumentPrivilege.PrintAllowLevel 属性

获取和设置文档权限的打印级别。就像 Adobe Professional 的打印允许设置。0: 无。1: 低分辨率 (150 dpi)。2: 高分辨率。

```csharp
public int PrintAllowLevel { get; set; }
```

## 备注

如果属性的值为 -1，则级别未定义。
