## PdfViewer.PrintAsImage 属性

设置或获取 PdfViewer 以图像形式打印的模式。

```csharp
public bool PrintAsImage { get; set; }
```

## 备注

如果为 true，则始终以图像形式打印（生成要打印的图像）。如果为 false，则直接打印到设备，如果所有功能都受支持。如果文档包含不支持的功能，系统可能会自动决定以图像形式打印。默认值为 false。
