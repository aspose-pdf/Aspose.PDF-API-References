## PdfViewer.AutoResize 属性

获取或设置一个布尔值，指示文件是否以优化大小打印。如果为 false，则页面不进行缩放打印。如果为 true，则页面进行缩放以适应可打印区域。

```csharp
public bool AutoResize { get; set; }
```
