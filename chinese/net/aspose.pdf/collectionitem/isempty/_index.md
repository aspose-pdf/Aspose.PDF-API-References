## CollectionItem.IsEmpty 属性

获取一个值，指示集合项是否为空。

```csharp
public bool IsEmpty { get; }
```

### 返回值

如果集合项为空，则为 true；否则为 false。

## 备注

如果集合项不包含任何值，包括字符串值、双精度值、整数值和日期值，则此属性返回 true。如果集合项中存在任何这些值类型，则此属性返回 false。
