## Color.FromCmyk 方法

根据 CMYK 颜色组件获取有效的 pdf Color 对象。

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| c | Double | 青色组件（值 0 - 1）。 |
| m | Double | 品红色组件（值 0 - 1）。 |
| y | Double | 黄色组件（值 0 - 1）。 |
| k | Double | 黑色组件（值 0 - 1）。 |

### 返回值

每个组件值在 [0..1] 范围内的 Color 对象。
