## 颜色相等运算符

如果两个颜色相等，则返回 true。

```csharp
public static bool operator ==(Color x, Color y)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | Color | 第一个颜色对象。 |
| y | Color | 第二个颜色对象。 |

### 返回值

如果颜色对象相等，则返回 true。
