## Color 不等于运算符

```csharp
public static bool operator !=(Color x, Color y)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x | Color | 第一个 Color 对象。 |
| y | Color | 第二个 Color 对象。 |

### 返回值

如果 Color 对象不相等，则返回 true。
