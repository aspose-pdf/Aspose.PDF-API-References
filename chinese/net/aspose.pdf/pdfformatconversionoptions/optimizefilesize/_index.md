## PdfFormatConversionOptions.OptimizeFileSize 属性

获取或设置一个标志，该标志启用/禁用特殊转换模式，以获取具有减小文件大小的 PDF/A 文档。现在，这个标志影响 PDF 文档中使用的字体的优化，未来可能还会使用这个标志来开启其他数据结构（如图形）的优化。设置这个标志和模式可能会显著减少文件大小，但同时也可能显著降低转换性能。

```csharp
public bool OptimizeFileSize { get; set; }
```
