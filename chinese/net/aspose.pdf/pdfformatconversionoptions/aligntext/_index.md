## PdfFormatConversionOptions.AlignText property

此标志控制转换后文档中的文本对齐方式。默认情况下，文档转换 不会影响文本对齐并保持文本不变。但在某些情况下，字体替换 会导致转换后的文档中的文本重叠或多余空格。设置此标志时 将执行特殊对齐操作。此标志应仅针对存在重叠文本或额外文本空格问题的文档设置 导致使用此标志会降低 性能，并且在某些情况下可能会损坏文本内容。

public bool AlignText { get; set; }
