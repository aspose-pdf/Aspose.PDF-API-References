## PdfFormatConversionOptions.AlignText property

此标志控制转换文档中的文本对齐方式。默认情况下，文档转换 不会影响文本对齐并保持文本不变。但在某些情况下，字体替代 会导致转换后的文档中出现文本重叠或多余空格。当此标志设置 时，将执行特殊对齐操作。这个标志应该只为documents 设置，它有重叠文本或额外文本空间的问题，导致使用这个标志会降低 性能，在某些情况下可能会损坏文本内容。

public bool AlignText { get; set; }
