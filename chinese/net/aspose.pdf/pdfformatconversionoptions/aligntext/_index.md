## PdfFormatConversionOptions.AlignText 属性

此标志控制转换文档中的文本对齐方式。默认情况下，文档转换不会影响文本对齐，并保持文本原样。但是在某些情况下，字体替换会导致转换文档中的文本重叠或额外空格。当设置此标志时，将执行特殊的对齐操作。此标志仅应在文档存在文本重叠或额外文本空格问题时设置，因为使用此标志会降低性能，并且在某些情况下可能会损坏文本内容。

```csharp
public bool AlignText { get; set; }
```
