## HtmlSaveOptions.HtmlMarkupGenerationModes 枚举

有时会存在对生成的 HTML 的特定要求。此枚举定义了在将 PDF 转换为 HTML 时可以使用的 HTML 准备模式，以满足这些特定要求。

```csharp
public enum HtmlMarkupGenerationModes
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| WriteAllHtml | `0` | 默认模式，任何特定要求均不存在。将生成包含所有 HTML 部分的输出，而无需任何特殊的额外处理。 |
| WriteOnlyBodyContent | `1` | 将剥离所有位于 HTML 主体外的 HTML 内容，即只保留位于 \c \ \c \ 标签内的内容 |
