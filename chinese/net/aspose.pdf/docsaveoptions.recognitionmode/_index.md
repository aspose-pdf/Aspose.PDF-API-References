## DocSaveOptions.RecognitionMode enumeration

允许控制如何将 PDF 文档转换为文字处理文档。

```csharp
public enum RecognitionMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Textbox | `0` | 此模式快速且有利于最大限度地保留 PDF 文件的原始外观， ，但结果文档的可编辑性可能会受到限制。 |
| Flow | `1` | 全识别模式，引擎进行分组和多级分析，还原 原始文档作者的意图，并产生最大可编辑的文档。 缺点是输出文档可能看起来与原始PDF文件不同。 |
| EnhancedFlow | `2` | 支持表格识别的新 Flow 模式的早期 alfa 版本。 |

### 评论

使用Textbox当生成的文档不会 进一步被大量编辑时的模式。当没有太多事情要做时，文本框很容易修改。

使用Flow输出文档需要进一步编辑时的模式。 流模式下的段落和 texlines 允许轻松修改文本，但 unupported 格式化对象看起来会比在Textbox模式。
