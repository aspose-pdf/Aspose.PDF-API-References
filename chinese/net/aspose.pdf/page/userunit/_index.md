## Page.UserUnit 属性

获取或设置 UserUnit 值。一个正数，给出默认用户空间单位的大小，以 1 / 72 英寸的倍数表示。默认值为 1。请设置零或负值以清除此页面中的条目。

```csharp
public double UserUnit { get; set; }
```
