## LineBreakStyle 枚举

表示文件的可能换行样式。

```csharp
public enum LineBreakStyle
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Windows | `0` | 选择 Windows 字符串换行样式（CRLF）。 |
