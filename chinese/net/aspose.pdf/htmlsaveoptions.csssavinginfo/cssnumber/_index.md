## HtmlSaveOptions.CssSavingInfo.CssNumber 字段

由转换器设置。在转换过程中会创建多个 CSS 文件。此属性显示在转换过程中保存的 CSS 文件的序号。它可以在自定义代码的逻辑中用于决定如何处理或保存 CSS 内容

```csharp
public int CssNumber;
```
