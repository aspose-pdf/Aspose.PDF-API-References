## LoadOptions.ResourceLoadingResult.MIMETypeIfKnown 字段

有时，了解加载资源的 MIME 类型对转换器是有用的。您可以在此参数中提供 MIME 类型（如果在加载后已知）。当 MIME 类型未知或不需要提供时，请将参数保持为 null。

```csharp
public string MIMETypeIfKnown;
```
