## HorizontalAlignment 枚举

描述水平对齐。

```csharp
public enum HorizontalAlignment
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 无对齐。 |
| Left | `1` | 左对齐。 |
| Center | `2` | 居中对齐。 |
| Right | `3` | 右对齐。 |
| Justify | `4` | 两端对齐。文本将在左右边距上对齐。 |
| FullJustify | `5` | 类似于“Justify”对齐，除了最后一行在“Justify”模式下仅左对齐，而在“FullJustify”模式下所有行将左右对齐。 |
