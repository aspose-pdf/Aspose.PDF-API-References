## ParagraphPositioningMode 枚举

指定用于确定元素在页面上位置的变体。

```csharp
public enum ParagraphPositioningMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | `0` | 位置由之前放置的元素确定。添加元素时会考虑到后续元素的位置。 |
| Absolute | `1` | 位置由 Left 和 Top 值指定，不依赖于之前的元素，也不影响后续元素的位置。 |
