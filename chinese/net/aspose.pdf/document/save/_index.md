## Save() {#save}

增量保存文档（即使用增量更新技术）。

```csharp
public void Save()
```

### 评论

为了逐步保存文档，我们应该打开文档文件进行写入。 因此 Document 必须使用可写流初始化，就像在下一个代码片段中一样： Document doc = new Document(new FileStream("document.pdf", FileMode.Open, FileAccess.ReadWrite)); // 进行一些更改并保存文档增量 doc.Save();
