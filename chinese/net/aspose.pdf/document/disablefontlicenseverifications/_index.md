## Document.DisableFontLicenseVerifications 属性

如果这些操作被该字体的许可证禁止，则无法执行与字体相关的许多操作。例如，如果许可证规则禁止将某些字体嵌入到 PDF 文档中，则无法将该字体嵌入到 PDF 文档中。此标志用于禁用当前 PDF 文档中所有字体的任何许可证限制。使用此标志时请小心。当设置此标志时，意味着设置此标志的人对可能的许可证/法律违规行为承担全部责任。因此，他自担风险。强烈建议仅在您完全确信自己没有违反版权法时使用此标志。默认值为 false。

```csharp
public bool DisableFontLicenseVerifications { get; set; }
```
