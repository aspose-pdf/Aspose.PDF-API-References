## Document.DisableFontLicenseVerifications property

如果该字体的许可证禁止这些操作，则无法执行该字体的许多操作。 例如，如果许可规则禁用该字体的嵌入，则某些字体无法嵌入到 PDF 文档中。 此标志用于禁用当前 PDF 文档中所有字体的任何许可限制。 使用此标志时要小心。当它被设置时，意味着设置这个标志的人， 对自己可能违反许可/法律的行为承担全部责任。 所以他自己承担风险。 强烈建议仅在您完全确信自己没有违反 版权法时才使用此标志。 默认为假。

```csharp
public bool DisableFontLicenseVerifications { get; set; }
```
