## Document.IgnoreCorruptedObjects property

获取或设置忽略源文件中错误的标志。 当源文档中的页面复制到目标文档中时，如果源文件中的某些对象在此标志为 false 时已损坏，则复制过程将停止，但异常 。 示例:dest.Pages.Add(src.Pages); 如果此标志设置为 true，则损坏的对象将被替换为空值。 默认情况下:true。

```csharp
public bool IgnoreCorruptedObjects { get; set; }
```
