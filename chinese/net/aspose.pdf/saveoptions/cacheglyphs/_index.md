## SaveOptions.CacheGlyphs 属性

获取或设置布尔值，指示在准备 aps 页面时是否将字体字形缓存。提高将 PDF 转换为其他格式的性能，但增加内存消耗。

```csharp
public bool CacheGlyphs { get; set; }
```
