## ImageDeleteAction enumeration

从集合中删除图像时对图像对象执行的操作。如果图像对象被删除

```csharp
public enum ImageDeleteAction
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| KeepContents | `0` | 图像将从集合中删除。如果页面内容包含对图像的引用，它们将不会被删除。文件可能无效。 |
| None | `1` | 图像将从集合和页面内容中删除，但图像对象不会被删除。文件大小不会减少。 |
| ForceDelete | `2` | 图像将从集合中删除，图像对象将从文档中删除。如果存在对同一对象的其他引用，则文档可能已损坏。 |
| Check | `3` | 图像将从集合中删除，并且仅当没有其他页面对图像的其他引用时才会删除图像对象。与 ForceDelete 选项相比，这可能需要更多时间。 |
