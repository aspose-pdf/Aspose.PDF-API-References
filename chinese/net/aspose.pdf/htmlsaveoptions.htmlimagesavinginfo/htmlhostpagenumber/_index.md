## HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber 字段

告知自定义代码保存的图像属于生成的 HTML 页面文件集的哪一页。如果关闭了按页面拆分，则此值始终为 '1'，因为在这种情况下只生成一个 HTML 页面。

```csharp
public int HtmlHostPageNumber;
```
