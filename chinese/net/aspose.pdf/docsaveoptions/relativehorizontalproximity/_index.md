## DocSaveOptions.RelativeHorizontalProximity property

在 Pdf 中，单词可以用打印单词 的运算符内部表示，通过独立打印它们的字母或音节。因此，为了检测单词，有时我们需要检测实际上是单词的独立字符组 。 此设置定义文本元素（字母、音节）之间的空间宽度 在识别源 PDF 中的单词期间必须将其视为单词之间的距离。 （至少在字母之间存在这个宽度的空格意味着 文本元素属于不同的单词）。 以字体大小为标准 - 1.0 表示假定单词字体大小的 100%。 注意！仅在源 PDF 包含特定很少使用的字体 无法从字体计算最佳值的情况下使用。 因此，在绝大多数情况下，此参数不会改变结果文档中的任何内容。

```csharp
public float RelativeHorizontalProximity { get; set; }
```
