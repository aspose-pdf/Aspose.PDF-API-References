## DocSaveOptions.ConvertType3Fonts 属性

获取或设置 Type3 字体的转换。在 Type 3 字体中，字形应由图形操作符的流定义。这意味着在 DOC/DOCX 输出中，我们看到的是图像而不是文本。将此标志设置为 true 以将 Type3 字体转换为 TTF，并在生成的文件中获取文本。

```csharp
public bool ConvertType3Fonts { get; set; }
```
