## DocSaveOptions.ReSaveFonts 属性

获取或设置重新保存字体的过程。如果设置为 true，我们将在每一页重新加载字体，以避免之前字体属性的影响，并从头加载新创建的字体。如果您想提高性能，请将此选项设置为 false。默认值为 true;

```csharp
public bool ReSaveFonts { get; set; }
```
