## SvgSaveOptions.CustomStrategyOfEmbeddedImagesSaving field

此字段可以包含保存策略 在转换期间必须使用（如果存在） 用于自定义处理创建引用的外部图像 嵌入保存的 SVG 的文件（如嵌入的 BMP 或 JPEG）。 该策略必须处理资源 并返回表示生成的 SVG 中保存的资源 的所需 URI 的字符串。 如果出于某种原因必须由转换器的代码本身而不是自定义代码来处理这个或那个文件， 请在“imageSavingInfo”的自定义代码标志“CustomProcessingCancelled”中设置参数的变量 它向转换器发出信号:处理该资源 的所有必要步骤必须在转换器本身中完成，就好像没有任何外部自定义代码一样。

```csharp
public EmbeddedImagesSavingStrategy CustomStrategyOfEmbeddedImagesSaving;
```
