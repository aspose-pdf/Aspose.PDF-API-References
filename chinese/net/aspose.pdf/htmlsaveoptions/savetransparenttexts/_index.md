## HtmlSaveOptions.SaveTransparentTexts 字段

Pdf 可以包含可以选择到剪贴板的透明文本（通常发生在文档包含图像和从中提取的 OCR 文本时）。此设置告诉转换器我们是否需要将这些文本作为透明可选择文本保存到结果 HTML 中

```csharp
public bool SaveTransparentTexts;
```
