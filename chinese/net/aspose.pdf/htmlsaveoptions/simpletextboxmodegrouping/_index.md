## HtmlSaveOptions.SimpleTextboxModeGrouping 属性

此属性指定将字形和单词按顺序分组为字符串。例如，标签和单词在转换后的 HTML 中的顺序不同，而您希望它们匹配。仅当 FixedLayout 属性的值为 true 时，此参数才会应用于文档。

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
