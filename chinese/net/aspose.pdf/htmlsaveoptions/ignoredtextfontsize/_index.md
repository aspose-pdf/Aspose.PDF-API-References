## HtmlSaveOptions.IgnoredTextFontSize 属性

指定大小或更小的文本将在转换过程中被忽略。我们不会删除这些文本，我们忽略它并且不将其传输到输出文件

```csharp
public float? IgnoredTextFontSize { get; set; }
```
