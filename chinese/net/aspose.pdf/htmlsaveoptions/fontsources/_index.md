## HtmlSaveOptions.FontSources 属性

预保存字体的字体来源。

```csharp
public FontSourceCollection FontSources { get; }
```

## 备注

字体可以预先保存以用于缓存目的，然后传递到 Html 转换过程中。例如，在文档拆分场景中以及使用单一字体集在多个线程中处理文档页面时，这可能会很有用。
