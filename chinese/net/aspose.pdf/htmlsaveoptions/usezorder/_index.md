## HtmlSaveOptions.UseZOrder property

如果属性 UseZORder 设置为 true，则图形和文本将添加到生成的 HTML 文档 相应地在原始 PDF 文档中的 Z 顺序。如果此属性为 false，则所有图形都将 作为单层放置，这可能会对重叠对象造成一些不必要的影响。

public bool UseZOrder { get; set; }
