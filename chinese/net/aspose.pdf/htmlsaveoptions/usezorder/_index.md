## HtmlSaveOptions.UseZOrder 属性

如果属性 UseZOrder 设置为 true，则图形和文本将根据原始 PDF 文档中的 Z 顺序添加到生成的 HTML 文档中。如果该属性为 false，则所有图形将作为单个层放置，这可能会导致重叠对象的一些不必要效果。

```csharp
public bool UseZOrder { get; set; }
```
