## HtmlSaveOptions.PreventGlyphsGrouping property

该属性开启文本字形不会被分组为单词和字符串的模式 该模式允许在页面上定位字形时保持最大精度，它可以 用于转换带有音符或字形的文档，应该单独放置 只有当 FixedLayout 属性的值为 true 时，此参数才会应用于文档。

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
