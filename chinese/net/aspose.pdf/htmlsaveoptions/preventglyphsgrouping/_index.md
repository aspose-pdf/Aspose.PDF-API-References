## HtmlSaveOptions.PreventGlyphsGrouping 属性

此属性切换到文本字形不会被分组为单词和字符串的模式。此模式允许在页面上保持字形定位的最大精度，并且可以用于转换包含音乐符号或应彼此分开的字形的文档。仅当 FixedLayout 属性的值为 true 时，此参数才会应用于文档。

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
