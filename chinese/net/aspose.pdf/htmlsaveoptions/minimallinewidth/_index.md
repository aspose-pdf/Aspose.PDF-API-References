## HtmlSaveOptions.MinimalLineWidth 属性

此属性设置图形路径线的最小宽度。如果线的厚度小于 1px，Adobe Acrobat 会将其四舍五入到此值。因此，此属性可用于模拟 HTML 浏览器的此行为。

```csharp
public float MinimalLineWidth { get; set; }
```
