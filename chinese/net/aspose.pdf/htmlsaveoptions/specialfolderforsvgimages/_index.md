## HtmlSaveOptions.SpecialFolderForSvgImages field

获取或设置目录的路径，如果在将文档保存为 HTML 的过程中遇到 ，则必须仅将 SVG 图像保存到该目录。如果参数为空或 null 则 SVG 文件（如果有）将与其他图像文件（靠近输出文件）一起保存 或图像的特殊文件夹中（如果它在 SpecialImagesFolderIfAny 选项中指定）。 如果 CustomImageSavingStrategy 属性成功用于处理相关图像文件，则不会影响任何内容。

```csharp
public string SpecialFolderForSvgImages;
```
