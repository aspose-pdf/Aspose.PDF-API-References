## HtmlSaveOptions.CustomResourceSavingStrategy 字段

此字段可以包含在转换过程中必须使用的保存策略（如果存在），用于自定义处理与保存的 HTML 节点相关的创建的引用资源文件（如图像和字体）。该策略必须处理资源并返回表示生成的 HTML 中保存的资源的期望 URL 的字符串。

```csharp
public ResourceSavingStrategy CustomResourceSavingStrategy;
```
