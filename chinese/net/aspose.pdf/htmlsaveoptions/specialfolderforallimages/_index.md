## HtmlSaveOptions.SpecialFolderForAllImages 字段

获取或设置必须保存任何图像的目录路径，如果在将文档保存为 HTML 时遇到这些图像。如果参数为空或为 null，则图像文件（如果有）将与链接到 HTML 的其他文件一起保存。如果成功使用 CustomImageSavingStrategy 属性处理相关图像文件，则不会影响任何内容。

```csharp
public string SpecialFolderForAllImages;
```
