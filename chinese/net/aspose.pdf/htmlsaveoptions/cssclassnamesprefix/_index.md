## HtmlSaveOptions.CssClassNamesPrefix 字段

当 PDFtoHTML 转换器生成结果 CSS 时，生成并使用的 CSS 类名（类似于 ".stl_01 {}" ... ".stl_NN {}"）在结果 CSS 中使用。此属性允许强制设置类名前缀。例如，如果您希望所有类名以 'my_prefix_' 开头（即类似于 'my_prefix_1' ... 'my_prefix_NNN'），则只需在转换之前将 'my_prefix_' 分配给此属性。如果此属性保持不变（即值为 null），则转换器将自行生成类名（类似于 ".stl_01 {}" ... ".stl_NN {}"）。

```csharp
public string CssClassNamesPrefix;
```
