## HtmlSaveOptions.ConvertMarkedContentToLayers property

如果属性 ConvertMarkedContentToLayers 设置为 true，则 PDF 中的所有元素被标记 内容（层）将被放入 HTML div 中，“data-pdflayer”属性指定层名称。 该层名称将从 PDF 的可选属性中提取标记的内容。 如果此属性为 false（默认情况下），则不会从 PDF 标记的内容创建任何图层。

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
