## HtmlSaveOptions.PagesFlowTypeDependsOnViewersScreenSize 字段

如果属性 'SplitOnPages=false'，则表示所有输入 PDF 页面的整个 HTML 将放入一个大的结果 HTML 文件中。此标志定义结果 HTML 是否以这样的方式生成，即表示 PDF 页面的区域在结果 HTML 中的流动将依赖于查看者的屏幕分辨率。假设查看者一侧的屏幕宽度足够大，可以将 2 页或更多页面水平放置在一起。如果此标志设置为 true，则将利用此机会（尽可能多的页面将水平显示在一起，然后下一组水平页面将显示在第一组下方）。否则，页面将以这样的方式流动：下一页总是在上一页下方。

```csharp
public bool PagesFlowTypeDependsOnViewersScreenSize;
```
