## HtmlSaveOptions.RenderTextAsImage 属性

如果属性 RenderTextAsImage 设置为 true，则源中的文本将成为 HTML 中的图像。这可能有助于使文本不可选择或 HTML 文本未正确呈现。

```csharp
public bool RenderTextAsImage { get; set; }
```
