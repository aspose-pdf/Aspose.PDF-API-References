## HtmlSaveOptions.SaveFullFont 属性

指示将保存完整字体，仅支持 True Type 字体。默认情况下 SaveFullFont = false，转换器保存显示文档文本所需的初始字体的子集。

```csharp
public bool SaveFullFont { get; set; }
```
