## HtmlSaveOptions.IgnoreResourceFontErrors 属性

获取或设置与缺少字体相关的错误将被忽略的指示。 true - 表示将忽略缺少字体的错误。 在处理过程中，将跳过引用不正确资源的文本段。 默认值为 false

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
