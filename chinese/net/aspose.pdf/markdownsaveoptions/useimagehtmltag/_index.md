## MarkdownSaveOptions.UseImageHtmlTag 属性

获取和设置允许使用 img 标签将图像插入文本的左侧和右侧。在这种情况下，在 markdown 查看器中，文本将围绕图像换行。

```csharp
public bool UseImageHtmlTag { get; set; }
```
