## PdfFormatConversionOptions.SegmentAlignStrategy 枚举

描述用于对齐文档文本段落的策略。目前仅支持将段落恢复到原始边界的策略。未来可能会添加其他策略。

```csharp
public enum SegmentAlignStrategy : byte
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 无。 |
| RestoreSegmentBounds | `1` | 恢复段落边界。 |
