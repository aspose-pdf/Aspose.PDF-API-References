## BaseParagraph.ZIndex 属性

获取或设置一个 int 值，该值指示图形的 Z 顺序。具有较大 ZIndex 的图形将放置在具有较小 ZIndex 的图形上方。ZIndex 可以是负数。具有负 ZIndex 的图形将放置在页面文本后面。

```csharp
public int ZIndex { get; set; }
```
