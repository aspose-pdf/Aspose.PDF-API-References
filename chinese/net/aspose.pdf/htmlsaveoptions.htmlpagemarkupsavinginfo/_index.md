## HtmlSaveOptions.HtmlPageMarkupSavingInfo class

如果 HtmlSaveOptions 的 SplitToPages 属性，则在 PDF 转换为 HTML 期间会创建多个 HTML 文件（每个转换的页面一个 HTML 文件） 。 此类表示与自定义保存一个 HTML 页面的标记相关的数据集 在 PDF 到 HTML 的转换过程中

```csharp
public class HtmlPageMarkupSavingInfo
```
