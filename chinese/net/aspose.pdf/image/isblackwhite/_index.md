## Image.IsBlackWhite 属性

获取或设置一个布尔值，指示图像是否被强制为黑白。如果使用的是 CCITT 子格式的 TIFF 图像，则此属性必须设置为 true。

```csharp
public bool IsBlackWhite { get; set; }
```
