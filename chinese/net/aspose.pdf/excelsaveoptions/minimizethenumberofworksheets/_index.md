## ExcelSaveOptions.MinimizeTheNumberOfWorksheets 属性

如果需要最小化结果工作簿中的工作表数量，请设置为 true。默认值为 false；这意味着将每个 PDF 页面保存为单独的工作表。

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
