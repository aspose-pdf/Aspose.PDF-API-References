## Stamp.Rotate 属性

根据 [`Rotation`](../../rotation/) 值设置或获取印章内容的旋转。注意。此属性用于设置角度，这些角度是 90 度的倍数（0、90、180、270 度）。要设置任意角度，请使用 RotateAngle 属性。如果 ArbitraryAngle 设置的角度不是 90 的倍数，则 Rotate 属性返回 Rotation.None。

```csharp
public Rotation Rotate { get; set; }
```
