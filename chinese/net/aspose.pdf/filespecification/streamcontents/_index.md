## FileSpecification.StreamContents 属性

获取文件内容作为流。内容不会加载到内存中，这样可以减少内存使用。但此流不支持定位和 Length 属性。如果您需要这些功能，请改用 Contents 属性。

```csharp
public Stream StreamContents { get; }
```
