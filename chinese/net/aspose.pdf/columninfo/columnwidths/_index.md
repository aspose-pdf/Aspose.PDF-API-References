## ColumnInfo.ColumnWidths 属性

获取或设置一个包含列宽的字符串。每列的值应由空格分隔。默认单位是点，但也支持厘米、英寸和可用宽度的百分比。例如，"120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
