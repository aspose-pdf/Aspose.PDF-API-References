## TextStamp.TreatYIndentAsBaseLine property

定义放置文本的坐标原点。 如果 TreatYIndentAsBaseLine = true （当 Draw = true 时默认） YIndent 值将被视为文本基线。 如果 TreatYIndentAsBaseLine = false （当 Draw = false 时默认） YIndent 值将被视为文本的底部（下降线）。
