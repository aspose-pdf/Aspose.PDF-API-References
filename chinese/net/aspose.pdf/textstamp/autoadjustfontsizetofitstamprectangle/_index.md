## TextStamp.AutoAdjustFontSizeToFitStampRectangle 属性

如果启用，字体大小将自动调整以适应大小为: [`Width`](../width/) 和 [`Height`](../height/) 的印章矩形。默认宽度和高度源自页面矩形。

```csharp
public bool AutoAdjustFontSizeToFitStampRectangle { get; set; }
```
