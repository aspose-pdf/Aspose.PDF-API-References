## FloatingBox.IsNeedRepeating 属性

获取或设置一个布尔值，指示段落是否需要在下一页重复。默认值为 false。该属性仅在段落本身和其 ReferenceParagraphID 所引用的对象都包含在 RepeatingRows 中时有效。

```csharp
public bool IsNeedRepeating { get; set; }
```
