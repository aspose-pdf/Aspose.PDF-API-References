## UnifiedSaveOptions.ExtractOcrSublayerOnly 属性

此属性启用从具有 OCR 子层的 PDF 文档中提取图像或文本的功能。

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### 属性值

`true` 文本将被提取到结果文档中；否则为 `false`。
