## PdfSaveOptions.DefaultFontName 属性

用于计算机上缺失字体的默认字体名称。当保存为 PDF 的 PDF 文档包含在文档本身和设备上不可用的字体时，API 会用默认字体替换这些字体（如果在设备上找到具有 `DefaultFontName` 的字体）

```csharp
public string DefaultFontName { get; set; }
```
