## PdfANonSpecificationFlags.CheckDifferentNamesInFontDictionaries 属性

一些 PDF 文档包含在内部数据中具有不同名称的字体。使用此标志会强制在 BaseFont 和 FontDescriptor.FontName 字段不同的情况下执行特殊处理逻辑。

```csharp
public bool CheckDifferentNamesInFontDictionaries { get; set; }
```
