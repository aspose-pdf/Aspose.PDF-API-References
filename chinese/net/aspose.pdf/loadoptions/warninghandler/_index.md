## LoadOptions.WarningHandler 属性

用于处理生成的任何警告的回调。WarningHandler 返回 ReturnAction 枚举项，指定继续或中止。继续是默认操作，加载操作将继续，但用户也可以返回中止，在这种情况下，加载操作应停止。

```csharp
public IWarningCallback WarningHandler { get; set; }
```
