## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace 属性

一些字体不提供某些文本符号的 Unicode 信息。这种信息的缺失会导致错误“文本无法映射到 Unicode”。使用此标志将非链接符号映射到 Unicode “空格”（代码 32）。

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
