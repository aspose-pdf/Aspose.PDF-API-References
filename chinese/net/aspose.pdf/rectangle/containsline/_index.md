## Rectangle.ContainsLine 方法

确定矩形是否包含由两个点表示的线。

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| x1 | Double | 线的起点的 X 坐标。 |
| y1 | Double | 线的起点的 Y 坐标。 |
| x2 | Double | 线的终点的 X 坐标。 |
| y2 | Double | 线的终点的 Y 坐标。 |

### 返回值

`true` 如果矩形包含该线；否则，`false`。
