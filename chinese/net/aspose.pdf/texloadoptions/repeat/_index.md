## TeXLoadOptions.Repeat 属性

获取/设置指示是否需要在输入 TeX 文件中存在引用的情况下运行 TeX 作业两次的标志。一般来说，当引擎在排版过程中收集一些数据并在第一次运行时将其存储在辅助文件中时，这种行为是有用的。在第二次运行时，引擎以某种方式使用这些数据。

```csharp
public bool Repeat { get; set; }
```
