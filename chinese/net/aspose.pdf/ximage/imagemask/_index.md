## XImage.ImageMask 属性

获取一个标志，指示图像是否应被视为图像蒙版（参见 8.9.6，“蒙版图像”）。如果该标志为真，则 BitsPerComponent 的值应为 1，且不应指定 Mask 和 ColorSpace；未蒙版区域应使用当前非描边颜色进行绘制。默认值：false。

```csharp
public bool ImageMask { get; }
```

### 属性值

如果图像是图像蒙版，则为真。
