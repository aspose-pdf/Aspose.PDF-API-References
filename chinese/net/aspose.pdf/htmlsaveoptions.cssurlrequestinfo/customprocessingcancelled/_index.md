## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled 字段

如果自定义代码无法或不应定义将在生成的 HTML 中用于引用该 CSS 的 URL，则应设置此字段。如果为 'true'，则 CSS 文件将以标准方式保存在标准位置。

```csharp
public bool CustomProcessingCancelled;
```
