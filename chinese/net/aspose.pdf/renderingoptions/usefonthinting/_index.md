## RenderingOptions.UseFontHinting 属性

使用此标志可以开启字体提示机制。字体提示是使用数学指令来调整轮廓字体的显示。在某些情况下，开启此标志可能会解决文本可读性的问题。目前，使用此标志仅对 TTF 字体有效，如果这些字体在源文档中使用。

```csharp
public bool UseFontHinting { get; set; }
```
