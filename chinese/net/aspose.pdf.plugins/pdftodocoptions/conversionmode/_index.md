## PdfToDocOptions.ConversionMode 属性

允许控制 PDF 文档如何转换为文字处理文档。

```csharp
public ConversionMode ConversionMode { get; set; }
```

## 备注

当生成的文档不需要进一步大量编辑时，请使用文本框模式。文本框在不需要做很多修改时容易修改。

当输出文档需要进一步编辑时，请使用流模式。流模式中的段落和文本行允许轻松修改文本，但不支持的格式对象在文本框模式下看起来会更糟。
