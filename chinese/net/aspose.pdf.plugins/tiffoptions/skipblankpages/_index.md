## TiffOptions.SkipBlankPages 属性

获取或设置一个值，指示是否跳过空白页。

```csharp
public bool SkipBlankPages { get; set; }
```

### 属性值

`true` 如果需要跳过空白页；否则为 `false`。

## 备注

默认值为 false
