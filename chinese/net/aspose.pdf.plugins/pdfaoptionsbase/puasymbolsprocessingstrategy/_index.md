## PdfAOptionsBase.PuaSymbolsProcessingStrategy 属性

获取或设置 PDF 文档中处理私有使用区 (PUA) 符号的策略。

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### 属性值

处理 PUA 符号的策略。对于 B 级符合性文档，默认值为 None，对于 A 级符合性文档，默认值为 SurroundPuaTextWithEmptyActualText。
