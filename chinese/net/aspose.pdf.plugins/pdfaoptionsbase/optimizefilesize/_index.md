## PdfAOptionsBase.OptimizeFileSize 属性

获取或设置一个值，指示在 PDF/A 转换过程中是否尝试减少文件大小。

```csharp
public bool OptimizeFileSize { get; set; }
```

### 属性值

`true` 如果文件大小应该尽可能减少；否则，`false`。

## 备注

当设置为 `true` 时，转换过程将尝试最小化生成的文件大小。这可能会影响转换过程的性能。
