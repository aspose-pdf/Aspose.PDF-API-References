## PdfAOptionsBase.FontEmbeddingOptions 属性

获取处理无法嵌入文档的字体的选项。

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### 属性值

字体嵌入选项。

## 备注

PDF/A 标准要求所有字体必须嵌入文档。此属性提供处理无法嵌入某些字体的情况的选项，因为这些字体在目标 PC 上缺失。
