## ConversionMode 枚举

定义输出文档的转换模式。

```csharp
public enum ConversionMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| TextBox | `0` | 此模式快速且适合最大限度地保留 PDF 文件的原始外观，但生成文档的可编辑性可能有限。 |
| Flow | `1` | 完全识别模式，引擎执行分组和多级分析，以恢复原始文档作者的意图并生成最大可编辑的文档。缺点是输出文档可能与原始 PDF 文件看起来不同。 |
| EnhancedFlow | `2` | 支持表格识别的替代 Flow 模式。 |
