## PdfToXlsOptions.InsertBlankColumnAtFirst 属性

如果需要将空列插入为工作表的第一列，请设置为 true。默认值为 false；这意味着不会插入空列。

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
