### Annot {#Annot}
```
public static final StructureTypeStandard Annot
```


(Аннотация; PDF 1.5) Связь между частью содержимого ILSE и соответствующей аннотацией PDF. Аннотацию следует использовать для всех аннотаций PDF, кроме аннотаций ссылок и аннотаций виджетов.

### Art {#Art}
```
public static final StructureTypeStandard Art
```


(Статья) Относительно самодостаточный текст, составляющий единое повествование или изложение. Статьи должны быть непересекающимися; то есть они не должны содержать других статей в качестве составных элементов.

### BibEntry {#BibEntry}
```
public static final StructureTypeStandard BibEntry
```


(Библиографическая запись) Ссылка, идентифицирующая внешний источник некоторого цитируемого контента. Он может содержать метку (тип структуры Lbl) в качестве дочернего элемента.

Хотя библиографическая запись, скорее всего, будет включать составные части, идентифицирующие автора цитируемого контента, работу, издателя и т. д., на этом уровне детализации не определены стандартные типы структур.

(Тело списка) Описательное содержимое элемента списка. Например, в списке словарей он содержит определение термина. Он может либо содержать содержимое напрямую, либо иметь другие BLSE, возможно, включая вложенные списки, в качестве дочерних элементов.

### NonStruct {#NonStruct}
```
public static final StructureTypeStandard NonStruct
```


(Неструктурный элемент) Группирующий элемент, не имеющий внутренней структурной значимости; он служит исключительно для целей группировки. Этот тип элемента отличается от раздела (тип структуры Div) тем, что он не должен интерпретироваться или экспортироваться в другие форматы документа; однако его потомки должны обрабатываться нормально.

### Note {#Note}
```
public static final StructureTypeStandard Note
```


(Примечание) Элемент пояснительного текста, такой как сноска или концевая сноска, на которую ссылаются в теле документа. Он может иметь метку (тип структуры Lbl) в качестве дочернего элемента. Примечание может быть включено как дочерний элемент структурного элемента в основной текст, который ссылается на него, или оно может быть включено в другом месте (например, в разделе концевых сносок) и доступно посредством ссылки (тип структуры Ссылка).

Тегированный PDF не предписывает размещение сносок в порядке содержимого страницы. Они могут быть либо встроенными, либо в конце страницы, на усмотрение ответственного автора.

(Частный элемент) Элемент группировки, содержащий частный контент, принадлежащий производящему его приложению. Структурное значение этого типа элемента не определено и должно полностью определяться соответствующим автором. Ни элемент Private, ни его потомки не должны интерпретироваться или экспортироваться в другие форматы документов.

### Quote {#Quote}
```
public static final StructureTypeStandard Quote
```


(Цитата) Встроенная часть текста, приписываемая кому-либо, кроме автора окружающего текста.

Цитируемый текст должен быть встроен в один абзац. Это отличается от блочного элемента BlockQuote, который состоит из одного или нескольких полных абзацев (или других элементов, представленных так, как если бы они были полными абзацами).

### RB {#RB}
```
public static final StructureTypeStandard RB
```


(Основной текст Ruby) Полноразмерный текст, к которому применяется аннотация ruby. RB может содержать текст, другие встроенные элементы или их смесь. Он может иметь атрибут RubyAlign.

### RP {#RP}
```
public static final StructureTypeStandard RP
```


(Ruby пунктуация) Пунктуация вокруг текста аннотации ruby. Он используется только в том случае, если аннотацию ruby нельзя правильно отформатировать в стиле ruby, и вместо этого она отформатирована как обычный комментарий или когда она отформатирована как warichu. Он содержит текст (обычно одну ЛЕВУЮ или ПРАВУЮ СКОБКУ или аналогичный символ скобок).

### RT {#RT}
```
public static final StructureTypeStandard RT
```


(Текст аннотации Ruby) Текст меньшего размера, который должен быть помещен рядом с базовым текстом ruby. Он может содержать текст, другие встроенные элементы или их смесь. Он может иметь атрибуты RubyAlign и RubyPosition.

### Ruby {#Ruby}
```
public static final StructureTypeStandard Ruby
```
