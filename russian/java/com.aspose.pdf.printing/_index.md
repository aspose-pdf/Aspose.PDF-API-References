**com.aspose.pdf.printing** package предоставляет классы, которые используются для настройки процесса печати.


## Классы

| Учебный класс | Описание |
| --- | --- |
| [DuplexKind](../com.aspose.pdf.printing/duplexkind) | Задает настройку двусторонней печати принтера. |
| [PdfPrintRange](../com.aspose.pdf.printing/pdfprintrange) | Указывает часть документа для печати. |
| [PdfPrinterResolution](../com.aspose.pdf.printing/pdfprinterresolution) | Представляет разрешение, поддерживаемое принтером. |
| [PdfPrinterResolutionKind](../com.aspose.pdf.printing/pdfprinterresolutionkind) | Указывает разрешение принтера. |
| [PdfPrinterSettings](../com.aspose.pdf.printing/pdfprintersettings) | Указывает информацию о способе печати документа, включая принтер, на котором он печатается. |
| [PrintPageSettings](../com.aspose.pdf.printing/printpagesettings) | Указывает параметры, которые применяются к одной печатной странице. |
| [PrintPaperSize](../com.aspose.pdf.printing/printpapersize) | Задает размер листа бумаги. |
| [PrintPaperSource](../com.aspose.pdf.printing/printpapersource) | Указывает лоток для бумаги, из которого принтер получает бумагу. |
| [PrintPaperSourceKind](../com.aspose.pdf.printing/printpapersourcekind) | Стандартные источники бумаги. |
| [PrinterMargins](../com.aspose.pdf.printing/printermargins) | Задает размеры полей печатной страницы. |
| [PrinterPaperKind](../com.aspose.pdf.printing/printerpaperkind) | Определяет стандартные размеры бумаги. |
