**com.aspose.pdf** является корневым пакетом для всех классов библиотеки Aspose.PDF for Java, которые находятся либо непосредственно в ней, например**Document** или косвенно через несколько подпакетов.

1) классы почти для всех типов аннотаций, которые может иметь документ.

2) классы, которые описывают формы (стандартные, статические, динамические) и различные типы полей, такие как текстовое поле, список, переключатель и т. д.

3) классы, которые помогают работать с логической структурой документа, которая представляет организацию документа по главам и разделам или идентификацию специальных элементов, таких как рисунки, таблицы и сноски.

4) классы, которые позволяют извлекать текст, добавлять текст, манипулировать существующим текстом документа. Он также содержит классы, позволяющие извлекать, заменять, подставлять шрифты документа.

5) классы для работы с различными типами действий, назначениями и другими особенностями документа, которые традиционно называют интерактивными средствами обеспечения взаимодействия пользователя с ним.

6) классы, которые позволяют чередовать поведение функций извлечения текста и манипулирования. В пакете com.aspose.pdf.text есть дополнительные классы.

7) дополнительные классы поддержки для DOM.

8) классы для вариантов преобразования PDF/A.
