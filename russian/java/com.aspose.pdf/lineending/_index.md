Перечисляет стили окончания линии, которые будут использоваться при рисовании линии.
## Поля

| Поле | Описание |
| --- | --- |
| [Butt](#Butt) | Короткая линия в конечной точке, перпендикулярная самой линии. |
| [Circle](#Circle) | Круг, заполненный внутренним цветом аннотации, если таковой имеется. |
| [ClosedArrow](#ClosedArrow) | Две короткие линии, встречающиеся под острым углом, как в стиле OpenArrow, и соединенные третьей линией, чтобы сформировать треугольный замкнутый наконечник стрелки, заполненный внутренним цветом аннотации, если таковой имеется. |
| [Diamond](#Diamond) | Ромбовидная форма, заполненная внутренним цветом аннотации, если таковая имеется. |
| [EnumSeparatorCharArray](#EnumSeparatorCharArray) |  |
| [None](#None) | Нет окончания строки. |
| [OpenArrow](#OpenArrow) | Две короткие линии, сходящиеся под острым углом, образуют открытый наконечник стрелы. |
| [RClosedArrow](#RClosedArrow) | Треугольная замкнутая стрелка в обратном направлении от ClosedArrow. |
| [ROpenArrow](#ROpenArrow) | Две короткие линии в обратном направлении от OpenArrow. |
| [Slash](#Slash) | Короткая линия в конечной точке примерно на 30 градусов по часовой стрелке от перпендикуляра к самой линии. |
| [Square](#Square) | Квадрат, заполненный внутренним цветом аннотации, если он есть. |
