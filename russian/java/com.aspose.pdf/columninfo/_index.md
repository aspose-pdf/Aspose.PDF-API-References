Получает или задает строку, содержащую расстояние между столбцами. Значение каждого интервала должно быть отделено пробелом. Единицей измерения по умолчанию является точка, но также поддерживаются сантиметры и дюймы. Например, «120 2,5 см 1,5 дюйма».
