**com.aspose.pdf.text**Пакет предоставляет классы, которые позволяют извлекать текст, добавлять текст, манипулировать существующим текстом документа. Он также содержит классы, позволяющие извлекать, заменять, подставлять шрифты документа.

| [CustomFontSubstitutionBase.OriginalFontSpecification](../com.aspose.pdf.text/customfontsubstitutionbase.originalfontspecification) | Представляет исходную спецификацию шрифта. |
| [ExtractedTextPostProcessor](../com.aspose.pdf.text/extractedtextpostprocessor) | Выполняет постобработку экстрагированного текста |
| [FontSourceCollection](../com.aspose.pdf.text/fontsourcecollection) | Представляет коллекцию источников шрифтов. |
| [FontSubstitution](../com.aspose.pdf.text/fontsubstitution) | Только для внутреннего использования |

| [FontTypes](../com.aspose.pdf.text/fonttypes) | Перечисление поддерживаемых типов шрифтов. |

| [TextProcessingContext](../com.aspose.pdf.text/textprocessingcontext) | Представляет контекст обработки текста |
