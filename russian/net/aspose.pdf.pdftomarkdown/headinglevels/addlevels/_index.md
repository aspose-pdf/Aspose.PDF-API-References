## Метод HeadingLevels.AddLevels

Добавляет уровни заголовков. Коллекция размеров шрифта должна быть отсортирована по убыванию.

```csharp
public void AddLevels(ICollection<double> fontSizes)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fontSizes | ICollection`1 | Значения должны быть отсортированы в порядке убывания. |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentException | Если значение не отсортировано или значение меньше одного. |
