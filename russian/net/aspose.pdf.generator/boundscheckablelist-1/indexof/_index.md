## BoundsCheckableList&lt;T&gt;.IndexOf метод

Ищет указанный объект и возвращает индекс первого вхождения с нулевой базой в пределах всего System.Collections.Generic.List.

```csharp
public int IndexOf(T item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | T | Объект, который нужно найти в System.Collections.Generic.List. Значение может быть null для ссылочных типов. |

### Возвращаемое значение

Индекс первого вхождения *item* с нулевой базой в пределах всего System.Collections.Generic.List, если найден; в противном случае –1.
