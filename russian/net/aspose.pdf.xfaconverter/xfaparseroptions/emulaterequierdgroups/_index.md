## Свойство XfaParserOptions.EmulateRequierdGroups

Если это свойство истинно, то будут нарисованы дополнительные красные прямоугольники для обязательных "исключенных групп" Xfa. Это свойство было введено из-за отсутствия аналогов исключенных групп во время преобразования представления форм Xfa в стандарт. По умолчанию оно ложно.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
