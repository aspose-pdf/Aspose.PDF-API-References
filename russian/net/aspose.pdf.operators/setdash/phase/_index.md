## Свойство SetDash.Phase

Фаза штриха. Перед началом штриховки пути массив штрихов должен быть циклично обработан, суммируя длины штрихов и промежутков. Когда накопленная длина равна значению, указанному фазой штриха, начинается штриховка пути, и массив штрихов будет использоваться циклично с этого момента.

```csharp
public int Phase { get; set; }
```
