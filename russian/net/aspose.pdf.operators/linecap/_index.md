## Перечисление LineCap

Стиль окончания линии должен указывать форму, которая будет использоваться на концах открытых подпутей (и штрихов, если таковые имеются), когда они обводятся.

```csharp
public enum LineCap
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| ButtCap | `0` | Прямое окончание. Обводка будет квадратной на конце пути. Не будет выступа за конец пути. |
| RoundCap | `1` | Закругленное окончание. Полукруглая дуга с диаметром, равным ширине линии, будет нарисована вокруг конца и будет заполнена. |
| SquareCap | `2` | Выступающее квадратное окончание. Обводка будет продолжаться за конец пути на расстояние, равное половине ширины линии, и будет квадратной. |
