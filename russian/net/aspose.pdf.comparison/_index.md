## Классы

| Класс | Описание |
| --- | --- |
| [ComparisonOptions](./comparisonoptions/) | Представляет класс параметров сравнения PDF-документов. |
| [DiffOperation](./diffoperation/) | Представляет класс операции различий. |
| [DocumentComparisonStatistics](./documentcomparisonstatistics/) | Представляет класс статистики сравнения документов. |
| [GraphicalPdfComparer](./graphicalpdfcomparer/) | Представляет класс для графического сравнения PDF-документов. Должен использоваться для поиска небольших изменений, в основном графического характера. Для сравнения изменений текстового содержимого используйте другие классы сравнения PDF. |
| [HtmlDiffOutputGenerator](./htmldiffoutputgenerator/) | Представляет класс для генерации HTML-представления различий текстов. Удаленные разрывы строк обозначаются знаком абзаца. |
| [ImagesDifference](./imagesdifference/) | Представляет класс результата сравнения двух страниц PDF. |
| [JsonDiffOutputGenerator](./jsondiffoutputgenerator/) | Представляет класс для отображения результатов сравнения PDF-документов или страниц в формате JSON. |
| [MarkdownDiffOutputGenerator](./markdowndiffoutputgenerator/) | Представляет класс для генерации markdown-представления различий текстов. Из-за синтаксиса markdown невозможно показать изменения в символах пробела. Выбор изменений приводит к добавлению символов пробела вокруг форматирования, в противном случае просмотрщик markdown не сможет правильно отобразить текст. Удаленные разрывы строк обозначаются - знаком абзаца. |
| [OutputTextStyle](./outputtextstyle/) | Представляет класс набора стилей для маркировки изменений текста. |
| [PdfOutputGenerator](./pdfoutputgenerator/) | Представляет класс для генерации PDF-представления различий текстов. |
| [SideBySideComparisonOptions](./sidebysidecomparisonoptions/) | Представляет класс параметров для сравнения документов с боковым выводом. |
| [SideBySidePdfComparer](./sidebysidepdfcomparer/) |  |
| [TextItemComparisonStatistics](./textitemcomparisonstatistics/) | Представляет класс статистики сравнения текста. |
| [TextPdfComparer](./textpdfcomparer/) | Представляет класс для сравнения двух страниц PDF или PDF-документов. |
| [TextStyle](./textstyle/) | Представляет класс стиля текста. |
