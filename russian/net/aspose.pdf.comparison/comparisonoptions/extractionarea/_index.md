## Свойство ComparisonOptions.ExtractionArea

Получите и установите прямоугольную область, в которой будет сравниваться текст страниц. Эта опция не может быть установлена вместе с [`ExcludeTables`](../excludetables/), [`ExcludeAreas1`](../excludeareas1/) и [`ExcludeAreas2`](../excludeareas2/) опциями.

```csharp
public Rectangle ExtractionArea { get; set; }
```
