## Свойство ComparisonOptions.ExcludeTables

Получите и установите параметр, который определяет, исключены ли таблицы из сравнения. Этот параметр не может быть установлен вместе с параметром [`ExtractionArea`](../extractionarea/). Значение по умолчанию - `false`.

```csharp
public bool ExcludeTables { get; set; }
```
