## Свойство ComparisonOptions.ExcludeAreas1

Получите и установите исключенные области. Используется для первой страницы или документа в методе сравнения. Эта опция может быть установлена вместе с [`ExcludeTables`](../excludetables/). Эта опция не может быть установлена вместе с опцией [`ExtractionArea`](../extractionarea/).

```csharp
public Rectangle[] ExcludeAreas1 { get; set; }
```
