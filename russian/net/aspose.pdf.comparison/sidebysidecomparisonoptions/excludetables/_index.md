## Свойство SideBySideComparisonOptions.ExcludeTables

Получите и установите параметр, который определяет, исключаются ли таблицы из сравнения. Этот параметр не может быть установлен вместе с [`ComparisonArea1`](../comparisonarea1/) и [`ComparisonArea2`](../comparisonarea2/). Значение по умолчанию - `false`.

```csharp
public bool ExcludeTables { get; set; }
```
