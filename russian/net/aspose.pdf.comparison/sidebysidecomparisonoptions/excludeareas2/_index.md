## Свойство SideBySideComparisonOptions.ExcludeAreas2

Получите и установите исключенные области. Используется для второй страницы или документа в методе сравнения. Эта опция может быть установлена вместе с [`ExcludeTables`](../excludetables/). Эта опция не может быть установлена вместе с опцией [`ComparisonArea2`](../comparisonarea2/).

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
