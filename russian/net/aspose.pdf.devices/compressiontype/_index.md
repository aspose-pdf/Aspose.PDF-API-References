## Перечисление CompressionType

Используется для указания значения параметра, передаваемого устройству Tiff изображения.

```csharp
public enum CompressionType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| LZW | `0` | Указывает схему сжатия LZW. Может быть передан кодировщику Tiff в качестве параметра, относящегося к категории сжатия. |
| CCITT4 | `1` | Указывает схему сжатия CCITT4. Может быть передан кодировщику CCITT4 в качестве параметра, относящегося к категории сжатия. |
| CCITT3 | `2` | Указывает схему сжатия CCITT3. Может быть передан кодировщику CCITT3 в качестве параметра, относящегося к категории сжатия. |
| RLE | `3` | Указывает схему сжатия RLE. Может быть передан кодировщику RLE в качестве параметра, относящегося к категории сжатия. |
| None | `4` | Указывает отсутствие сжатия. Может быть передан кодировщику Tiff в качестве параметра, относящегося к категории сжатия. |
