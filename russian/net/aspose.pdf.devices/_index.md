Пространство имен **Aspose.Pdf.Devices** предоставляет классы, которые используются для представления документа в виде изображения(ий) или простого текста. Таким образом, документ может быть отправлен на текстовые или различные графические устройства, что означает, что мы хотим получить его текстовое или графическое представление.

## Классы

| Класс | Описание |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате bmp. |
| [Device](./device/) | Абстрактный класс для всех типов устройств. Устройство используется для представления pdf документа в некотором формате. Например, страницу документа можно представить в виде изображения или текста. |
| [DicomDevice](./dicomdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате Dicom. |
| [DocumentDevice](./documentdevice/) | Абстрактный класс для всех устройств, которые используются для обработки всего pdf документа. |
| [EmfDevice](./emfdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате emf. |
| [GifDevice](./gifdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате gif. |
| [ImageDevice](./imagedevice/) | Абстрактный класс для устройств изображения. |
| [JpegDevice](./jpegdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате jpeg. |
| [Margins](./margins/) | Этот класс представляет поля изображения. |
| [PageDevice](./pagedevice/) | Абстрактный класс для всех устройств, которые используются для обработки определенной страницы pdf документа. |
| [PngDevice](./pngdevice/) | Представляет устройство изображения, которое помогает сохранить страницы pdf документа в формате png. |
| [Resolution](./resolution/) | Представляет класс для хранения разрешения изображения. |
| [TextDevice](./textdevice/) | Представляет класс для преобразования страниц pdf документа в текст. |
| [ThumbnailDevice](./thumbnaildevice/) | Представляет устройство изображения, которое сохраняет страницы pdf документа в виде миниатюрного изображения. |
| [TiffDevice](./tiffdevice/) | Этот класс помогает сохранить страницы pdf документа по одной в одно изображение tiff. |
| [TiffSettings](./tiffsettings/) | Этот класс представляет настройки для импорта pdf в Tiff. |
## Перечисление

| Перечисление | Описание |
| --- | --- |
| [ColorDepth](./colordepth/) | Используется для указания значения параметра, передаваемого устройству изображения Tiff. |
| [CompressionType](./compressiontype/) | Используется для указания значения параметра, передаваемого устройству изображения Tiff. |
| [FormPresentationMode](./formpresentationmode/) | Используется для указания режима представления формы при печати или преобразовании pdf документов в изображения. |
| [ShapeType](./shapetype/) | Этот перечисляемый тип представляет тип формы для извлеченных изображений. |
