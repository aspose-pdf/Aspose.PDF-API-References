## Свойство RunCreateRequest.MaxCompletionTokens

Получает или задает максимальное количество токенов завершения, которые могут быть использованы в ходе выполнения. Выполнение будет стараться использовать только указанное количество токенов завершения на протяжении нескольких этапов выполнения. Если выполнение превышает указанное количество токенов завершения, выполнение завершится со статусом неполное. См. incomplete_details для получения дополнительной информации.

```csharp
public int? MaxCompletionTokens { get; set; }
```
