## Свойство RunCreateRequest.MaxPromptTokens

Получает или задает максимальное количество токенов подсказки, которые могут быть использованы в ходе выполнения. Выполнение будет прилагать все усилия, чтобы использовать только указанное количество токенов подсказки на протяжении нескольких этапов выполнения. Если выполнение превышает указанное количество токенов подсказки, выполнение завершится со статусом неполное. См. incomplete_details для получения дополнительной информации.

```csharp
public int? MaxPromptTokens { get; set; }
```
