## Свойство VectorStore.Metadata

Получает или устанавливает набор из 16 пар ключ-значение, которые могут быть прикреплены к векторному хранилищу. Это может быть полезно для хранения дополнительной информации о векторном хранилище в структурированном формате. Ключи могут иметь максимальную длину 64 символа, а значения могут иметь максимальную длину 512 символов.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
