## VectorStore class

Помощник для создания векторного хранилища с file_ids и его привязки к этому потоку. Максимальное количество векторных хранилищ, которые могут быть привязаны к потоку, составляет 1.

```csharp
public class VectorStore
```

| [FileIds](../../aspose.pdf.ai/vectorstore/fileids/) { get; set; } | Получает или задает список идентификаторов файлов для добавления в векторное хранилище. Максимальное количество файлов в векторном хранилище составляет 10000. |
| [Metadata](../../aspose.pdf.ai/vectorstore/metadata/) { get; set; } | Получает или задает набор из 16 пар ключ-значение, которые могут быть привязаны к векторному хранилищу.
