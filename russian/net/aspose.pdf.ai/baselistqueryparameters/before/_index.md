## Свойство BaseListQueryParameters.Before

Получает или устанавливает курсор для использования в пагинации. before - это идентификатор объекта, который определяет ваше место в списке. Например, если вы делаете запрос списка и получаете 100 объектов, заканчивающихся на obj_foo, ваш последующий вызов может включать before=obj_foo, чтобы получить предыдущую страницу списка.

```csharp
public string Before { get; set; }
```
