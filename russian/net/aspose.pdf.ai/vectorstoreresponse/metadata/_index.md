## Свойство VectorStoreResponse.Metadata

Получает или устанавливает набор из 16 пар ключ-значение, которые могут быть прикреплены к объекту. Это может быть полезно для хранения дополнительной информации об объекте в структурированном формате. Ключи могут иметь максимальную длину 64 символа, а значения - максимальную длину 512 символов.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
