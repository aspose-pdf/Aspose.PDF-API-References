## Свойство CompletionCreateRequest.TopP

Получает или устанавливает альтернативу выборке с температурой, называемую ядерной выборкой, где модель учитывает результаты токенов с вероятностной массой top_p. Таким образом, 0.1 означает, что учитываются только токены, составляющие верхние 10% вероятностной массы.

```csharp
public double? TopP { get; set; }
```
