## Свойство TruncationStrategy.StrategyType

Получает или устанавливает стратегию усечения, которую следует использовать для потока. Значение по умолчанию - auto. Если установлено значение last_messages, поток будет усечен до n самых последних сообщений в потоке. При установке значения auto сообщения в середине потока будут удалены, чтобы соответствовать длине контекста модели, max_prompt_tokens.

```csharp
public string StrategyType { get; set; }
```
