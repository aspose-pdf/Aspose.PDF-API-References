## Свойство FileSearch.VectorStores

Получает или устанавливает помощника для создания векторного хранилища с file_ids и прикрепления его к этому потоку. Максимальное количество векторных хранилищ, которое может быть прикреплено к потоку, составляет 1.

public List<VectorStore> VectorStores { get; set; }
