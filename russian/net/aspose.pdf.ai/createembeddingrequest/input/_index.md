## Свойство CreateEmbeddingRequest.Input

Получает или устанавливает входной текст для встраивания, закодированный в виде строки или массива токенов. Чтобы встроить несколько входных данных в одном запросе, передайте массив строк или массив массивов токенов. Входные данные не должны превышать максимальное количество входных токенов для модели (8192 токена для text-embedding-ada-002), не могут быть пустой строкой, и любой массив должен иметь размерность 2048 или меньше.

```csharp
public string Input { get; set; }
```
