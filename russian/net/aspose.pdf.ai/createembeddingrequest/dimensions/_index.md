## Свойство CreateEmbeddingRequest.Dimensions

Получает или задает количество измерений, которые должны иметь результирующие выходные встраивания. Поддерживается только в моделях text-embedding-3 и более поздних.

```csharp
public int? Dimensions { get; set; }
```
