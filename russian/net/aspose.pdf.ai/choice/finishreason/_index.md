## Свойство Choice.FinishReason

Получает или устанавливает причину, по которой модель прекратила генерацию токенов. Это произойдет, если модель достигнет естественной точки остановки или заданной длины последовательности остановки, если максимальное количество токенов, указанное в запросе, было достигнуто.

```csharp
public string FinishReason { get; set; }
```
