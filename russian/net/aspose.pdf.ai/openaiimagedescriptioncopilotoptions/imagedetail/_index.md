## Свойство OpenAIImageDescriptionCopilotOptions.ImageDetail

Получает или устанавливает уровень детализации изображения, если это указано пользователем. "low" использует меньше токенов, вы можете выбрать высокое разрешение, используя "high". Если не установлено, по умолчанию используется "auto".

```csharp
public string ImageDetail { get; set; }
```
