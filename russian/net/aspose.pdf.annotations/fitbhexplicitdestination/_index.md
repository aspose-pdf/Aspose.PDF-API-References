## Класс FitBHExplicitDestination

Представляет явное назначение, которое отображает страницу с вертикальной координатой top, расположенной на верхнем крае окна, и содержимое страницы увеличено ровно настолько, чтобы поместить всю ширину его ограничивающего прямоугольника в окне. Значение null для top указывает, что текущее значение этого параметра должно быть сохранено без изменений.

```csharp
public sealed class FitBHExplicitDestination : ExplicitDestination
```
