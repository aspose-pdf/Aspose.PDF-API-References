## Dash(int, int) {#constructor}

Конструктор для Dash. Определяет пунктирную границу с заданными штрихами и промежутками, которые остаются неизменными для всей пунктирной границы.

```csharp
public Dash(int on, int off)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| on | Int32 | Длина штриха. |
| off | Int32 | Длина промежутка. |

## Dash(int[]) {#constructor_1}

Конструктор для Dash. Определяет шаблон штрихов и промежутков, который будет использоваться для рисования пунктирной границы.

```csharp
public Dash(int[] pattern)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pattern | Int32[] | Массив штрихов (минимум два значения), определяющий шаблон штрихов и промежутков, который будет использоваться для рисования пунктирной границы. |
