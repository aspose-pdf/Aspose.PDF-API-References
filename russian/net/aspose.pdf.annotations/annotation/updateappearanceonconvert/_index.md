## Свойство Annotation.UpdateAppearanceOnConvert

Если истинно, внешний вид аннотации будет обновлен перед преобразованием документа PF в изображение. Это позволяет правильно конвертировать поля, но, вероятно, требует больше времени.

```csharp
public static bool UpdateAppearanceOnConvert { get; set; }
```
