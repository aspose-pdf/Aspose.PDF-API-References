## Индексатор PDF3DCrossSectionArray

Получает или задает [`PDF3DCrossSection`](../../pdf3dcrosssection/) по указанному индексу.

```csharp
public PDF3DCrossSection this[int index] { get; set; }
```

| Параметр | Описание |
| --- | --- |
| index | Индекс. |

### Возвращаемое значение

Сечение.

### Исключения

| исключение | условие |
| --- | --- |
| IndexOutOfRangeException | Неверный индекс: индекс должен быть в диапазоне [1..n], где n равно количеству сечений. |
