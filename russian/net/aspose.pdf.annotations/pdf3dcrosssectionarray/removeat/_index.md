## Метод PDF3DCrossSectionArray.RemoveAt

Удаляет сечение из массива по указанному индексу.

```csharp
public void RemoveAt(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс удаляемого сечения в массиве. |

### Исключения

| исключение | условие |
| --- | --- |
| IndexOutOfRangeException | Неверный индекс: индекс должен быть в диапазоне [1..n], где n равно количеству сечений. |
