## AppearanceDictionary indexer

Представляет удобную форму для получения потоков внешнего вида.

```csharp
public XForm this[string key] { get; set; }
```

| Параметр | Описание |
| --- | --- |
| key | Представляет путь к потоку появления. Если словарь внешнего вида имеет подсловари, то путь должен состоять из 2 частей ([`Keys`](../keys)), else path имеет только одну часть. |
