## Свойство PopupAnnotation.Parent

Получает или задает родительскую аннотацию, с которой будет связана эта всплывающая аннотация. Если эта запись присутствует, содержимое, M, C и T родительской аннотации будут переопределять соответствующие значения самой всплывающей аннотации.

```csharp
public Annotation Parent { get; set; }
```
