## FitBExplicitDestination class

Представляет явное назначение, которое отображает страницу с ее содержимым, увеличенным настолько, чтобы его ограничивающая рамка полностью помещалась в окне как по горизонтали, так и по вертикали. Если требуемые коэффициенты увеличения по горизонтали и вертикали различаются, используйте меньший из двух, центрируя ограничивающую рамку внутри окна в другом измерении.

```csharp
public sealed class FitBExplicitDestination : ExplicitDestination
```
