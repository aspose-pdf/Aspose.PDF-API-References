## Перечисление PrinterMarksKind

Указывает типы печатных знаков, которые должны быть добавлены в документ.

```csharp
[Flags]
public enum PrinterMarksKind
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Указывает, что печатные знаки не должны добавляться. |
| TrimMarks | `1` | Указывает, что должны быть добавлены обрезные знаки. |
| BleedMarks | `2` | Указывает, что должны быть добавлены знаки обреза. |
| RegistrationMarks | `4` | Указывает, что должны быть добавлены регистрационные знаки. |
| ColorBars | `8` | Указывает, что должны быть добавлены цветные полосы. |
| PageInformation | `10` | Указывает, что должна быть добавлена информация о странице. |
| All | `1F` | Указывает, что должны быть добавлены все типы печатных знаков. |

## Примечания

Это перечисление имеет атрибут FlagsAttribute, который позволяет побитово комбинировать его значения членов.
