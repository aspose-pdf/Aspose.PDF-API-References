## Свойство Measure.NumberFormat.ForceDenominator

Если FractionDisplayment равно ShowAsFraction, это значение определяет, может ли дробь быть сокращена или нет. Если значение истинно, дробь может не быть сокращена.

```csharp
public bool ForceDenominator { get; set; }
```
