## HighlightingMode enumeration

Перечисляет режим выделения аннотации, визуальный эффект, который будет использоваться при нажатии или удерживании кнопки мыши внутри ее активной области.

```csharp
public enum HighlightingMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Без выделения. |
| Invert | `1` | Инвертировать содержимое прямоугольника аннотации. |
| Outline | `2` | Инвертировать границу аннотации. |
| Push | `3` | Отображение нижнего вида аннотации, если таковая имеется. Если внешний вид вниз не определен, сместите содержимое прямоугольника аннотации, чтобы оно отображалось так, как если бы оно было протолкнуто ниже поверхности страницы. |
| Toggle | `4` | То же, что и Push (что предпочтительнее). |
