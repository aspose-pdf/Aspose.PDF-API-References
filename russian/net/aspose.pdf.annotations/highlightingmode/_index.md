## Перечисление HighlightingMode

Перечисляет режим выделения аннотации, визуальный эффект, который будет использоваться, когда кнопка мыши нажата или удерживается внутри его активной области.

```csharp
public enum HighlightingMode
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| None | `0` | Без выделения. |
| Invert | `1` | Инвертировать содержимое прямоугольника аннотации. |
| Outline | `2` | Инвертировать границу аннотации. |
| Push | `3` | Отобразить внешний вид аннотации при нажатии, если таковой имеется. Если внешний вид при нажатии не определен, сместить содержимое прямоугольника аннотации так, чтобы оно выглядело так, как будто его нажимают ниже поверхности страницы. |
| Toggle | `4` | То же самое, что и Push (что предпочтительнее). |
