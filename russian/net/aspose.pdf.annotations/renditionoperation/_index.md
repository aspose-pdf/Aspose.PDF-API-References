## Перечисление RenditionOperation

Операция, которую необходимо выполнить при срабатывании действия.

```csharp
public enum RenditionOperation
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| PlayStop | `0` | Если с аннотацией не связана никакая рендеринг, воспроизвести указанную рендеринг, связав ее с аннотацией. Если рендеринг уже связана с аннотацией, она должна быть остановлена, и новая рендеринг должна быть связана с аннотацией. |
| Stop | `1` | Остановить любую рендеринг, воспроизводимую в связи с аннотацией. |
| Pause | `2` | Приостановить любую рендеринг, воспроизводимую в связи с аннотацией. |
| Resume | `3` | Возобновить любую рендеринг, воспроизводимую в связи с аннотацией. |
| PlayResume | `4` | Воспроизвести указанную рендеринг, связав ее с аннотацией. Если рендеринг уже связана с аннотацией, возобновить рендеринг, если она приостановлена. |
| Undefined | `-1` | Операция не определена. |
