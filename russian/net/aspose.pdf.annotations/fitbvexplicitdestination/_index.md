## FitBVExplicitDestination class

Представляет явное назначение, которое отображает страницу с горизонтальной координатой left, расположенной на левом краю окна, и содержимое страницы увеличено ровно настолько, чтобы поместить всю высоту его ограничивающего прямоугольника в окне. Значение null для left указывает, что текущее значение этого параметра должно оставаться неизменным.

```csharp
public sealed class FitBVExplicitDestination : ExplicitDestination
```
