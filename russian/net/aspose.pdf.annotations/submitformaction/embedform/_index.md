## Поле SubmitFormAction.EmbedForm

Если установлено, то запись F отправленного FDF будет спецификацией файла, содержащей встроенный файловый поток, представляющий PDF-файл, из которого отправляется FDF.

```csharp
public const int EmbedForm;
```
