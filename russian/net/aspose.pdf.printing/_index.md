Пространство имен **Aspose.Pdf.Printing** предоставляет классы и функции для обработки операций печати PDF, включая настройки принтера, настройки страниц и расширения для управления конфигурациями, связанными с печатью.

## Классы

| Класс | Описание |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Предоставляет данные для события [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Указывает настройки, которые применяются к одной напечатанной странице. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Представляет методы расширения для [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Указывает размер листа бумаги. |
| [PaperSizeExtensions](./papersizeextensions/) | Представляет методы расширения для [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Представляет стандартные размеры бумаги. |
| [PaperSource](./papersource/) | Указывает лоток бумаги, из которого принтер получает бумагу. |
| [PaperSourceExtensions](./papersourceextensions/) | Представляет методы расширения для [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Предоставляет набор предопределенных экземпляров [`PaperSource`](../aspose.pdf.printing/papersource/), представляющих общие источники бумаги. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Предоставляет данные для события [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Представляет разрешение, поддерживаемое принтером. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Представляет методы расширения для [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Указывает информацию о том, как документ печатается, включая принтер, который его печатает. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Представляет методы расширения для [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Предоставляет данные для событий [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) и [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) класса [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
## Перечисление
