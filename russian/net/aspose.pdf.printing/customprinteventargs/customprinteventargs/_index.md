## Конструктор CustomPrintEventArgs

Инициализирует [`CustomPrintEventArgs`](../) с заданными настройками принтера и страницы.

```csharp
public CustomPrintEventArgs(string fileName, PrinterSettings printerSettings, 
    PageSettings pageSettings)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fileName | String | Имя файла, который печатается. |
| printerSettings | PrinterSettings | Информация о принтере, на котором должен быть напечатан документ. |
| pageSettings | PageSettings | Настройки, которые должны быть применены к каждой странице документа. |
