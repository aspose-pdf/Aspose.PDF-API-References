## Метод XFormPlacement.AddOnPage

Добавляет текущий элемент на страницу. Если нужно добавить много элементов, лучше использовать [`AddGraphics`](../../../aspose.pdf/page/addgraphics/).

```csharp
public override void AddOnPage(Page destination)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| destination | Page | Целевая страница |
