## Свойство SvgExtractionOptions.StrictExtractionAreaBoundCheck

Получает и устанавливает опцию для строгой проверки, находятся ли подпути в указанном прямоугольнике в [`ExtractionAreaBound`](../extractionareabound/). Если установлено значение false, то подпути, которые не полностью включены в [`ExtractionAreaBound`](../extractionareabound/), будут извлечены. Значение по умолчанию - `True`.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
