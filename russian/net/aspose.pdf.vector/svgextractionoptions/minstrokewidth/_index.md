## Свойство SvgExtractionOptions.MinStrokeWidth

Получает или задает минимальную ширину штриха, которая будет использоваться в результирующем SVG. Если в PDF используется более тонкая ширина штриха, она будет заменена на эту ширину. Значение по умолчанию - 0.5.

```csharp
public double MinStrokeWidth { get; set; }
```

## Примечания

Значение выражается в преобразованных единицах пользовательского пространства страницы PDF. По умолчанию 1 единица пользовательского пространства равна 1/72 дюйма (0.35 мм), но это может быть переопределено документом PDF. Преобразования могут повлиять на фактическую минимальную ширину в сгенерированном SVG.
