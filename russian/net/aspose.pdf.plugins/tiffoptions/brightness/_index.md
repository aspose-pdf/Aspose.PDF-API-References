## Свойство TiffOptions.Brightness

Получает или устанавливает границу значения преобразования цветов в белый и черный. Этот параметр может быть применен с EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle или ColorDepth.Format1bpp == 1

```csharp
public float Brightness { get; set; }
```

### Значение свойства

Значение яркости должно быть в диапазоне от 0 до 1. По умолчанию значение равно 0.33f
