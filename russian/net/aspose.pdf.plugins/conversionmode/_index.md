## Перечисление ConversionMode

Определяет режим конвертации выходного документа.

```csharp
public enum ConversionMode
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| TextBox | `0` | Этот режим быстрый и хорошо подходит для максимального сохранения оригинального вида PDF-файла, но редактируемость полученного документа может быть ограничена. |
| Flow | `1` | Режим полной распознаваемости, движок выполняет группировку и многоуровневый анализ, чтобы восстановить намерение автора оригинального документа и создать максимально редактируемый документ. Недостаток в том, что выходной документ может выглядеть иначе, чем оригинальный PDF-файл. |
| EnhancedFlow | `2` | Альтернативный режим Flow, который поддерживает распознавание таблиц. |
