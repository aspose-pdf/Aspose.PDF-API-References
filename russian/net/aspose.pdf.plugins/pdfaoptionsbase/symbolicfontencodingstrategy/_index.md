## Свойство PdfAOptionsBase.SymbolicFontEncodingStrategy

Получает или устанавливает стратегию кодирования символических шрифтов при конвертации в формат PDF/A.

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### Значение свойства

Стратегия кодирования символического шрифта.

## Примечания

Это свойство позволяет вам контролировать, какая подтаблица CMap будет скопирована в результирующий документ в случаях, когда оригинальный символический шрифт TrueType в исходном документе содержит несколько подтаблиц CMap.
