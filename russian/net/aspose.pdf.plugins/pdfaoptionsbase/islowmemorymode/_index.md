## Свойство PdfAOptionsBase.IsLowMemoryMode

Получает или задает значение, указывающее, включен ли режим низкой памяти во время процесса конвертации PDF/A.

```csharp
public bool IsLowMemoryMode { get; set; }
```

### Значение свойства

`true`, если режим низкой памяти включен; в противном случае `false`.
