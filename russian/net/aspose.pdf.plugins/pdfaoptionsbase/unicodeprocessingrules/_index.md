## Свойство PdfAOptionsBase.UnicodeProcessingRules

Получает или задает правила обработки таблиц ToUnicode CMap и не связанных с символами Unicode в процессе конвертации PDF/A.

```csharp
public ToUnicodeProcessingRules UnicodeProcessingRules { get; set; }
```

### Значение свойства

Экземпляр [`ToUnicodeProcessingRules`](../../../aspose.pdf/tounicodeprocessingrules/), который определяет правила обработки проблемных случаев с символами Unicode.
