## Перечисление ValidationMethod

Представляет перечисление, определяющее метод, используемый для проверки сертификатов.

```csharp
public enum ValidationMethod
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Auto | `0` | Автоматически определяет лучший метод для проверки сертификатов. |
| Ocsp | `1` | Использует Протокол статуса онлайн-сертификатов (OCSP) для проверки сертификатов. OCSP — это протокол, который предоставляет статус проверки сертификата, запрашивая его непосредственно у выдавшего удостоверяющего центра (CA). |
| Crl | `2` | Проверяет сертификаты с использованием метода списка отозванных сертификатов (CRL). |
| All | `3` | Использует все доступные методы (OCSP и CRL) для проверки сертификатов. |
