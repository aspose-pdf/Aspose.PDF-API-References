## Перечисление ValidationMode

Указывает режим проверки для процессов проверки подписи PDF.

```csharp
public enum ValidationMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Представляет режим, в котором проверка не выполняется. |
| OnlyCheck | `1` | Представляет режим, в котором проверка выполняется, но ее результат не влияет на проверку цифровой подписи. Вы можете самостоятельно проверить результат проверки. |
| Strict | `2` | Представляет режим, в котором проверка выполняется, и ее результат влияет на проверку цифровой подписи. Если сертификат не может быть проверен, то цифровая подпись будет считаться недействительной. Вы можете самостоятельно проверить результат проверки. |
