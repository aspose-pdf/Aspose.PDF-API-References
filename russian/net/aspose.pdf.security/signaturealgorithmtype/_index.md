## Перечисление SignatureAlgorithmType

Перечисляет типы алгоритмов подписи, используемых для цифровых подписей.

```csharp
public enum SignatureAlgorithmType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Ecdsa | `0` | Алгоритм цифровой подписи на основе эллиптической кривой (ECDSA), используемый для цифровых подписей. |
| Rsa | `1` | Алгоритм Ривеста–Шамира–Адлемана (RSA), используемый для цифровых подписей. |
| Dsa | `2` | Алгоритм цифровой подписи (DSA), используемый для цифровых подписей. |
| Timestamp | `3` | Метка времени, используемая в качестве алгоритма подписи для обеспечения цифровых подписей. |
| Unknown | `4` | Указывает, что тип алгоритма подписи неизвестен. |
