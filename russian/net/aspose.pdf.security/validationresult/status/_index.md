## Свойство ValidationResult.Status

Получает статус процесса проверки сертификата.

```csharp
public ValidationStatus Status { get; }
```

## Примечания

Свойство Status указывает на результат проверки сертификата. Возможные значения определены в перечислении [`ValidationStatus`](../../validationstatus/), такие как Valid, Invalid или Undefined. Оно дает представление о том, прошел ли сертификат проверки или нет.
