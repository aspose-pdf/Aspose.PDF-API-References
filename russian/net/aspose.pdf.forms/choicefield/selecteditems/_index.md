## Свойство ChoiceField.SelectedItems

Получает или устанавливает массив выбранных элементов. Для списка с множественным выбором массив содержит более одного элемента. Для списка с одиночным выбором он содержит один элемент.

```csharp
public virtual int[] SelectedItems { get; set; }
```
