## Свойство IconFit.SpreadOnBorder

Если истинно, указывает, что внешний вид кнопки должен быть масштабирован, чтобы полностью вписаться в границы аннотации, не принимая во внимание ширину линии границы.

```csharp
public bool SpreadOnBorder { get; set; }
```
