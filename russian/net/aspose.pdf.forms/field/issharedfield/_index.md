## Свойство Field.IsSharedField

Свойство для поддержки Generator. Используется, когда поле добавляется в заголовок или нижний колонтитул. Если true, это поле будет создано один раз, и его внешний вид будет виден на всех страницах документа. Если false, будет создано отдельное поле для каждой страницы документа.

```csharp
public bool IsSharedField { get; set; }
```
