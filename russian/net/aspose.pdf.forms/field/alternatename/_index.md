## Свойство Field.AlternateName

Получает или устанавливает альтернативное имя поля (Альтернативное имя поля, которое будет использоваться вместо фактического имени поля, где бы поле ни идентифицировалось в пользовательском интерфейсе). Альтернативное имя используется в качестве подсказки поля в Adobe Acrobat.

```csharp
public string AlternateName { get; set; }
```
