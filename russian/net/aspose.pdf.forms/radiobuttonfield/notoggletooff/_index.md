## Свойство RadioButtonField.NoToggleToOff

Получает или устанавливает флаг, который позволяет радиокнопке не иметь выбранного значения. Если `true`, то в любой момент времени должна быть выбрана ровно одна радиокнопка; выбор в данный момент выбранной кнопки не имеет эффекта. Если `false`, нажатие на выбранную кнопку снимает с нее выбор, оставляя ни одной кнопки невыбранной.

```csharp
public bool NoToggleToOff { get; set; }
```

## Примечания

Некоторые PDF-ридеры (включая Adobe Acrobat) могут игнорировать состояние `false` флага.
