## Signature.ShowProperties property

Принудительно показать/скрыть свойства подписи. В случае, если ShowProperties имеет значение true, поле подписи имеет предопределенный формат отображения (строки для представления): --------------------- ---------------------- Цифровая подпись {субъект сертификата} Дата: {signature.Date} Причина: {signature.Reason} Местоположение: { подпись.Расположение} ---------------------------------------------------------- где {X} является заполнителем для значения X. Также подпись может иметь изображение, в этом случае перечисленные строки размещаются поверх изображения. ShowProperties по умолчанию имеет значение true.

```csharp
public bool ShowProperties { get; set; }
```
