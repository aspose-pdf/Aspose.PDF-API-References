## AddOption(string) {#addoption}

Добавляет новый флажок в группу флажков, в которой в любой момент может быть отмечен не более одного флажка. Новый флажок добавляется в конец группы.

```csharp
public void AddOption(string optionName)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| optionName | String | Значение опции, представленной добавленным флажком. |
