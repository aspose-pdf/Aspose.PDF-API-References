## Свойство BarcodeField.ECC

Получает целочисленное значение, представляющее коэффициент коррекции ошибок. Для PDF417 должно быть от 0 до 8. Для QRCode должно быть от 0 до 3 (0 для 'L', 1 для 'M', 2 для 'Q' и 3 для 'H').

```csharp
public int ECC { get; }
```
