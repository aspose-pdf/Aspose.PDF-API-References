## Метод Form.AddFieldAppearance

Добавляет дополнительный внешний вид поля на указанной странице документа в указанном месте.

```csharp
public void AddFieldAppearance(Field field, int pageNumber, Rectangle rect)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| field | Field | Поле, внешний вид которого должен быть добавлен на форму. |
| pageNumber | Int32 | Номер страницы, на которой поле должно быть размещено. |
| rect | Rectangle | Прямоугольник, в котором будет размещено поле. |
