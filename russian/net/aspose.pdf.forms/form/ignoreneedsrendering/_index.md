## Свойство Form.IgnoreNeedsRendering

Если это свойство истинно, значение ключа NeedsRendering будет игнорироваться при конвертации формы XFA в стандартную форму. По умолчанию оно ложно.

```csharp
public bool IgnoreNeedsRendering { get; set; }
```
