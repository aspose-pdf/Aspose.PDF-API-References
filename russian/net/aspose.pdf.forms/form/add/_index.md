## Add(Field, string, int) {#add}

Добавляет в форму новое поле; Если это поле уже размещено на другой или этой форме, создается копия поля.

```csharp
public Field Add(Field field, string partialName, int pageNumber)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| field | Field | Имя поля. |
| partialName | String | Имя поля в форме. |
| pageNumber | Int32 | Номер страницы, на которую будет добавлено поле. |

### Возвращаемое значение

Возвращается добавленное поле. Если копия поля была создана, она будет возвращена.
