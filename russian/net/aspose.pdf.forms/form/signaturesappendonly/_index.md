## Свойство Form.SignaturesAppendOnly

Если установлено, документ содержит подписи, которые могут быть недействительными, если файл сохранен (записан) таким образом, что изменяет его предыдущие содержимое, в отличие от инкрементального обновления.

```csharp
public bool SignaturesAppendOnly { get; set; }
```
