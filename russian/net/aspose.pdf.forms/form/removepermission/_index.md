## Свойство Form.RemovePermission

Если это свойство истинно, словарь "Perms" будет удален из pdf документа после конвертации динамических документов в стандартные. Словарь "Perms" может содержать правила, которые мешают отображению выбора обязательных полей в Adobe Acrobat reader. По умолчанию это значение ложное.

```csharp
public bool RemovePermission { get; set; }
```
