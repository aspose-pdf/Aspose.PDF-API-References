## Свойство SignatureField.Signature

Получает объект подписи. Этот объект содержит данные подписи в соответствии со стандартами криптографии с открытым ключом. Классы [`PKCS1`](../../pkcs1/), [`PKCS7`](../../pkcs7/) и [`PKCS7Detached`](../../pkcs7detached/) представляют все поддерживаемые типы объектов подписи.

```csharp
public Signature Signature { get; }
```
