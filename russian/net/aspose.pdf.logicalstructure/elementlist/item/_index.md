## Метод ElementList.Item

Извлекает элемент по заданному индексу.

```csharp
public abstract Element Item(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс в списке элементов. |

### Возвращаемое значение

Элемент с указанным индексом в коллекции. Если *index* больше или равен количеству элементов в списке, возвращается null.
