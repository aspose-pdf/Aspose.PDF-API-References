## Поле StructureTypeStandard.TBody

(Группа строк тела таблицы; PDF 1.5) Группа строк, которые составляют основную часть тела таблицы. Если таблица разбита на несколько страниц, область тела может быть разделена на границе строки. Таблица может иметь несколько элементов TBody, чтобы позволить рисовать границу или фон для набора строк.

```csharp
public static readonly StructureTypeStandard TBody;
```
