## StructureTypeStandard.TOC field

(Оглавление) Список, состоящий из записей элементов оглавления (тип структуры TOCI) и/или других вложенных записей оглавления (TOC).

Запись TOC, которая включает только записи TOCI, представляет собой плоскую иерархию. Запись TOC, которая включает в себя другие вложенные записи TOC (и, возможно, записи TOCI), представляет собой более сложную иерархию. В идеале иерархия записи TOC верхнего уровня отражает структуру основной части документа.
