## StructureTypeStandard.Ruby field

(Ruby; PDF 1.5) Побочное примечание (аннотация), написанное шрифтом меньшего размера и помещенное рядом с основным текстом, к которому оно относится. Элемент Ruby может также содержать элементы RB, RT и RP.

(Ruby) Обертка всей сборки ruby. Он должен содержать один элемент RB, за которым следует либо элемент RT, либо группа из трех элементов, состоящая из RP, RT и RP. Элементы Ruby и элементы их содержимого не должны разбиваться на несколько строк.
