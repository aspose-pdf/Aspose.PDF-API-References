## Поле StructureTypeStandard.TFoot

(Группа строк нижнего колонтитула таблицы; PDF 1.5) Группа строк, которые составляют нижний колонтитул таблицы. Если таблица разбита на несколько страниц, эти строки могут быть перерисованы внизу каждого фрагмента таблицы (хотя существует только один элемент TFoot.)

public static readonly StructureTypeStandard TFoot;
