## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace property

Некоторые шрифты не предоставляют информацию об юникодах для некоторых текстовых символов. Это отсутствие информации вызывает ошибку «Текст не может быть сопоставлен с Unicode». Используйте этот флаг для сопоставления несвязанных символов с «пробелом» Unicode (код 32).
