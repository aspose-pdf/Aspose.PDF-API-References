**Aspose.Pdf.Оптимизация** - это пространство имен для классов, управляющих процессом оптимизации документа.

## Классы

| Класс | Описание |
| --- | --- |
| [ImageCompressionOptions](./imagecompressionoptions/) | Класс содержит набор параметров для сжатия изображений. |
| [OptimizationOptions](./optimizationoptions/) | Класс, который описывает алгоритм оптимизации документа. Экземпляр этого класса может быть использован в качестве параметра метода OptimizeResources(). |
## Перечисление

| Перечисление | Описание |
| --- | --- |
| [ImageCompressionVersion](./imagecompressionversion/) | Описывает версии алгоритма сжатия изображений. |
| [ImageEncoding](./imageencoding/) | Типы кодирования изображений. |
