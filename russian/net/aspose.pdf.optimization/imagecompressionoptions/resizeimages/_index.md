## Свойство ImageCompressionOptions.ResizeImages

Если этот флаг установлен в true и CompressImages равно true, изображения будут изменены в размере, если разрешение изображения больше, чем указанный параметр MaxResolution.

```csharp
public bool ResizeImages { get; set; }
```
