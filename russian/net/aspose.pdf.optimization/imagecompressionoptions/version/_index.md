## ImageCompressionOptions.Version property

Версия алгоритма сжатия. Возможные значения:1. стандартное сжатие, 2. быстрое (улучшенное сжатие, которое быстрее стандартного, но может быть применимо не ко всем изображениям), 3. смешанное (стандартное сжатие применяется к изображениям, которые не могут быть сжаты более быстрым алгоритмом, это может дать наилучшее сжатие, но более медленное, чем "быстрый" алгоритм. Версия "Быстрая" не применима для изменения размера изображений (будет использоваться стандартный метод). По умолчанию "Стандартная".

public ImageCompressionVersion Version { get; set; }
