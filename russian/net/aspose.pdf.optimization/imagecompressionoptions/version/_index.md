## Свойство ImageCompressionOptions.Version

Версия алгоритма сжатия. Возможные значения: 1. стандартное сжатие, 2. быстрое (улучшенное сжатие, которое быстрее стандартного, но может не применяться ко всем изображениям), 3. смешанное (стандартное сжатие применяется к изображениям, которые не могут быть сжаты более быстрым алгоритмом, это может дать лучшее сжатие, но медленнее, чем "быстрый" алгоритм. Версия "Fast" не применима для изменения размера изображений (будет использоваться стандартный метод). По умолчанию "Standard".

```csharp
public ImageCompressionVersion Version { get; set; }
```
