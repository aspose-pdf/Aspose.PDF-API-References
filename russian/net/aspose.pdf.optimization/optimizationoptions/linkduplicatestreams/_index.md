## Свойство OptimizationOptions.LinkDuplicateStreams

Если этот флаг установлен в true, потоки ресурсов будут проанализированы. Если найдены дублирующиеся потоки (т.е. если содержимое потоков одинаково), то эти потоки будут сохранены как один объект. Это позволяет уменьшить размер документа в некоторых случаях (например, когда один и тот же документ был конкатенирован несколько раз).

```csharp
public bool LinkDuplicateStreams { get; set; }
```
