## OptimizationOptions.RemoveUnusedObjects property

Если для этого флага установлено значение true, все объекты документа будут проверены, а неиспользуемые объекты (т.е. объекты, на которые нет ссылок) будут удалены из документа.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
