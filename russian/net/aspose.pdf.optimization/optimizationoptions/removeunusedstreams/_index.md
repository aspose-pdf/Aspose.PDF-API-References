## Свойство OptimizationOptions.RemoveUnusedStreams

Если этот флаг установлен в true, каждый ресурс проверяется на его использование. Если ресурс никогда не используется, то он удаляется. Это может уменьшить размер документа, например, когда страницы были извлечены из документа.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
