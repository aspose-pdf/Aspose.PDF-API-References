## Пространства имен

| Пространство имен | Описание |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | **Aspose.Pdf** — это корневое пространство имен для всех классов библиотеки Aspose.Pdf, которые находятся либо непосредственно в нем, такие как **Document**, либо косвенно через несколько подпространств имен. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | Пространство имен **Aspose.Pdf.AI** предоставляет классы для функциональности ИИ, включая API-клиентов и интеллектуальных помощников. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | Пространство имен **Aspose.Pdf.Annotations** предоставляет классы для работы с различными типами действий, назначений и другими функциями документа, которые традиционно называются интерактивными, позволяя пользователю взаимодействовать с ним. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | Пространство имен **Aspose.Pdf.Comparison** предоставляет классы, которые используются для сравнения документов. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | Пространство имен **Aspose.Pdf.DataEditor** содержит инструменты для редактирования данных внутри документа. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | Пространство имен **Aspose.Pdf.Devices** предоставляет классы, которые используются для представления документа в виде изображения или простого текста. Таким образом, документ может быть отправлен на текстовые или различные графические устройства, что означает, что мы хотим получить его текстовое или графическое представление. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | Пространство имен **Aspose.Pdf.Drawing** предоставляет классы, описывающие графические примитивы, такие как дуга, линия, круг и т. д. Соответствующие объекты могут быть использованы для создания новых документов. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | Пространство имен **Aspose.Pdf.Facades** предоставляет классы, которые изначально пришли из Aspose.Pdf.Kit. Эти классы используются для манипуляции документами, выполняя операции, такие как конкатенация, штампование, подпись, аннотирование и т. д., но на высоком уровне без доступа к внутренней структуре документа. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | Пространство имен **Aspose.Pdf.Forms** содержит классы, которые описывают формы (стандартные, статические, динамические) и различные типы полей, такие как текстовое поле, список, радиокнопка и т. д. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | **Aspose.Pdf.Tagged.LogicalStructure** — это пространство имен для классов логической структуры PDF. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | Пространство имен **Aspose.Pdf.Multithreading** предоставляет классы, которые предназначены для использования в многопоточных сценариях. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | **Aspose.Pdf.Operators** — это пространство имен для реализаций операторов. Эти классы описывают операторы, используемые в содержимом страниц PDF. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | **Aspose.Pdf.Optimization** — это пространство имен для классов, управляющих процессом оптимизации документа. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | **Aspose.Pdf.Plugins** — это корневое пространство имен для всех классов Aspose.Pdf.Plugins, которые находятся либо непосредственно в нем, такие как **PdfOrganizer**, либо косвенно через несколько подпространств имен. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | Пространство имен **Aspose.Pdf.Printing** предоставляет классы и функции для обработки операций печати PDF, включая настройки принтера, настройки страниц и расширения для управления конфигурациями, связанными с печатью. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | **Aspose.Pdf.Sanitization** — это пространство имен для операций по санации. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | Пространство имен **Aspose.Pdf.Security** содержит классы, используемые для шифрования и цифровой подписи. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | Пространство имен **Aspose.Pdf.Structure** предоставляет классы, которые помогают работать с логической структурой документа, представляющей организацию документа на главы и разделы или идентификацию специальных элементов, таких как рисунки, таблицы и сноски. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | **Aspose.Pdf.Tagged** — это пространство имен для классов, поддерживающих документы Tagged PDF. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | Пространство имен **Aspose.Pdf.Text** предоставляет классы, которые позволяют извлекать текст, добавлять текст, манипулировать существующим текстом документа. Оно также содержит классы, которые позволяют извлекать, заменять, подменять шрифты документа. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | **Aspose.Pdf.Vector** — это корневое пространство имен для графических операций. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | Пространство имен **Aspose.Pdf.XfaConverter** предоставляет классы, которые позволяют конвертировать XFA. |
