## Перечисление DigestHashAlgorithm

Представляет тип алгоритма, который сопоставляет данные с "хешем"

```csharp
public enum DigestHashAlgorithm
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Auto | `0` | Автоматическая настройка алгоритма хеширования по усмотрению алгоритма подписи. Для EDCSA значение по умолчанию определяется размером ключа. Значение по умолчанию для неподписанного PKCS7 - Sha1. |
| Sha1 | `1` | SHA-1. Алгоритм безопасного хеширования 1. Это значение по умолчанию для неподписанного PKCS7. |
| Sha256 | `2` | SHA-256. Алгоритм безопасного хеширования 2. Это значение по умолчанию для подписанного PKCS7. |
| Sha384 | `3` | SHA-384. Алгоритм безопасного хеширования 2 |
| Sha512 | `4` | SHA-512. Алгоритм безопасного хеширования 2 |
| Sha3_256 | `5` | SHA3-256. Алгоритм безопасного хеширования 3 |
| Sha3_384 | `6` | SHA3-384. Алгоритм безопасного хеширования 3 |
| Sha3_512 | `7` | SHA3-512. Алгоритм безопасного хеширования 3 |
