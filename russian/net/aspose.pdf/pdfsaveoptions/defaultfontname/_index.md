## PdfSaveOptions.DefaultFontName property

Имя шрифта, используемое по умолчанию для шрифтов, отсутствующих на компьютере. Когда документ PDF, сохраненный в формате PDF, содержит шрифты, недоступные в самом документе и на устройстве, API заменяет эти шрифты шрифтом по умолчанию (если шрифт с`DefaultFontName` находится на устройстве)

```csharp
public string DefaultFontName { get; set; }
```
