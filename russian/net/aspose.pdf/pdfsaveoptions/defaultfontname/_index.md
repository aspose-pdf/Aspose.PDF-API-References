## Свойство PdfSaveOptions.DefaultFontName

Имя шрифта, используемое по умолчанию для шрифтов, отсутствующих на компьютере. Когда PDF-документ, который сохраняется в PDF, содержит шрифты, которые недоступны в самом документе и на устройстве, API заменяет эти шрифты на шрифт по умолчанию (если шрифт с `DefaultFontName` найден на устройстве).

```csharp
public string DefaultFontName { get; set; }
```
