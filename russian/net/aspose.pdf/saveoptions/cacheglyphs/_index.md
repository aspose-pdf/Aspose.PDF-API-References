## Свойство SaveOptions.CacheGlyphs

Получает или устанавливает логическое значение, которое указывает, будут ли глифы шрифта кэшироваться при подготовке страниц aps. Улучшает производительность конвертации pdf в другие форматы, но увеличивает потребление памяти.

```csharp
public bool CacheGlyphs { get; set; }
```
