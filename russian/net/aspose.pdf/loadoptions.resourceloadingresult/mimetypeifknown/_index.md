## Поле LoadOptions.ResourceLoadingResult.MIMETypeIfKnown

Иногда знание о MIME-типе загружаемого ресурса полезно для конвертера. Вы можете предоставить MIME-тип (если он известен после загрузки) в этом параметре. Пожалуйста, оставьте параметр равным null, когда MIME-тип неизвестен или его не нужно указывать.

```csharp
public string MIMETypeIfKnown;
```
