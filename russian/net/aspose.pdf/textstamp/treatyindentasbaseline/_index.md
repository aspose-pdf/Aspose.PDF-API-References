## TextStamp.TreatYIndentAsBaseLine property

Определяет начало координат для размещения текста. Если TreatYIndentAsBaseLine = true (по умолчанию, когда Draw = true), значение YIndent будет рассматриваться как базовая линия текста. Если TreatYIndentAsBaseLine = false (по умолчанию, когда Draw = false), значение YIndent будет рассматриваться как нижнее ( линия спуска) текста.

public bool TreatYIndentAsBaseLine { get; set; }
