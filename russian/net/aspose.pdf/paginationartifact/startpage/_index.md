## Свойство PaginationArtifact.StartPage

Получает или задает номер начальной страницы для артефакта. Значение должно быть больше или равно 1. Если установлено значение меньше 1, оно будет скорректировано до 1.

```csharp
public int StartPage { get; set; }
```
