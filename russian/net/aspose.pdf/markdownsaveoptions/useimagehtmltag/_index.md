## Свойство MarkdownSaveOptions.UseImageHtmlTag

Получает и устанавливает разрешение на использование тега img для вставки изображений слева и справа от текста. В этом случае, в просмотрщике markdown, текст будет обвиваться вокруг изображения.

```csharp
public bool UseImageHtmlTag { get; set; }
```
