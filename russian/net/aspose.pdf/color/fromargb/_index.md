## FromArgb(int, int, int) {#fromargb}

Получает действительный объект цвета pdf из компонентов цвета RGB.

```csharp
public static Color FromArgb(int r, int g, int b)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| r | Int32 | Компонент красного цвета (значение 0 - 255). |
| g | Int32 | Компонент зеленого цвета (значение 0 - 255). |
| b | Int32 | Компонент синего цвета (значение 0 - 255). |

### Возвращаемое значение

Объект цвета с каждым значением компонента в диапазоне [0..255].
