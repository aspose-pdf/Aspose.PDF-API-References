## HtmlSaveOptions.RasterImagesSavingModes enumeration

Преобразованный PDF может содержать растровые изображения (.png, *.jpeg и т.д.) Это перечисление определяет методы обработки растровых изображений при преобразовании PDF в HTML

```csharp
public enum RasterImagesSavingModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| AsPngImagesEmbeddedIntoSvg | `0` | для каждого отдельного растрового файла будет сгенерировано изображение-оболочка SVG, и растровое изображение будет встроено в виде строк в кодировке Base64 в это изображение SVG |
| AsExternalPngFilesReferencedViaSvg | `1` | отдельные растровые изображения будут разделены как файлы PNG, но будут ссылаться на них через перенос изображений SVG, т.е. будут сгенерированы один файл PNG и один SVG для каждого растрового изображения, и каждый из таких SVG будет содержать ссылки на соответствующий файл PNG |
| AsEmbeddedPartsOfPngPageBackground | `2` | Будет создан один большой фоновый файл PNG для каждой страницы результатов. Растровые изображения будут встроены в этот файл и обработаны как области этого изображения. Никакие внешние файлы PNG для каждого изображения не будут генерироваться, только один файл PNG на странице будет присутствовать в наборе файлов результатов преобразования. |
