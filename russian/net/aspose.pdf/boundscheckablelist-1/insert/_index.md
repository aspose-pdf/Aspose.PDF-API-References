## BoundsCheckableList&lt;T&gt;.Insert метод

Вставляет элемент в System.Collections.Generic.List по указанному индексу.

```csharp
public void Insert(int index, T item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс с нуля, по которому элемент должен быть вставлен. |
| item | T | Объект для вставки. Значение может быть null для ссылочных типов. |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | *index* меньше 0. -или- *index* больше Count. |
