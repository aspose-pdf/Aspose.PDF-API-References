## BoundsCheckableList&lt;T&gt;.Contains метод

```csharp
public bool Contains(T item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | T | Объект, который нужно найти в System.Collections.Generic.List. Значение может быть null для ссылочных типов. |
