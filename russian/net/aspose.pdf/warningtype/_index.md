## Перечисление WarningType

Enum, представляющий тип предупреждения.

```csharp
public enum WarningType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SourceFileCorruption | `0` | Файл поврежден. |
| DataLoss | `1` | Текст/график/изображение или другие данные полностью отсутствуют в дереве документа после загрузки или в созданном документе после сохранения. |
| MajorFormattingLoss | `2` | Значительные потери форматирования по сравнению с оригинальным документом. Это для случаев, когда потеря форматирования значительна, но данные все еще присутствуют. |
| MinorFormattingLoss | `3` | Незначительные потери форматирования по сравнению с оригинальным документом. Это для незначительных потерь точности. |
| CompatibilityIssue | `4` | Известная проблема, которая не позволит открыть документ определенными пользовательскими агентами или предыдущими версиями пользовательских агентов. |
| InvalidInputStreamType | `5` | Неверный тип входного потока |
| UnexpectedContent | `99` | Файл содержит неожиданный контент. |
