## Перечисление HtmlSaveOptions.AntialiasingProcessingType

Этот перечисляемый тип описывает возможные меры сглаживания при конвертации

```csharp
public enum AntialiasingProcessingType
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| NoAdditionalProcessing | `0` | не используется специальная обработка сглаживания. Это оптимальный вариант для подавляющего большинства документов, и он не требует дополнительного времени при конвертации |
| TryCorrectResultHtml | `1` | В этом случае конвертер пытается обнаружить места с соседними графическими элементами фона и корректировать результат HTML соответствующим образом. Этот вариант позволяет улучшить результат экспорта для документов, содержащих фоны, состоящие из нескольких соседних графических элементов (для таких документов рендереры PDF, например, Acrobat Reader, обычно пытаются сгладить границы элементов при рендеринге. С этим вариантом конвертер имитирует такое поведение рендереров PDF. Этот вариант позволяет улучшить макет результата экспорта для некоторых специфических документов (которые используют такие составные фоны), но требует дополнительного времени для обработки (обычно около 10-15% дополнительного времени). Поэтому использование этого режима в общем случае не рекомендуется. |
