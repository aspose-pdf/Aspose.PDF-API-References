## Метод Metered.SetMeteredKey

Устанавливает общедоступный и частный ключ для учета. Если вы покупаете лицензию на учет, при запуске приложения этот API должен быть вызван, обычно этого достаточно. Однако, если постоянно не удается загрузить данные о потреблении и превышено 24 часа, лицензия будет установлена в статус оценки, чтобы избежать такой ситуации, вы должны регулярно проверять статус лицензии, если он в статусе оценки, вызовите этот API снова.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| publicKey | String | общедоступный ключ |
| privateKey | String | частный ключ |
