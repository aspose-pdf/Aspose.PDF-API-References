## Свойство Stamp.Background

Устанавливает или получает логическое значение, которое указывает, что содержимое штампа используется в качестве фона. Если значение истинно, содержимое штампа располагается внизу. По умолчанию значение ложно, содержимое штампа располагается сверху.

```csharp
public bool Background { get; set; }
```
