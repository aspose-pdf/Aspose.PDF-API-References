## Свойство Stamp.OutlineOpacity

Получает или устанавливает значение, указывающее непрозрачность контура штампа. Значение находится в диапазоне от 0.0 до 1.0. По умолчанию значение равно 1.0.

```csharp
public double OutlineOpacity { get; set; }
```
