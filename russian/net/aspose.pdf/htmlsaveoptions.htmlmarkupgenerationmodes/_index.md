## HtmlSaveOptions.HtmlMarkupGenerationModes enumeration

```csharp
public enum HtmlMarkupGenerationModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| WriteAllHtml | `0` | Режим по умолчанию, какие-либо особые требования отсутствуют. Будет сгенерирован вывод, который будет содержать все части HTML без какой-либо специальной дополнительной обработки. |
