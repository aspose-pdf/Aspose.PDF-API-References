## Метод OptimizedMemoryStream.Read

При переопределении в производном классе считывает последовательность байтов из текущего потока и перемещает позицию внутри потока на количество считанных байтов.

```csharp
public override int Read(byte[] buffer, int offset, int count)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| buffer | Byte[] | Массив байтов. Когда этот метод возвращает значение, буфер содержит указанный массив байтов с значениями |
| offset | Int32 | Смещение в байтах с нулевой базой, с которого следует начать хранить данные, считанные из текущего потока. |
| count | Int32 | Максимальное количество байтов, которые должны быть считаны из текущего потока. |

### Возвращаемое значение

Общее количество байтов, считанных в буфер. Это может быть меньше запрашиваемого количества байтов, если такое количество байтов в данный момент недоступно, или ноль (0), если конец потока был достигнут.
