## Перечисление PasswordType

Этот перечисляемый тип представляет известные типы паролей, используемые для защищенных паролем pdf-документов.

```csharp
public enum PasswordType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Pdf документ не защищен паролем. |
| User | `1` | Pdf документ был открыт с использованием пароля для открытия документа (ограниченный доступ). |
| Owner | `2` | Pdf документ был открыт с использованием пароля для изменения разрешений (полный доступ). |
| Inaccessible | `3` | Pdf документ защищен паролем, но пароли пользователя и владельца не пусты, и ни один из паролей не был определен или предоставленный пароль был неверным. Поэтому невозможно определить тип пароля. |
