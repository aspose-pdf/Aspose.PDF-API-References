## Перечисление прав

Это перечисление представляет права пользователя для pdf.

```csharp
[Flags]
public enum Permissions
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| PrintDocument | `4` | (Обработчики безопасности версии 2) Печать документа. (Обработчики безопасности версии 3 или выше) Печать документа (возможно, не на самом высоком уровне качества, в зависимости от того, установлено ли также PrintingQuality). |
| ModifyContent | `8` | Изменение содержимого документа с помощью операций, отличных от тех, которые контролируются ModifyTextAnnotations, FillForm и 11. |
| ExtractContent | `10` | (Обработчики безопасности версии 2) Копирование или иное извлечение текста и графики из документа, включая извлечение текста и графики (в поддержку доступности для пользователей с ограниченными возможностями или для других целей). (Обработчики безопасности версии 3 или выше) Копирование или иное извлечение текста и графики из документа с помощью операций, отличных от тех, которые контролируются ExtractContentWithDisabilities. |
| ModifyTextAnnotations | `20` | Добавление или изменение текстовых аннотаций, заполнение интерактивных полей формы и, если также установлено ModifyContent, создание или изменение интерактивных полей формы (включая поля для подписи). |
| FillForm | `100` | (Обработчики безопасности версии 3 или выше) Заполнение существующих интерактивных полей формы (включая поля для подписи), даже если ModifyTextAnnotations не установлено. |
| ExtractContentWithDisabilities | `200` | (Обработчики безопасности версии 3 или выше) Извлечение текста и графики (в поддержку доступности для пользователей с ограниченными возможностями или для других целей). |
| AssembleDocument | `400` | (Обработчики безопасности версии 3 или выше) Сборка документа (вставка, поворот или удаление страниц и создание закладок или миниатюр), даже если ModifyContent не установлено. |
| PrintingQuality | `800` | (Обработчики безопасности версии 3 или выше) Печать документа в представление, из которого может быть создана точная цифровая копия содержимого PDF. Когда этот бит не установлен (и бит 3 установлен), печать ограничивается низкоуровневым представлением внешнего вида, возможно, ухудшенного качества. |
