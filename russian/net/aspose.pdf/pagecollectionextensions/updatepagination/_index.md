## Метод PageCollectionExtensions.UpdatePagination

Обновляет номера страниц и даты в заголовке и подвале для всех страниц. Это будет работать, если документ имеет хотя бы один артефакт пагинации с данными специальных настроек. Все страницы в коллекции будут обновлены с использованием исходного артефакта в соответствии с его настройками.

```csharp
public static void UpdatePagination(this PageCollection pageCollection)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pageCollection | PageCollection | Коллекция страниц. |
