## Поле HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber

Устанавливается конвертером. Если установлено свойство SplitToPages, то во время конвертации создается несколько HTML-файлов (по одному HTML-файлу на каждую конвертированную страницу). Это свойство сообщает пользовательскому коду, с какой страницы оригинального PDF был создан сохраненный HTML-разметка. Если номер оригинальной страницы по какой-то причине неизвестен или SplitOnPages=false, то это свойство всегда содержит '0', что сигнализирует о том, что конвертер не может предоставить точный номер страницы оригинального PDF для предоставленного файла HTML-разметки.

```csharp
public int PdfHostPageNumber;
```
