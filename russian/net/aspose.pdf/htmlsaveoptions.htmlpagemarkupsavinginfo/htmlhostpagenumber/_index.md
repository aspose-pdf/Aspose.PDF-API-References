## Поле HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber

Устанавливается конвертером. Если установлено свойство SplitToPages, то во время конвертации создается несколько HTML-файлов (по одному HTML-файлу на каждую конвертированную страницу). Это свойство содержит порядковый номер сохраненного HTML-файла. Свойство может быть использовано в логике пользовательского кода для определения того, как обрабатывать или куда сохранять HTML-страницу, и если разделение на страницы отключено, это значение всегда будет равно '1', так как в таком случае генерируется только одна большая HTML-страница для всего исходного документа.

```csharp
public int HtmlHostPageNumber;
```
