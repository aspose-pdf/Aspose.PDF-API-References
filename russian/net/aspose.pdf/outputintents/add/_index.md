## Метод OutputIntents.Add

Добавляет выходное намерение в коллекцию.

```csharp
public void Add(OutputIntent item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | OutputIntent | Выходное намерение, которое нужно добавить в коллекцию. |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | *item* равно null. |
| InvalidOperationException | Документ, содержащий коллекцию, не имеет каталога для доступа к OutputIntents. |
