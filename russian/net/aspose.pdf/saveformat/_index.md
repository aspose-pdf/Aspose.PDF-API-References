### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Pdf | `0` | означает сохранение без изменения формата, т.е. как PDF, используйте это вместо 'SaveFormat.None', который устарел |
| None | `0` | означает сохранение без изменения формата, т.е. как PDF. Это устаревший вариант и будет удален в конечном итоге, пожалуйста, используйте вместо этого 'SaveFormat.Pdf' |
| Doc | `1` | означает сохранение в формате DOC |
| Xps | `2` | означает сохранение в формате XPS |
| Html | `3` | означает сохранение в формате HTML |
| Xml | `4` | означает сохранение в формате XML |
| TeX | `5` | означает сохранение в формате TEX, т.е. формате, подходящем для текстового редактора Latex |
| DocX | `6` | означает сохранение в формате DOCX |
| Svg | `7` | означает сохранение в формате SVG |
| MobiXml | `8` | означает сохранение в формате MobiXML (специальный формат электронных книг) |
| Excel | `9` | означает сохранение в формате MsExcel |
| Epub | `10` | означает сохранение в формате EPUB (специальный формат электронных книг) |
| Pptx | `14` | означает сохранение в формате PPTX |
| Aps | `15` | Сохранение в виде файла APS XML. |
| PdfXml | `16` | Внутренняя структура PDF-документа в формате XML |
| Ps | `17` | означает сохранение в формате PostScript. |
| Eps | `18` | означает сохранение в формате Encapsulated PostScript. |
| Markdown | `19` | означает сохранение в формате Markdown. |
