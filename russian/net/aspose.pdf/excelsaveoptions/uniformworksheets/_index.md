## Свойство ExcelSaveOptions.UniformWorksheets

Установите true для использования равномерного деления столбцов по всему документу. Значение по умолчанию - false; это означает, что деление столбцов будет независимым для каждой страницы.

```csharp
public bool UniformWorksheets { get; set; }
```
