## ExcelSaveOptions.UniformWorksheets property

Установите значение true для использования равномерного разделения столбцов в документе. Значение по умолчанию - false; это означает, что разделение столбцов будет независимым для каждой страницы.

public bool UniformWorksheets { get; set; }
