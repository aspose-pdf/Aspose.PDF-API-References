## Свойство ExcelSaveOptions.InsertBlankColumnAtFirst

Установите значение true, если вам нужно вставить пустой столбец в качестве первого столбца листа. Значение по умолчанию - false; это означает, что пустой столбец не будет вставлен.

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
