## Метод Image.GetMimeType

Возвращает mime-тип для изображения.

```csharp
public static string GetMimeType(Image i)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| i | Image | Объект изображения. |

### Возвращаемое значение
