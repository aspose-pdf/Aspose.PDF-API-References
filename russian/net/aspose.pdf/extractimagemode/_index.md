## Перечисление ExtractImageMode

Определяет различные режимы, которые могут быть использованы при извлечении изображений из документов.

```csharp
public enum ExtractImageMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| DefinedInResources | `0` | Определяет режим извлечения изображений, при котором извлекаются все изображения, определенные в ресурсах для конкретной страницы. |
| ActuallyUsed | `1` | Определяет режим извлечения изображений, при котором извлекаются только те изображения, которые фактически отображаются на странице. |
