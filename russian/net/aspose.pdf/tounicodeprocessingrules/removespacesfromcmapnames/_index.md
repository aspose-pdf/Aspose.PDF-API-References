## Свойство ToUnicodeProcessingRules.RemoveSpacesFromCMapNames

Некоторые шрифты имеют карты кодов символов ToUnicode с пробелами в именах. Эти пробелы могут вызывать ошибки при сопоставлении текста в юникоде. Этот флаг указывает на необходимость удалить пробелы из имен карт кодов символов ToUnicode. По умолчанию false.

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
