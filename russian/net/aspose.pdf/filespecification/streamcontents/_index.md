## Свойство FileSpecification.StreamContents

Получает содержимое файла в виде потока. Содержимое не загружается в память, что позволяет уменьшить использование памяти. Но этот поток не поддерживает позиционирование и свойство Length. Если вам нужны эти функции, пожалуйста, используйте свойство Contents вместо этого.

```csharp
public Stream StreamContents { get; }
```
