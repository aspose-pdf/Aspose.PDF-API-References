## Перечисление HeadingRecognitionStrategy

Представляет типы стратегий распознавания заголовков.

```csharp
public enum HeadingRecognitionStrategy
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Outlines | `0` | Представляет стратегию распознавания заголовка с помощью контуров. |
| Heuristic | `1` | Представляет стратегию распознавания заголовка с помощью эвристических правил и статистики размера шрифта. |
| Auto | `2` | Обеспечивает автоматический выбор стратегии распознавания заголовка. Это опция по умолчанию. Если документ содержит закладки, будет выбрана стратегия Outlines, в противном случае - Heuristic. |
| None | `3` | Не распознавать заголовки. Эта опция может быть полезна в сложных форматированных документах. |
