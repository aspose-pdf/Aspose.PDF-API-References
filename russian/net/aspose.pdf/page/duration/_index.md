## Page.Duration property

Получает установленную продолжительность отображения страницы. Это время в секундах, в течение которого страница должна отображаться во время презентации. Возвращает -1, если продолжительность не определена.

```csharp
public double Duration { get; set; }
```

### Примеры

Пример демонстрирует, как получить продолжительность страницы

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
int pageRect = page.Duration;
```
