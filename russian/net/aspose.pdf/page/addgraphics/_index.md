## Метод Page.AddGraphics

Добавляет графику на страницу. Работает быстрее, чем добавление элементов по одному с помощью метода [`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/).

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| elements | GraphicElementCollection | Коллекция графики. |
| rectangle | Rectangle | Элементы будут добавлены на страницу, если они находятся внутри области прямоугольника. Если прямоугольник равен null, все графические элементы будут добавлены |
