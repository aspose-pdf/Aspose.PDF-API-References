## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled field

Должно быть установлено пользовательским кодом, если он не может или не должен определять URL, который будет использоваться в сгенерированном HTML для ссылки на этот CSS. Если это 'true', то файл CSS будет сохранен стандартным образом в стандартном месте.

```csharp
public bool CustomProcessingCancelled;
```
