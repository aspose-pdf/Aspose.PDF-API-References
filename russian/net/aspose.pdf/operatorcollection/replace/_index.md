## Метод OperatorCollection.Replace

Заменить операторы в коллекции на другие операторы.

```csharp
public void Replace(IList<Operator> operators)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| operators | IList`1 | Список операторов, который заменит операторы, в настоящее время содержащиеся в коллекции. Каждый оператор из списка должен иметь правильный индекс в диапазоне [1..N], где N — количество операторов в коллекции |
