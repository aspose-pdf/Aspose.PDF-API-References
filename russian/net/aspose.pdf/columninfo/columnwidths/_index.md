## Свойство ColumnInfo.ColumnWidths

Получает или задает строку, содержащую ширину столбцов. Значение каждого столбца должно быть разделено пробелом. Значение по умолчанию - пункт, но также поддерживаются см, дюймы и процент от доступной ширины. Например, "120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
