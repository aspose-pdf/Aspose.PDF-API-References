## Свойство ColumnInfo.ColumnSpacing

Получает или устанавливает строку, содержащую расстояние между колонками. Значение каждого расстояния должно быть разделено пробелом. Значение по умолчанию - пункт, но также поддерживаются см и дюймы. Например, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Примечания

Если это свойство не установлено, будет использовано значение по умолчанию 0 для каждого расстояния.
