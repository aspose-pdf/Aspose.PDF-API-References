## ColumnInfo.ColumnSpacing property

Получает или задает строку, содержащую расстояние между столбцами. Значение каждого интервала должно быть отделено пробелом. Единицей по умолчанию является точка, , но также поддерживаются см и дюймы. Например, «120 2,5 см 1,5 дюйма».

```csharp
public string ColumnSpacing { get; set; }
```

### Примечания

Если это свойство не установлено, для каждого интервала будет использоваться значение по умолчанию 0.
