## HtmlSaveOptions.PartsEmbeddingModes enumeration

Это перечисление перечисляет возможные способы встраивания файлов, на которые есть ссылки в HTML Позволяет контролировать, будут ли файлы, на которые ссылаются (HTML, шрифты, изображения, CSS) будут встроены в основной файл HTML или будут сгенерированы как отдельные бинарные объекты

```csharp
public enum PartsEmbeddingModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| EmbedAllIntoHtml | `0` | Принудительно встраивает все указанные файлы (Css, изображения, шрифты) в сгенерированную разметку HTML (т.е. в сам HTML) Этот подход создает один файл HTML , но общий размер вывода становится больше (поскольку используется кодировка двоичных файлов Base64) и не все браузеры (особенно устаревшие) успешно обрабатывают двоичные файлы, встроенные в HTML. Но это позволяет получить HTML, содержащий весь результат, без каких-либо дополнительных файлов. |
| EmbedCssOnly | `1` | Принудительно разделяет все файлы, на которые ссылаются, кроме CSS (изображения и шрифты) Т.е. CSS будет встроен в результирующий HTML и все другие файлы, на которые ссылаются( Изображения и шрифты) будут обрабатываться как внешние части Он генерирует HTML, который подходит для широкого набора браузеров |
| NoEmbedding | `2` | Принуждает разделять ссылочные файлы (Css, изображения, шрифты) Этот подход генерирует набор файлов, но общий размер вывода становится меньше (поскольку не используется кодировка двоичных файлов Base64) Также такой подход генерирует HTML, который подходит для широкого набора браузеров |
