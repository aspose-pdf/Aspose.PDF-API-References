## Перечисление HtmlSaveOptions.PartsEmbeddingModes

Этот перечисляемый тип перечисляет возможные режимы встраивания файлов, на которые ссылается HTML. Он позволяет контролировать, будут ли ссылки на файлы (HTML, шрифты, изображения, CSS) встроены в основной HTML-файл или будут сгенерированы как отдельные бинарные сущности.

```csharp
public enum PartsEmbeddingModes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| EmbedAllIntoHtml | `0` | Обязывает встроить все ссылки на файлы (CSS, изображения, шрифты) в сгенерированный HTML-разметку (т.е. в сам HTML). Этот подход генерирует один HTML-файл, но общий размер выходных данных становится больше (поскольку используется кодирование Base64 для бинарных данных), и не все браузеры (особенно устаревшие) успешно обрабатывают бинарные данные, встроенные в HTML. Но это позволяет получить HTML, который содержит весь результат, без каких-либо дополнительных файлов. |
| EmbedCssOnly | `1` | Обязывает оставить отдельно все ссылки на файлы, кроме CSS (изображения и шрифты). Т.е. CSS будет встроен в результирующий HTML, а все другие ссылки на файлы (изображения и шрифты) будут обработаны как внешние части. Это генерирует HTML, который подходит для широкого набора браузеров. |
| NoEmbedding | `2` | Обязывает оставить отдельно ссылки на файлы (CSS, изображения, шрифты). Этот подход генерирует набор файлов, но общий размер выходных данных становится меньше (поскольку не используется кодирование Base64 для бинарных данных). Также такой подход генерирует HTML, который подходит для широкого набора браузеров. |
