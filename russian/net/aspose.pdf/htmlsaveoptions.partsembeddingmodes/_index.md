## HtmlSaveOptions.PartsEmbeddingModes enumeration

Это перечисление перечисляет возможные режимы встраивания файлов, на которые есть ссылки в HTML Оно позволяет контролировать, будут ли файлы, на которые ссылаются (HTML, шрифты, изображения, CSS) , внедряться в основной файл HTML или будут генерироваться как отдельные двоичные объекты

```csharp
public enum PartsEmbeddingModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| EmbedAllIntoHtml | `0` | Принудительно встраивает все указанные файлы (CSS, изображения, шрифты) в сгенерированную HTML-разметку (т.е. в сам HTML) не все браузеры (особенно устаревшие) успешно обрабатывают двоичные файлы, встроенные в HTML. Но это позволяет получить HTML, содержащий весь результат, без дополнительных файлов. |
| EmbedCssOnly | `1` | Принудительно разделяет все файлы, на которые ссылаются, кроме CSS (изображения и шрифты) Т.е. CSS будет встроен в результирующий HTML, а все другие файлы, на которые есть ссылки (изображения и шрифты) будут обрабатываться как внешние части Он генерирует HTML, который подходит для широкого набора браузеров |
| NoEmbedding | `2` | Принудительно разделяет ссылочные файлы (Css, изображения, шрифты) Этот подход генерирует набор файлов, но общий размер вывода становится меньше (поскольку не используется кодировка двоичных файлов Base64) Также такой подход генерирует HTML, который подходит для широкий набор браузеров |
