## Свойство XImage.ImageMask

Получает флаг, указывающий, следует ли рассматривать изображение как маску изображения (см. 8.9.6, "Маскированные изображения"). Если этот флаг истинный, значение BitsPerComponent должно быть 1, а Mask и ColorSpace не должны быть указаны; немаскированные области должны быть закрашены с использованием текущего неконтурного цвета. Значение по умолчанию: false.

```csharp
public bool ImageMask { get; }
```

### Значение свойства

True, если изображение является маской изображения.
