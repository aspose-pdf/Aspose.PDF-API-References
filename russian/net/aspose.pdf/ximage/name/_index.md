## Свойство XImage.Name

Получает или устанавливает имя изображения. Пожалуйста, обратите внимание, что если вы измените имя изображения, на которое есть ссылки в содержимом страницы, документ может стать некорректным. В этом случае используйте метод XImage.Rename.

```csharp
public string Name { get; set; }
```
