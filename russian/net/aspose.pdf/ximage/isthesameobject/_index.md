## Метод XImage.IsTheSameObject

Возвращает true, если обе ссылки на изображения указывают на один и тот же объект.

```csharp
public bool IsTheSameObject(XImage image)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| image | XImage | Изображение, которое будет сравниваться с "этим" изображением. |

### Возвращаемое значение

Булевое значение, которое равно true, если ссылки на изображения указывают на один и тот же объект.
