## Перечисление XslFoLoadOptions.ParsingErrorsHandlingTypes

Исходный документ XSLFO может содержать ошибки форматирования. Этот перечисляемый тип перечисляет возможные стратегии обработки таких ошибок форматирования

```csharp
public enum ParsingErrorsHandlingTypes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| TryIgnore | `0` | В этом случае конвертер будет проинструктирован попытаться продолжить конвертацию и игнорировать найденные ошибки форматирования. В этом случае успех не гарантирован, серьезные проблемы могут возникнуть позже в конвертере, и в таком случае будет выброшено исключение с списком найденных ошибок форматирования. |
| ThrowExceptionImmediately | `1` | В этом случае конвертация будет немедленно остановлена, и исключение будет выброшено сразу после обнаружения первой ошибки форматирования |
| InvokeCustomHandler | `2` | Это самый гибкий метод - пользовательский код должен предоставить (в свойстве WarningCallback) специальный обработчик, который будет вызван при обнаружении ошибки форматирования. Этот обработчик может, например, регистрировать или подсчитывать ошибки и будет принимать решение о том, можно ли продолжить обработку для этой или той ошибки. |
