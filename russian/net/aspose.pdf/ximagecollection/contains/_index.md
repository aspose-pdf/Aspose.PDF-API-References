## Метод XImageCollection.Contains

Определяет, содержит ли коллекция конкретное значение.

```csharp
public bool Contains(XImage item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | XImage | Объект, который нужно найти в коллекции. |

### Возвращаемое значение

true, если элемент найден в коллекции; в противном случае false.
