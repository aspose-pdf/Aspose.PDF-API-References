### Примечания

В основном используется объект[`ImagePlacementAbsorber`](../imageplacementabsorber)в сценарии поиска изображений. Когда поиск завершен, вхождения представлены[`ImagePlacement`](../imageplacement)объектами, которые[`ImagePlacements`](./imageplacements)Коллекция содержит. Объект[`ImagePlacement`](../imageplacement)предоставляет доступ к свойствам размещения изображения:размерам, разрешению и т. д.

### Примеры

Пример демонстрирует, как найти изображения на первой странице документа PDF и получить свойства размещения изображений.

```csharp
// Открыть документ
ocument doc = new Document(@"D:\Tests\input.pdf");

// Создаем объект ImagePlacementAbsorber для размещения изображения search

// Принять поглотитель для первой страницы
oc.Pages[1].Accept(abs);

// Показать свойства размещения изображения для всех размещений
oreach (ImagePlacement imagePlacement in abs.ImagePlacements)
     
   Console.Out.WriteLine("image width:" + imagePlacement.Rectangle.Width);
   Console.Out.WriteLine("image height:" + imagePlacement.Rectangle.Height);
   Console.Out.WriteLine("image LLX:" + imagePlacement.Rectangle.LLX);
   Console.Out.WriteLine("image LLY:" + imagePlacement.Rectangle.LLY);
   Console.Out.WriteLine("image horizontal resolution:" + imagePlacement.Resolution.X);
   Console.Out.WriteLine("image vertical resolution:" + imagePlacement.Resolution.Y);
```
