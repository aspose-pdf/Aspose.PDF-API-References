## Document.MergeOptions.IsNeedPageTreeBalance property

Получает и устанавливает требование для балансировки дерева страниц. Все дерево страниц в результирующем документе будет сбалансировано. Оно создает сбалансированное дерево страниц для ускорения доступа к страницам.

```csharp
public bool IsNeedPageTreeBalance { get; set; }
```
