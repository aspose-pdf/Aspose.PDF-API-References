## Метод Rectangle.Contains

Определяет, находится ли заданная точка внутри прямоугольника.

```csharp
public bool Contains(Point point, bool inclusive = false)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| point | Point | Точка для проверки. |
| inclusive | Boolean | inclusive: точка может находиться на границе |

### Возвращаемое значение

True, если точка находится внутри прямоугольника.
