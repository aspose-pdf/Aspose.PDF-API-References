## Метод Rectangle.IsIntersect

Определяет, пересекается ли этот прямоугольник с другим прямоугольником.

```csharp
public bool IsIntersect(Rectangle otherRect)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| otherRect | Rectangle | Пересечение будет проверяться с указанным прямоугольником. |

### Возвращаемое значение

True, если этот прямоугольник пересекается с указанным прямоугольником. В противном случае false.
