## Свойство DocSaveOptions.ReSaveFonts

Получает или задает процедуру повторного сохранения шрифтов. Если установлено значение true, мы перезагружаем шрифты на каждой странице, чтобы избежать влияния предыдущих свойств шрифтов, и загружаем вновь созданный шрифт с нуля. Установите этот параметр в false, если хотите улучшить производительность. Значение по умолчанию - true;

```csharp
public bool ReSaveFonts { get; set; }
```
