## Свойство DocSaveOptions.ConvertType3Fonts

Получает или устанавливает преобразование для шрифтов Type3. В шрифтах Type 3 глифы должны быть определены потоками графических операторов. Это означает, что в выходных файлах DOC/DOCX мы видим изображения вместо текста. Установите этот флаг в true, чтобы преобразовать шрифты Type3 в TTF и получить текст в результирующем файле.

```csharp
public bool ConvertType3Fonts { get; set; }
```
