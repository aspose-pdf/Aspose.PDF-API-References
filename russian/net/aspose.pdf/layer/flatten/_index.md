## Метод Layer.Flatten

```csharp
public void Flatten(bool cleanupContentStream)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| cleanupContentStream | Boolean | Указывает, следует ли удалять маркеры группы необязательного контента из потока контента. |

## Замечания

Установка параметра *cleanupContentStream* в значение false ускоряет процесс уплощения.
