## FloatingBox.IsNeedRepeating property

Получает или задает логическое значение, указывающее, нужно ли повторять абзац на следующей странице. Значение по умолчанию — false. Атрибут действителен, только если сам абзац и объект, на который ссылается его ReferenceParagraphID, включены в RepeatingRows.

public bool IsNeedRepeating { get; set; }
