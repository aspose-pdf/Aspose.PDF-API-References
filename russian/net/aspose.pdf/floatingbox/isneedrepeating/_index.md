## Свойство FloatingBox.IsNeedRepeating

Получает или устанавливает логическое значение, которое указывает, нужно ли повторять абзац на следующей странице. Значение по умолчанию - false. Атрибут действителен только в том случае, если сам абзац и объект, на который ссылается его ReferenceParagraphID, оба включены в RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
