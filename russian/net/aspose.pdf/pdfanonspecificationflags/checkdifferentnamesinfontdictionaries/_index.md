## Свойство PdfANonSpecificationFlags.CheckDifferentNamesInFontDictionaries

Некоторые PDF-документы содержат шрифты, которые имеют разные названия во внутренних данных. Использование этого флага обеспечивает специальную логику обработки для случаев, когда поля BaseFont и FontDescriptor.FontName различны.

```csharp
public bool CheckDifferentNamesInFontDictionaries { get; set; }
```
