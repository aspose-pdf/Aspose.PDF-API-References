## HtmlSaveOptions.FontSavingModes enumeration

Перечисляет режимы, которые можно использовать для сохранения шрифтов ссылки в сохраненном PDF

```csharp
public enum FontSavingModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| AlwaysSaveAsWOFF | `0` | Все указанные шрифты будут сохранены и указаны как WOFF-шрифты |
| AlwaysSaveAsTTF | `1` | Все указанные шрифты будут сохранены и указаны как TTF-шрифты |
| AlwaysSaveAsEOT | `2` | Все указанные шрифты будут сохранены и указаны как EOT-шрифты |
| SaveInAllFormats | `3` | Все указанные шрифты будут сохранены (и указаны в CSS) как 3 независимых файла:EOT, TTH, WOFF. Увеличивает размер выходных данных, но делает вывод пригодным для подавляющего большинства веб-браузеров |
| DontSave | `4` | Все указанные шрифты не будут сохранены. |
