## HtmlSaveOptions.FontSavingModes перечисление

Перечисляет режимы, которые могут быть использованы для сохранения шрифтов, упомянутых в сохраненном PDF.

```csharp
public enum FontSavingModes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| AlwaysSaveAsWOFF | `0` | Все упомянутые шрифты будут сохранены и упомянуты как WOFF-шрифты. |
| AlwaysSaveAsTTF | `1` | Все упомянутые шрифты будут сохранены и упомянуты как TTF-шрифты. |
| AlwaysSaveAsEOT | `2` | Все упомянутые шрифты будут сохранены и упомянуты как EOT-шрифты. |
| SaveInAllFormats | `3` | Все упомянутые шрифты будут сохранены (и упомянуты в CSS) как 3 независимых файла: EOT, TTH, WOFF. Это увеличивает размер выходных данных, но делает выходные данные подходящими для подавляющего большинства веб-браузеров. |
| DontSave | `4` | Все упомянутые шрифты не будут сохранены. |
