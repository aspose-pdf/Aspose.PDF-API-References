## Перечисление SaveOptions.HtmlBorderLineType

Представляет типы линий, которые могут быть использованы в результирующем документе для рисования границ или других линий

```csharp
public enum HtmlBorderLineType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Линия не будет показана. |
| Dotted | `1` | Будет показана пунктирная линия. |
| Dashed | `2` | Будет показана штриховая линия. |
| Solid | `3` | Будет показана сплошная линия. |
| Double | `4` | Будет показана двойная линия. |
| Groove | `5` | Будет показана линия в виде канавки. |
| Ridge | `6` | Будет показана линия в виде выступа. |
| Inset | `7` | Будет показана линия в виде врезки. |
| Outset | `8` | Будет показана линия в виде выступа. |
