## RenderingOptions.ConvertFontsToUnicodeTTF property

Указывает, что все шрифты будут преобразованы в версии TTF Unicode. Это полезно для совместимости и для оптимизации использования шрифта, потому что каждый новый шрифт TTF будет иметь не все символы из исходного шрифта, а только символы, которые используются в тексте.

public bool ConvertFontsToUnicodeTTF { get; set; }
