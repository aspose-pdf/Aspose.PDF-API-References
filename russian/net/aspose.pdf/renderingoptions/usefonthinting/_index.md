## Свойство RenderingOptions.UseFontHinting

Использование этого флага включает механизм подсказки шрифтов. Подсказка шрифтов — это использование математических инструкций для настройки отображения контурного шрифта. В некоторых случаях включение этого флага может решить проблемы с читаемостью текста. В настоящее время использование этого флага может оказать эффект только для шрифтов TTF, если эти шрифты используются в исходном документе.

```csharp
public bool UseFontHinting { get; set; }
```
