## Перечисление HorizontalAlignment

Описывает горизонтальное выравнивание.

```csharp
public enum HorizontalAlignment
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Нет выравнивания. |
| Left | `1` | Выравнивание по левому краю. |
| Center | `2` | Центрированное выравнивание. |
| Right | `3` | Выравнивание по правому краю. |
| Justify | `4` | Выравнивание по ширине. Текст будет выровнен по левому и правому краям. |
| FullJustify | `5` | Похоже на выравнивание 'Justify', за исключением того, что последняя строка будет выровнена только по левому краю в режиме 'Justify', в то время как в режиме 'FullJustify' все строки будут выровнены по левому и правому краям. |
