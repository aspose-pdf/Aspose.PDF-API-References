## Table.ImportArray method

Импортирует одномерный массив данных в таблицу. Импорт идет по одной ячейке на каждый элемент массива и начинается со строки и столбца, определенных в параметрах. Если во время импорта обнаруживается, что необходимые строки все еще отсутствуют (т.е. целевая таблица слишком мала, чтобы поглотить все данные), будут созданы необходимые строки

```csharp
public void ImportArray(object[] importedArray, int firstFilledRow, int firstFilledColumn, 
    bool isLeftColumnsFilled)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| importedArray | Object[] | импортированные данные, нули будут импортированы как пустые строки |
| firstFilledRow | Int32 | определить номер первой целевой строки в целевой таблице, из которой начнется импорт. Если количество строк в целевой таблице меньше требуемого, недостающие строки будут созданы первыми. |
| firstFilledColumn | Int32 | указывает номер первого целевого столбца в целевой таблице, столбец должен присутствовать в целевой таблице перед началом импорта |
| isLeftColumnsFilled | Boolean | Если 'isLeftColumnsFilled'=false, то во второй и всех последующих заполненных строках будут пропущены ячейки слева от firstFilledColumn |
