## Метод Table.ImportArray

Импортирует одномерный массив данных в таблицу. Импорт происходит по одной ячейке для каждого элемента массива и начинается с строки и столбца, определенных в параметрах. Во время импорта, если будет обнаружено, что необходимые строки все еще отсутствуют (т.е. целевая таблица слишком мала, чтобы вместить все данные), необходимые строки будут созданы

```csharp
public void ImportArray(object[] importedArray, int firstFilledRow, int firstFilledColumn, 
    bool isLeftColumnsFilled)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| importedArray | Object[] | импортированные данные, нулевые значения будут импортированы как пустые строки |
| firstFilledRow | Int32 | определяет номер первой целевой строки в целевой таблице, с которой начнется импорт. Если количество строк в целевой таблице меньше необходимого, сначала будут созданы недостающие строки. |
| firstFilledColumn | Int32 | указывает номер первого целевого столбца в целевой таблице, столбец должен присутствовать в целевой таблице до начала импорта |
| isLeftColumnsFilled | Boolean | Если 'isLeftColumnsFilled'=false, то во втором и всех последующих заполненных строках ячейки, которые находятся слева от firstFilledColumn, будут пропущены |
