## Перечисление ImageDeleteAction

Действие, которое выполняется с объектом изображения, когда изображение удаляется из коллекции. Если объект изображения удален

```csharp
public enum ImageDeleteAction
```

### Значения

| Название | Значение | Описание |
| --- | --- | --- |
| KeepContents | `0` | Изображение будет удалено из коллекции. Если содержимое страницы содержит ссылки на изображение, они не будут удалены. Документ может стать недействительным. |
| None | `1` | Изображение будет удалено из коллекции и из содержимого страницы, но объект изображения не будет удален. Размер файла не уменьшится. |
| ForceDelete | `2` | Изображение будет удалено из коллекции, и объект изображения будет удален из документа. Если существуют другие ссылки на тот же объект, документ может быть поврежден. |
| Check | `3` | Изображение будет удалено из коллекции, и объект изображения будет удален только в том случае, если нет других ссылок на изображение с других страниц. Это может занять больше времени по сравнению с опцией ForceDelete. |
