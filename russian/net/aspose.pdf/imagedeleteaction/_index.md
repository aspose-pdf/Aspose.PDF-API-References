## ImageDeleteAction enumeration

Действие, которое выполняется с объектом изображения при удалении изображения из коллекции. Если объект изображения удален

```csharp
public enum ImageDeleteAction
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| KeepContents | `0` | Изображение будет удалено из коллекции. Если содержимое страницы содержит ссылки на изображение, они не будут удалены. Документ может стать недействительным. |
| None | `1` | Изображение будет удалено из коллекции и из содержимого страницы, но объект изображения не будет удален. Размер файла не будет уменьшен. |
| ForceDelete | `2` | Изображение будет удалено из коллекции, а объект изображения будет удален из документа. Если существуют другие ссылки на тот же объект, документ может быть поврежден. |
| Check | `3` | Изображение будет удалено из коллекции и объект изображения будет удален только в том случае, если нет других ссылок на изображение с других страниц. Это может потребовать больше времени по сравнению с опцией ForceDelete. |
