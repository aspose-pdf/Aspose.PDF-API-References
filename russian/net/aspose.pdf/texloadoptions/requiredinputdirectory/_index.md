## Свойство TeXLoadOptions.RequiredInputDirectory

Получает/устанавливает требуемый каталог ввода TeX. Требуемый ввод - это файлы, которые каким-либо образом включены в основной файл .tex, например, пакеты, для которых нет встроенной поддержки.

public ITeXInputDirectory RequiredInputDirectory { get; set; }
