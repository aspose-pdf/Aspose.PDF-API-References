## TeXLoadOptions.Repeat property

Возвращает/устанавливает флаг, указывающий, нужно ли дважды запускать задание TeX в случае, например, во входном TeX есть ссылки файл(ы). В общем, такое поведение полезно, когда движок собирает некоторые данные в процессе верстки и сохраняет их во вспомогательном файле, все при первом запуске. И при втором запуске движок каким-то образом использует эти данные.

```csharp
public bool Repeat { get; set; }
```
