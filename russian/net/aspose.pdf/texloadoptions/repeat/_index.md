## Свойство TeXLoadOptions.Repeat

Получает/устанавливает флаг, указывающий, необходимо ли запускать задачу TeX дважды в случае, если, например, в входных TeX файлах есть ссылки. В общем, такое поведение полезно, когда движок собирает некоторые данные в процессе верстки и сохраняет их в вспомогательном файле при первом запуске. А при втором запуске движок каким-то образом использует эти данные.

```csharp
public bool Repeat { get; set; }
```
