## Метод IBoundsCheckableItem.CheckBounds

Проверяет, помещается ли элемент в заданные размеры контейнера (включительно).

```csharp
public bool CheckBounds(double containerWidth, double containerHeight)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| containerWidth | Double | Ширина контейнера. |
| containerHeight | Double | Высота контейнера. |

### Возвращаемое значение

True, если помещается; в противном случае false.
