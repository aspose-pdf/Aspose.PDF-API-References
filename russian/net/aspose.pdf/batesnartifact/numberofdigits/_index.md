## Свойство BatesNArtifact.NumberOfDigits

Получает или устанавливает количество цифр для нумерации Bates. Значение должно быть в диапазоне от 3 до 15 включительно. Если установлено значение меньше 3, оно будет скорректировано до 3. Если установлено значение больше 15, оно будет скорректировано до 15. Значение по умолчанию равно 6.

```csharp
public int NumberOfDigits { get; set; }
```
