## Свойство BaseParagraph.ZIndex

Получает или устанавливает целочисленное значение, которое указывает порядок Z графика. График с большим ZIndex будет размещен поверх графика с меньшим ZIndex. ZIndex может быть отрицательным. График с отрицательным ZIndex будет размещен позади текста на странице.

```csharp
public int ZIndex { get; set; }
```
