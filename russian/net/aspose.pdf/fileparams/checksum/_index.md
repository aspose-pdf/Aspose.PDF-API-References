## Свойство FileParams.CheckSum

16-байтовая строка, которая является контрольной суммой байтов не сжатого встроенного файла. Контрольная сумма вычисляется с помощью стандартного алгоритма MD5 для дайджеста сообщений на байтах потока встроенного файла.

```csharp
public string CheckSum { get; }
```
