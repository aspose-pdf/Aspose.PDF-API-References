## Метод PageLabelCollection.RemoveLabel

Удалить метку по индексу страницы (индекс страницы начинается с 0).

```csharp
public bool RemoveLabel(int pageIndex)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pageIndex | Int32 | Индекс страницы, на которой метка должна быть удалена. |

### Возвращаемое значение

true, если операция была выполнена успешно.
