## Перечисление CollectionFieldSubtype

Представляет параметр подтипа поля в коллекции схем.

```csharp
public enum CollectionFieldSubtype
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | `0` | Подтип не определен. |
| S | `1` | Тип текста. Данные поля должны храниться как строка текста PDF. |
| D | `2` | Тип даты. Данные поля должны храниться как строка даты PDF. |
| N | `3` | Тип числа. Данные поля должны храниться как число PDF. |
| F | `4` | Данные поля должны быть именем файла встроенного потока файла, как указано в записи UF спецификации файла, если она присутствует; в противном случае по записи F спецификации файла |
| Desc | `5` | Данные поля должны быть именем файла встроенного потока файла, как указано в записи UF спецификации файла, если она присутствует; в противном случае по записи F спецификации файла |
| ModDate | `6` | Данные поля должны быть датой изменения встроенного потока файла, как указано в записи ModDate в словаре параметров встроенного файла. |
| CreationDate | `7` | Данные поля должны быть датой создания встроенного потока файла, как указано в записи CreationDate во встроенном файле |
| Size | `8` | Данные поля должны быть размером встроенного файла, как указано в записи Size в словаре параметров встроенного файла |
| CompressedSize | `9` | (PDF 2.0) Данные поля - это длина встроенного потока файла, как указано в записи Length в словаре встроенного потока файла, и два значения должны быть идентичны. |
