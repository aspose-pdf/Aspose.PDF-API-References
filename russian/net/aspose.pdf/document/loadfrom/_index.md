## Метод Document.LoadFrom

Загружает файл, конвертируя его в PDF.

```csharp
public void LoadFrom(string filename, LoadOptions options)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| filename | String | Путь к файлу, который нужно открыть. |
| options | LoadOptions | Параметры загрузки. |

### Исключения

| исключение | условие |
| --- | --- |
| [PdfException](../../pdfexception/) | Если файл не может быть загружен. |
| FileNotFoundException | Если файл не найден. |
