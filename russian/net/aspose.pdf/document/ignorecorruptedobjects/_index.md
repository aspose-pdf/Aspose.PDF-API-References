## Document.IgnoreCorruptedObjects property

Получает или устанавливает флаг игнорирования ошибок в исходных файлах. Когда страницы исходного документа копируются в конечный документ, процесс копирования останавливается с исключением , если некоторые объекты в исходных файлах повреждены, когда этот флаг установлен в false. пример: dest.Pages.Add(src.Pages); Если для этого флага установлено значение true, поврежденные объекты будут заменены пустыми значениями. По умолчанию: true.

```csharp
public bool IgnoreCorruptedObjects { get; set; }
```
