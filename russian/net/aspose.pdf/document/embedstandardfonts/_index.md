## Document.EmbedStandardFonts property

Свойство, указывающее, что документ должен включать все стандартные шрифты Type1 , флаг IsEmbedded которого установлен в true. Все шрифты PDF могут быть встроены в документ просто путем установки флага IsEmbedded в значение true, но стандартные шрифты PDF Type1 являются исключением из этого правила. Встраивание шрифта Standard Type1 требует много времени, поэтому для встраивания этих шрифтов необходимо не только установить флаг IsEmbedded в true для указанного шрифта, но и установить дополнительный флаг на уровне документа - EmbedStandardFonts = true; Это свойство можно установить только один раз для всех шрифтов. По умолчанию false.

```csharp
public bool EmbedStandardFonts { get; set; }
```
