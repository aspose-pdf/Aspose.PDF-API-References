## Свойство Document.EmbedStandardFonts

Свойство, которое объявляет, что документ должен встраивать все стандартные шрифты Type1, у которых установлен флаг IsEmbedded в true. Все шрифты PDF могут быть встроены в документ просто путем установки флага IsEmbedded в true, но стандартные шрифты Type1 PDF являются исключением из этого правила. Встраивание стандартных шрифтов Type1 требует много времени, поэтому для встраивания этих шрифтов необходимо не только установить флаг IsEmbedded в true для указанного шрифта, но и установить дополнительный флаг на уровне документа - EmbedStandardFonts = true; Это свойство может быть установлено только один раз для всех шрифтов. По умолчанию false.

```csharp
public bool EmbedStandardFonts { get; set; }
```
