## Свойство Document.AllowReusePageContent

Позволяет объединять содержимое страниц для оптимизации размера документа. Если используется, то разные, но дублирующиеся страницы могут ссылаться на один и тот же объект содержимого. Обратите внимание, что этот режим может вызвать побочные эффекты, такие как изменение содержимого страницы при изменении другой страницы.

```csharp
public bool AllowReusePageContent { get; set; }
```
