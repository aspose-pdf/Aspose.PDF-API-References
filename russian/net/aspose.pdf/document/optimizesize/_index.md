## Document.OptimizeSize property

Получает или устанавливает флаг оптимизации. Когда страницы добавляются в документ, одинаковые потоки ресурсов в результирующем файле объединяются в один объект PDF, если этот флаг установлен. Это позволяет уменьшить результирующий размер файла, но может привести к замедлению выполнения и увеличению требований к памяти. Значение по умолчанию:false.

public bool OptimizeSize { get; set; }
