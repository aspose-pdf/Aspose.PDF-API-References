## Document.Optimize method

Линеаризовать документ, чтобы - как можно быстрее открыть первую страницу; - максимально быстро отображать следующую страницу или переходить по ссылке на следующую страницу; - отображать страницу постепенно по мере ее поступления, когда данные для страницы доставляются по медленному каналу (сначала отображать наиболее полезные данные); - разрешить взаимодействие с пользователем, например переход по ссылке, даже до того, как вся страница будет получена и отображена. Вызов этого метода фактически не сохраняет документ. Напротив, документ подготовлен только для того, чтобы иметь оптимизированную структуру, вызовите затем Сохранить, чтобы получить оптимизированный документ.

```csharp
public void Optimize()
```
