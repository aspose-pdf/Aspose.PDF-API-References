## Свойство HtmlSaveOptions.RenderTextAsImage

Если атрибут RenderTextAsImage установлен в true, текст из источника становится изображением в HTML. Может быть полезно, чтобы сделать текст невыбираемым или если HTML текст не отображается правильно.

```csharp
public bool RenderTextAsImage { get; set; }
```
