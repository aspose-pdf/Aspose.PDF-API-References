## HtmlSaveOptions.CustomStrategyOfCssUrlCreation field

Это поле может содержать пользовательский метод, возвращающий URL-адрес (или шаблон URL-адреса, если включена многостраничная генерация - см. подробности ниже) предмета CSS в том виде, в каком он должен быть помещен в сгенерированный результирующий HTML. Fe, если вы хотите, чтобы конвертер поместил какой-то конкретный URL-адрес вместо стандартного Имя файла CSS в сгенерированный CSS, тогда вам нужно просто создать и поместить в это свойство метод который генерирует желаемый URL. Если установлен флаг 'SplitCssIntoPages', то эта пользовательская стратегия (если есть) должна возвращать не точный URL CSS, а строку шаблона, которая (после замена заполнителя номером страницы с помощью функции string.Format() внутри конвертера) может быть преобразована в URL-адрес для той или иной URL-адрес CSS страницы. Примеры ожидаемой возвращаемой строки в таком случае: 'SomeTargetLocation-page_{0}.css','../PartHandlers/GetCss.aspx?DocumentId=45654&amp;CssPage ={0}')

```csharp
public CssUrlMakingStrategy CustomStrategyOfCssUrlCreation;
```
