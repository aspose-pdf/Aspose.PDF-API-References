## Свойство HtmlSaveOptions.MinimalLineWidth

Этот атрибут устанавливает минимальную ширину линии графического пути. Если толщина линии меньше 1px, Adobe Acrobat округляет ее до этого значения. Таким образом, этот атрибут можно использовать для эмуляции этого поведения в HTML-браузерах.

```csharp
public float MinimalLineWidth { get; set; }
```
