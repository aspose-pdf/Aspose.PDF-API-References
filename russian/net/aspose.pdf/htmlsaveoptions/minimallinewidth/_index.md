## HtmlSaveOptions.MinimalLineWidth property

Этот атрибут устанавливает минимальную ширину линии графического контура. Если толщина линии меньше 1 пикселя, Adobe Acrobat округляет ее до этого значения. Таким образом, этот атрибут может использоваться для эмуляции этого поведения для HTML-браузеров.

public float MinimalLineWidth { get; set; }
