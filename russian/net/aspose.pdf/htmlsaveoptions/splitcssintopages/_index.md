## HtmlSaveOptions.SplitCssIntoPages property

Когда выбран многостраничный режим (т.е. 'SplitIntoPages' равен 'true'), тогда этот атрибут определяет, должен ли быть создан отдельный CSS-файл для каждой HTML-страницы результата. По умолчанию этот атрибут равен false, поэтому будет создан один большой общий CSS для всех создаваемых страниц. Суммарный размер всех CSS, созданных в этом режиме (один CSS на страницу), обычно намного больше, чем размер одного большого файла CSS, потому что в первом случае Классы CSS дублируются в этом случае в нескольких файлах CSS для каждой страницы. Таким образом, этот параметр хуже использовать только тогда, когда Вы заинтересованы в дальнейшей обработке каждой страницы HTML самостоятельно, и, следовательно, размер CSS из каждая отдельная страница, разобранная на части, является наиболее важной проблемой.

public bool SplitCssIntoPages { get; set; }
