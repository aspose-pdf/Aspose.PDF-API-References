## Свойство HtmlSaveOptions.SplitCssIntoPages

Когда выбран режим многопстраничного документа (т.е. 'SplitIntoPages' равно 'true'), это свойство определяет, следует ли создавать отдельный CSS-файл для каждой результирующей HTML-страницы. По умолчанию это свойство равно false, поэтому будет создан один большой общий CSS для всех созданных страниц. Суммарный размер всех CSS, сгенерированных в этом режиме (один CSS на страницу), обычно намного больше размера одного большого CSS-файла, потому что в первом случае CSS-классы дублируются в нескольких CSS-файлах для каждой страницы. Поэтому эта настройка хуже использовать только тогда, когда вас интересует дальнейшая обработка каждой HTML-страницы независимо, и, следовательно, размер CSS каждой страницы отдельно является наиболее критическим вопросом.

```csharp
public bool SplitCssIntoPages { get; set; }
```
