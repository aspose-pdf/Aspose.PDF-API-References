## HtmlSaveOptions.TrySaveTextUnderliningAndStrikeoutingInCss field

PDF сам по себе не содержит маркеров подчеркивания для текстов. Это эмулируется линией, расположенной под текстом. Эта опция позволяет конвертеру попытаться угадать, что эта или та линия является подчеркиванием текста, и поместить эту информацию в CSS вместо графического рисования подчеркивания.

```csharp
public bool TrySaveTextUnderliningAndStrikeoutingInCss;
```
