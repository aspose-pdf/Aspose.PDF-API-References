## HtmlSaveOptions.CustomCssSavingStrategy field

Это поле может содержать стратегию сохранения , которая должна использоваться (если присутствует) при преобразовании из Pdf в Html для обработка сохранения CSS-файлов, связанных с созданным HTML-документом в целом или с его страницами (если создается несколько HTML-страниц) Если вы хотите обрабатывать CSS-файл каким-то определенным образом, просто пожалуйста, создайте соответствующий метод и назначьте делегата, созданного из него, этому свойству.

```csharp
public CssSavingStrategy CustomCssSavingStrategy;
```
