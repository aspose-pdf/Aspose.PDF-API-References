## HtmlSaveOptions.CustomCssSavingStrategy field

Это поле может содержать стратегию сохранения , которая должна использоваться (если присутствует) во время преобразования Pdf в HTML для управления сохранением CSS, связанных с созданным HTML-документом целиком или с его страницами (если создается несколько HTML-страниц) Если хотите обрабатывать файл CSS определенным образом, просто создайте соответствующий метод и назначьте делегата, созданного из него, этому свойству.

```csharp
public CssSavingStrategy CustomCssSavingStrategy;
```
