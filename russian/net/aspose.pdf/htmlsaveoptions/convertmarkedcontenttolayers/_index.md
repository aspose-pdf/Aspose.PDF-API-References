## HtmlSaveOptions.ConvertMarkedContentToLayers property

Если для атрибута ConvertMarkedContentToLayers установлено значение true, то все элементы внутри PDF-файла с пометкой содержимое (слой) будут помещены в HTML-элемент div с данными Атрибут -pdflayer, указывающий имя слоя. Имя этого слоя будет извлечено из необязательных свойств маркированного содержимого PDF. Если этот атрибут имеет значение false (по умолчанию), то никакие слои не будут созданы из содержимого, отмеченного PDF.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
