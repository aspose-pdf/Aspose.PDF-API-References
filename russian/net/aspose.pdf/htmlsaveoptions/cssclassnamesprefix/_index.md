## HtmlSaveOptions.CssClassNamesPrefix field

Когда конвертер PDFtoHTML генерирует результирующие CSS, имена классов CSS (что-то вроде ".stl_01 {}" ... ".stl_NN {}) генерируются и используются в результирующем CSS. Это свойство позволяет принудительно установить префикс имени класса Например, если вы хотите, чтобы все имена классов начинались с 'my_prefix_' (т.е. были чем-то вроде 'my_prefix_1'... 'my_prefix_NNN'), , тогда просто назначьте 'my_prefix_' этому свойству перед преобразованием. Если это свойство останется нетронутым (т.е. null будет оставлен как значение ), тогда конвертер сам сгенерирует имена классов (это будет что-то вроде ".stl_01 {}" ... ".stl_NN {}")

public string CssClassNamesPrefix;
