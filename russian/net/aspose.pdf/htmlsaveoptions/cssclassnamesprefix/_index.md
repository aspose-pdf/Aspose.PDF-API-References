## Поле HtmlSaveOptions.CssClassNamesPrefix

Когда конвертер PDFtoHTML генерирует результирующие CSS, генерируются и используются имена классов CSS (что-то вроде ".stl_01 {}" ... ".stl_NN {}"). Это свойство позволяет принудительно установить префикс имени класса. Например, если вы хотите, чтобы все имена классов начинались с 'my_prefix_' (т.е. были чем-то вроде 'my_prefix_1' ... 'my_prefix_NNN'), просто присвойте 'my_prefix_' этому свойству перед конвертацией. Если это свойство останется нетронутым (т.е. значение будет равно null), то конвертер сам сгенерирует имена классов (это будет что-то вроде ".stl_01 {}" ... ".stl_NN {}")

```csharp
public string CssClassNamesPrefix;
```
