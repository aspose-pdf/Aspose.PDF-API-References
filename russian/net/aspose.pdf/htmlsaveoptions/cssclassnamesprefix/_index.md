## HtmlSaveOptions.CssClassNamesPrefix field

Когда конвертер PDFtoHTML генерирует результирующие CSS, имена классов CSS (что-то вроде ".stl_01 {}" ... ".stl_NN {}) генерируются и используются в результирующем CSS Это свойство позволяет принудительно установить префикс имени класса Например, если Вы хотите, чтобы все имена классов начинались с 'my_prefix_' (т.е. что-то вроде 'my_prefix_1' ... 'my_prefix_NNN' ), тогда просто присвойте 'my_prefix_' этому свойству перед преобразованием. Если это свойство останется нетронутым (т.е. в качестве значения будет оставлено значение null ), то конвертер сам сгенерирует имена классов (это будет что-то вроде ".stl_01 {}" ... ".stl_NN { }")

```csharp
public string CssClassNamesPrefix;
```
