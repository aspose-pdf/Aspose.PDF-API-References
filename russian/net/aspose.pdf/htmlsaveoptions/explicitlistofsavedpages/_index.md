## HtmlSaveOptions.ExplicitListOfSavedPages property

С помощью этого свойства вы можете явно определить какие страницы документа должны быть преобразованы. Страницы в этом списке должны иметь номера, начинающиеся с 1. Т.е. корректные номера страниц должны быть взяты из диапазона (1...[NumberOfPagesInConvertedDocument]) Порядок появления страниц в этом списке не влияет на их порядок в результирующей HTML-странице(ах) - в результирующих страницах всегда будет идти по порядку в котором они присутствуют в исходном PDF. Если этот список пуст (по умолчанию), все страницы будут преобразованы. количество страниц в документе]) будет выдано исключение.
