## Свойство HtmlSaveOptions.UseZOrder

Если атрибут UseZOrder установлен в true, графика и текст добавляются в результирующий HTML-документ в соответствии с Z-порядком в оригинальном PDF-документе. Если этот атрибут равен false, вся графика помещается в один слой, что может вызвать некоторые нежелательные эффекты для перекрывающихся объектов.

```csharp
public bool UseZOrder { get; set; }
```
