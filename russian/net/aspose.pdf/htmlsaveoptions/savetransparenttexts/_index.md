## HtmlSaveOptions.SaveTransparentTexts field

Pdf может содержать прозрачные тексты, которые можно выбрать для буфера обмена (обычно это происходит, когда документ содержит изображения и извлеченные из него тексты с помощью OCR). Эта настройка указывает конвертеру, нужно ли сохранять такие тексты как прозрачные выбираемые тексты в результирующем HTML.

```csharp
public bool SaveTransparentTexts;
```
