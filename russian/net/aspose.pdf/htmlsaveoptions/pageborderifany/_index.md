## Поле HtmlSaveOptions.PageBorderIfAny

Этот атрибут представляет собой набор настроек, используемых для рисования границы (если таковая имеется) в результирующем HTML-документе вокруг области, представляющей исходную PDF-страницу. По сути, это касается отображения краев бумаги страницы, а не границы страницы, упомянутой в самой PDF-странице.

```csharp
public BorderInfo PageBorderIfAny;
```
