## HtmlSaveOptions.PreventGlyphsGrouping property

Этот атрибут включает режим, при котором текстовые глифы не будут группироваться в слова и строки Этот режим позволяет сохранять максимальную точность при позиционировании глифов на странице и его можно использовать для преобразования документов с нотами или глифами, которые должны располагаться отдельно друг от друга. Этот параметр будет применяться к документу только в том случае, если значение атрибута FixedLayout равно true.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
