## Свойство HtmlSaveOptions.PreventGlyphsGrouping

Этот атрибут включает режим, когда текстовые глифы не будут сгруппированы в слова и строки. Этот режим позволяет сохранить максимальную точность при позиционировании глифов на странице, и его можно использовать для конвертации документов с музыкальными нотами или глифами, которые должны быть размещены отдельно друг от друга. Этот параметр будет применен к документу только в том случае, если значение атрибута FixedLayout равно true.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
