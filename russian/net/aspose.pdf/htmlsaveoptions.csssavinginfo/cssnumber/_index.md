## Поле HtmlSaveOptions.CssSavingInfo.CssNumber

Устанавливается конвертером. Во время конвертации создается несколько CSS-файлов. Это свойство показывает порядковый номер сохраненного CSS-файла во время конвертации. Его можно использовать в логике пользовательского кода, чтобы решить, как обрабатывать или куда сохранять содержимое CSS

```csharp
public int CssNumber;
```
