## PdfFileSignature.GetSignNames method

Получает имена всех непустых подписей.

```csharp
public IList<string> GetSignNames(bool onlyActive = true)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| onlyActive | Boolean | если true, вернуть только активные подписи; в противном случае вернуть все подписи. |

### Возвращаемое значение

Возвращает IList&lt;string&gt;.
