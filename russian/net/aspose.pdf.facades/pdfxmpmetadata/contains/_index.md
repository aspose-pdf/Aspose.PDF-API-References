
| Параметр | Тип | Описание |
| --- | --- | --- |
| key | String | Ключ, который будет проверяться. |

### Возвращаемое значение

### Примеры

```csharp

xmp.BindPdf("input.pdf");
xmp.Add("xmp:Nickname", "Nickname1");
if (!xmp.Contains("xmp:Nickname"))
  Console.WriteLine("Key does not exists");
```
