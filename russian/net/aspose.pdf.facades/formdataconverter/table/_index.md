## FormDataConverter.Table property

Получает или задает средний контейнер данных, один DataTable. Должен быть определен перед преобразованием данных из одного формата в другой. Должны быть определены столбцы и имя таблицы таблицы данных. TableName — это имя таблицы в базе данных. Каждый столбец ColumnName является полным именем поля PDF. Заголовок каждого столбца - это имя столбца таблицы в базе данных. Если имя поля совпадает с именем столбца таблицы name, Заголовок указывать не нужно.
