## FormDataConverter.Table property

Получает или задает средний контейнер данных, один DataTable. Он должен быть определен перед преобразованием данных из одного формата в другой формат. Должны быть определены столбцы и TableName таблицы DataTable. database. Имя столбца каждого столбца — это полное имя поля PDF-файла. Caption каждого столбца — это имя столбца таблицы в базе данных. Если имя поля совпадает с именем таблицы column , заголовок указывать не нужно.
