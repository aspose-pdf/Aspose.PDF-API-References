## FormEditor.SetFieldCombNumber method

Устанавливает количество гребней для обычного однострочного текстового поля (поле автоматически делится на столько равноотстоящих позиций или гребней, , как значение параметра combNumber).

```csharp
public bool SetFieldCombNumber(string fieldName, int combNumber)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fieldName | String | Полное имя поля. |
| combNumber | Int32 | Количество сот, на которые нужно разделить поле. |

### Возвращаемое значение

В случае успеха вернуть true, иначе false.

### Примеры

```csharp
FormEditor formEditor = new FormEditor("PdfWithAcroForm.pdf", "FormEditor_SetFieldComb.pdf"));
formEditor.SetFieldCombNumber("textCombField", 5);
```
