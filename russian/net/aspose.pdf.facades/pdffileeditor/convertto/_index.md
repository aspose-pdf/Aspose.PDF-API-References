## Свойство PdfFileEditor.ConvertTo

Устанавливает формат PDF файла. Результирующий файл будет сохранен в указанном формате файла. Если это свойство не указано, файл будет сохранен в формате PDF по умолчанию без конвертации.

```csharp
public PdfFormat ConvertTo { set; }
```
