## Свойство PdfFileEditor.KeepFieldsUnique

Если истинно, то имена полей будут уникальными при конкатенации форм. Суффиксы будут добавлены к именам полей, шаблон суффикса может быть указан в свойстве UniqueSuffix.

```csharp
public bool KeepFieldsUnique { get; set; }
```
