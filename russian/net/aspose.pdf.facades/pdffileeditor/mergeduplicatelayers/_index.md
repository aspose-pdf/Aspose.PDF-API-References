## PdfFileEditor.MergeDuplicateLayers property

Необязательное содержимое объединенных документов с одинаковыми именами будет объединено в один слой результирующего документа, если это свойство истинно. В противном случае слои с одинаковыми именами будут сохранены как разные слои в результирующем документе.

public bool MergeDuplicateLayers { get; set; }
