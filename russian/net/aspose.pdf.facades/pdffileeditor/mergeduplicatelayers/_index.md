## Свойство PdfFileEditor.MergeDuplicateLayers

Необязательное содержимое объединенных документов с одинаковыми именами будет объединено в один слой в результирующем документе, если это свойство истинно. В противном случае слои с одинаковыми именами будут сохранены как разные слои в результирующем документе.

```csharp
public bool MergeDuplicateLayers { get; set; }
```
