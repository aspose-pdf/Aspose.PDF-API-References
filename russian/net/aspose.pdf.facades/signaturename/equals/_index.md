## Метод SignatureName.Equals

Определяет, равны ли этот экземпляр и указанный объект.

```csharp
public override bool Equals(object obj)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | Object | Объект, который нужно сравнить с текущим экземпляром. |

### Возвращаемое значение

True, если указанный объект равен текущему экземпляру; в противном случае - false.
