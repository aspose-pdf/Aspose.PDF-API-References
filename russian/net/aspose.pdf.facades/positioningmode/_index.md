## PositioningMode enumeration

Определяет режим позиционирования. Возможные значения включают Legacy (обратная совместимость) и Current (обновленный метод расчета позиции текста)

```csharp
public enum PositioningMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Legacy | `0` | Устаревшее позиционирование текста |
| ModernLineSpacing | `1` | Обновлен межстрочный интервал, расчет положения по вертикали производится по старым правилам (т.е. текст позиционируется относительно левого нижнего угла указанного прямоугольника) |
| Current | `2` | Обновлен межстрочный интервал и расчет вертикального положения выполняется на основе левого верхнего угла, а не левого нижнего. |
