## Перечисление PositioningMode

Определяет режим позиционирования. Возможные значения включают Legacy (обратная совместимость) и Current (обновленный метод расчета позиции текста)

```csharp
public enum PositioningMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Legacy | `0` | Устаревшее позиционирование текста |
| ModernLineSpacing | `1` | Обновленный межстрочный интервал, вертикальное позиционирование рассчитывается по старым правилам (т.е. текст позиционируется относительно нижнего левого угла указанного прямоугольника) |
| Current | `2` | Обновленный межстрочный интервал и вертикальное позиционирование рассчитывается на основе верхнего левого угла, а не нижнего левого. |
