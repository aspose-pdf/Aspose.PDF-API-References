## Перечисление DefaultMetadataProperties

Перечисление стандартных свойств XMP.

```csharp
public enum DefaultMetadataProperties
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Advisory | `0` | Свойство xmp:Advisory. Неупорядоченный массив, указывающий свойства, которые были изменены вне приложения для авторинга. Каждый элемент должен содержать одно /// пространство имен и XPath, разделенные одним ASCII-пробелом |
| BaseURL | `1` | Свойство xmp:BaseURL. Базовый URL для относительных URL в содержимом документа. Если этот документ содержит интернет-ссылки, и эти ссылки относительные, /// они относительны к этому базовому URL. Это свойство предоставляет стандартный способ интерпретации встроенных относительных URL инструментами. /// Инструменты веб-авторинга должны устанавливать значение на основе их представления о том, где URL будут интерпретированы |
| CreateDate | `2` | Свойство xmp:CreateDate. Дата и время, когда ресурс был изначально создан. |
| CreatorTool | `3` | Свойство xmp:CreatorTool. Название первого известного инструмента, использованного для создания ресурса. |
| Identifier | `4` | Свойство xmp:Identifier. Неупорядоченный массив текстовых строк, которые однозначно идентифицируют ресурс в данном контексте |
| MetadataDate | `5` | Свойство xmp:MetadataDate. Дата и время, когда любые метаданные для этого ресурса были в последний раз изменены |
| ModifyDate | `6` | Свойство xmp:ModifyDate. Дата и время, когда ресурс был в последний раз изменен. |
| Nickname | `7` | Свойство xmp:Nickname. Краткое неформальное название для ресурса. |
| Thumbnails | `8` | Свойство xmp:Thumbnails. Альтернативный массив изображений миниатюр для файла, которые могут отличаться по характеристикам, таким как размер или кодирование изображения. |
