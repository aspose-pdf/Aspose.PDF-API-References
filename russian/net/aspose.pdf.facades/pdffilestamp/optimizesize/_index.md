## Свойство PdfFileStamp.OptimizeSize

Получает или устанавливает флаг оптимизации. Равные потоки ресурсов в результирующем файле объединяются в один объект PDF, если этот флаг установлен. Это позволяет уменьшить размер результирующего файла, но может привести к более медленному выполнению и большим требованиям к памяти. Значение по умолчанию: false.

```csharp
public bool OptimizeSize { get; set; }
```
