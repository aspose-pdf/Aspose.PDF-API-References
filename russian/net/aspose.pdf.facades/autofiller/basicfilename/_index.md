## Свойство AutoFiller.BasicFileName

Получает или устанавливает базовое имя файла, если будет сгенерировано много маленьких файлов. Сгенерированный файл будет выглядеть как "BasicFileName0","BasicFileName1",... Оно работает с другим свойством [`GeneratingPath`](../generatingpath/)GeneratingPath.

```csharp
public string BasicFileName { get; set; }
```
