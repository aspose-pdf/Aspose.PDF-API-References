## PdfViewer.AutoResize property

Получает или задает логическое значение, указывающее, будет ли файл распечатан с оптимизированным размером.  Если false, напечатать страницу без масштабирования страницы. Если true, напечатать страницу с масштабированием, чтобы оно соответствовало области печати.

public bool AutoResize { get; set; }
