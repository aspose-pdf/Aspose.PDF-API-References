## PdfViewer.AutoResize property

Получает или задает логическое значение, указывающее, следует ли печатать файл с оптимизированным размером.  Если false печатать страницу без масштабирования страницы. Если true, распечатайте страницу с масштабированием, чтобы соответствовать области печати.

public bool AutoResize { get; set; }
