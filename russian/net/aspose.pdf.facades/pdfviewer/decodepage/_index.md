## Метод PdfViewer.DecodePage

Декодирует страницу одного Pdf файла.

```csharp
public Bitmap DecodePage(int pageNumber)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pageNumber | Int32 | Номер страницы одного Pdf файла, который должен быть между 1 и PageCount. |

### Возвращаемое значение

возвращает изображение страницы Pdf.
