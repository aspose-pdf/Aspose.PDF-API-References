Устанавливает или получает режим печати PdfViewer как изображения.


### Примечания

При значении true всегда печатается как изображение (создается изображение, которое печатается) При значении false печатается непосредственно на устройство, если все функции поддерживаются. Если документ содержит неподдерживаемые функции, система может автоматически принять решение о печати в виде изображения. Значение по умолчанию — false.
