## Событие PdfViewer.StartPage

Происходит перед началом печати страницы.

```csharp
public event EventHandler<StartEndPageEventArgs> StartPage;
```

## Примечания

Это событие вызывается для каждой страницы, которая собирается быть напечатанной. Его можно использовать для выполнения пользовательских действий перед началом печати страницы.
