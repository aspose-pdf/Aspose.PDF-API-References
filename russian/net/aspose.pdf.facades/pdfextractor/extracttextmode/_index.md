## PdfExtractor.ExtractTextMode property

Устанавливает режим извлечения результата текста.

```csharp
public int ExtractTextMode { get; set; }
```

### Стоимость имущества

0 — это чисто текстовый режим, а 1 — режим необработанного упорядочения. По умолчанию 0.

### Примеры

Пример демонстрирует`ExtractTextMode`использование свойства в сценарии извлечения текста.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractTextMode = 1;
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```
