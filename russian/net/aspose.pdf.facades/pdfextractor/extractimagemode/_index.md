## Свойство PdfExtractor.ExtractImageMode

Устанавливает режим для процесса извлечения изображений.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## Примечания

Значение по умолчанию - ExtractImageMode.DefinedInResources, которое извлекает все изображения, определенные в ресурсах. Для извлечения фактически отображаемых изображений следует использовать режим ExtractImageMode.ActuallyUsed.
