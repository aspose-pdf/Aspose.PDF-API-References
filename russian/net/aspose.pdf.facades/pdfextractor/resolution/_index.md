## Свойство PdfExtractor.Resolution

Устанавливает или получает разрешение для извлеченных изображений. Значение по умолчанию - 150. Изображения с большим значением разрешения более четкие. Однако увеличение значения разрешения приводит к увеличению времени и памяти, необходимых для извлечения изображений. Обычно для получения четкого изображения достаточно установить разрешение на 150 или 300.

```csharp
public int Resolution { get; set; }
```
