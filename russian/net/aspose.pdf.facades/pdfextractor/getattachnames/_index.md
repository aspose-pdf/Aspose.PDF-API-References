## PdfExtractor.GetAttachNames method

Возвращает список вложений в файле PDF. Примечание. Перед использованием этого метода необходимо вызвать ExtractAttachments.

```csharp
public IList<string> GetAttachNames()
```

### Возвращаемое значение

Список вложений

### Примеры

Пример демонстрирует, как извлечь имена вложений из файла PDF.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(TestSettings.GetInputFile("sample.pdf"));
extractor.ExtractAttachment();
IList attachments = extractor.GetAttachNames();
foreach (string name in attachments)
	Console.WriteLine(name);
```
