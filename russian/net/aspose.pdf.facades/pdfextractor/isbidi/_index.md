## Свойство PdfExtractor.IsBidi

Истинно, когда текст содержит еврейские или арабские символы. Этот случай должен рассматриваться особенно, потому что функции строк изменяют свое поведение и начинают обрабатывать текст справа налево (за исключением чисел и других не текстовых символов).

```csharp
public bool IsBidi { get; }
```
