## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable property

Получает значение, указывающее, что замена неизбежна.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

### Примечания

Возвращает true, если замена была запрошена из-за отсутствия исходного шрифта или если исходный шрифт не может использоваться в контексте какой-либо задачи. В случае, если пользователь игнорирует флаг и не заменяет шрифт - выполняется процедура замены шрифта по умолчанию. Но это дает пользователю возможность изменить стандартную процедуру замены шрифта и установить лучший шрифт для системы. Возвращает false, если исходный шрифт присутствует, допустим, но пользователю разрешено заменить его.
