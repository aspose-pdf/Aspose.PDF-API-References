## Метод TextSegmentCollection.Contains

Определяет, содержит ли коллекция конкретное значение.

```csharp
public bool Contains(TextSegment item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | TextSegment | Объект, который нужно найти в коллекции |

### Возвращаемое значение

true, если item найден в коллекции; в противном случае false.
