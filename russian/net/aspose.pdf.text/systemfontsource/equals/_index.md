## Метод SystemFontSource.Equals

Проверьте, равны ли объекты источника системного шрифта.

```csharp
public override bool Equals(object obj)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | Object | Объект источника системного шрифта, который будет сравнен. |

### Возвращаемое значение

True, если оба объекта являются источниками системного шрифта, в противном случае false.
