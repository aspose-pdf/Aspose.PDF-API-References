## Перечисление TextReplaceOptions.Scope

Область, в которой применяется операция замены текста REPLACE_FIRST по умолчанию. Этот устаревший параметр был сохранен для совместимости. Он влияет на PdfContentEditor и не оказывает влияния на TextFragmentAbsorber.

```csharp
public enum Scope
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| REPLACE_FIRST | `0` | Заменить только первое вхождение текста на каждой из затронутых страниц |
| REPLACE_ALL | `1` | Заменить все вхождения текста на всех затронутых страницах |
