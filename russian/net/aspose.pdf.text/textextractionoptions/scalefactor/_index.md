## TextExtractionOptions.ScaleFactor property

Получает или задает коэффициент, который будет применяться для масштабирования размера шрифта при извлечении в чистом режиме. Установка меньшего значения приводит к увеличению пробелов в извлеченном тексте. Значение по умолчанию 1 - без масштабирования; Установка значения в ноль позволяет алгоритму автоматически выбирать масштабирование.
