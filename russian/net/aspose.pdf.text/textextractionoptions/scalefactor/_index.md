## Свойство TextExtractionOptions.ScaleFactor

Получает или устанавливает коэффициент, который будет применяться для масштабирования размера шрифта во время извлечения в чистом режиме. Установка меньшего значения приводит к большему количеству пробелов в извлеченном тексте. Значение по умолчанию - 1 - без масштабирования; Установка значения в ноль позволяет алгоритму автоматически выбирать масштабирование.

```csharp
public double ScaleFactor { get; set; }
```
