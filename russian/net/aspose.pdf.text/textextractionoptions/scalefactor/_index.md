## TextExtractionOptions.ScaleFactor property

Получает или задает коэффициент, который будет применяться для масштабирования размера шрифта во время извлечения в чистом режиме. Установка меньшего значения приводит к большему количеству пробелов в извлеченном тексте. Значение по умолчанию: 1 — без масштабирования; Установка нулевого значения позволяет выбирать алгоритм масштабирование автоматически.

public double ScaleFactor { get; set; }
