## Свойство TextFragmentAbsorber.HasErrors

Значение указывает, были ли найдены ошибки во время извлечения текста. Поиск ошибок будет выполнен только если TextSearchOptions.LogTextExtractionErrors = true; И это может снизить производительность.

```csharp
public bool HasErrors { get; }
```
