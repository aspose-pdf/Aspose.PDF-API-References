## Свойство TextParagraph.SubsequentLinesIndent

Получает или устанавливает значение отступа для последующих строк. Если установлено ненулевое значение, оно имеет преимущество над значением FormattingOptions.SubsequentLinesIndent.

```csharp
public float SubsequentLinesIndent { get; set; }
```
