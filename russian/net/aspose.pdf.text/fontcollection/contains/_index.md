## Contains(string) {#contains_1}

Проверяет, существует ли шрифт в коллекции шрифтов.

```csharp
public bool Contains(string name)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | String | Название шрифта. |

### Возвращаемое значение

True в случае, если коллекция содержит шрифт с указанным названием.

## Contains(Font) {#contains}

Определяет, содержит ли коллекция конкретное значение.

```csharp
public bool Contains(Font item)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| item | Font | Объект, который нужно найти в коллекции |

### Возвращаемое значение

true, если элемент найден в коллекции; в противном случае false.
