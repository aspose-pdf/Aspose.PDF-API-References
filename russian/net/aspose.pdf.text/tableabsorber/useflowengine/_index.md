## Свойство TableAbsorber.UseFlowEngine

* Включите альтернативный движок распознавания таблиц, который превосходит в многочисленных сценариях и способен распознавать таблицы без границ. Пока не поддерживает редактирование таблиц и получение стилей текста. Значение по умолчанию - false;

public bool UseFlowEngine { get; set; }
