## Свойство TextFragmentState.CoordinateOrigin

Получает или устанавливает текстовый CoordinateOrigin. Если CoordinateOrigin равен Descender, координата Y текста соответствует самой низкой точке шрифта. Если CoordinateOrigin равен BaseLine, координата Y текста соответствует базовой линии шрифта. Значение по умолчанию - Descender. Если значение Descent шрифта слишком велико, текст может отображаться выше, чем другие шрифты. В этом случае можно выбрать CoordinateOrigin BaseLine для лучшего отображения текста.

```csharp
public override CoordinateOrigin CoordinateOrigin { get; set; }
```
