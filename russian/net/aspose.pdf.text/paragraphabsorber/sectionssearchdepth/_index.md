## ParagraphAbsorber.SectionsSearchDepth property

Получает или задает значение, указывающее, сколько раз будет выполняться последовательный поиск более мелких элементов структуры. единицы (столбцы).

```csharp
public int SectionsSearchDepth { get; set; }
```

### Примечания

Увеличение этого значения может привести к незначительному снижению производительности без видимых изменений в результатах поиска. Уменьшение этого значения может привести к некорректному определению абзацев в разделах. Мы не рекомендуем устанавливать значение меньше значения по умолчанию, если вы не желание получить только «грубые» элементы структуры страницы.
