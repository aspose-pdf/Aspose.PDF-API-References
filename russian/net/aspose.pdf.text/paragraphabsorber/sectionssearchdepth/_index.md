## ParagraphAbsorber.SectionsSearchDepth property

Получает или задает значение, указывающее, сколько раз будет выполняться последовательный поиск более мелких элементов структуры. Глубина поиска по умолчанию равна 3. Это означает три поиска разделов, разделенных по горизонтали (заголовки, абзацы и т. д.), и три поиска разделов, разделенных по вертикали (столбцы).

```csharp
public int SectionsSearchDepth { get; set; }
```

### Примечания

Увеличение этого значения может привести к незначительному снижению производительности без видимых изменений в результатах поиска. Уменьшение этого значения может привести к некорректному определению абзацев в разделах. Мы не рекомендуем устанавливать значение меньше значения по умолчанию, если вы не хотите получать только "грубые" элементы структуры страницы.
