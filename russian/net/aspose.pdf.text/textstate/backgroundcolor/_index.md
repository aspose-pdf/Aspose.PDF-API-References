## Свойство TextState.BackgroundColor

Устанавливает цвет фона текста.

```csharp
public virtual Color BackgroundColor { get; set; }
```

## Примечания

Обратите внимание, что значение не сохраняется как характеристика текста в документе. Геттер свойства BackgroundColor работает для объекта в случае, если он был явно установлен ранее с помощью сеттера BackgroundColor для этих объектов. Свойство используется во время выполнения в контексте текущего процесса генерации/модификации.
