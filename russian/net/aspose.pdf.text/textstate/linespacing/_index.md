## Свойство TextState.LineSpacing

Получает или устанавливает межстрочное расстояние текста.

```csharp
public virtual float LineSpacing { get; set; }
```

## Примечания

Обратите внимание, что значение не сохраняется как характеристика текста в документе. Геттер свойства LineSpacing работает для объекта в случае, если он был явно установлен ранее с помощью сеттера LineSpacing для этих объектов. Свойство используется во время выполнения в контексте текущего процесса генерации/модификации.
