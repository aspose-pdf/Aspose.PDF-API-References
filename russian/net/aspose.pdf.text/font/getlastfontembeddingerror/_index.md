## Метод Font.GetLastFontEmbeddingError

Цель этого метода - вернуть описание ошибки, если попытка встроить шрифт не удалась. Если нет случаев ошибки, он возвращает пустую строку.

```csharp
public string GetLastFontEmbeddingError()
```

### Возвращаемое значение

Описание ошибки
