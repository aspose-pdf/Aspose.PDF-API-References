## Метод Font.Save

Сохраняет шрифт в поток. Обратите внимание, что шрифт сохраняется в промежуточный формат TTF, предназначенный для использования только в преобразованной копии оригинального документа. Файл шрифта не предназначен для использования вне контекста оригинального документа.

```csharp
public void Save(Stream stream)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| stream | Stream | Поток для сохранения шрифта. |
