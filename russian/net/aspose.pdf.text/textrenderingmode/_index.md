## TextRenderingMode enumeration

Режим рендеринга текста, Tmode, определяет, должен ли показ текста вызывать обводку контуров глифа, заливку, использование в качестве границы отсечения или некоторую комбинацию этих трех.

```csharp
public enum TextRenderingMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| FillText | `0` | Заполнить текст. |
| StrokeText | `1` | Обводка текста. |
| FillThenStrokeText | `2` | Заполните, затем обведите текст. |
| Invisible | `3` | Ни заливки, ни обводки текста (невидимый). |
| FillTextAndAddPathToClipping | `4` | Заполнить текст и добавить к контуру для обрезки (см. 9.3.6, «Режим рендеринга текста»). |
| StrokeTextAndAddPathToClipping | `5` | Обведите текст и добавьте к контуру для обрезки. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Заполните, затем обведите текст и добавьте к контуру для обрезки. |
| AddPathToClipping | `7` | Добавить текст к контуру для обрезки. |
