## Свойство SystemFontsSubstitution.DefaultFont

Получает или задает шрифт замещения по умолчанию. Шрифт используется, когда не найдено других действительных замен, но начальный шрифт принадлежит целевой категории замещения ([`FontCategories`](../fontcategories/)).

```csharp
public Font DefaultFont { get; set; }
```
