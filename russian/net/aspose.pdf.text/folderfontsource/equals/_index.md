## Метод FolderFontSource.Equals

Проверьте, равны ли объекты источника шрифтов папки.

```csharp
public override bool Equals(object obj)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | Object | Объект источника шрифтов папки, который будет сравнен. |

### Возвращаемое значение

True, если оба объекта являются источниками шрифтов папки, нацеленных на одну и ту же папку.
