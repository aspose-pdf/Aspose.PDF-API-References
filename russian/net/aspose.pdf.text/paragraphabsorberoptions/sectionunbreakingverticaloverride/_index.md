## Свойство ParagraphAbsorberOptions.SectionUnbreakingVerticalOverride

Получает или задает максимальную длину зоны с 'уровнем заполнения ноль', которая не будет рассматриваться как фактический вертикальный разрыв секции (как доля ширины / высоты страницы). Значение по умолчанию - 0.005.

```csharp
public double SectionUnbreakingVerticalOverride { get; set; }
```
