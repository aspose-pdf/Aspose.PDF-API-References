## Свойство ParagraphAbsorberOptions.SectionUnbreakingHorizontalOverride

Получает или устанавливает максимальную длину зоны с 'нулевым уровнем заполнения', которая не будет рассматриваться как фактический горизонтальный разрыв секции (как доля ширины / высоты страницы). Значение по умолчанию - 0.005.

```csharp
public double SectionUnbreakingHorizontalOverride { get; set; }
```
