## Свойство TextAbsorber.TextSearchOptions

Получает или устанавливает параметры поиска текста.

```csharp
public virtual TextSearchOptions TextSearchOptions { get; set; }
```

## Примечания

Позволяет определить прямоугольник, который ограничивает извлекаемый текст. По умолчанию прямоугольник пуст. Это означает, что границы страницы только определяют область извлечения текста.
