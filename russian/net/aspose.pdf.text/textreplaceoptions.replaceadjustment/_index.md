## TextReplaceOptions.ReplaceAdjustment enumeration

Определяет действие, которое будет выполнено после замены фрагмента текста на более короткий. None - никаких действий, замененный текст может перекрывать остальную часть строки; AdjustSpaceWidth - пытается изменить пробелы между словами, чтобы сохранить длину строки; WholeWordsHyphenation - пытается распределить слова между строками абзаца, чтобы сохранить правое поле абзаца; ShiftRestOfLine - сдвигает остаток строки в соответствии с изменением длины текста, длина строки может быть изменена; Значение по умолчанию — ShiftRestOfLine.

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Никаких действий, замененный текст может перекрывать остальную часть строки |
| AdjustSpaceWidth | `1` | Пытается настроить пробелы между словами, чтобы сохранить длину строки |
| WholeWordsHyphenation | `2` | Пытается распределить слова между строками абзаца, чтобы сохранить правое поле абзаца |
| ShiftRestOfLine | `4` | (по умолчанию) Смещает остальную часть строки в соответствии с изменением длины текста, длина строки может быть изменена |
