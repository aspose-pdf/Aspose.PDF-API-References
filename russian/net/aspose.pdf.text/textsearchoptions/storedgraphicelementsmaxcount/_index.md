## TextSearchOptions.StoredGraphicElementsMaxCount property

Получает или задает значение, которое ограничивает поиск связанной с текстом графики (подчеркивания, фона и т. д.) на странице для указанного количества элементов. Значение по умолчанию — 250. Установите меньшее значение в случае проблем с производительностью, попробуйте большее значение в если некоторые графические элементы не были найдены.

```csharp
public int StoredGraphicElementsMaxCount { get; set; }
```
