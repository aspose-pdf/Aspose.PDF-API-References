## Свойство TextSearchOptions.IgnoreResourceFontErrors

Получает или устанавливает указание на то, что ошибки, связанные с отсутствием шрифта, будут игнорироваться поглотителем текстовых (фрагментов). true - означает, что ошибки отсутствия шрифта будут игнорироваться. Текстовые сегменты, которые ссылаются на некорректные ресурсы, будут пропущены во время обработки. false (по умолчанию) - ошибка отсутствия шрифта завершит обработку, вызвав исключение.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
