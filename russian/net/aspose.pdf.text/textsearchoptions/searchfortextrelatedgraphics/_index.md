## Свойство TextSearchOptions.SearchForTextRelatedGraphics

Получает или устанавливает значение, которое позволяет искать графику, связанную с текстом (подчеркивание, фон и т. д.) во время поиска текста. true - поиск графики, связанной с текстом, будет выполнен (значение по умолчанию). false - графические элементы, которые могут присутствовать в исходном документе, будут проигнорированы. Установите это в случае проблем с производительностью или отсутствия необходимости обрабатывать подчеркивание, фон или обрезку.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
