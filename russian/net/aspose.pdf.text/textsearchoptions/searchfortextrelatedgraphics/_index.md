## TextSearchOptions.SearchForTextRelatedGraphics property

Получает или задает значение, разрешающее поиск связанной с текстом графики (подчеркивание, фон и т. д.) во время поиска текста. true — будет выполняться поиск связанной с текстом графики (значение по умолчанию). false — графические элементы, которые могут присутствовать в исходном документе. будет игнорироваться. Установите это в случае проблем с производительностью или отсутствия необходимости обработки подчеркивания, фона или обрезки.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
