## Свойство TextSearchOptions.UseFontEngineEncoding

Получает или устанавливает указание на то, что текст будет искаться с использованием кодировки шрифта. true - означает, что будет использоваться кодировка шрифта (попробуйте это, если поиск текста не удался из-за несовершенной кодировки в документе) false - означает, что будет использоваться кодировка шрифта документа (значение по умолчанию)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
