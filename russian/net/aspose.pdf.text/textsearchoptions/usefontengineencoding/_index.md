## TextSearchOptions.UseFontEngineEncoding property

Получает или задает индикацию того, что текст будет искаться с использованием кодировки механизма шрифтов. true — означает, что будет использоваться кодировка механизма шрифтов (попробуйте это, если текстовый поиск не удается из-за несовершенной кодировки в документе) false — означает, что кодировка шрифта документа будет использоваться (значение по умолчанию)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
