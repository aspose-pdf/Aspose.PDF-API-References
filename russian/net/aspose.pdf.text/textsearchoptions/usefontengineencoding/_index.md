## TextSearchOptions.UseFontEngineEncoding property

Получает или задает указание на то, что текст будет искаться с использованием кодировки механизма шрифтов. true - означает, что будет использоваться кодировка шрифтового движка (попробуйте, если текстовый поиск не работает из-за несовершенной кодировки в документе) false - означает, что будет использоваться кодировка шрифта документа ( значение по умолчанию)

public bool UseFontEngineEncoding { get; set; }
