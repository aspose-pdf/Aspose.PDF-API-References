## Метод FontRepository.LoadFonts

Загружает системные установленные шрифты и стандартные шрифты Pdf. Этот метод был разработан для ускорения процесса загрузки шрифтов. По умолчанию шрифты загружаются при первом запросе на любой шрифт. Использование этого метода загружает системные и стандартные шрифты Pdf немедленно перед открытием любого документа Pdf.

```csharp
public static void LoadFonts()
```
