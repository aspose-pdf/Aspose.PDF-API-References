## Свойство Element.ActualText

(Необязательный; PDF 1.4) Текст, который является точной заменой для структурного элемента и его дочерних элементов. Этот текст замены (который должен применяться к как можно меньшему объему содержимого) полезен при извлечении содержимого документа в поддержку доступности для пользователей с ограниченными возможностями или для других целей.

```csharp
public virtual string ActualText { get; set; }
```
