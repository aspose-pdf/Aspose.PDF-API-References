Пространство имен **Aspose.Pdf.Structure** предоставляет классы, которые помогают работать с логической структурой документа, представляющей организацию документа на главы и разделы или идентификацию специальных элементов, таких как фигуры, таблицы и сноски.

## Классы

| Класс | Описание |
| --- | --- |
| [Element](./element/) | Класс, представляющий базовый элемент логической структуры. |
| [ElementCollection](./elementcollection/) | Коллекция базовых элементов логической структуры. |
| [FigureElement](./figureelement/) | Класс, представляющий фигуру логической структуры. |
| [RootElement](./rootelement/) | Корневой элемент структуры. |
| [StructElement](./structelement/) | Общий элемент структуры. |
| [TextElement](./textelement/) | Общий текстовый элемент логической структуры документа. |
