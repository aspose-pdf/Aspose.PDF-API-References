## StringCollection::crend method


Возвращает обратный итератор к элементу, следующему за последним элементом перевёрнутого контейнера. Он соответствует элементу, предшествующему первому элементу неперевёрнутого контейнера. Этот элемент служит заполнителем; попытка доступа к нему приводит к неопределённому поведению.

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crend() const noexcept
```


### ReturnValue

Итератор, указывающий на теоретический элемент с const‑квалификатором, предшествующий первому элементу контейнера.
