## StringCollection::crbegin method


Возвращает обратный итератор к первому элементу перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [crend()](../crend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crbegin() const noexcept
```


### ReturnValue

Итератор, указывающий на последний элемент контейнера, объявленный как const.
