## OptimizationOptions::get_CompressAllContentStreams method


Если установлено в **true**

, все несжатые потоки содержимого страниц будут сжаты с помощью фильтра FlateDecode во время выполнения [Document::OptimizeResources()](../../../aspose.pdf/document/optimizeresources/). По умолчанию **false**

для сохранения обратной совместимости.

bool Aspose::Pdf::Optimization::OptimizationOptions::get_CompressAllContentStreams() const
