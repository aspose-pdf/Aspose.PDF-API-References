## ImageCompressionVersion enum


Описывает версии алгоритма сжатия изображений.

```cpp
enum class ImageCompressionVersion
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Standard | 0 | Алгоритм Standard. Значение по умолчанию. |
| Fast | 2 | Улучшенный алгоритм быстрее, чем стандартный, но применим не во всех случаях. |
| Смешанный | 3 | Используйте быстрый алгоритм, когда это возможно, и стандартный для остальных случаев. Может быть медленнее, чем "Fast", но может обеспечить лучшую компрессию. |
