## XfaParserOptions::get_EmulateRequierdGroups method


Если это свойство истинно, то для требуемых Xfa \"исключённых групп\" будут нарисованы дополнительные красные прямоугольники. Это свойство было введено из‑за отсутствия аналогов исключённых групп при конвертации Xfa‑представления форм в стандарт. По умолчанию значение ложно.
