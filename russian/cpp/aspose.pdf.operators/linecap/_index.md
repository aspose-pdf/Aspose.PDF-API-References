## LineCap enum


Стиль окончания линии должен определять форму, которая будет использоваться на концах открытых подпутей (и пунктировок, если есть) при их обводке.

```cpp
enum class LineCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ButtCap | 0 | Окончание Butt. Обводка должна заканчиваться квадратом в конечной точке пути. Не должно быть проекции за конец пути. |
| RoundCap | 1 | Окончание Round. Полукруглая дуга с диаметром, равным ширине линии, будет нарисована вокруг конечной точки и заполнена. |
| SquareCap | 2 | Выступающее квадратное окончание. Обводка должна продолжаться за конечную точку пути на расстояние, равное половине ширины линии, и быть обрезана квадратом. |
