## LineJoin enum


Стиль соединения линии должен определять форму, используемую на углах обводимых путей.

```cpp
enum class LineJoin
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| MiterJoin | 0 | Miter join. Внешние края штрихов двух сегментов должны быть продлены до их встречи под углом, как в рамке картины. Если сегменты встречаются под слишком острым углом, определяемым параметром ограничения фаски (см. 8.4.3.5, "Miter Limit"), вместо этого следует использовать соединение с фаской. |
| RoundJoin | 1 | Round join. Дуга круга с диаметром, равным ширине линии, должна быть нарисована вокруг точки, где встречаются два сегмента, соединяя внешние края штрихов обоих сегментов. Эта фигурка в виде куска пирога должна быть заполнена, образуя скруглённый угол. |
| BevelJoin | 2 | Bevel join. Оба сегмента должны быть завершены заглушками (см. 8.4.3.3, "Line Cap Style") и образовавшийся вырез за пределами концов сегментов должен быть заполнен треугольником. |
