## Font::Save method


Сохраняет шрифт в поток. [Note](../../../aspose.pdf/note/) что шрифт сохраняется во временный формат TTF, предназначенный только для использования в преобразованной копии оригинального документа. Файл шрифта не предназначен для использования вне контекста оригинального документа.
