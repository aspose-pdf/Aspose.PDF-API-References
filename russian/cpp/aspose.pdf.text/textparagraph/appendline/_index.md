## TextParagraph::AppendLine(const System::SharedPtr\<TextFragment\>\&, const System::SharedPtr\<TextState\>\&, float) method

```cpp
void Aspose::Pdf::Text::TextParagraph::AppendLine(const System::SharedPtr<TextFragment> &line, const System::SharedPtr<TextState> &textState, float lineSpacing)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| строка | const System::SharedPtr\<TextFragment\>\& | Текст новой строки. |
| textState | const System::SharedPtr\<TextState\>\& | [Text](../../) состояние новой строки. |
| lineSpacing | float | Дополнительный интервал (по умолчанию 0.0 и соответствует высоте строки по умолчанию). Значение интервала добавляется к интервалу строк по умолчанию для конкретной строки, поэтому вы можете указать 12.0, чтобы получить пустую строку ПОСЛЕ текстовой строки, нарисованной шрифтом 12pt. |
