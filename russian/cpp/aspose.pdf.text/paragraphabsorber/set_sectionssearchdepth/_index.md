## ParagraphAbsorber::set_SectionsSearchDepth method


Устанавливает значение, указывающее, сколько раз будут выполнены последовательные поиски более мелких элементов структуры. Глубина поиска по умолчанию равна 3. Это означает три поиска горизонтально разделённых разделов (заголовки, абзацы и т.д.) и три поиска вертикально разделённых (колонки).

```cpp
void Aspose::Pdf::Text::ParagraphAbsorber::set_SectionsSearchDepth(int32_t value)
```

## Примечания


Увеличение этого значения может привести к небольшому снижению производительности без видимых изменений в результатах поиска. Уменьшение этого значения может привести к неправильному определению абзацев в секциях. Мы не рекомендуем устанавливать значение ниже значения по умолчанию, если вы не хотите получать только «грубые» элементы структуры страницы.
