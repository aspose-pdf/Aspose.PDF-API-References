## SubstitutionFontCategories enum


Представляет категории шрифтов, которые могут быть заменены.

```cpp
enum class SubstitutionFontCategories
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| TheSameNamedEmbeddedFonts | 0 | Шрифты, названные так же, как системные шрифты. Такие шрифты обычно безопасно заменять шрифтами с тем же названием из системы. |
| AllEmbeddedFonts | 1 | Все встроенные шрифты заменяются. |
