## TextFragmentState::set_CoordinateOrigin method


Устанавливает текст [CoordinateOrigin](../../coordinateorigin/). Если [CoordinateOrigin](../../coordinateorigin/) имеет значение Descender, координата Y текста соответствует самой нижней точке шрифта. Если [CoordinateOrigin](../../coordinateorigin/) имеет значение BaseLine, координата Y текста соответствует базовой линии шрифта. Значение по умолчанию — Descender. Если значение Descent шрифта слишком велико, текст может отображаться выше, чем другие шрифты. В этом случае можно выбрать [CoordinateOrigin](../../coordinateorigin/) BaseLine для лучшего отображения текста.
