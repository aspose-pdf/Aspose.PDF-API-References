## LanguageTransformation enum


Режимы трансформации языка.

```cpp
enum class LanguageTransformation
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Выполняется трансформация языка по умолчанию. |
| ExactlyAsISee | 1 | Трансформация языка выполняется так же, как в текстовом редакторе. Обычно это означает, что текст в документе будет выглядеть точно так же, как вы видите его в коде. Но без гарантий. |
| None | 2 | Трансформация языка не выполняется. |
