## TextSearchOptions::get_SearchForTextRelatedGraphics method


Возвращает значение, позволяющее выполнять поиск графических элементов, связанных с текстом (подчеркивание, фон и т.п.) во время поиска текста. true — поиск графики, связанной с текстом, будет выполнен (значение по умолчанию). false — графические элементы, которые могут присутствовать в исходном документе, будут игнорироваться. Установите это при проблемах с производительностью или если нет необходимости обрабатывать подчеркивание, фон или обрезку.
