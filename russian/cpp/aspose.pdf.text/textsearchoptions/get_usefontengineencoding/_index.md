## TextSearchOptions::get_UseFontEngineEncoding method


Возвращает признак, что поиск текста будет выполнен с использованием кодировки шрифтового движка. true — означает, что будет использована кодировка шрифтового движка (попробуйте, если поиск текста не удаётся из‑за несовершенной кодировки в документе). false — означает, что будет использована кодировка шрифтов документа (значение по умолчанию).
