## TextSearchOptions::get_IgnoreShadowText method


Возвращает признак, что фрагменты текста, представляющие тень обычного текста, будут игнорироваться при поиске. true — означает, что теневой текст не будет найден (используйте, если поиск текста возвращает дублированные фрагменты в близких позициях). false — означает, что теневой текст будет найден вместе с обычным (значение по умолчанию).
