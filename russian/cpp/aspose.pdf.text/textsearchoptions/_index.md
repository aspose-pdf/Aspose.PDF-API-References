## TextSearchOptions class


Представляет параметры поиска текста.

```cpp
class TextSearchOptions : public Aspose::Pdf::Text::TextOptions
```

## Методы

| Метод | Описание |
| --- | --- |
| [get_IgnoreResourceFontErrors](./get_ignoreresourcefonterrors/)() const | Возвращает признак, что ошибки, связанные с отсутствием шрифта, будут игнорироваться поглотителем текста (фрагмента). true — означает, что ошибки отсутствия шрифта будут игнорироваться. Сегменты [Text](../), ссылающиеся на некорректные ресурсы, будут пропущены при обработке. false (по умолчанию) — ошибка отсутствия шрифта завершит обработку, выбросив исключение. |
| [get_IgnoreShadowText](./get_ignoreshadowtext/)() const | Возвращает признак, что фрагменты текста, представляющие тень обычного текста, будут игнорироваться при поиске. true — означает, что теневой текст не будет найден (используйте, если поиск текста возвращает дублированные фрагменты в близких позициях). false — означает, что теневой текст будет найден вместе с обычным (значение по умолчанию). |

Это может снизить производительность. false (по умолчанию) — запись ошибок не будет происходить. |

Установите это при проблемах с производительностью или если нет необходимости обрабатывать подчеркивание, фон или обрезку. |
| [get_SearchInAnnotations](./get_searchinannotations/)() const | Возвращает значение, позволяющее выполнять поиск текста в [Annotations](../../aspose.pdf.annotations/). true — текст будет искаться в [Annotations](../../aspose.pdf.annotations/). false — текст в [Annotations](../../aspose.pdf.annotations/) не будет обрабатываться [TextFragmentAbsorber](../textfragmentabsorber/). |
| [get_StoredGraphicElementsMaxCount](./get_storedgraphicelementsmaxcount/)() const | Возвращает значение, ограничивающее поиск графических элементов, связанных с текстом (подчеркивание, фон и т.п.) на странице указанным числом элементов. По умолчанию — 250. Установите меньшее значение в случае проблем с производительностью, попробуйте большее, если некоторые графические элементы не найдены. |
| [get_UseFontEngineEncoding](./get_usefontengineencoding/)() const | Возвращает признак, что поиск текста будет выполнен с использованием кодировки шрифтового движка. true — означает, что будет использована кодировка шрифтового движка (попробуйте, если поиск текста не удаётся из‑за несовершенной кодировки в документе). false — означает, что будет использована кодировка шрифтов документа (значение по умолчанию). |
