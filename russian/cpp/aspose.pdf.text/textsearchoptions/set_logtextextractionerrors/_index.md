## TextSearchOptions::set_LogTextExtractionErrors method


Устанавливает признак, что ошибки извлечения текста (декодирования) будут записываться в поглотитель текста (фрагмента). true — означает, что ошибки извлечения (декодирования) будут записаны. Это может снизить производительность. false (по умолчанию) — запись ошибок не будет происходить.
