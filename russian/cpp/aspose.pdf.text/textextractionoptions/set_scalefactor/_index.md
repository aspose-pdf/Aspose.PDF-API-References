## TextExtractionOptions::set_ScaleFactor method


Устанавливает коэффициент, который будет применён для масштабирования размера шрифта во время извлечения в чистом режиме. Уменьшение значения приводит к большему количеству пробелов в извлечённом тексте. Значение по умолчанию — 1, без масштабирования; установка значения в ноль позволяет алгоритму выбирать масштаб автоматически.
