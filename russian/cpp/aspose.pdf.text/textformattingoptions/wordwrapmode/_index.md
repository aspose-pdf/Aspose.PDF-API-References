## WordWrapMode enum


Определяет стратегии переноса слов.

```cpp
enum class WordWrapMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| NoWrap | 0 | Перенос не выполняется. |
| DiscretionaryHyphenation | 1 | Выполняется дискреционный перенос. Позволяет разбивать слова посередине. |
| ByWords | 2 | Перенос строк оборачивает только полные слова. Если полное слово нельзя перенести, предпринимается попытка использовать дискреционный перенос. |
| Undefined | 3 | [WordWrapMode](./) не установлен. Будут использованы стратегии переноса верхнего уровня структуры (ячейка таблицы, абзац и т.д.). |
