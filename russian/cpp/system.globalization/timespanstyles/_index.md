## TimeSpanStyles enum


Указывает параметры форматирования, которые настраивают разбор строки для методов, преобразующих строковое представление временного интервала в объект Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Временной интервал, представленный строкой inptut, считается отрицательным только если присутствует знак минуса. |
| AssumeNegative | 1 | Временной интервал, представленный строкой inptut, всегда считается отрицательным. |
