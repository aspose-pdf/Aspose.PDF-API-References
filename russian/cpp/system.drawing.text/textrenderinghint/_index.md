## TextRenderingHint enum


Указывает качество отображения текста.

```cpp
enum class TextRenderingHint
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SystemDefault | 0 | Глиф с подсказкой рендеринга по умолчанию. |
| SingleBitPerPixelGridFit | 1 | Битовая карта глифа с подсказкой. |

| AntiAlias | 4 | Битовая карта глифа со сглаживанием без подсказки. |
| ClearTypeGridFit | 5 | Битовая карта глифа CT с подсказкой. |
