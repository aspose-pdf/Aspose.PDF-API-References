## Array::cbegin method


Возвращает итератор к первому элементу контейнера, объявленному как const. Если контейнер пуст, возвращаемый итератор будет равен [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Итератор, указывающий на первый элемент контейнера, объявленный как const.
