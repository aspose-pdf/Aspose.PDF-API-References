## Array::begin() const method


Возвращает итератор на первый элемент const‑квалифицированного контейнера. Если контейнер пуст, возвращённый итератор будет равен [end()](../end/).

```cpp
const_iterator System::Array<T>::begin() const noexcept
```


### ReturnValue

Итератор, указывающий на первый элемент константно-квалифицированного контейнера.
