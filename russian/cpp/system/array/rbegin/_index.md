## Array::rbegin() const method


Возвращает обратный итератор к первому элементу перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращённый итератор равен [rend()](../rend/).

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue

Итератор, указывающий на последний элемент контейнера с const‑квалификатором.
