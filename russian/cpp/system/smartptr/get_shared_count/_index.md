## SmartPtr::get_shared_count method


Получает количество существующих shared‑указателей на объект, на который ссылаются, включая текущий. Проверяется, что текущий указатель находится в режиме shared.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Количество существующих shared‑указателей на объект, на который есть ссылка, если таковые имеются. Если указатель равен null, возвращает 0.
