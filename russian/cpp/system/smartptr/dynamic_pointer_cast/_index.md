## SmartPtr::dynamic_pointer_cast method


Преобразует указатель к другому типу с помощью dynamic_cast над объектом, на который он указывает.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::dynamic_pointer_cast() const
```


| Параметр | Описание |
| --- | --- |
| Y | Целевой тип указательного объекта. |

### ReturnValue

Указатель изменённого типа, который всегда находится в режиме shared.
