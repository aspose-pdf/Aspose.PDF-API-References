## SmartPtr::static_pointer_cast method


Преобразует указатель к другому типу с помощью static_cast на указываемом объекте.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::static_pointer_cast() const
```


| Параметр | Описание |
| --- | --- |
| Y | Целевой тип указательного объекта. |

### ReturnValue

Указатель изменённого типа, который всегда находится в режиме shared.
