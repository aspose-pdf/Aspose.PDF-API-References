## WeakReference class


Представляет слабую ссылку, которая ссылается на объект, но при этом позволяет удалить этот объект.

```cpp
template<typename...>class WeakReference
```


| Параметр | Описание |
| --- | --- |
| Аргументы | Параметры шаблона слабой ссылки, объявляющие тип ссылочного объекта; количество аргументов может быть нулем или одним. |
