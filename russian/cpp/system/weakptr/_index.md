## WeakPtr class


Подкласс [System::SmartPtr](../smartptr/), который переводит себя в слабый режим при конструировании. Обратите внимание, что этот класс не гарантирует, что его экземпляр всегда будет оставаться в слабом режиме, так как [set_Mode()](../smartptr/set_mode/) всё ещё доступен. Этот тип представляет собой указатель для управления удалением другого объекта. Его следует создавать в стеке и передавать в функции либо по значению, либо по константной ссылке.

```cpp
template<class T>class WeakPtr : public System::SmartPtr<T>
```


| Параметр | Описание |
| --- | --- |
| T | Тип указываемого. |
