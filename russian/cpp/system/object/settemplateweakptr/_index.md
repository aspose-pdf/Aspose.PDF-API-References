## Object::SetTemplateWeakPtr method


Устанавливает n‑й аргумент шаблона как слабый указатель (а не shared). Позволяет переключать указатели в контейнерах в режим weak.

```cpp
virtual void System::Object::SetTemplateWeakPtr(uint32_t argument)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| argument | uint32_t | [Index](../../index/) аргумента шаблона, который следует рассматривать как слабый указатель. |
