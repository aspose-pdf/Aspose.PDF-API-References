## DateTime::operator+ method


Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, равное сумме значения текущего объекта и указанного интервала времени.

```cpp
DateTime System::DateTime::operator+(TimeSpan value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | TimeSpan | Временной интервал, который следует добавить к значению, представленному текущим объектом |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, равное сумме значения текущего объекта и **value**.
