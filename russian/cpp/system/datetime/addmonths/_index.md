## DateTime::AddMonths method


Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества месяцев.

```cpp
DateTime System::DateTime::AddMonths(int value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | int | Количество месяцев для добавления; значение может быть положительным или отрицательным. |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** месяцев.
