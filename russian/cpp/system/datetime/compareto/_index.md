## DateTime::CompareTo method


Сравнивает два значения даты и времени, представленные текущим объектом и указанным экземпляром класса [DateTime](../), и возвращает значение, указывающее относительные позиции значений на временной шкале.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | DateTime | Экземпляр класса [DateTime](../) для сравнения с текущим объектом. |

### ReturnValue

Значение, меньшее 0, если текущий объект представляет значение, предшествующее значению, представленному **value**; 0, если значения, представленные обоими объектами, одинаковы; значение, большее 0, если значение, представленное текущим объектом, позже значения, представленного **value**.
