## DateTime::IsDaylightSavingTime method


Определяет, попадает ли значение даты и времени, представленного текущим объектом, в диапазон перехода на летнее время для текущего часового пояса.

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### ReturnValue

True, если значение даты и времени, указанное текущим объектом, попадает в диапазон летнего времени, иначе — false.
