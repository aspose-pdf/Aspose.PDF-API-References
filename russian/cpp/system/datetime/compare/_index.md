## DateTime::Compare method


Сравнивает два значения, представленные указанными экземплярами класса [DateTime](../), и возвращает значение, указывающее относительные позиции значений на временной шкале.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| t1 | DateTime | Первый сравниваемый операнд |
| t2 | DateTime | Второй сравниваемый операнд |

### ReturnValue

Значение, меньшее 0, если **t1** происходит раньше, чем **t2**; 0, если **t1** совпадает с **t2**; значение, большее 0, если **t1** происходит позже, чем **t2**.
