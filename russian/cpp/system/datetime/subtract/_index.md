## DateTime::Subtract(DateTime) const method


Возвращает экземпляр класса [TimeSpan](../../timespan/), представляющий временной интервал между значениями даты и времени, представленными текущим и указанным объектами.

```cpp
TimeSpan System::DateTime::Subtract(DateTime value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | DateTime | Экземпляр класса [DateTime](../), отмечающий один конец интервала, который необходимо вычислить. |

### ReturnValue

Экземпляр класса [TimeSpan](../../timespan/), представляющий временной интервал между значениями даты и времени, представленными текущим объектом и **value**.

```cpp

```


| Параметр | Тип | Описание |
| --- | --- | --- |
| длительность | TimeSpan | Интервал времени для вычитания |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является результатом вычитания **duration** из значения, представленного текущим объектом.
