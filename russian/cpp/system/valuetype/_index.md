## ValueType class


Базовый класс для типов‑значений с наследованием от [Object](../object/), сокращённым по соображениям производительности. Этот тип должен быть размещён в стеке и передаваться в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../smartptr/) для управления объектами этого типа.

```cpp
class ValueType
```
