## System::Set method


Реализует перевод шаблона 'var'.

```cpp
template<class ExpressionT> bool System::Set(ExpressionT &var, const ExpressionT &value)
```


| Параметр | Описание |
| --- | --- |
| ExpressionT | Тип переменной. |

| Параметр | Тип | Описание |
| --- | --- | --- |
| var | ExpressionT\& | Ссылка на переменную, которую нужно инициализировать. |
| value | const ExpressionT\& | Значение, которое будет присвоено переменной. |

### ReturnValue

Всегда истинно (этот шаблон всегда срабатывает).
