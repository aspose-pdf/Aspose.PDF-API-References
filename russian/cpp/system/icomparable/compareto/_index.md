## IComparable::CompareTo method


Сравнивает текущий объект с указанным объектом.

```cpp
virtual int System::IComparable<T>::CompareTo(T other)=0
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| другое | T | Объект для сравнения с текущим объектом |

### ReturnValue

Отрицательное число, если текущий объект предшествует указанному объекту в порядке сортировки; 0, если сравниваемые объекты равны; положительное число, если текущий объект следует за указанным объектом в порядке сортировки
