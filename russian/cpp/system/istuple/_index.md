## System::IsTuple method

```cpp
bool System::IsTuple(const SharedPtr<Object> &object, int32_t length)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| объект | const SharedPtr\<Object\>\& | Объект для проверки. |
| длина | int32_t | ожидаемая длина кортежа |

### ReturnValue

является ли объект кортежем и его длина равна ожидаемой
