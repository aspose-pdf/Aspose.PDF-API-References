## IConvertible::ToUInt64 method


Преобразует значение этого экземпляра в эквивалентный 64-битный uint32_teger с использованием указанной информации о форматировании, зависящей от культуры.

```cpp
virtual uint64_t System::IConvertible::ToUInt64(System::SharedPtr<System::IFormatProvider> provider)=0
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | System::SharedPtr\<System::IFormatProvider\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), предоставляющая культурно-зависимые сведения о форматировании. |

### ReturnValue

64‑битное беззнаковое целое значение, эквивалентное значению данного экземпляра.
