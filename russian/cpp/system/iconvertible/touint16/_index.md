## IConvertible::ToUInt16 method


Преобразует значение этого экземпляра в эквивалентный 16-битный uint32_teger с использованием указанной информации о форматировании, зависящей от культуры.

```cpp
virtual uint16_t System::IConvertible::ToUInt16(System::SharedPtr<System::IFormatProvider> provider)=0
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| provider | System::SharedPtr\<System::IFormatProvider\> | Реализация интерфейса [System::IFormatProvider](../../iformatprovider/), предоставляющая культурно-зависимые сведения о форматировании. |

### ReturnValue

16‑битное беззнаковое целое значение, эквивалентное значению этого экземпляра.
