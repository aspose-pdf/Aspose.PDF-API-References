## Index class


Представляет индекс в коллекции. Индекс может быть от начала или от конца. Этот тип следует выделять в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../smartptr/) для управления объектами этого типа.

```cpp
class Index : public System::Details::BoxableObjectBase
```
