## String::IsNullOrWhiteSpace method


Указывает, является ли указанная строка null, пустой или состоит только из пробельных символов.



| Параметр | Тип | Описание |

| str | const String\& | [String](../) для проверки. |

### ReturnValue

true, если строка является нулевой или пустой, или состоит исключительно из пробельных символов, иначе false.
