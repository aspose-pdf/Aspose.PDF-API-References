## String::end method


Возвращает указатель на конец фактического буфера строки. Никогда ничего не переаллочирует. Не гарантирует, что буфер будет нуль-терминирован.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Указатель на первый элемент после фактического буфера строки, нуль-терминированный или нет.
