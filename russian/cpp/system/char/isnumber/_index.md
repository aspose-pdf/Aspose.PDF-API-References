## Char::IsNumber(char_t) method


Определяет, классифицируется ли указанный символ как число.

```cpp

```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

Истина, если указанный символ является числом, иначе — ложь.
