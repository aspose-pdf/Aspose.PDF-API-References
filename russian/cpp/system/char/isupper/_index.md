## Char::IsUpper(char_t) method


Определяет, является ли указанный символ классифицированным как заглавная буква.

```cpp

```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True если указанный символ является заглавной буквой, иначе - false
