## Char::IsControl(char_t) method


Определяет, классифицируется ли указанный символ как управляющий символ Unicode.

```cpp
static bool System::Char::IsControl(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

Истина, если указанный символ является управляющим символом Unicode, иначе — false
