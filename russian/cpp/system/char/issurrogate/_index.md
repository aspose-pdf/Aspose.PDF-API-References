## Char::IsSurrogate(char_t) method


Определяет, является ли указанный символ кодовой единицей суррогата UTF-16.

```cpp
static bool System::Char::IsSurrogate(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ |

### ReturnValue

Истина, если указанный символ является кодовой единицей суррогата UTF-16, иначе — ложь.
