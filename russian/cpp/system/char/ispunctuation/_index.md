## Char::IsPunctuation(char_t) method


Определяет, классифицируется ли указанный символ как символ пунктуации.

```cpp
static bool System::Char::IsPunctuation(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

true, если указанный символ является знаковым знаком препинания, иначе — false
