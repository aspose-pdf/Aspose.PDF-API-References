## Char::IsDigit(char_t) method


Определяет, классифицируется ли указанный символ как десятичная цифра.

```cpp
static bool System::Char::IsDigit(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True, если указанный символ является десятичной цифрой, иначе - false
