## Char::IsWhiteSpace(char_t) method


Определяет, является ли указанный символ классифицированным как пробельный символ.

```cpp
static bool System::Char::IsWhiteSpace(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

true, если указанный символ является символом пробела, иначе — false
