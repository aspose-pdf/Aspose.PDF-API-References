## SocketType enum


Перечисляет типы сокетов.

```cpp
enum class SocketType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Stream | 1 | Тип, поддерживающий надёжные двунаправленные потоковые байты, основанные на соединении, без дублирования данных и без сохранения границ. |
| Dgram | 2 | Тип, поддерживающий датаграммы, которые являются безсоединительными, ненадёжными сообщениями фиксированной максимальной длины. |
| Raw | 3 | Тип, поддерживающий доступ к базовому транспортному протоколу. |
| Rdm | 4 | Тип, поддерживающий безсоединительные, ориентированные на сообщения, надёжно доставляемые сообщения и сохраняющий границы сообщений в данных. |
| Seqpacket | 5 | Тип, обеспечивающий ориентированную на соединение и надёжную двунаправленную передачу упорядоченных потоков байтов по сети. |
| Неизвестно | n/a | Неизвестный тип. |
