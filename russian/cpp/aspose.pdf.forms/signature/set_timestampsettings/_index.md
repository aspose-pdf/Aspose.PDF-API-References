## Signature::set_TimestampSettings method


Получает/устанавливает настройки метки времени.

```cpp
void Aspose::Pdf::Forms::Signature::set_TimestampSettings(const System::SharedPtr<Aspose::Pdf::TimestampSettings> &value)
```

## Примечания


Если свойство установлено и сертификат передан в конструктор класса [Signature](../), цифровая подпись будет снабжена встроенной временной меткой. Если свойство установлено, но сертификат не передан в конструктор класса [Signature](../), временная метка будет добавлена как отдельная подпись.
