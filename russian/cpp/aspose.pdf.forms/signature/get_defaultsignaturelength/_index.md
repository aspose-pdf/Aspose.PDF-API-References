## Signature::get_DefaultSignatureLength method


Получает значение длины данных подписи по умолчанию в байтах.

```cpp
int32_t Aspose::Pdf::Forms::Signature::get_DefaultSignatureLength() const
```

## Примечания


Это оценка длины подписи в байтах. Используется для подписи через [CustomSignHash](../), если установлен параметр [AvoidEstimatingSignatureLength](../). Значение по умолчанию — 3000.
