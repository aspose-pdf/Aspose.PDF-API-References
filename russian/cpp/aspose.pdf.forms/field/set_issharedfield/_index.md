## Field::set_IsSharedField method


Свойство для поддержки Generator. Используется, когда поле добавляется в заголовок или нижний колонтитул. Если true, это поле будет создано один раз, и его внешний вид будет виден на всех страницах документа. Если false, отдельное поле будет создано для каждой страницы документа.
