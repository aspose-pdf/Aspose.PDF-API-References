## CancellationTokenRegistration::Dispose method


Удаляет регистрацию и удаляет обратный вызов из связанного [CancellationTokenSource](../../cancellationtokensource/). После вызова этого метода зарегистрированный обратный вызов больше не будет вызываться, когда связанный [CancellationTokenSource](../../cancellationtokensource/) будет отменён.

```cpp
void System::Threading::CancellationTokenRegistration::Dispose()
```

## Примечания



Безопасно вызывать этот метод несколько раз — последующие вызовы не окажут никакого эффекта.
