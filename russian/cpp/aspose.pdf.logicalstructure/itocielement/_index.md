## ITociElement class


Представляет элемент, который может функционировать как запись оглавления (TOCI) в тегированном PDF‑документе.

```cpp
class ITociElement : public virtual System::Object
```

## Методы

| Метод | Описание |
| --- | --- |
| virtual [get_GetElement](./get_getelement/)() | Получает базовый PDF‑элемент, представляющий эту структуру TOCI. |

Этот интерфейс абстрагирует как стандартные элементы TOCI, так и элементы списка (LI), используемые во вложенных структурах оглавления.
