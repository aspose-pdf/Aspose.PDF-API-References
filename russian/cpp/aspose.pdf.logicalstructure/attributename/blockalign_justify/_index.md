## BlockAlign_Justify field


Атрибут BlockAlign: Justify — Дети выравниваются по обоим краям (до и после) прямоугольника содержимого ячейки таблицы. Первый дочерний элемент размещается как описано для Before, а последний — как описано для After, при равных промежутках между дочерними элементами. Если дочерний элемент один, он выравнивается только по переднему краю, как для Before.
