## Примечания


([Table](../../../aspose.pdf/table/) оглавления) Список, состоящий из записей элементов оглавления (тип структуры TOCI) и/или других вложенных записей оглавления (TOC).

Запись TOC, содержащая только записи TOCI, представляет плоскую иерархию. Запись TOC, включающая другие вложенные записи TOC (и возможно записи TOCI), представляет более сложную иерархию. Идеально, если иерархия записи верхнего уровня TOC отражает структуру основной части документа.
