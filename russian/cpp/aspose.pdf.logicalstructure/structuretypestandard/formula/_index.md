## Примечания


(Formula) Математическая формула.

Этот тип структуры полезен только для идентификации всего элемента содержимого как формулы. Ни один стандартный тип структуры не определён для идентификации отдельных компонентов внутри формулы. С точки зрения форматирования формула должна обрабатываться аналогично рисунку (тип структуры Figure).
