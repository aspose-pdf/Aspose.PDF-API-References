## TraceLevel enum


Указывает, какие сообщения выводить для классов [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) и System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Off | 0 | Не выводить сообщения трассировки и отладки. |
| Ошибка | 1 | Выводить сообщения обработки ошибок. |
| Предупреждение | 2 | Выводить предупреждения и сообщения обработки ошибок. |
| Info | 3 | Выводить информационные сообщения, предупреждения и сообщения обработки ошибок. |
| Verbose | 4 | Выводить все сообщения отладки и трассировки. |
