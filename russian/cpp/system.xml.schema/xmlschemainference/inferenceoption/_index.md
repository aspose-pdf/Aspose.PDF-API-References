## InferenceOption enum


Влияет на информацию о повторяемости и типах, выводимую классом [XmlSchemaInference](../) для элементов и атрибутов в XML‑документе.

```cpp
enum class InferenceOption
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Restricted | 0 | Указывает, что для конкретного элемента или атрибута должна быть выведена более строгая декларация схемы. |
| Relaxed | 1 | Указывает, что для конкретного элемента или атрибута должна быть выведена менее строгая декларация схемы. |
