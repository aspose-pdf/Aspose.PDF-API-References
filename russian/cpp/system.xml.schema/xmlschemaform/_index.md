## XmlSchemaForm enum


Указывает, необходимо ли квалифицировать атрибуты или элементы префиксом пространства имён.

```cpp
enum class XmlSchemaForm
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Форма элемента и атрибута не указана в схеме. |
| Квалифицировано | 1 | Элементы и атрибуты должны быть квалифицированы префиксом пространства имён. |
| Неквалифицировано | 2 | Элементы и атрибуты не обязаны быть квалифицированы префиксом пространства имён. |
