## XmlSchemaMinInclusiveFacet class


Представляет элемент **minInclusive** из XML [Schema](../), определённый World Wide [Web](../../system.web/) Consortium (W3C). Этот класс можно использовать для указания ограничения минимального значения элемента simpleType. Значение элемента должно быть больше или равно значению элемента **minInclusive**.

```cpp
class XmlSchemaMinInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```

Объекты этого класса должны выделяться только с помощью функции [System::MakeObject()](../../system/makeobject/).
