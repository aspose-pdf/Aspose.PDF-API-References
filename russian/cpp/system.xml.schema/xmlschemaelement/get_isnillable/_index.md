## XmlSchemaElement::get_IsNillable method


Возвращает информацию, указывающую, может ли **xsi:nil** встречаться в данных экземпляра. Указывает, может ли элементу быть присвоено явное значение nil.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Если nillable равно **true**, это позволяет экземпляру элемента иметь атрибут **nil**, установленный в **true**. Атрибут **nil** определён как часть пространства имён XML [Schema](../../) для экземпляров. Значение по умолчанию — **false**. Необязательно.
