## XmlSchemaComplexType::get_BlockResolved method


Возвращает значение после того, как тип был скомпилирован в набор информации после проверки схемы (infoset). Это значение указывает, как тип применяется, когда в документе экземпляра используется **xsi:type**.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

Значение пост‑валидационного infoset. Значение по умолчанию — **BlockDefault** на элементе **schema**.
