## XmlSchemaComplexType::get_IsAbstract method


Возвращает информацию, определяющую, может ли элемент **complexType** использоваться в документе экземпляра.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Если **true**, элемент не может использовать этот **complexType** напрямую и должен использовать сложный тип, производный от этого **complexType**. По умолчанию **false**. Необязательно.
