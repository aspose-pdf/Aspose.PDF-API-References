## PdfFileSecurity::get_AllowExceptions method


Если это значение установлено в true, при ошибке операции будет выброшено исключение. В противном случае метод возвращает false при неудаче, и последнее исключение можно проверить с помощью свойства LastException.

```cpp
bool Aspose::Pdf::Facades::PdfFileSecurity::get_AllowExceptions()
```


## Deprecated
Это свойство устарело и не может использоваться для разрешения выбрасывания исключений.
