## PdfExtractor::set_ExtractImageMode method

```cpp
void Aspose::Pdf::Facades::PdfExtractor::set_ExtractImageMode(Aspose::Pdf::ExtractImageMode value)
```

## Примечания


Значение по умолчанию — [ExtractImageMode.DefinedInResources](../../../aspose.pdf/extractimagemode/), которое извлекает все изображения, определённые в ресурсах.

Для извлечения фактически отображаемых изображений следует использовать режим [ExtractImageMode.ActuallyUsed](../../../aspose.pdf/extractimagemode/).
