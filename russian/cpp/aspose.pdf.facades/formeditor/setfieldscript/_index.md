## FormEditor::SetFieldScript method


Устанавливает JavaScript для поля PushButton. Если ранее существовал JavaScript, он будет заменён новым.

```cpp
bool Aspose::Pdf::Facades::FormEditor::SetFieldScript(const System::String &fieldName, const System::String &script)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| fieldName | const System::String\& | Полностью квалифицированное имя поля. |
| скрипт | const System::String\& | Java-скрипт, который будет добавлен/размещён в поле push button. |

### ReturnValue

true если скрипт поля был успешно установлен.
