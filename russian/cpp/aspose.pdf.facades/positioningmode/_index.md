## PositioningMode enum


Определяет режим позиционирования. Возможные значения включают Legacy (обратная совместимость) и Current (обновлённый метод расчёта позиции текста).

```cpp
enum class PositioningMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Legacy | 0 | Устаревшее позиционирование текста. |
| ModernLineSpacing | 1 | Обновлённый межстрочный интервал, вычисление вертикального положения выполняется по старым правилам (т. е. текст позиционируется относительно нижнего левого угла указанного прямоугольника) |
| Текущий | 2 | Обновлённый межстрочный интервал и вычисление вертикального положения выполняются на основе левого верхнего угла, а не левого нижнего. |
