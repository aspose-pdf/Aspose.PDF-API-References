## DocumentPrivilege::set_ChangeAllowLevel method


Получает и задает уровень изменения привилегий документа. Аналогично настройкам Changes Allowed в Adobe Professional. 0: Нет. 1: Вставка, удаление и вращение страниц. 2: Заполнение полей формы и подпись существующих полей подписи. 3: Комментирование, заполнение полей формы и подпись существующих полей подписи. 4: Всё, кроме извлечения страниц.

```cpp
void Aspose::Pdf::Facades::DocumentPrivilege::set_ChangeAllowLevel(int32_t value)
```

## Примечания


Если у свойства значение -1, то уровень не определён.
