## PdfFileEditor::get_AllowConcatenateExceptions method


Если установлено значение true, исключения выбрасываются при возникновении ошибки. Иначе исключения не выбрасываются, и методы возвращают false при неудаче.

```cpp
bool Aspose::Pdf::Facades::PdfFileEditor::get_AllowConcatenateExceptions()
```


## Deprecated
