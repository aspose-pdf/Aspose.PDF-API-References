## PdfFileEditor::get_CorruptedItems method


Массив обнаруженных проблем, возникших при выполнении конкатенации. Для каждого повреждённого документа, переданного в функцию [Concatenate()](../concatenate/) создаётся новая запись [CorruptedItem](../corrupteditem/). Это свойство может использоваться только когда CorruptedFileAction имеет значение ConcatenateIgnoringCorrupted.
