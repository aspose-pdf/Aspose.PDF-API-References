## SizeF class


Представляет пару значений с одинарной точностью с плавающей запятой, которые задают ширину и высоту изображения. Этот тип следует выделять в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class SizeF
```
