## SystemFonts class


Предоставляет набор заранее созданных объектов [Font](../font/), представляющих шрифты, используемые для отображения текста в элементах интерфейса [Windows](../../system.windows/). Это статический тип без сервисов экземпляров. Вы никогда не должны создавать его экземпляры каким-либо способом.

```cpp
class SystemFonts : public System::Object
```
