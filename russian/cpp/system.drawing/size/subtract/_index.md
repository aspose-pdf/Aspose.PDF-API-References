## Size::Subtract method


Возвращает новый объект [Size](../), который является результатом вычитания **size2** из **size1**, т.е. значение его ширины — результат вычитания значения ширины **size2's** из значения ширины **size1's**, а значение высоты — результат вычитания значения высоты **size2's** из значения высоты **size1's**.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| size1 | const Size\& | Объект [Size](../), из которого вычитать |
| size2 | const Size\& | Объект [Size](../) для вычитания |

### ReturnValue

Объект [Size](../), который является результатом вычитания **size2** из **size1**
