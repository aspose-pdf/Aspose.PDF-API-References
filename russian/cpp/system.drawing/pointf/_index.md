## PointF class


Представляет пару координат X и Y с одинарной точностью в виде чисел с плавающей точкой для точки на двумерной плоскости. Этот тип следует размещать в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class PointF
```
