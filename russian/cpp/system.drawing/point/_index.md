## Point class


Представляет пару целочисленных координат X и Y точки на двумерной плоскости. Этот тип следует размещать в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class Point
```
