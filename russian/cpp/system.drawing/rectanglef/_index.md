## RectangleF class


Представляет прямоугольную область изображения, определяемую координатами X и Y верхнего левого угла с одинарной точностью и его шириной и высотой. Этот тип должен выделяться в стеке и передаваться в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class RectangleF
```
