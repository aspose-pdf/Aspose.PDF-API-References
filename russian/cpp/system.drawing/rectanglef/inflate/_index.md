## RectangleF::Inflate(const SizeF\&) method

```cpp
void System::Drawing::RectangleF::Inflate(const SizeF &size)
```

| size | const SizeF\& | Объект [SizeF](../../sizef/) определяющий величины, на которые следует увеличить ширину и высоту прямоугольника |
