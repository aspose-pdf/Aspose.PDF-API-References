## CharacterRange class


Представляет диапазон позиций символов в строке. Этот тип следует выделять в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class CharacterRange
```
