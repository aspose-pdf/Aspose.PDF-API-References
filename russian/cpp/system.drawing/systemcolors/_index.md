## SystemColors class


Класс, предоставляющий набор заранее созданных объектов [Color](../color/), представляющих цвета элементов отображения [Windows](../../system.windows/). Это статический тип без сервисов экземпляров. Вы никогда не должны создавать его экземпляры каким-либо способом.

```cpp
class SystemColors
```
