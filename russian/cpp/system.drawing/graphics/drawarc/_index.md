## Graphics::DrawArc(const SharedPtr\<Pen\>\&, float, float, float, float, float, float) method


Рисует указанный дугу с использованием указанного пера на поверхности, представленной текущим объектом.



| Параметр | Тип | Описание |
| --- | --- | --- |
| перо | const SharedPtr\<Pen\>\& | Перо, используемое при рисовании дуги |
| x | float | Координата X левого верхнего угла прямоугольника, определяющего эллипс |
| y | float | Координата Y левого верхнего угла прямоугольника, определяющего эллипс |
| width | float | Ширина прямоугольника, определяющего эллипс |
| height | float | Высота прямоугольника, определяющего эллипс |
| startAngle | float | Угол в градусах, измеряемый по часовой стрелке от оси X до начальной точки дуги |
| sweepAngle | float | Угол в градусах, измеряемый по часовой стрелке от **startAngle** до конечной точки дуги |
