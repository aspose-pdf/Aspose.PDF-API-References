## Graphics::FillPie(const SharedPtr\<Brush\>\&, float, float, float, float, float, float) method


Заполняет указанный сектор, используя указанную кисть, на поверхности, представленной текущим объектом.

```cpp

```


| Параметр | Тип | Описание |
| --- | --- | --- |
| brush | const SharedPtr\<Brush\>\& | Кисть, используемая при заполнении сектора |
| x | float | Координата X левого верхнего угла прямоугольника, определяющего эллипс |
| y | float | Координата Y левого верхнего угла прямоугольника, определяющего эллипс |
| width | float | Ширина прямоугольника, определяющего эллипс |
| height | float | Высота прямоугольника, определяющего эллипс |
| startAngle | float | Угол в градусах, измеряемый по часовой стрелке от оси X до начальной точки сектора |
| sweepAngle | float | Угол в градусах, измеряемый по часовой стрелке от **startAngle** до конечной точки сектора |
