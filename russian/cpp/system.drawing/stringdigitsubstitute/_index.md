## StringDigitSubstitute enum


Указывает, как цифры в строке заменяются в соответствии с локалью или языком.

```cpp
enum class StringDigitSubstitute
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| User | 0 | Пользовательская подстановка. |
| None | 1 | Подстановка отключена. |
| National | 2 | Заменять цифры, соответствующие официальному национальному языку локали пользователя. |
| Traditional | 3 | Заменять цифры, соответствующие родному письму или языку пользователя, которые могут отличаться от официального национального языка локали пользователя. |
