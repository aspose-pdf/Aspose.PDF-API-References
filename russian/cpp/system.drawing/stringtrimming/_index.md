## StringTrimming enum


Указывает, как символы должны обрезаться в строке, которая не помещается в форму разметки.

```cpp
enum class StringTrimming
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Без обрезки. |
| Character | 1 | Обрезать до ближайшего символа. |
| Word | 2 | Обрезать до ближайшего слова. |
| EllipsisCharacter | 3 | Обрезать до ближайшего символа и вставить многоточие в конец строки. |
| EllipsisWord | 4 | Обрезать до ближайшего слова и вставить многоточие в конец строки. |
| EllipsisPath | 5 | Центр удаляется из обрезанных строк и заменяется многоточием. Сохраняйте как можно большую часть последнего сегмента строки, разделённого слешем. |
