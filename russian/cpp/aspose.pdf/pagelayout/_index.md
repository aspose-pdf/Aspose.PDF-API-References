## PageLayout enum


Описывает макет страницы.

```cpp
enum class PageLayout
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SinglePage | 0 | Одна страница. |
| OneColumn | 1 | Отображать страницы в одной колонке. |
| TwoColumnLeft | 2 | Отображать страницы в двух колонках, при этом нечётные страницы слева. |
| TwoColumnRight | 3 | Отображать страницы в двух колонках, при этом нечётные страницы находятся справа. |
| TwoPageLeft | 4 | Отображать две страницы одновременно, при этом нечётные страницы слева. |
| TwoPageRight | 5 | Отображать две страницы одновременно, при этом нечётные страницы справа. |
| Default | 6 | Разметка по умолчанию. |
