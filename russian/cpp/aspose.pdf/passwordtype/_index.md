## PasswordType enum


Этот enum представляет известные типы паролей, используемых для защищённых паролем PDF‑документов.

```cpp
enum class PasswordType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | [Pdf](../) документ не защищён паролем. |
| User | 1 | [Pdf](../) документ был открыт с использованием пароля открытия документа (ограниченный доступ). |
| Owner | 2 | [Pdf](../) документ был открыт с использованием пароля изменения прав (полный доступ). |
| Inaccessible | 3 | [Pdf](../) документ защищён паролем, но оба пароля пользователя и владельца не пусты, и ни один из паролей не был определён, либо введённый пароль неверен. Поэтому невозможно определить тип пароля. |
