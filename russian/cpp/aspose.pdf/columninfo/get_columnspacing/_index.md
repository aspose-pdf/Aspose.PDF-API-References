## ColumnInfo::get_ColumnSpacing method


Возвращает строку, содержащую расстояние между колонками. Значения каждого расстояния должны разделяться пробелом. Единицей измерения по умолчанию является пункт, но также поддерживаются см и дюймы. Например, "120 2.5cm 1.5inch".

```cpp
System::String Aspose::Pdf::ColumnInfo::get_ColumnSpacing() const
```

## Примечания


Если это свойство не задано, значение по умолчанию 0 будет использовано для каждого расстояния.
