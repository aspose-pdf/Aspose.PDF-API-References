## ParagraphPositioningMode enum


Указывает вариант определения расположения элемента на странице.

```cpp
enum class ParagraphPositioningMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Расположение определяется ранее размещёнными элементами. При определении расположения последующих элементов учитывается добавление нового элемента. |
| Absolute | 1 | Расположение задаётся значениями [Left](../left/) и Top, не зависит от предыдущих элементов и не влияет на расположение последующих. |
