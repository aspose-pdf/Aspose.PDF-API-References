## CollectionItem::get_IsEmpty method


Получает значение, указывающее, пустой ли элемент коллекции.

```cpp
bool Aspose::Pdf::CollectionItem::get_IsEmpty()
```


### ReturnValue

true, если элемент коллекции пуст; в противном случае — false.
## Примечания


Это свойство возвращает true, если элемент коллекции не содержит никаких значений, включая строковые значения, значения типа double, целочисленные значения и значения даты. Если любой из этих типов значений присутствует в элементе коллекции, свойство возвращает false.
