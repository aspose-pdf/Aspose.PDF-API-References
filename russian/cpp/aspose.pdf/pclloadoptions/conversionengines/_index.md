## ConversionEngines enum


Перечисляет движки конвертации, которые могут быть использованы для преобразования.

```cpp
enum class ConversionEngines
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| LegacyEngine | 0 | Legacy conversion engine — это движок конвертации, который использовался некоторое время и используется по умолчанию. |
| NewEngine | 1 | Новый движок конвертации в настоящее время находится в состоянии A‑testing. Мы открыли его в публичном API, чтобы получить обратную связь. В настоящее время рекомендуется использовать его только тогда, когда старый движок конвертации не может конвертировать документ по некоторым причинам. |
