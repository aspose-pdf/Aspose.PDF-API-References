## Permissions enum


Этот enum представляет разрешения пользователя для PDF.

```cpp
enum class Permissions
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| PrintDocument | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 2) Печать документа. ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Печать документа (возможно не в самом высоком качестве, в зависимости от того, установлен ли также [PrintingQuality](./)). |
| ModifyContent | n/a | Изменять содержимое документа операциями, отличными от тех, которые контролируются [ModifyTextAnnotations](./), [FillForm](./) и 11. |
| ExtractContent | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 2) Копировать или иным способом извлекать текст и графику из документа, включая извлечение текста и графики (в поддержку доступности для пользователей с ограниченными возможностями или для других целей). ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Копировать или иным способом извлекать текст и графику из документа операциями, отличными от контролируемых [ExtractContentWithDisabilities](./). |
| ModifyTextAnnotations | n/a | Добавлять или изменять текстовые аннотации, заполнять интерактивные поля формы и, если также установлен [ModifyContent](./), создавать или изменять интерактивные поля формы (включая поля подписи). |
| FillForm | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Заполнять существующие интерактивные поля формы (включая поля подписи), даже если [ModifyTextAnnotations](./) отключён. |
| ExtractContentWithDisabilities | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Извлекать текст и графику (в поддержку доступности для пользователей с ограниченными возможностями или для других целей). |
| AssembleDocument | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Собирать документ (вставлять, вращать или удалять страницы и создавать закладки или миниатюры), даже если [ModifyContent](./) отключён. |
| PrintingQuality | n/a | ([Security](../../aspose.pdf.security/) обработчики версии 3 или выше) Печатать документ в представление, из которого можно создать точную цифровую копию содержимого PDF. Когда этот бит отключён (и установлен бит 3), печать ограничивается низкоуровневым представлением внешнего вида, возможно с ухудшенным качеством. |
