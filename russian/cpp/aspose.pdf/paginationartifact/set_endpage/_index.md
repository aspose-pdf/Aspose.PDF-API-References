## PaginationArtifact::set_EndPage method


Устанавливает номер конечной страницы артефакта. Значение должно быть больше или равно 0. Если установить значение меньше 0, оно будет скорректировано до 0. Значение по умолчанию 0 означает отсутствие границ конечной страницы.
