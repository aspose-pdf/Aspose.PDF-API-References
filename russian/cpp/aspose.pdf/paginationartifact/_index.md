## PaginationArtifact class


Представляет абстрактный базовый класс для артефактов нумерации в документе.

```cpp
class PaginationArtifact : public Aspose::Pdf::Artifact
```

## Методы

| Метод | Описание |
| --- | --- |
| [get_EndPage](./get_endpage/)() const | Получает номер конечной страницы артефакта. Значение должно быть больше или равно 0. Если установить значение меньше 0, оно будет скорректировано до 0. Значение по умолчанию 0 означает отсутствие границ конечной страницы. |
| [get_StartPage](./get_startpage/)() const | Получает номер начальной страницы артефакта. Значение должно быть больше или равно 1. Если установить значение меньше 1, оно будет скорректировано до 1. |

| [set_EndPage](./set_endpage/)(int32_t) | Устанавливает номер конечной страницы артефакта. Значение должно быть больше или равно 0. Если установить значение меньше 0, оно будет скорректировано до 0. Значение по умолчанию 0 означает отсутствие границ конечной страницы. |
| [set_StartPage](./set_startpage/)(int32_t) | Устанавливает номер начальной страницы артефакта. Значение должно быть больше или равно 1. Если установить значение меньше 1, оно будет скорректировано до 1. |
| [set_Subset](./set_subset/)(Aspose::Pdf::Subset) | Устанавливает подмножество страниц, к которым применяется артефакт (например, все страницы, четные страницы, нечетные страницы). |
