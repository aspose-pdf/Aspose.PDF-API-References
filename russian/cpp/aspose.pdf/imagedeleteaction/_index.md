## ImageDeleteAction enum


Действие, которое выполняется с объектом изображения, когда изображение удаляется из коллекции. Если объект изображения удалён.

```cpp
enum class ImageDeleteAction
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| KeepContents | 0 | [Image](../image/) будет удалён из коллекции. Если содержимое страницы содержит ссылки на изображение, они не будут удалены. [Document](../document/) может стать недействительным. |
| None | 1 | [Image](../image/) будет удалён из коллекции и из содержимого страницы, но объект изображения не будет удалён. Размер файла не уменьшится. |
| ForceDelete | 2 | [Image](../image/) будет удалён из коллекции, и объект изображения будет удалён из документа. Если существуют другие ссылки на тот же объект, документ может быть повреждён. |
| Check | 3 | [Image](../image/) будет удалён из коллекции, и объект изображения будет удалён только если нет других ссылок на изображение с других страниц. Это может потребовать больше времени по сравнению с опцией ForceDelete. |
