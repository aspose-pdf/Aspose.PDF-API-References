## EmphasisStyle enum


Определяет доступные стили сериализации для выделения и сильного выделения. Для спецификации см. CommonMark — Выделение и сильное выделение.

```cpp
enum class EmphasisStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Звёздочка | 0 | Используйте '*' соответственно '**' для (сильно) выделенного текста. |
| Подчеркивание | 1 | Используйте '_' соответственно '__' для (сильно) выделенного текста. |
