## XImage::get_ImageMask method


Получает флаг, указывающий, следует ли рассматривать изображение как маску изображения (см. 8.9.6, \"Masked Images\"). Если этот флаг истинный, значение BitsPerComponent должно быть 1, а Mask и [ColorSpace](../../colorspace/) не должны быть указаны; немаскированные области должны быть закрашены текущим цветом без обводки. Значение по умолчанию: false.

```cpp
bool Aspose::Pdf::XImage::get_ImageMask()
```

## Примечания


True, если изображение является маской изображения.
