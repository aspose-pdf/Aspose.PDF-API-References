## AutoTaggingSettings::get_HeadingRecognitionStrategy method


Получает стратегию, используемую для распознавания заголовков в документе во время автоматической разметки.

```cpp
Aspose::Pdf::HeadingRecognitionStrategy Aspose::Pdf::AutoTaggingSettings::get_HeadingRecognitionStrategy() const
```

## Примечания


Свойство [HeadingRecognitionStrategy](../../headingrecognitionstrategy/) определяет, как заголовки идентифицируются в документе. Доступные стратегии включают распознавание заголовков на основе оглавления, эвристического анализа или автоматического обнаружения. Установка этого свойства в значение [HeadingRecognitionStrategy::None](../../headingrecognitionstrategy/) отключает распознавание заголовков.
