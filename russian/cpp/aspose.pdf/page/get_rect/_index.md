## Page::get_Rect method


Получает прямоугольник страницы. При получении: возвращается crop box страницы, если он указан, иначе возвращается media box страницы. При установке: всегда устанавливается media box страницы. Обратите внимание, что это свойство не учитывает вращение страницы. Чтобы получить прямоугольник страницы с учётом вращения, используйте ActualRect.

```cpp
System::SharedPtr<Aspose::Pdf::Rectangle> Aspose::Pdf::Page::get_Rect()
```
