## Page::set_IsAddParagraphsAfterLast method


Устанавливает добавление абзацев после последнего абзаца страницы.

```cpp
void Aspose::Pdf::Page::set_IsAddParagraphsAfterLast(bool value)
```

## Примечания


Значение указывает, будут ли абзацы добавлены после последнего абзаца страницы. [Paragraphs](../../paragraphs/) будут добавлены после последнего абзаца страницы, если значение истинно.
