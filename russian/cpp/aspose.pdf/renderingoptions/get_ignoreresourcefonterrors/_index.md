## RenderingOptions::get_IgnoreResourceFontErrors method


Возвращает указание, что ошибки, связанные с отсутствием шрифта, будут игнорироваться. true — означает, что ошибки отсутствия шрифта будут игнорироваться. Сегменты [Text](../../../aspose.pdf.text/) которые ссылаются на некорректные ресурсы, будут пропускаться во время обработки. По умолчанию false.

```cpp
bool Aspose::Pdf::RenderingOptions::get_IgnoreResourceFontErrors() const
```
