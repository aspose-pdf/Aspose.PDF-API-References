## RenderingOptions::set_IgnoreResourceFontErrors method


Устанавливает указание, что ошибки, связанные с отсутствием шрифта, будут игнорироваться. true — означает, что ошибки отсутствия шрифта будут игнорироваться. Сегменты [Text](../../../aspose.pdf.text/) которые ссылаются на некорректные ресурсы, будут пропускаться во время обработки. По умолчанию false.

void Aspose::Pdf::RenderingOptions::set_IgnoreResourceFontErrors(bool value)
