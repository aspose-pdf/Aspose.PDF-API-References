## Document::set_DisableFontLicenseVerifications method


Многие операции с шрифтом не могут быть выполнены, если эти операции запрещены лицензией данного шрифта. Например, некоторые шрифты нельзя встроить в PDF‑документ, если правила лицензии запрещают встраивание этого шрифта. Этот флаг используется для отключения любых лицензионных ограничений для всех шрифтов в текущем PDF‑документе. Будьте осторожны при использовании этого флага. Когда он установлен, это означает, что человек, устанавливающий флаг, берёт на себя всю ответственность за возможные нарушения лицензий/законов. Таким образом, он действует на свой собственный риск. Настоятельно рекомендуется использовать этот флаг только тогда, когда вы полностью уверены, что не нарушаете авторское право. По умолчанию false.
