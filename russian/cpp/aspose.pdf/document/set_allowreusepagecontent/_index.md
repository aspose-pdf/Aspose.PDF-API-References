## Document::set_AllowReusePageContent method


Позволяет объединять содержимое страниц для оптимизации размера документа. При использовании разные, но дублированные страницы могут ссылаться на один и тот же объект контента. Обратите внимание, что этот режим может вызвать побочные эффекты, такие как изменение содержимого одной страницы при изменении другой.
