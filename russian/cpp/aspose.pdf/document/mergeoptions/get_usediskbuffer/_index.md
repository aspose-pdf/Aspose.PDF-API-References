## MergeOptions::get_UseDiskBuffer method


Если эта опция используется, то целевой документ будет периодически сохраняться на диск, а дальнейшая конкатенация будет применяться к нему как инкрементные обновления. Значение по умолчанию — **false**.

```cpp
bool Aspose::Pdf::Document::MergeOptions::get_UseDiskBuffer() const
```

## Примечания


**This option is ignored when calling a non-static merge method.**
