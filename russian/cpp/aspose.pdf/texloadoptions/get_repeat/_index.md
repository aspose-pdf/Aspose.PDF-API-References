## TeXLoadOptions::get_Repeat method


Получает/устанавливает флаг, указывающий, необходимо ли запускать задание TeX дважды в случае, например, когда во входных TeX‑файлах есть ссылки. Как правило, такое поведение полезно, когда движок собирает некоторые данные в процессе наборa и сохраняет их во вспомогательный файл при первом запуске. А при втором запуске движок каким‑то образом использует эти данные.
