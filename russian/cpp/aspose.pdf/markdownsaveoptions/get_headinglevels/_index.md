## MarkdownSaveOptions::get_HeadingLevels method


Определяет ожидаемые уровни заголовков, используемые в стратегии распознавания заголовков по размеру шрифта. Если значение этого свойства установлено, то стратегия распознавания заголовков [HeadingRecognitionStrategy::Heuristic](../../headingrecognitionstrategy/) будет выбрана при установке стратегий [PdfToMarkdown::HeadingRecognitionStrategy::Auto](../), даже если документ содержит закладки.
