## HtmlImageType enum


перечисляет возможные типы файлов изображений, которые могут быть сохранены как внешние ресурсы во время преобразования [Pdf](../../) в Html.

```cpp
enum class HtmlImageType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Jpeg | 0 | Формат JPEG. |
| Png | 1 | Формат PNG. |
| Bmp | 2 | Формат BMP. |
| Gif | 3 | Формат GIF. |
| Tiff | 4 | Формат TIFF. |
| Svg | 5 | Формат SVG. |
| ZippedSvg | 6 | Сжатый формат Svg. |
| Неизвестно | 7 | Неизвестно — означает, что конвертер не может самостоятельно определить тип содержимого. |
