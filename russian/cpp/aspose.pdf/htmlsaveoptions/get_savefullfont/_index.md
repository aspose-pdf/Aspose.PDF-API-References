## HtmlSaveOptions::get_SaveFullFont method


Указывает, что будет сохраняться полный шрифт, поддерживает только True Type шрифты. По умолчанию SaveFullFont = false, и конвертер сохраняет подмножество исходного шрифта, необходимое для отображения текста документа.
