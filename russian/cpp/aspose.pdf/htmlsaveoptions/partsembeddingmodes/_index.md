## PartsEmbeddingModes enum


Этот enum перечисляет возможные режимы встраивания файлов, указанных в HTML It позволяет контролировать, будут ли ссылки на файлы (HTML, Fonts,Images, CSSes) встроены в основной HTML‑файл или созданы как отдельные бинарные сущности.

```cpp
enum class PartsEmbeddingModes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| EmbedAllIntoHtml | 0 | Принудительно внедряет все ссылки на файлы (CSS, изображения, шрифты) в генерируемую разметку HTML (т.е. в сам HTML). Этот подход создаёт один HTML‑файл, но общий размер вывода становится больше (из‑за использования Base64‑кодирования бинарных данных), и не все браузеры (особенно устаревшие) успешно обрабатывают встроенные в HTML бинарные данные. Однако позволяет получить HTML, содержащий весь результат без дополнительных файлов. |
| EmbedCssOnly | 1 | Обеспечивает отделение всех ссылочных файлов, кроме CSS (изображений и шрифтов). То есть CSS будет встроен в результирующий HTML, а все остальные ссылочные файлы (изображения и шрифты) будут обработаны как внешние части. Генерирует HTML, подходящий для широкого набора браузеров. |
| NoEmbedding | 2 | Обеспечивает отделение ссылочных файлов (Css,Images,Fonts). Этот подход генерирует набор файлов, но общий размер вывода становится меньше (поскольку не используется кодирование Base64 бинарных данных). Также такой подход генерирует HTML, подходящий для широкого набора браузеров. |
