## HtmlSaveOptions::get_SplitCssIntoPages method


Когда выбран режим мультистраничности (т.е. 'SplitIntoPages' = 'true'), этот атрибут определяет, следует ли создавать отдельный CSS‑файл для каждой результирующей HTML‑страницы. По умолчанию атрибут имеет значение false, поэтому создаётся один большой общий CSS для всех созданных страниц. Суммарный размер всех CSS‑файлов, генерируемых в этом режиме (по одному CSS на страницу), обычно значительно превышает размер одного большого CSS‑файла, потому что в первом случае классы CSS дублируются в нескольких файлах для каждой страницы. Поэтому данная настройка целесообразна только тогда, когда вам требуется последующая обработка каждой HTML‑страницы независимо, и, следовательно, размер CSS каждой отдельной страницы является критическим фактором.
