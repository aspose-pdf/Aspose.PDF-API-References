## Rectangle::ContainsLine method


Определяет, содержит ли прямоугольник линию, представленную двумя точками.

```cpp
bool Aspose::Pdf::Rectangle::ContainsLine(double x1, double y1, double x2, double y2)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| x1 | double | Координата X начальной точки линии. |
| y1 | double | Координата Y начальной точки линии. |
| x2 | double | Координата X конечной точки линии. |
| y2 | double | Координата Y конечной точки линии. |

### ReturnValue

**true** if the rectangle contains the line; otherwise, **false**.
