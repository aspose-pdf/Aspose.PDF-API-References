## BaseParagraph::set_ZIndex method


Устанавливает целочисленное значение, указывающее порядок Z графика. График с большим ZIndex будет размещён поверх графика с меньшим ZIndex. ZIndex может быть отрицательным. График с отрицательным ZIndex будет размещён за текстом на странице.
