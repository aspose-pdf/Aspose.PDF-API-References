## BaseParagraph::get_ZIndex method


Возвращает целочисленное значение, указывающее порядок Z графика. График с большим ZIndex будет размещён поверх графика с меньшим ZIndex. ZIndex может быть отрицательным. График с отрицательным ZIndex будет находиться за текстом на странице.

int32_t Aspose::Pdf::BaseParagraph::get_ZIndex() const
