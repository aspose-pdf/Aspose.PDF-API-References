## FontEmbeddingOptions class


Стандарт PDF/A требует, чтобы все шрифты были встроены в документ. Этот класс включает флаги для случаев, когда невозможно встроить некоторый шрифт, потому что этот шрифт отсутствует на целевом ПК.

```cpp
class FontEmbeddingOptions : public System::Object
```
