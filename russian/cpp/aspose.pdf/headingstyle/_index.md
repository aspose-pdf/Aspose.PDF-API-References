## HeadingStyle enum


Определяет доступные стили сериализации заголовков. Для спецификации см. CommonMark — заголовки ATX, соответственно CommonMark — заголовки Setext.

```cpp
enum class HeadingStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Atx | 0 | Добавляет префикс к заголовкам с помощью '#'. |
| Setext | 1 | Подчеркивает заголовки уровней 1 и 2 символами "======" и соответственно "------". |
