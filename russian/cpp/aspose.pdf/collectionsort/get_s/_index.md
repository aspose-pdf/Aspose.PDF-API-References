## CollectionSort::get_S method


Получает имя или имена полей, которые интерактивный процессор PDF будет использовать для сортировки элементов в коллекции. Если значение является именем, оно идентифицирует поле, описанное в словаре родительской коллекции. Если значение является массивом, каждый элемент массива должен быть именем, которое идентифицирует поле, описанное в словаре родительской коллекции. Формат массива используется, чтобы позволить дополнительным полям участвовать в сортировке, где каждое дополнительное поле используется для разбивки связей. Более конкретно, если несколько словарей элементов коллекции имеют одинаковое значение для первого поля, указанного в массиве, значения последующих полей, указанных в массиве, будут использоваться для сортировки, пока не будет определён уникальный порядок или пока не исчерпаются указанные поля.

const System::ArrayPtr<System::String> & Aspose::Pdf::CollectionSort::get_S() const
