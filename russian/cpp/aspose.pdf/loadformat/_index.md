## LoadFormat enum


Указывает формат загрузки.

```cpp
enum class LoadFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| CGM | 0 | означает загрузку документа в формате CGM |
| HTML | 1 | означает загрузку документа в формате HTML |
| EPUB | 2 | означает загрузку документа в формате EPUB (специальный формат электронных книг) |
| XML | 3 | означает загрузку документа в формате XML (специальный XML, представляющий логическую структуру PDF‑документа) |
| XSLFO | 4 | означает загрузку документа в формате XSLFO |
| PCL | 5 | означает загрузку документа в формате PCL |
| XPS | 6 | означает загрузку документа в формате XPS |
| TEX | 7 | означает загрузку документа в формате TEX — формат редактора LaTeX |
| SVG | 8 | означает загрузку документа в формате SVG — формат текстового редактора LaTeX |
| MHT | 9 | означает загрузку документа в формате MHT (это упакованный формат HTML) |
| PS | 10 | означает загрузку документа в формате PS (формат документа PostScript) |
| MD | 11 | означает, что документ загружен в формате MD (markdown). |
| TXT | 12 | означает, что документ загружен в формате TXT. |
| APS | 13 | означает загрузку документа в формате APS. |
| PDFXML | 14 | Внутренняя структура PDF‑документа в формате XML. |
| OFD | 15 | означает загрузку документа в формате [OFD](../../aspose.pdf.ofd/). |
| DJVU | 16 | означает загрузку документа в формате Djvu. |
| CDR | 17 | означает загрузку документа в формате CDR. |
| RTF | 18 | означает загрузку документа в формате RTF. |
