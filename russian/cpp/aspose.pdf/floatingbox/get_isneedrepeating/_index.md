## FloatingBox::get_IsNeedRepeating method


Получает значение типа bool, указывающее, нужно ли повторять абзац на следующей странице. Значение по умолчанию — false. Атрибут действителен только тогда, когда сам абзац и объект, на который ссылается его ReferenceParagraphID, оба включены в RepeatingRows.

bool Aspose::Pdf::FloatingBox::get_IsNeedRepeating() const
