## DocSaveOptions::get_ConvertType3Fonts method


Получает конвертацию для шрифтов Type3. В шрифтах Type 3 глифы определяются потоками графических операторов. Это означает, что в выводе DOC/DOCX мы видим изображения вместо текста. Установите этот флаг в true, чтобы конвертировать шрифты Type3 в TTF и получить текст в результирующем файле.

bool Aspose::Pdf::DocSaveOptions::get_ConvertType3Fonts() const
