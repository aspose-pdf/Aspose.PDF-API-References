## DocSaveOptions::get_ReSaveFonts method


Получает процедуру повторного сохранения шрифтов. Если установить в true, шрифты будут перезагружаться на каждой странице, чтобы избежать влияния предыдущих свойств шрифта и загрузить вновь созданный шрифт с нуля. Установите эту опцию в false, если хотите повысить производительность. Значение по умолчанию — true.

bool Aspose::Pdf::DocSaveOptions::get_ReSaveFonts() const
