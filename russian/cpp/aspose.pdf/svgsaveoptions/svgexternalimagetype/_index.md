## SvgExternalImageType enum


перечисляет возможные типы файлов изображений, которые могут быть сохранены как внешние ресурсы во время преобразования [Pdf](../../) в SVG

```cpp
enum class SvgExternalImageType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Jpeg | 0 | Формат JPEG. |
| Png | 1 | Формат PNG. |
| Bmp | 2 | Формат BMP. |
| Gif | 3 | Формат GIF. |
| Tiff | 4 | Формат TIFF. |
| Неизвестно | 5 | Неизвестно — означает, что конвертер не может самостоятельно определить тип содержимого. |
