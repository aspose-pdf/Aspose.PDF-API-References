## BatesNArtifact::get_NumberOfDigits method


Получает количество цифр для нумерации Бейтса. Значение должно быть в диапазоне от 3 до 15 включительно. Если установить значение меньше 3, оно будет скорректировано до 3. Если установить значение больше 15, оно будет скорректировано до 15. Значение по умолчанию — 6.

int32_t Aspose::Pdf::BatesNArtifact::get_NumberOfDigits() const
