## HorizontalAlignment enum


Описывает горизонтальное выравнивание.

```cpp
enum class HorizontalAlignment
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Без выравнивания. |
| Left | 1 | Выравнивание по левому краю. |
| Center | 2 | Выравнивание [Center](../center/). |
| Right | 3 | Выравнивание по правому краю. |
| Justify | 4 | Выравнивание по ширине. [Text](../../aspose.pdf.text/) будет выравнено по обоим краям — левому и правому. |
| FullJustify | 5 | Похоже на выравнивание 'Justify', за исключением того, что последняя строка будет выровнена только по левому краю в режиме 'Justify', тогда как в режиме 'FullJustify' все строки будут выровнены и по левому, и по правому краю. |
