## BorderSide enum


Перечисляет стороны границы.

```cpp
enum class BorderSide
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Нет границы. |
| Left | 1 | [Left](../left/) граница. |
| Верх | 2 | Верхняя граница. |
| Right | 4 | [Right](../right/) граница. |
| Низ | 8 | Нижняя граница. |
| Все | 15 | Границы со всех сторон. Имеет то же значение, что и "Box", и будет заменён на "Box" через год. Поэтому, пожалуйста, используйте "Box", если возможно. |
| Box | 15 | Границы четырёх сторон. |
