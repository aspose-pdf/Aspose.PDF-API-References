## DigestHashAlgorithm enum


Представляет тип алгоритма, отображающего данные в "hash".

```cpp
enum class DigestHashAlgorithm
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Auto | 0 | Автоматическая настройка алгоритма хеширования по усмотрению алгоритма подписи. Для EDCSA значение по умолчанию определяется размером ключа. Значение по умолчанию для неотсоединённого PKCS7 — Sha1. |
| Sha1 | 1 | SHA-1. Secure Hash Algorithm 1. Это значение по умолчанию для неотсоединённого PKCS7. |
| Sha256 | 2 | SHA-256. Secure Hash Algorithm 2. Это значение по умолчанию для отсоединённого PKCS7. |
| Sha384 | 3 | SHA-384. Secure Hash Algorithm 2. |
| Sha512 | 4 | SHA-512. Secure Hash Algorithm 2. |
| Sha3_256 | 5 | SHA3-256. Secure Hash Algorithm 3. |
| Sha3_384 | 6 | SHA3-384. Secure Hash Algorithm 3. |
| Sha3_512 | 7 | SHA3-512. Алгоритм безопасного хеширования 3. |
