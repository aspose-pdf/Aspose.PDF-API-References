## WarningType enum


Enum представлял тип предупреждения.

```cpp
enum class WarningType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SourceFileCorruption | 0 | Файл повреждён. |
| DataLoss | 1 | Текст/диаграмма/изображение или другие данные полностью отсутствуют либо в дереве документа после загрузки, либо в созданном документе после сохранения. |
| MajorFormattingLoss | 2 | Существенные потери форматирования по сравнению с оригинальным документом. Это относится к случаям, когда потеря форматирования значительна, но данные всё ещё присутствуют. |
| MinorFormattingLoss | 3 | Незначительные потери форматирования по сравнению с оригинальным документом. Это относится к небольшим потерям точности. |
| CompatibilityIssue | 4 | Известная проблема, которая может помешать открытию документа некоторыми пользовательскими агентами или их предыдущими версиями. |
| InvalidInputStreamType | 5 | Недопустимый тип входного потока. |
| UnexpectedContent | 99 | Файл содержит неожиданное содержимое. |
