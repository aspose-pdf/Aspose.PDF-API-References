## PageMode enum


Класс описывает используемые компоненты страницы документа.

```cpp
enum class PageMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| UseNone | 0 | Не использовать никаких компонентов. |
| UseOutlines | 1 | [Document](../document/) контур виден. |
| UseThumbs | 2 | Миниатюры видимы. |
| FullScreen | 3 | FullScreenFull-screen режим, без строки меню, элементов управления окном или любого другого видимого окна. |
| UseOC | 4 | Панель группы необязательного содержимого видима. |
| UseAttachments | 5 | Панель вложений видима. |
