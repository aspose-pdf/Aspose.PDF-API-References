## ExtractImageMode enum


Определяет различные режимы, которые могут использоваться при извлечении изображений из документов.

```cpp
enum class ExtractImageMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| DefinedInResources | 0 | Определяет режим извлечения изображений, при котором извлекаются все изображения, определённые в ресурсах для конкретной страницы. |
| ActuallyUsed | 1 | Определяет режим извлечения изображений, при котором извлекаются только те изображения, которые действительно отображаются на странице. |
