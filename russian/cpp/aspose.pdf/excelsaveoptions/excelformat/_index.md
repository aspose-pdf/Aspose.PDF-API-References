## ExcelFormat enum


Позволяет указать формат файлов .xlsx, .xls/xml или csv. Значение по умолчанию — XLSX;.

```cpp
enum class ExcelFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| XMLSpreadSheet2003 | 0 | Формат Excel 2003 XML. |
| XLSX | 1 | Формат файла Office Open XML (.xlsx). |
| CSV | 2 | Формат файла со значениями, разделёнными запятыми (CSV). |
| XLSM | 3 | Формат файла Office Open XML (.xlsm) с поддержкой макросов. |
| ODS | 4 | Электронная таблица OpenDocument. |
