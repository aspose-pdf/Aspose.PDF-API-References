## HtmlLoadOptions::get_CreateLogicalStructure method


Получает значение, указывающее, следует ли создавать логическую структуру в результирующем PDF‑документе.

```cpp
bool Aspose::Pdf::HtmlLoadOptions::get_CreateLogicalStructure() const
```

## Примечания


Когда установлено в **true**

, логическая структура документа создаётся, что может улучшить доступность и сделать документ более подходящим для программ чтения с экрана и других вспомогательных технологий.
