## HtmlLoadOptions::set_CreateLogicalStructure method


Устанавливает значение, указывающее, следует ли создавать логическую структуру в результирующем PDF‑документе.

```cpp
void Aspose::Pdf::HtmlLoadOptions::set_CreateLogicalStructure(bool value)
```

## Примечания


Когда установлено в **true**

, логическая структура документа создаётся, что может улучшить доступность и сделать документ более подходящим для программ чтения с экрана и других вспомогательных технологий.
