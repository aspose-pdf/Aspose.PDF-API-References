Пространство имён **[Aspose.Pdf.Devices](./)** предоставляет классы, которые используются для представления документа в виде изображения(й) или простого текста. Таким образом, документ может быть отправлен на текстовые или различные графические устройства, что означает, что мы хотим получить его текстовое или графическое представление.

## Классы

| Класс | Описание |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат bmp. |
| [Device](./device/) | Абстрактный класс для всех типов устройств. [Device](./device/) используется для представления PDF‑документа в некотором формате. Например, страница документа может быть представлена как изображение или текст. |
| [DicomDevice](./dicomdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат Dicom. |
| [DocumentDevice](./documentdevice/) | Абстрактный класс для всех устройств, который используется для обработки всего PDF‑документа. |
| [EmfDevice](./emfdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат emf. |
| [GifDevice](./gifdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат gif. |
| [ImageDevice](./imagedevice/) | Абстрактный класс для устройств изображения. |
| [JpegDevice](./jpegdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат jpeg. |
| [Margins](./margins/) | Этот класс представляет поля изображения. |
| [PageDevice](./pagedevice/) | Абстрактный класс для всех устройств, который используется для обработки определённой страницы PDF‑документа. |
| [PngDevice](./pngdevice/) | Представляет устройство изображения, которое помогает сохранять страницы PDF‑документа в формат png. |
| [Resolution](./resolution/) | Представляет класс для хранения разрешения изображения. |
| [TextDevice](./textdevice/) | Представляет класс для преобразования страниц PDF‑документа в текст. |
| [ThumbnailDevice](./thumbnaildevice/) | Представляет устройство изображения, которое сохраняет страницы PDF‑документа в виде миниатюрного изображения. |
| [TiffDevice](./tiffdevice/) | Этот класс помогает сохранять pdf‑документ постранично в одно tiff‑изображение. |
| [TiffSettings](./tiffsettings/) | Этот класс представляет настройки импорта pdf в Tiff. |
