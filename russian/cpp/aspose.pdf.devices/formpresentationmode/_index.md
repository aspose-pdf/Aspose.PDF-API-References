## FormPresentationMode enum


Используется для указания режима представления формы при печати или преобразовании в pdf‑документы изображений.

```cpp
enum class FormPresentationMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Production | 0 | Указывает режим представления формы Production (используется по умолчанию). |
| Editor | 1 | Указывает режим представления формы Editor. |
