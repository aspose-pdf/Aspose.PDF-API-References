## TiffSettings::get_Brightness method


Получает или задаёт границу значения преобразования цветов в белый и чёрный. Этот параметр может использоваться с EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle или [ColorDepth.Format1bpp](../../colordepth/) == 1.

```cpp
float Aspose::Pdf::Devices::TiffSettings::get_Brightness() const
```

## Примечания


Значение яркости должно находиться в диапазоне от 0 до 1. По умолчанию значение равно 0.33f
