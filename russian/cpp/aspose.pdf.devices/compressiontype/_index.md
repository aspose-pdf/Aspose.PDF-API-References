## CompressionType enum


Используется для указания значения параметра, передаваемого устройству изображения Tiff.

```cpp
enum class CompressionType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| LZW | 0 | Указывает схему сжатия LZW. Может быть передан кодировщику Tiff как параметр, относящийся к категории Compression. |
| CCITT4 | 1 | Указывает схему сжатия CCITT4. Может быть передан энкодеру CCITT4 в качестве параметра, принадлежащего категории Compression. |
| CCITT3 | 2 | Указывает схему сжатия CCITT3. Может быть передан энкодеру CCITT3 в качестве параметра, принадлежащего категории Compression. |
| RLE | 3 | Указывает схему сжатия RLE. Может быть передан энкодеру RLE в качестве параметра, принадлежащего категории Compression. |
| None | 4 | Указывает отсутствие сжатия. Может быть передан энкодеру Tiff в качестве параметра, принадлежащего категории compression. |
