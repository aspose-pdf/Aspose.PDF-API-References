| [WebServiceAttribute](./webserviceattribute/) | Добавляет дополнительную информацию к XML‑службе [Web](../system.web/). Объекты этого класса должны создаваться только с помощью функции [System::MakeObject()](../system/makeobject/). Никогда не создавайте экземпляр этого типа в стеке или с использованием оператора new, так как это приведёт к ошибкам выполнения и/или сбоям утверждений.
