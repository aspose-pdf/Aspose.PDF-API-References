## FileNotFoundException typedef


Исключение, которое выбрасывается, когда попытка доступа к файлу, не существующему на диске, завершается неудачей. Никогда не оборачивайте экземпляры класса [FileNotFoundException](./) в [System::SmartPtr](../../system/smartptr/).

```cpp
using System::IO::FileNotFoundException =  System::ExceptionWrapper<Details_FileNotFoundException>
```
