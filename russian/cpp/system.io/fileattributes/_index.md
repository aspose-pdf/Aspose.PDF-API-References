## FileAttributes enum


Представляет атрибуты каталога или файла.

```cpp
enum class FileAttributes
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Архив | 32 | Элемент файловой системы является кандидатом на удаление или резервное копирование. |
| Сжатый | 2048 | Элемент файловой системы сжат. |
| Device | 64 | Не используется. |
| Каталог | 16 | Элемент файловой системы является каталогом. |
| Зашифрованный | 16384 | Элемент файловой системы зашифрован. |
| Скрытый | 2 | Элемент файловой системы скрыт. |
| IntegrityStream | 32768 | Элемент файловой системы содержит поддержку целостности данных. |
| Normal | 128 | Элемент файловой системы является файлом без специальных атрибутов. |
| NotContentIndexed | 8192 | Элемент файловой системы не будет индексироваться службой индексирования содержимого. |
| NoScrubData | 131072 | Элемент файловой системы не включён в сканирование целостности данных. |
| Офлайн | 4096 | Элемент файловой системы находится в офлайн-режиме, и его данные недоступны сразу. |
| ReadOnly | 1 | Сущность файловой системы является только для чтения. |
| ReparsePoint | 1024 | Сущность файловой системы — это файл, содержащий точку повторного разбора. |
| SparseFile | 512 | Сущность файловой системы является разрежённым файлом. |
| System | 4 | Сущность файловой системы является системным файлом. |
| Temporary | 256 | Сущность файловой системы является временным файлом. |
| Недействителен | 0 | Недопустимое значение. |
