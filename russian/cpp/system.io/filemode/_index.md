## FileMode enum


Указывает, как должен быть открыт файл.

```cpp
enum class FileMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| CreateNew | 1 | Создать новый файл. Если файл уже существует, генерируется исключение. |
| Создать | 2 | Создать новый файл. Если файл уже существует, он перезаписывается. |
| Open | 3 | Открыть существующий файл. Если файл не существует, генерируется исключение. |
| OpenOrCreate | 4 | Открыть существующий файл или создать новый, если он не существует. |
| Обрезка | 5 | Открыть существующий файл и усечь его, чтобы он стал пустым. Если файл не существует, генерируется исключение. |
| Добавить | 6 | Открыть существующий файл и переместить указатель в конец, либо создать новый, если он не существует. |
