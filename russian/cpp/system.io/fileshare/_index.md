## FileShare enum


Указывает, какой тип доступа могут иметь другие объекты [FileStream](../filestream/) к открываемому файлу.

```cpp
enum class FileShare
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Нет доступа. |
| Read | 1 | Только для чтения. |
| Write | 2 | Только для записи. |
| ReadWrite | 3 | Доступ для чтения и записи. |
| Удалить | 4 | Файл может быть удалён. |
| Наследуемый | 16 | Делает дескриптор файла наследуемым дочерними процессами. |
