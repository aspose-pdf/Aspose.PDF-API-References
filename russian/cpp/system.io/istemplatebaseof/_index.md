## IsTemplateBaseOf typedef


Представляет аналог std::is_base_of<Base, Derived>, определяющий наследование неинстанцированного шаблонного класса Base от инстанцированного шаблонного класса Derived. Не будет работать при множественном наследовании или непубличном наследовании от Base.

```cpp
using System::Details::IsTemplateBaseOf =  typename IsTemplateBaseOfImpl<Base, Derived>::type
```
