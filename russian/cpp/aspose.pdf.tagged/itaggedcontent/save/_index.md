## ITaggedContent::Save method


Сохраняет текущее состояние помеченного содержимого в связанный PDF‑документ.

```cpp
virtual void Aspose::Pdf::Tagged::ITaggedContent::Save()=0
```

## Примечания


Этот метод гарантирует, что все элементы помеченного содержимого правильно обновлены и сохранены в PDF‑документе. Он выполняет необходимые операции, такие как обновление MCID для элементов MCR, установка операторов BDC и обеспечение соответствия стандартам PDF/UA.
