## ITaggedContent::SetLanguage method




```cpp
virtual void Aspose::Pdf::Tagged::ITaggedContent::SetLanguage(System::String lang)=0
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| lang | System::String | Идентификатор языка должен быть либо пустой строкой, указывающей, что язык неизвестен, либо языковым тегом (Language-Tag), определённым в RFC 3066, Tags for the Identification of Languages. |
## Примечания


Устанавливает естественный язык для PDF‑документа.

Идентификатор языка, который указывает естественный язык для всего текста в документе, за исключением случаев, когда он переопределяется спецификациями языка для структурных элементов или помеченного содержимого.
