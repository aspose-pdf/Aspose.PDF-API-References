## AutoSizeMode enum


Определяет, как работает автоматический размер.

```cpp
enum class AutoSizeMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) растёт и сжимается по мере необходимости. Его нельзя изменять вручную. |
| GrowOnly | 1 | [Control](../control/) растёт по мере необходимости, но не может сжиматься ниже минимального размера. Его можно изменять вручную. |
