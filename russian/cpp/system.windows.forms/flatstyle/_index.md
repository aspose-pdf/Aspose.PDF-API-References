## FlatStyle enum


Определяет, будет ли элемент управления отрисован плоским или трёхмерным.

```cpp
enum class FlatStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) отображается плоско. |
| Popup | 1 | [Control](../control/) отображается плоско, если только мышь не перемещается над ним, когда он отображается в 3D-стиле. |
| Standard | 2 | [Control](../control/) отображается в 3D-стиле. |
| System | 3 | Стиль управляется ОС. |
