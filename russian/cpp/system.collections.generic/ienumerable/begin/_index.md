## IEnumerable::begin() method


Получает итератор, указывающий на первый элемент (если он есть) коллекции. Этот итератор нельзя использовать для изменения ссылочного объекта, потому что [GetEnumerator()](../getenumerator/) возвращает копию объекта T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```


### ReturnValue

Итератор, указывающий на первый элемент (если он есть) коллекции
