## IEnumerable::end() method


Получает итератор, указывающий сразу после последнего элемента (если он существует) в коллекции. Этот итератор нельзя использовать для изменения ссылочного объекта, потому что [GetEnumerator()](../getenumerator/) возвращает копию-объект типа T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```
