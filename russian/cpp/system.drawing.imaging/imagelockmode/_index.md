## ImageLockMode enum


Указывает свойства области изображения, которая блокируется.

```cpp
enum class ImageLockMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ReadOnly | 1 | Область заблокирована только для чтения. |
| WriteOnly | 2 | Область заблокирована только для записи. |
| ReadWrite | n/a | Область заблокирована как для чтения, так и для записи. |
| UserInputBuffer | 4 | Изображение находится в пользовательском буфере ввода, доступ к которому контролируется пользователем. |
