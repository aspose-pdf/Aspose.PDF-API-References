## PixelFormat enum


Указывает формат цветовых данных пикселя.

```cpp
enum class PixelFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Indexed | 65536 | Указывает, что данные пикселя содержат индексированные цветовые значения, что означает, что они являются индексом к цветам в системной таблице цветов. |
| Gdi | 131072 | Указывает, что данные пикселя содержат цвета GDI. |
| Alpha | 262144 | Указывает, что данные пикселя содержат альфа-значения, которые не предумножены. |
| PAlpha | 524288 | Указывает, что данные пикселя содержат предумноженные альфа-значения. |
| Extended | 1048576 | Зарезервировано. |
| Canonical | 2097152 | Указывает формат пикселя 32 бита на пиксель с 24-битной глубиной цвета и 8-битным альфа-каналом. |
| Неопределено | 0 | Указывает, что формат пикселя не определён. |
| DontCare | 0 | Формат пикселя не указан. |
| Format1bppIndexed | n/a | Указывает, что формат пикселя — 1 бит на пиксель индексированный цвет. |
| Format4bppIndexed | n/a | Указывает, что формат пикселя — 4 бита на пиксель индексированный цвет. |
| Format8bppIndexed | n/a | Указывает, что формат пикселя — 8 бит на пиксель индексированный цвет. |
| Format16bppGrayScale | n/a | Указывает, что формат пикселя — 16 бит на пиксель. Информация о цвете указывает 65536 оттенков серого. |
| Format16bppRgb555 | n/a | Указывает, что формат пикселя — 16 бит на пиксель, по 5 бит для компонентов красного, зелёного и синего, а оставшийся бит не используется. |
| Format16bppRgb565 | n/a | Указывает, что формат пикселя — 16 бит на пиксель, по 5 бит для красного, 6 бит для зелёного и 5 бит для синего компонентов. |
| Format16bppArgb1555 | n/a | Указывает, что формат пикселя — 16 бит на пиксель, по 5 бит для компонентов красного, зелёного и синего и 1 бит для альфа‑канала. |
| Format24bppRgb | n/a | Указывает, что формат пикселя — 24 бита на пиксель, по 8 бит для компонентов красного, зелёного и синего. |
| Format32bppRgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель, по 8 бит для компонентов красного, зелёного и синего, а оставшиеся 8 бит не используются. |
| Format32bppArgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель, по 8 бит для компонентов красного, зелёного и синего и 8 бит для альфа‑канала. |
| Format32bppPArgb | n/a | Указывает, что формат пикселя — 32 бита на пиксель, по 8 бит для компонентов красного, зелёного и синего и 8 бит для альфа‑канала. Компоненты красного, зелёного и синего предварительно умножаются на значение альфа‑канала. |
| Format48bppRgb | n/a | Указывает, что формат пикселя — 48 бит на пиксель, по 16 бит для компонентов красного, зелёного и синего. |
| Format64bppArgb | n/a | Указывает, что формат пикселя — 64 бита на пиксель, по 16 бит для компонентов красного, зелёного и синего и 16 бит для альфа‑канала. |
| Format64bppPArgb | n/a | Указывает, что формат пикселя — 64 бита на пиксель, по 16 бит для компонентов красного, зелёного и синего и 16 бит для альфа‑канала. Компоненты красного, зелёного и синего предварительно умножаются на значение альфа‑канала. |
| Format32bppCMYK | n/a | Указывает, что формат пикселя — 32 бита на пиксель, по 8 бит для компонентов голубого, пурпурного, жёлтого и чёрного. |
| Max | 16 | Максимальное значение этого перечисления. |
