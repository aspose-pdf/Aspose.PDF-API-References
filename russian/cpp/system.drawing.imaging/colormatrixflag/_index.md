## ColorMatrixFlag enum


Указывает типы изображений и цветов, которые будут затронуты настройками коррекции цвета и градаций серого объекта [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | И цвета, и значения градаций серого подлежат коррекции цвета. |
| SkipGrays | 1 | Значения градаций серого не корректируются по цвету. |
| AltGrays | 2 | Только значения градаций серого корректируются по цвету. |
