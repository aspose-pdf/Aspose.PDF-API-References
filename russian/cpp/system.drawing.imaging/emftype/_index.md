## EmfType enum


Указывает типы записей, которые размещаются в файле EMF.

```cpp
enum class EmfType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) расширенный метафайл. Содержит команды GDI. Метапфайлы этого типа называют EMF‑файлом. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) расширенный метафайл plus. Содержит команды GDI+. Метапфайлы этого типа называют EMF+‑файлом. |
| EmfPlusDual | n/a | Двойной [Windows](../../system.windows/) расширенный метафайл. Содержит эквивалентные команды GDI и GDI+. Метапфайлы этого типа называют EMF+‑файлом. |
