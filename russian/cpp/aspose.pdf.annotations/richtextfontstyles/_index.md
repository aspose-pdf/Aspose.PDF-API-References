## RichTextFontStyles enum


Параметры стилизации фрагментов текста в RichText.

```cpp
enum class RichTextFontStyles
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ClearExisting | 1 | Если установлен, очищает все существующие стили перед применением дополнительных. При сочетании с другими флагами стилей (например, [Bold](./)), сначала сбрасывает стили, затем применяет указанные. Без этого флага новые стили добавляются к существующим. |
| Bold | 2 | Опция, указывающая жирный стиль. |
| Italic | 4 | Опция, указывающая курсив. |
| Underline | 8 | Опция, указывающая подчеркивание. |
