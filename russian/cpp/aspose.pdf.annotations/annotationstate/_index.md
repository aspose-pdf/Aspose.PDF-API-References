## AnnotationState enum


Перечисление состояний, в которые может быть установлена оригинальная аннотация.

enum class AnnotationState

| Имя | Значение | Описание |
| --- | --- | --- |
| Неопределено | 0 | Состояние не определено. |
| Отмечено | 1 | Аннотация была отмечена пользователем. |
| Не отмечено | 2 | Аннотация не была отмечена пользователем. |
| Accepted | 3 | Пользователь согласен с изменением. |
| Отклонено | 4 | Пользователь не согласен с изменением. |

| Завершено | 6 | Изменение было завершено. |
| None | 7 | Пользователь не указал ничего относительно изменения. |
