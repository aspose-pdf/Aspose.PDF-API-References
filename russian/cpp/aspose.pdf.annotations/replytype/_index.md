## ReplyType enum


Перечисляет виды отношений (\"reply type\") между аннотацией и той, которая указана в InReplyTo.

```cpp
enum class ReplyType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Неопределено | 0 | Неопределённое отношение. |
| Reply | 1 | Аннотация считается ответом на аннотацию, указанную в InReplyTo. Приложения‑просмотрщики не должны отображать ответы на аннотацию по отдельности, а объединять их в виде ветвящихся комментариев. |
| Group | 2 | Аннотация сгруппирована с аннотацией, указанной в InReplyTo. |
