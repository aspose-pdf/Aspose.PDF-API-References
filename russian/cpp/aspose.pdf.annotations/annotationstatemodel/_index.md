## AnnotationStateModel enum


Модель состояния, соответствующая состоянию аннотации.

```cpp
enum class AnnotationStateModel
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Неопределено | 0 | Не определённая модель состояния. |
| Отмечено | 1 | Аннотация была отмечена (или снята) пользователем. |
