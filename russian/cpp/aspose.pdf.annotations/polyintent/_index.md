## PolyIntent enum


Перечисляет намерения полигональной или полилинейной аннотации.

```cpp
enum class PolyIntent
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Неопределено | 0 | Неопределённое состояние. |
| PolygonCloud | 1 | Означает, что аннотация предназначена для работы в виде облачного объекта. |
| PolyLineDimension | 2 | Указывает, что полилинейная аннотация предназначена для работы в качестве измерения. |
| PolygonDimension | 3 | Указывает, что полигональная аннотация предназначена для работы в качестве измерения. |
