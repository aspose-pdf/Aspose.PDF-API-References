## ExplicitDestinationType enum


Перечисляет типы явных пунктов назначения.

```cpp
enum class ExplicitDestinationType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| XYZ | 0 | Отображает страницу, при этом координаты (left, top) размещаются в левом верхнем углу окна, а содержимое страницы масштабируется коэффициентом zoom. Значение null для любого из параметров left, top или zoom указывает, что текущие значения этих параметров должны оставаться неизменными. Значение zoom, равное 0, имеет то же значение, что и null. |
| Fit | 1 | Отображает страницу, масштабируя её содержимое настолько, чтобы вся страница полностью помещалась в окне как по горизонтали, так и по вертикали. Если требуемые коэффициенты масштабирования по горизонтали и вертикали различаются, используется меньший из них, при этом страница центрируется в окне по другой оси. |
| FitH | 2 | Отображает страницу, при этом вертикальная координата top размещается у верхнего края окна, а содержимое страницы масштабируется настолько, чтобы вся ширина страницы полностью помещалась в окне. Значение null для top указывает, что текущий параметр должен оставаться неизменным. |
| FitV | 3 | Отображает страницу, при этом горизонтальная координата left размещается у левого края окна, а содержимое страницы масштабируется настолько, чтобы вся высота страницы полностью помещалась в окне. Значение null для left указывает, что текущий параметр должен оставаться неизменным. |
| FitR | 4 | Отображает страницу, масштабируя её содержимое настолько, чтобы прямоугольник, заданный координатами left, bottom, right и top, полностью помещался в окне как по горизонтали, так и по вертикали. Если требуемые коэффициенты масштабирования по горизонтали и вертикали различаются, используется меньший из них, при этом прямоугольник центрируется в окне по другой оси. Значение null для любого из параметров может привести к непредсказуемому поведению. |
| FitB | 5 | Отображает страницу, масштабируя её содержимое настолько, чтобы её ограничивающий прямоугольник полностью помещался в окне как по горизонтали, так и по вертикали. Если требуемые коэффициенты масштабирования по горизонтали и вертикали различаются, используется меньший из них, при этом ограничивающий прямоугольник центрируется в окне по другой оси. |
| FitBH | 6 | Отображает страницу, при этом вертикальная координата top размещается у верхнего края окна, а содержимое страницы масштабируется настолько, чтобы вся ширина её ограничивающего прямоугольника полностью помещалась в окне. Значение null для top указывает, что текущий параметр должен оставаться неизменным. |
| FitBV | 7 | Отобразить страницу так, чтобы горизонтальная координата left была расположена у левого края окна, а содержимое страницы увеличивалось ровно настолько, чтобы вся высота её ограничивающего прямоугольника помещалась в окно. Значение null для left указывает, что текущее значение этого параметра должно оставаться без изменений. |
