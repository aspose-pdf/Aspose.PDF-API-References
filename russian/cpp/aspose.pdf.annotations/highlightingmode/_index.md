## HighlightingMode enum


Перечисляет режим подсветки аннотации, визуальный эффект, который будет использоваться при нажатии или удержании кнопки мыши внутри её активной области.

```cpp
enum class HighlightingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Без подсветки. |
| Инвертировать | 1 | Инвертировать содержимое прямоугольника аннотации. |
| Контур | 2 | Инвертировать границу аннотации. |
| Вытолкнуть | 3 | Отобразить нижнее представление аннотации, если оно есть. Если нижнее представление не определено, сместить содержимое прямоугольника аннотации так, чтобы казалось, что оно вытолкнуто под поверхность страницы. |
| Переключить | 4 | То же, что и Вытолкнуть (что предпочтительно). |
