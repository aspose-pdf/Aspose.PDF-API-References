## PrinterMarksKind enum


Указывает типы печатных меток, которые следует добавить в документ.

```cpp
enum class PrinterMarksKind
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Указывает, что метки принтера не будут добавлены. |
| TrimMarks | n/a | Указывает, что метки обрезки будут добавлены. |
| BleedMarks | n/a | Указывает, что метки обрезки будут добавлены. |
| RegistrationMarks | n/a | Указывает, что регистрационные метки будут добавлены. |
| ColorBars | n/a | Указывает, что цветные полосы будут добавлены. |
| PageInformation | n/a | Указывает, что информация о странице будет добавлена. |
| Все | n/a | Указывает, что будут добавлены все типы меток принтера. |
