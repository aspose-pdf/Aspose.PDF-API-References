## RenditionOperation enum


Операция, которую следует выполнить при срабатывании действия.

```cpp
enum class RenditionOperation
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| PlayStop | 0 | Если с аннотацией не связан ни один рендеринг, воспроизведите указанный рендеринг, связав его с аннотацией. Если рендеринг уже связан с аннотацией, он будет остановлен, и новый рендеринг будет связан с аннотацией. |
| Stop | 1 | Остановить любой рендеринг, воспроизводимый в связке с аннотацией. |
| Pause | 2 | Приостановить любой рендеринг, воспроизводимый в связке с аннотацией. |
| Возобновление | 3 | Возобновить любой рендеринг, воспроизводимый в связке с аннотацией. |
| PlayResume | 4 | Воспроизвести указанный рендеринг, связав его с аннотацией. Если рендеринг уже связан с аннотацией, возобновить рендеринг, если он приостановлен. |
| Неопределено | -1 | Операция не определена. |
