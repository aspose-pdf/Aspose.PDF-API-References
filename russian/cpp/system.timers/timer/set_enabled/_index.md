## Timer::set_Enabled method


Запускает или останавливает таймер. Запуск таймера не перезапускает отсчёт времени, если таймер уже работает.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | bool | true, чтобы запустить таймер, false, чтобы остановить. |
