## TiffOptions::get_Brightness method


Получает или задает границу значений преобразования цветов в белый и черный. Этот параметр может использоваться с EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle или ColorDepth.Format1bpp == 1.

```cpp
float Aspose::Pdf::LowCode::TiffOptions::get_Brightness()
```

## Примечания


Значение яркости должно находиться в диапазоне от 0 до 1. По умолчанию значение равно 0.33f
