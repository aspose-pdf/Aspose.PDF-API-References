## PdfAOptionsBase::set_SymbolicFontEncodingStrategy method


Устанавливает стратегию кодирования символических шрифтов при конвертации в формат PDF/A.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_SymbolicFontEncodingStrategy(const System::SharedPtr<PdfASymbolicFontEncodingStrategy> &value)
```

## Примечания


Стратегия кодирования символических шрифтов.

Это свойство позволяет вам контролировать, какая подтаблица CMap будет скопирована в результирующий документ в случаях, когда исходный символический шрифт TrueType в исходном документе содержит несколько подтаблиц CMap.
