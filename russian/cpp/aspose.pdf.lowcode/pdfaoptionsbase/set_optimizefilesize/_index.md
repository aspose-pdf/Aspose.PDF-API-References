## PdfAOptionsBase::set_OptimizeFileSize method


Устанавливает значение, указывающее, следует ли пытаться уменьшить размер файла во время процесса конвертации PDF/A.

```cpp
void Aspose::Pdf::LowCode::PdfAOptionsBase::set_OptimizeFileSize(bool value)
```

## Примечания


**true**

если размер файла следует уменьшать насколько возможно; в противном случае **false**

.

Когда установлено в **true**

, процесс конвертации попытается минимизировать размер получаемого файла. Это может повлиять на производительность процесса конвертации.
