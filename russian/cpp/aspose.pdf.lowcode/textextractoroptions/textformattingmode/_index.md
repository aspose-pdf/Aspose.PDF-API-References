## TextFormattingMode enum


Определяет различные режимы, которые могут использоваться при преобразовании PDF‑документа в текст. См. класс [TextExtractorOptions](../).

```cpp
enum class TextFormattingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Pure | 0 | Представляет текстовое содержимое PDF с несколькими процедурами форматирования. Это включает учёт относительных позиций и добавление дополнительных пробелов для выравнивания текста по ширине страницы. |
| Raw | 1 | Представляет содержимое PDF без изменений, то есть без форматирования. |
| Plain | 2 | Представляет текстовое содержимое PDF с учётом относительного позиционирования фрагментов текста, но без добавления дополнительных пробелов. |
