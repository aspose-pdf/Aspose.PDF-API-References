## SoapParameterStyle enum


Перечисляет форматы параметров в SOAP‑сообщении.

```cpp
enum class SoapParameterStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Default | 0 | Если атрибут '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' не применён к классу, то значение по умолчанию — 'Wrapped'. |
| Bare | 1 | Параметры размещаются в XML‑элементах, следующих за элементом 'Body'. |
| Wrapped | 2 | Параметры инкапсулируются в единственном XML‑элементе, следующем за элементом 'Body'. |
