## VerificationState enum


Указывает состояние процесса проверки цифровой подписи в PDF‑документе.

```cpp
enum class VerificationState
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Действителен | 0 | Представляет состояние, указывающее, что процесс проверки цифровой подписи завершён успешно, и подпись действительна. |
| Недействителен | 1 | Представляет состояние, указывающее, что процесс проверки цифровой подписи завершился неудачей, и подпись считается недействительной. |
| Неопределено | 2 | Представляет состояние, при котором результат процесса проверки цифровой подписи не определён. |
