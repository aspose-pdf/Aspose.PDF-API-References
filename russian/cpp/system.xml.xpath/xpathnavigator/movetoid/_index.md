## XPathNavigator::MoveToId method


При переопределении в производном классе перемещается к узлу, имеющему атрибут типа **ID**, значение которого совпадает с указанным [String](../../../system/string/).

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| id | String | [String](../../../system/string/), представляющий значение **ID** узла, к которому вы хотите переместиться. |

### ReturnValue

**true** if the [XPathNavigator](../) is successful moving; otherwise, **false**. If **false**, the position of the navigator is unchanged.
