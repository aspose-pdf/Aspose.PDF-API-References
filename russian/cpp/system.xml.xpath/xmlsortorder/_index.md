## XmlSortOrder enum


Указывает порядок сортировки.

```cpp
enum class XmlSortOrder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| По возрастанию | 1 | Узлы сортируются по возрастанию. Например, если числа 1,2,3 и 4 сортируются по возрастанию, они отображаются как 1,2,3,4. |
| По убыванию | 2 | Узлы сортируются по убыванию. Например, если числа 1,2,3 и 4 сортируются по убыванию, они отображаются как 4,3,2,1. |
