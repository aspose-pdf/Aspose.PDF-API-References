## LineCap enum


Указывает доступные стили окончаний, которыми объект [Pen](../../system.drawing/pen/) может завершать линию.

```cpp
enum class LineCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Плоский | 0 | Плоская головка линии. |
| Квадрат | 1 | Квадратная головка линии. |
| Round | 2 | Круглая головка линии. |
| Треугольник | 3 | Треугольная головка линии. |
| НетЯкоря | 16 | Нет якоря. |
| КвадратныйЯкорь | 17 | Головка линии с квадратным якорем. |
| КруглыйЯкорь | 18 | Головка линии с круглым якорем. |
| АлмазныйЯкорь | 19 | Головка линии с алмазным якорем. |
| СтрелочныйЯкорь | 20 | Головка линии со стрелочным якорем. |
| МаскаЯкоря | 240 | Маска, используемая для проверки, является ли головка линии якорной. |
| Custom | 255 | Пользовательская головка. |
