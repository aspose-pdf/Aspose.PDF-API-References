## SmoothingMode enum


Указывает, какой тип сглаживания (антиалиасинга) применяется к линиям, кривым и краям заполненных областей.

```cpp
enum class SmoothingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Недействителен | n/a | Недопустимое значение. |
| Default | n/a | Режим по умолчанию, без антиалиасинга. |
| HighSpeed | n/a | Без антиалиасинга, высокая скорость. |
| HighQuality | n/a | С антиалиасингом, низкая скорость. |
| None | n/a | Без антиалиасинга. |
| AntiAlias | n/a | Отрисовка с антиалиасингом. |
