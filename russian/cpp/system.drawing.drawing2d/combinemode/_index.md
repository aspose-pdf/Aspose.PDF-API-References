## CombineMode enum


Указывает, как комбинируются области отсечения.

```cpp
enum class CombineMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Заменить | 0 | Одна область отсечения заменяется другой. |
| Intersect | 1 | Две области отсечения комбинируются путём взятия их пересечения. |
| Объединение | 2 | Две области отсечения комбинируются путём объединения обеих. |
| Xor | 3 | Две области отсечения комбинируются, оставляя только площадь, заключённую в одной из областей, но не в обеих. |
| Исключить | 4 | Две области отсечения комбинируются, беря площадь первой области, не пересекающуюся со второй. |
| Дополнение | 5 | Две области отсечения комбинируются, беря площадь второй области, не пересекающуюся с первой. |
