## DashCap enum


Указывает тип заглушки, используемой на обоих концах штриха в пунктирной линии.

```cpp
enum class DashCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Плоский | 0 | Квадратное окончание, которое делает оба конца штриха квадратными. |
| Round | 2 | Круглое окончание, которое скругляет оба конца штриха. |
| Треугольник | 3 | Треугольное окончание, которое направлено к обоим концам штриха. |
