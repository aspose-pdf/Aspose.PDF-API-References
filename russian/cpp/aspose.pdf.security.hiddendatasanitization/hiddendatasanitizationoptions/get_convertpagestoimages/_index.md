## HiddenDataSanitizationOptions::get_ConvertPagesToImages method


Возвращает параметр конвертации страниц в изображения. Если этот параметр включён, параметр ImageCompressionOptions будет игнорироваться. Параметр должен быть включён вручную при использовании метода [All](../all/), если он требуется. Конвертация страниц в изображения будет происходить после очистки основных скрытых данных, которые контролируются другими параметрами.
