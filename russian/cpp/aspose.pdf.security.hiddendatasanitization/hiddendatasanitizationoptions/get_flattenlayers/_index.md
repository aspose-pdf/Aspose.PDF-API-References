## HiddenDataSanitizationOptions::get_FlattenLayers method


Возвращает параметр, позволяющий уплощать слои в PDF‑документе. При включении все слои документа объединяются в один слой, удаляя их отдельную структуру. Этот параметр полезен для очистки документов, упрощая их содержимое и гарантируя отсутствие скрытых данных в слоях.
