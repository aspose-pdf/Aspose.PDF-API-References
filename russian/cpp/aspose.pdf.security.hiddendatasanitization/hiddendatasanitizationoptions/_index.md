## HiddenDataSanitizationOptions class


Представляет параметры конфигурации для очистки скрытых данных в документе.

```cpp
class HiddenDataSanitizationOptions : public System::Object
```

## Методы

| Метод | Описание |
| --- | --- |
| static [All](./all/)() | Создаёт новый экземпляр класса [HiddenDataSanitizationOptions](./) со всеми параметрами, установленными для очистки. Это включает включение удаления аннотаций, JavaScript, метаданных, вложений, индекса поиска, личной информации, уплощения форм и слоёв, при этом отключая возможность конвертировать страницы в изображения. Дополнительные конфигурации, такие как [ImageCompressionOptions](../) или [ConvertPagesToImages](../), могут быть изменены вручную после получения экземпляра, так как они не активированы по умолчанию. |
| [get_ConvertPagesToImages](./get_convertpagestoimages/)() const | Получает параметр конвертации страниц в изображения. Если этот параметр включён, параметр ImageCompressionOptions будет игнорироваться. Параметр должен быть включён вручную при использовании метода [All](./all/), если он требуется. Конвертация страниц в изображения произойдёт после очистки основных скрытых данных, что контролируется другими параметрами. |
| [get_FlattenForms](./get_flattenforms/)() const | Возвращает значение, указывающее, следует ли уплощать формы в документе во время процесса очистки. Уплощение форм преобразует интерактивные поля формы в статический контент, делая их не редактируемыми или заполняемыми. |
| [get_FlattenLayers](./get_flattenlayers/)() const | Возвращает параметр, позволяющий уплощать слои в PDF‑документе. При включении все слои документа объединяются в один слой, удаляя их отдельную структуру. Этот параметр полезен для очистки документов, упрощая их содержимое и гарантируя отсутствие скрытых данных в слоях. |
| [get_ImageCompressionOptions](./get_imagecompressionoptions/)() const | Возвращает параметр конвертации изображений документа. Параметр необходимо включать вручную при использовании метода [All](./all/), если это требуется. |

| [get_RemoveAttachments](./get_removeattachments/)() const | Возвращает параметр, позволяющий удалять все вложенные файлы из документа. При включении он обеспечивает удаление всех вложений в PDF во время процесса очистки. |
| [get_RemoveJavaScriptsAndActions](./get_removejavascriptsandactions/)() const | Возвращает значение, указывающее, следует ли удалять JavaScript и связанные с ним действия из документа. Этот параметр полезен для устранения потенциальных уязвимостей безопасности, вызываемых встроенными скриптами. |
| [get_RemoveMetadata](./get_removemetadata/)() const | Возвращает параметр, позволяющий удалять метаданные из документа. Если установлен в true, такие метаданные, как свойства документа и дополнительная встроенная информация, будут удалены во время очистки. |
