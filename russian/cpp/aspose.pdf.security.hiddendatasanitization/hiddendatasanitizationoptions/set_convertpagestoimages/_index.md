## HiddenDataSanitizationOptions::set_ConvertPagesToImages method


Устанавливает параметр конвертации страниц в изображения. Если этот параметр включён, параметр ImageCompressionOptions будет игнорироваться. Параметр необходимо включать вручную при использовании метода [All](../all/), если он требуется. Конвертация страниц в изображения будет выполнена после очистки основных скрытых данных, контролируемых другими параметрами.
