## XmlElement::set_IsEmpty method


Устанавливает формат тега элемента.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | bool | Значение для установки. При установке этого значения, если оно установлено в **true**, дочерние элементы удаляются, и элемент сериализуется в формате короткого тега. Если установлено в **false**, значение изменяется (независимо от того, содержит элемент контент или нет); если элемент пуст, он сериализуется в длинном формате. |
