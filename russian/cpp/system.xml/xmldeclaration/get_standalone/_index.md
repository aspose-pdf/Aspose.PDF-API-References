## XmlDeclaration::get_Standalone method


Возвращает значение атрибута standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Допустимыми значениями являются **yes**, если все объявления сущностей, требуемые XML‑документом, содержатся в документе, или **no**, если требуется внешнее определение типа документа (DTD). Если атрибут standalone отсутствует в объявлении XML, этот метод возвращает [String::Empty](../../../system/string/empty/).
