## XmlWriterSettings::set_NamespaceHandling method


Устанавливает значение, указывающее, должен ли [XmlWriter](../../xmlwriter/) удалять дублирующие объявления пространств имён при записи XML‑содержимого. Поведение по умолчанию заключается в том, что запись выводит все объявления пространств имён, присутствующие в разрешателе пространств имён записи.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | System::Xml::NamespaceHandling | Значение для установки. |
