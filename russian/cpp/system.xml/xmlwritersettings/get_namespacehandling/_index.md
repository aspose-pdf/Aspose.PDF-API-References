## XmlWriterSettings::get_NamespaceHandling method


Возвращает значение, указывающее, должен ли [XmlWriter](../../xmlwriter/) удалять дублирующие объявления пространств имён при записи XML‑содержимого. Поведение по умолчанию заключается в том, что запись выводит все объявления пространств имён, присутствующие в разрешателе пространств имён записи.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

Перечисление [NamespaceHandling](../../namespacehandling/), используемое для указания, следует ли удалять дублирующие объявления пространств имён в [XmlWriter](../../xmlwriter/).
