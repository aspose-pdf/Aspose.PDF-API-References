## XmlNode::Normalize method


Помещает все узлы [XmlText](../../xmltext/) на полной глубине поддерева под этим [XmlNode](../) в «нормальную» форму, где только разметка (то есть теги, комментарии, инструкции обработки, секции CDATA и ссылки на сущности) разделяют узлы [XmlText](../../xmltext/), то есть рядом не находятся соседние узлы [XmlText](../../xmltext/).
