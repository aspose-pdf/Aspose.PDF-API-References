## XmlNamespaceManager::RemoveNamespace method


Удаляет указанный пространство имён для данного префикса.

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| префикс | String | Префикс для пространства имён. |
| uri | String | Пространство имён, которое нужно удалить для указанного префикса. Удаляемое пространство имён берётся из текущей области пространства имён. Пространства имён за пределами текущей области игнорируются. |
