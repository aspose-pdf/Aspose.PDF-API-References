## XmlReader::ReadString method


При переопределении в производном классе читает содержимое элемента или текстового узла как строку. Однако рекомендуется использовать метод [XmlReader::ReadElementContentAsString](../readelementcontentasstring/), поскольку он предоставляет более простой способ выполнения этой операции.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```


### ReturnValue

Содержимое элемента или пустая строка.
