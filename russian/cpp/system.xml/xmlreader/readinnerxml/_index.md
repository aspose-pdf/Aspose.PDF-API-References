## XmlReader::ReadInnerXml method


При переопределении в производном классе считывает всё содержимое, включая разметку, в виде строки.

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```


### ReturnValue

Весь XML‑контент, включая разметку, в текущем узле. Если у текущего узла нет дочерних элементов, возвращается пустая строка. Если текущий узел не является ни элементом, ни атрибутом, возвращается пустая строка.
