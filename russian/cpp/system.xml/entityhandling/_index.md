## EntityHandling enum


Указывает, как [XmlTextReader](../xmltextreader/) или [XmlValidatingReader](../xmlvalidatingreader/) обрабатывают сущности.

```cpp
enum class EntityHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ExpandEntities | 1 | Расширяет все сущности и возвращает расширенные узлы. |
| ExpandCharEntities | 2 | Расширяет символьные сущности и возвращает общие сущности как узлы [XmlNodeType::EntityReference](../xmlnodetype/). |
