## XmlReaderSettings::get_MaxCharactersInDocument method


Возвращает значение, указывающее максимальное допустимое количество символов в XML‑документе. Нулевое (0) значение означает отсутствие ограничений размера XML‑документа. Ненулевое значение задаёт максимальный размер в символах.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

Максимальное допустимое количество символов в XML‑документе. По умолчанию 0.
