## XmlConvert::VerifyXmlChars method


Возвращает переданную строку, если все символы и символы суррогатных пар в аргументе строки являются допустимыми XML‑символами, в противном случае генерируется [XmlException](../../xmlexception/) с информацией о первом найденном недопустимом символе.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| content | const String\& | [String](../../../system/string/) содержащий символы для проверки. |

### ReturnValue

Переданная строка, если все символы и символы суррогатных пар в аргументе строки являются допустимыми XML‑символами, в противном случае генерируется [XmlException](../../xmlexception/) с информацией о первом найденном недопустимом символе.
