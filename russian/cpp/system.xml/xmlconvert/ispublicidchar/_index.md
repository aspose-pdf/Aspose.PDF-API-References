## XmlConvert::IsPublicIdChar method


Возвращает переданный экземпляр символа, если символ в аргументе является допустимым символом публичного идентификатора, иначе **nullptr**.

```cpp
static bool System::Xml::XmlConvert::IsPublicIdChar(char16_t ch)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| ch | char16_t | Объект [Char](../../../system/char/) для проверки. |

### ReturnValue

Переданный символ, если он является допустимым символом публичного идентификатора, иначе **nullptr**.
