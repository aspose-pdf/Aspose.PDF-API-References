## XmlTextWriter::LookupPrefix method


Возвращает ближайший префикс, определённый в текущей области пространства имён для указанного URI пространства имён.

```cpp
String System::Xml::XmlTextWriter::LookupPrefix(String ns) override
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| ns | String | URI пространства имён, префикс которого вы хотите найти. |

### ReturnValue

Соответствующий префикс. Или **nullptr**, если в текущей области не найдено подходящего URI пространства имён.
