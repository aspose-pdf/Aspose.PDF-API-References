## XmlTextWriter::get_Indentation method


Возвращает количество IndentChars, которое следует записывать для каждого уровня иерархии, когда [XmlTextWriter::set_Formatting](../set_formatting/) установлен в [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```


### ReturnValue

Количество **IndentChars** для каждого уровня. По умолчанию равно 2.
