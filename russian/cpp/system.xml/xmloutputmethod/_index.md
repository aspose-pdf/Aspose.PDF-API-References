## XmlOutputMethod enum


Указывает метод, используемый для сериализации вывода [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Xml | 0 | Сериализовать в соответствии с правилами XML 1.0. |
| Html | 1 | Сериализовать в соответствии с правилами HTML, определёнными XSLT. |
| Text | 2 | Сериализовать только текстовые блоки. |
| AutoDetect | 3 | Используйте правила XSLT для выбора между методами вывода [XmlOutputMethod::Xml](./) и [XmlOutputMethod::Html](./) во время выполнения. |
