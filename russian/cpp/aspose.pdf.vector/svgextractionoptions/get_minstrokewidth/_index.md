## SvgExtractionOptions::get_MinStrokeWidth method


Получает минимальную ширину штриха, которая будет использована в результирующем SVG. Если в PDF используется более тонкая ширина штриха, она будет заменена этой шириной. Значение по умолчанию — 0.5.

```cpp
double Aspose::Pdf::Vector::SvgExtractionOptions::get_MinStrokeWidth() const
```

## Примечания


Значение выражается в преобразованных единицах пользовательского пространства конвертируемой страницы PDF. По умолчанию 1 единица пользовательского пространства равна 1/72 дюйма (0,35 мм), но это может быть переопределено документом PDF. Преобразования могут влиять на фактическую минимальную ширину в генерируемом SVG.
