## SvgExtractionOptions::set_UnpackPageContentXForm method


Получает и задает флаг, определяющий, следует ли распаковывать найденные на страницах XFrom. Элементы XFrom могут оказаться в разных SVG‑файлах. Распаковываются только XForms, отрисованные инструкциями Do из содержимого страницы. Вложенные XForms не распаковываются.
