## Enumeração LineBreakStyle

Representa os possíveis estilos de quebra de linha para um arquivo.

```csharp
public enum LineBreakStyle
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Windows | `0` | Seleciona o estilo de quebra de string do Windows (CRLF). |
| Unix | `1` | Seleciona o estilo de quebra de string do Unix (LF). Também usado para OSX. |
| Auto | `2` | Seleciona automaticamente o tipo de quebra de string com base no sistema operacional. |
