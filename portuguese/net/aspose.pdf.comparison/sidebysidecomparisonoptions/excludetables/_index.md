## Propriedade SideBySideComparisonOptions.ExcludeTables

Obtenha e defina a opção que determina se as tabelas são excluídas da comparação. Esta opção não pode ser definida juntamente com [`ComparisonArea1`](../comparisonarea1/) e [`ComparisonArea2`](../comparisonarea2/). O valor padrão é `false`.

```csharp
public bool ExcludeTables { get; set; }
```
