## Método ImagesDifference.DifferenceToImage

Converte o array de diferenças em uma imagem bitmap usando as cores especificadas.

```csharp
public Bitmap DifferenceToImage(Color color, Color backgroundColor)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| color | Color | A cor para diferenças não zero. |
| backgroundColor | Color | A cor de fundo para diferenças zero. |

### Valor de Retorno

Uma imagem bitmap representando o array de diferenças.
