## Classe MarkdownDiffOutputGenerator

Representa uma classe para gerar a representação markdown das diferenças de textos. Devido à sintaxe markdown, não é possível mostrar mudanças nos caracteres de espaço em branco. A seleção de mudanças faz com que caracteres de espaço em branco sejam adicionados ao redor da formatação, caso contrário, o visualizador markdown não exibirá corretamente o texto. Quebras de linha deletadas são indicadas pelo marcador de parágrafo.

```csharp
public class MarkdownDiffOutputGenerator : IFileOutputGenerator, IStringOutputGenerator
```
