## Propriedade ComparisonOptions.ExcludeTables

Obtenha e defina a opção que determina se as tabelas são excluídas da comparação. Esta opção não pode ser definida juntamente com a opção [`ExtractionArea`](../extractionarea/). O valor padrão é `false`.

```csharp
public bool ExcludeTables { get; set; }
```
