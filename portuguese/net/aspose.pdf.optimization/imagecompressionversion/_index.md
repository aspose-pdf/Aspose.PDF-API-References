## Enumeração ImageCompressionVersion

Descreve versões do algoritmo de compressão de imagem.


### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Standard | `0` | Algoritmo padrão. Valor padrão. |
| Fast | `2` | Algoritmo melhorado mais rápido que o padrão, mas aplicável nem todos os casos. |
| Mixed | `3` | Use o algoritmo rápido quando possível e o padrão para outros casos. Pode ser mais lento que "Fast", mas pode produzir melhor compressão. |
