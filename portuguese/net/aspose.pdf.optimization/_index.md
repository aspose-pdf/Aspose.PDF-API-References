O **Aspose.Pdf.Optimization** é um namespace para classes que gerenciam o processo de otimização de documentos.

## Classes

| Classe | Descrição |
| --- | --- |
| [ImageCompressionOptions](./imagecompressionoptions/) | A classe contém um conjunto de opções para compressão de imagem. |
| [OptimizationOptions](./optimizationoptions/) | Classe que descreve o algoritmo de otimização de documentos. A instância desta classe pode ser usada como parâmetro do método OptimizeResources(). |
## Enumeração

| Enumeração | Descrição |
| --- | --- |
| [ImageCompressionVersion](./imagecompressionversion/) | Descreve versões do algoritmo de compressão de imagem. |
| [ImageEncoding](./imageencoding/) | Tipos de codificação de imagem. |
