## Propriedade ImageCompressionOptions.ResizeImages

Se esta flag estiver definida como verdadeira e CompressImages for verdadeira, as imagens serão redimensionadas se a resolução da imagem for maior do que o parâmetro MaxResolution especificado.

```csharp
public bool ResizeImages { get; set; }
```
