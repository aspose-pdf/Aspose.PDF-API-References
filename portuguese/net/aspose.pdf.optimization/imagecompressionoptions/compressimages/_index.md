## Propriedade ImageCompressionOptions.CompressImages

Se esta flag estiver definida como verdadeira, as imagens serão comprimidas no documento. O nível de compressão é especificado com a propriedade ImageQuality.

```csharp
public bool CompressImages { get; set; }
```
