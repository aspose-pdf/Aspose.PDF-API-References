## Propriedade OptimizationOptions.RemoveUnusedObjects

Se esta flag estiver definida como verdadeira, todos os objetos do documento serão verificados e objetos não utilizados (ou seja, objetos que não têm nenhuma referência) serão removidos do documento.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
