## Propriedade OptimizationOptions.LinkDuplcateStreams

Se esta flag estiver definida como verdadeira, os fluxos de recursos serão analisados. Se fluxos duplicados forem encontrados (ou seja, se o conteúdo do fluxo for igual), então esses fluxos serão armazenados como um único objeto. Isso permite diminuir o tamanho do documento em alguns casos (por exemplo, quando o mesmo documento foi concatenado várias vezes).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
