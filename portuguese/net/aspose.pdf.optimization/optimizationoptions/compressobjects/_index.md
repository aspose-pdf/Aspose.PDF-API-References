## Propriedade OptimizationOptions.CompressObjects

Se esta flag estiver definida como `true`, os objetos Pdf serão compactados em Objest Streams e comprimidos para reduzir o tamanho do arquivo pdf.

```csharp
public bool CompressObjects { get; set; }
```
