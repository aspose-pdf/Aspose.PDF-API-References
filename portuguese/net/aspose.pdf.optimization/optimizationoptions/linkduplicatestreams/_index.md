## Propriedade OptimizationOptions.LinkDuplicateStreams

Se esta flag estiver definida como verdadeira, os streams de recurso serão analisados. Se streams duplicados forem encontrados (ou seja, se o conteúdo do stream for igual), então esses streams serão armazenados como um objeto. Isso permite diminuir o tamanho do documento em alguns casos (por exemplo, quando o mesmo documento foi concatenado várias vezes).

```csharp
public bool LinkDuplicateStreams { get; set; }
```
