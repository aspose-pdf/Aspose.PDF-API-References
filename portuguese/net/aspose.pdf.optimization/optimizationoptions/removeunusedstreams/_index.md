## Propriedade OptimizationOptions.RemoveUnusedStreams

Se esta flag estiver definida como verdadeira, cada recurso é verificado quanto ao seu uso. Se o recurso nunca for usado, então o recurso é removido. Isso pode diminuir o tamanho do documento, por exemplo, quando páginas foram extraídas do documento.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
