## Enumeração TextRenderingMode

O modo de renderização de texto, Tmode, determina se a exibição de texto deve causar contornos de glifos a serem traçados, preenchidos, usados como um limite de recorte ou alguma combinação dos três.

```csharp
public enum TextRenderingMode
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| FillText | `0` | Preencher texto. |
| StrokeText | `1` | Traçar texto. |
| FillThenStrokeText | `2` | Preencher, então traçar texto. |
| Invisible | `3` | Nem preencher nem traçar texto (invisível). |
| FillTextAndAddPathToClipping | `4` | Preencher texto e adicionar ao caminho para recorte (veja 9.3.6, "Modo de Renderização de Texto"). |
| StrokeTextAndAddPathToClipping | `5` | Traçar texto e adicionar ao caminho para recorte. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Preencher, então traçar texto e adicionar ao caminho para recorte. |
| AddPathToClipping | `7` | Adicionar texto ao caminho para recorte. |
