## Propriedade TextSearchOptions.SearchForTextRelatedGraphics

Obtém ou define o valor que permite a busca por gráficos relacionados ao texto (sublinhados, fundo etc.) durante a busca de texto. true - a busca por gráficos relacionados ao texto será realizada (valor padrão). false - elementos gráficos que podem estar presentes no documento de origem serão ignorados. Defina isso em caso de problemas de desempenho ou se não houver necessidade de lidar com sublinhados, fundo ou recortes.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
