## Propriedade TextSearchOptions.IgnoreShadowText

Obtém ou define a indicação de que fragmentos de texto representando a sombra do texto normal serão ignorados durante a busca. true - significa que o texto sombra não será encontrado (tente isso se a busca de texto retornar fragmentos duplicados em posições próximas) false - significa que o texto sombra será encontrado assim como o texto normal (valor padrão)

```csharp
public bool IgnoreShadowText { get; set; }
```
