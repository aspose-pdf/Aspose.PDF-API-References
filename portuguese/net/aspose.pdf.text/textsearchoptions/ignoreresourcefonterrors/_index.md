## Propriedade TextSearchOptions.IgnoreResourceFontErrors

Obtém ou define a indicação de que erros relacionados à ausência de fonte serão ignorados pelo absorvedor de texto (fragmento). true - significa que erros de ausência de fonte serão ignorados. Segmentos de texto que se referem a recursos incorretos serão pulados durante o processamento. false (padrão) - erro de ausência de fonte encerrará o processamento lançando uma exceção.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
