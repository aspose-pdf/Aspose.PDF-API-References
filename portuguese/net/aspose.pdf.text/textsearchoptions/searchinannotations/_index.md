## Propriedade TextSearchOptions.SearchInAnnotations

Obtém ou define o valor que permite a busca de texto em Anotações. true - o texto será buscado em Anotações. false - o texto em Anotações não será analisado pelo TextFragmentAbsorber.

public bool SearchInAnnotations { get; set; }
