## Propriedade TextSearchOptions.LogTextExtractionErrors

Obtém ou define a indicação de que erros de extração (decodificação) de texto serão registrados no absorvedor de texto (fragmento). true - significa que erros de extração (decodificação) de texto serão registrados. Isso pode diminuir o desempenho. false (padrão) - sem registro de erro.

```csharp
public bool LogTextExtractionErrors { get; set; }
```
