## Propriedade TextAbsorber.Errors

Lista de [`TextExtractionError`](../../textextractionerror/) objetos. Contém informações sobre erros encontrados durante a extração de texto. A busca por erros será realizada apenas se TextSearchOptions.LogTextExtractionErrors = true; e isso pode diminuir o desempenho.

```csharp
public List<TextExtractionError> Errors { get; }
```
