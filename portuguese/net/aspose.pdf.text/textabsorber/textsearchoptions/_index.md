## Propriedade TextAbsorber.TextSearchOptions

Obtém ou define opções de busca de texto.

```csharp
public virtual TextSearchOptions TextSearchOptions { get; set; }
```

## Observações

Permite definir um retângulo que delimita o texto extraído. Por padrão, o retângulo está vazio. Isso significa que os limites da página apenas definem a região de extração de texto.
