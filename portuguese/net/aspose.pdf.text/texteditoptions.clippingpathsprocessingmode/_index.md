## Enumeração TextEditOptions.ClippingPathsProcessingMode

Modos de processamento de caminhos de recorte

```csharp
public enum ClippingPathsProcessingMode
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| KeepIntact | `0` | Mantém os caminhos de recorte do layout da página original. (Padrão) |
| Expand | `1` | O caminho de recorte original será expandido no caso de o texto editado exigir mais espaço. |
| Remove | `2` | O caminho de recorte original será removido no caso de o texto editado exigir mais espaço. Atenção: Devido à interação entre os caminhos de recorte, a remoção deles pode levar a resultados inesperados no layout da página. |
