## Propriedade ParagraphAbsorberOptions.SectionUnbreakingHorizontalOverride

Obtém ou define o comprimento máximo da zona com 'nível de preenchimento zero' que não será tratada como uma quebra de seção horizontal real (como uma fração da largura / altura de uma página). O valor padrão é 0.005.

```csharp
public double SectionUnbreakingHorizontalOverride { get; set; }
```
