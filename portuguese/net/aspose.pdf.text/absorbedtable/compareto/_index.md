## Método AbsorbedTable.CompareTo

Compara o objeto AbsorbedTable atual com outro objeto AbsorbedTable e retorna um inteiro que indica se o objeto atual precede, segue ou ocorre na mesma posição na ordem de classificação que o outro objeto.

```csharp
public int CompareTo(AbsorbedTable other)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| other | AbsorbedTable | Um objeto AbsorbedTable para comparar com este objeto. |

### Valor de Retorno

Um valor que indica a ordem relativa dos objetos sendo comparados. O valor de retorno tem os seguintes significados: -1: Este objeto precede o outro objeto na ordem de classificação. 0: Este objeto ocorre na mesma posição na ordem de classificação que o outro objeto. 1: Este objeto segue o outro objeto na ordem de classificação.
