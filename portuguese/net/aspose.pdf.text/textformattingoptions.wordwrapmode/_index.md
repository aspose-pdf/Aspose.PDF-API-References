## Enumeração TextFormattingOptions.WordWrapMode

Define estratégias de quebra de palavras

```csharp
public enum WordWrapMode
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| NoWrap | `0` | Nenhuma quebra é realizada |
| DiscretionaryHyphenation | `1` | Hifenização discrecionária é realizada. Permite quebrar palavras no meio. |
| ByWords | `2` | A quebra de palavras envolve apenas palavras completas. Se a palavra completa não puder ser quebrada, tenta usar hifenização discrecionária |
| Undefined | `3` | WordWrapMode não definido. As estratégias de quebra da estrutura de nível superior (célula de tabela, parágrafo, etc.) serão usadas. |
