## Enumeração TextReplaceOptions.Scope

Escopo onde a operação de substituição de texto é aplicada REPLACE_FIRST por padrão Esta opção obsoleta foi mantida para compatibilidade. Ela afeta o PdfContentEditor e não tem efeito sobre o TextFragmentAbsorber.

```csharp
public enum Scope
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| REPLACE_FIRST | `0` | Substituir apenas a primeira ocorrência do texto em cada uma das páginas afetadas |
| REPLACE_ALL | `1` | Substituir todas as ocorrências de texto em todas as páginas afetadas |
