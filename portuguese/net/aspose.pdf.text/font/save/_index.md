## Método Font.Save

Salva a fonte no fluxo. Observe que a fonte é salva em um formato TTF intermediário destinado a ser usado apenas em uma cópia convertida do documento original. O arquivo de fonte não é destinado a ser usado fora do contexto do documento original.

```csharp
public void Save(Stream stream)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| stream | Stream | Fluxo para salvar a fonte. |
