## Propriedade TextFormattingOptions.HyphenSymbol

Obtém ou define o símbolo de hífen que é usado no processo de hifenização.

```csharp
public string HyphenSymbol { get; set; }
```

## Observações

Para eliminar a representação do hífen (com o procedimento de quebra ainda em vigor), defina uma string vazia string.Empty para HyphenSymbol.
