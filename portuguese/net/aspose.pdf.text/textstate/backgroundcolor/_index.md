## Propriedade TextState.BackgroundColor

Define a cor de fundo do texto.

```csharp
public virtual Color BackgroundColor { get; set; }
```

## Observações

Observe que o valor não é preservado como uma característica de texto dentro do documento. O getter da propriedade BackgroundColor funciona para um objeto caso tenha sido definido explicitamente anteriormente com o setter BackgroundColor para esse objeto. A propriedade é usada em tempo de execução no contexto do processo atual de geração/modificação.
