## Propriedade TextState.CoordinateOrigin

Obtém ou define a origem das coordenadas do texto. Se a origem das coordenadas for Descender, a coordenada Y do texto corresponde ao ponto mais baixo da fonte. Se a origem das coordenadas for BaseLine, a coordenada Y do texto corresponde à linha de base da fonte. O valor padrão é Descender. Se o valor de Descent da fonte for muito grande, o texto pode ser renderizado mais alto do que outras fontes. Nesse caso, a origem das coordenadas BaseLine pode ser selecionada para uma melhor renderização do texto.

public virtual CoordinateOrigin CoordinateOrigin { get; set; }
