Obtém ou define o alinhamento horizontal para o texto.

```csharp
public virtual HorizontalAlignment HorizontalAlignment { get; set; }
```

## Observações

HorizontalAlignment.None é igual a HorizontalAlignment.Left. Observe que a propriedade TextState.HorizontalAlignment funciona apenas em cenários de geração de novos documentos.
