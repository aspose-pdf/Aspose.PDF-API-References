## Propriedade TextState.LineSpacing

Obtém ou define o espaçamento entre linhas do texto.

```csharp
public virtual float LineSpacing { get; set; }
```

## Observações

Observe que o valor não é preservado como uma característica do texto dentro do documento. O getter da propriedade LineSpacing funciona para um objeto caso tenha sido definido explicitamente anteriormente com o setter de LineSpacing para esse objeto. A propriedade é usada pelo tempo de execução no contexto do processo atual de geração/modificação.
