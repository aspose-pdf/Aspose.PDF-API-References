## Propriedade TextExtractionOptions.ScaleFactor

Obtém ou define o fator que será aplicado para escalar o tamanho da fonte durante a extração em modo puro. Um valor menor leva a mais espaços no texto extraído. O valor padrão é 1 - sem escalonamento; Definir o valor como zero permite que o algoritmo escolha o escalonamento automaticamente.

public double ScaleFactor { get; set; }
