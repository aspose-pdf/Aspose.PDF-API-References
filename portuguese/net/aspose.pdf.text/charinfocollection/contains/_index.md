## Método CharInfoCollection.Contains

Determina se a coleção contém um valor específico.

```csharp
public bool Contains(CharInfo item)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| item | CharInfo | O objeto a ser localizado na coleção |

### Valor de Retorno

true se o item for encontrado na coleção; caso contrário, false.
