## Propriedade TextFragmentState.TabStops

Obtém tabulações para o texto.

```csharp
public TabStops TabStops { get; }
```

## Observações

Observe que a propriedade Tabstops funciona apenas em cenários de geração de novos documentos. As tabulações podem ser adicionadas durante a inicialização do [`TextFragment`](../../textfragment/). As tabulações devem ser construídas antes do texto.
