## Propriedade TextFragmentState.HorizontalAlignment

Obtém ou define o alinhamento horizontal para o texto.

```csharp
public override HorizontalAlignment HorizontalAlignment { get; set; }
```

## Observações

HorizontalAlignment.None é igual a HorizontalAlignment.Left. Observe que a propriedade TextFragmentState.VerticalAlignment funciona apenas em cenários de geração de novos documentos.
