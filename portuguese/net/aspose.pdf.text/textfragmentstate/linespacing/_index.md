## Propriedade TextFragmentState.LineSpacing

Obtém ou define o espaçamento entre linhas do texto.

```csharp
public override float LineSpacing { get; set; }
```

## Observações

Observe que o valor não é preservado como uma característica de texto dentro do documento. O getter da propriedade LineSpacing funciona para um objeto caso tenha sido definido explicitamente anteriormente com o setter LineSpacing para esses objetos. A propriedade é usada pelo tempo de execução no contexto do processo atual de geração/modificação.
