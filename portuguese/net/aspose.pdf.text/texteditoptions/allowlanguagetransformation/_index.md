## Propriedade TextEditOptions.AllowLanguageTransformation

Obtém ou define o valor que permite o uso de transformação de linguagem durante a adição ou edição de texto. true - a transformação de linguagem será aplicada se necessário (valor padrão). false - a transformação de linguagem NÃO será aplicada.

```csharp
public bool AllowLanguageTransformation { get; set; }
```
