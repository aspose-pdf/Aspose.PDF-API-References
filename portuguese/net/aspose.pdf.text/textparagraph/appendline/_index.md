## AppendLine(string, float) {#appendline_6}

Anexa linha de texto.

```csharp
public void AppendLine(string line, float lineSpacing)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| line | String | O texto da nova linha. |
| lineSpacing | Single | Espaçamento adicional (0.0 é o padrão e corresponde à altura padrão da linha de texto). O valor do espaçamento é adicionado ao espaçamento padrão da linha para a linha particular, então você pode especificar 12.0 para obter uma linha vazia DEPOIS de uma linha de texto desenhada com fonte de 12pt. |
