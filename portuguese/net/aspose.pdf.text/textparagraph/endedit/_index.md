## Método TextParagraph.EndEdit

Encerra a edição do TextParagraph.

```csharp
public void EndEdit()
```

## Observações

Melhora o desempenho da população do TextParagraph. Qualquer cálculo de layout é suspenso até que o método EndEdit seja invocado. Observe que a invocação do método não pode ser aninhada.
