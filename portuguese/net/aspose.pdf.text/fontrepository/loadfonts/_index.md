## Método FontRepository.LoadFonts

Carrega fontes instaladas no sistema e fontes Pdf padrão. Este método foi projetado para acelerar o processo de carregamento de fontes. Por padrão, as fontes são carregadas na primeira solicitação de qualquer fonte. O uso deste método carrega fontes do sistema e fontes Pdf padrão imediatamente antes de qualquer documento Pdf ser aberto.

public static void LoadFonts()
