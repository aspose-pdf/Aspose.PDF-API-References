## Indexador PDF3DCrossSectionArray

Obtém ou define o [`PDF3DCrossSection`](../../pdf3dcrosssection/) no índice especificado.

```csharp
public PDF3DCrossSection this[int index] { get; set; }
```

| Parâmetro | Descrição |
| --- | --- |
| index | O índice. |

### Valor de Retorno

Seção transversal.

### Exceções

| exceção | condição |
| --- | --- |
| IndexOutOfRangeException | Índice inválido: o índice deve estar na faixa [1..n] onde n é igual à contagem de seções transversais. |
