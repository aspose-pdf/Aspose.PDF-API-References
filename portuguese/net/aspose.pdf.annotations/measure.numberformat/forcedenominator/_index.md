## Propriedade Measure.NumberFormat.ForceDenominator

Se FractionDisplayment for ShowAsFraction, este valor determina se a fração pode ou não ser reduzida. Se o valor for verdadeiro, a fração pode não ser reduzida.

public bool ForceDenominator { get; set; }
