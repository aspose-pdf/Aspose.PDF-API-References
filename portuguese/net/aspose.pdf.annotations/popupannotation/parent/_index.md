## Propriedade PopupAnnotation.Parent

Obtém ou define a anotação pai com a qual esta anotação pop-up deve ser associada. Se esta entrada estiver presente, as entradas Contents, M, C e T da anotação pai devem substituir as da própria anotação pop-up.

```csharp
public Annotation Parent { get; set; }
```
