## Campo SubmitFormAction.EmbedForm

Se definido, a entrada F do FDF enviado deverá ser uma especificação de arquivo contendo um fluxo de arquivo incorporado representando o arquivo PDF do qual o FDF está sendo enviado.

```csharp
public const int EmbedForm;
```
