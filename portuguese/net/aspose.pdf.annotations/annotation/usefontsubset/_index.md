## Propriedade Annotation.UseFontSubset

Se esta propriedade estiver definida como verdadeira, as fontes serão adicionadas ao documento como subconjuntos. O valor padrão é verdadeiro.

```csharp
public static bool UseFontSubset { get; set; }
```
