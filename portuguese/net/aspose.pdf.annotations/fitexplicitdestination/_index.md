## Classe FitExplicitDestination

Representa um destino explícito que exibe a página com seu conteúdo ampliado o suficiente para caber toda a página na janela, tanto horizontal quanto verticalmente. Se os fatores de ampliação horizontal e vertical necessários forem diferentes, use o menor dos dois, centralizando a página na janela na outra dimensão.

```csharp
public sealed class FitExplicitDestination : ExplicitDestination
```
