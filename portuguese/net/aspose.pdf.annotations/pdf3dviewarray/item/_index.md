## Indexador PDF3DViewArray

Obtém ou define o [`PDF3DView`](../../pdf3dview/) para visualizar o array no índice especificado.

```csharp
public PDF3DView this[int index] { get; set; }
```

| Parâmetro | Descrição |
| --- | --- |
| index | O índice. |

### Valor de Retorno

PDF3DView.

### Exceções

| exceção | condição |
| --- | --- |
| IndexOutOfRangeException | Índice inválido: o índice deve estar na faixa [1..n] onde n é igual ao número de visualizações. |
