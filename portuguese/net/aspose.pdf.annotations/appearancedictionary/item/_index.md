## Indexador AppearanceDictionary

Representa uma forma conveniente de obter fluxos de aparência.

```csharp
public XForm this[string key] { get; set; }
```

| Parâmetro | Descrição |
| --- | --- |
| key | Representa o caminho para o fluxo de aparência. Se o dicionário de aparência tiver subdicionários, então o caminho deve conter 2 partes ([`Keys`](../keys/)), caso contrário, o caminho tem apenas uma parte. |

### Valor de Retorno

Objeto XForm (fluxo de aparência) que corresponde à chave dada.
