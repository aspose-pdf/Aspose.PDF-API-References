## Método XYZExplicitDestination.CreateDestination

Crie um destino para a localização especificada da página considerando a rotação da página, se necessário.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| page | Page | Página de destino. |
| left | Double | Posição à esquerda na página. |
| top | Double | Posição no topo da página. |
| zoom | Double | Fator de zoom (0 para padrão). |
| considerRotation | Boolean | Se verdadeiro, a posição será recalculada de acordo com a rotação da página. |

### Valor de Retorno

Objeto de destino.
