## Enumeração PrinterMarksKind

Especifica os tipos de marcas de impressora a serem adicionadas a um documento.

```csharp
[Flags]
public enum PrinterMarksKind
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Nenhum | `0` | Especifica que nenhuma marca de impressora deve ser adicionada. |
| MarcasDeCorte | `1` | Especifica que marcas de corte devem ser adicionadas. |
| MarcasDeSangria | `2` | Especifica que marcas de sangria devem ser adicionadas. |
| MarcasDeRegistro | `4` | Especifica que marcas de registro devem ser adicionadas. |
| BarrasDeCor | `8` | Especifica que barras de cor devem ser adicionadas. |
| InformaçõesDaPágina | `10` | Especifica que informações da página devem ser adicionadas. |
| Todas | `1F` | Especifica que todos os tipos de marcas de impressora devem ser adicionados. |

## Observações

Esta enumeração possui um atributo FlagsAttribute que permite uma combinação bit a bit de seus valores de membro.
