## Propriedade AnnotationElement.AlternateDescriptions

Obtém ou define as Descrições Alternativas para a anotação. Texto que deve ser exibido para a anotação ou, se este tipo de anotação não exibir texto, uma descrição alternativa do conteúdo da anotação em forma legível por humanos.

```csharp
public string AlternateDescriptions { get; set; }
```
