## Campo StructureTypeStandard.Formula

(Fórmula) Uma fórmula matemática.

Este tipo de estrutura é útil apenas para identificar um elemento de conteúdo inteiro como uma fórmula. Nenhum tipo de estrutura padrão é definido para identificar componentes individuais dentro da fórmula. Do ponto de vista de formatação, a fórmula deve ser tratada de maneira semelhante a uma figura (tipo de estrutura Figura).

```csharp
public static readonly StructureTypeStandard Formula;
```
