## Campo StructureTypeStandard.BibEntry

(Entrada de bibliografia) Uma referência identificando a fonte externa de algum conteúdo citado. Pode conter um rótulo (estrutura do tipo Lbl) como filho.

Embora uma entrada de bibliografia provavelmente inclua partes componentes identificando o autor, obra, editora do conteúdo citado, nenhum tipo de estrutura padrão é definido neste nível de detalhe.

```csharp
public static readonly StructureTypeStandard BibEntry;
```
