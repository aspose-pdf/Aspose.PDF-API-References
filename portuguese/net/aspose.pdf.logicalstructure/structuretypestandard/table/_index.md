## Campo StructureTypeStandard.Table

(Tabela) Um layout bidimensional de células de dados retangulares, possivelmente tendo uma subestrutura complexa. Contém uma ou mais linhas de tabela (tipo de estrutura TR) como filhos; ou um cabeçalho de tabela opcional (tipo de estrutura THead) seguido por um ou mais elementos de corpo de tabela (tipo de estrutura TBody) e um rodapé de tabela opcional (tipo de estrutura TFoot). Além disso, uma tabela pode ter uma legenda (tipo de estrutura Caption) como seu primeiro ou último filho.

```csharp
public static readonly StructureTypeStandard Table;
```
