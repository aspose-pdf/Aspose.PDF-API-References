## Campo StructureTypeStandard.NonStruct

(Elementos não estruturais) Um elemento de agrupamento que não possui significado estrutural inerente; serve exclusivamente para fins de agrupamento. Este tipo de elemento difere de uma divisão (tipo de estrutura Div) na medida em que não deve ser interpretado ou exportado para outros formatos de documento; no entanto, seus descendentes devem ser processados normalmente.

```csharp
public static readonly StructureTypeStandard NonStruct;
```
