## Campo StructureTypeStandard.TBody

(Grupo de linhas do corpo da tabela; PDF 1.5) Um grupo de linhas que constituem a parte principal do corpo de uma tabela. Se a tabela for dividida em várias páginas, a área do corpo pode ser separada em um limite de linha. Uma tabela pode ter múltiplos elementos TBody para permitir o desenho de uma borda ou fundo para um conjunto de linhas.

```csharp
public static readonly StructureTypeStandard TBody;
```
