## Método ElementList.Item

Recupera um elemento no índice dado.

```csharp
public abstract Element Item(int index)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | O índice na lista de elementos. |

### Valor de Retorno

O Elemento com o índice especificado na coleção. Se *index* for maior ou igual ao número de elementos na lista, isso retorna null.
