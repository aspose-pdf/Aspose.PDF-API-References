## Indexador ElementList

Obtém um elemento no índice dado.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Parâmetro | Descrição |
| --- | --- |
| index | O índice na lista de elementos. |

### Valor de Retorno

O Elemento com o índice especificado na coleção. Se o índice for maior ou igual ao número de elementos na lista, isso retorna null.
