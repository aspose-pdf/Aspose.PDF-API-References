## Campo AttributeName.TextAlign_Justify

Atributo TextAlign: Justify - Alinhado com as bordas inicial e final, com espaçamento interno dentro de cada linha expandido, se necessário, para alcançar tal alinhamento. A última (ou única) linha deve ser alinhada apenas com a borda inicial.

```csharp
public static readonly AttributeName TextAlign_Justify;
```
