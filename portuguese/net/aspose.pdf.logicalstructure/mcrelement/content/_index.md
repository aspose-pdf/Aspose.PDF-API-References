## Propriedade MCRElement.Content

Obtém o conteúdo de texto para o elemento de referência de conteúdo marcado.

Para o objeto recém-criado que implementa [`ITextElement`](../../itextelement/). É nulo em outros casos.

```csharp
public string Content { get; }
```

### Valor da Propriedade

Conteúdo de texto para o elemento de referência de conteúdo marcado.
