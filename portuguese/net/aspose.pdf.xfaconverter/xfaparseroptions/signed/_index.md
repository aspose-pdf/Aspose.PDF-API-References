## Propriedade XfaParserOptions.Signed

Se esta propriedade for verdadeira, o documento será convertido usando o fluxo de formulário xfa (se existir). Se for falsa, o fluxo de formulário xfa será ignorado. Esta propriedade foi introduzida porque não está claro como calcular a soma de verificação que é usada para verificar a assinatura.

```csharp
public bool Signed { get; set; }
```
