## Propriedade XfaParserOptions.EmulateRequierdGroups

Se esta propriedade for verdadeira, retângulos vermelhos adicionais serão desenhados para grupos "excluídos" Xfa obrigatórios. Esta propriedade foi introduzida devido à ausência de análogos de grupos excluídos durante a conversão da representação Xfa de formulários para o padrão. É falsa por padrão.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
