O **Aspose.Pdf.Vector** é um namespace raiz para operações gráficas.

## Classes

| Classe | Descrição |
| --- | --- |
| [GraphicElement](./graphicelement/) | Representa a classe base para objetos gráficos na página. |
| [GraphicElementCollection](./graphicelementcollection/) | Representa a coleção de [`GraphicElement`](../aspose.pdf.vector/graphicelement/). |
| [GraphicsAbsorber](./graphicsabsorber/) | Representa um objeto absorvedor de elementos gráficos. Realiza busca gráfica e fornece acesso aos resultados da busca através da coleção [`Elements`](../aspose.pdf.vector/graphicsabsorber/elements/). |
| [GraphicState](./graphicstate/) | Representa o estado gráfico do atual [`GraphicElement`](../aspose.pdf.vector/graphicelement/). |
| [SubPath](./subpath/) | Representa um objeto gráfico vetorial na página. Basicamente, objetos gráficos vetoriais são representados por dois grupos de SubPaths. Um deles é representado por um conjunto de linhas e curvas. Outros são apresentados como retângulos e podem às vezes ser confundidos. Normalmente, é uma área retangular que tem uma cor, mas muito frequentemente esse retângulo é colocado no início da página e define todo o espaço da página em branco. Assim, você obtém o SubPath, mas visualmente você só vê o texto na página. |
| [SubPathGroup](./subpathgroup/) | Representa uma classe para um grupo de contêineres de elementos gráficos. Objetos da classe têm uma caixa delimitadora para contabilizar o tamanho do grupo. |
| [SvgExtractionOptions](./svgextractionoptions/) | Representa uma classe de opções para extrair gráficos vetoriais da página do documento pdf. |
| [SvgExtractor](./svgextractor/) | Representa uma classe para extração de imagens SVG da página. |
| [XFormPlacement](./xformplacement/) | Representa a colocação do XForm. Se o XForm for exibido na página mais de 1 vez, todas as XformPlacements associadas a este XForm terão elementos gráficos comuns, mas estados gráficos diferentes. |
