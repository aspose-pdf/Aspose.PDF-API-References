## Propriedade SvgExtractionOptions.GroupStrength

Obtém e define uma opção A força de agrupamento de subcaminhos em imagens. Permite configurar o grau de agrupamento de subcaminhos. O intervalo de valores é de 0 a 1. Um valor de 0 corresponde à opção [`ExtractEverySubPathToSvg`](../extracteverysubpathtosvg/) sendo habilitada. Um valor de 1 criará uma única imagem para todos os caminhos vetoriais na página. A opção tem efeito quando [`AutoGrouping`](../autogrouping/) é falso. O valor padrão é `0.8`.

public double GroupStrength { get; set; }
