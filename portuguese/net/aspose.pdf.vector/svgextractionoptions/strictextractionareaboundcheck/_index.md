## Propriedade SvgExtractionOptions.StrictExtractionAreaBoundCheck

Obtém e define uma opção para verificar estritamente se os subcaminhos estão dentro do retângulo especificado em [`ExtractionAreaBound`](../extractionareabound/). Se definido como falso, os subcaminhos que não estão completamente incluídos em [`ExtractionAreaBound`](../extractionareabound/) serão extraídos. O valor padrão é `True`.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
