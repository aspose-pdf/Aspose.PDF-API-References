## Propriedade SvgExtractionOptions.UnpackPageContentXForm

Obtém e define uma flag que determina se o XFrom encontrado nas páginas deve ser desempacotado ou não. Elementos XFrom podem acabar em diferentes arquivos SVG. Apenas XForms que são renderizados por declarações Do do conteúdo da página são desempacotados. XForms aninhados não são desempacotados.

```csharp
public bool UnpackPageContentXForm { get; set; }
```
