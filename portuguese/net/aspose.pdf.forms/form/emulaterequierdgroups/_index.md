## Propriedade Form.EmulateRequierdGroups

Se esta propriedade for verdadeira, retângulos de contorno vermelhos adicionais serão desenhados para os contêineres de elementos exclGroup Xfa obrigatórios. Esta propriedade foi introduzida devido à ausência de análogos para o exclGroup durante a conversão da representação Xfa de formulários para o padrão. É falsa por padrão.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
