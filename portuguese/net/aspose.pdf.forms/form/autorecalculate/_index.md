## Propriedade Form.AutoRecalculate

Se definido, todos os campos do formulário serão recalculados quando qualquer campo for alterado. O valor padrão é verdadeiro. Defina como falso para aumentar o desempenho ao preencher o formulário com uma grande quantidade de campos calculados.

```csharp
public bool AutoRecalculate { get; set; }
```
