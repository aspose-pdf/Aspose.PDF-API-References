## Método Form.RemoveFieldAppearance

Remove a aparência do campo no índice especificado. Se apenas uma aparência de filho restar, o método a incorpora ao campo.

```csharp
public void RemoveFieldAppearance(Field field, int appearanceIndex)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| field | Field | Campo com aparências. |
| appearanceIndex | Int32 | Índice das aparências. |
