## Propriedade Form.IgnoreNeedsRendering

Se esta propriedade for verdadeira, o valor da chave NeedsRendering será ignorado durante a conversão do formulário XFA para o formulário Padrão. É falso por padrão.

```csharp
public bool IgnoreNeedsRendering { get; set; }
```
