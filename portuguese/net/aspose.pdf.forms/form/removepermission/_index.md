## Propriedade Form.RemovePermission

Se esta propriedade for verdadeira, o dicionário "Perms" será removido do documento pdf após a conversão de documentos dinâmicos para padrão. O dicionário "Perms" pode conter regras que perturbam a exibição da seleção de campos obrigatórios no Adobe Acrobat Reader. É falso por padrão.

```csharp
public bool RemovePermission { get; set; }
```
