## Propriedade BarcodeField.XSymHeight

Obtém a distância vertical entre dois módulos de código de barras, medida em pixels. A razão XSymHeight/XSymWidth deve ser um valor inteiro. Para PDF417, a faixa de razão aceitável é de 1 a 4. Para QRCode e DataMatrix, essa razão deve ser sempre 1

public int XSymHeight { get; }
