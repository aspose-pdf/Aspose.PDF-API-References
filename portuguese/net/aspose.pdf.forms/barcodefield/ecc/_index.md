## Propriedade BarcodeField.ECC

Obtém um valor inteiro representando o coeficiente de correção de erro. Para PDF417, deve ser de 0 a 8. Para QRCode, deve ser de 0 a 3 (0 para 'L', 1 para 'M', 2 para 'Q' e 3 para 'H').

public int ECC { get; }
