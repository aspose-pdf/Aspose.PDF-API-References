## Propriedade RadioButtonField.NoToggleToOff

Obtém ou define a flag que permite que o botão de opção não tenha valor selecionado. Se `true`, exatamente um botão de opção deve estar selecionado o tempo todo; selecionar o botão atualmente selecionado não tem efeito. Se `false`, clicar no botão selecionado o desmarca, deixando nenhum botão selecionado.

```csharp
public bool NoToggleToOff { get; set; }
```

## Observações

Alguns leitores de PDF (incluindo o Adobe Acrobat) podem ignorar o estado `false` da flag.
