## Propriedade Signature.DefaultSignatureLength

Obtém ou define o comprimento padrão para os dados da assinatura em bytes.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Observações

Esta é uma estimativa do comprimento da assinatura em bytes. Usado para assinar via [`CustomSignHash`](../customsignhash/) se o parâmetro [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) estiver definido. O valor padrão é 3000.
