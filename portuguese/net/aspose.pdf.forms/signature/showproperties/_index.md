## Propriedade Signature.ShowProperties

Forçar a mostrar/ocultar propriedades da assinatura. No caso de ShowProperties ser verdadeiro, o campo de assinatura tem um formato predefinido de aparência (strings para representar): ------------------------------------------- Digitally signed by {certificate subject} Date: {signature.Date} Reason: {signature.Reason} Location: {signature.Location} ------------------------------------------- onde {X} é um espaço reservado para o valor X. Além disso, a assinatura pode ter uma imagem, nesse caso as strings listadas são colocadas sobre a imagem. ShowProperties é verdadeiro por padrão.

```csharp
public bool ShowProperties { get; set; }
```
