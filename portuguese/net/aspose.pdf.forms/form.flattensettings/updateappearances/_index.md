## Propriedade Form.FlattenSettings.UpdateAppearances

Se definida, todas as aparências de campo serão regeneradas antes do achatamento. Esta opção pode ajudar se o campo estiver incorretamente achatado. Esta opção pode diminuir o desempenho. Por padrão, definida como falsa.

```csharp
public bool UpdateAppearances { get; set; }
```
