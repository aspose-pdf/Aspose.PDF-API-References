## Propriedade Field.IsSharedField

Propriedade para suporte ao Generator. Usado quando o campo é adicionado ao cabeçalho ou rodapé. Se verdadeiro, este campo será criado uma vez e sua aparência será visível em todas as páginas do documento. Se falso, um campo separado será criado para cada página do documento.

```csharp
public bool IsSharedField { get; set; }
```
