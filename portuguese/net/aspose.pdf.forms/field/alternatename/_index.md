## Propriedade Field.AlternateName

Obtém ou define o nome alternativo do campo (Um nome alternativo do campo que deve ser usado no lugar do nome real do campo sempre que o campo for identificado na interface do usuário). O nome alternativo é usado como dica de ferramenta do campo no Adobe Acrobat.

public string AlternateName { get; set; }
