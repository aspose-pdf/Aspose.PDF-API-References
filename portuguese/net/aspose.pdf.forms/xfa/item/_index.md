## Indexador XFA

Obtém ou define o valor do nó de dados de acordo com *caminho*.

```csharp
public string this[string path] { get; set; }
```

| Parâmetro | Descrição |
| --- | --- |
| caminho | Caminho do nó de dados, por exemplo, form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Certifique-se de incluir índices mesmo que os dados contenham apenas ocorrências únicas de cada nó, ou seja, escreva node1[0].node2[0]... em vez de node1.node2... |

### Valor de Retorno

Valor do nó de dados.
