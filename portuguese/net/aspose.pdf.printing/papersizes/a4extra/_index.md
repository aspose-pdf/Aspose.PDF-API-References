## Campo PaperSizes.A4Extra

Papel A4 extra (236 mm por 322 mm).

```csharp
public static readonly PaperSize A4Extra;
```

## Observações

Este valor é específico para o driver PostScript e é usado apenas por impressoras Linotronic para ajudar a economizar papel.
