O **namespace Aspose.Pdf.Printing** fornece classes e funcionalidades para lidar com operações de impressão de PDF, incluindo configurações de impressora, configurações de página e extensões para gerenciar configurações relacionadas à impressão.

## Classes

| Classe | Descrição |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Fornece dados para o evento [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Especifica configurações que se aplicam a uma única página impressa. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Representa métodos de extensão para [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Especifica o tamanho de uma folha de papel. |
| [PaperSizeExtensions](./papersizeextensions/) | Representa métodos de extensão para [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Representa os tamanhos de papel padrão. |
| [PaperSource](./papersource/) | Especifica a bandeja de papel da qual a impressora obtém papel. |
| [PaperSourceExtensions](./papersourceextensions/) | Representa métodos de extensão para [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Fornece um conjunto de instâncias pré-definidas de [`PaperSource`](../aspose.pdf.printing/papersource/) representando fontes de papel comuns. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Fornece dados para o evento [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Representa a resolução suportada por uma impressora. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Representa métodos de extensão para [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Especifica informações sobre como um documento é impresso, incluindo a impressora que o imprime. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Representa métodos de extensão para [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Fornece dados para os eventos [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) e [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) da classe [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
## Enumeração

| Enumeração | Descrição |
| --- | --- |
| [Duplex](./duplex/) | Especifica a configuração de duplex da impressora. |

| [PrinterResolutionKind](./printerresolutionkind/) | Especifica uma resolução de impressora. |
| [PrintRange](./printrange/) | Especifica a opção que designa a parte do documento a ser impressa. |
