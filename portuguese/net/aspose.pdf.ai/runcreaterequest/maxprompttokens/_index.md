## Propriedade RunCreateRequest.MaxPromptTokens

Obtém ou define o número máximo de tokens de prompt que podem ser usados ao longo da execução. A execução fará o melhor esforço para usar apenas o número de tokens de prompt especificado, em várias rodadas da execução. Se a execução exceder o número de tokens de prompt especificado, a execução terminará com o status incompleto. Veja incomplete_details para mais informações.

```csharp
public int? MaxPromptTokens { get; set; }
```
