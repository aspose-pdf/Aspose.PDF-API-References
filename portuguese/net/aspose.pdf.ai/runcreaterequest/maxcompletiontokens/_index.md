## Propriedade RunCreateRequest.MaxCompletionTokens

Obtém ou define o número máximo de tokens de conclusão que podem ser usados ao longo da execução. A execução fará o possível para usar apenas o número de tokens de conclusão especificado, em várias etapas da execução. Se a execução exceder o número de tokens de conclusão especificado, a execução terminará com o status incompleto. Veja incomplete_details para mais informações.

```csharp
public int? MaxCompletionTokens { get; set; }
```
