## Propriedade CompletionCreateRequest.Stream

Obtém ou define se deve usar streaming. Se definido, deltas de mensagens parciais serão enviados, como no ChatGPT. Tokens serão enviados como eventos enviados pelo servidor apenas de dados à medida que se tornam disponíveis, com o stream terminado por uma mensagem de dados: [DONE].

public bool? Stream { get; set; }
