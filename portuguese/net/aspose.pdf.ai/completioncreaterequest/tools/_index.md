## Propriedade CompletionCreateRequest.Tools

Obtém ou define uma lista de ferramentas que o modelo pode chamar. Atualmente, apenas funções são suportadas como uma ferramenta. Use isso para fornecer uma lista de funções para as quais o modelo pode gerar entradas JSON. Um máximo de 128 funções é suportado.

```csharp
public List<Tool> Tools { get; set; }
```
