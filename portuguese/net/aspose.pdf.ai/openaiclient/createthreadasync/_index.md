## Método OpenAIClient.CreateThreadAsync

Cria um novo thread de forma assíncrona.

```csharp
public Task<ThreadResponse> CreateThreadAsync(ThreadCreateRequest threadCreateRequest, 
    CancellationToken? cancellationToken = default)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| threadCreateRequest | ThreadCreateRequest | O objeto de solicitação contendo detalhes para criar o thread. |
| cancellationToken | Nullable`1 | Um token para cancelar a operação. |

### Valor de Retorno

Uma tarefa que representa a operação assíncrona. O resultado da tarefa contém a resposta da criação do thread.
