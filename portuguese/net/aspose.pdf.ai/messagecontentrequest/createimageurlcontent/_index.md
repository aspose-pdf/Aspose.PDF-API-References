## Método MessageContentRequest.CreateImageUrlContent

Cria um conteúdo de URL de imagem para uma mensagem.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| imageUrl | String | A URL da imagem. |
| detail | String | O detalhe da URL da imagem. Especifica o nível de detalhe da imagem se especificado pelo usuário. "low" usa menos tokens, você pode optar por alta resolução usando "high". O padrão é "auto". |

### Valor de Retorno

Uma nova instância de MessageContentRequest com conteúdo de ImageUrl.
