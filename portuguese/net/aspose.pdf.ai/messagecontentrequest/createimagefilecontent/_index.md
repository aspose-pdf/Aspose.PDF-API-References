## Método MessageContentRequest.CreateImageFileContent

Cria um conteúdo de arquivo de imagem para uma mensagem.

```csharp
public static MessageContentRequest CreateImageFileContent(string fileId, string detail = "auto")
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| fileId | String | O ID do arquivo de imagem. |
| detail | String | O detalhe do arquivo de imagem. Especifica o nível de detalhe da imagem se especificado pelo usuário. "low" usa menos tokens, você pode optar por alta resolução usando "high". O padrão é "auto". |

### Valor de Retorno

Uma nova instância de MessageContentRequest com conteúdo de ImageFile.
