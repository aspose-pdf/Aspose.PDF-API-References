## Propriedade CreateChatCompletionChunkResponse.Usage

Obtém ou define um campo opcional que estará presente apenas quando você definir stream_options: {"include_usage": true} em sua solicitação. Quando presente, contém um valor nulo, exceto para o último chunk, que contém as estatísticas de uso de tokens para toda a solicitação.

public Usage Usage { get; set; }
