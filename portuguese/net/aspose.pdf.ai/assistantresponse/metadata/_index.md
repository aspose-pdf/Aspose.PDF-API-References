## Propriedade AssistantResponse.Metadata

Obtém ou define um conjunto de 16 pares chave-valor que podem ser anexados a um objeto. Isso pode ser útil para armazenar informações adicionais sobre o objeto em um formato estruturado. As chaves podem ter no máximo 64 caracteres de comprimento e os valores podem ter no máximo 512 caracteres de comprimento.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
