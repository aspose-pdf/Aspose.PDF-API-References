## Propriedade CompletionResponse.SystemFingerprint

Obtém ou define a impressão digital que representa a configuração do backend com a qual o modelo é executado. Pode ser usada em conjunto com o parâmetro de solicitação seed para entender quando alterações no backend foram feitas que podem impactar o determinismo.

public string SystemFingerprint { get; set; }
