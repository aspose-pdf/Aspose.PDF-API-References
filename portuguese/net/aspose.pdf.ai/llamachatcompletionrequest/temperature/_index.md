## Propriedade LlamaChatCompletionRequest.Temperature

Define ou obtém a temperatura de amostragem a ser usada, entre 0 e 2. Valores mais altos, como 0.8, tornarão a saída mais aleatória, enquanto valores mais baixos, como 0.2, a tornarão mais focada e determinística. O valor padrão é 1.

```csharp
public double? Temperature { get; set; }
```
