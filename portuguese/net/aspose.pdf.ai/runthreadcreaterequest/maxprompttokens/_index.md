## Propriedade RunThreadCreateRequest.MaxPromptTokens

Obtém ou define o número máximo de tokens de prompt que podem ser usados durante a execução. A execução fará o possível para usar apenas o número de tokens de prompt especificado, em várias rodadas da execução. Se a execução exceder o número de tokens de prompt especificado, a execução terminará com o status incompleto. Veja incomplete_details para mais informações.

```csharp
public int? MaxPromptTokens { get; set; }
```
