## Propriedade RunThreadCreateRequest.TopP

Obtém ou define um valor que é uma alternativa à amostragem com temperatura, chamada amostragem de núcleo, onde o modelo considera os resultados dos tokens com massa de probabilidade top_p. Assim, 0.1 significa que apenas os tokens que compõem os 10% superiores da massa de probabilidade são considerados. Geralmente, recomendamos alterar isso ou a temperatura, mas não ambos.

```csharp
public double? TopP { get; set; }
```
