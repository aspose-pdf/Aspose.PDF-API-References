## Namespaces

| Namespace | Descrição |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | O **Aspose.Pdf** é um namespace raiz para todas as classes da biblioteca Aspose.Pdf que estão diretamente nele, como **Document**, ou indiretamente através de vários subnamespaces. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | O namespace **Aspose.Pdf.AI** fornece classes para funcionalidades de IA, incluindo clientes de API e assistentes inteligentes. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | O namespace **Aspose.Pdf.Annotations** fornece classes para trabalhar com vários tipos de ações, destinos e outras características do documento que tradicionalmente são chamadas de interativas, permitindo que o usuário se comunique com ele. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | O namespace **Aspose.Pdf.Comparison** fornece classes que são usadas para comparação de documentos. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | O **Aspose.Pdf.DataEditor** contém ferramentas para editar dados dentro de um documento. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | O namespace **Aspose.Pdf.Devices** fornece classes que são usadas para representar documentos como imagem(ns) ou texto simples. Assim, o documento pode ser enviado para dispositivos textuais ou gráficos variados, o que significa que queremos obter sua representação textual ou gráfica. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | O namespace **Aspose.Pdf.Drawing** fornece classes que descrevem primitivas gráficas como arco, linha, círculo etc. Objetos correspondentes podem ser usados para criar novos documentos. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | O namespace **Aspose.Pdf.Facades** fornece classes que originalmente vieram do Aspose.Pdf.Kit. Essas classes são usadas para manipular documentos realizando operações como concatenar, carimbar, assinar, anotar etc., mas em um nível alto, sem acesso à estrutura interna de um documento. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | O namespace **Aspose.Pdf.Forms** possui classes que descrevem formulários (padrão, estático, dinâmico) e vários tipos de campos como caixa de texto, caixa de lista, botão de opção etc. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | O **Aspose.Pdf.Tagged.LogicalStructure** é um namespace para classes de Estrutura Lógica de PDF. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | O namespace **Aspose.Pdf.Multithreading** fornece classes que são destinadas a serem usadas em cenários multithread. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | O **Aspose.Pdf.Operators** é um namespace para implementações de Operadores. Essas classes descrevem operadores usados no conteúdo das páginas PDF. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | O **Aspose.Pdf.Optimization** é um namespace para classes que gerenciam o processo de otimização de documentos. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | O **Aspose.Pdf.Plugins** é um namespace raiz para todas as classes do Aspose.Pdf.Plugins que estão diretamente nele, como **PdfOrganizer**, ou indiretamente através de vários subnamespaces. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | O namespace **Aspose.Pdf.Printing** fornece classes e funcionalidades para lidar com operações de impressão de PDF, incluindo configurações de impressora, configurações de página e extensões para gerenciar configurações relacionadas à impressão. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | O **Aspose.Pdf.Sanitization** é um namespace para operações de sanitização. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | O namespace **Aspose.Pdf.Security** contém classes usadas para criptografia e assinatura digital. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | O namespace **Aspose.Pdf.Structure** fornece classes que ajudam a trabalhar com a estrutura lógica de um documento, que apresenta a organização do documento em capítulos e seções ou a identificação de elementos especiais, como figuras, tabelas e notas de rodapé. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | O **Aspose.Pdf.Tagged** é um namespace para classes que suportam documentos PDF Marcados. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | O namespace **Aspose.Pdf.Text** fornece classes que permitem extrair texto, adicionar texto, manipular texto existente de um documento. Também contém classes que permitem extrair, substituir, substituir fontes de um documento. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | O **Aspose.Pdf.Vector** é um namespace raiz para operações gráficas. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | O namespace **Aspose.Pdf.XfaConverter** fornece classes que permitem converter XFA. |
