## Classe CompromiseCheckResult

Representa uma classe para verificar assinaturas digitais de documentos quanto a comprometimento.

```csharp
public sealed class CompromiseCheckResult
```

## Propriedades

| Nome | Descrição |
| --- | --- |
| [HasCompromisedSignatures](../../aspose.pdf.signatures/compromisecheckresult/hascompromisedsignatures/) { get; } | Indica se há assinaturas digitais comprometidas no documento. Retorna verdadeiro se pelo menos uma assinatura estiver comprometida; caso contrário, falso. |
| [SignaturesCoverage](../../aspose.pdf.signatures/compromisecheckresult/signaturescoverage/) { get; } | Obtém o estado de cobertura das assinaturas digitais em um documento. Se for igual a Indefinido, então uma das assinaturas está comprometida. |
