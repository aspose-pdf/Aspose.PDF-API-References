O **namespace Aspose.Pdf.Security** contém classes usadas para criptografia e assinatura digital.

## Classes

| Classe | Descrição |
| --- | --- |
| [DsaAlgorithmInfo](./dsaalgorithminfo/) | Representa uma classe para as informações sobre o algoritmo de assinatura DSA. |
| [EcdsaAlgorithmInfo](./ecdsaalgorithminfo/) | Representa uma classe para as informações sobre o algoritmo de assinatura ECDSA. |
| [KeyedSignatureAlgorithmInfo](./keyedsignaturealgorithminfo/) | Representa uma classe para informações sobre um algoritmo de assinatura com chave. |
| [RsaAlgorithmInfo](./rsaalgorithminfo/) | Representa uma classe para as informações sobre o algoritmo de assinatura RSA. |
| [SignatureAlgorithmInfo](./signaturealgorithminfo/) | Representa uma classe para informações sobre um algoritmo de assinatura, incluindo seu tipo, padrão criptográfico e algoritmo de hash de digestão. |
| [SignatureLengthMismatchException](./signaturelengthmismatchexception/) | Representa erros que ocorrem durante a assinatura de PDF. Ocorre se [`SignHash`](../aspose.pdf.forms/signhash/) for usado para assinar um documento e o comprimento real da assinatura for maior do que o especificado na opção [`DefaultSignatureLength`](../aspose.pdf.forms/signature/defaultsignaturelength/). |
| [TimestampAlgorithmInfo](./timestampalgorithminfo/) | Representa uma classe para as informações sobre o algoritmo de assinatura de timestamp. |
| [UnknownSignatureAlgorithmInfo](./unknownsignaturealgorithminfo/) | Representa uma classe para as informações sobre algoritmos de assinatura desconhecidos. |
| [ValidationOptions](./validationoptions/) | Representa opções para validar uma assinatura digital em um documento PDF. |
| [ValidationResult](./validationresult/) | Representa o resultado de um processo de validação para um certificado. |
