## Campo SignatureAlgorithmInfo.DigestHashAlgorithm

Obtém o algoritmo de hash de digestão usado para a assinatura. Para um carimbo de data/hora, este é o algoritmo de hash de digestão com o qual o hash do conteúdo do documento é assinado.
