## Enumeração ValidationStatus

Representa o status de validação de uma validação de certificado.

```csharp
public enum ValidationStatus
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Válido | `0` | Indica que o certificado foi validado com sucesso. |
| Inválido | `1` | Indica que a validação do certificado falhou. |
| Indefinido | `2` | Indica que o processo de validação foi inconclusivo ou não foi realizado. |

## Observações

Esta enumeração define os possíveis resultados da validação de certificados: - Válido: Indica que o certificado foi validado com sucesso. - Inválido: Indica que a validação do certificado falhou. - Indefinido: Indica que o processo de validação foi inconclusivo ou não foi realizado.
