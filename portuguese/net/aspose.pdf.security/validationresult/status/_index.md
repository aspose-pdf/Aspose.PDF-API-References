## Propriedade ValidationResult.Status

Obtém o status do processo de validação de um certificado.

```csharp
public ValidationStatus Status { get; }
```

## Observações

A propriedade Status indica o resultado da validação do certificado. Os valores possíveis são definidos na enumeração [`ValidationStatus`](../../validationstatus/), como Válido, Inválido ou Indefinido. Ela fornece uma visão sobre se o certificado passou ou não nas verificações de validação.
