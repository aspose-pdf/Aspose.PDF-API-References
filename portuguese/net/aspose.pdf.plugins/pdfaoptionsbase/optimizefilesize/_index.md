## Propriedade PdfAOptionsBase.OptimizeFileSize

Obtém ou define um valor que indica se deve tentar reduzir o tamanho do arquivo durante o processo de conversão para PDF/A.

```csharp
public bool OptimizeFileSize { get; set; }
```

### Valor da Propriedade

`true` se o tamanho do arquivo deve ser reduzido o máximo possível; caso contrário, `false`.

## Observações

Quando definido como `true`, o processo de conversão tentará minimizar o tamanho do arquivo resultante. Isso pode afetar o desempenho do processo de conversão.
