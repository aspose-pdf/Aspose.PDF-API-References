## Propriedade PdfAOptionsBase.UnicodeProcessingRules

Obtém ou define as regras para processar tabelas CMap ToUnicode e não vinculadas a símbolos Unicode durante o processo de conversão PDF/A.

```csharp
public ToUnicodeProcessingRules UnicodeProcessingRules { get; set; }
```

### Valor da Propriedade

Uma instância de [`ToUnicodeProcessingRules`](../../../aspose.pdf/tounicodeprocessingrules/) que define as regras para lidar com os casos problemáticos com caracteres Unicode.
