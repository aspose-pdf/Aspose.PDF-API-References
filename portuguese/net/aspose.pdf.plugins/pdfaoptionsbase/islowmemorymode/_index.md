## Propriedade PdfAOptionsBase.IsLowMemoryMode

Obtém ou define um valor que indica se o modo de baixa memória está ativado durante o processo de conversão para PDF/A.

```csharp
public bool IsLowMemoryMode { get; set; }
```

### Valor da Propriedade

`true` se o modo de baixa memória estiver ativado; caso contrário, `false`.
