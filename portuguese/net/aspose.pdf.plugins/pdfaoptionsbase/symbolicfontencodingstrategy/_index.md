## Propriedade PdfAOptionsBase.SymbolicFontEncodingStrategy

Obtém ou define a estratégia para codificação de fontes simbólicas ao converter para o formato PDF/A.

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### Valor da Propriedade

A estratégia de codificação de fontes simbólicas.

## Observações

Esta propriedade permite que você controle qual sub-tabela CMap seria copiada para o documento resultante nos casos em que a fonte simbólica TrueType original no documento de origem contém várias sub-tabelas CMap.
