## Propriedade PdfAOptionsBase.FontEmbeddingOptions

Obtém as opções para processar fontes que não podem ser incorporadas ao documento.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Valor da Propriedade

As opções de incorporação de fontes.

## Observações

O padrão PDF/A exige que todas as fontes devem ser incorporadas ao documento. Esta propriedade fornece opções para lidar com casos em que não é possível incorporar algumas fontes porque estão ausentes no PC de destino.
