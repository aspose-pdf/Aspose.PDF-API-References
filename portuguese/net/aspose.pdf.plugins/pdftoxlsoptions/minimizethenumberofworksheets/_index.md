## Propriedade PdfToXlsOptions.MinimizeTheNumberOfWorksheets

Defina como verdadeiro se você precisar minimizar o número de planilhas no livro de trabalho resultante. O valor padrão é falso; isso significa salvar cada página PDF como uma planilha separada.

public bool MinimizeTheNumberOfWorksheets { get; set; }
