## Construtor do FormExporterValuesToCsvOptions

Inicializa uma nova instância do objeto [`FormExporterValuesToCsvOptions`](../), na qual os campos cujos dados serão exportados e o separador para os dados exportados são especificados.

```csharp
public FormExporterValuesToCsvOptions(SelectField selectField, char delimeter = ',')
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| selectField | SelectField | Delegado que determina as propriedades de qual(is) campo(s) definir valores. |
| delimeter | Char | Especifica o delimitador a ser usado nos dados exportados entre os valores. O valor padrão é ',' |
