## Propriedade SetDash.Phase

Fase do traço. Antes de começar a traçar um caminho, o array de traços deve ser ciclado, somando os comprimentos dos traços e lacunas. Quando o comprimento acumulado igualar o valor especificado pela fase do traço, o traçado do caminho deve começar e o array de traços deve ser usado ciclicamente a partir desse ponto.

```csharp
public int Phase { get; set; }
```
