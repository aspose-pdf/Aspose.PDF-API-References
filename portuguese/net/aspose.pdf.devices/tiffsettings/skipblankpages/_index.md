## Propriedade TiffSettings.SkipBlankPages

Obtém ou define um valor que indica se deve pular páginas em branco.

```csharp
public bool SkipBlankPages { get; set; }
```

### Valor da Propriedade

`true` se precisar pular páginas em branco; caso contrário, `false`.

## Observações

O valor padrão é falso
