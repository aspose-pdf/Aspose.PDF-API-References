## Método PdfFileSignature.GetAccessPermissions

Retorna o valor das permissões de acesso de um documento certificado pelo tipo de assinatura MDP.

```csharp
public DocMDPAccessPermissions GetAccessPermissions()
```

### Valor de Retorno

Se o documento estiver sendo certificado, retorna o valor das permissões de acesso; caso contrário, uma exceção é lançada.
