## Propriedade Bookmark.Action

Obtém ou define a ação vinculada ao marcador. Se PageNumber estiver presente, a ação não pode ser especificada. O tipo de ação inclui: "GoTo", "GoToR", "Launch", "Named".

```csharp
public string Action { get; set; }
```
