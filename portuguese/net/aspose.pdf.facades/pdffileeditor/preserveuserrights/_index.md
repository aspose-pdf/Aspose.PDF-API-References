## Propriedade PdfFileEditor.PreserveUserRights

Se verdadeiro, os direitos do usuário do primeiro documento são aplicados ao documento concatenado. Os direitos do usuário de todos os outros documentos são ignorados.

public bool PreserveUserRights { get; set; }
