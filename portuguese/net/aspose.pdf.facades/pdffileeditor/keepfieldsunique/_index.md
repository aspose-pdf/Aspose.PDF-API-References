## Propriedade PdfFileEditor.KeepFieldsUnique

Se verdadeiro, os nomes dos campos serão tornados únicos quando os formulários forem concatenados. Sufixos serão adicionados aos nomes dos campos, o modelo de sufixo pode ser especificado na propriedade UniqueSuffix.

```csharp
public bool KeepFieldsUnique { get; set; }
```
