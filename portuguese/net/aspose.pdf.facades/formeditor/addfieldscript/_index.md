## Método FormEditor.AddFieldScript

Adicione JavaScript para um campo PushButton. Se um evento antigo existir, um novo evento é adicionado após ele.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| fieldName | String | O nome do campo totalmente qualificado. |
| script | String | O script Java a ser adicionado/colocado em um campo de botão de pressão. |

### Valor de Retorno

Verdadeiro no caso de o script ter sido adicionado com sucesso.
