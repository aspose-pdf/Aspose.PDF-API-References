## Método FormEditor.SetFieldScript

Defina JavaScript para um campo PushButton. Se um JavaScript antigo existia, ele será substituído pelo novo.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| fieldName | String | O nome do campo totalmente qualificado. |
| script | String | O script Java a ser adicionado/colocado em um campo de botão de pressão. |

### Valor de Retorno

true se o script do campo foi definido com sucesso.
