## Propriedade FormDataConverter.Table

Obtém ou define o contêiner de dados intermediário, um DataTable. Deve ser definido antes de converter dados de um formato para outro formato. As Colunas e o TableName do DataTable devem ser definidos. O TableName é o nome da Tabela no banco de dados. O ColumnName de cada coluna é o nome do campo qualificado do pdf. O Caption de cada coluna é o nome da coluna da tabela no banco de dados. Se o nome do campo for o mesmo que o nome da coluna da tabela, o Caption não precisa ser especificado.

```csharp
public DataTable Table { get; set; }
```
