## Método PdfContentEditor.CreateSound

Cria Anotações de Som.

```csharp
public void CreateSound(Rectangle rect, string filePath, string name, int page, string rate)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| rect | Rectangle | O retângulo da anotação que define a localização da anotação na página. |
| filePath | String | O caminho do arquivo de som. |
| name | String | O nome de um ícone a ser usado na exibição da anotação, incluindo: Speaker e Mic. |
| page | Int32 | A página na qual a anotação de Som é criada. |
| rate | String | A taxa de amostragem, em amostras por segundo. |
