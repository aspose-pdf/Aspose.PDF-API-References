## Método AutoFiller.ImportDataTable

Importa dados do tipo DataTable. O nome de cada coluna do dataTable deve ser o mesmo que o nome de um campo do pdf modelo, respeitando maiúsculas e minúsculas.

```csharp
public void ImportDataTable(DataTable dataTable)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| dataTable | DataTable | O dataTable que contém todos os dados do banco de dados ou de outra fonte de dados. |
