## Evento PdfViewer.StartPage

Ocorre antes de uma página começar a ser impressa.

```csharp
public event EventHandler<StartEndPageEventArgs> StartPage;
```

## Observações

Este evento é acionado para cada página que está prestes a ser impressa. Pode ser usado para realizar ações personalizadas antes que a impressão da página comece.
