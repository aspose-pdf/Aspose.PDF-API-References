## Propriedade PdfViewer.AutoResize

Obtém ou define um valor bool que indica se o arquivo deve ser impresso com tamanho otimizado. Se falso, imprime a página sem escalonamento. Se verdadeiro, imprime a página com escalonamento para se ajustar à área imprimível.

```csharp
public bool AutoResize { get; set; }
```
