## Evento PdfViewer.EndPage

Ocorre quando a impressão de uma página termina no PdfViewer.

```csharp
public event EventHandler<StartEndPageEventArgs> EndPage;
```

## Observações

Este evento é acionado após uma página ter sido processada no PdfViewer. Pode ser usado para realizar ações ou atualizações após a impressão de uma página.
