## Propriedade PdfViewer.PrintAsImage

Define ou obtém um modo para o PdfViewer imprimir como imagem.

```csharp
public bool PrintAsImage { get; set; }
```

## Observações

Se verdadeiro, imprime sempre como imagem (gera imagem que é impressa). Se falso, imprime diretamente no dispositivo se todos os recursos forem suportados. Caso o documento contenha recursos não suportados, o sistema pode decidir automaticamente imprimir como imagem. O valor padrão é falso.
