## Método PdfFileEditor.ContentsResizeParameters.ContentSizePercent

Cria parâmetros de redimensionamento com o tamanho de conteúdo especificado em porcentagens do tamanho inicial da página. As margens são calculadas automaticamente.

```csharp
public static ContentsResizeParameters ContentSizePercent(double width, double height)
```

| Parâmetro | Tipo | Descrição |

| width | Double | Nova largura do conteúdo em porcentagens. |
| height | Double | Nova altura do conteúdo em porcentagens. |

Novos parâmetros de redimensionamento.
