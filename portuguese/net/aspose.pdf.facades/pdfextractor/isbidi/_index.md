## Propriedade PdfExtractor.IsBidi

É verdadeiro quando o texto contém símbolos hebraicos ou árabes. Este caso deve ser considerado especialmente porque as funções de string mudam seu comportamento e começam a processar o texto da direita para a esquerda (exceto números e outros caracteres não textuais).

```csharp
public bool IsBidi { get; }
```
