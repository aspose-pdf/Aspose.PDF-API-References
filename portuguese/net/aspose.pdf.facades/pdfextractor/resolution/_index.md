## Propriedade PdfExtractor.Resolution

Define ou obtém a resolução para imagens extraídas. O valor padrão é 150. Imagens que têm um valor de resolução maior são mais nítidas. No entanto, aumentar o valor da resolução resulta em um aumento do tempo e da memória necessários para extrair imagens. Geralmente, para obter uma imagem nítida, é suficiente definir a resolução para 150 ou 300.

public int Resolution { get; set; }
