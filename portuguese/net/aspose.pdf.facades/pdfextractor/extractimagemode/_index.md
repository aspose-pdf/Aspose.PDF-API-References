## Propriedade PdfExtractor.ExtractImageMode

Define o modo para o processo de extração de imagens.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## Observações

O valor padrão é ExtractImageMode.DefinedInResources, que extrai todas as imagens definidas nos recursos. Para extrair imagens realmente exibidas, o modo ExtractImageMode.ActuallyUsed deve ser utilizado.
