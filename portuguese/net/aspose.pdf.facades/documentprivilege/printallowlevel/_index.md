## Propriedade DocumentPrivilege.PrintAllowLevel

Obtém e define o nível de impressão do privilégio do documento. Assim como as configurações de Impressão Permitida do Adobe Professional. 0: Nenhum. 1: Baixa Resolução (150 dpi). 2: Alta Resolução.

```csharp
public int PrintAllowLevel { get; set; }
```

## Observações

Se a propriedade tiver um valor de -1, então o nível é indefinido.
