## Propriedade DocumentPrivilege.CopyAllowLevel

Obtém e define o nível de cópia do privilégio do documento. Assim como as configurações de permissão do Adobe Professional. 0: Nenhum. 1: Habilitar acesso ao texto para dispositivos de leitura de tela para deficientes visuais. 2: Habilitar cópia de texto, imagens e outros conteúdos.

```csharp
public int CopyAllowLevel { get; set; }
```

## Observações

Se a propriedade tiver um valor de -1, então o nível é indefinido.
