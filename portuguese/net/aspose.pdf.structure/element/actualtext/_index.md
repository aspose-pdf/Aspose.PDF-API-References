## Propriedade Element.ActualText

(Opicional; PDF 1.4) Texto que é uma substituição exata para o elemento de estrutura e seus filhos. Este texto de substituição (que deve se aplicar a uma parte tão pequena do conteúdo quanto possível) é útil ao extrair o conteúdo do documento em apoio à acessibilidade para usuários com deficiência ou para outros fins.

```csharp
public virtual string ActualText { get; set; }
```
