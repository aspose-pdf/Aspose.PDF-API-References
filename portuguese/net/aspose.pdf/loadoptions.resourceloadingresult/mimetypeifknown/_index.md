## Campo LoadOptions.ResourceLoadingResult.MIMETypeIfKnown

Às vezes, o conhecimento sobre o tipo MIME do recurso carregado é útil para o conversor. Você pode fornecer o tipo MIME (se for conhecido após o carregamento) neste parâmetro. Por favor, deixe o parâmetro igual a nulo quando o tipo MIME for desconhecido ou não for necessário fornecê-lo.

```csharp
public string MIMETypeIfKnown;
```
