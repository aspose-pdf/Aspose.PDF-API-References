## LoadOptions.ResourceLoadingResult.LoadingCancelled field

Às vezes, por algumas razões, o carregamento não deve ocorrer código personalizado. Nesse caso, defina este sinalizador como Verdadeiro. Nesse caso, o conversor tentará usar o carregador de recursos padrão interno para obter esse resultado (como se comporta na situação em que a estratégia personalizada não é fornecida).

```csharp
public bool LoadingCancelled;
```
