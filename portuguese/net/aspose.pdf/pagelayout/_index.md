## Enumeração PageLayout

Descreve o layout da página.

```csharp
public enum PageLayout
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| SinglePage | `0` | Página única. |
| OneColumn | `1` | Exibir as páginas em uma coluna. |
| TwoColumnLeft | `2` | Exibir as páginas em duas colunas, com páginas de números ímpares à esquerda. |
| TwoColumnRight | `3` | Exibir as páginas em duas colunas, com páginas de números ímpares à direita. |
| TwoPageLeft | `4` | Exibir as páginas duas de cada vez, com páginas de números ímpares à esquerda. |
| TwoPageRight | `5` | Exibir as páginas duas de cada vez, com páginas de números ímpares à direita. |
| Default | `6` | Layout padrão. |
