## Método Matrix.Skew

Cria matriz para o ângulo de rotação dado.

```csharp
public static Matrix Skew(double alpha, double beta)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| alpha | Double | Ângulo de inclinação x em radianos. |
| beta | Double | Ângulo de inclinação y em radianos. |

### Valor de Retorno

Matriz de transformação.

## Exemplos

```csharp
Matrix m = Matrix.Skew(Math.PI / 2, Math.PI / 2);
```
