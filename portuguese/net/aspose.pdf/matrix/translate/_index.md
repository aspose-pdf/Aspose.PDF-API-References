## Método Matrix.Translate

Translada uma matriz pela quantidade especificada na direção x e y.

```csharp
public static Matrix Translate(double dx, double dy, Matrix source)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| dx | Double | A quantidade a ser traduzida na direção x. |
| dy | Double | A quantidade a ser traduzida na direção y. |
| source | Matrix | A matriz a ser traduzida. |

### Valor de Retorno

Uma nova matriz que é o resultado da tradução.
