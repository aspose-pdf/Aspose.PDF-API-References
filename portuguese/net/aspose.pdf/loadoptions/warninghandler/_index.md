## Propriedade LoadOptions.WarningHandler

Callback para lidar com quaisquer avisos gerados. O WarningHandler retorna um item do enum ReturnAction especificando Continue ou Abort. Continue é a ação padrão e a operação de Load continua, no entanto, o usuário também pode retornar Abort, caso em que a operação de Load deve cessar.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
