## Propriedade OptimizedMemoryStream.CanRead

Quando substituída em uma classe derivada, obtém um valor que indica se o fluxo atual suporta leitura.

```csharp
public override bool CanRead { get; }
```

### Valor de Retorno

true se o fluxo suporta leitura; caso contrário, false.
