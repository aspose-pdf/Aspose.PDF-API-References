## Método OptimizedMemoryStream.Write

Quando substituído em uma classe derivada, grava uma sequência de bytes no fluxo atual e avança a posição atual dentro deste fluxo pelo número de bytes gravados.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| buffer | Byte[] | Um array de bytes. Este método copia *count* bytes de *buffer* para o fluxo atual. |
| offset | Int32 | O deslocamento de byte baseado em zero em *buffer* onde começar a copiar bytes para o fluxo atual. |
| count | Int32 | O número de bytes a serem gravados no fluxo atual. |
