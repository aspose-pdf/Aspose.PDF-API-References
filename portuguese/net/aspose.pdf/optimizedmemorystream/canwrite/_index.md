## Propriedade OptimizedMemoryStream.CanWrite

Quando substituída em uma classe derivada, obtém um valor que indica se o fluxo atual suporta gravação.

```csharp
public override bool CanWrite { get; }
```

### Valor de Retorno

true se o fluxo suporta gravação; caso contrário, false.
