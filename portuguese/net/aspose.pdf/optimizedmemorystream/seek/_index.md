## Método OptimizedMemoryStream.Seek

Quando substituído em uma classe derivada, define a posição dentro do fluxo atual.

```csharp
public override long Seek(long offset, SeekOrigin origin)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| offset | Int64 | Um deslocamento em bytes relativo ao parâmetro *origin*. |
| origin | SeekOrigin | Um valor do tipo SeekOrigin que indica o ponto de referência usado para obter a nova posição. |

### Valor de Retorno

A nova posição dentro do fluxo atual.
