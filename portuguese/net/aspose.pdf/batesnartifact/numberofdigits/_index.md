## Propriedade BatesNArtifact.NumberOfDigits

Obtém ou define o número de dígitos para numeração Bates. O valor deve estar entre 3 e 15, inclusive. Se um valor menor que 3 for definido, ele será ajustado para 3. Se um valor maior que 15 for definido, ele será ajustado para 15. O valor padrão é 6.

```csharp
public int NumberOfDigits { get; set; }
```
