## Método DestinationCollection.GetExplicitDestination

Retorna o destino explícito pelo nome.

```csharp
public ExplicitDestination GetExplicitDestination(string destinameName, bool useCache)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| destinameName | String | O nome do destino. |
| useCache | Boolean | Determina se a versão em cache da coleção é usada ou não. |

### Valor de Retorno

O objeto ExplicitDestination para o destino encontrado; caso contrário, nulo.
