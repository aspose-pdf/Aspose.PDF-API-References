## Método DestinationCollection.GetPageNumber

Retorna o número da página do destino pelo nome.

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| destinameName | String | O nome do destino. |
| useCache | Boolean | Determina se a versão em cache da coleção é usada ou não. |

### Valor de Retorno

O número da página se o destino foi encontrado; caso contrário, -1.
