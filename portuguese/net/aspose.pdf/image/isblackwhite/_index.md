## Propriedade Image.IsBlackWhite

Obtém ou define um valor bool que indica se a imagem é forçada a ser preto e branco. Se uma imagem TIFF do subformato CCITT for usada, essa propriedade deve ser definida como verdadeira.

```csharp
public bool IsBlackWhite { get; set; }
```
