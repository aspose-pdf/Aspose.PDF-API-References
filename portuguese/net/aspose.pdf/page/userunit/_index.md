## Propriedade Page.UserUnit

Obtém ou define o valor de UserUnit. Um número positivo que indica o tamanho das unidades de espaço do usuário padrão, em múltiplos de 1 / 72 polegada. O valor padrão é 1. Por favor, defina um valor zero ou negativo para limpar esta entrada na página.

public double UserUnit { get; set; }
