## Método Page.AddGraphics

Adiciona gráficos à página. Funciona mais rápido do que adicionar elementos um a um com o método [`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/).

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| elements | GraphicElementCollection | Coleção de gráficos. |
| rectangle | Rectangle | Os elementos serão adicionados à página se estiverem dentro da área do retângulo. Se o retângulo for nulo, todos os elementos gráficos serão adicionados |
