## Propriedade Page.IsAddParagraphsAfterLast

Obtém ou define a adição de parágrafos após o último parágrafo da página

```csharp
public bool IsAddParagraphsAfterLast { get; set; }
```

### Valor da Propriedade

O valor indica se os parágrafos serão adicionados após o último parágrafo da página. Os parágrafos serão adicionados após o último parágrafo da página se o valor for verdadeiro.
