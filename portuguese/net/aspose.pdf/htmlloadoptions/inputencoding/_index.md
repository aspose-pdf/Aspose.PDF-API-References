## Propriedade HtmlLoadOptions.InputEncoding

Obtém ou define o atributo que especifica a codificação usada para este documento no momento da análise. Se este atributo for nulo, a codificação será determinada a partir do atributo de conjunto de caracteres do documento.

```csharp
public string InputEncoding { get; set; }
```
