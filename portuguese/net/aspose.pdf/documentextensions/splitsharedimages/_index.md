## Método DocumentExtensions.SplitSharedImages

```csharp
public static void SplitSharedImages(this Document doc, Page page_1, Page page_2)
```

| Parâmetro | Tipo | Descrição |

| doc | Documento | O documento contendo ambas as coleções. |
| page_1 | Página | Primeira página para comparação. |
