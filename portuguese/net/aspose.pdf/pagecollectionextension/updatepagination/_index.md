## Método PageCollectionExtension.UpdatePagination

Atualiza os números e datas das páginas do cabeçalho e rodapé para todas as páginas. Isso funcionará se o documento tiver pelo menos um artefato de paginação com dados de configurações especiais. Todas as páginas na coleção serão atualizadas com o artefato de origem de acordo com suas configurações.

```csharp
public static void UpdatePagination(this PageCollection pageCollection)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| pageCollection | PageCollection | A coleção de páginas. |
