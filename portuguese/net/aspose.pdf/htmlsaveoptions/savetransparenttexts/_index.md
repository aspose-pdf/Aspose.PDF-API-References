## Campo HtmlSaveOptions.SaveTransparentTexts

O PDF pode conter textos transparentes que podem ser selecionados para a área de transferência (geralmente isso acontece quando o documento contém imagens e textos extraídos por OCR). Esta configuração informa ao conversor se precisamos salvar esses textos como textos transparentes selecionáveis no HTML resultante.

```csharp
public bool SaveTransparentTexts;
```
