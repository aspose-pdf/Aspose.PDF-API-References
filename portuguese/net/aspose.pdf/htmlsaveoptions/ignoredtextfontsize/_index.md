## Propriedade HtmlSaveOptions.IgnoredTextFontSize

Texto com o tamanho especificado ou menor será ignorado durante a conversão. Não removemos esse texto, nós o ignoramos e não o transferimos para o arquivo de saída

```csharp
public float? IgnoredTextFontSize { get; set; }
```
