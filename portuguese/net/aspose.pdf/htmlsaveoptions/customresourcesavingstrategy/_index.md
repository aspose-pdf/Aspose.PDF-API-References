## Campo HtmlSaveOptions.CustomResourceSavingStrategy

Este campo pode conter a estratégia de salvamento que deve ser usada (se presente) durante a conversão para o manuseio personalizado de arquivos de recursos referenciados criados (como imagens e fontes) relacionadas a nós do HTML salvo. Essa estratégia deve processar recursos e retornar uma string que representa a URL desejável do recurso salvo no HTML gerado.

```csharp
public ResourceSavingStrategy CustomResourceSavingStrategy;
```
