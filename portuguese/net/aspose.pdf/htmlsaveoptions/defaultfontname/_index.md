## Propriedade HtmlSaveOptions.DefaultFontName

Especifica o nome de uma fonte instalada que é usada para substituir qualquer fonte de documento que não esteja incorporada e não esteja instalada no sistema. Se nulo, então a fonte de substituição padrão é usada.

```csharp
public string DefaultFontName { get; set; }
```

### Valor da Propriedade

Nome da fonte
