## Propriedade HtmlSaveOptions.FontSources

Fontes de fontes pré-salvas.

```csharp
public FontSourceCollection FontSources { get; }
```

## Observações

As fontes podem ser salvas preliminarmente para fins de cache e, em seguida, passadas para o processo de conversão Html. Por exemplo, pode ser útil em cenários de divisão de documentos e processamento de páginas de documentos em múltiplas threads com um único conjunto de fontes.
