## Propriedade HtmlSaveOptions.SimpleTextboxModeGrouping

Este atributo especifica um agrupamento sequencial de glifos e palavras em strings. Por exemplo, tags e palavras têm uma ordem diferente no HTML convertido e você quer que elas coincidam. Este parâmetro será aplicado ao documento apenas quando o valor do atributo FixedLayout for verdadeiro.

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
