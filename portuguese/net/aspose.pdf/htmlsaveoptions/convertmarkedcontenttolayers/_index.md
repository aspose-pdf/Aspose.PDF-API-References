## Propriedade HtmlSaveOptions.ConvertMarkedContentToLayers

Se o atributo ConvertMarkedContentToLayers estiver definido como verdadeiro, então todos os elementos dentro de um conteúdo marcado em PDF (camada) serão colocados em um div HTML com o atributo "data-pdflayer" especificando um nome de camada. Este nome de camada será extraído das propriedades opcionais do conteúdo marcado em PDF. Se este atributo for falso (por padrão), então nenhuma camada será criada a partir do conteúdo marcado em PDF.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
