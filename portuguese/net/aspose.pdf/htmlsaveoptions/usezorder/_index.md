## Propriedade HtmlSaveOptions.UseZOrder

Se o atributo UseZOrder estiver definido como verdadeiro, gráficos e texto são adicionados ao documento HTML resultante de acordo com a ordem Z no documento PDF original. Se este atributo for falso, todos os gráficos são colocados como uma única camada, o que pode causar alguns efeitos desnecessários para objetos sobrepostos.

```csharp
public bool UseZOrder { get; set; }
```
