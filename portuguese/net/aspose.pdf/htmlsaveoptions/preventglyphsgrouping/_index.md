## Propriedade HtmlSaveOptions.PreventGlyphsGrouping

Este atributo ativa o modo quando os glifos de texto não serão agrupados em palavras e strings. Este modo permite manter a máxima precisão durante o posicionamento dos glifos na página e pode ser usado para converter documentos com notas musicais ou glifos que devem ser colocados separadamente uns dos outros. Este parâmetro será aplicado ao documento apenas quando o valor do atributo FixedLayout for verdadeiro.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
