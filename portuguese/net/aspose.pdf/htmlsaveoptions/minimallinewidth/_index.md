## Propriedade HtmlSaveOptions.MinimalLineWidth

Este atributo define a largura mínima da linha do caminho gráfico. Se a espessura da linha for inferior a 1px, o Adobe Acrobat arredonda para este valor. Portanto, este atributo pode ser usado para emular esse comportamento em navegadores HTML.

```csharp
public float MinimalLineWidth { get; set; }
```
