## Campo HtmlSaveOptions.TrySaveTextUnderliningAndStrikeoutingInCss

O PDF em si não contém marcadores de sublinhado para textos. É emulado com uma linha situada sob o texto. Esta opção permite que o conversor tente adivinhar que esta ou aquela linha é um sublinhado de texto e coloque essa informação no CSS em vez de desenhar o sublinhado graficamente

```csharp
public bool TrySaveTextUnderliningAndStrikeoutingInCss;
```
