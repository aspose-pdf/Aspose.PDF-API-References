## Propriedade HtmlSaveOptions.SplitIntoPages

Obtém ou define a flag que indica se cada página do documento de origem será convertida em seu próprio documento HTML de destino, ou seja, se o HTML resultante será dividido em várias páginas HTML.

public bool SplitIntoPages { get; set; }
