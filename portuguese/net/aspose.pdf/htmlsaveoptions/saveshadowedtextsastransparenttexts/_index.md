## Campo HtmlSaveOptions.SaveShadowedTextsAsTransparentTexts

O PDF pode conter textos que são sombreados por outros elementos (por exemplo, por imagens), mas podem ser selecionados para a área de transferência no Acrobat Reader (geralmente isso acontece quando o documento contém imagens e textos OCR extraídos dele). Esta configuração informa ao conversor se precisamos salvar esses textos como textos selecionáveis transparentes no HTML resultante para imitar o comportamento do Acrobat Reader (caso contrário, esses textos geralmente são salvos como ocultos, não disponíveis para cópia para a área de transferência).

```csharp
public bool SaveShadowedTextsAsTransparentTexts;
```
