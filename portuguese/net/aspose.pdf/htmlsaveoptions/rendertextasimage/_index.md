## Propriedade HtmlSaveOptions.RenderTextAsImage

Se o atributo RenderTextAsImage estiver definido como verdadeiro, o texto da fonte se torna uma imagem em HTML. Pode ser útil para tornar o texto não selecionável ou quando o texto HTML não é renderizado corretamente.

```csharp
public bool RenderTextAsImage { get; set; }
```
