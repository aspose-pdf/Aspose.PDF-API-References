## Propriedade HtmlSaveOptions.IgnoreResourceFontErrors

Obtém ou define a indicação de que erros relacionados à ausência de fonte serão ignorados. true - significa que erros de ausência de fonte serão ignorados. Segmentos de texto que se referem a recursos incorretos serão pulados durante o processamento. false por padrão

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
