## Propriedade HtmlSaveOptions.SaveFullFont

Indica que a fonte completa será salva, suporta apenas Fontes True Type. Por padrão, SaveFullFont = falso e o conversor salva o subconjunto da fonte inicial necessário para exibir o texto do documento.

```csharp
public bool SaveFullFont { get; set; }
```
