## Enumeração PdfFormatConversionOptions.SegmentAlignStrategy

Descreve estratégias usadas para alinhar segmentos de texto do documento. Agora, apenas a estratégia para restaurar segmentos aos limites originais é suportada. No futuro, outras estratégias podem ser adicionadas.

```csharp
public enum SegmentAlignStrategy : byte
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Nenhum | `0` | Nenhum. |
| RestaurarLimitesSegmento | `1` | Restaurar limites do segmento. |
