## Método Rectangle.ContainsPoint

Determina se o ponto dado está contido dentro do retângulo.

```csharp
public bool ContainsPoint(double x, double y)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| x | Double | Coordenada X do ponto. |
| y | Double | Coordenada Y do ponto. |

### Valor de Retorno

`true` se o ponto estiver contido dentro do retângulo; caso contrário, `false`.
