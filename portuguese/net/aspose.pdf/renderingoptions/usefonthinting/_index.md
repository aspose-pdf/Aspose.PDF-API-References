## Propriedade RenderingOptions.UseFontHinting

O uso desta flag ativa o mecanismo de hinting de fonte. O hinting de fonte é o uso de instruções matemáticas para ajustar a exibição de uma fonte contornada. Em alguns casos, ativar esta flag pode resolver problemas de legibilidade do texto. No momento atual, o uso desta flag pode ter efeito apenas para fontes TTF, se essas fontes forem usadas no documento de origem.

```csharp
public bool UseFontHinting { get; set; }
```
