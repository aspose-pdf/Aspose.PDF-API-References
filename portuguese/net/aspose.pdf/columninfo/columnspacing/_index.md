## Propriedade ColumnInfo.ColumnSpacing

Obtém ou define uma string que contém o espaçamento entre colunas. O valor de cada espaçamento deve ser separado por espaço em branco. A unidade padrão é ponto, mas cm e polegada também são suportados. Por exemplo, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Observações

Se esta propriedade não for definida, o valor padrão 0 será usado para cada espaçamento.
