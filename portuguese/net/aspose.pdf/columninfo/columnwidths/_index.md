## Propriedade ColumnInfo.ColumnWidths

Obtém ou define uma string que contém a largura das colunas. O valor de cada coluna deve ser separado por espaço. A unidade padrão é ponto, mas cm, polegada e porcentagem da largura disponível também são suportados. Por exemplo, "120 2.5cm 1.5inch"

public string ColumnWidths { get; set; }
