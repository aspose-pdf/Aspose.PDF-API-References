| [ColumnCount](../../aspose.pdf/columninfo/columncount/) { get; set; } | Obtém ou define um valor int que indica o número de colunas. |
| [ColumnSpacing](../../aspose.pdf/columninfo/columnspacing/) { get; set; } | Obtém ou define uma string que contém o espaçamento entre colunas. | [ColumnWidths](../../aspose.pdf/columninfo/columnwidths/) { get; set; } | Obtém ou define uma string que contém a largura das colunas. O valor de cada coluna deve ser separado por espaço em branco. A unidade padrão é ponto, mas cm, polegada e porcentagem da largura disponível também são suportados. Por exemplo, "120 2.5cm 1.5inch" |
