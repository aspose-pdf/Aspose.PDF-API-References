## Campo HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled

Deve ser definido por código personalizado se não puder ou não deve definir a URL que será usada no HTML gerado para referência desse CSS. Se for 'true', o arquivo CSS será salvo de maneira padrão no local padrão.

```csharp
public bool CustomProcessingCancelled;
```
