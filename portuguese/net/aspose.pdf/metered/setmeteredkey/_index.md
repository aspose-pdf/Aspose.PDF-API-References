## Método Metered.SetMeteredKey

Define a chave pública e privada metered. Se você comprar uma licença metered, ao iniciar o aplicativo, esta API deve ser chamada, normalmente, isso é suficiente. No entanto, se sempre falhar ao enviar dados de consumo e exceder 24 horas, a licença será definida como status de avaliação; para evitar esse caso, você deve verificar regularmente o status da licença; se estiver no status de avaliação, chame esta API novamente.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| publicKey | String | chave pública |
| privateKey | String | chave privada |
