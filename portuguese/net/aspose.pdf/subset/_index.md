## Enumeração Subconjunto

Representa o subconjunto de páginas ao qual um artefato de paginação pode se aplicar.

```csharp
public enum Subset
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Todos | `0` | Indica que o artefato se aplica a todas as páginas. |
| Par | `1` | Indica que o artefato se aplica apenas às páginas de número par. |
| Ímpar | `2` | Indica que o artefato se aplica apenas às páginas de número ímpar. |
