## Propriedade PdfFormatConversionOptions.AlignText

Esta flag controla o alinhamento do texto no documento convertido. Por padrão, a conversão de documentos não afeta o alinhamento do texto e deixa o texto como está. Mas em alguns casos, a substituição de fonte causa sobreposição de texto ou espaços extras no documento convertido. Quando esta flag é definida, operações especiais de alinhamento serão realizadas. Esta flag deve ser definida apenas para documentos que têm problemas com texto sobreposto ou espaços extras de texto, pois o uso desta flag diminui o desempenho e, em alguns casos, pode corromper o conteúdo do texto.

public bool AlignText { get; set; }
