## Propriedade PdfFormatConversionOptions.OutputIntent

Obtém ou define o [`OutputIntent`](../../outputintent/) para a conversão de formato PDF.

```csharp
public OutputIntent OutputIntent { get; set; }
```

## Observações

O `OutputIntent` especifica o dispositivo ou condição de saída pretendido para o qual o documento PDF está sendo preparado. Ele é usado para garantir que as cores no documento sejam renderizadas corretamente no dispositivo de destino.
