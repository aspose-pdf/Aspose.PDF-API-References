## Propriedade PdfFormatConversionOptions.OptimizeFileSize

Obtém ou define um indicador que ativa/desativa o modo de conversão especial para obter um documento PDF/A com tamanho reduzido. Atualmente, esse indicador impacta na otimização das fontes utilizadas no documento PDF, e, possivelmente, no futuro, esse indicador também será utilizado para ativar a otimização de outras estruturas de dados, como gráfico. O conjunto desse indicador e modo pode reduzir significativamente o tamanho do arquivo, mas, ao mesmo tempo, pode diminuir significativamente o desempenho da conversão.

```csharp
public bool OptimizeFileSize { get; set; }
```
