## Método Color.FromCmyk

Obtém um objeto Color pdf válido a partir dos componentes de cor CMYK.

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| c | Double | O componente de cor Ciano (valor 0 - 1). |
| m | Double | O componente de cor Magenta (valor 0 - 1). |
| y | Double | O componente de cor Amarelo (valor 0 - 1). |
| k | Double | O componente de cor Preto (valor 0 - 1). |

### Valor de Retorno

Objeto Color com cada valor de componente na faixa [0..1].
