## Propriedade FileParams.CheckSum

Uma string de 16 bytes que é a soma de verificação dos bytes do arquivo incorporado descompactado. A soma de verificação é calculada aplicando o algoritmo padrão MD5 de resumo de mensagem aos bytes do fluxo do arquivo incorporado.

public string CheckSum { get; }
