## Método PageCollection.IndexOf

Retorna o índice da página especificada.

```csharp
public int IndexOf(Page entity)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| entity | Page | Objeto Page. Os números das páginas começam a partir de 1. |

### Valor de Retorno

Índice da página na coleção.

## Observações

Os números das páginas começam a partir de 1. Retorna 0 caso a coleção não contenha a página.
