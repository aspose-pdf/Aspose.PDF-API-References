## Observações

Use a classe ComHelper para carregar um documento de um arquivo ou stream em um objeto Documento em uma aplicação COM. A classe Documento fornece um construtor padrão para criar um novo documento e também fornece construtores sobrecarregados para carregar um documento de um arquivo ou stream. Se você estiver usando Aspose.Words de uma aplicação .NET, pode usar todos os construtores do Documento diretamente, mas se estiver usando Aspose.Pdf de uma aplicação COM, apenas o construtor padrão do Documento está disponível.
