## Campo SvgSaveOptions.TreatTargetFileNameAsDirectory

Esta opção define se será criado um diretório de destino (caso ainda não exista) com o mesmo nome do arquivo de saída solicitado, em vez do próprio arquivo de saída solicitado. Assim, esse diretório conterá todas as imagens SVG de saída das páginas (como descrito abaixo). Se não, os arquivos de saída das páginas, além do primeiro, serão criados exatamente no diretório solicitado como o arquivo de saída principal, mas conterão no nome do arquivo o sufixo _[2...n], que é definido pelo número da página, por exemplo, se você definir o arquivo de saída "C:\AsposeTests\output.svg" e a saída contiver vários arquivos svg das páginas, então os arquivos das páginas também serão criados no diretório "C:\AsposeTests\" e terão os nomes 'output.svg', 'output_2.svg', 'output_3.svg', etc.

```csharp
public bool TreatTargetFileNameAsDirectory;
```
