## Campo SvgSaveOptions.CompressOutputToZipArchive

Especifica se a saída será criada como um zip-archive. Consulte o comentário sobre as opções 'TreatTargetFileNameAsDirectory' para ver as regras de nomenclatura dos arquivos svg das páginas para documentos de origem multipágina, que também se aplicam ao conjunto compactado de arquivos de saída.
