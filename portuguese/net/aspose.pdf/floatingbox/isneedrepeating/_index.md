## Propriedade FloatingBox.IsNeedRepeating

Obtém ou define um valor bool que indica se o parágrafo precisa ser repetido na próxima página. O valor padrão é falso. O atributo é válido apenas quando o próprio parágrafo e o objeto ao qual seu ReferenceParagraphID se refere estão incluídos em RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
