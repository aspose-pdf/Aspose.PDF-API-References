## Campo HtmlSaveOptions.HtmlImageSavingInfo.ParentType

A imagem salva pode pertencer ao HTML em si ou pode ser extraída do SVG incorporado ao HTML. Esta propriedade pode informar ao código personalizado qual é o tipo de pai da imagem processada. É definida pelo conversor e pode ser usada no código personalizado para decidir o que deve ser feito com essa imagem (por exemplo, o código personalizado pode decidir onde salvar a imagem ou como ela deve ser referenciada no conteúdo do pai).

```csharp
public ImageParentTypes ParentType;
```
