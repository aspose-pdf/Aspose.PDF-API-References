## Campo HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber

Informa ao código personalizado a que página do conjunto gerado de arquivos de página HTML a imagem salva pertence. Se a divisão em páginas estiver desativada, este valor sempre contém '1', pois, nesse caso, apenas uma página HTML é gerada.

```csharp
public int HtmlHostPageNumber;
```
