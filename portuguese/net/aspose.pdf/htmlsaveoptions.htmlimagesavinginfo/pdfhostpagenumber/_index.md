## Campo HtmlSaveOptions.HtmlImageSavingInfo.PdfHostPageNumber

Informa ao código personalizado a que página do documento PDF original a imagem salva pertence. Como é possível que nem todas as páginas do documento original sejam salvas, este valor nos informa sobre o número da página host no PDF original. Se o número da página original por algum motivo for desconhecido, ele sempre retorna '1'

```csharp
public int PdfHostPageNumber;
```
