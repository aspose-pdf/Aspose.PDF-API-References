## Propriedade ExcelSaveOptions.UniformWorksheets

Defina como verdadeiro para usar a divisão uniforme de colunas através do documento. O valor padrão é falso; isso significa que a divisão de colunas será independente para cada página.

```csharp
public bool UniformWorksheets { get; set; }
```
