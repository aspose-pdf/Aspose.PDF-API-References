## Campo HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber

Definido pelo conversor. Se a propriedade SplitToPages estiver definida, então vários arquivos HTML (um arquivo HTML por página convertida) são criados durante a conversão. Esta propriedade informa ao código personalizado de qual página do PDF original foi criado o HTML-markup salvo. Se o número da página original por algum motivo for desconhecido ou SplitOnPages=false, então esta propriedade sempre contém '0', o que sinaliza que o conversor não pode fornecer o número exato da página do PDF original para o arquivo HTML-markup fornecido.

```csharp
public int PdfHostPageNumber;
```
