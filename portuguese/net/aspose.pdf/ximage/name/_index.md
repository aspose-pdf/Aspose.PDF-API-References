## Propriedade XImage.Name

Obtém ou define o nome da imagem. Observe que, se você alterar o nome da imagem que possui referências no conteúdo da página, o documento pode se tornar incorreto. Por favor, use o método XImage.Rename neste caso.

```csharp
public string Name { get; set; }
```
