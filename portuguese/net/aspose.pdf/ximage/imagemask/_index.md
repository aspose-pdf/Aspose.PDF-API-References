## Propriedade XImage.ImageMask

Obtém uma flag indicando se a imagem deve ser tratada como uma máscara de imagem (veja 8.9.6, "Imagens Mascaradas"). Se essa flag for verdadeira, o valor de BitsPerComponent deve ser 1 e Mask e ColorSpace não devem ser especificados; áreas não mascaradas devem ser pintadas usando a cor atual não traçada. Valor padrão: falso.

```csharp
public bool ImageMask { get; }
```

### Valor da Propriedade

Verdadeiro se a imagem for uma máscara de imagem.
