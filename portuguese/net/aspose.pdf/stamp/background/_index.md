## Propriedade Stamp.Background

Define ou obtém um valor bool que indica que o conteúdo está carimbado como fundo. Se o valor for verdadeiro, o conteúdo do carimbo é colocado na parte inferior. Por padrão, o valor é falso, o conteúdo do carimbo é colocado na parte superior.

```csharp
public bool Background { get; set; }
```
