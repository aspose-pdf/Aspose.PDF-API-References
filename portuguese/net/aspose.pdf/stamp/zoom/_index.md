## Propriedade Stamp.Zoom

Fator de zoom do carimbo. Permite escalar o carimbo. Observe que o par de propriedades ZoomX e ZoomY permite definir o fator de zoom para cada eixo separadamente. A configuração desta propriedade altera tanto as propriedades ZoomX quanto ZoomY. Se ZoomX e ZoomY forem diferentes, então a propriedade Zoom retorna o valor de ZoomX.

```csharp
public double Zoom { get; set; }
```
