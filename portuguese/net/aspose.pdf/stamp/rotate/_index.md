## Propriedade Stamp.Rotate

Define ou obtém a rotação do conteúdo do carimbo de acordo com os valores de [`Rotation`](../../rotation/). Nota. Esta propriedade é para definir ângulos que são múltiplos de 90 graus (0, 90, 180, 270 graus). Para definir um ângulo arbitrário, use a propriedade RotateAngle. Se o ângulo definido por ArbitraryAngle não for múltiplo de 90, a propriedade Rotate retorna Rotation.None.

```csharp
public Rotation Rotate { get; set; }
```
