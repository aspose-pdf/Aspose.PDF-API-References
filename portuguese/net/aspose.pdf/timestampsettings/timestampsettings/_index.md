public TimestampSettings(string serverUrl, string basicAuthCredentials,

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| serverUrl | String | A URL do servidor de timestamp. |

| digestHashAlgorithm | DigestHashAlgorithm | O nome do algoritmo de hash, se omitido, então sha1 é usado. O valor padrão é `SHA256`. |
