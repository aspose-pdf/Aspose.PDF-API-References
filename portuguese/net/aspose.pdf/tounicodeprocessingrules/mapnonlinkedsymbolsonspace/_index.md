## Propriedade ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace

Algumas fontes não fornecem informações sobre unicodes para alguns símbolos de texto. Essa falta de informação gera um erro "Texto não pode ser mapeado para Unicode". Use esta flag para mapear símbolos não vinculados no unicode "espaço" (código 32).

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
