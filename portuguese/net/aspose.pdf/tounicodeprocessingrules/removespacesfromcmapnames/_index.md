## Propriedade ToUnicodeProcessingRules.RemoveSpacesFromCMapNames

Algumas fontes têm mapas de código de caractere ToUnicode com espaços nos nomes. Esses espaços podem causar erros com o mapeamento de texto unicode. Esta flag ordena a remoção de espaços dos nomes dos mapas de código de caractere ToUnicode. Por padrão, falso.

public bool RemoveSpacesFromCMapNames { get; set; }
