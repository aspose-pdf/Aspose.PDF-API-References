## Propriedades

| Nome | Descrição |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | Obtém uma flag indicando se o processador PDF interativo deve fornecer suporte para editar o valor do campo. Valor padrão: false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | Obtém o tipo de um valor de campo em uma coleção de esquema. Este campo descreve o tipo de valor correspondente a [`Subtype`](./subtype/). |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | Obtém o nome textual do campo que deve ser apresentado ao usuário pelo processador PDF interativo |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | Obtém a ordem relativa do nome do campo na interface do usuário. Os campos devem ser classificados pelo processador PDF interativo em ordem crescente. |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | Obtém o subtipo de um valor de campo em uma coleção de esquema. O subtipo de campo de coleção ou campo relacionado a arquivo que este dicionário descreve. Esta entrada identifica o tipo de dado que deve ser armazenado no campo. |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | Obtém a visibilidade inicial do campo na interface do usuário. Valor padrão: true. |
