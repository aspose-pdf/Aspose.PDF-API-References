## Propriedade PdfSaveOptions.DefaultFontName

Nome da fonte usado por padrão para fontes que estão ausentes no computador. Quando o documento PDF que está sendo salvo em PDF contém fontes que não estão disponíveis no próprio documento e no dispositivo, a API substitui essas fontes pela fonte padrão (se a fonte com `DefaultFontName` for encontrada no dispositivo)

```csharp
public string DefaultFontName { get; set; }
```
