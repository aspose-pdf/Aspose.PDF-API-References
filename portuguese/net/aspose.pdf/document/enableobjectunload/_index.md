## Propriedade Document.EnableObjectUnload

Obtém ou define a flag que permite que o documento seja parcialmente descarregado da memória. Isso permite reduzir o uso de memória, mas pode ter um efeito negativo no desempenho.

```csharp
public bool EnableObjectUnload { get; set; }
```
