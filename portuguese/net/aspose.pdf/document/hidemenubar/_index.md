## Propriedade Document.HideMenubar

Obtém ou define a flag que especifica se a barra de menu deve ser ocultada quando o documento está ativo.

```csharp
public bool HideMenubar { get; set; }
```

O exemplo demonstra como obter a flag HideMenubar:

```csharp
Document document = new Document("sample.pdf");
bool value = document.HideMenubar;
```
