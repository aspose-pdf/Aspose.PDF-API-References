## Propriedade Document.HideWindowUI

Obtém ou define a flag que especifica se os elementos da interface do usuário devem ser ocultos quando o documento está ativo.

```csharp
public bool HideWindowUI { get; set; }
```

## Exemplos

O exemplo demonstra como obter a flag HideWindowUI:

```csharp
Document document = new Document("sample.pdf");
bool value = document.HideWindowUI;
```
