## Propriedade Document.CryptoAlgorithm

Obtém as configurações de segurança se o documento estiver criptografado. Se o documento não estiver criptografado, a exceção correspondente será gerada no .net 1.1 ou CryptoAlgorithm será nulo para outras versões do .net.

```csharp
public CryptoAlgorithm? CryptoAlgorithm { get; }
```
