## Método XImageCollection.Contains

Determina se a coleção contém um valor específico.

```csharp
public bool Contains(XImage item)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| item | XImage | O objeto a ser localizado na coleção. |

### Valor de Retorno

true se o item for encontrado na coleção; caso contrário, false.
