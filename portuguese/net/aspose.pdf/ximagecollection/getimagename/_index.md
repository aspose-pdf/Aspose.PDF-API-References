## Método XImageCollection.GetImageName

Retorna o nome na lista de imagens que é a chave da imagem dada.

```csharp
public string GetImageName(XImage image)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| image | XImage | Imagem a ser pesquisada. |

### Valor de Retorno

Nome (chave) da imagem encontrada; null se a imagem não foi encontrada.
