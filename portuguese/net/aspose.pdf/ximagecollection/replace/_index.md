## Replace(int, Stream, int, bool) {#replace_2}

Substituir imagem na coleção por outra imagem.

```csharp
public void Replace(int index, Stream stream, int quality, bool isBlackAndWhite)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| index | Int32 | Índice do item da coleção que será substituído no intervalo [1..contagem de imagens]. |
| stream | Stream | Stream contendo dados da imagem (no formato JPEG). |
| quality | Int32 | Qualidade da compressão JPEG, em porcentagem (valores válidos são 0..100). |
| isBlackAndWhite | Boolean | Se verdadeiro, a imagem é comprimida com o método de compressão CCITT, que fornece melhor compressão para imagens em preto e branco. Pode ser usado apenas para imagens em preto e branco. |
