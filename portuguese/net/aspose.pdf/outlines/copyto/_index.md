## Método Outlines.CopyTo

```csharp
public abstract void CopyTo(OutlineItemCollection[] array, int arrayIndex)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| array | OutlineItemCollection[] | O System.Array unidimensional que é o destino. Deve ter indexação baseada em zero. |
| arrayIndex | Int32 | O índice baseado em zero no array onde a cópia começa. |
