## Propriedade DocSaveOptions.ReSaveFonts

Obtém ou define o procedimento para re-salvar fontes. Se definido como verdadeiro, recarregamos fontes em cada página para evitar a influência de propriedades de fonte anteriores e carregamos a nova fonte criada do zero. Defina esta opção como falsa se você quiser melhorar o desempenho. O valor padrão é verdadeiro;

```csharp
public bool ReSaveFonts { get; set; }
```
