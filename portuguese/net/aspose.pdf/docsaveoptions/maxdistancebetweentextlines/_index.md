## Propriedade DocSaveOptions.MaxDistanceBetweenTextLines

Este parâmetro é usado para agrupar linhas de texto em parágrafos. Determina quão distantes podem estar duas linhas de texto relativas. Especificado em centenas de porcento da altura das linhas de texto.

public float MaxDistanceBetweenTextLines { get; set; }
