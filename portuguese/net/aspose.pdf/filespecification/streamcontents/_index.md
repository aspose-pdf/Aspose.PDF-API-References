## Propriedade FileSpecification.StreamContents

Obtém o conteúdo do arquivo como um fluxo. O conteúdo não é carregado na memória, o que permite diminuir o uso de memória. Mas este fluxo não suporta posicionamento e a propriedade Length. Se você precisar desses recursos, use a propriedade Contents em vez disso.

```csharp
public Stream StreamContents { get; }
```
