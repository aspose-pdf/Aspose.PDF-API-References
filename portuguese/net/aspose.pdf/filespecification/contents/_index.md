## Propriedade FileSpecification.Contents

Obtém ou define o arquivo de conteúdo. Esta propriedade retorna dados carregados na memória, o que pode causar uma exceção de falta de memória para grandes dados. Para diminuir o uso de memória, utilize StreamContents.

```csharp
public Stream Contents { get; set; }
```
