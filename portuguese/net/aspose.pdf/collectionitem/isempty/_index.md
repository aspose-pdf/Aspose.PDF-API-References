## Propriedade CollectionItem.IsEmpty

Obtém um valor que indica se o item da coleção está vazio.

```csharp
public bool IsEmpty { get; }
```

### Valor de Retorno

true se o item da coleção estiver vazio; caso contrário, false.

## Observações

Esta propriedade retorna true se o item da coleção não contiver nenhum valor, incluindo valores de string, valores duplos, valores inteiros e valores de data. Se algum desses tipos de valor estiver presente no item da coleção, esta propriedade retorna false.
