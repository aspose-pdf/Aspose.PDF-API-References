## Enumeração ParagraphPositioningMode

Especifica a variante para determinar a localização do elemento na página.

```csharp
public enum ParagraphPositioningMode
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Default | `0` | A localização é determinada pelos elementos previamente colocados. A adição de um elemento é levada em conta ao determinar a localização dos elementos subsequentes. |
| Absolute | `1` | A localização é especificada pelos valores Left e Top, não depende de elementos anteriores e não afeta a localização dos subsequentes. |
