## Propriedade TextStamp.WordWrap

Define a quebra de palavras. Se esta propriedade for definida como verdadeira e o valor da Largura for especificado, o texto será quebrado em várias linhas para se ajustar à largura especificada. Valor padrão: falso.

public bool WordWrap { get; set; }
