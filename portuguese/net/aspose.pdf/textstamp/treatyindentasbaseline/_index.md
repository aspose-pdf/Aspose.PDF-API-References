## Propriedade TextStamp.TreatYIndentAsBaseLine

Define a origem das coordenadas para posicionar o texto. Se TreatYIndentAsBaseLine = true (padrão quando Draw = true) o valor de YIndent será tratado como a linha de base do texto. Se TreatYIndentAsBaseLine = false (padrão quando Draw = false) o valor de YIndent será tratado como a linha de descida inferior do texto.

```csharp
public bool TreatYIndentAsBaseLine { get; set; }
```
