## Propriedade TextStamp.Draw

Esta propriedade determina como o carimbo é desenhado na página. Se Draw = true, o carimbo é desenhado como operadores gráficos e se draw = false, então o carimbo é desenhado como texto.

```csharp
public bool Draw { get; set; }
```
