## Propriedade TextStamp.AutoAdjustFontSizeToFitStampRectangle

Se habilitada, o tamanho da fonte será ajustado automaticamente para caber no retângulo do carimbo de tamanho: [`Width`](../width/) e [`Height`](../height/). A largura e altura padrão são derivadas do retângulo da página.

```csharp
public bool AutoAdjustFontSizeToFitStampRectangle { get; set; }
```
