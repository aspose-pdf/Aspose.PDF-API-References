## Propriedade SaveOptions.CacheGlyphs

Obtém ou define um valor booleano que indica se os glifos da fonte serão armazenados em cache ao preparar páginas aps. Melhora o desempenho da conversão de pdf para outros formatos, mas aumenta o consumo de memória.

```csharp
public bool CacheGlyphs { get; set; }
```
