## Propriedade SaveOptions.WarningHandler

Callback para lidar com quaisquer avisos gerados. O WarningHandler retorna um item do enum ReturnAction especificando Continue ou Abort. Continue é a ação padrão e a operação de Save continua, no entanto, o usuário também pode retornar Abort, caso em que a operação de Save deve cessar.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
