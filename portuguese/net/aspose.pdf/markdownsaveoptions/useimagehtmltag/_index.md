## Propriedade MarkdownSaveOptions.UseImageHtmlTag

Obtém e define a permissão para usar uma tag img para inserir imagens à esquerda e à direita do texto. Nesse caso, no visualizador de markdown, o texto irá se ajustar em torno da imagem.

```csharp
public bool UseImageHtmlTag { get; set; }
```
