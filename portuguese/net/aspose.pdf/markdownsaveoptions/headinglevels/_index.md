## Propriedade MarkdownSaveOptions.HeadingLevels

Define os níveis de cabeçalho esperados a serem usados na estratégia de reconhecimento de cabeçalhos FontSize. Se o valor desta propriedade estiver definido, a estratégia de reconhecimento de cabeçalho !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic será selecionada quando as estratégias !:PdfToMarkdown.HeadingRecognitionStrategy.Auto estiverem definidas, mesmo que o documento contenha marcadores.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
