## Enumeração DigestHashAlgorithm

Representa o tipo de algoritmo que mapeia dados para um "hash"

```csharp
public enum DigestHashAlgorithm
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| Auto | `0` | Configuração automática do algoritmo de hash a critério do algoritmo de assinatura. Para EDCSA, o valor padrão é determinado pelo tamanho da chave. O valor padrão para um PKCS7 não destacado é Sha1. |
| Sha1 | `1` | SHA-1. Algoritmo de Hash Seguro 1. É um valor padrão para um PKCS7 não destacado. |
| Sha256 | `2` | SHA-256. Algoritmo de Hash Seguro 2. É um valor padrão para um PKCS7 destacado. |
| Sha384 | `3` | SHA-384. Algoritmo de Hash Seguro 2 |
| Sha512 | `4` | SHA-512. Algoritmo de Hash Seguro 2 |
| Sha3_256 | `5` | SHA3-256. Algoritmo de Hash Seguro 3 |
| Sha3_384 | `6` | SHA3-384. Algoritmo de Hash Seguro 3 |
| Sha3_512 | `7` | SHA3-512. Algoritmo de Hash Seguro 3 |
