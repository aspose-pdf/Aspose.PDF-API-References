## Enumeração CollectionFieldSubtype

Representa o parâmetro de subtipo de um campo em uma coleção de esquema.

```csharp
public enum CollectionFieldSubtype
```

### Valores

| Nome | Valor | Descrição |
| --- | --- | --- |
| None | `0` | O subtipo não está definido. |
| S | `1` | Um tipo de texto. Os dados do campo devem ser armazenados como uma string de texto PDF. |
| D | `2` | Um tipo de data. Os dados do campo devem ser armazenados como uma string de data PDF. |
| N | `3` | Um tipo de número. Os dados do campo devem ser armazenados como um número PDF. |
| F | `4` | Os dados do campo devem ser o nome do arquivo do fluxo de arquivo incorporado, conforme identificado pela entrada UF da especificação do arquivo, se presente; caso contrário, pela entrada F da especificação do arquivo |
| Desc | `5` | Os dados do campo devem ser o nome do arquivo do fluxo de arquivo incorporado, conforme identificado pela entrada UF da especificação do arquivo, se presente; caso contrário, pela entrada F da especificação do arquivo |
| ModDate | `6` | Os dados do campo devem ser a data de modificação do fluxo de arquivo incorporado, conforme identificado pela entrada ModDate no dicionário de parâmetros do arquivo incorporado. |
| CreationDate | `7` | Os dados do campo devem ser a data de criação do fluxo de arquivo incorporado, conforme identificado pela entrada CreationDate no arquivo incorporado |
| Size | `8` | Os dados do campo devem ser o tamanho do arquivo incorporado, conforme identificado pela entrada Size no dicionário de parâmetros do arquivo incorporado |
| CompressedSize | `9` | (PDF 2.0) Os dados do campo são o comprimento do fluxo de arquivo incorporado, conforme identificado pela entrada Length no dicionário do fluxo de arquivo incorporado, e os dois valores devem ser idênticos. |
