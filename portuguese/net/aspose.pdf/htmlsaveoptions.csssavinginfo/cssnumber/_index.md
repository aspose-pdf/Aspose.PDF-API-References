## Campo HtmlSaveOptions.CssSavingInfo.CssNumber

Definido pelo conversor. Durante a conversão, vários arquivos CSS são criados. Esta propriedade mostra a ordem do arquivo CSS salvo durante a conversão. Pode ser usada na lógica do código personalizado para decidir como processar ou onde salvar o conteúdo CSS

```csharp
public int CssNumber;
```
