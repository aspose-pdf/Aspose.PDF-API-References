## Método ITaggedContent.SetLanguage

Define a língua natural para o documento pdf.

Um identificador de língua que deve especificar a língua natural para todo o texto no documento, exceto onde for substituído por especificações de língua para elementos de estrutura ou conteúdo marcado.

```csharp
public void SetLanguage(string lang)
```

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| lang | String | Um identificador de língua deve ser ou a string de texto vazia, para indicar que a língua é desconhecida, ou um Language-Tag conforme definido na RFC 3066, Tags for the Identification of Languages. |
