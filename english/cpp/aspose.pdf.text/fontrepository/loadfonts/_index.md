## FontRepository::LoadFonts method


Loads system installed fonts and standard [Pdf](../../../aspose.pdf/) fonts. This method was designed to speed up font loading process. By default fonts are loaded on first request for any font. Use of this method loads system and standard [Pdf](../../../aspose.pdf/) fonts immediately before any [Pdf](../../../aspose.pdf/) document was open.
