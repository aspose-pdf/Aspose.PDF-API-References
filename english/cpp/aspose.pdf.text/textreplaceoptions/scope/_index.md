```cpp
enum class Scope
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| REPLACE_FIRST | 0 | Replace only first occurrence of the text on each of affected pages. |
| REPLACE_ALL | 1 | Replace all text occurrences on all affected pages. |
