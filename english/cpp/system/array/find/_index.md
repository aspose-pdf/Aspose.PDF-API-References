## Array::Find method

| Parameter | Type | Description |
| --- | --- | --- |
| arr | System::ArrayPtr\<T\> | [Array](../) to search an element in |
| match | System::Predicate\<T\> | A predicate that defines the conditions to match array elements against |

### ReturnValue
