## MarkdownSaveOptions::set_HeadingLevels method


Defines expected heading levels to use in FontSize recognition headers strategy. If this property value is set, then header recognition [PdfToMarkdown::HeadingRecognitionStrategy::Heuristic](../../headingrecognitionstrategy/) strategy will be selected when set [PdfToMarkdown::HeadingRecognitionStrategy::Auto](../../headingrecognitionstrategy/) strategies even if the document contains bookmarks.
