## HtmlFragment::set_HtmlLoadOptions method


Sets [HtmlLoadOptions](../../htmlloadoptions/) that will be used for loading (and rendering) of HTML into this instance of class. Please use it when it's necessary use specific setting for import of HTML for this or that instance (f.e when this or that instance should use specific BasePath for imported HTML or should use specific loader of external resources) If parameter is default (null), then standard HTML loading options will be used.
