## CollectionField class


Represents a document collection schema field class.

```cpp
class CollectionField : public System::Object
```

## Methods

| Method | Description |
| --- | --- |
| [get_E](./get_e/)() const | Gets a flag indicating whether the interactive PDF processor should provide support for editing the field value. Default value: false. |
| [get_FiledType](./get_filedtype/)() const | Gets the type of a field value in a schema collection. This field describes the value type corresponding to [Subtype](../). |
| [get_N](./get_n/)() const | Gets the textual field name that shall be presented to the user by the interactive PDF processor. |
| [get_O](./get_o/)() const | Gets the relative order of the field name in the user interface. Fields shall be sorted by the interactive PDF processor in ascending order. |
| [get_Subtype](./get_subtype/)() const | Gets the subtype of a field value in a schema collection. The subtype of collection field or file-related field that this dictionary describes. This entry identifies the type of data that shall be stored in the field. |
| [get_V](./get_v/)() const | Gets the initial visibility of the field in the user interface. Default value: true. |
