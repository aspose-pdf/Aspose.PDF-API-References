## Form::get_NeedsRendering method


Gets a value indicating whether the document requires the removal of the dynamic [XFA](../../xfa/) form. This property was introduced to determine if [IgnoreNeedsRendering](../) should be used to remove the [XFA](../../xfa/) form in cases where the [XFA](../../xfa/) form is present and [NeedsRendering](../) is false.
