## XmlSchemaMinLengthFacet class


Represents the **minLength** element from XML [Schema](../) as specified by the World Wide [Web](../../system.web/) Consortium (W3C). This class can be used to specify a restriction on the minimum length of the data value of a **simpleType** element. The length must be greater than the value of the **minLength** element.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
