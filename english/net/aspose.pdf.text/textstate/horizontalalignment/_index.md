## TextState.HorizontalAlignment property

Gets or sets horizontal alignment for the text.

```csharp
public virtual HorizontalAlignment HorizontalAlignment { get; set; }
```

## Remarks

HorizontalAlignment.None is equal to HorizontalAlignment.Left. Note that TextState.HorizontalAlignment property works in new document generation scenarios only.
