## AppearanceDictionary.Keys property

Gets keys of the dictionary. If appearance dictionary has subditionaries, then `Keys` contains (N&#x7C;R&#x7C;D).state values, where N - normal appearance, R - rollover appearance, D - down appearance and state - the name of the state (e.g. On, Off for checkboxes).
