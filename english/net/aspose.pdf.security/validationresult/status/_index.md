## ValidationResult.Status property

Gets the status of the validation process for a certificate.


## Remarks

The Status property indicates the outcome of the certificate validation. Possible values are defined in the [`ValidationStatus`](../../validationstatus/) enumeration, such as Valid, Invalid, or Undefined. It provides an insight into whether the certificate passed the validation checks or not.
