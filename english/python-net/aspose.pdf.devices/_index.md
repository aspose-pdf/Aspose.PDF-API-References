The aspose.pdf.devices namespace provides classes which are used for representing document as image(s) or a plain text. So document can be sent on to textual or various graphic devices which means we want to get it textual or graphic representation.

## Classes
| Class | Description |
| :- | :- |
|[BmpDevice](/pdf/python-net/aspose.pdf.devices/bmpdevice/)|Represents image device that helps to save pdf document pages into bmp.|
|[Device](/pdf/python-net/aspose.pdf.devices/device/)|Abstract class for all types of devices. Device is used to represent pdf document in some format.<br/>            For example, document page can be represented as image or text.|
|[DicomDevice](/pdf/python-net/aspose.pdf.devices/dicomdevice/)|Represents image device that helps to save pdf document pages into Dicom format.|
|[DocumentDevice](/pdf/python-net/aspose.pdf.devices/documentdevice/)|Abstract class for all devices which is used to process the whole pdf document.|
|[EmfDevice](/pdf/python-net/aspose.pdf.devices/emfdevice/)|Represents image device that helps to save pdf document pages into emf.|
|[GifDevice](/pdf/python-net/aspose.pdf.devices/gifdevice/)|Represents image device that helps to save pdf document pages into gif.|
|[ImageDevice](/pdf/python-net/aspose.pdf.devices/imagedevice/)|An abstract class for image devices.|
|[JpegDevice](/pdf/python-net/aspose.pdf.devices/jpegdevice/)|Represents image device that helps to save pdf document pages into jpeg.|
|[Margins](/pdf/python-net/aspose.pdf.devices/margins/)|This class represents margins of an image.|
|[PageDevice](/pdf/python-net/aspose.pdf.devices/pagedevice/)|Abstract class for all devices which is used to process certain page the pdf document.|
|[PngDevice](/pdf/python-net/aspose.pdf.devices/pngdevice/)|Represents image device that helps to save pdf document pages into png.|
|[Resolution](/pdf/python-net/aspose.pdf.devices/resolution/)|Represents class for holding image resolution.|
|[TextDevice](/pdf/python-net/aspose.pdf.devices/textdevice/)|Represents class for converting pdf document pages into text.|
|[ThumbnailDevice](/pdf/python-net/aspose.pdf.devices/thumbnaildevice/)|Represents image device that save pdf document pages into Thumbnail image.|
|[TiffDevice](/pdf/python-net/aspose.pdf.devices/tiffdevice/)|This class helps to save pdf document page by page into the one tiff image.|
|[TiffSettings](/pdf/python-net/aspose.pdf.devices/tiffsettings/)|This class represents settings for importing pdf to Tiff.|
