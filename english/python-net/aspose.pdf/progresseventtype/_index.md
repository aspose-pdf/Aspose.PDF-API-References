## ProgressEventType enumeration

This enum describes possible progress event types<br/>            that can occure during conversion

## Members
| Member name | Description |
| :- | :- |
|TOTAL_PROGRESS|means that occured event informs about total progress of conversion|
|SOURCE_PAGE_ANALYSED|means that occured event informs about end of analysis of one of pages before conversion|
|RESULT_PAGE_CREATED|means that occured event informs about creation of one result page before phisical export|
|RESULT_PAGE_SAVED|means that occured event informs about finished saving of one result page|
