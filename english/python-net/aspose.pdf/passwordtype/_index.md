## PasswordType enumeration

This enum represents  known password types used for password protected pdf documents.

## Members
| Member name | Description |
| :- | :- |
|NONE|Pdf document is not password protected.|
|USER|Pdf document was opened using document open password (restricted access).|
|OWNER|Pdf document was opened using change permissions password (full access).|
|INACCESSIBLE|Pdf document is password protected but both user and owner passwords are not empty and <br/>            none of the passwords was defined or supplied password was incorrect. So it impossible to deduce the type of the password.|
