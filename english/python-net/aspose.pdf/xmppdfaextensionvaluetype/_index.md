## XmpPdfAExtensionValueType class

The PDF/A ValueType schema is required for all property value types which are not defined in the XMP 2004 specification, i.e. for value types outside of the following list:<br/>            - Array types (these are container types which may contain one or more fields): Alt, Bag, Seq<br/>            - Basic value types: Boolean, (open and closed) Choice, Date, Dimensions, Integer, Lang Alt, Locale, MIMEType, ProperName, Real, Text, Thumbnail, URI, URL, XPath<br/>            - Media Management value types: AgentName, RenditionClass, ResourceEvent, ResourceRef, Version<br/>            - Basic Job/Workflow value type: Job<br/>            - EXIF schema value types: Flash, CFAPattern, DeviceSettings, GPSCoordinate, OECF/SFR, Rational<br/>            Schema namespace URI: http://www.aiim.org/pdfa/ns/type#<br/>            Required schema namespace prefix: pdfaType
