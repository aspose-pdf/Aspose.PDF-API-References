## PageCoordinateType enumeration

Describes page coordinate type.

## Members
| Member name | Description |
| :- | :- |
|MEDIA_BOX|The MediaBox is used to specify the width and height of the page. <br/>            For the average user, this probably equals the actual page size. <br/>            The MediaBox is the largest page box in a PDF. The other page boxes can equal the <br/>            size of the MediaBox but they cannot be larger.|
|CROP_BOX|The CropBox defines the region to which the page contents are to be clipped. <br/>            Acrobat uses this size for screen display and printing.|
