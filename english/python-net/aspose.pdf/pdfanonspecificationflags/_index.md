## PdfANonSpecificationFlags class

This class holds flags to control PDF/A conversion for cases when source PDF document doesn't <br/>            correspond to PDF specification. If flags of this clas are used it decreases performance <br/>            but it's necessary when source PDF document can't be convert into PDF/A format by usual way. <br/>            By default all flags are set to false.
