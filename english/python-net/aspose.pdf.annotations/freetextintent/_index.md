## FreeTextIntent enumeration

Enumerates the intents of the free text annotation.

## Members
| Member name | Description |
| :- | :- |
|UNDEFINED|Not defined state.|
|FREE_TEXT_CALLOUT|Means that the annotation is intended to function as a callout.|
|FREE_TEXT_TYPE_WRITER|Means that the annotation is intended to function as a click-to-type or typewriter object.|
