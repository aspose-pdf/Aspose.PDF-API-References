## PolyIntent enumeration

Enumerates the intents of the polygon or polyline annotation.

## Members
| Member name | Description |
| :- | :- |
|UNDEFINED|Undefined state.|
|POLYGON_CLOUD|Means that the annotation is intended to function as a cloud object.|
|POLY_LINE_DIMENSION|Indicates that the polyline annotation is intended to function as a dimension.|
|POLYGON_DIMENSION|Indicates that the polygon annotation is intended to function as a dimension.|
