## DocumentPrivilege class

Represents the privileges for accessing Pdf file. Refer to[PdfFileSecurity](/pdf/python-net/aspose.pdf.facades/pdffilesecurity/).<br/>             There are 4 ways using this class:<br/>             1.Using predefined privilege directly.<br/>             2.Based on a predefined privilege and change some specifical permissions.<br/>             3.Based on a predefined privilege and change some specifical Adobe Professional permissions combination.<br/>             4.Mixes the way2 and way3.
