## SubmitFormFlag enumeration

Enumeration of possible submit form flags.

## Members
| Member name | Description |
| :- | :- |
|FDF|Data will be returned in FDF format.|
|HTML|Data will be returned in HTML format.|
|XFDF|Data will be returned in XFDF format.|
|FDF_WITH_COMMENTS|Data will be returned in FDF format with comments.|
|XFDF_WITH_COMMENTS|Data will be returned in XFDF format with comments.|
|PDF|Whole PDF file will be submitted.|
