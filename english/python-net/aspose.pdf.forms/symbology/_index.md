## Symbology enumeration

A (Barcode) Symbology defines the technical details of a particular type of barcode:<br/>            the width of the bars, character set, method of encoding, checksum specifications, etc.

## Members
| Member name | Description |
| :- | :- |
|PDF417|A multi-row, variable-length 2D symbology with high data capacity and error-correction capability.|
|QR_CODE|QR code (short for Quick Response) is a specific matrix barcode (2D code), <br/>            readable by dedicated QR barcode readers and camera phones.|
|DATA_MATRIX|A 2D matrix symbology consisting of black and white square modules arranged in either a square or <br/>            rectangular pattern.|
