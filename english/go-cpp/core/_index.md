## Core PDF functions

|      Name      |  Description   |
| -------------- | -------------- |
| [New](./new/) | Create a new PDF-document. |
| [Open](./open/) | Open a PDF-document with filename. |
| [Save](./save/) | Save the previously opened PDF-document. |
| [SaveAs](./saveas/) | Save the previously opened PDF-document with new filename. |
| [Close](./close/) | Release allocated resources for PDF-document. |
| [SetLicense](./setlicense/) | Set license with filename. |
| [PageAdd](./pageadd/) | Add new page in PDF-document. |
| [PageInsert](./pageinsert/) | Insert new page at the specified position in PDF-document. |
| [PageDelete](./pagedelete/) | Delete specified page in PDF-document. |
| [PageCount](./pagecount/) | Return page count in PDF-document. |
| [ExtractText](./extracttext/) | Return the PDF-document contents as plain text. |


## Detailed Description

Core functions for working with PDF files.
