## XfaParserOptions.EmulateRequierdGroups özelliği

Bu özellik doğruysa, gerekli Xfa "hariç gruplar" için ek kırmızı dikdörtgenler çizilecektir. Bu özellik, formların Xfa temsilinin standart hale dönüştürülmesi sırasında hariç tutulan grupların analoğunun yokluğundan dolayı tanıtılmıştır. Varsayılan olarak yanlıştır.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
