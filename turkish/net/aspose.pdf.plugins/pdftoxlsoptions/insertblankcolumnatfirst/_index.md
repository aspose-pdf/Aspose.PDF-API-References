## PdfToXlsOptions.InsertBlankColumnAtFirst özelliği

Eğer boş sütunun çalışma sayfasının ilk sütunu olarak eklenmesini istiyorsanız true olarak ayarlayın. Varsayılan değer false'tur; bu da boş sütunun eklenmeyeceği anlamına gelir.

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
