## TiffOptions.Brightness özelliği

Beyaz ve siyah renklerin dönüşümünde bir değer sınırı alır veya ayarlar. Bu parametre, EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle veya ColorDepth.Format1bpp ile uygulanabilir == 1

```csharp
public float Brightness { get; set; }
```

### Özellik Değeri

Parlaklık değeri 0 ile 1 arasında olmalıdır. Varsayılan değer 0.33f'e eşittir.
