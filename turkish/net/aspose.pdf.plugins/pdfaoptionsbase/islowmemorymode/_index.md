## PdfAOptionsBase.IsLowMemoryMode özelliği

PDF/A dönüşüm sürecinde düşük bellek modunun etkin olup olmadığını belirten bir değeri alır veya ayarlar.

```csharp
public bool IsLowMemoryMode { get; set; }
```

### Özellik Değeri

`true` düşük bellek modu etkinse; aksi takdirde `false`.
