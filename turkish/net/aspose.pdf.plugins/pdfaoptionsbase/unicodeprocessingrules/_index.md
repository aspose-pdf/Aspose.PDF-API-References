## PdfAOptionsBase.UnicodeProcessingRules özelliği

```csharp
public ToUnicodeProcessingRules UnicodeProcessingRules { get; set; }
```

### Özellik Değeri

Sorunlu Unicode karakterleri ile başa çıkmak için kuralları tanımlayan bir [`ToUnicodeProcessingRules`](../../../aspose.pdf/tounicodeprocessingrules/) örneği.
