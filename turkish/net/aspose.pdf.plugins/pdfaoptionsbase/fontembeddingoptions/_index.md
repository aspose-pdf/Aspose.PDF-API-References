## PdfAOptionsBase.FontEmbeddingOptions özelliği

Belgeye gömülemeyen yazı tiplerini işlemek için seçenekleri alır.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Özellik Değeri

Yazı tipi gömme seçenekleri.

## Açıklamalar

PDF/A standardı, tüm yazı tiplerinin belgeye gömülmesi gerektiğini gerektirir. Bu özellik, bazı yazı tiplerinin hedef PC'de mevcut olmaması nedeniyle gömülmesinin mümkün olmadığı durumları yönetmek için seçenekler sunar.
