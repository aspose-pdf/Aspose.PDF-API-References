## PdfAOptionsBase.SymbolicFontEncodingStrategy özelliği

PDF/A formatına dönüştürülürken sembolik fontların kodlama stratejisini alır veya ayarlar.

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### Özellik Değeri

Sembolik font kodlama stratejisi.

## Açıklamalar

Bu özellik, kaynak belgede bulunan orijinal TrueType sembolik fontun birden fazla CMap alt tablosu içerdiği durumlarda, hangi CMap alt tablosunun sonuç belgesine kopyalanacağını kontrol etmenizi sağlar.
