## PdfToDocOptions.DönüşümModu özelliği

Bir PDF belgesinin nasıl bir kelime işleme belgesine dönüştürüleceğini kontrol etmeye olanak tanır.

```csharp
public ConversionMode ConversionMode { get; set; }
```

## Açıklamalar

Sonuçta oluşan belge daha fazla düzenlenmeyecekse TextBox modunu kullanın. TextBox'lar, yapılacak çok şey yoksa kolayca değiştirilebilir.

Çıktı belgesinin daha fazla düzenlenmesi gerekiyorsa Flow modunu kullanın. Flow modundaki paragraflar ve metin satırları, metni kolayca değiştirmeye olanak tanır, ancak desteklenmeyen biçimlendirme nesneleri, TextBox moduna göre daha kötü görünecektir.
