## TextReplaceOptions.ReplaceAdjustment enumerasyonu

Metin parçasının daha kısa bir şekilde değiştirilmesinden sonra yapılacak eylemi belirler. None - hiçbir eylem, değiştirilmiş metin satırın geri kalanıyla örtüşebilir; AdjustSpaceWidth - satır uzunluğunu korumak için kelimeler arasındaki boşlukları ayarlamaya çalışır; WholeWordsHyphenation - paragrafların doğru alanını korumak için kelimeleri paragraf satırları arasında dağıtmaya çalışır; ShiftRestOfLine - metnin değişen uzunluğuna göre satırın geri kalanını kaydırır, satırın uzunluğu değişebilir; Varsayılan değer ShiftRestOfLine'dır.

```csharp
[Flags]
public enum ReplaceAdjustment
```

### Değerler

| İsim | Değer | Açıklama |
| --- | --- | --- |
| None | `0` | Hiçbir eylem, değiştirilmiş metin satırın geri kalanıyla örtüşebilir |
| AdjustSpaceWidth | `1` | Satır uzunluğunu korumak için kelimeler arasındaki boşlukları ayarlamaya çalışır |
| WholeWordsHyphenation | `2` | Paragrafların doğru alanını korumak için kelimeleri paragraf satırları arasında dağıtmaya çalışır |
| IsFormFillingMode | `4` | Paragraf genişliğini kullanarak mevcut beyaz alanda kelimeleri yaymaya çalışır. Metin taşarsa, gizlenecektir. |
| ShiftRestOfLine | `8` | (Varsayılan) Metnin değişen uzunluğuna göre satırın geri kalanını kaydırır, satırın uzunluğu değişebilir |
