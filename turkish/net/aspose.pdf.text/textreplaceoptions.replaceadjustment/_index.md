## TextReplaceOptions.ReplaceAdjustment enumeration

Metin parçasının daha kısa olacak şekilde değiştirilmesinden sonra yapılacak eylemi belirler. Yok - eylem yok, değiştirilen metin satırın geri kalanıyla çakışabilir; AdjustSpaceWidth - satır uzunluğunu korumak için sözcükler arasındaki boşlukları ayarlamaya çalışır; WholeWordsHyphenation - sözcükleri dağıtmaya çalışır paragrafın sağ alanını korumak için paragraf satırları arasında; ShiftRestOfLine - değişen metnin uzunluğuna göre satırın geri kalanını kaydırır, satırın uzunluğu değiştirilebilir; Varsayılan değer ShiftRestOfLine'dır.

```csharp
[Flags]
public enum ReplaceAdjustment
```

### değerler

| İsim | Değer | Tanım |
| --- | --- | --- |
| None | `0` | Eylem yok, değiştirilen metin satırın geri kalanıyla çakışabilir |
| AdjustSpaceWidth | `1` | Satır uzunluğunu korumak için sözcükler arasındaki boşlukları ayarlamaya çalışır |
| WholeWordsHyphenation | `2` | Paragrafın sağ alanını korumak için sözcükleri paragraf satırları arasında dağıtmaya çalışır |
| ShiftRestOfLine | `4` | (Varsayılan) Değişen metnin uzunluğuna göre satırın geri kalanını kaydırır, satırın uzunluğu değişebilir |
