## ParagraphAbsorber class

Bölümler ve paragraflar gibi sayfa yapısı nesnelerinin bir emici nesnesini temsil eder. Metnin bölümleri ve paragrafları için arama yapar ve metin koordinat alanında onu tanımlayan dikdörtgenler ve polidonlara erişim sağlar. ### Notlar

Arama tamamlandığında[`PageMarkups`](./pagemarkups) koleksiyon şunları içerecektir[`PageMarkup`](../pagemarkup) koleksiyonlarına göre sayfa yapısını temsil eden nesneler[`MarkupSection`](../markupsection) ve[`MarkupParagraph`](../markupparagraph) . [`TextFragment`](../textfragment) nesne, arama oluşum metnine, metin özelliklerine erişim sağlar ve metni düzenlemeye ve metin durumunu (yazı tipi, yazı tipi boyutu, renk vb.) değiştirmeye izin verir.

### Örnekler

Örnek, ilk PDF belgesi sayfasında her paragrafın ilk metin bölümünün nasıl bulunacağını ve vurgulanacağını gösterir.

```csharp
// Belgeyi aç
Document doc = new Document("input.pdf");

// ParagraphAbsorber nesnesi oluştur
ParagraphAbsorber absorber = new ParagraphAbsorber();

// İlk sayfa için emiciyi kabul et
absorber.Visit(doc.Pages[1]);

// İlk sayfanın işaretleme nesnesini al
PageMarkup markup = absorber.PageMarkups[0];

// Her paragrafın ilk metin parçasını bulmak için sayfa metninin yapı öğeleri arasında dolaşın
foreach (MarkupSection section in markup.Sections)
{
    foreach (MarkupParagraph paragraph in section.Paragraphs)
    {
        TextFragment fragment = paragraph.Fragments[0];
        // Metin özelliklerini güncelle
        fragment.TextState.BackgroundColor = Color.LightBlue;
    }
}

// Belgeyi kaydet
doc.Save(GetOutputPath("output.pdf"));
```
