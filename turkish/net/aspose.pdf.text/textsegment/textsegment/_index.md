TextSegment nesnesi oluşturur.

```csharp
public TextSegment()
```

### Örnekler

Örnek, metin parçası nesnesinin nasıl oluşturulacağını, metin parçası koleksiyonuna bir metin parçasının nasıl ekleneceğini ve bunun Pdf sayfasına nasıl ekleneceğini gösterir.

```csharp
Document doc = new Document(inFile);
Page page = (Page)doc.Pages[1];

// metin parçası oluştur
TextFragment tf = new TextFragment("main text");
tf.Position = new Position(100, 600);

// metin özelliklerini ayarla
tf.TextState.FontSize = 5;
tf.TextState.Font = FontRepository.FindFont("TimesNewRoman");
tf.TextState.BackgroundColor = Color.LightGray;
tf.TextState.ForegroundColor = Color.Red;

// metin parçasının Segmentler koleksiyonuna bir segment daha ekleyin
TextSegment segment2 = new TextSegment();
segment2.Text = "another segment";

tf.Segments.Add(segment2);

// TextBuilder nesnesi oluştur
TextBuilder builder = new TextBuilder(page);

// metin parçasını Pdf sayfasına ekle
builder.AppendText(tf);

//belgeyi kaydet
doc.Save(outFile);
```
