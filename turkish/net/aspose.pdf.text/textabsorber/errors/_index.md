## TextAbsorber.Hatalar özelliği

[`TextExtractionError`](../../textextractionerror/) nesnelerinin listesi. Metin çıkarımı sırasında bulunan hatalar hakkında bilgi içerir. Hataları arama, yalnızca TextSearchOptions.LogTextExtractionErrors = true olduğunda gerçekleştirilecektir; ve bu performansı düşürebilir.

```csharp
public List<TextExtractionError> Errors { get; }
```
