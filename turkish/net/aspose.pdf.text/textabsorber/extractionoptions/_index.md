## TextAbsorber.ExtractionOptions property

Metin çıkarma seçeneklerini alır veya ayarlar.

```csharp
public virtual TextExtractionOptions ExtractionOptions { get; set; }
```

### Notlar

Metin biçimlendirme modunu tanımlamaya izin verir[`TextExtractionOptions`](../../textextractionoptions) çıkarma sırasında. Varsayılan modPure

### Örnekler

Örnek, Saf metin biçimlendirme modunun nasıl ayarlanacağını ve metin çıkarmanın nasıl gerçekleştirileceğini gösterir.

```csharp
// belgeyi aç
Document doc = new Document(inFile);

// biçimlendirme ile metni çıkarmak için TextAbsorber nesnesi oluşturun
TextAbsorber absorber = new TextAbsorber();

// saf metin biçimlendirme modunu ayarla
absorber.ExtractionOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure);

// tüm belge sayfaları için emiciyi kabul et
doc.Pages.Accept(absorber);

// ayıklanan metni al
string extractedText = absorber.Text;
```
