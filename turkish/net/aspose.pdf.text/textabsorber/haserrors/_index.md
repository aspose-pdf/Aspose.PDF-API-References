## TextAbsorber.HasErrors özelliği

Değer, metin çıkarımı sırasında hataların bulunup bulunmadığını gösterir. Hataları arama, yalnızca TextSearchOptions.LogTextExtractionErrors = true ise yapılacaktır; ve bu performansı düşürebilir.

```csharp
public bool HasErrors { get; }
```
