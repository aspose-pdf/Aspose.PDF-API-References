## TextExtractionOptions.ScaleFactor özelliği

Saf modda çıkarım sırasında yazı tipi boyutunu ölçeklendirmek için uygulanacak faktörü alır veya ayarlar. Daha düşük bir değer ayarlamak, çıkarılan metinde daha fazla boşluk olmasına yol açar. Varsayılan değer 1'dir - ölçeklendirme yok; Sıfır değerini ayarlamak, algoritmanın otomatik olarak ölçeklendirme seçmesine olanak tanır.

```csharp
public double ScaleFactor { get; set; }
```
