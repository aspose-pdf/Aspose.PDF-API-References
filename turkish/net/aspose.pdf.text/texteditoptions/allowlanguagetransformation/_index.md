## TextEditOptions.AllowLanguageTransformation özelliği

Metin ekleme veya düzenleme sırasında dil dönüşümünün kullanımına izin veren değeri alır veya ayarlar. true - dil dönüşümü gerekli olduğunda uygulanacaktır (varsayılan değer). false - dil dönüşümü UYGULANMAYACAKTIR.

```csharp
public bool AllowLanguageTransformation { get; set; }
```
