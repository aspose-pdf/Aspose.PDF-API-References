## Font.IsEmbedded property

Yazı tipinin gömülü olup olmadığını gösteren bir değer alır veya ayarlar.

```csharp
public bool IsEmbedded { get; set; }
```

### Örnekler

Aşağıdaki örnek, bir yazı tipinin nasıl bulunacağını, onu gömülü olarak işaretlemeyi, belge sayfasında metin aramayı ve metin yazı tipini değiştirmeyi gösterir.

```csharp
// Font oluştur ve gömülü olarak işaretle
Font font = FontRepository.FindFont("Arial");
font.IsEmbedded = true;

// belgeyi aç
Document doc = new Document(@"D:\Tests\input.pdf");

// tüm "merhaba dünya" metin oluşumlarını bulmak için TextFragmentAbsorber nesnesi oluşturun
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");
// ilk sayfa için emiciyi kabul et
doc.Pages[1].Accept(absorber);

// ilk metin oluşumu için yazı tipini değiştir
absorber.TextFragments[1].TextState.Font = font;

// belgeyi kaydet
doc.Save(@"D:\Tests\output.pdf"); 
```
