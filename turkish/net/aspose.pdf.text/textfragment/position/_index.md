### Örnekler

Örnek, aşağıdakilerle temsil edilen bir metnin yerleşiminin nasıl görüntüleneceğini gösterir.[`TextFragment`](../../textfragment) nesne.

```csharp
// Belgeyi aç
Document doc = new Document(@"D:\Tests\input.pdf");

// Tüm "merhaba dünya" metin oluşumlarını bulmak için TextFragmentAbsorber nesnesi oluşturun

// İlk sayfa için emiciyi kabul et

// İlk metin oluşumunun metin ve yerleşim bilgilerini görüntüleyin

Console.Out.WriteLine(string.Format("fragment text: {0}", firstOccurrence.Text));
Console.Out.WriteLine(string.Format("fragment X indent: {0}", firstOccurrence.Position.XIndent));

```
