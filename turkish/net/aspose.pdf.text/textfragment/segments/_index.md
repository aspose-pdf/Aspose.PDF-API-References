## TextFragment.Segments property

### Notlar

Birkaç kelimeyle,[`TextSegment`](../../textsegment) nesneler çocuklarıdır[`TextFragment`](../../textfragment) object. Gelişmiş kullanıcılar, daha karmaşık metin düzenleme senaryoları gerçekleştirmek için doğrudan segmentlere erişebilir. Ayrıntılar için lütfen şuna bakın:[`TextFragment`](../../textfragment) nesne açıklaması.

### Örnekler

Örnek, tüm[`TextSegment`](../../textsegment) içindeki nesneler[`TextFragment`](../../textfragment) .

```csharp
// Belgeyi aç
Document doc = new Document(@"D:\Tests\input.pdf");

// Tüm "merhaba dünya" metin oluşumlarını bulmak için TextFragmentAbsorber nesnesi oluşturun
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");

// İlk sayfa için emiciyi kabul et
doc.Pages[1].Accept(absorber);

// Tüm metin segmentlerinde gezinin ve metin ve yerleşim bilgilerini çıkarın
foreach (TextSegment segment in absorber.TextFragments[1].Segments)
{
    Console.Out.WriteLine(string.Format("segment text: {0}", segment.Text));
    Console.Out.WriteLine(string.Format("segment X indent: {0}", segment.Position.XIndent));
    Console.Out.WriteLine(string.Format("segment Y indent: {0}", segment.Position.YIndent));
}

```
