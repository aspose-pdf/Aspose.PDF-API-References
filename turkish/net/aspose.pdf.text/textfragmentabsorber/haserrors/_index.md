## TextFragmentAbsorber.HasErrors özelliği

Değer, metin çıkarımı sırasında hataların bulunup bulunmadığını gösterir. Hataları aramak yalnızca TextSearchOptions.LogTextExtractionErrors = true ise yapılacaktır; ve bu performansı düşürebilir.

```csharp
public bool HasErrors { get; }
```
