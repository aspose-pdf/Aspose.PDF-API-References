## TextFragmentState.CoordinateOrigin özelliği

Metin CoordinateOrigin'ini alır veya ayarlar. Eğer CoordinateOrigin Descender ise, metnin Y koordinatı fontun en düşük noktasına karşılık gelir. Eğer CoordinateOrigin BaseLine ise, metnin Y koordinatı fontun temel hattına karşılık gelir. Varsayılan değer Descender'dır. Eğer fontun Descent değeri çok büyükse, metin diğer fontlardan daha yüksek bir şekilde render edilebilir. Bu durumda, daha iyi metin render'ı için CoordinateOrigin BaseLine seçilebilir.

```csharp
public override CoordinateOrigin CoordinateOrigin { get; set; }
```
