## TextSearchOptions.SearchInAnnotations özelliği

Anotasyonlarda metin aramaya izin veren değeri alır veya ayarlar. true - metin Anotasyonlarda aranacaktır. false - Anotasyonlardaki metin TextFragmentAbsorber tarafından ayrıştırılmayacaktır.

```csharp
public bool SearchInAnnotations { get; set; }
```
