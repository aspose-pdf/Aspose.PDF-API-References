## TextSearchOptions.UseFontEngineEncoding özelliği

Metnin font motoru kodlaması kullanılarak aranacağını belirten bir değer alır veya ayarlar. true - font motoru kodlamasının kullanılacağını belirtir (metin araması belgedeki kusurlu kodlama nedeniyle başarısız olursa bunu deneyin) false - belgenin font kodlamasının kullanılacağını belirtir (varsayılan değer)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
