## TextSearchOptions.IgnoreResourceFontErrors özelliği

Yazı (parça) emici tarafından fontun yokluğuna ilişkin hataların göz ardı edileceğini belirten bir değer alır veya ayarlar. true - font yokluğu hatalarının göz ardı edileceği anlamına gelir. Yanlış kaynaklara atıfta bulunan metin segmentleri işleme sırasında atlanacaktır. false (varsayılan) - font yokluğu hatası, istisna fırlatarak işlemi sonlandıracaktır.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
