## TextSearchOptions.LogTextExtractionErrors özelliği

Metin çıkarma (kodlama) hatalarının metin (parça) emiciye kaydedileceğini belirten bir değer alır veya ayarlar. true - metin çıkarma (kodlama) hatalarının kaydedileceği anlamına gelir. Performansı düşürebilir. false (varsayılan) - hata kaydı yoktur.

```csharp
public bool LogTextExtractionErrors { get; set; }
```
