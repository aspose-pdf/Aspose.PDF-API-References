## TextSearchOptions.IgnoreShadowText özelliği

Normal metnin gölgesini temsil eden metin parçalarının arama sırasında göz ardı edileceğini belirten bir işaret alır veya ayarlar. true - gölge metnin bulunmayacağı anlamına gelir (metin araması yakın konumlarda tekrar eden parçalar döndürüyorsa bunu deneyin) false - gölge metnin normal metinle birlikte bulunacağı anlamına gelir (varsayılan değer)

```csharp
public bool IgnoreShadowText { get; set; }
```
