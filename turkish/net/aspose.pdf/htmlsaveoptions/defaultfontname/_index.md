## HtmlSaveOptions.DefaultFontName özelliği

Sistemde yüklü olmayan ve gömülü olmayan herhangi bir belge fontunu değiştirmek için kullanılan yüklü bir fontun adını belirtir. Null ise varsayılan değiştirme fontu kullanılır.

```csharp
public string DefaultFontName { get; set; }
```
