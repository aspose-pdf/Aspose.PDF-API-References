## HtmlSaveOptions.CssClassNamesPrefix alanı

PDFtoHTML dönüştürücüsü sonuç CSS'leri oluşturduğunda, CSS sınıf adları (".stl_01 {}" ... ".stl_NN {}" gibi) oluşturulur ve sonuç CSS'sinde kullanılır. Bu özellik, zorla sınıf adı ön eki ayarlamaya olanak tanır. Örneğin, tüm sınıf adlarının 'my_prefix_' ile başlamasını istiyorsanız (yani 'my_prefix_1' ... 'my_prefix_NNN' gibi), bu özelliğe 'my_prefix_' atamanız yeterlidir. Bu özellik dokunulmamış kalırsa (yani değeri null olarak bırakılırsa), dönüştürücü sınıf adlarını kendisi oluşturacaktır (bu, ".stl_01 {}" ... ".stl_NN {}" gibi bir şey olacaktır).

```csharp
public string CssClassNamesPrefix;
```
