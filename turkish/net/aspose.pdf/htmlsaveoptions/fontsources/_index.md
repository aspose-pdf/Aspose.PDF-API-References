## HtmlSaveOptions.FontSources özelliği

Önceden kaydedilmiş fontların kaynakları.

```csharp
public FontSourceCollection FontSources { get; }
```

## Açıklamalar

Fontlar, öncelikle önbellek amacıyla kaydedilebilir ve ardından Html dönüştürme sürecine geçirilebilir. Örneğin, bu, belge bölme senaryosunda ve belge sayfalarını tek bir font seti ile birden fazla iş parçacığında işleme alırken faydalı olabilir.
