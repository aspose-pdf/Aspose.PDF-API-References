## HtmlSaveOptions.SaveFullFont özelliği

Tam fontun kaydedileceğini belirtir, yalnızca True Type Font'ları destekler. Varsayılan olarak SaveFullFont = false'dur ve dönüştürücü, belgenin metnini görüntülemek için gereken başlangıç fontunun alt kümesini kaydeder.

```csharp
public bool SaveFullFont { get; set; }
```
