## HtmlSaveOptions.ConvertMarkedContentToLayers özelliği

ConvertMarkedContentToLayers niteliği true olarak ayarlandığında, bir PDF işaretli içerik (katman) içindeki tüm öğeler, "data-pdflayer" niteliğine sahip bir HTML div'ine yerleştirilecektir. Bu katman adı, PDF işaretli içeriğin isteğe bağlı özelliklerinden çıkarılacaktır. Bu nitelik false (varsayılan olarak) ise, PDF işaretli içerikten hiçbir katman oluşturulmayacaktır.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
