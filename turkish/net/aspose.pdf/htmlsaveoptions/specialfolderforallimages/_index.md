## HtmlSaveOptions.SpecialFolderForAllImages alanı

HTML olarak belge kaydedilirken karşılaşılan herhangi bir görüntünün kaydedilmesi gereken dizinin yolunu alır veya ayarlar. Parametre boş veya null ise, görüntü dosyaları (varsa) HTML ile bağlantılı diğer dosyalarla birlikte kaydedilecektir. İlgili görüntü dosyasını işlemek için CustomImageSavingStrategy özelliği başarıyla kullanıldıysa, bu durum hiçbir şeyi etkilemez.

```csharp
public string SpecialFolderForAllImages;
```
