## HtmlSaveOptions.MinimalLineWidth özelliği

Bu nitelik grafik yol çizgisinin minimum genişliğini ayarlar. Eğer çizgi kalınlığı 1px'den az ise Adobe Acrobat bunu bu değere yuvarlar. Bu nedenle, bu nitelik HTML tarayıcıları için bu davranışı taklit etmek amacıyla kullanılabilir.

```csharp
public float MinimalLineWidth { get; set; }
```
