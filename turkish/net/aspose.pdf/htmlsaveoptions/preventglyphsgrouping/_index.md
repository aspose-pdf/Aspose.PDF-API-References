## HtmlSaveOptions.PreventGlyphsGrouping özelliği

Bu özellik, metin gliflerinin kelimeler ve dizeler halinde gruplanmayacağı modu açar. Bu mod, sayfadaki gliflerin konumlandırılması sırasında maksimum hassasiyeti korumaya olanak tanır ve müzik notaları veya birbirinden ayrı yerleştirilmesi gereken glifler içeren belgelerin dönüştürülmesi için kullanılabilir. Bu parametre, yalnızca FixedLayout özelliğinin değeri true olduğunda belgeye uygulanacaktır.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
