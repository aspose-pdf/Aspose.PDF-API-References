## HtmlSaveOptions.SpecialFolderForSvgImages alanı

Belge HTML olarak kaydedilirken karşılaşılan SVG görüntülerinin yalnızca kaydedilmesi gereken dizinin yolunu alır veya ayarlar. Parametre boş veya null ise, SVG dosyaları (varsa) diğer resim dosyalarıyla birlikte (çıkış dosyasının yanına) veya özel resim klasöründe (eğer SpecialImagesFolderIfAny seçeneğinde belirtilmişse) kaydedilecektir. İlgili resim dosyasını işlemek için CustomImageSavingStrategy özelliği başarıyla kullanıldıysa, bu durum hiçbir şeyi etkilemez.

```csharp
public string SpecialFolderForSvgImages;
```
