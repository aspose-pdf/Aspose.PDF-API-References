## HtmlSaveOptions.CustomCssSavingStrategy alanı

Bu alan, Pdf'den Html'ye dönüştürme sırasında (varsa) kullanılacak olan CSS'lerin, oluşturulan HTML belgesinin tamamı veya sayfalarıyla ilgili olarak kaydedilmesi için gereken kaydetme stratejisini içerebilir (birden fazla HTML sayfası oluşturulursa). CSS dosyasını belirli bir şekilde işlemek istiyorsanız, lütfen ilgili yöntemi oluşturun ve buraya oluşturduğunuz delegasyonu atayın.

```csharp
public CssSavingStrategy CustomCssSavingStrategy;
```
