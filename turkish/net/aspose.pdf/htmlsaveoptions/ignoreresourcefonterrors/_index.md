## HtmlSaveOptions.IgnoreResourceFontErrors özelliği

Yazı tipinin yokluğuna ilişkin hataların göz ardı edileceğini belirtir. true - yazı tipinin yokluğuna ilişkin hataların göz ardı edileceği anlamına gelir. Yanlış kaynaklara atıfta bulunan metin segmentleri işleme sırasında atlanacaktır. varsayılan olarak false

public bool IgnoreResourceFontErrors { get; set; }
