## HtmlSaveOptions.RenderTextAsImage özelliği

RenderTextAsImage niteliği true olarak ayarlandığında, kaynak metin HTML'de bir resim haline gelir. Metni seçilemez hale getirmek veya HTML metni düzgün bir şekilde render edilmediğinde faydalı olabilir.

```csharp
public bool RenderTextAsImage { get; set; }
```
