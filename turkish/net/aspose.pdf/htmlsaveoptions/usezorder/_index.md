## HtmlSaveOptions.UseZOrder özelliği

UseZOrder niteliği true olarak ayarlandığında, grafikler ve metin, orijinal PDF belgesindeki Z sırasına göre sonuç HTML belgesine eklenir. Bu nitelik false ise, tüm grafikler tek bir katman olarak yerleştirilir, bu da üst üste binen nesneler için bazı gereksiz etkilere neden olabilir.

```csharp
public bool UseZOrder { get; set; }
```
