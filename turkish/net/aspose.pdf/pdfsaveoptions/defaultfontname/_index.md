## PdfSaveOptions.DefaultFontName özelliği

Bilgisayarda mevcut olmayan yazı tipleri için varsayılan olarak kullanılan yazı tipi adı. PDF'ye kaydedilen PDF belgesi, belgede ve cihazda mevcut olmayan yazı tiplerini içeriyorsa, API bu yazı tiplerini varsayılan yazı tipi ile değiştirir (eğer cihazda `DefaultFontName` ile bulunan bir yazı tipi varsa).

```csharp
public string DefaultFontName { get; set; }
```
