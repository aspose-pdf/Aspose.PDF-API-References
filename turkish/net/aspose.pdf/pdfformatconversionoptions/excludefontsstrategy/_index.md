## PdfFormatConversionOptions.ExcludeFontsStrategy özelliği

Gereksiz fontları hariç tutma ve belge dosya boyutunu azaltma stratejileri. Bu parametre yalnızca [`OptimizeFileSize`](../optimizefilesize/) bayrağı true olarak ayarlandığında anlam kazanır. Varsayılan olarak SubsetFonts ve RemoveDuplicatedFonts stratejilerinin kombinasyonu kullanılır.

```csharp
public RemoveFontsStrategy ExcludeFontsStrategy { get; set; }
```
