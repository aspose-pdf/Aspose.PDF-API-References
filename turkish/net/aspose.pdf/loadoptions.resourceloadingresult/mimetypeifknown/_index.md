## LoadOptions.ResourceLoadingResult.MIMETypeIfKnown alanı

Yüklenen kaynağın MIME türü hakkında bilgi, dönüştürücü için bazen faydalıdır. Yükleme sonrasında MIME türü biliniyorsa, bu parametreye sağlayabilirsiniz. MIME türü bilinmiyorsa veya sağlanması gerekmiyorsa, lütfen parametreyi null olarak bırakın.

```csharp
public string MIMETypeIfKnown;
```
