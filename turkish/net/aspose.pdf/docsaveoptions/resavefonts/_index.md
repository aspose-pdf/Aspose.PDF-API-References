## DocSaveOptions.ReSaveFonts özelliği

Fontları yeniden kaydetme prosedürünü alır veya ayarlar. true olarak ayarlandığında, önceki font özelliklerinin etkisini önlemek için her sayfada fontları yeniden yükleriz ve yeni oluşturulan fontu sıfırdan yükleriz. Performansı artırmak istiyorsanız bu seçeneği false olarak ayarlayın. Varsayılan değer true'dur;

```csharp
public bool ReSaveFonts { get; set; }
```
