## DocSaveOptions.ConvertType3Fonts özelliği

Type3 yazı tipleri için dönüşümü alır veya ayarlar. Type 3 yazı tiplerinde, glifler grafik operatörlerinin akışlarıyla tanımlanmalıdır. Bu, DOC/DOCX çıktısında metin yerine resimler gördüğümüz anlamına gelir. Type3 yazı tiplerini TTF'ye dönüştürmek ve sonuç dosyasında metin almak için bu bayrağı true olarak ayarlayın.

```csharp
public bool ConvertType3Fonts { get; set; }
```
