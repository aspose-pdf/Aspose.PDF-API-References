## UnifiedSaveOptions.ExtractOcrSublayerOnly özelliği

Bu özellik, OCR alt katmanına sahip PDF belgeleri için resim veya metin çıkarma işlevselliğini etkinleştirir.

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### Özellik Değeri

`true` metin sonuç belgesinde çıkarılacaktır; aksi takdirde, `false`.
