## XslFoLoadOptions.XsltArgumentList özelliği

Mevcut xls parametrelerine değer eklemek için XsltArgumentList. XLS dosyasında değeri olmayan 'animal' parametresi var: XsltArgumentList args = new XsltArgumentList(); args.AddParam("animal", "", "cat"); şimdi dönüştürücü, XLS dosyasında 'cat' değerine sahip bir 'animal' parametresi olduğunu varsayıyor.

```csharp
public XsltArgumentList XsltArgumentList { get; set; }
```
