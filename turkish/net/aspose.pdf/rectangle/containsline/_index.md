## Rectangle.ContainsLine metodu

Dikdörtgenin iki nokta ile temsil edilen bir çizgiyi içerip içermediğini belirler.

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| x1 | Double | Çizginin başlangıç noktasının X koordinatı. |
| y1 | Double | Çizginin başlangıç noktasının Y koordinatı. |
| x2 | Double | Çizginin bitiş noktasının X koordinatı. |
| y2 | Double | Çizginin bitiş noktasının Y koordinatı. |

### Dönüş Değeri

Dikdörtgen çizgiyi içeriyorsa `true`; aksi takdirde `false`.
