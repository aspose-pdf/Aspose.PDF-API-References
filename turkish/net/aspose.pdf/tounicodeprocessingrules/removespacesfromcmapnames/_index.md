## ToUnicodeProcessingRules.RemoveSpacesFromCMapNames özelliği

Bazı yazı tipleri, adlarında boşluklar bulunan ToUnicode karakter kodu haritalarına sahiptir. Bu boşluklar, unicode metin eşleştirmesi ile hatalara neden olabilir. Bu bayrak, ToUnicode karakter kodu haritalarının adlarından boşlukları kaldırmayı emreder. Varsayılan olarak false.

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
