## OptimizedMemoryStream.CanWrite özelliği

Türetilmiş bir sınıfta geçersiz kılındığında, mevcut akışın yazmayı destekleyip desteklemediğini belirten bir değer alır.

```csharp
public override bool CanWrite { get; }
```

### Dönüş Değeri

Akış yazmayı destekliyorsa true; aksi takdirde false.
