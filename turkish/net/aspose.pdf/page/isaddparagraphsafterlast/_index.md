## Page.IsAddParagraphsAfterLast özelliği

Sayfanın son paragrafından sonra paragrafların eklenmesini alır veya ayarlar

```csharp
public bool IsAddParagraphsAfterLast { get; set; }
```

### Özellik Değeri

Değer, sayfanın son paragrafından sonra paragrafların eklenip eklenmeyeceğini belirtir. Değer true ise, sayfanın son paragrafından sonra paragraflar eklenecektir.
