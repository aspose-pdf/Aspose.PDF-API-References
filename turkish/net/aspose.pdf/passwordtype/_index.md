## PasswordType enumerasyonu

Bu enum, şifre korumalı pdf belgeleri için kullanılan bilinen şifre türlerini temsil eder.

```csharp
public enum PasswordType
```

### Değerler

| İsim | Değer | Açıklama |
| --- | --- | --- |
| None | `0` | Pdf belgesi şifre korumalı değildir. |
| User | `1` | Pdf belgesi, belge açma şifresi (kısıtlı erişim) kullanılarak açılmıştır. |
| Owner | `2` | Pdf belgesi, izinleri değiştirme şifresi (tam erişim) kullanılarak açılmıştır. |
| Inaccessible | `3` | Pdf belgesi şifre korumalıdır ancak hem kullanıcı hem de sahip şifreleri boş değildir ve tanımlanan şifrelerden hiçbiri doğru değildir. Bu nedenle şifre türünü belirlemek imkansızdır. |
