## RenderingOptions.ConvertFontsToUnicodeTTF özelliği

Tüm fontların TTF unicode sürümlerine dönüştürüleceğini belirtir. Bu, uyumluluk nedenleri için ve font kullanımını optimize etmek için faydalıdır, çünkü her yeni TTF font, kaynak fonttan tüm sembolleri değil, yalnızca metinde kullanılan sembolleri içerecektir.

```csharp
public bool ConvertFontsToUnicodeTTF { get; set; }
```
