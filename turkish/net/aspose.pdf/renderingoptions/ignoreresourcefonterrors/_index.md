## RenderingOptions.IgnoreResourceFontErrors özelliği

Yazı tipinin yokluğuna ilişkin hataların göz ardı edileceğini belirten bir değer alır veya ayarlar. true - yazı tipinin yokluğuna ilişkin hataların göz ardı edileceği anlamına gelir. Yanlış kaynaklara atıfta bulunan metin segmentleri işleme sırasında atlanacaktır. varsayılan olarak false

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
