## TextStamp.TreatYIndentAsBaseLine özelliği

Metin yerleştirmek için koordinat başlangıcını tanımlar. Eğer TreatYIndentAsBaseLine = true (varsayılan olarak Draw = true olduğunda) YIndent değeri metin taban çizgisi olarak kabul edilecektir. Eğer TreatYIndentAsBaseLine = false (varsayılan olarak Draw = false olduğunda) YIndent değeri metnin alt (iniş çizgisi) olarak kabul edilecektir.

```csharp
public bool TreatYIndentAsBaseLine { get; set; }
```
