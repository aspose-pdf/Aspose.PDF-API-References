## MarkdownSaveOptions.HeadingLevels özelliği

FontSize tanıma başlıkları stratejisinde kullanılacak beklenen başlık seviyelerini tanımlar. Bu özellik değeri ayarlandığında, başlık tanıma !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic stratejisi, belge yer imleri içerse bile !:PdfToMarkdown.HeadingRecognitionStrategy.Auto stratejileri ayarlandığında seçilecektir.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
