## MarkdownSaveOptions.ResourcesDirectoryName özelliği

Belge kaynaklarını, örneğin resimleri kaydetmek için dizin adını alır ve ayarlar. Değer belirtilmezse, resimler markdown dosyasının bulunduğu dizine yazılacaktır. Bu bir yol değil, sadece bir isimdir! Bu dizin, kaydedilen markdown dosyasının bulunduğu dizinde otomatik olarak oluşturulacaktır.

```csharp
public string ResourcesDirectoryName { get; set; }
```
