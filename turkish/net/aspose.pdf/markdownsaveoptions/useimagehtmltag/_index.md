## MarkdownSaveOptions.UseImageHtmlTag özelliği

Metnin soluna ve sağına resim eklemek için bir img etiketinin kullanımına izin verme durumunu alır ve ayarlar. Bu durumda, markdown görüntüleyicisinde metin resmin etrafında sarılacaktır.

```csharp
public bool UseImageHtmlTag { get; set; }
```
