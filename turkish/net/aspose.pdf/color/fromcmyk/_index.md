## Color.FromCmyk metodu

CMYK renk bileşenlerinden geçerli pdf Renk nesnesi alır.

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| c | Double | Cyan renk bileşeni (değer 0 - 1). |
| m | Double | Magenta renk bileşeni (değer 0 - 1). |
| y | Double | Sarı renk bileşeni (değer 0 - 1). |
| k | Double | Anahtar renk bileşeni (değer 0 - 1). |

### Dönüş Değeri

Her bileşen değeri [0..1] aralığında olan Renk nesnesi.
