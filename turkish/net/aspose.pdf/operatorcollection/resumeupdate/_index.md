## ResumeUpdate(bool) {#resumeupdate_1}

Belge güncellemesini yeniden başlatır. Bekleyen değişiklikler varsa içerik akışını günceller. İptal etme parametresi true ise tüm operatörleri "değiştirildi" olarak işaretler.

```csharp
public void ResumeUpdate(bool updateAll)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| updateAll | Boolean | Eğer true ise, koleksiyondaki tüm operatörler güncellenmiş olarak işaretlenir. |
