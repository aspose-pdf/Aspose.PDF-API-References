## ExcelSaveOptions.InsertBlankColumnAtFirst özelliği

Eğer çalışma sayfasının ilk sütunu olarak boş sütun eklemek istiyorsanız true olarak ayarlayın. Varsayılan değer false'tur; bu da boş sütunun eklenmeyeceği anlamına gelir.

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
