## ExcelSaveOptions.UniformWorksheets özelliği

Belge boyunca uniform sütun bölümü kullanmak için true olarak ayarlayın. Varsayılan değer false'tur; bu, sütun bölümünün her sayfa için bağımsız olacağı anlamına gelir.

```csharp
public bool UniformWorksheets { get; set; }
```
