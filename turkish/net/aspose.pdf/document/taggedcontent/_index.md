## Document.TaggedContent property

TaggedPdf içeriğine erişim sağlar.

```csharp
public ITaggedContent TaggedContent { get; }
```

### Örnekler

Örnek, başlık, paragraflar ve resimler içeren yeni belge oluşturmak için etiketli içeriğin nasıl kullanılacağını gösterir.

```csharp
// Yeni belge oluştur
Document document = new Document();

// etiketli içeriği al
ITaggedContent taggedContent = document.TaggedContent;

// Belge için dili ayarla
taggedContent.SetLanguage("en-US");

// PDF belgesi için başlık ayarla
taggedContent.SetTitle("Example document");

// Bölüm oluşturma ve ekleme
SectElement sect = taggedContent.CreateSectElement();
taggedContent.RootElement.AppendChild(sect);

// Başlık Oluştur
HeaderElement h1 = taggedContent.CreateHeaderElement(1);
h1.SetText("The Header");
sect.AppendChild(h1);

// Paragraf oluştur
ParagraphElement p = taggedContent.CreateParagraphElement();
p.SetTag("Paragraph");
p.SetText("The text of paragraph.");
sect.AppendChild(p);

// İllüstrasyon oluştur
IllustrationElement figure1 = taggedContent.CreateFigureElement();
sect.AppendChild(figure1);
figure1.AlternativeText = "Figure 1";
figure1.Title = "Image 1";
figure1.SetTag("Fig");
figure1.SetImage("path/of/image.jpg");

// Belgeyi kaydet
document.Save("example.pdf");
```
