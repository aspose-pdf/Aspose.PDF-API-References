## Document.IgnoreCorruptedObjects özelliği

Kaynak dosyalardaki hataları yok sayma bayrağını alır veya ayarlar. Kaynak belgeden hedef belgeye sayfalar kopyalandığında, bu bayrak yanlışsa kaynak dosyalardaki bazı nesneler bozulduğunda kopyalama işlemi bir istisna ile durdurulur. örnek: dest.Pages.Add(src.Pages); Bu bayrak doğru olarak ayarlandığında, bozulmuş nesneler boş değerlerle değiştirilir. Varsayılan: doğru.

```csharp
public bool IgnoreCorruptedObjects { get; set; }
```
