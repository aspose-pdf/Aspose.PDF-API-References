## HtmlSaveOptions.CssSavingInfo.CssNumber alanı

Dönüştürücü tarafından ayarlanır. Dönüşüm sırasında birkaç CSS dosyası oluşturulur. Bu özellik, dönüşüm sırasında kaydedilen CSS dosyasının sırasını gösterir. CSS içeriğini nasıl işleneceğine veya nereye kaydedileceğine karar vermek için özel kodun mantığında kullanılabilir.

```csharp
public int CssNumber;
```
