## CollectionItem.IsEmpty özelliği

Koleksiyon öğesinin boş olup olmadığını belirten bir değer alır.

```csharp
public bool IsEmpty { get; }
```

### Dönüş Değeri

Koleksiyon öğesi boşsa true; aksi takdirde false.

## Açıklamalar

Bu özellik, koleksiyon öğesi herhangi bir değer içermiyorsa, string değerler, double değerler, tam sayı değerleri ve tarih değerleri dahil olmak üzere, true döner. Eğer bu değer türlerinden herhangi biri koleksiyon öğesinde mevcutsa, bu özellik false döner.
