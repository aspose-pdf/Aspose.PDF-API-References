## PdfViewer.PrintAsImage özelliği

PdfViewer'ın görüntü olarak yazdırma modunu ayarlar veya alır.

```csharp
public bool PrintAsImage { get; set; }
```

## Açıklamalar

Eğer doğruysa her zaman görüntü olarak yazdırır (yazdırılan görüntüyü oluşturur). Eğer yanlışsa, tüm özellikler destekleniyorsa doğrudan cihaza yazdırır. Belge desteklenmeyen özellikler içeriyorsa sistem otomatik olarak görüntü olarak yazdırmayı seçebilir. Varsayılan değer yanlıştır.
