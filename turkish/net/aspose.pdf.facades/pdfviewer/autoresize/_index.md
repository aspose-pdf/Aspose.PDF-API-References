## PdfViewer.AutoResize özelliği

Dosyanın optimize edilmiş boyutla yazdırılıp yazdırılmayacağını belirten bir bool değeri alır veya ayarlar. Eğer false ise sayfa ölçeklendirmeden yazdırılır. Eğer true ise yazdırılabilir alana sığacak şekilde ölçeklendirilerek yazdırılır.

```csharp
public bool AutoResize { get; set; }
```
