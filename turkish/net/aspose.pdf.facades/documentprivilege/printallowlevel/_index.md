## DocumentPrivilege.PrintAllowLevel özelliği

Belgelerin baskı seviyesini alır ve ayarlar. Adobe Professional'ın Baskıya İzin Verilen ayarları gibi. 0: Yok. 1: Düşük Çözünürlük (150 dpi). 2: Yüksek Çözünürlük.

```csharp
public int PrintAllowLevel { get; set; }
```

## Açıklamalar

Eğer özelliğin değeri -1 ise, o zaman seviye tanımsızdır.
