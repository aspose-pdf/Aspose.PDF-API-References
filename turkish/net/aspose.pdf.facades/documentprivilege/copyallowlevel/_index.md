## DocumentPrivilege.CopyAllowLevel özelliği

Belgenin ayrıcalıklarının kopyalama seviyesini alır ve ayarlar. Adobe Professional'ın izin ayarları gibi. 0: Yok. 1: Görme engelliler için ekran okuyucu cihazlar için metin erişimini etkinleştir. 2: Metin, resimler ve diğer içeriklerin kopyalanmasını etkinleştir.

```csharp
public int CopyAllowLevel { get; set; }
```

## Açıklamalar

Eğer özelliğin değeri -1 ise, o zaman seviye tanımsızdır.
