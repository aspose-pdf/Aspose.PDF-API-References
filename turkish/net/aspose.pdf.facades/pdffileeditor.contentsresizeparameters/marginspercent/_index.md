## PdfFileEditor.ContentsResizeParameters.MarginsPercent metodu

Yeniden boyutlandırma parametreleri oluşturur. Kenar boşlukları, başlangıç sayfa boyutunun yüzdesi olarak belirtilir.

```csharp
public static ContentsResizeParameters MarginsPercent(double left, double right, double top, 
    double bottom)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| left | Double | Sol kenar boşluğu (sayfa genişliğinin yüzdesi olarak). |
| right | Double | Sağ kenar boşluğu (sayfa yüksekliğinin yüzdesi olarak). |
| top | Double | Üst kenar boşluğu (sayfa yüksekliğinin yüzdesi olarak). |
| bottom | Double | Alt kenar boşluğu (sayfa yüksekliğinin yüzdesi olarak). |

### Dönüş Değeri

Yeni yeniden boyutlandırma parametrelerini döndürür.
