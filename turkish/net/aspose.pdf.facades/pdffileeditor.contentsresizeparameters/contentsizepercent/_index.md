## PdfFileEditor.ContentsResizeParameters.ContentSizePercent yöntemi

Belirtilen içerik boyutunu başlangıç sayfa boyutunun yüzdesi olarak ayarlayan yeniden boyutlandırma parametreleri oluşturur. Kenar boşlukları otomatik olarak hesaplanır.

```csharp
public static ContentsResizeParameters ContentSizePercent(double width, double height)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| width | Double | Yeni içerik genişliği yüzdesi. |
| height | Double | Yeni içerik yüksekliği yüzdesi. |

### Dönüş Değeri

Yeni yeniden boyutlandırma parametreleri.
