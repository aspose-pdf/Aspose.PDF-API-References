## PdfFileMend.IsWordWrap özelliği

AddText yöntemlerinde kelime sarma durumunu belirten bir bool değeri ayarlar. Değer true ise, FormattedText içindeki metin kelime sarma yapar. Varsayılan olarak, değer false'tur.

public bool IsWordWrap { set; }
