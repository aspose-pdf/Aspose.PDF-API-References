## AutoFiller.BasicFileName özelliği

Birçok küçük dosya oluşturulacaksa temel dosya adını alır veya ayarlar. Oluşturulan dosya "BasicFileName0","BasicFileName1",... gibi olacaktır. Diğer bir özellik olan [`GeneratingPath`](../generatingpath/) ile çalışır.

```csharp
public string BasicFileName { get; set; }
```
