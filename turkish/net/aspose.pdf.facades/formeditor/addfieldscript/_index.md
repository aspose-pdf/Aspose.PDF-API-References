## FormEditor.AddFieldScript metodu

Bir PushButton alanı için JavaScript ekleyin. Eski bir olay varsa, yeni olay onun sonrasına eklenir.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fieldName | String | Tam nitelikli alan adı. |
| script | String | Bir push button alanına eklenecek/yerleştirilecek Java script. |

### Dönüş Değeri

Script başarıyla eklendiğinde true döner.
