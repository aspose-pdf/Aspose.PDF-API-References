## FormEditor.SetFieldCombNumber method

Normal bir tek satırlık metin alanı için tarak sayısını ayarlar (alan otomatik olarak birçok eşit aralıklı konuma veya taraklara bölünür, tarakNumarası parametresinin değeri kadar).

```csharp
public bool SetFieldCombNumber(string fieldName, int combNumber)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| fieldName | String | Nitelikli alan adı. |
| combNumber | Int32 | Alanı bölmek için tarak sayısı. |

### Geri dönüş değeri

Başarılı olursa, true, aksi takdirde false döndürün.

### Örnekler

```csharp
FormEditor formEditor = new FormEditor("PdfWithAcroForm.pdf", "FormEditor_SetFieldComb.pdf"));
formEditor.SetFieldCombNumber("textCombField", 5);
```
