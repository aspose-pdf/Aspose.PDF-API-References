## FormEditor.SetFieldScript metodu

Bir PushButton alanı için JavaScript ayarlayın. Eski JavaScript varsa, yeni ile değiştirilecektir.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fieldName | String | Tam nitelikli alan adı. |
| script | String | Bir push button alanına eklenmesi/yerleştirilmesi gereken Java script. |

### Dönüş Değeri

Alan scripti başarıyla ayarlandıysa true döner.
