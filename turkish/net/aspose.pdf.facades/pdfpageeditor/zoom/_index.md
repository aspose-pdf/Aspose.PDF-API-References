## PdfPageEditor.Zoom property

Yakınlaştırma katsayısını alır veya ayarlar. 1.0 değeri %100'e karşılık gelir. Varsayılan değer 1.0'dır. Aşağıdaki örnek, belge sayfalarının yakınlaştırmasının nasıl değiştirileceğini gösterir.

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
editor.Zoom = 0.5f;
```

```csharp
public float Zoom { get; set; }
```
