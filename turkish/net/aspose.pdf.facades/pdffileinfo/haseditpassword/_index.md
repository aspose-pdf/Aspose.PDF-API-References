## PdfFileInfo.HasEditPassword özelliği

İzinleri veya belge güvenlik özelliğini değiştirmek için şifre gerekiyorsa true döner. Bu özelliğin yalnızca geçerli bir şifre [`PdfFileInfo`](../) yapıcısında sağlandığında okunabileceğine dikkat edin. PasswordType Erişilemez olduğunda (geçersiz bir şifre sağlandığı anlamına gelir) bu özelliği okumak [`InvalidPasswordException`](../../../aspose.pdf/invalidpasswordexception/) ile başarısız olacaktır.

```csharp
public bool HasEditPassword { get; }
```
