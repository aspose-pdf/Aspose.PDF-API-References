## CreateWebLink(Rectangle, string, int, Color, Enum[]) {#createweblink_2}

PDF belgesinde bir web bağlantısı oluşturur.

```csharp
public void CreateWebLink(Rectangle rect, string url, int originalPage, Color clr, 
    Enum[] actionName)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| rect | Rectangle | Etkin tıklama için dikdörtgen. |
| url | String | Web bağlantısı hedefi. |
| originalPage | Int32 | Web bağlantısına bağlı dikdörtgenin oluşturulacağı orijinal sayfa sayısı. |
| clr | Color | Etkin tıklama için dikdörtgenin rengi. |
| actionName | Enum[] | Acrobat görüntüleyicide menü öğelerinin yürütülmesine karşılık gelen eylemler dizisi (PredefinedAction enum üyeleri). |

### Örnekler

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateWebLink(new System.Drawing.Rectangle(0, 0, 100, 100),
    "http://www.aspose.com", 1, System.Drawing.Color.Red,
    new Enum[] { PredefinedAction.FirstPage, PredefinedAction.PrintDialog });
editor.Save("example_out.pdf");
```
