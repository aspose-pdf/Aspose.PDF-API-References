## CreateLocalLink(Rectangle, int, int, Color, Enum[]) {#createlocallink_2}

PDF belgesinde yerel bir bağlantı oluşturur.

```csharp
public void CreateLocalLink(Rectangle rect, int desPage, int originalPage, Color clr, 
    Enum[] actionName)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| rect | Rectangle | Etkin tıklama için dikdörtgen. |
| desPage | Int32 | Hedef sayfa. |
| originalPage | Int32 | Yerel bağlantıyla bağlı dikdörtgenin oluşturulacağı orijinal sayfa sayısı. |
| clr | Color | Etkin tıklama için dikdörtgenin rengi. |
| actionName | Enum[] | Acrobat görüntüleyicide menü öğelerinin yürütülmesine karşılık gelen eylemler dizisi (PredefinedAction enum üyeleri). |

### Örnekler

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateLocalLink(new System.Drawing.Rectangle(0, 0, 100, 100),
    2, 1, System.Drawing.Color.Red,
    new Enum[] { PredefinedAction.FirstPage, PredefinedAction.PrintDialog });
editor.Save("example_out.pdf");
```
