## TruncationStrategy.StrategyType özelliği

İlgili iş parçacığı için kullanılacak kesme stratejisini alır veya ayarlar. Varsayılan auto'dur. last_messages olarak ayarlandığında, iş parçacığı en son n mesajla kesilecektir. auto olarak ayarlandığında, iş parçacığının ortasındaki mesajlar modelin bağlam uzunluğuna uyması için atılacaktır, max_prompt_tokens.

```csharp
public string StrategyType { get; set; }
```
