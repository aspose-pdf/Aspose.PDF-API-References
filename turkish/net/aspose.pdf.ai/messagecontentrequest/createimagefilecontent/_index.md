## MessageContentRequest.CreateImageFileContent metodu

Bir mesaj için bir resim dosyası içeriği oluşturur.

```csharp
public static MessageContentRequest CreateImageFileContent(string fileId, string detail = "auto")
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| fileId | String | Resim dosyasının kimliği. |
| detail | String | Resim dosyasının detayı. Kullanıcı tarafından belirtilmişse resmin detay seviyesini belirtir. "low" daha az token kullanır, "high" ile yüksek çözünürlük seçeneğine geçebilirsiniz. Varsayılan "auto"dur. |
