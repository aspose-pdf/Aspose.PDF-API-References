## MessageContentRequest.CreateImageUrlContent yöntemi

Bir mesaj için bir resim URL içeriği oluşturur.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| imageUrl | String | Resmin URL'si. |
| detail | String | Resim URL'sinin detayı. Kullanıcı tarafından belirtilmişse resmin detay seviyesini belirtir. "low" daha az token kullanır, "high" ile yüksek çözünürlük seçebilirsiniz. Varsayılan "auto"dur. |

### Dönüş Değeri

ImageUrl içeriği ile yeni bir MessageContentRequest örneği.
