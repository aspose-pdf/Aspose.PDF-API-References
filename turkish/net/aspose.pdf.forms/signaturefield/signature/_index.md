## SignatureField.Signature özelliği

İmza nesnesini alır. Bu nesne, açık anahtar kriptografik standartları ile ilgili imza verilerini içerir. Sınıflar [`PKCS1`](../../pkcs1/), [`PKCS7`](../../pkcs7/) ve [`PKCS7Detached`](../../pkcs7detached/) desteklenen tüm imza nesne türlerini temsil eder.

```csharp
public Signature Signature { get; }
```
