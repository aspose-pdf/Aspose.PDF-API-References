## Signature.DefaultSignatureLength özelliği

İmza verisinin varsayılan uzunluğunu bayt cinsinden alır veya ayarlar.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Açıklamalar

Bu, imzanın bayt cinsinden uzunluğunun bir tahminidir. [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) parametresi ayarlandığında [`CustomSignHash`](../customsignhash/) aracılığıyla imzalamak için kullanılır. Varsayılan değer 3000'dir.
