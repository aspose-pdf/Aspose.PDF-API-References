## Form.FlattenSettings.UpdateAppearances özelliği

Ayarlandığında, tüm alan görünümleri düzleştirmeden önce yeniden oluşturulacaktır. Bu seçenek, alanın yanlış bir şekilde düzleştirilmesi durumunda yardımcı olabilir. Bu seçenek performansı düşürebilir. Varsayılan olarak false olarak ayarlanmıştır.

```csharp
public bool UpdateAppearances { get; set; }
```
