## XFA dizinleyici

Veritabanı düğüm değerini *yol* göre alır veya ayarlar.

```csharp
public string this[string path] { get; set; }
```

| Parametre | Açıklama |
| --- | --- |
| yol | Veritabanı düğüm yolu, örneğin form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Her düğümün yalnızca bir kez bulunduğu durumlarda bile indeksleri dahil etmeyi unutmayın, yani node1[0].node2[0]... yerine node1.node2... yazın. |

### Dönüş Değeri

Veritabanı düğüm değeri.
