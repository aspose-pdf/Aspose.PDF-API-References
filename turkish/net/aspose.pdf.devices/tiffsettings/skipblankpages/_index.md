## TiffSettings.SkipBlankPages özelliği

Boş sayfaların atılıp atılmayacağını belirten bir değeri alır veya ayarlar.

```csharp
public bool SkipBlankPages { get; set; }
```

### Özellik Değeri

Boş sayfaların atılması gerekiyorsa `true`; aksi takdirde `false`.

## Açıklamalar

Varsayılan değer false'dur.
