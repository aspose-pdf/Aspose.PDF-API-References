## SvgExtractionOptions.GroupStrength özelliği

Alt yolların görüntülere gruplama gücünü ayarlamak için bir seçenek alır ve ayarlar. Alt yolların gruplama derecesini yapılandırmanıza olanak tanır. Değer aralığı 0 ile 1 arasındadır. 0 değeri, [`ExtractEverySubPathToSvg`](../extracteverysubpathtosvg/) seçeneğinin etkin olduğu anlamına gelir. 1 değeri, sayfadaki tüm vektör yolları için tek bir görüntü oluşturur. Seçenek, [`AutoGrouping`](../autogrouping/) false olduğunda etkilidir. Varsayılan değer `0.8`'dir.

```csharp
public double GroupStrength { get; set; }
```
