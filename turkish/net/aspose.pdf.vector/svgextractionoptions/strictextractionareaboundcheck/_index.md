## SvgExtractionOptions.StrictExtractionAreaBoundCheck özelliği

ExtractionAreaBound'da belirtilen dikdörtgenin içinde alt yolların olup olmadığını kesin olarak kontrol etmek için bir seçenek alır ve ayarlar. Eğer false olarak ayarlanırsa, [`ExtractionAreaBound`](../extractionareabound/) içinde tamamen yer almayan alt yollar çıkarılacaktır. Varsayılan değer `True`'dur.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
