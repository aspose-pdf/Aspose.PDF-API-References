## SvgExtractionOptions.MinStrokeWidth özelliği

Sonuçta elde edilen SVG'de kullanılacak minimum stroke genişliğini alır veya ayarlar. PDF daha ince bir stroke genişliği kullanıyorsa, bu genişlikle değiştirilir. Varsayılan değer 0.5'tir.

```csharp
public double MinStrokeWidth { get; set; }
```

## Açıklamalar

Değer, dönüştürülen PDF sayfasının dönüştürülmüş kullanıcı alanı birimlerinde ifade edilir. Varsayılan olarak 1 kullanıcı alanı birimi 1/72 inç (0.35 mm) olarak kabul edilir, ancak bu PDF belgesi tarafından geçersiz kılınabilir. Dönüşümler, üretilen SVG'deki gerçek minimum genişliği etkileyebilir.
