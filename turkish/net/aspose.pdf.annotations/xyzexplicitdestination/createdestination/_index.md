## XYZExplicitDestination.CreateDestination metodu

Gerekirse sayfa döndürmesini dikkate alarak sayfanın belirtilen konumuna varış noktası oluşturur.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| Parametre | Tür | Açıklama |
| --- | --- | --- |
| page | Sayfa | Varış sayfası. |
| left | Çift | Sayfadaki sol konum. |
| top | Çift | Sayfadaki üst konum. |
| zoom | Çift | Yakınlaştırma faktörü (varsayılan için 0). |
| considerRotation | Boolean | Eğer doğruysa, konum sayfa döndürmesine göre yeniden hesaplanacaktır. |

### Dönüş Değeri

Varış noktası nesnesi.
