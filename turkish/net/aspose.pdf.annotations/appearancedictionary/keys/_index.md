## AppearanceDictionary.Keys özelliği

Sözlüğün anahtarlarını alır. Eğer görünüm sözlüğü alt sözlükler içeriyorsa, o zaman `Keys` (N&#x7C;R&#x7C;D).state değerlerini içerir; burada N - normal görünüm, R - rollover görünüm, D - down görünüm ve state - durumun adıdır (örneğin, onay kutuları için On, Off).

```csharp
public ICollection<string> Keys { get; }
```
