## HighlightingMode enumerasyonu

Notların vurgulama modunu, fare düğmesi aktif alanında basılı tutulduğunda veya basıldığında kullanılacak görsel etkiyi sıralar.

```csharp
public enum HighlightingMode
```

### Değerler

| İsim | Değer | Açıklama |
| --- | --- | --- |
| None | `0` | Vurgulama yok. |
| Invert | `1` | Not alanının içeriğini ters çevir. |
| Outline | `2` | Notun kenarlığını ters çevir. |
| Push | `3` | Notun aşağı görünümünü göster, eğer varsa. Eğer aşağı görünüm tanımlanmamışsa, not alanının içeriğini sayfanın yüzeyinin altına itilmiş gibi görünmesi için kaydır. |
| Toggle | `4` | Push ile aynı (tercih edilen). |
