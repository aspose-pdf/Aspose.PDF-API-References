## ImageCompressionOptions.ResizeImages özelliği

Bu bayrak true olarak ayarlandığında ve CompressImages true olduğunda, görüntüler belirtilen MaxResolution parametresinden daha yüksek bir çözünürlüğe sahipse yeniden boyutlandırılacaktır.

```csharp
public bool ResizeImages { get; set; }
```
