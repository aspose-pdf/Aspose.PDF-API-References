## ImageCompressionOptions.Version özelliği

Sıkıştırma algoritmasının versiyonu. Olası değerler: 1. standart sıkıştırma, 2. hızlı (standarttan daha hızlı olan geliştirilmiş sıkıştırma ancak tüm resimlere uygulanamayabilir), 3. karışık (hızlı algoritma ile sıkıştırılamayan resimlere standart sıkıştırma uygulanır, bu en iyi sıkıştırmayı verebilir ancak "hızlı" algoritmadan daha yavaştır. "Hızlı" versiyonu, resimleri yeniden boyutlandırmak için uygulanamaz (standart yöntem kullanılacaktır). Varsayılan "Standart"tır.

```csharp
public ImageCompressionVersion Version { get; set; }
```
