## OptimizationOptions.RemoveUnusedStreams özelliği

Eğer bu bayrak true olarak ayarlandıysa, her kaynak kullanımına göre kontrol edilir. Eğer kaynak hiç kullanılmadıysa, o zaman kaynak kaldırılır. Bu, örneğin sayfalar belgeden çıkarıldığında belge boyutunu azaltabilir.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
