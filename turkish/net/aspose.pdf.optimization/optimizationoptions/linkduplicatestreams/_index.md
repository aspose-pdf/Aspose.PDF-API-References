## OptimizationOptions.LinkDuplicateStreams özelliği

Bu bayrak true olarak ayarlandığında, Kaynak akışları analiz edilecektir. Eğer kopya akışlar bulunursa (yani akış içerikleri eşitse), bu akışlar bir nesne olarak saklanacaktır. Bu, bazı durumlarda belge boyutunu azaltmaya olanak tanır (örneğin, aynı belgenin birden fazla kez birleştirildiği durumlarda).

```csharp
public bool LinkDuplicateStreams { get; set; }
```
