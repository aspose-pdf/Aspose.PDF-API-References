## OptimizationOptions.RemoveUnusedObjects özelliği

Bu bayrak true olarak ayarlandığında, tüm belge nesneleri kontrol edilecek ve kullanılmayan nesneler (yani herhangi bir referansı olmayan nesneler) belgeden kaldırılacaktır.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
