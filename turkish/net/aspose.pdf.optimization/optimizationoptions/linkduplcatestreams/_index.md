## OptimizationOptions.LinkDuplcateStreams özelliği

Bu bayrak true olarak ayarlandığında, Kaynak akışları analiz edilecektir. Eğer kopya akışlar bulunursa (yani akış içerikleri eşitse), bu akışlar bir nesne olarak saklanacaktır. Bu, bazı durumlarda belge boyutunu azaltmaya olanak tanır (örneğin, aynı belge birden fazla kez birleştirildiğinde).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
