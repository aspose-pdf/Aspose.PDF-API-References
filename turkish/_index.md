## .NET için Aspose.PDF

{{% alert color="primary" %}} 

![Aspose.PDF for .NET Ürün Logosu](../aspose_pdf-for-net.png)

Aspose.PDF, geliştiricilerin ister basit ister karmaşık olsun, anında programlı olarak PDF belgeleri oluşturmasına olanak sağlamak için oluşturulmuş bir .NET bileşenidir. Aspose.PDF for .NET, geliştiricilerin PDF belgelerine tablolar, grafikler, resimler, köprüler, özel yazı tipleri ve daha fazlasını eklemesine olanak tanır. Ayrıca, PDF belgelerini sıkıştırmak da mümkündür. Aspose.PDF for .NET, güvenli PDF belgeleri geliştirmek için mükemmel güvenlik özellikleri sunar. Aspose.PDF for .NET'in en belirgin özelliği, hem API üzerinden hem de XML şablonlarından PDF belgelerinin oluşturulmasını desteklemesidir.

{{% /alert %}}

Bunlar, bazı yararlı kaynaklara bağlantılar:
- [Aspose.PDF for .NET API Referansı](/pdf/tr/net/)

## C++ için Aspose.PDF

{{% alert color="primary" %}} 

![Aspose.PDF for C++ Ürün Logosu](../aspose_pdf-for-cpp.png)

Aspose.PDF for C++, geliştiricilerin Adobe Acrobat kullanmadan PDF belgeleri oluşturmasını, okumasını ve değiştirmesini sağlayan yerel bir C++ kitaplığıdır. Aspose.PDF for C++, geliştiricilerin formlar oluşturmasına, metin eklemesine/düzenlemesine, PDF sayfalarını değiştirmesine, açıklama eklemesine, özel yazı tiplerini işlemesine ve çok daha fazlasına olanak tanır. Burada Aspose.PDF for C++'ın dokümantasyon bölümünde, API'nin tüm özelliklerini ve bu özelliklerin C++ uygulamanızda uygulanmasını göstermek için temel örnekleri keşfedebilirsiniz.
