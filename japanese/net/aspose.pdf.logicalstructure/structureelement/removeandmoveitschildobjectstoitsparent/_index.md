## StructureElement.RemoveAndMoveItsChildObjectsToItsParent メソッド

構造から要素を削除し、それに対する親オブジェクトからの参照、子オブジェクトからの参照、および文書からの対応するオブジェクトを削除します。削除されたオブジェクトの子オブジェクトを、削除されたオブジェクトのインデックスから始めて、その以前の親の子オブジェクトコレクションに挿入します。

```csharp
public void RemoveAndMoveItsChildObjectsToItsParent(
    bool checkIfChildObjectsCanBeMovedToParent = true)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| checkIfChildObjectsCanBeMovedToParent | Boolean | 削除されたオブジェクトの子オブジェクトをその親の子オブジェクトコレクションに挿入できるかどうかを確認します。 |
