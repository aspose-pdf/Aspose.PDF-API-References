## StructureTypeStandard.RB フィールド

（ルビー基本テキスト）ルビー注釈が適用されるフルサイズテキスト。RB にはテキスト、他のインライン要素、またはその両方の混合が含まれる場合があります。RubyAlign属性を持つことがあります。

```csharp
public static readonly StructureTypeStandard RB;
```
