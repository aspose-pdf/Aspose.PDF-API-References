## StructureTypeStandard.Ruby フィールド

(Ruby; PDF 1.5) 参照する基本テキストの隣に配置された小さいテキストサイズで書かれたサイドノート（注釈）。Ruby 要素には RB、RT、および RP 要素も含まれる場合があります。

(Ruby) 全体のルビーアセンブリを囲むラッパー。1 つの RB 要素の後に、RT 要素または RP、RT、および RP の 3 要素グループのいずれかが続く必要があります。Ruby 要素とそのコンテンツ要素は、複数行にわたって分割されてはなりません。

```csharp
public static readonly StructureTypeStandard Ruby;
```
