## StructureTypeStandard.Quote フィールド

(引用) 周囲のテキストの著者以外の誰かに帰属するインラインのテキスト部分です。

引用されたテキストは、単一の段落内にインラインで含まれている必要があります。これは、1つ以上の完全な段落（または完全な段落のように提示された他の要素）で構成されるブロックレベル要素 BlockQuote とは異なります。

```csharp
public static readonly StructureTypeStandard Quote;
```
