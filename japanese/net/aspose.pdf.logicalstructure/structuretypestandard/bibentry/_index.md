## StructureTypeStandard.BibEntry フィールド

(文献エントリ) 一部の引用されたコンテンツの外部ソースを特定する参照。子としてラベル (構造タイプ Lbl) を含むことがあります。

文献エントリには、引用されたコンテンツの著者、作品、出版社などを特定する構成要素が含まれる可能性がありますが、この詳細レベルで定義された標準構造タイプはありません。

```csharp
public static readonly StructureTypeStandard BibEntry;
```
