## AnnotationElement.AlternateDescriptions プロパティ

アノテーションの代替説明を取得または設定します。アノテーションに表示されるテキスト、またはこのタイプのアノテーションがテキストを表示しない場合は、アノテーションの内容の人間が読める形式の代替説明。

```csharp
public string AlternateDescriptions { get; set; }
```
