## MarkupParagraph.SecondaryPoints プロパティ

二次ポリゴンのポイントは段落の継続を説明します。段落が次の列またはページで継続される場合、null にはなりません。開始点は段落の左下隅です。次のポイントは反時計回りの順序です。

```csharp
public List<Point[]> SecondaryPoints { get; }
```
