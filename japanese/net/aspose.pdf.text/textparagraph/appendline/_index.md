## AppendLine(string, float) {#appendline_6}

テキスト行を追加します。

```csharp
public void AppendLine(string line, float lineSpacing)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| line | String | 新しい行のテキスト。 |
| lineSpacing | Single | 追加の間隔（0.0 がデフォルトで、デフォルトのテキスト行の高さに対応します）。間隔の値は特定の行のデフォルトの行間に追加されるため、12.0 を指定すると、12pt フォントで描画されたテキスト行の後に空の行が得られます。 |
