## TextReplaceOptions.AdjustmentNewLineSpacing プロパティ

置換調整が強制的に新しいテキスト行を作成する場合に使用される行間の値を取得または設定します。期待される値は、置換されたテキストのフォントサイズの倍率です。デフォルトは 1.2 です。

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
