## TextFragmentAbsorber.HasErrors プロパティ

値は、テキスト抽出中にエラーが見つかったかどうかを示します。エラーを検索するのは、TextSearchOptions.LogTextExtractionErrors = true の場合のみ実行され、パフォーマンスが低下する可能性があります。

```csharp
public bool HasErrors { get; }
```
