## TextEditOptions.AllowLanguageTransformation プロパティ

テキストの追加または編集中に言語変換の使用を許可する値を取得または設定します。 true - 必要に応じて言語変換が適用されます（デフォルト値）。 false - 言語変換は適用されません。

```csharp
public bool AllowLanguageTransformation { get; set; }
```
