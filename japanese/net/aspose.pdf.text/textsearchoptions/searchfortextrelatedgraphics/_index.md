## TextSearchOptions.SearchForTextRelatedGraphics プロパティ

テキスト検索中にテキスト関連グラフィック（下線、背景など）を検索することを許可する値を取得または設定します。 true - テキスト関連グラフィックの検索が実行されます（デフォルト値）。 false - ソースドキュメントに存在する可能性のあるグラフィック要素は無視されます。パフォーマンスの問題がある場合や、下線、背景、またはクリッピングを処理する必要がない場合は、これを設定します。

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
