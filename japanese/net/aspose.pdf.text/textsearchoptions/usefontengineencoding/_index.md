## TextSearchOptions.UseFontEngineEncoding プロパティ

テキストがフォントエンジンエンコーディングを使用して検索されることを示す設定を取得または設定します。true - フォントエンジンエンコーディングが使用されることを意味します（ドキュメント内の不完全なエンコーディングのためにテキスト検索が失敗した場合はこれを試してください）。false - ドキュメントフォントエンコーディングが使用されることを意味します（デフォルト値）。

```csharp
public bool UseFontEngineEncoding { get; set; }
```
