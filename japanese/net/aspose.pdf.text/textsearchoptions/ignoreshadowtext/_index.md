## TextSearchOptions.IgnoreShadowText プロパティ

通常のテキストの影を表すテキストフラグメントが検索中に無視されることを示す設定を取得または設定します。true - 影のテキストが見つからないことを意味します（テキスト検索が近い位置で重複したフラグメントを返す場合はこれを試してください）。false - 影のテキストが通常のテキストとともに見つかることを意味します（デフォルト値）。

```csharp
public bool IgnoreShadowText { get; set; }
```
