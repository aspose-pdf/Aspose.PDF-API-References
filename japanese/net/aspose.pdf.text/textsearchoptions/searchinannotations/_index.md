## TextSearchOptions.SearchInAnnotations プロパティ

注釈内のテキストを検索することを許可する値を取得または設定します。true - テキストは注釈内で検索されます。false - 注釈内のテキストは TextFragmentAbsorber によって解析されません。

```csharp
public bool SearchInAnnotations { get; set; }
```
