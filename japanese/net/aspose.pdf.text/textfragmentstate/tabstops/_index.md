## TextFragmentState.TabStops プロパティ

テキストのタブストップを取得します。

```csharp
public TabStops TabStops { get; }
```

## 備考

Tabstops プロパティは、新しいドキュメント生成シナリオでのみ機能することに注意してください。タブストップは、[`TextFragment`](../../textfragment/) の初期化中に追加される場合があります。タブストップは、テキストの前に構築する必要があります。
