## TextFragmentState.IsFitRectangle メソッド

入力文字列が定義された長方形内に配置できるかどうかを確認します。

```csharp
public bool IsFitRectangle(string str, Rectangle rect)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| str | String | チェックする文字列。 |
| rect | Rectangle | チェックする長方形。 |

### 戻り値

文字列が長方形に収まる場合は true; そうでない場合は false。
