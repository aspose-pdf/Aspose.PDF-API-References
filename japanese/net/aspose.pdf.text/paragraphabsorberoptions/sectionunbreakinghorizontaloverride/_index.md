## ParagraphAbsorberOptions.SectionUnbreakingHorizontalOverride プロパティ

「ゼロ充填レベル」のゾーンの最大長を取得または設定します。これは、実際の水平セクションブレークとして扱われないものです（ページの幅/高さの割合として）。デフォルト値は 0.005 です。

```csharp
public double SectionUnbreakingHorizontalOverride { get; set; }
```
