## Font.Save メソッド

フォントをストリームに保存します。フォントは、元のドキュメントの変換されたコピーでのみ使用されることを意図した中間TTF形式で保存されることに注意してください。フォントファイルは、元のドキュメントのコンテキスト外で使用されることを意図していません。

```csharp
public void Save(Stream stream)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| stream | Stream | フォントを保存するストリーム。 |
