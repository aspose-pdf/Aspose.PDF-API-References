## SystemFontsSubstitution.DefaultFont プロパティ

デフォルトの代替フォントを取得または設定します。このフォントは、他の有効な代替フォントが見つからなかった場合に使用されますが、初期フォントがターゲット代替カテゴリ ([`FontCategories`](../fontcategories/)) に属している場合に使用されます。

```csharp
public Font DefaultFont { get; set; }
```
