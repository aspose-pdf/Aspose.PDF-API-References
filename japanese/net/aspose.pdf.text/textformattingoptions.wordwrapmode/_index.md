## TextFormattingOptions.WordWrapMode 列挙型

```csharp
public enum WordWrapMode
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| NoWrap | `0` | 折り返しは行われません |
| DiscretionaryHyphenation | `1` | 裁量によるハイフネーションが行われます。単語を途中で分割することができます。 |
| ByWords | `2` | 単語の折り返しは完全な単語のみを折り返します。完全な単語が折り返せない場合は、裁量によるハイフネーションを使用しようとします |
| Undefined | `3` | WordWrapMode が設定されていません。上位構造（テーブルセル、段落など）の折り返し戦略が使用されます。 |
