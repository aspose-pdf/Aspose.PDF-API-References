## TextState.LineSpacing プロパティ

テキストの行間を取得または設定します。

```csharp
public virtual float LineSpacing { get; set; }
```

## 備考

値はドキュメント内のテキスト特性として保持されないことに注意してください。LineSpacing プロパティのゲッターは、以前に LineSpacing セッターで明示的に設定されたオブジェクトに対して機能します。このプロパティは、現在の生成/変更プロセスのコンテキストでランタイムによって使用されます。
