## TextState.BackgroundColor プロパティ

テキストの背景色を設定します。

```csharp
public virtual Color BackgroundColor { get; set; }
```

## 備考

値はドキュメント内のテキスト特性として保持されないことに注意してください。BackgroundColor プロパティのゲッターは、以前に BackgroundColor セッターで明示的に設定されたオブジェクトに対して機能します。このプロパティは、現在の生成/修正プロセスのコンテキストでランタイムによって使用されます。
