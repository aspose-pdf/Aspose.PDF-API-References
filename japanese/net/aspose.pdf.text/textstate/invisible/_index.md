## TextState.Invisible プロパティ

テキストの不可視性を取得または設定します。これは基本的に、[`RenderingMode`](../renderingmode/) 状態を反映していますが、特別なケース（クリッピングなど）を除きます。

```csharp
public virtual bool Invisible { get; set; }
```
