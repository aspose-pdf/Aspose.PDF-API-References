## LineBreakStyle 列挙型

ファイルの可能な改行スタイルを表します。

```csharp
public enum LineBreakStyle
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Windows | `0` | Windows の文字列ラップスタイル (CRLF) を選択します。 |
| Unix | `1` | Unix の文字列ラップスタイル (LF) を選択します。OSX にも使用されます。 |
| Auto | `2` | オペレーティングシステムに基づいて文字列ラッピングのタイプを自動的に選択します。 |
