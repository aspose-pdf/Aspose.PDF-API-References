## CompromiseCheckResult.HasCompromisedSignatures プロパティ

ドキュメントに侵害されたデジタル署名があるかどうかを示します。少なくとも1つの署名が侵害されている場合は true を返し、それ以外の場合は false を返します。

```csharp
public bool HasCompromisedSignatures { get; }
```
