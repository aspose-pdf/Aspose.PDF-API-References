## ImageCompressionOptions.CompressImages プロパティ

このフラグが true に設定されている場合、ドキュメント内の画像は圧縮されます。圧縮レベルは ImageQuality プロパティで指定されます。

```csharp
public bool CompressImages { get; set; }
```
