## OptimizationOptions.RemoveUnusedStreams プロパティ

このフラグが true に設定されている場合、すべてのリソースの使用状況がチェックされます。リソースが一度も使用されていない場合、そのリソースは削除されます。これは、ドキュメントからページが抽出された場合などに、ドキュメントのサイズを減少させる可能性があります。

```csharp
public bool RemoveUnusedStreams { get; set; }
```
