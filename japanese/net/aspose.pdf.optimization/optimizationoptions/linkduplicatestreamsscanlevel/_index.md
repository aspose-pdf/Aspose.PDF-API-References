## OptimizationOptions.LinkDuplicateStreamsScanLevel プロパティ

スキャンのレベル。より深いスキャン（高い値）は時間がかかりますが、より小さな結果ファイルを生成する可能性があります。デフォルト値: 10。

```csharp
public int LinkDuplicateStreamsScanLevel { get; set; }
```
