## OptimizationOptions.RemoveUnusedObjects プロパティ

このフラグが true に設定されている場合、すべてのドキュメントオブジェクトがチェックされ、未使用のオブジェクト（すなわち、参照を持たないオブジェクト）がドキュメントから削除されます。

```csharp
public bool RemoveUnusedObjects { get; set; }
```
