## OptimizationOptions.CompressObjects プロパティ

このフラグが `true` に設定されている場合、Pdf オブジェクトは Objest Streams にパックされ、PDF ファイルサイズを削減するために圧縮されます。

```csharp
public bool CompressObjects { get; set; }
```
