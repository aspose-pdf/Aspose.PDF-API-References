## SignatureLengthMismatchException クラス

PDF 署名中に発生するエラーを表します。[`SignHash`](../../aspose.pdf.forms/signhash/) を使用して文書に署名し、署名の実際の長さが [`DefaultSignatureLength`](../../aspose.pdf.forms/signature/defaultsignaturelength/) オプションで指定された長さを超える場合に発生します。

```csharp
public class SignatureLengthMismatchException : PdfException
```
