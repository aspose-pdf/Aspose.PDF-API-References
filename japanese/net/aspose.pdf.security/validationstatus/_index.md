## ValidationStatus 列挙型

証明書検証の検証ステータスを表します。

```csharp
public enum ValidationStatus
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Valid | `0` | 証明書が正常に検証されたことを示します。 |
| Invalid | `1` | 証明書の検証が失敗したことを示します。 |
| Undefined | `2` | 検証プロセスが結論に至らなかったか、実施されなかったことを示します。 |

## 備考

この列挙型は、証明書検証の可能な結果を定義します: - Valid: 証明書が正常に検証されたことを示します。 - Invalid: 証明書の検証が失敗したことを示します。 - Undefined: 検証プロセスが結論に至らなかったか、実施されなかったことを示します。
