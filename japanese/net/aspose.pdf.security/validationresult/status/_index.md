## ValidationResult.Status プロパティ

証明書の検証プロセスのステータスを取得します。

```csharp
public ValidationStatus Status { get; }
```

## 備考

Status プロパティは、証明書検証の結果を示します。可能な値は [`ValidationStatus`](../../validationstatus/) 列挙型で定義されており、Valid、Invalid、または Undefined などがあります。これは、証明書が検証チェックに合格したかどうかを示します。
