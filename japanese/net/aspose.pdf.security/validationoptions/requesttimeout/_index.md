## ValidationOptions.RequestTimeout プロパティ

検証プロセス中のネットワーク関連操作のタイムアウト期間をミリ秒単位で取得または設定します。RequestTimeout プロパティは、取り消しステータスや OCSP サーバーなどのオンラインリソースにアクセスする際に、システムがネットワーク応答を待機する最大時間を定義します。

```csharp
public int RequestTimeout { get; set; }
```
