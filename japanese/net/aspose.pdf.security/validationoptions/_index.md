## ValidationOptions クラス

PDF ドキュメント内のデジタル署名を検証するためのオプションを表します。

```csharp
public sealed class ValidationOptions
```

## コンストラクター

| 名前 | 説明 |
| --- | --- |
| [ValidationOptions](validationoptions/)() | デフォルトのコンストラクター。 |

## プロパティ

| 名前 | 説明 |
| --- | --- |
| [RequestTimeout](../../aspose.pdf.security/validationoptions/requesttimeout/) { get; set; } | 検証プロセス中のネットワーク関連操作のタイムアウト期間（ミリ秒）を取得または設定します。RequestTimeout プロパティは、オンラインリソース（取り消しステータスや OCSP サーバーなど）にアクセスする際に、システムがネットワーク応答を待機する最大時間を定義します。 |
| [ValidationMethod](../../aspose.pdf.security/validationoptions/validationmethod/) { get; set; } | 証明書を検証するために使用されるメソッドを取得または設定します。 |
| [ValidationMode](../../aspose.pdf.security/validationoptions/validationmode/) { get; set; } | PDF ドキュメント内のデジタル署名の検証モードを取得または設定します。ValidationMode プロパティは、検証プロセスの厳格さを決定します。 |
