## SignatureAlgorithmType 列挙型

デジタル署名に使用される署名アルゴリズムの種類を列挙します。

```csharp
public enum SignatureAlgorithmType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Ecdsa | `0` | デジタル署名に使用される楕円曲線デジタル署名アルゴリズム (ECDSA)。 |
| Rsa | `1` | デジタル署名に使用されるリベスト–シャミール–アドレマン (RSA) アルゴリズム。 |
| Dsa | `2` | デジタル署名に使用されるデジタル署名アルゴリズム (DSA)。 |
| Timestamp | `3` | デジタル署名を保証するための署名アルゴリズムとして使用されるタイムスタンプ。 |
| Unknown | `4` | 署名アルゴリズムの種類が不明であることを示します。 |
