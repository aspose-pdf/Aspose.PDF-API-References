## ReplyType 列挙型

注釈と InReplyTo で指定された注釈との間の関係の種類（「返信タイプ」）を列挙します。

```csharp
public enum ReplyType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Undefined | `0` | 未定義の関係。 |
| Reply | `1` | 注釈は InReplyTo で指定された注釈への返信と見なされます。ビューワーアプリケーションは、注釈への返信を個別に表示するのではなく、スレッドコメントの形式で一緒に表示する必要があります。 |
| Group | `2` | 注釈は InReplyTo で指定された注釈とグループ化されています。 |
