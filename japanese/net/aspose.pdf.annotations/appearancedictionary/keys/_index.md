## AppearanceDictionary.Keys プロパティ

辞書のキーを取得します。外観辞書にサブ辞書がある場合、`Keys` には (N&#x7C;R&#x7C;D).state 値が含まれます。ここで、N は通常の外観、R はロールオーバー外観、D はダウン外観、state は状態の名前（例：チェックボックスの On、Off）です。

```csharp
public ICollection<string> Keys { get; }
```
