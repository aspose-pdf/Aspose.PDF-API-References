## AnnotationFlags 列挙型

注釈のさまざまな特性を指定するフラグのセット。

```csharp
[Flags]
public enum AnnotationFlags
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Default | `0` | デフォルト値。 |
| Invisible | `1` | 設定されている場合、標準の注釈タイプのいずれにも属さず、注釈ハンドラーが利用できない場合は、注釈を表示しない。クリアされている場合は、存在する場合は外観辞書によって指定された外観ストリームを使用して、そのような未知の注釈を表示する。 |
| Hidden | `2` | 設定されている場合、注釈のタイプに関係なく、注釈を表示または印刷せず、ユーザーとの対話を許可しない。画面スペースが限られている場合、注釈を選択的に隠したり表示したりする機能は、外観ストリームと組み合わせて、オンラインヘルプシステムに似た補助的なポップアップ情報を表示するために使用できる。 |
| Print | `4` | 設定されている場合、ページが印刷されるときに注釈を印刷する。クリアされている場合、画面に表示されているかどうかに関係なく、注釈を決して印刷しない。これは、印刷されたページ上で意味のある目的を果たさないインタラクティブなボタンを表す注釈にとって便利である。 |
| NoZoom | `8` | 設定されている場合、ページの拡大率に合わせて注釈の外観をスケーリングしない。ページ上の注釈の位置（注釈の矩形の左上隅によって定義される）は、ページの拡大率に関係なく固定される。 |
| NoRotate | `10` | 設定されている場合、ページの回転に合わせて注釈の外観を回転しない。注釈の矩形の左上隅は、ページの回転に関係なく、ページ上の固定された位置に留まる。 |
| NoView | `20` | 設定されている場合、画面に注釈を表示せず、ユーザーとの対話を許可しない。注釈は印刷される可能性がある（Print フラグの設定に依存）が、画面表示およびユーザーとの対話の目的では隠されていると見なされるべきである。 |
| ReadOnly | `40` | 設定されている場合、注釈がユーザーと対話することを許可しない。注釈は表示または印刷される可能性がある（NoView および Print フラグの設定に依存）が、マウスクリックに応答したり、マウスの動きに応じて外観を変更したりしてはならない。このフラグはウィジェット注釈には無視され、その機能は関連するフォームフィールドの ReadOnly フラグに包含される。 |
| Locked | `80` | 設定されている場合、ユーザーが注釈を削除したり、そのプロパティ（位置やサイズを含む）を変更したりすることを許可しない。ただし、このフラグは注釈の内容の変更（フォームフィールドの値など）を制限しない。 |
| ToggleNoView | `100` | 設定されている場合、特定のイベントに対する NoView フラグの解釈を反転させる。典型的な使用法は、マウスカーソルがその上にあるときのみ表示される注釈を持つことである。 |
| LockedContents | `200` | 設定されている場合、ユーザーが注釈の内容を変更することを許可しない。このフラグは、注釈の削除や他の注釈プロパティ（位置やサイズなど）の変更を制限しない。 |
