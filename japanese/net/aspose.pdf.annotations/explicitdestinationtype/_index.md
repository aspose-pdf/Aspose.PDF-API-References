## ExplicitDestinationType 列挙型

```csharp
public enum ExplicitDestinationType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| XYZ | `0` | ウィンドウの左上隅に位置する座標 (左, 上) でページを表示し、ページの内容をズームファクターで拡大します。left、top、またはzoomのいずれかのパラメータがnullの場合、そのパラメータの現在の値が変更されずに保持されることを指定します。ズーム値が0の場合は、null値と同じ意味です。 |
| Fit | `1` | ページの内容がウィンドウ内に水平および垂直に収まるように、ちょうど十分に拡大してページを表示します。必要な水平および垂直の拡大率が異なる場合は、2つのうち小さい方を使用し、他の次元でページをウィンドウ内に中央に配置します。 |
| FitH | `2` | ウィンドウの上端に位置する垂直座標topでページを表示し、ページの内容がウィンドウ内にページの全幅を収めるのに十分に拡大します。topのnull値は、そのパラメータの現在の値が変更されずに保持されることを指定します。 |
| FitV | `3` | ウィンドウの左端に位置する水平座標leftでページを表示し、ページの内容がウィンドウ内にページの全高を収めるのに十分に拡大します。leftのnull値は、そのパラメータの現在の値が変更されずに保持されることを指定します。 |
| FitR | `4` | 左、下、右、上の座標で指定された矩形がウィンドウ内に水平および垂直に完全に収まるように、ページの内容をちょうど十分に拡大して表示します。必要な水平および垂直の拡大率が異なる場合は、2つのうち小さい方を使用し、他の次元で矩形をウィンドウ内に中央に配置します。パラメータのいずれかのnull値は、予測不可能な動作を引き起こす可能性があります。 |
| FitB | `5` | ページの内容がウィンドウ内に水平および垂直に完全に収まるように、バウンディングボックスをちょうど十分に拡大して表示します。必要な水平および垂直の拡大率が異なる場合は、2つのうち小さい方を使用し、他の次元でバウンディングボックスをウィンドウ内に中央に配置します。 |
| FitBH | `6` | ウィンドウの上端に位置する垂直座標topでページを表示し、ページの内容がウィンドウ内にバウンディングボックスの全幅を収めるのに十分に拡大します。topのnull値は、そのパラメータの現在の値が変更されずに保持されることを指定します。 |
| FitBV | `7` | ウィンドウの左端に位置する水平座標leftでページを表示し、ページの内容がウィンドウ内にバウンディングボックスの全高を収めるのに十分に拡大します。leftのnull値は、そのパラメータの現在の値が変更されずに保持されることを指定します。 |
