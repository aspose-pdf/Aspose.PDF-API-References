## FitHExplicitDestination クラス

ウィンドウの上端に位置する垂直座標 top でページを表示し、ページの内容がウィンドウ内に収まるように全幅が拡大された明示的な目的地を表します。top の null 値は、そのパラメータの現在の値が変更されずに保持されることを指定します。

```csharp
public sealed class FitHExplicitDestination : ExplicitDestination
```
