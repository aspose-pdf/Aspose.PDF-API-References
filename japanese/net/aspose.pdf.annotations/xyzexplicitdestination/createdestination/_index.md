## XYZExplicitDestination.CreateDestination メソッド

必要に応じてページの回転を考慮して、指定された位置にデスティネーションを作成します。

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| page | Page | デスティネーションページ。 |
| left | Double | ページ上の左位置。 |
| top | Double | ページ上の上位置。 |
| zoom | Double | ズーム係数（デフォルトは0）。 |
| considerRotation | Boolean | trueの場合、位置はページの回転に応じて再計算されます。 |

### 戻り値

デスティネーションオブジェクト。
