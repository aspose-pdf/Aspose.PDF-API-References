## SubmitFormAction.EmbedForm フィールド

設定されている場合、提出された FDF の F エントリは、FDF が提出される PDF ファイルを表す埋め込まれたファイルストリームを含むファイル仕様である必要があります。

```csharp
public const int EmbedForm;
```
