## FitBVExplicitDestination クラス

ウィンドウの左端に位置する水平座標 left でページを表示し、ページの内容がウィンドウ内のバウンディングボックスの全高に収まるように十分に拡大された明示的な目的地を表します。left の null 値は、そのパラメータの現在の値を変更せずに保持することを指定します。

```csharp
public sealed class FitBVExplicitDestination : ExplicitDestination
```
