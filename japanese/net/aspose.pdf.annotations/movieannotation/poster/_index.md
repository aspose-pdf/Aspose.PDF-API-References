## MovieAnnotation.Poster プロパティ

映画を表すポスター画像がどのように表示されるかを指定するフラグまたはストリームを取得または設定します。true の場合、ポスター画像は映画ファイルから取得されます。false の場合、ポスターは表示されません。

```csharp
public bool Poster { get; set; }
```
