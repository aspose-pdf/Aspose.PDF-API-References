## Measure.NumberFormat.ForceDenominator プロパティ

FractionDisplayment が ShowAsFraction の場合、この値は分数が簡約されるかどうかを決定します。値が true の場合、分数は簡約されない可能性があります。

```csharp
public bool ForceDenominator { get; set; }
```
