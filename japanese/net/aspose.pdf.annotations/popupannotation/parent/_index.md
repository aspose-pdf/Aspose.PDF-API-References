## PopupAnnotation.Parent プロパティ

ポップアップアノテーションが関連付けられる親アノテーションを取得または設定します。このエントリが存在する場合、親アノテーションの Contents、M、C、および T エントリはポップアップアノテーション自体のそれを上書きします。

```csharp
public Annotation Parent { get; set; }
```
