## FitVExplicitDestination クラス

ウィンドウの左端に水平座標 left が配置され、ページの内容がウィンドウ内にページ全体の高さを収めるのに十分な大きさに拡大されたページを表示する明示的な宛先を表します。left の null 値は、そのパラメータの現在の値を変更せずに保持することを指定します。

```csharp
public sealed class FitVExplicitDestination : ExplicitDestination
```
