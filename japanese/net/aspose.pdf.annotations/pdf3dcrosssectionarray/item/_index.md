## PDF3DCrossSectionArray インデクサ

指定されたインデックスの [`PDF3DCrossSection`](../../pdf3dcrosssection/) を取得または設定します。

```csharp
public PDF3DCrossSection this[int index] { get; set; }
```

| パラメーター | 説明 |
| --- | --- |
| index | インデックス。 |

### 戻り値

断面図。

### 例外

| 例外 | 条件 |
| --- | --- |
| IndexOutOfRangeException | 無効なインデックス: インデックスは [1..n] の範囲内である必要があります。ここで n は断面図の数に等しいです。 |
