## PDF3DViewArray インデクサー

指定されたインデックスで表示する [`PDF3DView`](../../pdf3dview/) を取得または設定します。

```csharp
public PDF3DView this[int index] { get; set; }
```

| パラメーター | 説明 |
| --- | --- |
| index | インデックス。 |

### 戻り値

PDF3DView。

### 例外

| 例外 | 条件 |
| --- | --- |
| IndexOutOfRangeException | 無効なインデックス: インデックスは [1..n] の範囲内である必要があります。ここで n はビューの数です。 |
