## Annotation.UpdateAppearanceOnConvert プロパティ

true の場合、注釈の外観は PF ドキュメントを画像に変換する前に更新されます。これにより、フィールドを正しく変換できますが、恐らくより多くの時間がかかります。

```csharp
public static bool UpdateAppearanceOnConvert { get; set; }
```
