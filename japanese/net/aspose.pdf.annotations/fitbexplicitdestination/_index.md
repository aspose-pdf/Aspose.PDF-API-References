## FitBExplicitDestination クラス

ウィンドウ内にバウンディングボックスが完全に収まるように、ページの内容が適切に拡大表示される明示的な目的地を表します。必要な水平方向および垂直方向の拡大率が異なる場合は、2つのうち小さい方を使用し、他の次元でウィンドウ内にバウンディングボックスを中央に配置します。

```csharp
public sealed class FitBExplicitDestination : ExplicitDestination
```
