## FitBHExplicitDestination クラス

ウィンドウの上端に位置する垂直座標 top を持つページを表示し、ページの内容がウィンドウ内のバウンディングボックスの全幅に収まるように十分に拡大された明示的な目的地を表します。top の null 値は、そのパラメータの現在の値を変更せずに保持することを指定します。

```csharp
public sealed class FitBHExplicitDestination : ExplicitDestination
```
