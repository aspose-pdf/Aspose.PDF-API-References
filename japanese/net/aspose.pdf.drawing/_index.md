**Aspose.Pdf.Drawing** 名前空間は、弧、線、円などのグラフィックプリミティブを説明するクラスを提供します。対応するオブジェクトは、新しいドキュメントを作成するために使用できます。

## クラス

| クラス | 説明 |
| --- | --- |
| [Arc](./arc/) | 弧を表します。 |
| [Circle](./circle/) | 円を表します。 |
| [Curve](./curve/) | ベジェ曲線を表します。 |
| [Ellipse](./ellipse/) | 楕円を表します。 |
| [GradientAxialShading](./gradientaxialshading/) | グラデーション軸シェーディングクラスを表します。 |
| [GradientRadialShading](./gradientradialshading/) | グラデーション放射状シェーディングタイプを表します。 |
| [Graph](./graph/) | グラフ - グラフィックス生成段落を表します。 |
| [Line](./line/) | 線を表します。 |
| [Path](./path/) | 弧を表します。 |
| [PatternColorSpace](./patterncolorspace/) | 基本パターンクラスを表します。 |
| [Rectangle](./rectangle/) | 矩形を表します。 |
| [Shape](./shape/) | 形状 - 基本グラフィックスオブジェクトを表します。 |
## 列挙型

| 列挙型 | 説明 |
| --- | --- |
| [ImageFormat](./imageformat/) | この列挙型は、画像フォーマットを表します。 |
