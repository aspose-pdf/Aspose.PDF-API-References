## FormEditor.SetFieldScript メソッド

PushButton フィールドのための JavaScript を設定します。古い JavaScript が存在する場合は、新しいもので置き換えられます。

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| fieldName | String | 完全修飾フィールド名。 |
| script | String | プッシュボタンフィールドに追加/配置される Java スクリプト。 |

### 戻り値

フィールドスクリプトが正常に設定された場合は true。
