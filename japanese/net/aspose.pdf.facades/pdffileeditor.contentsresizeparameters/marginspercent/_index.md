## PdfFileEditor.ContentsResizeParameters.MarginsPercent メソッド

リサイズパラメータを作成します。マージンは初期ページサイズのパーセントで指定されます。

```csharp
public static ContentsResizeParameters MarginsPercent(double left, double right, double top, 
    double bottom)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| left | Double | 左マージン（ページ幅のパーセント）。 |
| right | Double | 右マージン（ページ高さのパーセント）。 |
| top | Double | 上マージン（ページ高さのパーセント）。 |
| bottom | Double | 下マージン（ページ高さのパーセント）。 |

### 戻り値

新しいリサイズパラメータを返します。
