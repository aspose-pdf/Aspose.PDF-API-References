## AutoFiller.GeneratingPath プロパティ

多くの小さな PDF ファイルが生成される場合の小さな PDF ファイルの生成パスを取得または設定します。これは、別のプロパティ [`BasicFileName`](../basicfilename/)BasicFileName と連携します。4 つの出力モードのうちの 1 つです。

```csharp
public string GeneratingPath { get; set; }
```
