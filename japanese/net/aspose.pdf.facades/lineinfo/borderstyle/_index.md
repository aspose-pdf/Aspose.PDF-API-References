## LineInfo.BorderStyle プロパティ

線の境界スタイルを取得または設定します。0 は実線を表し、1 は破線を表し、2 はベベルを表し、3 は挿入を表し、4 は下線を表します。

```csharp
public int BorderStyle { get; set; }
```
