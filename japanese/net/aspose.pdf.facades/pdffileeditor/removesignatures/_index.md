## PdfFileEditor.RemoveSignatures プロパティ

true の場合、すべての署名がフィールドから削除されます（フィールドは残ります）。そうでない場合、無効な署名を取得する可能性があります。

```csharp
public bool RemoveSignatures { get; set; }
```
