## PdfFileEditor.OptimizeSize プロパティ

最適化フラグを取得または設定します。このフラグが設定されている場合、結果ファイル内の等しいリソースストリームは1つのPDFオブジェクトにマージされます。これにより、結果ファイルのサイズを減少させることができますが、実行速度が遅くなり、メモリ要件が大きくなる可能性があります。デフォルト値: false。

```csharp
public bool OptimizeSize { get; set; }
```
