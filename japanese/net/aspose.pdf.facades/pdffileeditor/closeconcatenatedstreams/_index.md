## PdfFileEditor.CloseConcatenatedStreams プロパティ

true に設定すると、操作後にストリームが閉じられます。

```csharp
public bool CloseConcatenatedStreams { get; set; }
```

## 例

```csharp
PdfFileEditor pfe = new PdfFileEditor();
pfe.CloseConcatenatedStreams = true;
```
