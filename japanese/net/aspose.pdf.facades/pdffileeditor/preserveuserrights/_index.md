## PdfFileEditor.PreserveUserRights プロパティ

true の場合、最初のドキュメントのユーザー権限が連結されたドキュメントに適用されます。他のすべてのドキュメントのユーザー権限は無視されます。

```csharp
public bool PreserveUserRights { get; set; }
```
