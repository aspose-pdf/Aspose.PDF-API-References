## PdfFileEditor.MergeDuplicateLayers プロパティ

このプロパティが true の場合、同じ名前の連結されたドキュメントのオプションの内容は、結果のドキュメントの1つのレイヤーにマージされます。 そうでない場合、同じ名前のレイヤーは結果のドキュメントに異なるレイヤーとして保存されます。

```csharp
public bool MergeDuplicateLayers { get; set; }
```
