## DocumentPrivilege.PrintAllowLevel プロパティ

ドキュメントの権限の印刷レベルを取得および設定します。Adobe Professional の Printing Allowed 設定と同様です。0: なし。1: 低解像度 (150 dpi)。2: 高解像度。

```csharp
public int PrintAllowLevel { get; set; }
```

## 備考

プロパティの値が -1 の場合、レベルは未定義です。
