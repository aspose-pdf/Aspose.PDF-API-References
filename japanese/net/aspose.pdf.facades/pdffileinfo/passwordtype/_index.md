## PdfFileInfo.PasswordType プロパティ

PdfFileInfo インスタンスを作成するために渡されたパスワードのタイプを返します。`PasswordType` にある可能な値を参照してください。PDF ドキュメントは、ユーザーパスワード（またはオープンパスワード）とオーナーパスワード（または権限、編集パスワード）の両方を使用して開くことができることに注意してください。

```csharp
public PasswordType PasswordType { get; }
```
