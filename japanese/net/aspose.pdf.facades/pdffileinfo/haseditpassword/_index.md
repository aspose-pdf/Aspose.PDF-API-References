## PdfFileInfo.HasEditPassword プロパティ

権限または文書セキュリティプロパティを変更するためにパスワードが必要な場合は true を返します。このプロパティは、[`PdfFileInfo`](../) コンストラクタに有効なパスワードが提供された場合のみ読み取ることができることに注意してください。PasswordType が Inaccessible の場合（無効なパスワードが提供されたことを意味します）、このプロパティを読み取ると [`InvalidPasswordException`](../../../aspose.pdf/invalidpasswordexception/) で失敗します。

```csharp
public bool HasEditPassword { get; }
```
