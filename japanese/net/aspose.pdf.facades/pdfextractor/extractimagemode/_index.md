## PdfExtractor.ExtractImageMode プロパティ

画像抽出プロセスのモードを設定します。

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## 備考

デフォルト値は ExtractImageMode.DefinedInResources で、リソースに定義されたすべての画像を抽出します。実際に表示されている画像を抽出するには、ExtractImageMode.ActuallyUsed モードを使用する必要があります。
