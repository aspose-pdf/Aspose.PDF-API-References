## PdfExtractor.IsBidi プロパティ

テキストにヘブライ語またはアラビア語の記号がある場合は真です。この場合は特に考慮する必要があります。なぜなら、文字列関数はその動作を変更し、テキストを右から左に処理し始めるからです（数字やその他の非テキスト文字を除く）。

```csharp
public bool IsBidi { get; }
```
