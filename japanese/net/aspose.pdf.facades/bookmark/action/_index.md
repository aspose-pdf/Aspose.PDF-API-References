## Bookmark.Action プロパティ

ブックマークにバインドされたアクションを取得または設定します。PageNumberが指定されている場合、アクションは指定できません。アクションタイプには、"GoTo"、"GoToR"、"Launch"、"Named"が含まれます。

```csharp
public string Action { get; set; }
```
