## PdfViewer.PrintAsImage プロパティ

PdfViewer が画像として印刷するモードを設定または取得します。

```csharp
public bool PrintAsImage { get; set; }
```

## 備考

true の場合は常に画像として印刷します（印刷される画像を生成します）。false の場合は、すべての機能がサポートされている場合、デバイスに直接印刷します。ドキュメントにサポートされていない機能が含まれている場合、システムは自動的に画像として印刷することを決定する場合があります。デフォルト値は false です。
