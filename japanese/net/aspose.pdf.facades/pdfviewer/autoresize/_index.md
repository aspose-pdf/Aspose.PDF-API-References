## PdfViewer.AutoResize プロパティ

ファイルが最適化されたサイズで印刷されるかどうかを示す bool 値を取得または設定します。 false の場合、ページスケーリングなしでページを印刷します。 true の場合、印刷可能な領域に合わせてスケーリングしてページを印刷します。

```csharp
public bool AutoResize { get; set; }
```
