## BarcodeField.ECCプロパティ

エラー訂正係数を表す整数値を取得します。PDF417の場合は0から8まで、QRCodeの場合は0から3まで（0は'L'、1は'M'、2は'Q'、3は'H'）です。

```csharp
public int ECC { get; }
```
