## BarcodeField.XSymHeight プロパティ

2 つのバーコードモジュール間の垂直距離をピクセル単位で取得します。比率 XSymHeight/XSymWidth は整数値でなければなりません。PDF417 の場合、許容される比率範囲は 1 から 4 です。QRCode と DataMatrix の場合、この比率は常に 1 でなければなりません。

```csharp
public int XSymHeight { get; }
```
