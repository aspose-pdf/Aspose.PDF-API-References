## ChoiceField.SelectedItems プロパティ

選択されたアイテムの配列を取得または設定します。マルチセレクトリストの場合、配列には複数のアイテムが含まれます。シングルセレクションリストの場合、単一のアイテムが含まれます。

```csharp
public virtual int[] SelectedItems { get; set; }
```
