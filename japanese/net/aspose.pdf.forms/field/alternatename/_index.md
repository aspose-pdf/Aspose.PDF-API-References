## Field.AlternateName プロパティ

フィールドの代替名を取得または設定します（ユーザーインターフェイスでフィールドが識別される場所で実際のフィールド名の代わりに使用される代替フィールド名）。代替名はAdobe Acrobatのフィールドツールチップとして使用されます。

```csharp
public string AlternateName { get; set; }
```
