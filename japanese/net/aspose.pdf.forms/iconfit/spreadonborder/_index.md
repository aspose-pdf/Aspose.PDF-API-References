## IconFit.SpreadOnBorder プロパティ

true の場合、ボタンの外観は、境界線の幅を考慮せずに、注釈の範囲内に完全に収まるようにスケーリングされることを示します。

```csharp
public bool SpreadOnBorder { get; set; }
```
