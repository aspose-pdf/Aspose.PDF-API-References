## Form.AutoRecalculate プロパティ

設定すると、任意のフィールドが変更されるとすべてのフォームフィールドが再計算されます。デフォルト値は true です。計算フィールドが大量にあるフォームを入力する際のパフォーマンスを向上させるために false に設定します。

```csharp
public bool AutoRecalculate { get; set; }
```
