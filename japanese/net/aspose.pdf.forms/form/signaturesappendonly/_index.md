## Form.SignaturesAppendOnly プロパティ

設定されている場合、ドキュメントには署名が含まれており、ファイルが以前の内容を変更する方法で保存（書き込み）されると無効になる可能性があります。増分更新とは対照的です。

```csharp
public bool SignaturesAppendOnly { get; set; }
```
