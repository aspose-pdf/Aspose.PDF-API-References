## Form.IgnoreNeedsRendering プロパティ

このプロパティが true の場合、NeedsRendering キーの値は XFA フォームから標準フォームへの変換中に無視されます。デフォルトでは false です。

```csharp
public bool IgnoreNeedsRendering { get; set; }
```
