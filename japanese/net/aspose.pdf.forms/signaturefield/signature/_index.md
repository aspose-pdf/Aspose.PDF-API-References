## SignatureField.Signature プロパティ

署名オブジェクトを取得します。このオブジェクトには公開鍵暗号基準に関する署名データが含まれています。クラス [`PKCS1`](../../pkcs1/)、[`PKCS7`](../../pkcs7/) および [`PKCS7Detached`](../../pkcs7detached/) は、すべてのサポートされている署名オブジェクトのタイプを表します。

```csharp
public Signature Signature { get; }
```
