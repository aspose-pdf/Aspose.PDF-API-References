## Form.FlattenSettings.UpdateAppearances プロパティ

設定すると、すべてのフィールドの外観がフラット化の前に再生成されます。このオプションは、フィールドが正しくフラット化されていない場合に役立つことがあります。このオプションはパフォーマンスを低下させる可能性があります。デフォルトでは false に設定されています。

```csharp
public bool UpdateAppearances { get; set; }
```
