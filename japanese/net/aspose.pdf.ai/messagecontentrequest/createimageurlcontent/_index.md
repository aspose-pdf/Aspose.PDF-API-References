## MessageContentRequest.CreateImageUrlContent メソッド

メッセージのための画像 URL コンテンツを作成します。

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| imageUrl | String | 画像の URL。 |
| detail | String | 画像 URL の詳細。ユーザーによって指定された場合、画像の詳細レベルを指定します。"low" はトークンを少なく使用し、"high" を使用することで高解像度を選択できます。デフォルトは "auto" です。 |

### 戻り値

ImageUrl コンテンツを持つ MessageContentRequest の新しいインスタンス。
