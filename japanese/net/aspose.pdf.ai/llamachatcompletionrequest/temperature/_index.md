## LlamaChatCompletionRequest.Temperature プロパティ

使用するサンプリング温度を設定または取得します。0 から 2 の間です。0.8 のような高い値は出力をよりランダムにし、0.2 のような低い値はより集中して決定論的にします。デフォルト値は 1 です。

```csharp
public double? Temperature { get; set; }
```
