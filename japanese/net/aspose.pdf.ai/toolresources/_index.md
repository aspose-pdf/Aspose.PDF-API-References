## ToolResources クラス

アシスタントのツールによって使用されるリソースのセットを表します。リソースはツールの種類に特有です。たとえば、code_interpreter ツールはファイル ID のリストを必要とし、file_search ツールはベクターストア ID のリストを必要とします。

```csharp
public class ToolResources
```
