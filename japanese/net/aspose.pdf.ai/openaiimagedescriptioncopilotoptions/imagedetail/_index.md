## OpenAIImageDescriptionCopilotOptions.ImageDetail プロパティ

ユーザーによって指定された場合、画像の詳細レベルを取得または設定します。"low" はトークンを少なく使用し、"high" を使用することで高解像度を選択できます。設定されていない場合はデフォルトで "auto" になります。

```csharp
public string ImageDetail { get; set; }
```
