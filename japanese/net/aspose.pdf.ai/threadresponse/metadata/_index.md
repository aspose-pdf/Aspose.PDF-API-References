## ThreadResponse.Metadata プロパティ

オブジェクトに添付できる 16 のキーと値のペアのセットを取得または設定します。これは、構造化された形式でオブジェクトに関する追加情報を保存するのに便利です。キーは最大 64 文字の長さで、値は最大 512 文字の長さです。

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
