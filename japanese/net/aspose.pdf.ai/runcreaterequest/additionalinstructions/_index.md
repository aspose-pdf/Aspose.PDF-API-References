## RunCreateRequest.AdditionalInstructions プロパティ

追加の指示を取得または設定します。実行の指示の最後に追加の指示を追加します。これは、他の指示を上書きすることなく、実行ごとに動作を変更するのに便利です。

```csharp
public string AdditionalInstructions { get; set; }
```
