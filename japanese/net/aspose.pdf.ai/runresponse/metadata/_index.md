## RunResponse.Metadata プロパティ

オブジェクトに添付できる 16 のキーと値のペアのセットを取得または設定します。これは、オブジェクトに関する追加情報を構造化された形式で保存するのに役立ちます。キーは最大 64 文字、値は最大 512 文字までです。

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
