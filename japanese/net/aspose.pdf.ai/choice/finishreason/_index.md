## Choice.FinishReason プロパティ

モデルがトークンの生成を停止した理由を取得または設定します。これは、モデルが自然な停止点に達した場合や、リクエストで指定された最大トークン数に達した場合に提供された停止シーケンスの長さで停止します。

```csharp
public string FinishReason { get; set; }
```
