## CompletionCreateRequest.Seed プロパティ

Seed 値を取得または設定します。この機能はベータ版です。指定された場合、当社のシステムは、同じシードとパラメータでの繰り返しリクエストが同じ結果を返すように、決定論的にサンプリングするために最善を尽くします。決定論は保証されておらず、バックエンドの変更を監視するために system_fingerprint レスポンスパラメータを参照する必要があります。

```csharp
public int? Seed { get; set; }
```
