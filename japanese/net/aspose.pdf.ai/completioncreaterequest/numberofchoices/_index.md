## CompletionCreateRequest.NumberOfChoices プロパティ

各入力メッセージに対して生成するチャット完了選択肢の数を取得または設定します。生成されたトークンの数に基づいて料金が発生することに注意してください。コストを最小限に抑えるために n を 1 に設定してください。

```csharp
public int? NumberOfChoices { get; set; }
```
