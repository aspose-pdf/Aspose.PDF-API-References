## CompletionCreateRequest.ResponseFormat プロパティ

モデルが出力する必要がある形式を指定するオブジェクトを取得または設定します。GPT-4 Turbo および gpt-3.5-turbo-1106 より新しいすべての GPT-3.5 Turbo モデルと互換性があります。{ "type": "json_object" } に設定すると、JSON モードが有効になり、モデルが生成するメッセージが有効な JSON であることが保証されます。

```csharp
public ResponseFormat ResponseFormat { get; set; }
```
