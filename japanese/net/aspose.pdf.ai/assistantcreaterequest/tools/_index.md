## AssistantCreateRequest.Tools プロパティ

アシスタントで有効なツールのリストを取得または設定します。アシスタントごとに最大128のツールを持つことができます。ツールは code_interpreter、file_search、または function のタイプであることができます。

```csharp
public List<Tool> Tools { get; set; }
```
