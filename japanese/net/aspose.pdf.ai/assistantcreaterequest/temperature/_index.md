## AssistantCreateRequest.Temperature プロパティ

0 と 2 の間で使用するサンプリング温度を取得または設定します。0.8 のような高い値は出力をよりランダムにし、0.2 のような低い値はより焦点を絞り、決定論的にします。

```csharp
public double? Temperature { get; set; }
```
