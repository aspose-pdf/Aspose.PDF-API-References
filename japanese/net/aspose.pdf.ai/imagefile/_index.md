
## プロパティ

| 名前 | 説明 |
| --- | --- |
| [Detail](../../aspose.pdf.ai/imagefile/detail/) { get; set; } | ユーザーによって指定された場合、画像の詳細レベルを取得または設定します。lowはトークンを少なく使用し、highを使用することで高解像度を選択できます。 |
| [FileId](../../aspose.pdf.ai/imagefile/fileid/) { get; set; } | メッセージの内容における画像のファイルIDを取得または設定します。ファイルの内容を後で表示する必要がある場合は、ファイルをアップロードする際にpurpose="vision"を設定してください。 |
