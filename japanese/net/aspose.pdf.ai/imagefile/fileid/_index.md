## ImageFile.FileId プロパティ

メッセージコンテンツ内の画像のファイル ID を取得または設定します。ファイルのコンテンツを後で表示する必要がある場合は、ファイルをアップロードする際に purpose="vision" を設定してください。

```csharp
public string FileId { get; set; }
```
