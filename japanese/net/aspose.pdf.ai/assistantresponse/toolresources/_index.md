## AssistantResponse.ToolResources プロパティ

アシスタントのツールによって使用されるリソースのセットを取得または設定します。リソースはツールの種類に特有です。たとえば、code_interpreter ツールはファイル ID のリストを必要とし、file_search ツールはベクターストア ID のリストを必要とします。

```csharp
public ToolResources ToolResources { get; set; }
```
