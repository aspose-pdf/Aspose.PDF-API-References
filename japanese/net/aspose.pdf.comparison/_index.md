## クラス

| クラス | 説明 |
| --- | --- |
| [ComparisonOptions](./comparisonoptions/) | PDF ドキュメント比較オプションクラスを表します。 |
| [DiffOperation](./diffoperation/) | 差分操作のクラスを表します。 |
| [DocumentComparisonStatistics](./documentcomparisonstatistics/) | ドキュメント比較統計クラスを表します。 |
| [GraphicalPdfComparer](./graphicalpdfcomparer/) | PDF ドキュメントを視覚的に比較するためのクラスを表します。主にグラフィカルな性質の小さな変更を検索するために使用されるべきです。テキストコンテンツの変更を比較するには、他の PDF 比較クラスを使用してください。 |
| [HtmlDiffOutputGenerator](./htmldiffoutputgenerator/) | テキストの違いの HTML 表現を生成するためのクラスを表します。削除された改行は段落マークで示されます。 |
| [ImagesDifference](./imagesdifference/) | 2 つの PDF ページを比較した結果クラスを表します。 |
| [JsonDiffOutputGenerator](./jsondiffoutputgenerator/) | PDF ドキュメントまたはページの比較結果を JSON 形式で表示するためのクラスを表します。 |
| [MarkdownDiffOutputGenerator](./markdowndiffoutputgenerator/) | テキストの違いの Markdown 表現を生成するためのクラスを表します。Markdown 構文のため、空白文字の変更を表示することはできません。変更の選択は、フォーマットの周りに空白文字を追加することを意味します。さもなければ、Markdown ビューアはテキストを正しく表示しません。削除された改行は - 段落マークで示されます。 |
| [OutputTextStyle](./outputtextstyle/) | テキスト変更をマークするためのスタイルセットクラスを表します。 |
| [PdfOutputGenerator](./pdfoutputgenerator/) | テキストの違いの PDF 表現を生成するためのクラスを表します。 |
| [SideBySideComparisonOptions](./sidebysidecomparisonoptions/) | サイドバイサイド出力でドキュメントを比較するためのオプションクラスを表します。 |
| [SideBySidePdfComparer](./sidebysidepdfcomparer/) |  |
| [TextItemComparisonStatistics](./textitemcomparisonstatistics/) | テキスト比較統計クラスを表します。 |
| [TextPdfComparer](./textpdfcomparer/) | 2 つの PDF ページまたは PDF ドキュメントを比較するためのクラスを表します。 |
| [TextStyle](./textstyle/) | テキストスタイルクラスを表します。 |
## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IFileOutputGenerator](./ifileoutputgenerator/) | テキスト間の違いをファイルに出力するためのインターフェースを表します。 |
| [IStringOutputGenerator](./istringoutputgenerator/) | テキスト間の違いを文字列に出力するためのインターフェースを表します。 |
