## SideBySideComparisonOptions.AdditionalChangeMarks プロパティ

追加の変更マーカーが表示されるかどうかを決定するプロパティを取得および設定します。設定すると、現在のページにはないが別のページに存在する変更マークが表示されます。変更が単語の間にある場合、マークは空白文字に対して正確に配置されないことがあります。デフォルト値は `false` です。

```csharp
public bool AdditionalChangeMarks { get; set; }
```
