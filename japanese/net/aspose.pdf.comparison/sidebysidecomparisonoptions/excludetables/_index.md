## SideBySideComparisonOptions.ExcludeTables プロパティ

テーブルが比較から除外されるかどうかを決定するオプションを取得および設定します。このオプションは [`ComparisonArea1`](../comparisonarea1/) および [`ComparisonArea2`](../comparisonarea2/) と一緒に設定することはできません。デフォルト値は `false` です。

```csharp
public bool ExcludeTables { get; set; }
```
