## Element.ActualText プロパティ

（オプション; PDF 1.4）構造要素とその子要素の正確な置き換えであるテキスト。この置き換えテキスト（できるだけ小さなコンテンツに適用されるべき）は、障害のあるユーザーへのアクセシビリティをサポートするためやその他の目的で文書の内容を抽出する際に便利です。

```csharp
public virtual string ActualText { get; set; }
```
