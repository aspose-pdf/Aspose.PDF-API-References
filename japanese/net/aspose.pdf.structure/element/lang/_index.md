## Element.Lang プロパティ

（オプション; PDF 1.4）ネストされた構造要素やマークされたコンテンツの言語仕様によって上書きされていない場合を除く、構造要素内のすべてのテキストの自然言語を指定する言語です。

```csharp
public virtual string Lang { get; set; }
```
