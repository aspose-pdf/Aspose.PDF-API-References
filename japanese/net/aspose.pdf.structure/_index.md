**Aspose.Pdf.Structure** 名前空間は、文書の論理構造を扱うためのクラスを提供し、文書を章やセクションに整理したり、図、表、脚注などの特別な要素を識別したりします。

## クラス

| クラス | 説明 |
| --- | --- |
| [Element](./element/) | 論理構造の基本要素を表すクラス。 |
| [ElementCollection](./elementcollection/) | 基本的な論理構造要素のコレクション。 |
| [FigureElement](./figureelement/) | 論理構造の図を表すクラス。 |
| [RootElement](./rootelement/) | ルート構造要素。 |
| [StructElement](./structelement/) | 一般的な構造要素。 |
| [TextElement](./textelement/) | 文書の論理構造の一般的なテキスト要素。 |
