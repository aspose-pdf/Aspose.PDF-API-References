## SetDash.Pattern プロパティ

ダッシュパターン。配列の要素は、交互のダッシュとギャップの長さを指定する数値でなければなりません。1 要素の配列の場合、ダッシュとギャップの長さは等しいです。

```csharp
public int[] Pattern { get; set; }
```
