## SetDash.Phaseプロパティ

ダッシュフェーズ。パスのストロークを開始する前に、ダッシュ配列はサイクルされ、ダッシュとギャップの長さが合計されます。累積長がダッシュフェーズで指定された値に等しくなると、パスのストロークが開始され、その時点からダッシュ配列は循環的に使用されます。

```csharp
public int Phase { get; set; }
```
