**Aspose.Pdf.Devices** 名前空間は、ドキュメントを画像またはプレーンテキストとして表現するために使用されるクラスを提供します。したがって、ドキュメントはテキストまたはさまざまなグラフィックデバイスに送信でき、テキストまたはグラフィック表現を取得したいことを意味します。

## クラス

| クラス | 説明 |
| --- | --- |
| [BmpDevice](./bmpdevice/) | pdf ドキュメントページを bmp に保存するのを助ける画像デバイスを表します。 |
| [Device](./device/) | すべてのタイプのデバイスのための抽象クラス。デバイスは、pdf ドキュメントをある形式で表現するために使用されます。たとえば、ドキュメントページは画像またはテキストとして表現できます。 |
| [DicomDevice](./dicomdevice/) | pdf ドキュメントページを Dicom 形式に保存するのを助ける画像デバイスを表します。 |
| [DocumentDevice](./documentdevice/) | pdf ドキュメント全体を処理するために使用されるすべてのデバイスのための抽象クラス。 |
| [EmfDevice](./emfdevice/) | pdf ドキュメントページを emf に保存するのを助ける画像デバイスを表します。 |
| [GifDevice](./gifdevice/) | pdf ドキュメントページを gif に保存するのを助ける画像デバイスを表します。 |
| [ImageDevice](./imagedevice/) | 画像デバイスのための抽象クラス。 |
| [JpegDevice](./jpegdevice/) | pdf ドキュメントページを jpeg に保存するのを助ける画像デバイスを表します。 |
| [Margins](./margins/) | このクラスは画像の余白を表します。 |
| [PageDevice](./pagedevice/) | pdf ドキュメントの特定のページを処理するために使用されるすべてのデバイスのための抽象クラス。 |
| [PngDevice](./pngdevice/) | pdf ドキュメントページを png に保存するのを助ける画像デバイスを表します。 |
| [Resolution](./resolution/) | 画像解像度を保持するためのクラスを表します。 |
| [TextDevice](./textdevice/) | pdf ドキュメントページをテキストに変換するためのクラスを表します。 |
| [ThumbnailDevice](./thumbnaildevice/) | pdf ドキュメントページをサムネイル画像に保存する画像デバイスを表します。 |
| [TiffDevice](./tiffdevice/) | このクラスは、pdf ドキュメントをページごとに 1 つの tiff 画像に保存するのを助けます。 |
| [TiffSettings](./tiffsettings/) | このクラスは、pdf を Tiff にインポートするための設定を表します。 |
## 列挙型

| 列挙型 | 説明 |
| --- | --- |
| [ColorDepth](./colordepth/) | Tiff 画像デバイスに渡されるパラメータ値を指定するために使用されます。 |
| [CompressionType](./compressiontype/) | Tiff 画像デバイスに渡されるパラメータ値を指定するために使用されます。 |
| [FormPresentationMode](./formpresentationmode/) | pdf ドキュメントを印刷または画像に変換する際のフォームプレゼンテーションモードを指定するために使用されます。 |
| [ShapeType](./shapetype/) | この列挙型は、抽出された画像の形状タイプを表します。 |
