## TiffSettings.SkipBlankPages プロパティ

空白ページをスキップするかどうかを示す値を取得または設定します。

```csharp
public bool SkipBlankPages { get; set; }
```

### プロパティ値

空白ページをスキップする必要がある場合は `true`；それ以外の場合は `false`。

## 備考

デフォルト値は false です
