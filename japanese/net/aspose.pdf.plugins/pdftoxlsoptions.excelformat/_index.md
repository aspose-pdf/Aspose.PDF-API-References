## PdfToXlsOptions.ExcelFormat 列挙型

.xlsx、.xls/xml または csv ファイル形式を指定できます。 デフォルト値は XLSX です。

```csharp
public enum ExcelFormat
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| XMLSpreadSheet2003 | `0` | Excel 2003 XML 形式 |
| XLSX | `1` | Office Open XML (.xlsx) ファイル形式 |
| CSV | `2` | カンマ区切り値 (CSV) ファイル形式 |
| XLSM | `3` | マクロ対応の Office Open XML (.xlsm) ファイル形式 |
| ODS | `4` | OpenDocument スプレッドシート |
