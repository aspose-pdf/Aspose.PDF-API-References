## PdfAOptionsBase.PuaSymbolsProcessingStrategy プロパティ

PDF ドキュメント内のプライベート ユース エリア (PUA) シンボルを処理するための戦略を取得または設定します。

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### プロパティ値

PUA シンボルを処理するための戦略。デフォルトは、レベル B 準拠ドキュメントの場合は None、レベル A 準拠ドキュメントの場合は SurroundPuaTextWithEmptyActualText です。
