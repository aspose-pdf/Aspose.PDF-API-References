## PdfAOptionsBase.SymbolicFontEncodingStrategy プロパティ

PDF/A 形式に変換する際のシンボリックフォントのエンコーディング戦略を取得または設定します。

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### プロパティ値

シンボリックフォントエンコーディング戦略。

## 備考

このプロパティを使用すると、元の TrueType シンボリックフォントがソースドキュメントに複数の CMap サブテーブルを含む場合に、結果ドキュメントにコピーされる CMap サブテーブルを制御できます。
