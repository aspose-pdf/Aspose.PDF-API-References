## TiffOptions.SkipBlankPages プロパティ

空白ページをスキップするかどうかを示す値を取得または設定します。

```csharp
public bool SkipBlankPages { get; set; }
```

### プロパティ値

`true` は空白ページをスキップする必要がある場合; それ以外は `false`。

## 備考

デフォルト値は false です。
