## TiffOptions.Brightness プロパティ

白と黒の色の変換の値の境界を取得または設定します。このパラメータは EncoderValue.CompressionCCITT4、EncoderValue.CompressionCCITT3、EncoderValue.CompressionRle または ColorDepth.Format1bpp == 1 と共に適用できます。

```csharp
public float Brightness { get; set; }
```

### プロパティ値

明るさの値は 0 から 1 の範囲である必要があります。デフォルト値は 0.33f に等しいです。
