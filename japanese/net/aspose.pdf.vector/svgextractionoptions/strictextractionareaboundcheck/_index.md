## SvgExtractionOptions.StrictExtractionAreaBoundCheck プロパティ

[`ExtractionAreaBound`](../extractionareabound/) で指定された矩形内にサブパスがあるかどうかを厳密にチェックするオプションを取得および設定します。false に設定すると、[`ExtractionAreaBound`](../extractionareabound/) に完全に含まれていないサブパスが抽出されます。デフォルト値は `True` です。

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
