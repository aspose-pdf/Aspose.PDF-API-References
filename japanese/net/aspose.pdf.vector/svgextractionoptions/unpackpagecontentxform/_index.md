## SvgExtractionOptions.UnpackPageContentXForm プロパティ

ページに見つかった XFrom を展開するかどうかを決定するフラグを取得および設定します。XFrom 要素は異なる SVG ファイルに含まれることがあります。ページコンテンツからの Do ステートメントによってレンダリングされた XForms のみが展開されます。ネストされた XForms は展開されません。

```csharp
public bool UnpackPageContentXForm { get; set; }
```
