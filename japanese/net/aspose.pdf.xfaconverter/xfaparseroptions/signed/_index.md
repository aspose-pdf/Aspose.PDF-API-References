## XfaParserOptions.Signed プロパティ

このプロパティが true の場合、ドキュメントは xfa フォームストリームを使用して変換されます（存在する場合）。false の場合、xfa フォームストリームは無視されます。このプロパティは、署名を確認するために使用されるチェックサムの計算方法が明確でないために導入されました。

```csharp
public bool Signed { get; set; }
```
