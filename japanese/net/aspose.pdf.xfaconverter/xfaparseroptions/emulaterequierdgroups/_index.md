## XfaParserOptions.EmulateRequierdGroups プロパティ

このプロパティが true の場合、必須の Xfa "除外グループ" のために追加の赤い矩形が描画されます。このプロパティは、標準へのフォームの Xfa 表現の変換中に除外グループの類似物が存在しないために導入されました。デフォルトでは false です。

```csharp
public bool EmulateRequierdGroups { get; set; }
```
