## FileParams.CheckSum プロパティ

圧縮されていない埋め込みファイルのバイトのチェックサムである 16 バイトの文字列。チェックサムは、埋め込みファイルストリームのバイトに標準の MD5 メッセージダイジェストアルゴリズムを適用することによって計算されます。

```csharp
public string CheckSum { get; }
```
