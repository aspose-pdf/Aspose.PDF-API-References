## ParagraphPositioningMode 列挙型

ページ上の要素の位置を決定するためのバリアントを指定します。

```csharp
public enum ParagraphPositioningMode
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Default | `0` | 位置は、以前に配置された要素によって決定されます。要素を追加することは、後続の要素の位置を決定する際に考慮されます。 |
| Absolute | `1` | 位置は Left および Top の値によって指定され、以前の要素には依存せず、後続の要素の位置にも影響を与えません。 |
