## ToUnicodeProcessingRules.RemoveSpacesFromCMapNames プロパティ

一部のフォントには、名前にスペースが含まれる ToUnicode 文字コードマップがあります。これらのスペースは、ユニコードテキストマッピングでエラーを引き起こす可能性があります。このフラグは、ToUnicode 文字コードマップの名前からスペースを削除するように指示します。デフォルトは false です。

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
