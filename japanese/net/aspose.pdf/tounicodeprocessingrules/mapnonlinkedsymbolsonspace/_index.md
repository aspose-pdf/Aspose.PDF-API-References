## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace プロパティ

一部のフォントは、いくつかのテキストシンボルのユニコードに関する情報を提供しません。この情報の欠如は、「テキストはユニコードにマッピングできません」というエラーを引き起こします。このフラグを使用して、非リンクシンボルをユニコードの「スペース」（コード 32）にマッピングします。

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
