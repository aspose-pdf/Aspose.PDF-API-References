## Color 不等号演算子

2 つの Color が等しくない場合は true を返します。

```csharp
public static bool operator !=(Color x, Color y)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | Color | 最初の Color オブジェクト。 |
| y | Color | 2 番目の Color オブジェクト。 |

### 戻り値

Color オブジェクトが等しくない場合は true。
