## Color.FromCmyk メソッド

CMYK カラーコンポーネントから有効な PDF Color オブジェクトを取得します。

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| c | Double | シアンカラーコンポーネント (値 0 - 1)。 |
| m | Double | マゼンタカラーコンポーネント (値 0 - 1)。 |
| y | Double | イエローカラーコンポーネント (値 0 - 1)。 |
| k | Double | キーカラーコンポーネント (値 0 - 1)。 |

### 戻り値

各コンポーネント値が [0..1] 範囲にある Color オブジェクト。
