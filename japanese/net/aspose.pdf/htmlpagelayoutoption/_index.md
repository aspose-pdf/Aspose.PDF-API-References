## HtmlPageLayoutOption 列挙型

他のオプションと共にフラグを指定し、ページのサイズとレイアウトを決定します。

```csharp
public enum HtmlPageLayoutOption
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | `0` | HtmlPageLayoutOption がページのサイズとレイアウトに影響を与えないことを示すデフォルト値です。 |
| FitToWidestContentWidth | `3` | ページの幅が指定されたページ幅ではなく、コンテンツサイズ自体から決定されることを示します。 |
| ScaleToPageWidth | `256` | ドキュメントの内容がページに合わせてスケーリングされることを示します。 |
