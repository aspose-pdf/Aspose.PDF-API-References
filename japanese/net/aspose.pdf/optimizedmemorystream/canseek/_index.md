## OptimizedMemoryStream.CanSeek プロパティ

派生クラスでオーバーライドされた場合、現在のストリームがシークをサポートしているかどうかを示す値を取得します。

```csharp
public override bool CanSeek { get; }
```

### 戻り値

ストリームがシークをサポートしている場合は true; それ以外の場合は false。
