## OptimizedMemoryStream.CanWrite プロパティ

派生クラスでオーバーライドされた場合、現在のストリームが書き込みをサポートしているかどうかを示す値を取得します。

```csharp
public override bool CanWrite { get; }
```

### 戻り値

ストリームが書き込みをサポートしている場合は true; それ以外の場合は false。
