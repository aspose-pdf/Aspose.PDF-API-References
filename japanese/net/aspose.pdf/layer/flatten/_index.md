## Layer.Flatten メソッド

指定されたレイヤーをフラット化します。

```csharp
public void Flatten(bool cleanupContentStream)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| cleanupContentStream | ブール | コンテンツストリームからオプションのコンテンツグループマーカーを削除するかどうかを指定します。 |

## 備考

*cleanupContentStream* パラメーターを false に設定すると、フラット化のプロセスが速くなります。
