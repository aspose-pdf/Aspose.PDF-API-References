## PdfFormatConversionOptions.OptimizeFileSize プロパティ

PDF/A ドキュメントのファイルサイズを削減するための特別な変換モードを有効/無効にするフラグを取得または設定します。現在、このフラグは PDF ドキュメントで使用されるフォントの最適化に影響を与え、将来的にはこのフラグがグラフィックなどの他のデータ構造の最適化を切り替えるためにも使用される可能性があります。このフラグとモードの設定はファイルサイズを大幅に削減する可能性がありますが、同時に変換のパフォーマンスを大幅に低下させる可能性があります。

```csharp
public bool OptimizeFileSize { get; set; }
```
