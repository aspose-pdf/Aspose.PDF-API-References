## SaveOptions.WarningHandler プロパティ

生成された警告を処理するためのコールバック。WarningHandler は、Continue または Abort のいずれかを指定する ReturnAction 列挙型のアイテムを返します。Continue はデフォルトのアクションであり、Save 操作は続行されますが、ユーザーは Abort を返すこともでき、その場合は Save 操作を中止する必要があります。

```csharp
public IWarningCallback WarningHandler { get; set; }
```
