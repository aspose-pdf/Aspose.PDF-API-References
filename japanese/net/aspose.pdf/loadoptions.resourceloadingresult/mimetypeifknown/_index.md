## LoadOptions.ResourceLoadingResult.MIMETypeIfKnown フィールド

読み込まれたリソースの MIME タイプに関する知識は、コンバータにとって有用です。このパラメータに、読み込み後に知られている場合は MIME タイプを提供できます。MIME タイプが不明な場合や提供する必要がない場合は、パラメータを null のままにしてください。

```csharp
public string MIMETypeIfKnown;
```
