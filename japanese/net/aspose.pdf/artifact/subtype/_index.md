## Artifact.Subtypeプロパティ

アーティファクトのサブタイプを取得します。アーティファクトが非標準のサブタイプを持つ場合、サブタイプの名前はCustomSubtypeを介して読み取ることができます。

```csharp
public ArtifactSubtype Subtype { get; set; }
```
