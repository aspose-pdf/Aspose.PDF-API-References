## FileSpecification.Contents プロパティ

コンテンツファイルを取得または設定します。このプロパティは、メモリにロードされたデータを返し、大きなデータの場合はメモリ不足例外を引き起こす可能性があります。メモリ使用量を減らすには、StreamContents を使用してください。

```csharp
public Stream Contents { get; set; }
```
