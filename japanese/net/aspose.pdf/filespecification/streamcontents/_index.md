## FileSpecification.StreamContents プロパティ

ファイルの内容をストリームとして取得します。内容はメモリに読み込まれず、メモリ使用量を減らすことができます。しかし、このストリームは位置指定と Length プロパティをサポートしていません。これらの機能が必要な場合は、代わりに Contents プロパティを使用してください。

```csharp
public Stream StreamContents { get; }
```
