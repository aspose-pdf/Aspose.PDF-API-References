## Rectangle.Contains メソッド

指定された点が長方形の内部にあるかどうかを判断します。

```csharp
public bool Contains(Point point, bool inclusive = false)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| point | Point | チェックする点。 |
| inclusive | Boolean | inclusive: 点はエッジ上にある可能性があります。 |

### 戻り値

点が長方形の内部にある場合は True。
