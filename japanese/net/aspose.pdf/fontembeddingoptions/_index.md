## FontEmbeddingOptions クラス

PDF/A 標準では、すべてのフォントを文書に埋め込む必要があります。このクラスには、宛先 PC にフォントが存在しないために一部のフォントを埋め込むことができない場合のフラグが含まれています。

```csharp
public class FontEmbeddingOptions
```
