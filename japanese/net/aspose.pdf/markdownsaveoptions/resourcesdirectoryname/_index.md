## MarkdownSaveOptions.ResourcesDirectoryName プロパティ

画像などのドキュメントリソースを保存するためのディレクトリ名を取得および設定します。値が指定されていない場合、画像はマークダウンファイル自体と同じディレクトリに書き込まれます。これはパスではなく、名前だけです！このディレクトリは、保存されたマークダウンファイルのあるディレクトリに自動的に作成されます。

```csharp
public string ResourcesDirectoryName { get; set; }
```
