## MarkdownSaveOptions.HeadingLevels プロパティ

FontSize 認識ヘッダー戦略で使用する期待される見出しレベルを定義します。このプロパティの値が設定されている場合、ヘッダー認識 !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic 戦略が選択され、ドキュメントにブックマークが含まれていても !:PdfToMarkdown.HeadingRecognitionStrategy.Auto 戦略が設定されます。

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
