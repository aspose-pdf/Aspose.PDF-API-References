## SaveOptions.ResourceSavingInfo.SupposedFileName フィールド

コンバータによって設定されます。コンバータからカスタムメソッドのコードに渡される想定ファイル名。カスタムコードでファイルをどのように処理するか、またはどこに保存するかを決定するために使用できます。

```csharp
public string SupposedFileName;
```
