## SaveOptions.ResourceSavingInfo.ResourceType プロパティ

コンバータによって設定されます。コンバータからカスタムメソッドのコードに渡される想定ファイル名。カスタムコードでそのファイルをどのように処理するか、またはどこに保存するかを決定するために使用できます。

```csharp
public NodeLevelResourceType ResourceType { get; }
```
