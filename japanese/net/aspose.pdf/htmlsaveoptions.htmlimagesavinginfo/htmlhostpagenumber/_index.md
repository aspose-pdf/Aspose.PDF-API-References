## HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber フィールド

保存された画像が生成された HTML ページファイルのセットのどのページに関連しているかをカスタムコードに伝えます。ページの分割がオフになっている場合、この値は常に '1' を含みます。なぜなら、その場合は HTML ページが 1 つだけ生成されるからです。

```csharp
public int HtmlHostPageNumber;
```
