## HtmlSaveOptions.HtmlImageSavingInfo.ParentType フィールド

保存された画像は HTML 自体に関連するか、HTML に埋め込まれた SVG から抽出されることがあります。このプロパティは、処理された画像の親のタイプをカスタムコードに伝えることができます。これはコンバーターによって設定され、カスタムコードでその画像に対して何をすべきかを決定するために使用できます（例えば、カスタムコードは画像をどこに保存するか、または親のコンテンツでどのように参照されるべきかを決定できます）。

```csharp
public ImageParentTypes ParentType;
```
