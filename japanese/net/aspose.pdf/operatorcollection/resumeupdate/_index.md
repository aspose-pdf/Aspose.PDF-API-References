## ResumeUpdate(bool) {#resumeupdate_1}

ドキュメントの更新を再開します。保留中の変更がある場合、コンテンツストリームを更新します。invalidate パラメーターが true の場合、すべてのオペレーターを「変更済み」としてマークします。

```csharp
public void ResumeUpdate(bool updateAll)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| updateAll | ブール | true の場合、コレクション内のすべてのオペレーターが更新済みとしてマークされます。 |
