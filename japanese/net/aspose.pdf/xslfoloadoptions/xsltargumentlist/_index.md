## XslFoLoadOptions.XsltArgumentList プロパティ

XsltArgumentList は既存の xls パラメータに値を挿入するためのもので、XLS ファイルには値のない 'animal' パラメータがあります: XsltArgumentList args = new XsltArgumentList(); args.AddParam("animal", "", "cat"); これでコンバータは XLS ファイルに値 'cat' の 'animal' パラメータがあると仮定します。

```csharp
public XsltArgumentList XsltArgumentList { get; set; }
```
