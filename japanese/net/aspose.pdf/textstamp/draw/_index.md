## TextStamp.Draw プロパティ

このプロパティは、スタンプがページにどのように描画されるかを決定します。Draw = true の場合、スタンプはグラフィックオペレーターとして描画され、draw = false の場合、スタンプはテキストとして描画されます。

```csharp
public bool Draw { get; set; }
```
