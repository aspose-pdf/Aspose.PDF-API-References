## TextStamp.AutoAdjustFontSizeToFitStampRectangle プロパティ

これが有効な場合、フォントサイズはサイズ [`Width`](../width/) と [`Height`](../height/) のスタンプ矩形に合わせて自動的に調整されます。デフォルトの幅と高さはページ矩形から導出されます。

```csharp
public bool AutoAdjustFontSizeToFitStampRectangle { get; set; }
```
