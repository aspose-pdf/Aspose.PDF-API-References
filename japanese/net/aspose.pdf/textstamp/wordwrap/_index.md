## TextStamp.WordWrap プロパティ

ワードラップを定義します。このプロパティが true に設定され、幅の値が指定されている場合、テキストは指定された幅に収まるように複数行に分割されます。デフォルト値: false。

```csharp
public bool WordWrap { get; set; }
```
