## XImage.IsTheSameObject メソッド

両方の画像が同じオブジェクトを参照している場合は true を返します。

```csharp
public bool IsTheSameObject(XImage image)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| image | XImage | "this" 画像と比較される画像。 |

### 戻り値

画像が同じオブジェクトを参照している場合は true のブール値。
