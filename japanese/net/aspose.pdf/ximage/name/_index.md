## XImage.Name プロパティ

画像名を取得または設定します。ページコンテンツに参照がある画像の名前を変更すると、ドキュメントが不正になる可能性があることに注意してください。この場合は XImage.Rename メソッドを使用してください。

```csharp
public string Name { get; set; }
```
