## ExcelSaveOptions.InsertBlankColumnAtFirst プロパティ

ワークシートの最初の列として空白の列を挿入する必要がある場合は true に設定します。デフォルト値は false で、空白の列は挿入されません。

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
