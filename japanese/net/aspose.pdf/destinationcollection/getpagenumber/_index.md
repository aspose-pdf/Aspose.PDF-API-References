## DestinationCollection.GetPageNumber メソッド

名前によって目的地のページ番号を返します。

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| destinameName | String | 目的地の名前。 |
| useCache | Boolean | キャッシュされたコレクションのバージョンが使用されるかどうかを決定します。 |

### 戻り値
