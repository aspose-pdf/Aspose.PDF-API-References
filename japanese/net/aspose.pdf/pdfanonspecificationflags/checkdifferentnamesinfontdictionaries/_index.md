## PdfANonSpecificationFlags.CheckDifferentNamesInFontDictionaries プロパティ

一部の PDF ドキュメントには、内部データで異なる名前を持つフォントが含まれています。このフラグを使用すると、BaseFont と FontDescriptor.FontName フィールドが異なる場合の特別な処理ロジックが強制されます。

```csharp
public bool CheckDifferentNamesInFontDictionaries { get; set; }
```
