## HtmlSaveOptions.CssSavingInfo.CssNumber フィールド

コンバータによって設定されます。変換中に複数の CSS ファイルが作成されます。このプロパティは、変換中に保存された CSS ファイルの順序を示します。これは、CSS コンテンツを処理する方法や保存先を決定するためのカスタムコードのロジックで使用できます。

```csharp
public int CssNumber;
```
