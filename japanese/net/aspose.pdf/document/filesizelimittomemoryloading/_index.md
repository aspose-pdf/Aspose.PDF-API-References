## Document.FileSizeLimitToMemoryLoading プロパティ

ファイル全体をメモリに読み込むためのファイルサイズ制限を取得および設定します。値はメガバイト単位で設定されます。デフォルト値は210 Mbです。

```csharp
public static int FileSizeLimitToMemoryLoading { get; set; }
```
