
| パラメーター | タイプ | 説明 |

| ownerPassword | 文字列 | オーナーパスワード。 |
| newUserPassword | 文字列 | 新しいユーザーパスワード。 |
| newOwnerPassword | 文字列 | 新しいオーナーパスワード。 |
