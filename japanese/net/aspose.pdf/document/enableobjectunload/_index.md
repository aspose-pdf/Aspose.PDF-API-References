## Document.EnableObjectUnload プロパティ

ドキュメントがメモリから部分的にアンロードされることを可能にするフラグを取得または設定します。これによりメモリ使用量を減少させることができますが、パフォーマンスに悪影響を及ぼす可能性があります。

```csharp
public bool EnableObjectUnload { get; set; }
```
