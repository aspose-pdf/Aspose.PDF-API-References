## Document.CryptoAlgorithm プロパティ

ドキュメントが暗号化されている場合、セキュリティ設定を取得します。ドキュメントが暗号化されていない場合、.net 1.1 では対応する例外が発生し、他の .net バージョンでは CryptoAlgorithm が null になります。

```csharp
public CryptoAlgorithm? CryptoAlgorithm { get; }
```
