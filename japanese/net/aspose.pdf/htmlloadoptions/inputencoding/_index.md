## HtmlLoadOptions.InputEncoding プロパティ

解析時にこのドキュメントに使用されるエンコーディングを指定する属性を取得または設定します。この属性が null の場合、エンコーディングはドキュメントの文字セット属性から決定されます。

```csharp
public string InputEncoding { get; set; }
```
