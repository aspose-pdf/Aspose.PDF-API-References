## ColumnInfo.ColumnWidths プロパティ

列の幅を含む文字列を取得または設定します。各列の値は空白で区切る必要があります。デフォルトの単位はポイントですが、cm、インチ、および利用可能な幅のパーセンテージもサポートされています。例えば、"120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
