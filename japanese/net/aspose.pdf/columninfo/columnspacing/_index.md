## ColumnInfo.ColumnSpacing プロパティ

列間の間隔を含む文字列を取得または設定します。各間隔の値は空白で区切る必要があります。デフォルト単位はポイントですが、cm とインチもサポートされています。例えば、"120 2.5cm 1.5inch"。

```csharp
public string ColumnSpacing { get; set; }
```

## 備考

このプロパティが設定されていない場合、各間隔にはデフォルト値 0 が使用されます。
