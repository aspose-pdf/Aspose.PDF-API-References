## BatesNArtifact.NumberOfDigits プロパティ

Bates 番号付けの桁数を取得または設定します。値は 3 から 15 の間である必要があります。3 未満の値が設定された場合は、3 に調整されます。15 を超える値が設定された場合は、15 に調整されます。デフォルト値は 6 です。

```csharp
public int NumberOfDigits { get; set; }
```
