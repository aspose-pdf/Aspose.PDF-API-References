## DocSaveOptions.MaxDistanceBetweenTextLines プロパティ

このパラメータは、テキスト行を段落にグループ化するために使用されます。相対的なテキスト行がどれくらい離れていることができるかを決定します。テキスト行の高さの百分の一で指定されます。

```csharp
public float MaxDistanceBetweenTextLines { get; set; }
```
