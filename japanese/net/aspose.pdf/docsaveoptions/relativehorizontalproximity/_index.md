## DocSaveOptions.RelativeHorizontalProximity プロパティ

Pdf では、単語は独立して文字や音節を印刷するオペレーターで内部的に表現される場合があります。したがって、単語を検出するためには、実際には単語である独立した文字のグループを検出する必要があります。この設定は、ソース PDF 内の単語の認識中に単語間の距離として扱われるべきテキスト要素（文字、音節）間のスペースの幅を定義します。（この幅以上の空白が文字間に存在する場合、テキスト要素は異なる単語に属することを意味します）。これはフォントサイズに対して規格化されており、1.0 は想定される単語のフォントサイズの 100% を意味します。注意！これは、ソース PDF に最適な値をフォントから計算できない特定のまれに使用されるフォントが含まれている場合にのみ使用されます。したがって、ほとんどのケースでは、このパラメーターは結果のドキュメントに何も変更を加えません。

```csharp
public float RelativeHorizontalProximity { get; set; }
```
