## Matrix.Translate メソッド

指定された量だけx方向とy方向にマトリックスを移動します。

```csharp
public static Matrix Translate(double dx, double dy, Matrix source)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| dx | Double | x方向に移動する量。 |
| dy | Double | y方向に移動する量。 |
| source | Matrix | 移動するマトリックス。 |

### 戻り値

移動の結果として得られる新しいマトリックス。
