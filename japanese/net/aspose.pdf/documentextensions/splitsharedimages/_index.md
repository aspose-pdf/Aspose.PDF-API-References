## DocumentExtensions.SplitSharedImages メソッド

リソース内の画像について、2 ページが共通の XImages をチェックし、類似のケースの場合は重複する XImages を作成することで分割します。

```csharp
public static void SplitSharedImages(this Document doc, Page page_1, Page page_2)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| doc | Document | 両方のコレクションを含むドキュメント。 |
| page_1 | Page | 比較のための最初のページ。 |
| page_2 | Page | 比較のための2 番目のページ。 |
