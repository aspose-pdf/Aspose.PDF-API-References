## OutlineCollection.VisibleCount プロパティ

Count はすべてのレベルでの可視の子孫アウトライン項目の数の合計です。注意：コレクション内のアイテムの数である Count と混同しないでください。

```csharp
public override int VisibleCount { get; }
```
