## Stamp.Zoom プロパティ

スタンプのズーム係数。スタンプをスケーリングすることができます。このプロパティのペアである ZoomX と ZoomY は、各軸ごとにズーム係数を設定することを可能にします。このプロパティを設定すると、ZoomX と ZoomY の両方のプロパティが変更されます。ZoomX と ZoomY が異なる場合、Zoom プロパティは ZoomX の値を返します。

```csharp
public double Zoom { get; set; }
```
