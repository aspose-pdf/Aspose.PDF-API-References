## Stamp.Rotate プロパティ

[`Rotation`](../../rotation/) 値に従ってスタンプコンテンツの回転を設定または取得します。注意。このプロパティは、90度の倍数である角度（0、90、180、270度）を設定するためのものです。任意の角度を設定するには、RotateAngleプロパティを使用します。ArbitraryAngleで設定された角度が90の倍数でない場合、RotateプロパティはRotation.Noneを返します。

```csharp
public Rotation Rotate { get; set; }
```
