## CollectionItem.HasName メソッド

指定された名前がコレクションアイテムに存在するかどうかを確認します。

```csharp
public bool HasName(string name)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| name | String | 確認する名前。 |

### 戻り値

コレクションアイテムに名前が存在する場合は true、そうでない場合は false。
