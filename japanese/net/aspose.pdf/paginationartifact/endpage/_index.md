## PaginationArtifact.EndPage プロパティ

アーティファクトの終了ページ番号を取得または設定します。値は 0 以上でなければなりません。0 未満の値が設定された場合は、0 に調整されます。デフォルト値の 0 は、終了ページの境界がないことを意味します。

```csharp
public int EndPage { get; set; }
```
