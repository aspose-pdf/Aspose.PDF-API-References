## CollectionSchema.HasNameメソッド

指定された名前がスキーマに存在するかどうかを判断します。

```csharp
public bool HasName(string name)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| name | String | チェックする名前。 |

### 戻り値

指定された名前がスキーマに存在する場合は`true`、それ以外の場合は`false`。
