## Page.AddGraphics メソッド

ページにグラフィックスを追加します。[`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/) メソッドで要素を1つずつ追加するよりも速く動作します。

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| elements | GraphicElementCollection | グラフィックスコレクション。 |
| rectangle | Rectangle | 要素は、矩形領域内にある場合にページに追加されます。矩形が null の場合、すべてのグラフィック要素が追加されます。 |
