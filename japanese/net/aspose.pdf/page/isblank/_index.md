## Page.IsBlank メソッド

ページが空白かどうかのフラグを取得します。

```csharp
public bool IsBlank(double fillThresholdFactor)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| fillThresholdFactor | Double | 検出の感度を管理するフィルしきい値の値。0.01以上である必要があります。 |

### 戻り値

True - ページが空白の場合; それ以外の場合は、false。
