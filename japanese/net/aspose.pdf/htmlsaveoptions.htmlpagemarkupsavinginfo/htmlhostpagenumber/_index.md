## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber フィールド

コンバータによって設定されます。SplitToPages プロパティが設定されている場合、変換中に複数の HTML ファイル（変換されたページごとに 1 つの HTML ファイル）が作成されます。このプロパティには保存された HTML ページのファイルの順序が含まれています。このプロパティは、HTML ページを処理する方法や保存先を決定するためにカスタムコードのロジックで使用できます。また、ページの分割がオフになっている場合、この値は常に '1' を含みます。なぜなら、その場合、全体のソースドキュメントに対して 1 つの大きな HTML ページのみが生成されるからです。

```csharp
public int HtmlHostPageNumber;
```
