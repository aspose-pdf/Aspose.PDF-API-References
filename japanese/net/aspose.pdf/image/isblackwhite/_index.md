## Image.IsBlackWhite プロパティ

画像が強制的に白黒であるかどうかを示すbool値を取得または設定します。CCITTサブフォーマットのTIFF画像が使用される場合、このプロパティはtrueに設定する必要があります。

```csharp
public bool IsBlackWhite { get; set; }
```
