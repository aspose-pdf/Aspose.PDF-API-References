## CollectionFieldSubtype 列挙型

スキームコレクション内のフィールドのサブタイプパラメータを表します。

```csharp
public enum CollectionFieldSubtype
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | `0` | サブタイプは定義されていません。 |
| S | `1` | テキストタイプ。フィールドデータはPDFテキスト文字列として保存されます。 |
| D | `2` | 日付タイプ。フィールドデータはPDF日付文字列として保存されます。 |
| N | `3` | 数値タイプ。フィールドデータはPDF数値として保存されます。 |
| F | `4` | フィールドデータは、存在する場合はファイル仕様のUFエントリによって識別される埋め込みファイルストリームのファイル名であり、そうでない場合はファイル仕様のFエントリによって識別されます。 |
| Desc | `5` | フィールドデータは、存在する場合はファイル仕様のUFエントリによって識別される埋め込みファイルストリームのファイル名であり、そうでない場合はファイル仕様のFエントリによって識別されます。 |
| ModDate | `6` | フィールドデータは、埋め込みファイルパラメータ辞書のModDateエントリによって識別される埋め込みファイルストリームの修正日です。 |
| CreationDate | `7` | フィールドデータは、埋め込みファイルのCreationDateエントリによって識別される埋め込みファイルストリームの作成日です。 |
| Size | `8` | フィールドデータは、埋め込みファイルパラメータ辞書のSizeエントリによって識別される埋め込みファイルのサイズです。 |
| CompressedSize | `9` | (PDF 2.0) フィールドデータは、埋め込みファイルストリーム辞書のLengthエントリによって識別される埋め込みファイルストリームの長さであり、2つの値は同一でなければなりません。 |
