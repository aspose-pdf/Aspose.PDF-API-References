## OutputIntent クラス

PDF ドキュメントの色特性を、ドキュメントが印刷されるターゲット出力デバイスまたは生産環境のそれと一致させる出力インテントを表します。

```csharp
public sealed class OutputIntent
```

## コンストラクター

| 名前 | 説明 |
| --- | --- |
| [OutputIntent](outputintent/)(string) | 指定された出力条件識別子で `OutputIntent` クラスの新しいインスタンスを初期化します。 |
