## TeXLoadOptions.RequiredInputDirectory プロパティ

TeX が必要とする入力ディレクトリを取得/設定します。必要な入力は、メインの .tex ファイルに何らかの形で含まれるファイル、例えば、組み込みサポートがないパッケージです。

```csharp
public ITeXInputDirectory RequiredInputDirectory { get; set; }
```
