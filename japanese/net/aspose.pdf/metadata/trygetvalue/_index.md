## Metadata.TryGetValue メソッド

辞書内のキーを見つけ、見つかった場合は値を取得しようとします。

```csharp
public bool TryGetValue(string key, out XmpValue value)
```

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| key | String | 辞書内で検索するキー。 |
| value | XmpValue& | 取得した値。 |

### 戻り値

キーが見つかった場合は true。
