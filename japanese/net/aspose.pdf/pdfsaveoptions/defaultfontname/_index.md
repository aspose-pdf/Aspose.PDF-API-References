## PdfSaveOptions.DefaultFontName プロパティ

コンピュータに存在しないフォントのデフォルトで使用されるフォント名。PDF に保存された PDF ドキュメントに、ドキュメント自体およびデバイス上で利用できないフォントが含まれている場合、API はこのフォントをデフォルトフォントに置き換えます（デバイス上に `DefaultFontName` のフォントが見つかった場合）。

```csharp
public string DefaultFontName { get; set; }
```
