## HtmlSaveOptions.SaveTransparentTexts フィールド

Pdf には、クリップボードに選択できる透明なテキストが含まれることがあります（通常、ドキュメントに画像とそこから抽出された OCR テキストが含まれている場合に発生します）。この設定は、変換ツールに対して、結果の HTML に透明な選択可能なテキストとしてそのようなテキストを保存する必要があるかどうかを指示します。

```csharp
public bool SaveTransparentTexts;
```
