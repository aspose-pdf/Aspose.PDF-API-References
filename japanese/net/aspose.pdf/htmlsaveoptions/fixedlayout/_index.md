## HtmlSaveOptions.FixedLayout プロパティ

HTML が固定レイアウトとして作成されるかどうかを示す値を取得または設定します。

```csharp
public bool FixedLayout { get; set; }
```

### プロパティ値

`true` は [固定レイアウト] の場合。そうでない場合は `false`。
