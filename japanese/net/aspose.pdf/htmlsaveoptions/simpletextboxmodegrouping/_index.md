## HtmlSaveOptions.SimpleTextboxModeGrouping プロパティ

この属性は、グリフと単語を文字列に順次グループ化することを指定します。たとえば、タグと単語は変換された HTML で異なる順序を持ち、それらを一致させたい場合です。このパラメータは、FixedLayout 属性の値が true の場合にのみドキュメントに適用されます。

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
