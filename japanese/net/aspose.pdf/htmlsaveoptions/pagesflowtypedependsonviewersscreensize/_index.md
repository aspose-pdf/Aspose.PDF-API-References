## HtmlSaveOptions.PagesFlowTypeDependsOnViewersScreenSize フィールド

属性 'SplitOnPages=false' の場合、すべての入力 PDF ページを表す全体の HTML が 1 つの大きな結果 HTML ファイルにまとめられます。このフラグは、結果 HTML が生成される方法を定義し、結果 HTML 内の PDF ページを表す領域のフローがビューアの画面解像度に依存するかどうかを決定します。ビューア側の画面の幅が十分に大きく、2 ページ以上を横方向に並べて表示できると仮定します。このフラグが true に設定されている場合、この機会が利用されます（できるだけ多くのページが横方向に並べて表示され、次の横方向のページグループが最初のものの下に表示されます）。そうでない場合、ページは次のページが常に前のページの下に流れるように流れます。

```csharp
public bool PagesFlowTypeDependsOnViewersScreenSize;
```
