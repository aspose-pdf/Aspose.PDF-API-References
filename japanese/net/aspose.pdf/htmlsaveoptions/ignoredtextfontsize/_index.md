## HtmlSaveOptions.IgnoredTextFontSize プロパティ

指定されたサイズ以下のテキストは、変換中に無視されます。このテキストは削除されるのではなく、無視され、出力ファイルに転送されません。

```csharp
public float? IgnoredTextFontSize { get; set; }
```
