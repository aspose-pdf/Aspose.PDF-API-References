## HtmlSaveOptions.MinimalLineWidth プロパティ

この属性はグラフィックパスの線の最小幅を設定します。線の太さが 1px 未満の場合、Adobe Acrobat はこの値に丸めます。したがって、この属性は HTML ブラウザでこの動作をエミュレートするために使用できます。

```csharp
public float MinimalLineWidth { get; set; }
```
