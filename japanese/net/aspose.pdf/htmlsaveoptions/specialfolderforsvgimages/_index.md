## HtmlSaveOptions.SpecialFolderForSvgImages フィールド

HTML としてドキュメントを保存する際に遭遇した場合、SVG 画像のみを保存するディレクトリへのパスを取得または設定します。パラメータが空または null の場合、SVG ファイル（ある場合）は、出力ファイルの近くにある他の画像ファイルと一緒に保存されるか、SpecialImagesFolderIfAny オプションで指定されている場合は画像用の特別なフォルダーに保存されます。関連する画像ファイルを処理するために CustomImageSavingStrategy プロパティが正常に使用された場合、何も影響しません。

```csharp
public string SpecialFolderForSvgImages;
```
