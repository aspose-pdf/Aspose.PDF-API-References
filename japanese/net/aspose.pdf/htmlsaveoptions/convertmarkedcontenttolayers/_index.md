## HtmlSaveOptions.ConvertMarkedContentToLayers プロパティ

属性 ConvertMarkedContentToLayers が true に設定されている場合、PDF マークされたコンテンツ（レイヤー）内のすべての要素は、"data-pdflayer" 属性を持つ HTML div に配置され、レイヤー名が指定されます。このレイヤー名は、PDF マークされたコンテンツのオプションプロパティから抽出されます。この属性が false（デフォルト）である場合、PDF マークされたコンテンツからはレイヤーは作成されません。

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
