## HtmlSaveOptions.DefaultFontName プロパティ

埋め込まれておらず、システムにインストールされていないドキュメントフォントを置き換えるために使用されるインストール済みフォントの名前を指定します。null の場合は、デフォルトの置き換えフォントが使用されます。

```csharp
public string DefaultFontName { get; set; }
```

### プロパティ値

フォント名
