## CollectionField クラス

ドキュメントコレクションスキーマフィールドクラスを表します。

```csharp
public class CollectionField
```

## プロパティ

| 名前 | 説明 |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | インタラクティブPDFプロセッサがフィールド値の編集をサポートするかどうかを示すフラグを取得します。デフォルト値: false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | スキーマコレクション内のフィールド値のタイプを取得します。このフィールドは、[`Subtype`](./subtype/)に対応する値のタイプを説明します。 |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | インタラクティブPDFプロセッサによってユーザーに提示されるテキストフィールド名を取得します |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | ユーザーインターフェースにおけるフィールド名の相対的な順序を取得します。フィールドはインタラクティブPDFプロセッサによって昇順にソートされます。 |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | スキーマコレクション内のフィールド値のサブタイプを取得します。この辞書が説明するコレクションフィールドまたはファイル関連フィールドのサブタイプです。このエントリは、フィールドに保存されるデータのタイプを特定します。 |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | ユーザーインターフェースにおけるフィールドの初期の可視性を取得します。デフォルト値: true。 |
