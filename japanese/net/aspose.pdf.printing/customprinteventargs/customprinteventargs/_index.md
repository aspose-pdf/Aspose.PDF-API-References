## CustomPrintEventArgs コンストラクタ

[`CustomPrintEventArgs`](../) を指定されたプリンターとページ設定で初期化します。

```csharp
public CustomPrintEventArgs(string fileName, PrinterSettings printerSettings, 
    PageSettings pageSettings)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| fileName | String | 印刷されるファイルの名前。 |
| printerSettings | PrinterSettings | ドキュメントが印刷されるプリンターに関する情報。 |
| pageSettings | PageSettings | ドキュメントの各ページに適用される設定。 |
