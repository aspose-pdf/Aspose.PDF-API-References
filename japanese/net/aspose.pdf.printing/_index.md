**Aspose.Pdf.Printing** 名前空間は、プリンタ設定、ページ設定、および印刷関連の構成を管理するための拡張を含む PDF 印刷操作を処理するためのクラスと機能を提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/) イベントのデータを提供します。 |
| [PageSettings](./pagesettings/) | 単一の印刷ページに適用される設定を指定します。 |
| [PageSettingsExtensions](./pagesettingsextensions/) | [`PageSettings`](../aspose.pdf.printing/pagesettings/) の拡張メソッドを表します。 |
| [PaperSize](./papersize/) | 用紙のサイズを指定します。 |
| [PaperSizeExtensions](./papersizeextensions/) | [`PaperSize`](../aspose.pdf.printing/papersize/) の拡張メソッドを表します。 |
| [PaperSizes](./papersizes/) | 標準の用紙サイズを表します。 |
| [PaperSource](./papersource/) | プリンタが用紙を取得するトレイを指定します。 |
| [PaperSourceExtensions](./papersourceextensions/) | [`PaperSource`](../aspose.pdf.printing/papersource/) の拡張メソッドを表します。 |
| [PaperSources](./papersources/) | 一般的な用紙ソースを表す事前定義された [`PaperSource`](../aspose.pdf.printing/papersource/) インスタンスのセットを提供します。 |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/) イベントのデータを提供します。 |
| [PrinterResolution](./printerresolution/) | プリンタがサポートする解像度を表します。 |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | [`PrinterResolution`](../aspose.pdf.printing/printerresolution/) の拡張メソッドを表します。 |
| [PrinterSettings](./printersettings/) | ドキュメントがどのように印刷されるかに関する情報を指定し、印刷するプリンタを含みます。 |
| [PrinterSettingsExtensions](./printersettingsextensions/) | [`PrinterSettings`](../aspose.pdf.printing/printersettings/) の拡張メソッドを表します。 |
| [StartEndPageEventArgs](./startendpageeventargs/) | [`PdfViewer`](../aspose.pdf.facades/pdfviewer/) クラスの [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) および [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) イベントのデータを提供します。 |
## 列挙型

| 列挙型 | 説明 |
| --- | --- |
| [Duplex](./duplex/) | プリンタの両面印刷設定を指定します。 |
| [PaperKind](./paperkind/) | 標準の用紙サイズを指定します。 |
| [PaperSourceKind](./papersourcekind/) | 標準の用紙ソース。 |
| [PrinterResolutionKind](./printerresolutionkind/) | プリンタ解像度を指定します。 |
| [PrintRange](./printrange/) | 印刷するドキュメントの部分を指定するオプションを指定します。 |
