**Aspose.Pdf.DataEditor** にはドキュメント内のデータを編集するためのツールが含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [CosPdfBoolean](./cospdfboolean/) | このクラスはブール型を表します。 |
| [CosPdfDictionary](./cospdfdictionary/) | オブジェクトの辞書にアクセスするためのクラスです。 |
| [CosPdfName](./cospdfname/) | このクラスは Pdf Name オブジェクトを表します。 |
| [CosPdfNumber](./cospdfnumber/) | このクラスは Pdf Number 型を表します。 |
| [CosPdfPrimitive](./cospdfprimitive/) | このクラスは基本の公開型 [`CosPdfPrimitive`](../aspose.pdf.dataeditor/cospdfprimitive/) を表します。 |
| [CosPdfString](./cospdfstring/) | このクラスは Pdf String オブジェクトを表します。 |
| [DictionaryEditor](./dictionaryeditor/) | ドキュメントのツリー辞書（ドキュメント辞書、ページ辞書、リソース辞書）にアクセスするためのクラスです。 |
