## DictionaryEditor.ContainsKey メソッド

```csharp
public bool ContainsKey(string key)
```

| パラメーター | 型 | 説明 |
| --- | --- | --- |
| key | String | [`DictionaryEditor`](../) で検索するキー。 |

### 戻り値

[`DictionaryEditor`](../) にキーを持つ編集可能な要素が含まれている場合は true; それ以外の場合は false。
