## 名前空間

| 名前空間 | 説明 |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | **Aspose.Pdf**は、**Document**のように直接含まれているか、いくつかのサブ名前空間を通じて間接的に含まれているAspose.Pdfライブラリのすべてのクラスのルート名前空間です。 |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | **Aspose.Pdf.AI**名前空間は、APIクライアントやインテリジェントアシスタントを含むAI機能のためのクラスを提供します。 |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | **Aspose.Pdf.Annotations**名前空間は、さまざまな種類のアクション、宛先、および従来インタラクティブと呼ばれるドキュメントの他の機能を操作するためのクラスを提供します。 |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | **Aspose.Pdf.Comparison**名前空間は、ドキュメントの比較に使用されるクラスを提供します。 |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | **Aspose.Pdf.DataEditor**には、ドキュメント内のデータを編集するためのツールが含まれています。 |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | **Aspose.Pdf.Devices**名前空間は、ドキュメントを画像またはプレーンテキストとして表現するためのクラスを提供します。これにより、ドキュメントをテキストまたはさまざまなグラフィックデバイスに送信でき、テキストまたはグラフィック表現を取得できます。 |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | **Aspose.Pdf.Drawing**名前空間は、円弧、線、円などのグラフィックプリミティブを説明するクラスを提供します。対応するオブジェクトは、新しいドキュメントを作成するために使用できます。 |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | **Aspose.Pdf.Facades**名前空間は、元々Aspose.Pdf.Kitから来たクラスを提供します。これらのクラスは、ドキュメントを操作し、結合、スタンプ、署名、注釈などの操作を高レベルで実行するために使用されますが、ドキュメントの内部構造にはアクセスしません。 |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | **Aspose.Pdf.Forms**名前空間には、フォーム（標準、静的、動的）やテキストボックス、リストボックス、ラジオボタンなどのさまざまなタイプのフィールドを説明するクラスがあります。 |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | **Aspose.Pdf.Tagged.LogicalStructure**は、PDF論理構造クラスのための名前空間です。 |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | **Aspose.Pdf.Multithreading**名前空間は、マルチスレッドシナリオで使用されることを意図したクラスを提供します。 |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | **Aspose.Pdf.Operators**は、オペレーターの実装のための名前空間です。これらのクラスは、PDFページコンテンツで使用されるオペレーターを説明します。 |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | **Aspose.Pdf.Optimization**は、ドキュメント最適化プロセスの管理のためのクラスの名前空間です。 |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | **Aspose.Pdf.Plugins**は、**PdfOrganizer**のように直接含まれているか、いくつかのサブ名前空間を通じて間接的に含まれているAspose.Pdf.Pluginsクラスのすべてのクラスのルート名前空間です。 |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | **Aspose.Pdf.Printing**名前空間は、プリンター設定、ページ設定、および印刷関連の構成を管理するための拡張を含むPDF印刷操作を処理するためのクラスと機能を提供します。 |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | **Aspose.Pdf.Sanitization**は、サニタイズ操作のための名前空間です。 |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | **Aspose.Pdf.Security**名前空間には、暗号化およびデジタル署名に使用されるクラスが含まれています。 |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | **Aspose.Pdf.Structure**名前空間は、ドキュメントの論理構造を操作するのに役立つクラスを提供します。これは、ドキュメントの章やセクションへの組織や、図、表、脚注などの特別な要素の識別を示します。 |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | **Aspose.Pdf.Tagged**は、タグ付きPDFドキュメントのサポートのためのクラスの名前空間です。 |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | **Aspose.Pdf.Text**名前空間は、テキストを抽出、追加、既存のテキストを操作するためのクラスを提供します。また、ドキュメントのフォントを抽出、置換、代替するためのクラスも含まれています。 |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | **Aspose.Pdf.Vector**は、グラフィック操作のためのルート名前空間です。 |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | **Aspose.Pdf.XfaConverter**名前空間は、XFAを変換するためのクラスを提供します。 |
