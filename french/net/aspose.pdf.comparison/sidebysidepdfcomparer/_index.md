## Méthodes

| Nom | Description |
| --- | --- |
| static [Compare](../../aspose.pdf.comparison/sidebysidepdfcomparer/compare/#compare)(Document, Document, string, SideBySideComparisonOptions) | Compare deux documents. Les pages sont comparées une par une. Les pages des documents comparés sont copiées l'une après l'autre dans le document résultant. D'abord la première page du premier document, puis la première page du deuxième document. Ensuite, la deuxième du premier document et puis la deuxième du deuxième document, etc. Vous pouvez l'ouvrir dans Adobe Acrobat en mode Deux pages pour voir les changements côte à côte. Les suppressions sont notées sur la page de gauche, et les insertions sont notées sur la page de droite. |
| static [Compare](../../aspose.pdf.comparison/sidebysidepdfcomparer/compare/#compare_1)(Page, Page, string, SideBySideComparisonOptions) | Compare deux pages. Le résultat est enregistré dans un document PDF dans lequel la première page est écrite en premier, puis la seconde. Vous pouvez l'ouvrir dans Adobe Acrobat en mode Deux pages pour voir les changements côte à côte. Les suppressions sont notées sur la page de gauche, et les insertions sont notées sur la page de droite. |
