## Propriété ComparisonOptions.ExcludeAreas1

Obtenez et définissez les zones à exclure. Utilisé pour la première page ou le document dans la méthode de comparaison. Cette option peut être définie avec [`ExcludeTables`](../excludetables/). Cette option ne peut pas être définie avec l'option [`ExtractionArea`](../extractionarea/).
