## Propriété ComparisonOptions.ExcludeTables

Obtenez et définissez l'option qui détermine si les tables sont exclues de la comparaison. Cette option ne peut pas être définie avec l'option [`ExtractionArea`](../extractionarea/). La valeur par défaut est `false`.

```csharp
public bool ExcludeTables { get; set; }
```
