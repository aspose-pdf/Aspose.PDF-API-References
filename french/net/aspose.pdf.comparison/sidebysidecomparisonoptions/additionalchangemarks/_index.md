## Propriété SideBySideComparisonOptions.AdditionalChangeMarks

Obtenez et définissez la propriété qui détermine si des marqueurs de changement supplémentaires sont affichés. Si défini, affiche les marques de changement qui ne sont pas sur la page actuelle mais qui sont présentes sur une autre page. Si le changement se situe entre des mots, la marque peut ne pas être positionnée exactement par rapport au caractère d'espace blanc. La valeur par défaut est `false`.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
