## Propriété SideBySideComparisonOptions.ExcludeTables

Obtenez et définissez l'option qui détermine si les tables sont exclues de la comparaison. Cette option ne peut pas être définie avec [`ComparisonArea1`](../comparisonarea1/) et [`ComparisonArea2`](../comparisonarea2/). La valeur par défaut est `false`.

```csharp
public bool ExcludeTables { get; set; }
```
