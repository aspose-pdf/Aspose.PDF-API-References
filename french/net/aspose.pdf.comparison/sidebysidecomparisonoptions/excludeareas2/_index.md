## Propriété SideBySideComparisonOptions.ExcludeAreas2

Obtenez et définissez les zones à exclure. Utilisé pour la deuxième page ou le document dans la méthode de comparaison. Cette option peut être définie avec [`ExcludeTables`](../excludetables/). Cette option ne peut pas être définie avec l'option [`ComparisonArea2`](../comparisonarea2/).

public Rectangle[] ExcludeAreas2 { get; set; }
