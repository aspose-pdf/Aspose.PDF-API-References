## Méthode AbsorbedCell.CompareTo

Compare l'objet AbsorbedCell actuel avec un autre objet AbsorbedCell et renvoie un entier qui indique si l'objet actuel précède, suit ou se trouve à la même position dans l'ordre de tri que l'autre objet.

```csharp
public int CompareTo(AbsorbedCell other)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| other | AbsorbedCell | Un objet AbsorbedCell à comparer avec cet objet. |

### Valeur de retour

Une valeur qui indique l'ordre relatif des objets comparés. La valeur de retour a les significations suivantes : -1 : Cet objet précède l'autre objet dans l'ordre de tri. 0 : Cet objet se trouve à la même position dans l'ordre de tri que l'autre objet. 1 : Cet objet suit l'autre objet dans l'ordre de tri.
