## Propriété TextState.BackgroundColor

Définit la couleur de fond du texte.

```csharp
public virtual Color BackgroundColor { get; set; }
```

## Remarques

Notez que la valeur n'est pas préservée en tant que caractéristique de texte dans le document. Le getter de la propriété BackgroundColor fonctionne pour un objet dans le cas où il a été explicitement défini auparavant avec le setter BackgroundColor pour cet objet. La propriété est utilisée par le runtime dans le contexte du processus de génération/modification actuel.
