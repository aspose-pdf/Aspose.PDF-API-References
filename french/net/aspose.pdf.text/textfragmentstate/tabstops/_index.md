## Propriété TextFragmentState.TabStops

Obtient les tabulations pour le texte.

```csharp
public TabStops TabStops { get; }
```

## Remarques

Notez que la propriété Tabstops fonctionne uniquement dans les scénarios de génération de nouveaux documents. Les tabulations peuvent être ajoutées lors de l'initialisation de [`TextFragment`](../../textfragment/). Les tabulations doivent être construites avant le texte.
