## TextReplaceOptions.ReplaceAdjustment enumeration

Détermine l'action qui sera effectuée après le remplacement du fragment de texte en plus court. Aucune - aucune action, le texte remplacé peut chevaucher le reste de la ligne ; AdjustSpaceWidth - essaie d'ajuster les espaces entre les mots pour conserver la longueur de la ligne ; WholeWordsHyphenation - essaie de distribuer les mots entre les lignes de paragraphe pour conserver le champ droit du paragraphe ; ShiftRestOfLine - décale le reste de la ligne en fonction de la longueur du texte, la longueur de la ligne peut être modifiée ; La valeur par défaut est ShiftRestOfLine.

```csharp
[Flags]
public enum ReplaceAdjustment
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Aucune action, le texte remplacé peut chevaucher le reste de la ligne |
| AdjustSpaceWidth | `1` | Essaie d'ajuster les espaces entre les mots pour conserver la longueur de ligne |
| WholeWordsHyphenation | `2` | Essaie de répartir les mots entre les lignes de paragraphe pour conserver le champ droit du paragraphe |
| ShiftRestOfLine | `4` | (par défaut) décale le reste de la ligne en fonction de la longueur du texte, la longueur de la ligne peut être modifiée |
