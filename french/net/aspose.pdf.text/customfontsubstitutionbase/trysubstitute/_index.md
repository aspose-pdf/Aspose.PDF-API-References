### Remarques

La classe CustomFontSubstitutionBase doit être héritée pour implémenter la logique de substitution de police personnalisée. La méthode TrySubstitute doit être remplacée correctement : Doit renvoyer true si une substitution est requise. substitutionFont doit être défini sur un objet Font valide. Doit renvoyer false si aucune substitution n'est requise. substitutionFont peut être défini sur null.
