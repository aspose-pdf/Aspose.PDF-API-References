## TextAbsorber.ExtractionOptions property

Obtient ou définit les options d'extraction de texte.

```csharp
public virtual TextExtractionOptions ExtractionOptions { get; set; }
```

### Remarques

Permet de définir le mode de formatage du texte[`TextExtractionOptions`](../../textextractionoptions) lors de l'extraction. Le mode par défaut estPure

### Exemples

L'exemple montre comment définir le mode de formatage de texte pur et effectuer l'extraction de texte.

```csharp
// ouvre le document
Document doc = new Document(inFile);

// crée un objet TextAbsorber pour extraire du texte avec mise en forme
TextAbsorber absorber = new TextAbsorber();

// définit le mode de formatage du texte pur
absorber.ExtractionOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure);

// accepte l'absorbeur pour toutes les pages du document
doc.Pages.Accept(absorber);

// récupère le texte extrait
string extractedText = absorber.Text;
```
