## Propriété TextAbsorber.TextSearchOptions

Obtient ou définit les options de recherche de texte.

```csharp
public virtual TextSearchOptions TextSearchOptions { get; set; }
```

## Remarques

Permet de définir un rectangle qui délimite le texte extrait. Par défaut, le rectangle est vide. Cela signifie que les limites de la page définissent uniquement la région d'extraction du texte.
