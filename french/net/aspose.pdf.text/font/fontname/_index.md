### Exemples

L'exemple montre comment rechercher du texte sur la première page et afficher le nom de la police d'une première occurrence de texte.

```csharp
// Ouvrir le document
Document doc = new Document(@"D:\Tests\input.pdf");

// Crée un objet TextFragmentAbsorber pour trouver toutes les occurrences de texte "hello world"
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");

// Accepte l'absorbeur pour la première page

// Afficher le nom de la police de la première occurrence de texte
```
