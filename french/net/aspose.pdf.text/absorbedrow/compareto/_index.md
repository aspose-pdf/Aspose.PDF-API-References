## Méthode AbsorbedRow.CompareTo

Compare l'objet AbsorbedRow actuel avec un autre objet AbsorbedRow et renvoie un entier qui indique si l'objet actuel précède, suit ou se trouve à la même position dans l'ordre de tri que l'autre objet.

```csharp
public int CompareTo(AbsorbedRow other)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| other | AbsorbedRow | Un objet AbsorbedRow à comparer avec cet objet. |

### Valeur de retour

Une valeur qui indique l'ordre relatif des objets comparés. La valeur de retour a les significations suivantes : -1 : Cet objet précède l'autre objet dans l'ordre de tri. 0 : Cet objet se trouve à la même position dans l'ordre de tri que l'autre objet. 1 : Cet objet suit l'autre objet dans l'ordre de tri.
