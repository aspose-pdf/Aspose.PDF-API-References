## Énumération TextEditOptions.LanguageTransformation

Modes de transformation de langue

```csharp
public enum LanguageTransformation
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | `0` | La transformation de langue par défaut est effectuée. |
| ExactlyAsISee | `1` | La transformation de langue est effectuée de la même manière que dans un éditeur de texte. Cela signifie généralement que le texte apparaîtra dans le document exactement comme vous le voyez dans le code. Mais aucune garantie. |
| None | `2` | Aucune transformation de langue n'est effectuée. |
