## Énumération TextFormattingOptions.WordWrapMode

Définit les stratégies de retour à la ligne

```csharp
public enum WordWrapMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| NoWrap | `0` | Aucun retour à la ligne n'est effectué |
| DiscretionaryHyphenation | `1` | Une césure discrétionnaire est effectuée. Permet de couper les mots au milieu. |
| ByWords | `2` | Le retour à la ligne ne coupe que des mots complets. Si le mot complet ne peut pas être coupé, tente d'utiliser la césure discrétionnaire |
| Undefined | `3` | WordWrapMode non défini. Les stratégies de retour à la ligne de la structure de niveau supérieur (cellule de tableau, paragraphe, etc.) seront utilisées. |
