## TextSearchOptions.SearchForTextRelatedGraphics property

Obtient ou définit la valeur qui permet de rechercher des graphiques liés au texte (souligné, arrière-plan, etc.) lors de la recherche de texte. true - la recherche de graphiques liés au texte sera effectuée (valeur par défaut). false - éléments graphiques pouvant être présents dans le document source sera ignoré. Définissez ceci en cas de problèmes de performances ou si vous n'avez pas besoin de gérer le soulignement, l'arrière-plan ou l'écrêtage.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
