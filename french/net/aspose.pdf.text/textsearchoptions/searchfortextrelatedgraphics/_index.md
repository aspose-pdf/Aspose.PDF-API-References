## Propriété TextSearchOptions.SearchForTextRelatedGraphics

Obtient ou définit une valeur qui permet de rechercher des graphiques liés au texte (soulignement, arrière-plan, etc.) lors de la recherche de texte. true - la recherche de graphiques liés au texte sera effectuée (valeur par défaut). false - les éléments graphiques qui peuvent être présents dans le document source seront ignorés. Définissez cela en cas de problèmes de performance ou s'il n'est pas nécessaire de gérer le soulignement, l'arrière-plan ou le recadrage.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
