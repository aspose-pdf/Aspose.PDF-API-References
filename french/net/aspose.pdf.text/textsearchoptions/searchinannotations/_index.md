## Propriété TextSearchOptions.SearchInAnnotations

Obtient ou définit la valeur qui permet de rechercher du texte dans les annotations. true - le texte sera recherché dans les annotations. false - le texte dans les annotations ne sera pas analysé par TextFragmentAbsorber.

public bool SearchInAnnotations { get; set; }
