## Propriété TextSearchOptions.IgnoreShadowText

Obtient ou définit l'indication que les fragments de texte représentant l'ombre du texte normal seront ignorés lors de la recherche. true - signifie que le texte d'ombre ne sera pas trouvé (essayez cela si la recherche de texte renvoie des fragments dupliqués sur des positions proches) false - signifie que le texte d'ombre sera trouvé ainsi que le texte normal (valeur par défaut)

```csharp
public bool IgnoreShadowText { get; set; }
```
