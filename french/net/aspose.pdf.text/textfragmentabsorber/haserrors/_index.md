## Propriété TextFragmentAbsorber.HasErrors

La valeur indique si des erreurs ont été trouvées lors de l'extraction de texte. La recherche d'erreurs ne sera effectuée que si TextSearchOptions.LogTextExtractionErrors = true ; et cela peut diminuer les performances.

public bool HasErrors { get; }
