## Propriété TextFragmentAbsorber.Errors

Liste des [`TextExtractionError`](../../textextractionerror/) objets. Elle contient des informations sur les erreurs trouvées lors de l'extraction de texte. La recherche d'erreurs ne sera effectuée que si TextSearchOptions.LogTextExtractionErrors = true ; et cela peut diminuer les performances.

public List<TextExtractionError> Errors { get; }
