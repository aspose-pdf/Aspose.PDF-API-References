## Méthode TextFragment.IsolateTextSegments

Obtient des [`TextSegment`](../../textsegment/)(s) représentant une partie spécifiée du texte de [`TextFragment`](../).

```csharp
public TextSegmentCollection IsolateTextSegments(int startIndex, int length)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| startIndex | Int32 | Position dans le texte à partir de laquelle les nouveaux [`TextSegment`](../../textsegment/)(s) commenceront. |
| length | Int32 | Longueur du texte qui sera isolé dans des [`TextSegment`](../../textsegment/)(s). |

### Valeur de retour

[`TextSegmentCollection`](../../textsegmentcollection/) contenant des segments de texte représentant une sous-chaîne de texte commençant à une position spécifiée et ayant une longueur spécifiée.
