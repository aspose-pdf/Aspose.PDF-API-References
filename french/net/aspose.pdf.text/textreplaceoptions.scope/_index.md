## Énumération TextReplaceOptions.Scope

Portée où l'opération de remplacement de texte est appliquée REPLACE_FIRST par défaut Cette option obsolète a été conservée pour des raisons de compatibilité. Elle affecte PdfContentEditor et n'a aucun effet sur TextFragmentAbsorber.

```csharp
public enum Scope
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| REPLACE_FIRST | `0` | Remplacer uniquement la première occurrence du texte sur chacune des pages affectées |
| REPLACE_ALL | `1` | Remplacer toutes les occurrences de texte sur toutes les pages affectées |
