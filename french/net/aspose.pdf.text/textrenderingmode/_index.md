## TextRenderingMode enumeration

Le mode de rendu du texte, Tmode, détermine si l'affichage du texte doit entraîner le trait, le remplissage, l'utilisation des contours du glyphe comme limite de découpage ou une combinaison des trois.

```csharp
public enum TextRenderingMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| FillText | `0` | Remplir le texte. |
| StrokeText | `1` | Barrez le texte. |
| FillThenStrokeText | `2` | Remplir, puis barrer le texte. |
| Invisible | `3` | Texte ni remplissage ni contour (invisible). |
| FillTextAndAddPathToClipping | `4` | Remplissez le texte et ajoutez-le au chemin pour le découpage (voir 9.3.6, "Mode de rendu du texte"). |
| StrokeTextAndAddPathToClipping | `5` | Barrez le texte et ajoutez-le au chemin pour le découpage. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Remplissez, puis tracez le texte et ajoutez-le au chemin pour le découpage. |
| AddPathToClipping | `7` | Ajouter du texte au chemin pour le découpage. |
