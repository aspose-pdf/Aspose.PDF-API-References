## Méthode AbsorbedTable.CompareTo

Compare l'objet AbsorbedTable actuel avec un autre objet AbsorbedTable et renvoie un entier qui indique si l'objet actuel précède, suit ou se trouve à la même position dans l'ordre de tri que l'autre objet.

```csharp
public int CompareTo(AbsorbedTable other)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| other | AbsorbedTable | Un objet AbsorbedTable à comparer avec cet objet. |

### Valeur de retour

Une valeur qui indique l'ordre relatif des objets comparés. La valeur de retour a les significations suivantes : -1 : Cet objet précède l'autre objet dans l'ordre de tri. 0 : Cet objet se trouve à la même position dans l'ordre de tri que l'autre objet. 1 : Cet objet suit l'autre objet dans l'ordre de tri.
