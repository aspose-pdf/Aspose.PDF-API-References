## Propriété PdfAOptionsBase.OptimizeFileSize

Obtient ou définit une valeur indiquant s'il faut essayer de réduire la taille du fichier pendant le processus de conversion PDF/A.

```csharp
public bool OptimizeFileSize { get; set; }
```

### Valeur de la propriété

`true` si la taille du fichier doit être réduite autant que possible ; sinon, `false`.

## Remarques

Lorsqu'il est défini sur `true`, le processus de conversion tentera de minimiser la taille du fichier résultant. Cela pourrait affecter les performances du processus de conversion.
