## PdfToXlsOptions.MinimizeTheNumberOfWorksheets property

Définissez true si vous devez minimiser le nombre de feuilles de calcul dans le classeur résultant. La valeur par défaut est false ; cela signifie enregistrer chaque page PDF en tant que feuille de calcul séparée.

public bool MinimizeTheNumberOfWorksheets { get; set; }
