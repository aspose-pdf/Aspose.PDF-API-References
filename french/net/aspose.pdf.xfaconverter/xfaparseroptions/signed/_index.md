## XfaParserOptions.Signed property

Si cette propriété est vraie, le document sera converti à l'aide du flux de formulaire xfa (s'il existe). Si elle est fausse, le flux de formulaire xfa sera ignoré. Cette propriété a été introduite car il n'est pas clair comment calculer la somme de contrôle celle utilisée pour vérifier la signature.
