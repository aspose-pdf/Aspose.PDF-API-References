## Propriété AssistantCreateRequest.ToolResources

Obtient ou définit les ressources utilisées par les outils de l'assistant. Les ressources sont spécifiques au type d'outil. Par exemple, l'outil code_interpreter nécessite une liste d'ID de fichiers, tandis que l'outil file_search nécessite une liste d'ID de magasin de vecteurs.

```csharp
public ToolResources ToolResources { get; set; }
```
