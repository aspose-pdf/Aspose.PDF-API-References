## Propriété CreateChatCompletionChunkResponse.Usage

Obtient ou définit un champ optionnel qui ne sera présent que lorsque vous définissez stream_options: {"include_usage": true} dans votre demande. Lorsqu'il est présent, il contient une valeur nulle sauf pour le dernier morceau qui contient les statistiques d'utilisation des jetons pour l'ensemble de la demande.

```csharp
public Usage Usage { get; set; }
```
