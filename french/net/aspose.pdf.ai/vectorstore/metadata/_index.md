## Propriété VectorStore.Metadata

Obtient ou définit un ensemble de 16 paires clé-valeur qui peuvent être attachées à un magasin de vecteurs. Cela peut être utile pour stocker des informations supplémentaires sur le magasin de vecteurs dans un format structuré. Les clés peuvent avoir une longueur maximale de 64 caractères et les valeurs peuvent avoir une longueur maximale de 512 caractères.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
