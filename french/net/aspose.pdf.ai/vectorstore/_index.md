## Classe VectorStore

Un helper pour créer un magasin de vecteurs avec des file_ids et l'attacher à ce thread. Il peut y avoir un maximum de 1 magasin de vecteurs attaché au thread.

```csharp
public class VectorStore
```

## Propriétés

| Nom | Description |
| --- | --- |
| [FileIds](../../aspose.pdf.ai/vectorstore/fileids/) { get; set; } | Obtient ou définit une liste d'ID de fichiers à ajouter au magasin de vecteurs. Il peut y avoir un maximum de 10000 fichiers dans un magasin de vecteurs. |
| [Metadata](../../aspose.pdf.ai/vectorstore/metadata/) { get; set; } | Obtient ou définit un ensemble de 16 paires clé-valeur qui peuvent être attachées à un magasin de vecteurs. Cela peut être utile pour stocker des informations supplémentaires sur le magasin de vecteurs dans un format structuré. Les clés peuvent avoir un maximum de 64 caractères et les valeurs peuvent avoir un maximum de 512 caractères. |
