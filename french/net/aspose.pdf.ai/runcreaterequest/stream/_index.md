## Propriété RunCreateRequest.Stream

Obtient ou définit si l'utilisation du streaming est activée. Si vrai, renvoie un flux d'événements qui se produisent pendant l'exécution en tant qu'événements envoyés par le serveur, se terminant lorsque l'exécution entre dans un état terminal avec un message de données : [DONE].

```csharp
public bool? Stream { get; set; }
```
