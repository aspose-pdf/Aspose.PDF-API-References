## Propriété CompletionCreateRequest.Stream

Obtient ou définit si l'utilisation du streaming est activée. Si défini, des deltas de message partiels seront envoyés, comme dans ChatGPT. Les tokens seront envoyés en tant qu'événements envoyés par le serveur uniquement en tant que données dès qu'ils seront disponibles, le flux étant terminé par un message de données : [DONE].

public bool? Stream { get; set; }
