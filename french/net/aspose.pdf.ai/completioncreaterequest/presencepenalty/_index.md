## Propriété CompletionCreateRequest.PresencePenalty

Obtient ou définit un nombre entre -2.0 et 2.0. Les valeurs positives pénalisent les nouveaux tokens en fonction de leur apparition dans le texte jusqu'à présent, augmentant la probabilité que le modèle parle de nouveaux sujets.

public double? PresencePenalty { get; set; }
