## Méthode MessageContentRequest.CreateImageUrlContent

Crée un contenu d'URL d'image pour un message.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Paramètre | Type | Description |
| --- | --- | --- |
| imageUrl | String | L'URL de l'image. |
| detail | String | Le détail de l'URL de l'image. Spécifie le niveau de détail de l'image si spécifié par l'utilisateur. "low" utilise moins de jetons, vous pouvez opter pour une haute résolution en utilisant "high". Par défaut, c'est "auto". |

### Valeur de retour

Une nouvelle instance de MessageContentRequest avec un contenu ImageUrl.
