## Propriété AssistantResponse.TopP

Obtient ou définit une alternative à l'échantillonnage avec température, appelée échantillonnage par noyau, où le modèle considère les résultats des jetons avec une masse de probabilité top_p. Ainsi, 0.1 signifie que seuls les jetons composant les 10% supérieurs de la masse de probabilité sont considérés. Nous recommandons généralement de modifier cela ou la température mais pas les deux.

```csharp
public double? TopP { get; set; }
```
