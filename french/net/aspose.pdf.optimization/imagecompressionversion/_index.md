## Énumération ImageCompressionVersion

Décrit les versions de l'algorithme de compression d'image.

```csharp
public enum ImageCompressionVersion
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Standard | `0` | Algorithme standard. Valeur par défaut. |
| Rapide | `2` | Algorithme amélioré plus rapide que le standard mais applicable pas dans tous les cas. |
| Mixte | `3` | Utiliser l'algorithme rapide lorsque possible et le standard pour les autres cas. Peut être plus lent que "Rapide" mais peut produire une meilleure compression. |
