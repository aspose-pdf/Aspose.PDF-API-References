## Propriété ImageCompressionOptions.ResizeImages

Si ce drapeau est défini sur vrai et que CompressImages est vrai, les images seront redimensionnées si la résolution de l'image est supérieure au paramètre MaxResolution spécifié.

```csharp
public bool ResizeImages { get; set; }
```
