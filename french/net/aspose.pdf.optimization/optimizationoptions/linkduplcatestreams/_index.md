## Propriété OptimizationOptions.LinkDuplcateStreams

Si ce drapeau est défini sur vrai, les flux de ressources seront analysés. Si des flux dupliqués sont trouvés (c'est-à-dire si le contenu des flux est égal), alors ces flux seront stockés comme un seul objet. Cela permet de réduire la taille du document dans certains cas (par exemple, lorsque le même document a été concaténé plusieurs fois).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
