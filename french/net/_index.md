## Espaces de noms

| Espace de noms | Description |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | Le **Aspose.Pdf** est un espace de noms racine pour toutes les classes de la bibliothèque Aspose.Pdf qui sont soit directement dans celui-ci comme **Document** ou indirectement à travers plusieurs sous-espaces de noms. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | L'espace de noms **Aspose.Pdf.AI** fournit des classes pour les fonctionnalités d'IA, y compris les clients API et les assistants intelligents. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | L'espace de noms **Aspose.Pdf.Annotations** fournit des classes pour travailler avec divers types d'actions, de destinations et d'autres fonctionnalités de document traditionnellement appelées interactives, permettant à l'utilisateur d'interagir avec celui-ci. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | L'espace de noms **Aspose.Pdf.Comparison** fournit des classes utilisées pour la comparaison de documents. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | Le **Aspose.Pdf.DataEditor** contient des outils pour éditer des données au sein d'un document. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | L'espace de noms **Aspose.Pdf.Devices** fournit des classes utilisées pour représenter un document sous forme d'image(s) ou de texte brut. Ainsi, le document peut être envoyé à des dispositifs textuels ou graphiques variés, ce qui signifie que nous voulons obtenir sa représentation textuelle ou graphique. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | L'espace de noms **Aspose.Pdf.Drawing** fournit des classes décrivant des primitives graphiques comme l'arc, la ligne, le cercle, etc. Les objets correspondants peuvent être utilisés pour créer de nouveaux documents. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | L'espace de noms **Aspose.Pdf.Facades** fournit des classes qui proviennent à l'origine de Aspose.Pdf.Kit. Ces classes sont utilisées pour manipuler des documents en effectuant des opérations telles que la concaténation, le timbrage, la signature, l'annotation, etc., mais à un niveau élevé sans accès à la structure interne d'un document. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | L'espace de noms **Aspose.Pdf.Forms** contient des classes qui décrivent des formulaires (standard, statique, dynamique) et divers types de champs comme la zone de texte, la liste déroulante, le bouton radio, etc. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | Le **Aspose.Pdf.Tagged.LogicalStructure** est un espace de noms pour les classes de structure logique PDF. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | L'espace de noms **Aspose.Pdf.Multithreading** fournit des classes destinées à être utilisées dans des scénarios multi-threadés. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | L'espace de noms **Aspose.Pdf.Operators** est un espace de noms pour les implémentations d'opérateurs. Ces classes décrivent les opérateurs utilisés dans le contenu des pages PDF. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | L'espace de noms **Aspose.Pdf.Optimization** est un espace de noms pour les classes de gestion du processus d'optimisation de document. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | Le **Aspose.Pdf.Plugins** est un espace de noms racine pour toutes les classes des classes Aspose.Pdf.Plugins qui sont soit directement dans celui-ci comme **PdfOrganizer** ou indirectement à travers plusieurs sous-espaces de noms. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | L'espace de noms **Aspose.Pdf.Printing** fournit des classes et des fonctionnalités pour gérer les opérations d'impression PDF, y compris les paramètres de l'imprimante, les paramètres de page et les extensions pour gérer les configurations liées à l'impression. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | L'espace de noms **Aspose.Pdf.Sanitization** est un espace de noms pour les opérations de désinfection. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | L'espace de noms **Aspose.Pdf.Security** contient des classes utilisées pour le chiffrement et la signature numérique. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | L'espace de noms **Aspose.Pdf.Structure** fournit des classes qui aident à travailler avec la structure logique d'un document qui présente l'organisation du document en chapitres et sections ou l'identification d'éléments spéciaux tels que des figures, des tableaux et des notes de bas de page. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | Le **Aspose.Pdf.Tagged** est un espace de noms pour les classes de support des documents PDF tagués. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | L'espace de noms **Aspose.Pdf.Text** fournit des classes qui permettent d'extraire du texte, d'ajouter du texte, de manipuler le texte existant d'un document. Il contient également des classes qui permettent d'extraire, de remplacer, de substituer les polices d'un document. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | Le **Aspose.Pdf.Vector** est un espace de noms racine pour les opérations graphiques. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | L'espace de noms **Aspose.Pdf.XfaConverter** fournit des classes qui permettent de convertir XFA. |
