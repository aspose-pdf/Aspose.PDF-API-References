## Méthode ElementList.Item

Récupère un élément à l'index donné.

```csharp
public abstract Element Item(int index)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| index | Int32 | L'index dans la liste des éléments. |

### Valeur de retour

L'élément avec l'index spécifié dans la collection. Si *index* est supérieur ou égal au nombre d'éléments dans la liste, cela renvoie null.
