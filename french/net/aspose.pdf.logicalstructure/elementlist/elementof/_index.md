## Indexeur ElementList

Obtient un élément à l'index donné.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Paramètre | Description |
| --- | --- |
| index | L'index dans la liste des éléments. |

### Valeur de retour

L'élément avec l'index spécifié dans la collection. Si l'index est supérieur ou égal au nombre d'éléments dans la liste, cela renvoie null.
