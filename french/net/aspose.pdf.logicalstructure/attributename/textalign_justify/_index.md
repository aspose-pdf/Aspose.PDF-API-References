## Champ AttributeName.TextAlign_Justify

Attribut TextAlign : Justify - Aligné avec les bords de début et de fin, avec un espacement interne dans chaque ligne élargi, si nécessaire, pour atteindre un tel alignement. La dernière (ou seule) ligne doit être alignée uniquement avec le bord de début.

```csharp
public static readonly AttributeName TextAlign_Justify;
```
