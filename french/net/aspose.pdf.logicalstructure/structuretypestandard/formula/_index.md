## Champ StructureTypeStandard.Formula

(Formule) Une formule mathématique.

Ce type de structure est utile uniquement pour identifier un élément de contenu entier comme une formule. Aucun type de structure standard n'est défini pour identifier les composants individuels au sein de la formule. Du point de vue du formatage, la formule doit être traitée de manière similaire à une figure (type de structure Figure).

```csharp
public static readonly StructureTypeStandard Formula;
```
