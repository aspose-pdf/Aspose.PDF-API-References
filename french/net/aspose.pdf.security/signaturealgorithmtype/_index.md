## Énumération SignatureAlgorithmType

Énumère les types d'algorithmes de signature utilisés pour les signatures numériques.

```csharp
public enum SignatureAlgorithmType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Ecdsa | `0` | L'algorithme de signature numérique à courbe elliptique (ECDSA) utilisé pour les signatures numériques. |
| Rsa | `1` | L'algorithme Rivest–Shamir–Adleman (RSA) utilisé pour les signatures numériques. |
| Dsa | `2` | L'algorithme de signature numérique (DSA) utilisé pour les signatures numériques. |
| Timestamp | `3` | Le Timestamp utilisé comme algorithme de signature pour garantir les signatures numériques. |
| Unknown | `4` | Indique que le type d'algorithme de signature est inconnu. |
