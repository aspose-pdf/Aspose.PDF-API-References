Le **namespace Aspose.Pdf.Devices** fournit des classes qui sont utilisées pour représenter un document sous forme d'image(s) ou de texte brut. Ainsi, le document peut être envoyé à des dispositifs textuels ou graphiques variés, ce qui signifie que nous voulons obtenir sa représentation textuelle ou graphique.

## Classes

| Classe | Description |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf en bmp. |
| [Device](./device/) | Classe abstraite pour tous les types de dispositifs. Le dispositif est utilisé pour représenter un document pdf dans un certain format. Par exemple, la page d'un document peut être représentée sous forme d'image ou de texte. |
| [DicomDevice](./dicomdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf au format Dicom. |
| [DocumentDevice](./documentdevice/) | Classe abstraite pour tous les dispositifs qui est utilisée pour traiter l'ensemble du document pdf. |
| [EmfDevice](./emfdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf en emf. |
| [GifDevice](./gifdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf en gif. |
| [ImageDevice](./imagedevice/) | Classe abstraite pour les dispositifs d'image. |
| [JpegDevice](./jpegdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf en jpeg. |
| [Margins](./margins/) | Cette classe représente les marges d'une image. |
| [PageDevice](./pagedevice/) | Classe abstraite pour tous les dispositifs qui est utilisée pour traiter une certaine page du document pdf. |
| [PngDevice](./pngdevice/) | Représente un dispositif d'image qui aide à enregistrer les pages du document pdf en png. |
| [Resolution](./resolution/) | Représente une classe pour contenir la résolution de l'image. |
| [TextDevice](./textdevice/) | Représente une classe pour convertir les pages du document pdf en texte. |
| [ThumbnailDevice](./thumbnaildevice/) | Représente un dispositif d'image qui enregistre les pages du document pdf en image miniature. |
| [TiffDevice](./tiffdevice/) | Cette classe aide à enregistrer le document pdf page par page dans une seule image tiff. |
| [TiffSettings](./tiffsettings/) | Cette classe représente les paramètres pour importer un pdf vers Tiff. |
## Énumération

| Énumération | Description |
| --- | --- |
| [ColorDepth](./colordepth/) | Utilisé pour spécifier la valeur du paramètre passée à un dispositif d'image Tiff. |
| [CompressionType](./compressiontype/) | Utilisé pour spécifier la valeur du paramètre passée à un dispositif d'image Tiff. |
| [FormPresentationMode](./formpresentationmode/) | Utilisé pour spécifier le mode de présentation du formulaire lors de l'impression ou de la conversion en image des documents pdf. |
| [ShapeType](./shapetype/) | Cette énumération représente le type de forme pour les images extraites. |
