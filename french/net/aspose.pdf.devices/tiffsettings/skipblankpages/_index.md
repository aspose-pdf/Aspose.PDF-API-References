## Propriété TiffSettings.SkipBlankPages

Obtient ou définit une valeur indiquant s'il faut ignorer les pages blanches.

```csharp
public bool SkipBlankPages { get; set; }
```

### Valeur de la propriété

`true` si besoin d'ignorer les pages blanches ; sinon, `false`.

## Remarques

La valeur par défaut est false
