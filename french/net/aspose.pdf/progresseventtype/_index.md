## Énumération ProgressEventType

Cet enum décrit les types d'événements de progression possibles qui peuvent se produire lors de la conversion

```csharp
public enum ProgressEventType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| TotalProgress | `0` | signifie que l'événement survenu informe sur la progression totale de la conversion |
| SourcePageAnalysed | `1` | signifie que l'événement survenu informe sur la fin de l'analyse de l'une des pages avant la conversion |
| ResultPageCreated | `2` | signifie que l'événement survenu informe sur la création d'une page de résultat avant l'exportation physique |
| ResultPageSaved | `3` | signifie que l'événement survenu informe sur la sauvegarde terminée d'une page de résultat |
