## DocSaveOptions.RecognitionMode enumeration

Permet de contrôler la façon dont un document PDF est converti en un document de traitement de texte.

```csharp
public enum RecognitionMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Textbox | `0` | Ce mode est rapide et bon pour préserver au maximum l'aspect original du fichier PDF, mais la possibilité de modification du document résultant pourrait être limitée. |
| Flow | `1` | Mode de reconnaissance complète, le moteur effectue un regroupement et une analyse à plusieurs niveaux pour restaurer l'intention de l'auteur du document d'origine et produire un document modifiable au maximum. L'inconvénient est que le document de sortie peut sembler différent du fichier PDF d'origine. |
| EnhancedFlow | `2` | Une première version alfa d'un nouveau mode Flow prenant en charge la reconnaissance des tables. |

### Remarques

Utilisez leTextbox mode lorsque le document résultant ne va pas être fortement modifié ultérieurement. Les zones de texte sont faciles à modifier lorsqu'il n'y a pas grand-chose à faire.

Utilisez leFlowlorsque le document de sortie nécessite d'autres modifications. Les paragraphes et les lignes de texte en mode flux permettent une modification facile du texte, mais les objets de mise en forme non mis à jour auront une apparence pire que dans leTextbox mode.
