## Énumération PageCoordinateType

Décrit le type de coordonnées de page.

```csharp
public enum PageCoordinateType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| MediaBox | `0` | Le MediaBox est utilisé pour spécifier la largeur et la hauteur de la page. Pour l'utilisateur moyen, cela équivaut probablement à la taille réelle de la page. Le MediaBox est la plus grande boîte de page dans un PDF. Les autres boîtes de page peuvent égaler la taille du MediaBox mais ne peuvent pas être plus grandes. |
| CropBox | `1` | Le CropBox définit la région à laquelle le contenu de la page doit être découpé. Acrobat utilise cette taille pour l'affichage à l'écran et l'impression. |
