## Méthode Metadata.TryGetValue

Essaie de trouver la clé dans le dictionnaire et récupère la valeur si trouvée.

```csharp
public bool TryGetValue(string key, out XmpValue value)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| key | String | Clé à rechercher dans le dictionnaire. |
| value | XmpValue& | Valeur récupérée. |

### Valeur de retour

true si la clé a été trouvée.
