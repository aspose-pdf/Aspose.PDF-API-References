## Propriétés

| Nom | Description |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | Obtient un indicateur indiquant si le processeur PDF interactif doit fournir un support pour l'édition de la valeur du champ. Valeur par défaut : false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | Obtient le type d'une valeur de champ dans une collection de schéma. Ce champ décrit le type de valeur correspondant à [`Subtype`](./subtype/). |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | Obtient le nom textuel du champ qui doit être présenté à l'utilisateur par le processeur PDF interactif |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | Obtient l'ordre relatif du nom du champ dans l'interface utilisateur. Les champs doivent être triés par le processeur PDF interactif par ordre croissant. |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | Obtient le sous-type d'une valeur de champ dans une collection de schéma. Le sous-type de champ de collection ou de champ lié au fichier que ce dictionnaire décrit. Cette entrée identifie le type de données qui doit être stocké dans le champ. |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | Obtient la visibilité initiale du champ dans l'interface utilisateur. Valeur par défaut : true. |
