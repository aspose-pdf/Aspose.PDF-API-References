## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled field

Doit être défini par du code personnalisé s'il ne peut pas ou ne doit pas définir l'URL qui sera utilisée dans le HTML généré pour référencer ce CSS. Si c'est 'true', alors le fichier CSS sera enregistré de manière standard à l'endroit standard.

```csharp
public bool CustomProcessingCancelled;
```
