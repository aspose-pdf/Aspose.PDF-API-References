## Champ LoadOptions.ResourceLoadingResult.ExceptionOfLoadingIfAny

Parfois, il est impossible de charger la ressource demandée pour une raison quelconque. L'indisponibilité de la ressource ne conduit souvent pas à un plantage de conversiov et le document résultant peut être créé quand même (mais peut-être dans une qualité un peu inférieure, sans images, etc.). Si une exception s'est produite lors du chargement, il suffit de l'attraper et de la mettre dans ce paramètre - parfois cette information est utile pour le convertisseur pour le rendu du résultat.

```csharp
public Exception ExceptionOfLoadingIfAny;
```
