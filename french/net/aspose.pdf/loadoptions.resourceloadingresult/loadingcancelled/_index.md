## LoadOptions.ResourceLoadingResult.LoadingCancelled field

Parfois, pour certaines raisons, le chargement ne doit pas se produire avec du code personnalisé. Dans ce cas, veuillez définir ce drapeau sur True. Dans ce cas, le convertisseur essaiera d'utiliser le chargeur de ressources par défaut interne pour obtenir ce résultat (comme il se comporte dans une situation où la stratégie personnalisée n'est pas fournie).

```csharp
public bool LoadingCancelled;
```
