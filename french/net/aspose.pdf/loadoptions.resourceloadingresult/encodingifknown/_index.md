## LoadOptions.ResourceLoadingResult.EncodingIfKnown field

Parfois, le codage de la ressource est connu après ou pendant le chargement. Dans ce cas, le code personnalisé peut fournir au convertisseur cette connaissance via ce paramètre. Vous pouvez laisser null dans ce paramètre si l'encodage est inconnu ou n'a pas d'importance.
