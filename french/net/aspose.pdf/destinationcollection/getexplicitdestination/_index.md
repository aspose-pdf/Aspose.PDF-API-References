```csharp
public ExplicitDestination GetExplicitDestination(string destinameName, bool useCache)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| destinameName | String | Le nom de la destination. |
| useCache | Boolean | Détermine si la version mise en cache de la collection est utilisée ou non. |
