## Méthode DestinationCollection.GetPageNumber

Renvoie le numéro de page de la destination par le nom.

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| destinameName | String | Le nom de la destination. |
| useCache | Boolean | Détermine si la version mise en cache de la collection est utilisée ou non. |

### Valeur de retour

Le numéro de page si la destination a été trouvée ; sinon, -1.
