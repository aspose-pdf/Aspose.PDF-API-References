## Énumération WarningType

Type d'avertissement représenté par l'énumération.

```csharp
public enum WarningType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| SourceFileCorruption | `0` | Le fichier est corrompu. |
| DataLoss | `1` | Du texte/graphique/image ou d'autres données sont complètement manquants soit de l'arbre de documents après chargement, soit du document créé après sauvegarde. |
| MajorFormattingLoss | `2` | Pertes de formatage majeures par rapport au document original. Cela concerne les occasions où la perte de formatage est substantielle mais que les données sont toujours présentes. |
| MinorFormattingLoss | `3` | Pertes de formatage mineures par rapport au document original. Cela concerne les pertes mineures de fidélité. |
| CompatibilityIssue | `4` | Problème connu qui empêchera le document d'être ouvert par certains agents utilisateurs, ou par des versions précédentes d'agents utilisateurs. |
| InvalidInputStreamType | `5` | Type de flux d'entrée invalide |
| UnexpectedContent | `99` | Le fichier contient un contenu inattendu. |
