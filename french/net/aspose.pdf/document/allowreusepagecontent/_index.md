## Propriété Document.AllowReusePageContent

Permet de fusionner le contenu des pages pour optimiser la taille du document. Si utilisé, alors des pages différentes mais dupliquées peuvent faire référence au même objet de contenu. Veuillez noter que ce mode peut provoquer des effets secondaires, comme le changement du contenu de la page lorsque l'autre page est modifiée.

```csharp
public bool AllowReusePageContent { get; set; }
```
