## Propriété Document.CryptoAlgorithm

Obtient les paramètres de sécurité si le document est chiffré. Si le document n'est pas chiffré, une exception correspondante sera levée dans .net 1.1 ou CryptoAlgorithm sera nul pour d'autres versions de .net.
