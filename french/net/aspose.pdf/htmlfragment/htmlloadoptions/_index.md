## Propriété HtmlFragment.HtmlLoadOptions

Obtient ou définit HtmlLoadOptions qui seront utilisées pour charger (et rendre) du HTML dans cette instance de la classe. Veuillez l'utiliser lorsqu'il est nécessaire d'utiliser un paramètre spécifique pour l'importation de HTML pour cette ou cette instance (par exemple lorsque cette ou cette instance doit utiliser un BasePath spécifique pour le HTML importé ou doit utiliser un chargeur spécifique de ressources externes). Si le paramètre est par défaut (null), alors les options de chargement HTML standard seront utilisées.

```csharp
public HtmlLoadOptions HtmlLoadOptions { get; set; }
```
