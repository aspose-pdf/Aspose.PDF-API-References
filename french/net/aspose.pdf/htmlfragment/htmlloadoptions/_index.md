## HtmlFragment.HtmlLoadOptions property

Obtient ou définit les HtmlLoadOptions qui seront utilisées pour le chargement (et le rendu) du HTML dans cette instance de classe. Veuillez l'utiliser lorsqu'il est nécessaire d'utiliser un paramètre spécifique pour l'importation de HTML pour telle ou telle instance (par exemple lorsque telle ou telle instance doit utilisez un BasePath spécifique pour le HTML importé ou utilisez un chargeur spécifique de ressources externes) Si le paramètre est par défaut (null), les options de chargement HTML standard seront utilisées.

public HtmlLoadOptions HtmlLoadOptions { get; set; }
