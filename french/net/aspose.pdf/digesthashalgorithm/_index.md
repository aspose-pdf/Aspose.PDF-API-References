## Énumération DigestHashAlgorithm

Représente le type d'algorithme qui mappe les données à un "hachage"

```csharp
public enum DigestHashAlgorithm
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Auto | `0` | Réglage automatique de l'algorithme de hachage à la discrétion de l'algorithme de signature. Pour EDCSA, la valeur par défaut est déterminée par la taille de la clé. La valeur par défaut pour un PKCS7 non détaché est Sha1. |
| Sha1 | `1` | SHA-1. Algorithme de hachage sécurisé 1. C'est une valeur par défaut pour un PKCS7 non détaché. |
| Sha256 | `2` | SHA-256. Algorithme de hachage sécurisé 2. C'est une valeur par défaut pour un PKCS7 détaché. |
| Sha384 | `3` | SHA-384. Algorithme de hachage sécurisé 2 |
| Sha512 | `4` | SHA-512. Algorithme de hachage sécurisé 2 |
| Sha3_256 | `5` | SHA3-256. Algorithme de hachage sécurisé 3 |
| Sha3_384 | `6` | SHA3-384. Algorithme de hachage sécurisé 3 |
| Sha3_512 | `7` | SHA3-512. Algorithme de hachage sécurisé 3 |
