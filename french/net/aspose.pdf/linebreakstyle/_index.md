## Énumération LineBreakStyle

Représente les styles de saut de ligne possibles pour un fichier.

```csharp
public enum LineBreakStyle
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Windows | `0` | Sélectionne le style de retour à la ligne Windows (CRLF). |
| Unix | `1` | Sélectionne le style de retour à la ligne Unix (LF). Utilisé également pour OSX. |
| Auto | `2` | Sélectionne automatiquement le type de retour à la ligne en fonction du système d'exploitation. |
