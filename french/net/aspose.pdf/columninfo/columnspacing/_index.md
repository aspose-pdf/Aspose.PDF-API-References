## Propriété ColumnInfo.ColumnSpacing

Obtient ou définit une chaîne qui contient l'espacement entre les colonnes. La valeur de chaque espacement doit être séparée par un espace. L'unité par défaut est le point, mais le cm et le pouce sont également pris en charge. Par exemple, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Remarques

Si cette propriété n'est pas définie, la valeur par défaut 0 sera utilisée pour chaque espacement.
