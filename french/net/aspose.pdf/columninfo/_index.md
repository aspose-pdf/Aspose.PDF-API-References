## Propriétés

| Nom | Description |
| --- | --- |
| [ColumnCount](../../aspose.pdf/columninfo/columncount/) { get; set; } | Obtient ou définit une valeur int qui indique le nombre de colonnes. |
| [ColumnSpacing](../../aspose.pdf/columninfo/columnspacing/) { get; set; } | Obtient ou définit une chaîne qui contient l'espacement entre les colonnes. La valeur de chaque espacement doit être séparée par un espace. L'unité par défaut est le point, mais le cm et le pouce sont également pris en charge. Par exemple, "120 2.5cm 1.5inch". |
| [ColumnWidths](../../aspose.pdf/columninfo/columnwidths/) { get; set; } | Obtient ou définit une chaîne qui contient la largeur des colonnes. La valeur de chaque colonne doit être séparée par un espace. L'unité par défaut est le point, mais le cm, le pouce et le pourcentage de la largeur disponible sont également pris en charge. Par exemple, "120 2.5cm 1.5inch" |
