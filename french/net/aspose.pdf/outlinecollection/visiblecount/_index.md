## Propriété OutlineCollection.VisibleCount

Count est la somme du nombre d'éléments de contour descendants visibles à tous les niveaux. Note : veuillez ne pas confondre avec Count qui est le nombre d'éléments dans la collection.

```csharp
public override int VisibleCount { get; }
```
