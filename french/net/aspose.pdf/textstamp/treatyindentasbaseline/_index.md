## TextStamp.TreatYIndentAsBaseLine property

Définit l'origine des coordonnées pour placer le texte. Si TreatYIndentAsBaseLine = true (par défaut lorsque Draw = true) La valeur YIndent sera traitée comme la ligne de base du texte. If TreatYIndentAsBaseLine = false (par défaut lorsque Draw = false) La valeur YIndent sera traitée comme bottom ( ligne de descente) du texte.

public bool TreatYIndentAsBaseLine { get; set; }
