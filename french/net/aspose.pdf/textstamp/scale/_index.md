## Propriété TextStamp.Scale

Définit l'échelle du texte. Si cette propriété est définie sur true et que la valeur de Width est spécifiée, le texte sera mis à l'échelle pour s'adapter à la largeur spécifiée.

```csharp
public bool Scale { get; set; }
```
