## Propriété XImage.Name

Obtient ou définit le nom de l'image. Veuillez noter que si vous changez le nom de l'image qui a des références dans le contenu de la page, le document peut devenir incorrect. Veuillez utiliser la méthode XImage.Rename dans ce cas.

```csharp
public string Name { get; set; }
```
