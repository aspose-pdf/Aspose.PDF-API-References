## Énumération Artifact.ArtifactType

Énumération des types d'artefacts possibles.

```csharp
public enum ArtifactType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Pagination | `0` | Artefacts de pagination. Caractéristiques de page auxiliaires telles que les en-têtes et les folios (numéros de page). |
| Mise en page | `1` | Artefacts de mise en page. Éléments typographiques ou de design purement cosmétiques tels que les règles de notes de bas de page ou les écrans de fond. |
| Page | `2` | Artefacts de page. Aides à la production extrinsèques au document lui-même, telles que les marques de coupe et les barres de couleur. |
| Arrière-plan | `3` | Artefacts d'arrière-plan. Images, motifs ou blocs colorés. |
| Indéfini | `4` | Le type d'artefact n'est pas défini ou inconnu. |
