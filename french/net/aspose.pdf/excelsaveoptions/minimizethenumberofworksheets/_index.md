## Propriété ExcelSaveOptions.MinimizeTheNumberOfWorksheets

Définissez sur vrai si vous devez minimiser le nombre de feuilles de calcul dans le classeur résultant. La valeur par défaut est fausse ; cela signifie enregistrer chaque page PDF en tant que feuille de calcul séparée.

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
