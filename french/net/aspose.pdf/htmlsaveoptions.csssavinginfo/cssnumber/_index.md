## Champ HtmlSaveOptions.CssSavingInfo.CssNumber

Défini par le convertisseur. Pendant la conversion, plusieurs fichiers CSS sont créés. Cette propriété montre l'ordre du fichier CSS enregistré pendant la conversion. Elle peut être utilisée dans la logique du code personnalisé pour décider comment traiter ou où enregistrer le contenu CSS

```csharp
public int CssNumber;
```
