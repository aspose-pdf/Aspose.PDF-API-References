## Classe HtmlSaveOptions.CssSavingInfo

Cette classe représente un ensemble de données liées à l'enregistrement personnalisé de CSS lors de la conversion de PDF au format HTML

```csharp
public class CssSavingInfo
```

## Champs

| Nom | Description |
| --- | --- |
| [ContentStream](../../aspose.pdf/htmlsaveoptions.csssavinginfo/contentstream) | Défini par le convertisseur. Représente le contenu binaire du CSS enregistré |
| [CssNumber](../../aspose.pdf/htmlsaveoptions.csssavinginfo/cssnumber) | Défini par le convertisseur. Lors de la conversion, plusieurs fichiers CSS sont créés. Cette propriété montre l'ordre du fichier CSS enregistré lors de la conversion. Elle peut être utilisée dans la logique du code personnalisé pour décider comment traiter ou où enregistrer le contenu CSS |
| [SupposedURL](../../aspose.pdf/htmlsaveoptions.csssavinginfo/supposedurl) | Défini par le convertisseur. Nom de fichier supposé qui passe du convertisseur au code de la méthode personnalisée. Peut être utilisé dans le code personnalisé pour décider comment traiter ou où enregistrer le contenu |
