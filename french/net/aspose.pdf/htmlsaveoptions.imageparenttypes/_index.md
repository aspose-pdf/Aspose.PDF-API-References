## Énumération HtmlSaveOptions.ImageParentTypes

Énumère les types possibles de parents d'images auxquels une image peut appartenir sur une page HTML ou à une image parent SVG

```csharp
public enum ImageParentTypes
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| HtmlPage | `0` | L'image appartient à la page HTML elle-même |
| SvgImage | `1` | L'image a été intégrée dans une image SVG |
