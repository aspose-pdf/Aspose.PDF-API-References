## XmpPdfAExtensionValueType class

Le schéma PDF/A ValueType est requis pour tous les types de valeur de propriété qui ne sont pas définis dans la spécification XMP 2004, c'est-à-dire pour les types de valeur en dehors de la liste suivante : - Types de tableau (il s'agit de types de conteneur pouvant contenir un ou plusieurs champs ) : Alt, Bag, Seq - Types de valeurs de base : booléen, (ouvert et fermé) Choice, Date, Dimensions, Integer, Lang Alt, Locale, MIMEType, ProperName, Real, Text, Thumbnail, URI, URL, XPath - Gestion des médias types de valeur : AgentName, RenditionClass, ResourceEvent, ResourceRef, Version - Type de valeur Job/Workflow de base : Job - Types de valeur de schéma EXIF : Flash, CFAPattern, DeviceSettings, GPSCoordinate, OECF/SFR, Rational Schema namespace URI : http://www. aiim.org/pdfa/ns/type# Préfixe d'espace de noms de schéma requis : pdfaType
