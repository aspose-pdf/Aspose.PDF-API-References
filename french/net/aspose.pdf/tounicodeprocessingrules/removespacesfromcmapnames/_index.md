## Propriété ToUnicodeProcessingRules.RemoveSpacesFromCMapNames

Certaines polices ont des cartes de codes de caractères ToUnicode avec des espaces dans les noms. Ces espaces pourraient provoquer des erreurs avec le mappage de texte unicode. Ce drapeau ordonne de supprimer les espaces des noms des cartes de codes de caractères ToUnicode. Par défaut faux.

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
