## Propriété CollectionItem.IsEmpty

Obtient une valeur indiquant si l'élément de collection est vide.

```csharp
public bool IsEmpty { get; }
```

### Valeur de retour

true si l'élément de collection est vide ; sinon, false.

## Remarques

Cette propriété retourne true si l'élément de collection ne contient aucune valeur, y compris des valeurs de chaîne, des valeurs double, des valeurs entières et des valeurs de date. Si l'un de ces types de valeur est présent dans l'élément de collection, cette propriété retourne false.
