## Champ HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber

Indique au code personnalisé à quelle page de l'ensemble généré de fichiers de pages HTML l'image sauvegardée appartient. Si le fractionnement par pages est désactivé, cette valeur contient toujours '1' car dans ce cas, une seule page HTML est générée.

```csharp
public int HtmlHostPageNumber;
```
