## Propriété TeXFragment.TeXLoadOptionsOfInstance

Obtient ou définit TeXLoadOptions qui seront utilisées pour le chargement (et le rendu) de LaTeX dans cette instance de classe. Veuillez l'utiliser lorsqu'il est nécessaire d'utiliser un paramètre spécifique pour l'importation de LaTeX pour cette instance ou celle-là (par exemple lorsque cette instance ou celle-là doit utiliser un BasePath spécifique pour le LaTeX importé ou doit utiliser un chargeur spécifique de ressources externes). Si le paramètre est par défaut (null), alors les options de chargement standard de LaTeX seront utilisées.

```csharp
public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
```
