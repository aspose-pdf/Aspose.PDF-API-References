## TeXFragment.TeXLoadOptionsOfInstance property

Obtient ou définit TeXLoadOptions qui sera utilisé pour le chargement (et le rendu) de LaTeX dans cette instance de classe. Veuillez l'utiliser quand il est nécessaire d'utiliser un paramètre spécifique pour l'importation de LaTeX pour telle ou telle instance (par exemple lorsque telle ou telle instance devrait utilisez un BasePath spécifique pour LaTeX importé ou utilisez un chargeur spécifique de ressources externes) Si le paramètre est par défaut (null), les options de chargement LaTeX standard seront utilisées.
