## Propriété FileSpecification.Contents

Obtient ou définit le fichier de contenu. Cette propriété retourne des données chargées en mémoire, ce qui peut provoquer une exception de mémoire insuffisante pour de grandes données. Pour réduire l'utilisation de la mémoire, veuillez utiliser StreamContents.

```csharp
public Stream Contents { get; set; }
```
