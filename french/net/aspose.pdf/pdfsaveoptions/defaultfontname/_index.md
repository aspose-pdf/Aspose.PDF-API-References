## Propriété PdfSaveOptions.DefaultFontName

Nom de la police utilisé par défaut pour les polices qui sont absentes sur l'ordinateur. Lorsque le document PDF qui est enregistré au format PDF contient des polices qui ne sont pas disponibles dans le document lui-même et sur l'appareil, l'API remplace ces polices par la police par défaut (si une police avec `DefaultFontName` est trouvée sur l'appareil).

```csharp
public string DefaultFontName { get; set; }
```
