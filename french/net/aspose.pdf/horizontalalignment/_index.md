## Énumération HorizontalAlignment

Décrit l'alignement horizontal.

```csharp
public enum HorizontalAlignment
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Aucun | `0` | Pas d'alignement. |
| Gauche | `1` | Aligné à gauche. |
| Centre | `2` | Alignement centré. |
| Droit | `3` | Aligné à droite. |
| Justifié | `4` | Alignement justifié. Le texte sera aligné sur les marges gauche et droite. |
| JustifiéComplet | `5` | Semblable à l'alignement 'Justifié', sauf que la toute dernière ligne sera seulement alignée à gauche en mode 'Justifié', tandis qu'en mode 'JustifiéComplet', toutes les lignes seront alignées à gauche et à droite. |
