## Classe PdfANonSpecificationFlags

Cette classe contient des indicateurs pour contrôler la conversion PDF/A dans les cas où le document PDF source ne correspond pas à la spécification PDF. Si les indicateurs de cette classe sont utilisés, cela diminue les performances, mais c'est nécessaire lorsque le document PDF source ne peut pas être converti en format PDF/A de manière habituelle. Par défaut, tous les indicateurs sont définis sur faux.
