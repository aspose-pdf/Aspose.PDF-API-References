## PdfANonSpecificationFlags class

Cette classe contient des indicateurs pour contrôler la conversion PDF/A dans les cas où le document PDF source ne correspond pas à la spécification PDF. Si des drapeaux de cette classe sont utilisés, cela diminue les performances mais c'est nécessaire lorsque le document PDF source ne peut pas être converti au format PDF/A de la manière habituelle. Par défaut, tous les drapeaux sont définis sur faux.
