## Énumération ParagraphPositioningMode

Spécifie la variante pour déterminer l'emplacement de l'élément sur la page.

```csharp
public enum ParagraphPositioningMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | `0` | L'emplacement est déterminé par les éléments placés précédemment. L'ajout d'un élément est pris en compte lors de la détermination de l'emplacement des éléments suivants. |
| Absolute | `1` | L'emplacement est spécifié par les valeurs Left et Top, ne dépend pas des éléments précédents et n'affecte pas l'emplacement des suivants. |
