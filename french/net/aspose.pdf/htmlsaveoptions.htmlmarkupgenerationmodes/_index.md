## HtmlSaveOptions.HtmlMarkupGenerationModes enumeration

Parfois, des exigences spécifiques au HTML créé sont présentes. Cette énumération définit les modes de préparation HTML qui peuvent être utilisés lors de la conversion de PDF en HTML pour répondre à ces exigences spécifiques.

```csharp
public enum HtmlMarkupGenerationModes
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| WriteAllHtml | `0` | Mode par défaut, aucune exigence spécifique n'est requise. Une sortie sera générée qui contiendra toutes les parties de HTML sans aucun traitement supplémentaire spécial. |
| WriteOnlyBodyContent | `1` | sera supprimé tout le contenu HTML qui est en dehors du corps de HTML, c'est-à-dire qu'il ne restera que le contenu qui est à l'intérieur tags |
