## Énumération HtmlSaveOptions.HtmlMarkupGenerationModes

Parfois, des exigences spécifiques pour le HTML créé sont présentes. Cet enum définit les modes de préparation du HTML qui peuvent être utilisés lors de la conversion de PDF en HTML pour répondre à ces exigences spécifiques.

```csharp
public enum HtmlMarkupGenerationModes
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| WriteAllHtml | `0` | Mode par défaut lorsque aucune exigence spécifique n'est présente. Un output sera généré qui contiendra toutes les parties du HTML sans aucun traitement supplémentaire spécial. |
| WriteOnlyBodyContent | `1` | Tout le contenu HTML qui est en dehors du corps du HTML sera supprimé, c'est-à-dire qu'il ne restera que le contenu qui est à l'intérieur des balises \c \ \c \ |
