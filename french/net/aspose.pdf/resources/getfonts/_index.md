## Méthode Resources.GetFonts

Renvoie la collection de polices. Si les ressources ne contiennent pas d'entrée de polices, elle sera créée en fonction du drapeau CreateIfAbsent.

```csharp
public FontCollection GetFonts(bool CreateIfAbsent)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| CreateIfAbsent | Boolean | Si ce drapeau est vrai, alors les polices seront créées si cette entrée est absente. |

### Valeur de retour

Collection de polices.
