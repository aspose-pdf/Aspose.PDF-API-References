## Énumération PclLoadOptions.ConversionEngines

Énumère les moteurs de conversion qui peuvent être utilisés pour la conversion

```csharp
public enum ConversionEngines
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| LegacyEngine | `0` | Le moteur de conversion Legacy est le moteur de conversion qui a été utilisé pendant un certain temps et est utilisé par défaut. |
| NewEngine | `1` | Le nouveau moteur de conversion est actuellement en état de test A. Nous l'ouvrons dans l'API publique pour obtenir des retours. Il est actuellement recommandé de l'utiliser uniquement lorsque l'ancien moteur de conversion ne peut pas convertir le document pour certaines raisons. |
