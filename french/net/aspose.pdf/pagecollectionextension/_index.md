## Classe PageCollectionExtension

Représente la méthode d'extension pour mettre à jour la pagination des en-têtes et des pieds de page.

```csharp
public static class PageCollectionExtension
```

## Méthodes

| Nom | Description |
| --- | --- |
| static [UpdatePagination](../../aspose.pdf/pagecollectionextension/updatepagination/)(this PageCollection) | Met à jour les numéros de page et les dates des en-têtes et des pieds de page pour toutes les pages. Cela fonctionnera si le document a au moins un artefact de pagination avec des données de paramètres spéciales. Toutes les pages de la collection seront mises à jour avec l'artefact source selon ses paramètres. |
