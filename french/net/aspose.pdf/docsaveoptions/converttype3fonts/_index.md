## Propriété DocSaveOptions.ConvertType3Fonts

Obtient ou définit la conversion pour les polices Type3. Dans les polices Type 3, les glyphes doivent être définis par des flux d'opérateurs graphiques. Cela signifie que dans la sortie DOC/DOCX, nous voyons des images au lieu de texte. Définissez ce drapeau sur vrai pour convertir les polices Type3 en TTF et obtenir du texte dans le fichier résultant.

```csharp
public bool ConvertType3Fonts { get; set; }
```
