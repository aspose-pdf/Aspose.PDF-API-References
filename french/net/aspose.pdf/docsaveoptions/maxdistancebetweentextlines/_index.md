## Propriété DocSaveOptions.MaxDistanceBetweenTextLines

Ce paramètre est utilisé pour regrouper les lignes de texte en paragraphes. Détermine la distance maximale entre deux lignes de texte relatives. Spécifié en centaines de pourcentages de la hauteur des lignes de texte.

```csharp
public float MaxDistanceBetweenTextLines { get; set; }
```
