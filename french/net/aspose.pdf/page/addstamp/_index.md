## Méthode Page.AddStamp

Mettre un tampon dans la page. Le tampon peut être un numéro de page, une image ou un simple texte, par exemple un logo.

```csharp
public void AddStamp(Stamp stamp)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| stamp | Stamp | Tampon à ajouter sur la page. Chaque tampon a ses coordonnées et ses propriétés correspondantes en fonction du type de tampon, c'est-à-dire une image ou une valeur de texte. |
