## Méthode Page.AddGraphics

Ajoute des graphiques à la page. Fonctionne plus rapidement que l'ajout d'éléments un par un avec la méthode [`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/).

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| elements | GraphicElementCollection | Collection de graphiques. |
| rectangle | Rectangle | Les éléments seront ajoutés à la page s'ils se trouvent à l'intérieur de la zone rectangulaire. Si le rectangle est nul, tous les éléments graphiques seront ajoutés |
