## Propriété Page.UserUnit

Obtient ou définit la valeur UserUnit. Un nombre positif donnant la taille des unités d'espace utilisateur par défaut, en multiples de 1 / 72 pouce. La valeur par défaut est 1. Veuillez définir une valeur nulle ou négative afin de supprimer cette entrée dans la page.

```csharp
public double UserUnit { get; set; }
```
