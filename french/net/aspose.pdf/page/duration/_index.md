## Page.Duration property

Obtient la durée d'affichage de la page définie. Il s'agit du temps en secondes pendant lequel la page doit être affichée lors de la présentation. Renvoie -1 si la durée n'est pas définie.

```csharp
public double Duration { get; set; }
```

### Exemples

L'exemple montre comment obtenir la durée de la page

```csharp

Page page = document.Pages[1];
int pageRect = page.Duration;
```
