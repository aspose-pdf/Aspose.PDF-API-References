## Propriété Page.IsAddParagraphsAfterLast

Obtient ou définit l'ajout de paragraphes après le dernier paragraphe de la page

```csharp
public bool IsAddParagraphsAfterLast { get; set; }
```

### Valeur de la propriété

La valeur indique si des paragraphes seront ajoutés après le dernier paragraphe de la page. Des paragraphes seront ajoutés après le dernier paragraphe de la page si la valeur est vraie.
