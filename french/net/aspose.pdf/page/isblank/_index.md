## Méthode Page.IsBlank

Obtient le drapeau indiquant si la page est vide ou non.

```csharp
public bool IsBlank(double fillThresholdFactor)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| fillThresholdFactor | Double | La valeur de seuil de remplissage qui gère la sensibilité de la détection. Doit être égale ou supérieure à 0.01. |

### Valeur de retour

Vrai - si la page est vide ; sinon, faux.
