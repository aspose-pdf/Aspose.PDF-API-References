## Propriété Stamp.Rotate

Définit ou obtient la rotation du contenu du tampon selon les valeurs de [`Rotation`](../../rotation/). Remarque. Cette propriété est pour définir des angles qui sont des multiples de 90 degrés (0, 90, 180, 270 degrés). Pour définir un angle arbitraire, utilisez la propriété RotateAngle. Si l'angle défini par ArbitraryAngle n'est pas un multiple de 90, alors la propriété Rotate renvoie Rotation.None.

```csharp
public Rotation Rotate { get; set; }
```
