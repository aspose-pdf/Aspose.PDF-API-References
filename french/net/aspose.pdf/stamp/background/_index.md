## Propriété Stamp.Background

Définit ou obtient une valeur booléenne qui indique que le contenu est estampillé en tant qu'arrière-plan. Si la valeur est vraie, le contenu du tampon est placé en bas. Par défaut, la valeur est fausse, le contenu du tampon est placé en haut.

```csharp
public bool Background { get; set; }
```
