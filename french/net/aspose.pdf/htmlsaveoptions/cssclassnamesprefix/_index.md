## HtmlSaveOptions.CssClassNamesPrefix field

Lorsque le convertisseur PDFtoHTML génère des CSS de résultat, les noms de classe CSS (quelque chose comme ".stl_01 {}" ... ".stl_NN {}) sont générés et utilisés dans le résultat CSS. Cette propriété permet de définir de force le nom de classe prefix Par exemple, si vous voulez que tous les noms de classe commencent par 'my_prefix_' (c'est-à-dire quelque chose comme 'my_prefix_1' ... 'my_prefix_NNN' ), puis attribuez simplement 'my_prefix_' à cette propriété avant la conversion. Si cette propriété restera intacte (c'est-à-dire null sera laissé comme value ), then converter générera lui-même des noms de classe (ce sera quelque chose comme ".stl_01 {}" ... ".stl_NN {}")
