## Propriété HtmlSaveOptions.ConvertMarkedContentToLayers

Si l'attribut ConvertMarkedContentToLayers est défini sur true, alors tous les éléments à l'intérieur d'un contenu marqué PDF (couche) seront placés dans une div HTML avec l'attribut "data-pdflayer" spécifiant un nom de couche. Ce nom de couche sera extrait des propriétés optionnelles du contenu marqué PDF. Si cet attribut est false (par défaut), alors aucune couche ne sera créée à partir du contenu marqué PDF.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
