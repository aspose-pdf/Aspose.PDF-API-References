## HtmlSaveOptions.ConvertMarkedContentToLayers property

Si l'attribut ConvertMarkedContentToLayers est défini sur true, tous les éléments d'un contenu PDF marqué (calque) seront placés dans une div HTML avec l'attribut "data-pdflayer" spécifiant un nom de calque. Ce nom de calque sera extrait des propriétés facultatives du PDF contenu marqué. Si cet attribut est faux (par défaut), aucune couche ne sera créée à partir du contenu marqué PDF.

public bool ConvertMarkedContentToLayers { get; set; }
