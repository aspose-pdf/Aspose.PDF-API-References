## HtmlSaveOptions.SpecialFolderForAllImages field

Obtient ou définit le chemin d'accès au répertoire dans lequel doivent être enregistrées les images si elles sont rencontrées lors de l'enregistrement du document au format HTML. Si le paramètre est vide ou null , les fichiers image (le cas échéant) seront enregistrés avec d'autres fichiers liés à HTML Cela n'affecte rien si la propriété CustomImageSavingStrategy a été utilisée avec succès pour traiter le fichier image pertinent.

public string SpecialFolderForAllImages;
