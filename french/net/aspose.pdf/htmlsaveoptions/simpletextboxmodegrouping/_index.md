## HtmlSaveOptions.SimpleTextboxModeGrouping property

Cet attribut spécifie un regroupement séquentiel de glyphes et de mots dans des chaînes Par exemple, les balises et les mots ont un ordre différent dans le code HTML converti et vous souhaitez qu'ils correspondent. Ce paramètre sera appliqué au document uniquement lorsque la valeur de l'attribut FixedLayout est true.
