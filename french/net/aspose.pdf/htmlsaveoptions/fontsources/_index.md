## Propriété HtmlSaveOptions.FontSources

Sources de polices préenregistrées.

```csharp
public FontSourceCollection FontSources { get; }
```

## Remarques

Les polices peuvent être enregistrées préalablement à des fins de cache et ensuite passées dans le processus de conversion Html. Par exemple, cela peut être utile dans un scénario de division de document et de traitement des pages de document dans plusieurs threads avec un seul ensemble de polices.
