## HtmlSaveOptions.SaveTransparentTexts field

Pdf peut contenir des textes transparents pouvant être sélectionnés dans le presse-papiers (généralement, cela se produit lorsque le document contient des images et des textes OCR extraits). Ce paramètre indique au convertisseur si nous devons enregistrer ces textes en tant que textes transparents sélectionnables dans le résultat HTML

```csharp
public bool SaveTransparentTexts;
```
