## Champ HtmlSaveOptions.SaveTransparentTexts

Le PDF peut contenir des textes transparents qui peuvent être sélectionnés dans le presse-papiers (cela se produit généralement lorsque le document contient des images et des textes OCR extraits de celui-ci). Ce paramètre indique au convertisseur si nous devons enregistrer ces textes en tant que textes transparents sélectionnables dans le HTML résultant.

```csharp
public bool SaveTransparentTexts;
```
