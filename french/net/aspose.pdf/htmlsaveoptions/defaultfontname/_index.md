## Propriété HtmlSaveOptions.DefaultFontName

Spécifie le nom d'une police installée qui est utilisée pour remplacer toute police de document qui n'est pas intégrée et qui n'est pas installée dans le système. Si null, alors la police de substitution par défaut est utilisée.

```csharp
public string DefaultFontName { get; set; }
```

### Valeur de la propriété

Nom de la police
