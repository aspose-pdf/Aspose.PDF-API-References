## Propriété HtmlSaveOptions.UseZOrder

Si l'attribut UseZOrder est défini sur true, les graphiques et le texte sont ajoutés au document HTML résultant conformément à l'ordre Z dans le document PDF original. Si cet attribut est faux, tous les graphiques sont placés comme une seule couche, ce qui peut causer des effets indésirables pour les objets superposés.

```csharp
public bool UseZOrder { get; set; }
```
