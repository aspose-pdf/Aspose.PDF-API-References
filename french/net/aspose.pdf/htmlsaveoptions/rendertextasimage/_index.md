## Propriété HtmlSaveOptions.RenderTextAsImage

Si l'attribut RenderTextAsImage est défini sur true, le texte de la source devient une image en HTML. Peut être utile pour rendre le texte non sélectionnable ou si le texte HTML n'est pas rendu correctement.

```csharp
public bool RenderTextAsImage { get; set; }
```
