## Propriété PaginationArtifact.EndPage

Obtient ou définit le numéro de page de fin pour l'artéfact. La valeur doit être supérieure ou égale à 0. Si une valeur inférieure à 0 est définie, elle sera ajustée à 0. La valeur par défaut de 0 signifie qu'il n'y a pas de limites de page de fin.

```csharp
public int EndPage { get; set; }
```
