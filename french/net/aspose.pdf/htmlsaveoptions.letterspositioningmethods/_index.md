## Énumération HtmlSaveOptions.LettersPositioningMethods

Il énumère les modes possibles de positionnement des lettres dans les mots dans le HTML résultant

```csharp
public enum LettersPositioningMethods
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| UseEmUnitsAndCompensationOfRoundingErrorsInCss | `0` | C'est la méthode par défaut. Elle utilise des unités EM et un algorithme spécial de compensation des erreurs d'arrondi. Elle est préférable pour une utilisation dans IE10.0 et les versions plus récentes et offre un meilleur redimensionnement des légendes lorsque le redimensionnement est nécessaire |
| UsePixelUnitsInCssLetterSpacingForIE | `1` | Elle permet d'obtenir parfois des résultats plus précis dans les anciennes versions du navigateur IE |
