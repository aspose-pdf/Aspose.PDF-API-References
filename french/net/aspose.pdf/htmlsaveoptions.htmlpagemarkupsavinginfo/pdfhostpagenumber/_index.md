## HtmlSaveOptions.HtmlPageMarkupSavingInfo.PdfHostPageNumber field

Défini par le convertisseur. Si la propriété SplitToPages est définie, plusieurs fichiers HTML (un fichier HTML par page convertie) sont créés lors de la conversion créée . Cette propriété indique au code personnalisé à partir de quelle page du PDF original a été créé le balisage HTML enregistré . Si le numéro de page d'origine pour une raison quelconque est inconnu ou SplitOnPages=false, alors cette propriété contient toujours '0' qui signale que le convertisseur ne peut pas fournir le numéro de page exact du PDF d'origine pour le fichier de balisage HTML fourni.

public int PdfHostPageNumber;
