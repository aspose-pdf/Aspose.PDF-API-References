## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber field

Défini par le convertisseur. Si la propriété SplitToPages est définie, plusieurs fichiers HTML (un fichier HTML par page convertie) sont créés lors de la conversion. Cette propriété contient l'ordinal du fichier de la page HTML enregistrée. La propriété peut être utilisée dans la logique du code personnalisé pour décider comment traiter ou où enregistrer la page HTML et Si le fractionnement sur les pages est désactivé, cette valeur contient toujours '1' puisque dans dans ce cas, une seule grande page HTML est générée pour l'ensemble du document source.
