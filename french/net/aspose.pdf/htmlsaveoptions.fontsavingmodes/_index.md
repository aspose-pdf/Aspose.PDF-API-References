## HtmlSaveOptions.FontSavingModes enumeration

Énumère les modes pouvant être utilisés pour enregistrer les polices référencées dans le PDF enregistré

```csharp
public enum FontSavingModes
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| AlwaysSaveAsWOFF | `0` | Toutes les polices référencées seront enregistrées et référencées en tant que WOFF-fonts |
| AlwaysSaveAsTTF | `1` | Toutes les polices référencées seront enregistrées et référencées en tant que TTF-fonts |
| AlwaysSaveAsEOT | `2` | Toutes les polices référencées seront enregistrées et référencées en tant que polices EOT |
| SaveInAllFormats | `3` | Toutes les polices référencées seront enregistrées (et référencées dans CSS) sous 3 fichiers indépendants : EOT, TTH, WOFF. Cela augmente la taille des données de sortie mais rend la sortie adaptée à la majorité écrasante des navigateurs Web |
| DontSave | `4` | Toutes les polices référencées ne seront pas enregistrées. |
