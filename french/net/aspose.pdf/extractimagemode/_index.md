## Énumération ExtractImageMode

Définit différents modes qui peuvent être utilisés lors de l'extraction d'images à partir de documents.

```csharp
public enum ExtractImageMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| DefinedInResources | `0` | Définit le mode d'extraction d'image dans lequel toutes les images définies dans les ressources pour une page particulière sont extraites. |
| ActuallyUsed | `1` | Définit le mode d'extraction d'image dans lequel seules les images qui sont réellement affichées sur une page sont extraites. |
