## Méthode XImageCollection.GetImageName

Renvoie le nom dans la liste des images qui est la clé de l'image donnée.

```csharp
public string GetImageName(XImage image)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| image | XImage | Image à rechercher. |

### Valeur de retour

Nom (clé) de l'image trouvée ; null si aucune image n'a été trouvée.
