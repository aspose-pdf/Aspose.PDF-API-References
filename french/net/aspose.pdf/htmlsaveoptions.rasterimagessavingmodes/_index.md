## HtmlSaveOptions.RasterImagesSavingModes enumeration

Le PDF converti peut contenir des images raster (.png, *.jpeg, etc.) Cette énumération définit les méthodes de traitement des images raster lors de la conversion du PDF en HTML

```csharp
public enum RasterImagesSavingModes
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| AsPngImagesEmbeddedIntoSvg | `0` | pour chaque fichier raster distinct sera généré une image SVG wrapper, et l'image raster sera intégrée en tant que chaînes codées en Base64 dans cette image SVG |
| AsExternalPngFilesReferencedViaSvg | `1` | des images raster distinctes seront mises à part en tant que fichiers PNG mais seront référencées via des images SVG enveloppantes, c'est-à-dire qu'un fichier PNG et un SVG seront générés pour chaque image raster, et chacun de ces SVG contiendra des liens vers le fichier PNG pertinent |
| AsEmbeddedPartsOfPngPageBackground | `2` | Sera généré un gros fichier d'arrière-plan PNG pour chaque page de résultats. Les images raster seront intégrées dans ce fichier et rendues en tant que régions de cette image. Aucun fichier PNG externe pour chaque image ne sera généré, un seul fichier PNG par page sera présent dans le jeu de résultats de conversion de fichiers. |
