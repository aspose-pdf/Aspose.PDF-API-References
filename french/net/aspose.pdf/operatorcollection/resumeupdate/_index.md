## ResumeUpdate(bool) {#resumeupdate_1}

Reprend la mise à jour du document. Met à jour le flux de contenu en cas de modifications en attente. Marque tous les opérateurs comme "modifiés" si le paramètre invalidate est vrai.

```csharp
public void ResumeUpdate(bool updateAll)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| updateAll | Boolean | Si vrai, tous les opérateurs dans la collection sont marqués comme mis à jour. |
