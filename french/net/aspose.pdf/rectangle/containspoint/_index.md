## Méthode Rectangle.ContainsPoint

Détermine si le point donné est contenu dans le rectangle.

```csharp
public bool ContainsPoint(double x, double y)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| x | Double | Coordonnée X du point. |
| y | Double | Coordonnée Y du point. |

### Valeur de retour

`true` si le point est contenu dans le rectangle ; sinon, `false`.
