## Méthode Rectangle.ContainsLine

Détermine si le rectangle contient une ligne représentée par deux points.

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| x1 | Double | La coordonnée X du point de départ de la ligne. |
| y1 | Double | La coordonnée Y du point de départ de la ligne. |
| x2 | Double | La coordonnée X du point d'arrivée de la ligne. |
| y2 | Double | La coordonnée Y du point d'arrivée de la ligne. |

### Valeur de retour

`true` si le rectangle contient la ligne ; sinon, `false`.
