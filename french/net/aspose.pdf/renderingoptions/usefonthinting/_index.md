## Propriété RenderingOptions.UseFontHinting

L'utilisation de ce drapeau active le mécanisme de hinting de police. Le hinting de police est l'utilisation d'instructions mathématiques pour ajuster l'affichage d'une police en contours. Dans certains cas, activer ce drapeau peut résoudre des problèmes de lisibilité du texte. À l'heure actuelle, l'utilisation de ce drapeau ne peut avoir d'effet que pour les polices TTF, si ces polices sont utilisées dans le document source.

```csharp
public bool UseFontHinting { get; set; }
```
