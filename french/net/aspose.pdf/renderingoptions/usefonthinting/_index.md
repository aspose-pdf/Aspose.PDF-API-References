## RenderingOptions.UseFontHinting property

L'utilisation de cet indicateur active le mécanisme d'indication de police. L'indication de police est l'utilisation d'instructions mathématiques pour ajuster l'affichage d'une police vectorielle. Dans certains cas, l'activation de cet indicateur peut résoudre des problèmes de lisibilité du texte. À l'heure actuelle, l'utilisation de cet indicateur ne peut avoir d'effet que pour les polices TTF, si ces polices sont utilisées dans le document source.

public bool UseFontHinting { get; set; }
