## Propriété RenderingOptions.ConvertFontsToUnicodeTTF

Indique que toutes les polices seront converties en versions TTF unicode. Cela est utile pour des raisons de compatibilité et pour optimiser l'utilisation des polices, car chaque nouvelle police TTF n'aura pas tous les symboles de la police source, mais uniquement les symboles qui sont utilisés dans le texte.

```csharp
public bool ConvertFontsToUnicodeTTF { get; set; }
```
