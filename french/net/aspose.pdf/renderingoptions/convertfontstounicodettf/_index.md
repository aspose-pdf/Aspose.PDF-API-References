## RenderingOptions.ConvertFontsToUnicodeTTF property

Indique que toutes les polices seront converties en versions unicode TTF. Ceci est utile pour des raisons de compatibilité et pour optimiser l'utilisation des polices, car chaque nouvelle police TTF n'aura pas tous les symboles de la police source, mais uniquement les symboles utilisés dans le texte.

public bool ConvertFontsToUnicodeTTF { get; set; }
