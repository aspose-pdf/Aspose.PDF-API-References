## Propriété RenderingOptions.IgnoreResourceFontErrors

Obtient ou définit l'indication que les erreurs liées à l'absence de police seront ignorées. true - signifie que les erreurs d'absence de police seront ignorées. Les segments de texte qui se réfèrent à des ressources incorrectes seront ignorés lors du traitement. false par défaut

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
