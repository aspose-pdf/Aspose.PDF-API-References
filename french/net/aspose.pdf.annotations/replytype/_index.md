## Énumération ReplyType

Énumère les types de relations (le "type de réponse") entre l'annotation et celle spécifiée par InReplyTo.

```csharp
public enum ReplyType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Undefined | `0` | Relation indéfinie. |
| Reply | `1` | L'annotation est considérée comme une réponse à l'annotation spécifiée par InReplyTo. Les applications de visualisation ne doivent pas afficher les réponses à une annotation individuellement mais ensemble sous forme de commentaires en fil. |
| Group | `2` | L'annotation est regroupée avec l'annotation spécifiée par InReplyTo. |
