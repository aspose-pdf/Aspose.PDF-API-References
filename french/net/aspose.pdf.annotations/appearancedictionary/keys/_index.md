## Propriété AppearanceDictionary.Keys

Obtient les clés du dictionnaire. Si le dictionnaire d'apparence a des sous-dictionnaires, alors `Keys` contient les valeurs (N&#x7C;R&#x7C;D).state, où N - apparence normale, R - apparence au survol, D - apparence enfoncée et state - le nom de l'état (par exemple On, Off pour les cases à cocher).

public ICollection<string> Keys { get; }
