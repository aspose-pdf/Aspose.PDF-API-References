## Indexeur AppearanceDictionary

Représente une forme pratique pour obtenir des flux d'apparence.

```csharp
public XForm this[string key] { get; set; }
```

| Paramètre | Description |
| --- | --- |
| key | Représente le chemin vers le flux d'apparence. Si le dictionnaire d'apparence a des sous-dictionnaires, alors le chemin doit contenir 2 parties ([`Keys`](../keys/)), sinon le chemin n'a qu'une seule partie. |

### Valeur de retour

Objet XForm (flux d'apparence) qui correspond à la clé donnée.
