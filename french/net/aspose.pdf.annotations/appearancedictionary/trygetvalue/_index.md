## Méthode AppearanceDictionary.TryGetValue

Essaie de trouver la clé dans le dictionnaire et récupère la valeur si elle est trouvée.

```csharp
public bool TryGetValue(string key, out XForm value)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| key | String | Clé à rechercher dans le dictionnaire. |
| value | XForm& | Valeur récupérée. |

### Valeur de retour

true si la clé a été trouvée.
