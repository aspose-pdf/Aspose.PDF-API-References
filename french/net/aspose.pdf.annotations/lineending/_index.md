## Énumération LineEnding

Énumère les styles de fin de ligne à utiliser pour dessiner la ligne.

```csharp
public enum LineEnding
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | `0` | Pas de fin de ligne. |
| Square | `1` | Un carré rempli de la couleur intérieure de l'annotation, le cas échéant. |
| Circle | `2` | Un cercle rempli de la couleur intérieure de l'annotation, le cas échéant. |
| Diamond | `3` | Une forme de diamant remplie de la couleur intérieure de l'annotation, le cas échéant. |
| OpenArrow | `4` | Deux lignes courtes se rejoignant en un angle aigu pour former une pointe de flèche ouverte. |
| ClosedArrow | `5` | Deux lignes courtes se rejoignant en un angle aigu comme dans le style OpenArrow et reliées par une troisième ligne pour former une pointe de flèche fermée triangulaire remplie de la couleur intérieure de l'annotation, le cas échéant. |
| Butt | `6` | Une courte ligne à l'extrémité perpendiculaire à la ligne elle-même. |
| ROpenArrow | `7` | Deux lignes courtes dans la direction inverse de OpenArrow. |
| RClosedArrow | `8` | Une pointe de flèche fermée triangulaire dans la direction inverse de ClosedArrow. |
| Slash | `9` | Une courte ligne à l'extrémité à environ 30 degrés dans le sens des aiguilles d'une montre par rapport à la perpendiculaire à la ligne elle-même. |
