## Méthode MarkupAnnotation.SetMarkedState

Définit l'état marqué et non marqué pour l'annotation. Notez que l'état est stocké dans une autre annotation de texte qui a des clés state et statemodel.

```csharp
public void SetMarkedState(bool marked)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| marked | Boolean | Vrai si définit l'état marqué, et faux si définit l'état non marqué. |
