## Indexeur PDF3DViewArray

Obtient ou définit le [`PDF3DView`](../../pdf3dview/) à afficher dans le tableau à l'index spécifié.

```csharp
public PDF3DView this[int index] { get; set; }
```

| Paramètre | Description |
| --- | --- |
| index | L'index. |

### Valeur de retour

### Exceptions

| exception | condition |
| --- | --- |
| IndexOutOfRangeException | Index invalide : l'index doit être dans la plage [1..n] où n est égal au nombre de vues. |
