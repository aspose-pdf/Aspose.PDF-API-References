## Énumération PrinterMarksKind

Spécifie les types de marques d'imprimante à ajouter à un document.

```csharp
[Flags]
public enum PrinterMarksKind
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Aucune | `0` | Spécifie qu'aucune marque d'imprimante ne doit être ajoutée. |
| MarquesDeDécoupe | `1` | Spécifie que des marques de découpe doivent être ajoutées. |
| MarquesDeSaignée | `2` | Spécifie que des marques de saignée doivent être ajoutées. |
| MarquesD'enregistrement | `4` | Spécifie que des marques d'enregistrement doivent être ajoutées. |
| BarresDeCouleur | `8` | Spécifie que des barres de couleur doivent être ajoutées. |
| InformationsSurLaPage | `10` | Spécifie que des informations sur la page doivent être ajoutées. |
| Tout | `1F` | Spécifie que tous les types de marques d'imprimante doivent être ajoutés. |

## Remarques

Cette énumération a un attribut FlagsAttribute qui permet une combinaison bit à bit de ses valeurs membres.
