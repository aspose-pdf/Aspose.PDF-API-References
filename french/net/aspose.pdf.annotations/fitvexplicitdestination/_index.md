## FitVExplicitDestination class

Représente une destination explicite qui affiche la page avec la coordonnée horizontale à gauche positionnée sur le bord gauche de la fenêtre et le contenu de la page agrandi juste assez pour tenir sur toute la hauteur de la page dans la fenêtre. Une valeur nulle pour gauche spécifie que la valeur actuelle de ce paramètre doit être conservée inchangée.

public sealed class FitVExplicitDestination : ExplicitDestination
