## ExplicitDestinationType enumeration

Énumère les types de destinations explicites.

public enum ExplicitDestinationType

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| XYZ | `0` | Affiche la page avec les coordonnées (gauche, haut) positionnées dans le coin supérieur gauche de la fenêtre et le contenu de la page agrandi par le facteur de zoom. Une valeur nulle pour l'un des parameters left, top ou zoom spécifie que la valeur actuelle de ce paramètre doit être conservée telle quelle. Une valeur de zoom de 0 a la même signification qu'une valeur nulle. |
| Fit | `1` | Affichez la page avec son contenu agrandi juste assez pour tenir toute la page dans la fenêtre à la fois horizontalement et verticalement. Si les facteurs d'agrandissement horizontal et vertical requis sont différents, utilisez le plus petit des deux, en centrant la page dans la fenêtre dans l'autre dimension. |
| FitH | `2` | Affiche la page avec la coordonnée verticale top positionnée sur le bord supérieur de la fenêtre et le contenu de la page agrandi juste assez pour tenir sur toute la largeur de la page dans la fenêtre. Une valeur nulle pour top indique que la valeur actuelle de ce paramètre doit être conservé tel quel. |
| FitV | `3` | Affiche la page avec la coordonnée horizontale gauche positionnée sur le bord gauche de la fenêtre et le contenu de la page agrandi juste assez pour tenir sur toute la hauteur de la page dans la fenêtre. Une valeur nulle pour gauche indique que la valeur actuelle de ce paramètre doit être conservé tel quel. |
| FitR | `4` | Affichez la page avec son contenu agrandi juste assez pour tenir dans le rectangle spécifié par les coordonnées gauche, bas, droite et en haut entièrement dans la fenêtre à la fois horizontalement et verticalement. Si les facteurs d'agrandissement horizontaux et verticaux requis sont différents, utilisez le plus petit de les deux, en centrant le rectangle dans la fenêtre dans l'autre dimension. Une valeur nulle pour any des paramètres peut entraîner un comportement imprévisible. |
| FitB | `5` | Affichez la page avec son contenu agrandi juste assez pour tenir entièrement son cadre de délimitation dans la fenêtre à la fois horizontalement et verticalement. Si les facteurs d'agrandissement horizontal et vertical requis sont différents, utilisez le plus petit des deux, en centrant le cadre de délimitation dans la fenêtre dans l'autre dimension. |
| FitBH | `6` | Affiche la page avec la coordonnée verticale top positionnée sur le bord supérieur de la fenêtre et le contenu de la page agrandi juste assez pour tenir sur toute la largeur de son cadre de délimitation dans la fenêtre. Une valeur nulle pour top indique que la valeur actuelle de ce paramètre doit être conservé tel quel. |
| FitBV | `7` | Affiche la page avec la coordonnée horizontale gauche positionnée sur le bord gauche de la fenêtre et le contenu de la page agrandi juste assez pour tenir sur toute la hauteur de son cadre de délimitation dans la fenêtre. Une valeur nulle pour gauche spécifie que la valeur actuelle de ce paramètre doit être conservé tel quel. |
