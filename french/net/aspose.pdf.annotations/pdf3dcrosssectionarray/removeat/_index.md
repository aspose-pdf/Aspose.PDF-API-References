## Méthode PDF3DCrossSectionArray.RemoveAt

```csharp
public void RemoveAt(int index)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| index | Int32 | L'index de la coupe transversale supprimée dans le tableau. |

### Exceptions

| exception | condition |
| --- | --- |
