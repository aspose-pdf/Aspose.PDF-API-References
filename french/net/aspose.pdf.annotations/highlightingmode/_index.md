## HighlightingMode enumeration

Énumère le mode de surbrillance de l'annotation, l'effet visuel à utiliser lorsque le bouton de la souris est enfoncé ou maintenu enfoncé dans sa zone active.

```csharp
public enum HighlightingMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Pas de surbrillance. |
| Invert | `1` | Inverser le contenu du rectangle d'annotation. |
| Outline | `2` | Inverser la bordure de l'annotation. |
| Push | `3` | Affiche l'apparence vers le bas de l'annotation, le cas échéant. Si aucune apparence vers le bas n'est définie, décalez le contenu du rectangle d'annotation pour qu'il apparaisse comme s'il était poussé sous la surface de la page. |
| Toggle | `4` | Identique à Push (qui est préféré). |
