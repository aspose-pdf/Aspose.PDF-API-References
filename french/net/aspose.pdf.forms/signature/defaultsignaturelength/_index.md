## Propriété Signature.DefaultSignatureLength

Obtient ou définit la longueur par défaut des données de signature en octets.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Remarques

Ceci est une estimation de la longueur de la signature en octets. Utilisé pour signer via [`CustomSignHash`](../customsignhash/) si le paramètre [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) est défini. La valeur par défaut est 3000.
