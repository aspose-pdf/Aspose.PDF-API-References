## Signature.ShowProperties property

Forcer à afficher/masquer les propriétés de signature. Si ShowProperties est vrai, le champ de signature a un format d'apparence prédéfini (chaînes à représenter) : --------------------- ---------------------- Signé numériquement par {sujet du certificat} Date : {signature.Date} Raison : {signature.Reason} Emplacement : { signature.Emplacement} --------------------------------------------------------- où {X} est un espace réservé pour la valeur X. La signature peut également avoir une image, dans ce cas, les chaînes répertoriées sont placées sur l'image. ShowProperties est vrai par défaut.
