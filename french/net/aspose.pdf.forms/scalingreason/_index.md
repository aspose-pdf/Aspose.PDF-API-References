## Énumération ScalingReason

Les circonstances dans lesquelles l'icône doit être mise à l'échelle à l'intérieur du rectangle d'annotation.

```csharp
public enum ScalingReason
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Toujours | `0` | Toujours mettre à l'échelle. |
| IconIsBigger | `1` | Mettre à l'échelle uniquement lorsque l'icône est plus grande que le rectangle d'annotation. |
| IconIsSmaller | `2` | Mettre à l'échelle uniquement lorsque l'icône est plus petite que le rectangle d'annotation. |
| Never | `3` | Ne jamais mettre à l'échelle. |
