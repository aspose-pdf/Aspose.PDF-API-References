## Indexeur XFA

Obtient ou définit la valeur du nœud de données selon *le chemin*.

```csharp
public string this[string path] { get; set; }
```

| Paramètre | Description |
| --- | --- |
| chemin | Chemin du nœud de données, par exemple form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Assurez-vous d'inclure les indices même si les données ne contiennent qu'une seule occurrence de chaque nœud, c'est-à-dire écrivez node1[0].node2[0]... au lieu de node1.node2... |

### Valeur de retour

Valeur du nœud de données.
