## Propriété Form.AutoRecalculate

Si elle est définie, tous les champs du formulaire seront recalculés lorsque n'importe quel champ est modifié. La valeur par défaut est true. Définir sur false afin d'augmenter les performances lors du remplissage du formulaire avec un grand nombre de champs calculés.

```csharp
public bool AutoRecalculate { get; set; }
```
