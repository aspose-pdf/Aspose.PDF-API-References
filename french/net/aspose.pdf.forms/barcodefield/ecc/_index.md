## BarcodeField.ECC property

Obtient une valeur entière représentant le coefficient de correction d'erreur. Pour PDF417, doit être compris entre 0 et 8. Pour QRCode, doit être compris entre 0 et 3 (0 pour « L », 1 pour « M », 2 pour « Q » et 3 pour « H »).
