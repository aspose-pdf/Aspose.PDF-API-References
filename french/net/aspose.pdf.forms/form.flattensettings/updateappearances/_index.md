## Propriété Form.FlattenSettings.UpdateAppearances

Si elle est définie, toutes les apparences de champ seront régénérées avant le aplatissement. Cette option peut aider si le champ est incorrectement aplati. Cette option peut diminuer les performances. Par défaut, elle est définie sur false.

public bool UpdateAppearances { get; set; }
