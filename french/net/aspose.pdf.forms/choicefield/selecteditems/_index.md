## Propriété ChoiceField.SelectedItems

Obtient ou définit un tableau d'éléments sélectionnés. Pour une liste multisélection, le tableau contient plus d'un élément. Pour une liste à sélection unique, il contient un seul élément.

```csharp
public virtual int[] SelectedItems { get; set; }
```
