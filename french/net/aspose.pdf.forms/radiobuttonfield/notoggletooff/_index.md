## Propriété RadioButtonField.NoToggleToOff

Obtient ou définit le drapeau qui permet au bouton radio de n'avoir aucune valeur sélectionnée. Si `true`, exactement un bouton radio doit être sélectionné à tout moment ; sélectionner le bouton actuellement sélectionné n'a aucun effet. Si `false`, cliquer sur le bouton sélectionné le désélectionne, laissant aucun bouton sélectionné.

```csharp
public bool NoToggleToOff { get; set; }
```

## Remarques

Certains lecteurs PDF (y compris Adobe Acrobat) peuvent ignorer l'état `false` du drapeau.
