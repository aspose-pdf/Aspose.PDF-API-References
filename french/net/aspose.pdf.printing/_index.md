L'espace de noms **Aspose.Pdf.Printing** fournit des classes et des fonctionnalités pour gérer les opérations d'impression PDF, y compris les paramètres de l'imprimante, les paramètres de page et les extensions pour gérer les configurations liées à l'impression.

## Classes

| Classe | Description |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Fournit des données pour l'événement [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Spécifie les paramètres qui s'appliquent à une seule page imprimée. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Représente des méthodes d'extension pour [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Spécifie la taille d'une feuille de papier. |
| [PaperSizeExtensions](./papersizeextensions/) | Représente des méthodes d'extension pour [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Représente les tailles de papier standard. |
| [PaperSource](./papersource/) | Spécifie le bac à papier à partir duquel l'imprimante obtient le papier. |
| [PaperSourceExtensions](./papersourceextensions/) | Représente des méthodes d'extension pour [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Fournit un ensemble d'instances prédéfinies de [`PaperSource`](../aspose.pdf.printing/papersource/) représentant des sources de papier courantes. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Fournit des données pour l'événement [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Représente la résolution prise en charge par une imprimante. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Représente des méthodes d'extension pour [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Spécifie des informations sur la manière dont un document est imprimé, y compris l'imprimante qui l'imprime. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Représente des méthodes d'extension pour [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Fournit des données pour les événements [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) et [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) de la classe [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
