L'**espace de noms Aspose.Pdf.Structure** fournit des classes qui aident à travailler avec une structure logique d'un document qui présente l'organisation du document en chapitres et sections ou l'identification d'éléments spéciaux tels que des figures, des tableaux et des notes de bas de page.

| Classe | Description |
| --- | --- |
| [Element](./element/) | Classe représentant l'élément de base de la structure logique. |
| [ElementCollection](./elementcollection/) | Collection d'éléments de base de la structure logique. |
| [FigureElement](./figureelement/) | Classe représentant une figure de structure logique. |
| [RootElement](./rootelement/) | Élément de structure racine. |
| [StructElement](./structelement/) | Élément de structure général. |
| [TextElement](./textelement/) | Élément de texte général de la structure logique du document. |
