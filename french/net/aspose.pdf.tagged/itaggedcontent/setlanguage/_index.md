## Méthode ITaggedContent.SetLanguage

Définit la langue naturelle pour le document pdf.

Un identifiant de langue qui doit spécifier la langue naturelle pour tout le texte dans le document, sauf là où il est remplacé par des spécifications de langue pour les éléments de structure ou le contenu marqué.

```csharp
public void SetLanguage(string lang)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| lang | String | Un identifiant de langue doit soit être la chaîne de texte vide, pour indiquer que la langue est inconnue, soit un Language-Tag tel que défini dans la RFC 3066, Tags for the Identification of Languages. |
