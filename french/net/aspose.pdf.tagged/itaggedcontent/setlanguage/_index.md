## ITaggedContent.SetLanguage method

Définit le langage naturel pour le document pdf.

Un identificateur de langue qui doit spécifier la langue naturelle de tout le texte du document, sauf lorsqu'il est remplacé par des spécifications de langue pour les éléments de structure ou le contenu marqué.

```csharp
public void SetLanguage(string lang)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| lang | String | Un identifiant de langue doit être soit la chaîne de texte vide, pour indiquer que la langue est inconnue, soit une balise de langue telle que définie dans la RFC 3066, Balises pour l'identification des langues. |
