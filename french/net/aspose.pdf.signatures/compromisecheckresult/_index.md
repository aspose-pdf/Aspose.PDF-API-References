## Classe CompromiseCheckResult

Représente une classe pour vérifier les signatures numériques des documents pour compromission.


| Nom | Description |
| --- | --- |
| [HasCompromisedSignatures](../../aspose.pdf.signatures/compromisecheckresult/hascompromisedsignatures/) { get; } | Indique s'il y a des signatures numériques compromises dans le document. Renvoie true si au moins une signature est compromise ; sinon, false. |
| [SignaturesCoverage](../../aspose.pdf.signatures/compromisecheckresult/signaturescoverage/) { get; } | Obtient l'état de couverture des signatures numériques dans un document. Si elle est égale à Undefined, alors l'une des signatures est compromise. |
