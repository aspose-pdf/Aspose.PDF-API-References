## Champ ViewerPreference.HideWindowUI

Un indicateur spécifiant s'il faut masquer les éléments de l'interface utilisateur dans la fenêtre du document (tels que les barres de défilement et les contrôles de navigation), ne laissant afficher que le contenu du document.

public const int HideWindowUI;
