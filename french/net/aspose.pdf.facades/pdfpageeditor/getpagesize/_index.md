## PdfPageEditor.GetPageSize method

Renvoie la taille de page de la page spécifiée.

```csharp
public PageSize GetPageSize(int page)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| page | Int32 | Index des pages. Les pages du document sont numérotées à partir de 1. |

### Return_Value

Le résultat est une instance de PageSize. Utilisez les propriétés Width et Height de l'objet renvoyé pour obtenir la largeur et la hauteur de la page.

### Exemples

L'exemple suivant illustre l'utilisation de la méthode GetPageSize :

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
PageSize size = editor.GetPageSize(1);
Console.WriteLine("Size of 1st page : " + size.Width + " x " + size.Height);
```
