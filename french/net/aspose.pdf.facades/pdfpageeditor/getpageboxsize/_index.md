## PdfPageEditor.GetPageBoxSize method

Renvoie la taille de la zone spécifiée dans le document.

```csharp
public Rectangle GetPageBoxSize(int page, string pageBoxName)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| page | Int32 | Index des pages. Les pages du document sont numérotées à partir de 1. |
| pageBoxName | String | Nom du type de boîte. Les valeurs valides sont : "art", "bleed", "crop", "media", "trim". |

### Return_Value

Rectangle contenant la case demandée.

### Exemples

L'exemple suivant montre comment obtenir la boîte multimédia de la 1ère page :

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
System.Drawing.Rectangle rect = editor.GetBoxSize(1, "media");
```
