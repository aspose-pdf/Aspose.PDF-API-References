## PdfPageEditor.GetPageRotation method

Renvoie la rotation de la page spécifiée.

```csharp
public int GetPageRotation(int page)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| page | Int32 | Index des pages. Les pages du document sont numérotées à partir de 1. |

### Return_Value

Rotation des pages en degrés.

### Exemples

L'exemple suivant montre comment obtenir une rotation de page :

```csharp
PdfPageEditor editor = new PdfPageEditor();

int rotation = editor.GetPageSize(1);
Console.WriteLine("Rotation of 1st page : " + rotation + " degrees");
```
