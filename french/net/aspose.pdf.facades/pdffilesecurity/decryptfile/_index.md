## PdfFileSecurity.DecryptFile method

Décrypte un document PDF crypté par mot de passe propriétaire. Si le document n'a pas de mot de passe propriétaire, il est autorisé à utiliser le mot de passe utilisateur. Lance une exception si le processus a échoué.

```csharp
public bool DecryptFile(string ownerPassword)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| ownerPassword | String | Mot de passe propriétaire. |

### Return_Value

Vrai pour le succès.
