## PdfViewer.PrintDocumentWithSetup method

Imprime le document PDF avec une boîte de dialogue de configuration. Choisissez une imprimante à l'aide de la boîte de dialogue.

```csharp
public void PrintDocumentWithSetup()
```

### Exemples

```csharp
[C#]
dfViewer viewer = new PdfViewer();
iewer.OpenPdfFile(@"d:\test.pdf");
iewer.AutoResize = true;         //imprimer le fichier avec la taille ajustée
iewer.AutoRotate = true;         //imprime le fichier avec une rotation ajustée
iewer.PrintPageDialog=false;// ne produit pas la boîte de dialogue de numéro de page lors de l'impression
iewer.PrintDocumentWithSetup();
iewer.ClosePdfFile();
```
