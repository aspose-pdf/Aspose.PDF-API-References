## Événement PdfViewer.StartPage

Se produit avant qu'une page ne commence à imprimer.

```csharp
public event EventHandler<StartEndPageEventArgs> StartPage;
```

## Remarques

Cet événement est déclenché pour chaque page qui est sur le point d'être imprimée. Il peut être utilisé pour effectuer des actions personnalisées avant le début de l'impression de la page.
