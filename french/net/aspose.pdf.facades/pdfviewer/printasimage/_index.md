## PdfViewer.PrintAsImage property

Définit ou obtient un mode pour PdfViewer à imprimer en tant qu'image.

```csharp
public bool PrintAsImage { get; set; }
```

### Remarques

Si vrai imprime toujours en tant qu'image (génère une image qui est imprimée) Si faux imprime directement sur l'appareil si toutes les fonctionnalités sont prises en charge. Dans le cas où le document contient des fonctionnalités non prises en charge, le système peut décider automatiquement d'imprimer en tant qu'image. La valeur par défaut est false.
