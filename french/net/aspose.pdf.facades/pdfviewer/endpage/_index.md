## Événement PdfViewer.EndPage

Se produit lorsque l'impression d'une page se termine dans le PdfViewer.

```csharp
public event EventHandler<StartEndPageEventArgs> EndPage;
```

## Remarques

Cet événement est déclenché après qu'une page a été traitée dans le PdfViewer. Il peut être utilisé pour effectuer des actions ou des mises à jour après l'impression d'une page.
