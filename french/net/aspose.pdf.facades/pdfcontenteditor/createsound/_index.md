## Méthode PdfContentEditor.CreateSound

Crée des annotations sonores.

```csharp
public void CreateSound(Rectangle rect, string filePath, string name, int page, string rate)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| rect | Rectangle | Le rectangle d'annotation définissant l'emplacement de l'annotation sur la page. |
| filePath | String | Le chemin du fichier audio. |
| name | String | Le nom d'une icône à utiliser pour afficher l'annotation, inclut : Haut-parleur et Micro. |
| page | Int32 | La page sur laquelle l'annotation sonore est créée. |
| rate | String | La fréquence d'échantillonnage, en échantillons par seconde. |
