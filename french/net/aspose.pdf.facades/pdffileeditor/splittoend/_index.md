
| Paramètre | Taper | La description |
| --- | --- | --- |
| inputStream | Stream | Flux de fichiers source Pdf. |
| location | Int32 | La position de fendage. |
| outputStream | Stream | Flux de fichiers Pdf de sortie. |

### Return_Value

Vrai pour le succès, ou faux.

### Remarques

Les flux ne sont PAS fermés après cette opération, sauf si CloseConcatedStreams est spécifié.

### Exemples

```csharp
PdfFileEditor pfe = new PdfFileEditor();
Stream sourceStream = new FileStream("file1.pdf", FileMode.Open, FileAccess.Read);
Stream outStream = new FileStream("out.pdf", FileMode.Create, FileAccess.Write);
pfe.SplitToEnd(sourceStream, 5, outStream);
```
