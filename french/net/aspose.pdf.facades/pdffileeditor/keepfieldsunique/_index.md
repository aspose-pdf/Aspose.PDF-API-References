## Propriété PdfFileEditor.KeepFieldsUnique

Si vrai, les noms de champs seront rendus uniques lorsque les formulaires sont concaténés. Des suffixes seront ajoutés aux noms de champs, le modèle de suffixe peut être spécifié dans la propriété UniqueSuffix.

```csharp
public bool KeepFieldsUnique { get; set; }
```
