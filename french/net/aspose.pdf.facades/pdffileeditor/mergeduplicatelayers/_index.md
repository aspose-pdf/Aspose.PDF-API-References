## PdfFileEditor.MergeDuplicateLayers property

Le contenu facultatif des documents concaténés avec des noms égaux sera fusionné en une seule couche dans le document résultant si cette propriété est vraie. Sinon, les calques portant le même nom seront enregistrés en tant que calques différents dans le document résultant.
