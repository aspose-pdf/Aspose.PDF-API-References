## PdfFileSignature.GetSignNames method

Obtient les noms de toutes les signatures non vides.

```csharp
public IList<string> GetSignNames(bool onlyActive = true)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| onlyActive | Boolean | si vrai, renvoie uniquement les signatures actives ; sinon, retournez toutes les signatures. |

### Return_Value

Renvoie un IList&lt;string&gt;.

### Exemples

