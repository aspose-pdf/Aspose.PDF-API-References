## Méthode FormEditor.SetFieldScript

Définir JavaScript pour un champ PushButton. Si un ancien JavaScript existait, il sera remplacé par le nouveau.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| fieldName | String | Le nom de champ entièrement qualifié. |
| script | String | Le script Java à ajouter/placer dans un champ de bouton poussoir. |

### Valeur de retour

true si le script du champ a été défini avec succès.
