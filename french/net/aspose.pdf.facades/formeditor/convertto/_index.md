## Propriété FormEditor.ConvertTo

Définit le format de fichier PDF. Le fichier résultant sera enregistré dans le format de fichier spécifié. Si cette propriété n'est pas spécifiée, le fichier sera enregistré dans le format PDF par défaut sans conversion.

public PdfFormat ConvertTo { set; }
