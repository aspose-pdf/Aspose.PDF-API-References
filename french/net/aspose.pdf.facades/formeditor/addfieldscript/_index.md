## Méthode FormEditor.AddFieldScript

Ajouter du JavaScript pour un champ PushButton. Si un ancien événement existe, un nouvel événement est ajouté après.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| fieldName | String | Le nom de champ entièrement qualifié. |
| script | String | Le script Java à ajouter/placer dans un champ de bouton poussoir. |

### Valeur de retour

Vrai en cas de succès de l'ajout du script.
