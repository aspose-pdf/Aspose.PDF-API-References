## CopyInnerField(string, string, int) {#copyinnerfield}

Copie un champ existant à la même position dans le numéro de page spécifié. Un nouveau document sera produit, qui contiendra tout ce que le document source a à l'exception du champ nouvellement copié.

```csharp
public void CopyInnerField(string fieldName, string newFieldName, int pageNum)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| fieldName | String | L'ancien nom de champ complet. |
| newFieldName | String | Le nouveau nom de champ complet. Si null, il sera défini comme fieldName + "~". |
| pageNum | Int32 | Le nombre de pages pour contenir le nouveau champ. Si -1, le nouveau champ sera copié sur la même page que l'ancien hébergé. |

### Exemples

```csharp
FormEditor formEditor = new FormEditor("PdfForm.pdf", "PdfForm_out.pdf");
// Crée une copie du champ de texte sur la pseconde page.
formEditor.CopyInnerField("textField", "textFieldCopy", 2);
```

## CopyInnerField(string, string, int, float, float) {#copyinnerfield_1}

Copie un champ existant vers une nouvelle position spécifiée à la fois par le numéro de page et les ordonnées. Un nouveau document sera produit, qui contiendra tout ce que contient le document source à l'exception du champ nouvellement copié.

```csharp
public void CopyInnerField(string fieldName, string newFieldName, int pageNum, float abscissa, 
    float ordinate)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| fieldName | String | L'ancien nom de champ complet. |
| newFieldName | String | Le nouveau nom de champ complet. Si null, il sera défini comme fieldName + "~". |
| pageNum | Int32 | Le nombre de pages pour contenir le nouveau champ. Si -1, le nouveau champ sera copié sur la même page que l'ancien hébergé. |
| abscissa | Single | L'abscisse du nouveau champ. Si -1, l'abscisse sera égale à celle d'origine. |
| ordinate | Single | L'ordonnée du nouveau champ. Si -1, l'ordonnée sera égale à celle d'origine. |

### Exemples

```csharp
FormEditor formEditor = new FormEditor("PdfForm.pdf", "PdfForm_out.pdf");
// Crée une copie du champ de texte sur la pseconde page.
formEditor.CopyInnerField("textField", "textFieldCopy", 2, 100, 200);
```
