## Énumération DefaultMetadataProperties

Énumération des propriétés XMP standard.

```csharp
public enum DefaultMetadataProperties
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Advisory | `0` | Propriété xmp:Advisory. Un tableau non ordonné spécifiant les propriétés qui ont été modifiées en dehors de l'application d'édition. Chaque élément doit contenir un seul /// espace de noms et XPath séparés par un espace ASCII |
| BaseURL | `1` | Propriété xmp:BaseURL. L'URL de base pour les URL relatives dans le contenu du document. Si ce document contient des liens Internet, et que ces liens sont relatifs, /// ils sont relatifs à cette URL de base. Cette propriété fournit un moyen standard pour que les URL relatives intégrées soient interprétées par les outils. /// Les outils de création Web doivent définir la valeur en fonction de leur notion de l'endroit où les URL seront interprétées |
| CreateDate | `2` | Propriété xmp:CreateDate. La date et l'heure auxquelles la ressource a été initialement créée. |
| CreatorTool | `3` | Propriété xmp:CreatorTool. Le nom du premier outil connu utilisé pour créer la ressource. |
| Identifier | `4` | Propriété xmp:Identifier. Un tableau non ordonné de chaînes de texte qui identifient sans ambiguïté la ressource dans un contexte donné |
| MetadataDate | `5` | Propriété xmp:MetadataDate. La date et l'heure auxquelles les métadonnées de cette ressource ont été modifiées pour la dernière fois |
| ModifyDate | `6` | Propriété xmp:ModifyDate. La date et l'heure auxquelles la ressource a été modifiée pour la dernière fois. |
| Nickname | `7` | Propriété xmp:Nickname. Un nom informel court pour la ressource. |
| Thumbnails | `8` | Propriété xmp:Thumbnails. Un tableau alternatif d'images miniatures pour un fichier, qui peut différer par des caractéristiques telles que la taille ou le codage de l'image. |
