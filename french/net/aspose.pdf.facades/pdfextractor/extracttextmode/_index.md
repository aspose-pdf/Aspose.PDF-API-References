## PdfExtractor.ExtractTextMode property

Définit le mode d'extraction du résultat du texte.

```csharp
public int ExtractTextMode { get; set; }
```

### Valeur de la propriété

0 est le mode texte pur et 1 est le mode de commande brut. La valeur par défaut est 0.

### Exemples

L'exemple montre la`ExtractTextMode` utilisation de la propriété dans le scénario d'extraction de texte.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractTextMode = 1;
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```
