## Méthode PdfFileEditor.ContentsResizeParameters.ContentSizePercent

Crée des paramètres de redimensionnement avec une taille de contenu spécifiée en pourcentages de la taille de page initiale. Les marges sont calculées automatiquement.

```csharp
public static ContentsResizeParameters ContentSizePercent(double width, double height)
```

| Paramètre | Type | Description |
| --- | --- | --- |
| width | Double | Nouvelle largeur de contenu en pourcentages. |
| height | Double | Nouvelle hauteur de contenu en pourcentages. |

### Valeur de retour

Nouveaux paramètres de redimensionnement.
