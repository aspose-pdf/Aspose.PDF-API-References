## Properti PdfAOptionsBase.FontEmbeddingOptions

Mendapatkan opsi untuk memproses font yang tidak dapat disematkan ke dalam dokumen.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Nilai Properti

Opsi penyematan font.

## Catatan

Standar PDF/A mengharuskan bahwa semua font harus disematkan ke dalam dokumen. Properti ini menyediakan opsi untuk menangani kasus ketika tidak mungkin untuk menyematkan beberapa font karena tidak ada di PC tujuan.
