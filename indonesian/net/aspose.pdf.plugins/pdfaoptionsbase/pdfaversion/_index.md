## Properti PdfAOptionsBase.PdfAVersion

Mengambil atau mengatur versi standar PDF/A yang akan digunakan untuk validasi atau konversi.

```csharp
public PdfAStandardVersion PdfAVersion { get; set; }
```

### Nilai Properti

Versi standar PDF/A. Ini bisa menjadi salah satu nilai dari enumerasi [`PdfAStandardVersion`](../../pdfastandardversion/).

## Catatan

Versi standar PDF/A digunakan untuk menentukan tingkat kepatuhan untuk validasi dan konversi PDF/A. Jika versi diatur ke Auto, sistem akan secara otomatis menentukan versi standar PDF/A yang sesuai untuk validasi berdasarkan metadata dokumen. Untuk proses konversi PDF/A, Auto secara default menggunakan versi standar PDF/A-1b.
