## Properti PdfToXlsOptions.MinimizeTheNumberOfWorksheets

Atur true jika Anda perlu meminimalkan jumlah lembar kerja di buku kerja yang dihasilkan. Nilai default adalah false; ini berarti menyimpan setiap halaman PDF sebagai lembar kerja terpisah.

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
