## Properti PdfToXlsOptions.InsertBlankColumnAtFirst

Atur true jika Anda perlu menyisipkan kolom kosong sebagai kolom pertama dari lembar kerja. Nilai default adalah false; ini berarti kolom kosong tidak akan disisipkan.

```csharp
public bool InsertBlankColumnAtFirst { get; set; }
```
