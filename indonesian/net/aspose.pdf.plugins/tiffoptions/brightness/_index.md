## Properti TiffOptions.Brightness

Mendapatkan atau mengatur batas nilai dari transformasi warna menjadi putih dan hitam. Parameter ini dapat diterapkan dengan EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle atau ColorDepth.Format1bpp == 1

```csharp
public float Brightness { get; set; }
```

### Nilai Properti

Nilai kecerahan harus berada dalam rentang dari 0 hingga 1. Secara default, nilai sama dengan 0.33f
