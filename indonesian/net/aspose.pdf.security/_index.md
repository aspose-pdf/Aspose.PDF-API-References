Namespace **Aspose.Pdf.Security** berisi kelas-kelas yang digunakan untuk enkripsi dan penandatanganan digital.

## Kelas

| Kelas | Deskripsi |
| --- | --- |
| [DsaAlgorithmInfo](./dsaalgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan DSA. |
| [EcdsaAlgorithmInfo](./ecdsaalgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan ECDSA. |
| [KeyedSignatureAlgorithmInfo](./keyedsignaturealgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan yang menggunakan kunci. |
| [RsaAlgorithmInfo](./rsaalgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan RSA. |
| [SignatureAlgorithmInfo](./signaturealgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan, termasuk jenisnya, standar kriptografi, dan algoritma hash digest. |
| [SignatureLengthMismatchException](./signaturelengthmismatchexception/) | Mewakili kesalahan yang terjadi selama penandatanganan PDF. Terjadi jika [`SignHash`](../aspose.pdf.forms/signhash/) digunakan untuk menandatangani dokumen dan panjang aktual dari tanda tangan lebih besar dari yang ditentukan dalam opsi [`DefaultSignatureLength`](../aspose.pdf.forms/signature/defaultsignaturelength/). |
| [TimestampAlgorithmInfo](./timestampalgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan timestamp. |
| [UnknownSignatureAlgorithmInfo](./unknownsignaturealgorithminfo/) | Mewakili kelas untuk informasi tentang algoritma tanda tangan yang tidak dikenal. |
| [ValidationOptions](./validationoptions/) | Mewakili opsi untuk memvalidasi tanda tangan digital dalam dokumen PDF. |
| [ValidationResult](./validationresult/) | Mewakili hasil dari proses validasi untuk sebuah sertifikat. |
## Enumerasi

| Enumerasi | Deskripsi |
| --- | --- |

| [ValidationMethod](./validationmethod/) | Mewakili enum yang mendefinisikan metode yang digunakan untuk validasi sertifikat. |
| [ValidationMode](./validationmode/) | Menentukan mode validasi untuk proses validasi tanda tangan PDF. |
| [ValidationStatus](./validationstatus/) | Mewakili status validasi dari validasi sertifikat. |
