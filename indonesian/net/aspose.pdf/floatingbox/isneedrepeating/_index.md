## Properti FloatingBox.IsNeedRepeating

Mendapatkan atau mengatur nilai bool yang menunjukkan apakah paragraf perlu diulang di halaman berikutnya. Nilai default adalah false. Atribut ini hanya berlaku ketika paragraf itu sendiri dan objek yang dirujuk oleh ReferenceParagraphID keduanya termasuk dalam RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
