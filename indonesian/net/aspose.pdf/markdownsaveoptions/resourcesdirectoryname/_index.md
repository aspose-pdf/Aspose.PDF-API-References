## Properti MarkdownSaveOptions.ResourcesDirectoryName

Mendapatkan dan mengatur nama direktori untuk menyimpan sumber daya dokumen seperti gambar. Jika nilai tidak ditentukan, maka gambar akan ditulis ke direktori yang sama dengan file markdown itu sendiri. Ini bukan jalur, ini hanya nama! Direktori ini akan secara otomatis dibuat di direktori dengan file markdown yang disimpan.

```csharp
public string ResourcesDirectoryName { get; set; }
```
