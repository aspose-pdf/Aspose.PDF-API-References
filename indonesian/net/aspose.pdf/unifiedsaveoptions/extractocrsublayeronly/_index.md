## Properti UnifiedSaveOptions.ExtractOcrSublayerOnly

Atribut ini mengaktifkan fungsionalitas untuk mengekstrak gambar atau teks untuk dokumen PDF dengan sublayer OCR.

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### Nilai Properti

`true` teks akan diekstrak dalam dokumen hasil; jika tidak, `false`.
