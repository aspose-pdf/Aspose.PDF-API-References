## Enumerasi LoadOptions.MarginsAreaUsageModes

Mewakili mode penggunaan area margin selama konversi (seperti HTML, EPUB dll), mendefinisikan perlakuan instruksi format yang diimpor terkait dengan penggunaan margin.

```csharp
public enum MarginsAreaUsageModes
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| PutContentOnMarginAreaIfNecessary | `0` | Dalam mode ini, konverter mematuhi format dokumen yang diimpor (mis. CSS dari HTML yang diimpor) dalam penggunaan area margin. Jadi, jika format dokumen yang diimpor memerlukan penggunaan area margin untuk rendering, konverter akan mengizinkan itu |
| NeverPutContentOnMarginArea | `1` | Mode ini secara ketat melarang penggunaan area margin, jadi, konverter tidak akan pernah menggunakan area margin untuk rendering, bahkan jika CSS atau format dokumen sumber mengizinkan atau memerlukan itu |
