## Properti HtmlSaveOptions.ConvertMarkedContentToLayers

Jika atribut ConvertMarkedContentToLayers diatur ke true maka semua elemen di dalam konten yang ditandai PDF (lapisan) akan dimasukkan ke dalam div HTML dengan atribut "data-pdflayer" yang menentukan nama lapisan. Nama lapisan ini akan diambil dari properti opsional konten yang ditandai PDF. Jika atribut ini false (secara default) maka tidak ada lapisan yang akan dibuat dari konten yang ditandai PDF.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
