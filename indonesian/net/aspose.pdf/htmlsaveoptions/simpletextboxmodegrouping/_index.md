## Properti HtmlSaveOptions.SimpleTextboxModeGrouping

Atribut ini menentukan pengelompokan berurutan dari glyph dan kata menjadi string. Misalnya, tag dan kata memiliki urutan yang berbeda dalam HTML yang dikonversi dan Anda ingin mereka cocok. Parameter ini akan diterapkan pada dokumen hanya ketika nilai atribut FixedLayout adalah true.

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
