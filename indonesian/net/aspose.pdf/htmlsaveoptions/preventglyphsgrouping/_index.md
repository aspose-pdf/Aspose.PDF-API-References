## Properti HtmlSaveOptions.PreventGlyphsGrouping

Atribut ini mengaktifkan mode ketika glyph teks tidak akan dikelompokkan menjadi kata dan string. Mode ini memungkinkan untuk menjaga presisi maksimum selama penempatan glyph di halaman dan dapat digunakan untuk mengonversi dokumen dengan notasi musik atau glyph yang harus ditempatkan terpisah satu sama lain. Parameter ini akan diterapkan pada dokumen hanya ketika nilai atribut FixedLayout adalah true.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
