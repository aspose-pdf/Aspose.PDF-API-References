## Field HtmlSaveOptions.CssClassNamesPrefix

Ketika konverter PDFtoHTML menghasilkan CSS hasil, nama kelas CSS (sesuatu seperti ".stl_01 {}" ... ".stl_NN {}") dihasilkan dan digunakan dalam CSS hasil. Properti ini memungkinkan untuk memaksa menetapkan awalan nama kelas. Misalnya, jika Anda ingin semua nama kelas dimulai dengan 'my_prefix_' (yaitu seperti 'my_prefix_1' ... 'my_prefix_NNN'), maka cukup tetapkan 'my_prefix_' ke properti ini sebelum konversi. Jika properti ini dibiarkan tidak tersentuh (yaitu null akan dibiarkan sebagai nilai), maka konverter akan menghasilkan nama kelas sendiri (itu akan menjadi sesuatu seperti ".stl_01 {}" ... ".stl_NN {}")
