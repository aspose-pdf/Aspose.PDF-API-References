## Field HtmlSaveOptions.SpecialFolderForAllImages

Mendapatkan atau mengatur jalur ke direktori di mana semua gambar harus disimpan jika ditemukan selama penyimpanan dokumen sebagai HTML. Jika parameter kosong atau null, maka file gambar (jika ada) akan disimpan bersama dengan file lain yang terhubung ke HTML. Ini tidak mempengaruhi apa pun jika properti CustomImageSavingStrategy berhasil digunakan untuk memproses file gambar yang relevan.

```csharp
public string SpecialFolderForAllImages;
```
