## Properti HtmlSaveOptions.IgnoreResourceFontErrors

Mendapatkan atau mengatur indikasi bahwa kesalahan terkait dengan ketidakhadiran font akan diabaikan. true - berarti bahwa kesalahan ketidakhadiran font akan diabaikan. Segmen teks yang merujuk pada sumber daya yang salah akan dilewati selama pemrosesan. false secara default

public bool IgnoreResourceFontErrors { get; set; }
