## Properti HtmlSaveOptions.FontSources

Sumber font dari font yang telah disimpan sebelumnya.

```csharp
public FontSourceCollection FontSources { get; }
```

## Catatan

Font dapat disimpan secara preliminer untuk tujuan cache dan kemudian diteruskan ke proses konversi Html. Misalnya, ini mungkin berguna dalam skenario pemisahan dokumen dan memproses halaman dokumen dalam beberapa thread dengan satu set font tunggal.
