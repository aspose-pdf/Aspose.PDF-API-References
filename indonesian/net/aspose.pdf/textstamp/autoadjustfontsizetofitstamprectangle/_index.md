## Properti TextStamp.AutoAdjustFontSizeToFitStampRectangle

Jika diaktifkan, ukuran font akan secara otomatis disesuaikan untuk memenuhi persegi panjang stempel dengan ukuran: [`Width`](../width/) dan [`Height`](../height/). Lebar dan tinggi default diambil dari persegi panjang halaman.

```csharp
public bool AutoAdjustFontSizeToFitStampRectangle { get; set; }
```
