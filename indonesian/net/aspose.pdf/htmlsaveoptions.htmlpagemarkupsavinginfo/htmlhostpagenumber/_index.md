## HtmlSaveOptions.HtmlPageMarkupSavingInfo.HtmlHostPageNumber field

Diatur oleh konverter. Jika properti SplitToPages diatur, maka beberapa file HTML (satu file HTML per halaman yang dikonversi) dibuat selama proses konversi. Properti ini berisi urutan file halaman HTML yang disimpan. Properti ini dapat digunakan dalam logika kode kustom untuk memutuskan bagaimana memproses atau di mana menyimpan halaman HTML dan jika pemisahan pada halaman dimatikan, nilai ini selalu berisi '1' karena dalam kasus seperti itu hanya satu halaman HTML besar yang dihasilkan untuk seluruh dokumen sumber.

```csharp
public int HtmlHostPageNumber;
```
