## HtmlSaveOptions.HtmlPageMarkupSavingInfo.CustomProcessingCancelled field

Harus diatur dalam kode kustom jika perlu. Flag ini harus diatur ke "true" dalam kode kustom jika karena alasan tertentu html-markup yang disuplai harus diproses bukan dengan kode kustom tetapi dengan kode konverter itu sendiri dengan cara standar untuk konverter. Jadi, pengaturan jika flag ini dalam kode kustom berarti bahwa kode kustom tidak memproses file yang dirujuk dan konverter harus menanganinya sendiri

```csharp
public bool CustomProcessingCancelled;
```
