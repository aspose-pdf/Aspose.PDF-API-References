## Metode Metered.SetMeteredKey

Mengatur kunci publik dan privat terukur. Jika Anda membeli lisensi terukur, saat memulai aplikasi, API ini harus dipanggil, biasanya ini sudah cukup. Namun, jika selalu gagal mengunggah data konsumsi dan melebihi 24 jam, lisensi akan diatur ke status evaluasi, untuk menghindari kasus tersebut, Anda harus secara teratur memeriksa status lisensi, jika statusnya evaluasi, panggil API ini lagi.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| publicKey | String | kunci publik |
| privateKey | String | kunci privat |
