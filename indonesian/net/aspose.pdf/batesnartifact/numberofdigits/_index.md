## Properti BatesNArtifact.NumberOfDigits

Mendapatkan atau mengatur jumlah digit untuk penomoran Bates. Nilai harus antara 3 dan 15 termasuk. Jika nilai kurang dari 3 diatur, itu akan disesuaikan menjadi 3. Jika nilai lebih dari 15 diatur, itu akan disesuaikan menjadi 15. Nilai default adalah 6.

```csharp
public int NumberOfDigits { get; set; }
```
