## Properti ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace

Beberapa font tidak menyediakan informasi tentang unicode untuk beberapa simbol teks. Kekurangan informasi ini menyebabkan kesalahan "Teks tidak dapat dipetakan ke Unicode". Gunakan bendera ini untuk memetakan simbol yang tidak terhubung pada unicode "spasi" (kode 32).

public bool MapNonLinkedSymbolsOnSpace { get; set; }
