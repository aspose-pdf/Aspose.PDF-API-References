```csharp
public OutputIntent OutputIntent { get; set; }
```

## Catatan

`OutputIntent` menentukan perangkat atau kondisi keluaran yang dimaksudkan untuk dokumen PDF yang sedang disiapkan. Ini digunakan untuk memastikan bahwa warna dalam dokumen dirender dengan benar pada perangkat target.
