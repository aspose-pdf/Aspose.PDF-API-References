## Properti PdfFormatConversionOptions.OptimizeFileSize

Mendapatkan atau mengatur sebuah flag yang mengaktifkan/mematikan mode konversi khusus untuk mendapatkan dokumen PDF/A dengan ukuran file yang lebih kecil. Sekarang flag ini mempengaruhi optimasi font yang digunakan dalam dokumen PDF, kemungkinan, di masa depan, flag ini juga akan digunakan untuk mengaktifkan optimasi untuk struktur data lain, seperti grafik. Pengaturan flag dan mode ini dapat secara signifikan mengurangi ukuran file tetapi pada saat yang sama dapat secara signifikan mengurangi kinerja konversi.

```csharp
public bool OptimizeFileSize { get; set; }
```
