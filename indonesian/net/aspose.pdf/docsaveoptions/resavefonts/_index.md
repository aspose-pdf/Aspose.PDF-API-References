## Properti DocSaveOptions.ReSaveFonts

Mendapatkan atau mengatur prosedur untuk menyimpan ulang font. Jika diatur ke true, kami memuat ulang font di setiap halaman untuk menghindari pengaruh properti font sebelumnya dan memuat font yang baru dibuat dari awal. Atur opsi ini ke false jika Anda ingin meningkatkan kinerja. Nilai default adalah true;

```csharp
public bool ReSaveFonts { get; set; }
```
