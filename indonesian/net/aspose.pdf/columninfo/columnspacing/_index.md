## Properti ColumnInfo.ColumnSpacing

Mendapatkan atau mengatur string yang berisi jarak antara kolom. Nilai setiap jarak harus dipisahkan oleh spasi. Satuan default adalah poin, tetapi cm dan inci juga didukung. Misalnya, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Keterangan

Jika properti ini tidak diatur, nilai default 0 akan digunakan untuk setiap jarak.
