## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled field

Harus diatur oleh kode kustom jika tidak dapat atau tidak boleh mendefinisikan URL yang akan digunakan dalam HTML yang dihasilkan untuk referensi CSS tersebut. Jika 'true', maka file CSS akan disimpan dengan cara standar di tempat standar.

```csharp
public bool CustomProcessingCancelled;
```
