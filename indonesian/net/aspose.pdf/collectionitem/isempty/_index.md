## Properti CollectionItem.IsEmpty

Mendapatkan nilai yang menunjukkan apakah item koleksi kosong.

```csharp
public bool IsEmpty { get; }
```

### Nilai Kembali

true jika item koleksi kosong; jika tidak, false.

## Catatan

Properti ini mengembalikan true jika item koleksi tidak mengandung nilai apa pun, termasuk nilai string, nilai ganda, nilai integer, dan nilai tanggal. Jika salah satu dari tipe nilai ini ada dalam item koleksi, properti ini mengembalikan false.
