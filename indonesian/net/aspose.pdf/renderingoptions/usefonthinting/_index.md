## Properti RenderingOptions.UseFontHinting

Penggunaan bendera ini mengaktifkan mekanisme hinting font. Font hinting adalah penggunaan instruksi matematis untuk menyesuaikan tampilan font outline. Dalam beberapa kasus, mengaktifkan bendera ini dapat menyelesaikan masalah dengan keterbacaan teks. Saat ini, penggunaan bendera ini hanya dapat memberikan efek untuk font TTF, jika font ini digunakan dalam dokumen sumber.

```csharp
public bool UseFontHinting { get; set; }
```
