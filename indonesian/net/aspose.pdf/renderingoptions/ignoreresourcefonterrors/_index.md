## Properti RenderingOptions.IgnoreResourceFontErrors

Mendapatkan atau mengatur indikasi bahwa kesalahan terkait ketidakhadiran font akan diabaikan. true - berarti bahwa kesalahan ketidakhadiran font akan diabaikan. Segmen teks yang merujuk pada sumber daya yang salah akan dilewati selama pemrosesan. false secara default

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
