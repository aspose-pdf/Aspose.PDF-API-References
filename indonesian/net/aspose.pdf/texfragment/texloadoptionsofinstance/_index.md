## Properti TeXFragment.TeXLoadOptionsOfInstance

Mendapatkan atau mengatur TeXLoadOptions yang akan digunakan untuk memuat (dan merender) LaTeX ke dalam instance kelas ini. Harap gunakan ini ketika perlu menggunakan pengaturan spesifik untuk impor LaTeX untuk instance ini atau itu (misalnya ketika instance ini atau itu harus menggunakan BasePath spesifik untuk LaTeX yang diimpor atau harus menggunakan pemuat spesifik untuk sumber daya eksternal). Jika parameter adalah default (null), maka opsi pemuatan LaTeX standar akan digunakan.

```csharp
public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
```
