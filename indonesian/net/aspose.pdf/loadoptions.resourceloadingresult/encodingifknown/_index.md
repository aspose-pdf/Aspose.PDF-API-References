## LoadOptions.ResourceLoadingResult.EncodingIfKnown field

Terkadang encoding dari sumber diketahui setelah atau selama pemuatan. Dalam kasus seperti itu, kode kustom dapat memberikan konverter pengetahuan tersebut melalui parameter ini. Anda dapat membiarkan null pada parameter ini jika encoding tidak diketahui atau tidak penting.

```csharp
public Encoding EncodingIfKnown;
```
