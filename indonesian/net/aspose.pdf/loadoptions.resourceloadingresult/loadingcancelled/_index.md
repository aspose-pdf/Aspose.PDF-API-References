## LoadOptions.ResourceLoadingResult.LoadingCancelled field

Terkadang karena beberapa alasan, pemuatan tidak boleh terjadi pada kode kustom. Dalam hal ini, silakan atur bendera ini sebagai True. Dalam hal ini, konverter akan mencoba menggunakan pemuat sumber daya default internal untuk mendapatkan hasil tersebut (seperti yang terjadi dalam situasi ketika strategi kustom tidak disuplai).

```csharp
public bool LoadingCancelled;
```
