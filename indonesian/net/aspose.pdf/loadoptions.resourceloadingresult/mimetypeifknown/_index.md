## LoadOptions.ResourceLoadingResult.MIMETypeIfKnown field

Terkadang pengetahuan tentang tipe MIME dari sumber yang dimuat berguna untuk konverter. Anda dapat memberikan tipe MIME (jika diketahui setelah pemuatan) dalam parameter ini. Harap biarkan parameter sama dengan null ketika tipe MIME tidak diketahui atau tidak perlu untuk menyediakannya.

```csharp
public string MIMETypeIfKnown;
```
