## Properti Document.OptimizeSize

Mendapatkan atau mengatur bendera optimasi. Ketika halaman ditambahkan ke dokumen, aliran sumber yang sama dalam file hasil digabungkan menjadi satu objek PDF jika bendera ini diatur. Ini memungkinkan untuk mengurangi ukuran file hasil tetapi dapat menyebabkan eksekusi yang lebih lambat dan kebutuhan memori yang lebih besar. Nilai default: false.

```csharp
public bool OptimizeSize { get; set; }
```
