## Properti Document.AllowReusePageContent

Memungkinkan untuk menggabungkan konten halaman untuk mengoptimalkan ukuran dokumen. Jika digunakan, maka halaman yang berbeda tetapi duplikat dapat merujuk ke objek konten yang sama. Harap dicatat bahwa mode ini dapat menyebabkan efek samping seperti mengubah konten halaman ketika halaman lain diubah.

```csharp
public bool AllowReusePageContent { get; set; }
```
