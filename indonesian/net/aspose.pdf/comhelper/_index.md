## Catatan

Gunakan kelas ComHelper untuk memuat dokumen dari file atau stream ke dalam objek Dokumen dalam aplikasi COM. Kelas Dokumen menyediakan konstruktor default untuk membuat dokumen baru dan juga menyediakan konstruktor yang di-overload untuk memuat dokumen dari file atau stream. Jika Anda menggunakan Aspose.Words dari aplikasi .NET, Anda dapat menggunakan semua konstruktor Dokumen secara langsung, tetapi jika Anda menggunakan Aspose.Pdf dari aplikasi COM, hanya konstruktor Dokumen default yang tersedia.
