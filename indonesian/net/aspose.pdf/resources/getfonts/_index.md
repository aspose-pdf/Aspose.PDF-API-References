## Metode Resources.GetFonts

Mengembalikan koleksi font. Jika sumber tidak mengandung entri font, itu akan dibuat tergantung pada flag CreateIfAbsent.

```csharp
public FontCollection GetFonts(bool CreateIfAbsent)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| CreateIfAbsent | Boolean | Jika flag ini benar, maka font akan dibuat jika entri ini tidak ada. |
