## Properti XfaParserOptions.Signed

Jika properti ini benar, maka dokumen akan dikonversi dengan menggunakan aliran formulir xfa (jika ada). Jika salah, maka aliran formulir xfa akan diabaikan. Properti ini diperkenalkan karena tidak jelas bagaimana cara menghitung jumlah cek yang digunakan untuk memeriksa tanda tangan.

public bool Signed { get; set; }
