## Properti XfaParserOptions.EmulateRequierdGroups

Jika properti ini benar, maka persegi panjang merah tambahan akan digambar untuk "kelompok yang dikecualikan" Xfa yang diperlukan. Properti ini diperkenalkan karena tidak adanya analog kelompok yang dikecualikan selama konversi representasi Xfa dari formulir ke standar. Secara default, ini adalah salah.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
