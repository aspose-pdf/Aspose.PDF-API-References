Namespace **Aspose.Pdf.Printing** menyediakan kelas dan fungsionalitas untuk menangani operasi pencetakan PDF, termasuk pengaturan printer, pengaturan halaman, dan ekstensi untuk mengelola konfigurasi terkait pencetakan.

## Kelas

| Kelas | Deskripsi |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Menyediakan data untuk acara [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Menentukan pengaturan yang berlaku untuk satu halaman yang dicetak. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Mewakili metode ekstensi untuk [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Menentukan ukuran selembar kertas. |
| [PaperSizeExtensions](./papersizeextensions/) | Mewakili metode ekstensi untuk [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Mewakili ukuran kertas standar. |
| [PaperSource](./papersource/) | Menentukan baki kertas dari mana printer mengambil kertas. |
| [PaperSourceExtensions](./papersourceextensions/) | Mewakili metode ekstensi untuk [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Menyediakan sekumpulan instance [`PaperSource`](../aspose.pdf.printing/papersource/) yang telah ditentukan sebelumnya yang mewakili sumber kertas umum. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Menyediakan data untuk acara [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Mewakili resolusi yang didukung oleh printer. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Mewakili metode ekstensi untuk [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Menentukan informasi tentang bagaimana dokumen dicetak, termasuk printer yang mencetaknya. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Mewakili metode ekstensi untuk [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Menyediakan data untuk acara [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) dan [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) dari kelas [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
