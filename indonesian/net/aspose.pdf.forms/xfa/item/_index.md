## Indeks XFA

Mendapatkan atau mengatur nilai node data sesuai dengan *jalur*.

```csharp
public string this[string path] { get; set; }
```

| Parameter | Deskripsi |
| --- | --- |
| jalur | Jalur node data, misalnya form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Pastikan untuk menyertakan indeks meskipun data hanya berisi satu kemunculan dari setiap node, yaitu tulis node1[0].node2[0]... alih-alih node1.node2... |
