## Properti RadioButtonField.NoToggleToOff

Mengambil atau mengatur bendera yang memungkinkan radiobutton tidak memiliki nilai yang dipilih. Jika `true`, tepat satu tombol radio harus dipilih setiap saat; memilih tombol yang saat ini dipilih tidak memiliki efek. Jika `false`, mengklik tombol yang dipilih akan membatalkan pemilihannya, meninggalkan tidak ada tombol yang dipilih.

```csharp
public bool NoToggleToOff { get; set; }
```

## Catatan

Beberapa pembaca PDF (termasuk Adobe Acrobat) mungkin mengabaikan status `false` dari bendera tersebut.
