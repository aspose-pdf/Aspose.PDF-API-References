## Properti SvgExtractionOptions.GroupStrength

Mendapatkan dan mengatur opsi Kekuatan pengelompokan subpath menjadi gambar. Memungkinkan Anda untuk mengonfigurasi tingkat pengelompokan subpath. Rentang nilai adalah dari 0 hingga 1. Nilai 0 sesuai dengan opsi [`ExtractEverySubPathToSvg`](../extracteverysubpathtosvg/) yang diaktifkan. Nilai 1 akan membuat satu gambar untuk semua jalur vektor di halaman. Opsi ini berpengaruh ketika [`AutoGrouping`](../autogrouping/) adalah false. Nilai default adalah `0.8`.

```csharp
public double GroupStrength { get; set; }
```
