## Properti DocumentPrivilege.PrintAllowLevel

Mendapatkan dan mengatur tingkat cetak dari hak dokumen. Sama seperti pengaturan Printing Allowed Adobe Professional. 0: Tidak ada. 1: Resolusi Rendah (150 dpi). 2: Resolusi Tinggi.

```csharp
public int PrintAllowLevel { get; set; }
```

## Catatan

Jika properti memiliki nilai -1, maka tingkatnya tidak terdefinisi.
