## Properti DocumentPrivilege.ChangeAllowLevel

Mengambil dan mengatur tingkat perubahan hak dokumen. Sama seperti pengaturan Changes Allowed di Adobe Professional. 0: Tidak ada. 1: Menyisipkan, Menghapus, dan Memutar halaman. 2: Mengisi bidang formulir dan menandatangani bidang tanda tangan yang ada. 3: Mengomentari, mengisi bidang formulir, dan menandatangani bidang tanda tangan yang ada. 4: Apa pun kecuali mengekstrak halaman.

```csharp
public int ChangeAllowLevel { get; set; }
```

## Catatan

Jika properti memiliki nilai -1, maka tingkatnya tidak terdefinisi.
