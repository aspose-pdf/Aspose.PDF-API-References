## Properti DocumentPrivilege.CopyAllowLevel

Mengambil dan mengatur tingkat salinan hak dokumen. Sama seperti pengaturan izin Adobe Professional. 0: Tidak ada. 1: Mengaktifkan akses teks untuk perangkat pembaca layar bagi penyandang disabilitas visual. 2: Mengaktifkan penyalinan teks, gambar, dan konten lainnya.

```csharp
public int CopyAllowLevel { get; set; }
```

## Catatan

Jika properti memiliki nilai -1, maka tingkatnya tidak terdefinisi.
