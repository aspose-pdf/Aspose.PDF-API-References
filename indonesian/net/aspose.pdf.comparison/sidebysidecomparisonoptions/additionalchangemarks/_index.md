## Properti SideBySideComparisonOptions.AdditionalChangeMarks

Dapatkan dan atur properti yang menentukan apakah penanda perubahan tambahan ditampilkan. Jika diatur, menampilkan tanda perubahan yang tidak ada di halaman saat ini tetapi ada di halaman lain. Jika perubahan terletak di antara kata-kata, tanda tersebut mungkin tidak diposisikan dengan tepat relatif terhadap karakter spasi. Nilai default adalah `false`.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
