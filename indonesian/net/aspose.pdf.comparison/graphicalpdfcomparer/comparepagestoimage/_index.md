## Metode GraphicalPdfComparer.ComparePagesToImage

Membandingkan halaman secara grafis. Hasil perbandingan ditempatkan dalam sebuah gambar.

```csharp
public void ComparePagesToImage(Page page1, Page page2, string resultImagePath)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| page1 | Page | Halaman pertama untuk dibandingkan. |
| page2 | Page | Halaman kedua untuk dibandingkan. |
| resultImagePath | String | Jalur ke file gambar target. |

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentException | Jika halaman yang dibandingkan memiliki ukuran yang berbeda. Jika resultImagePath adalah null atau string kosong. Ada format penyimpanan gambar yang tidak diketahui. |
