## Kelas FitHExplicitDestination

Mewakili tujuan eksplisit yang menampilkan halaman dengan koordinat vertikal atas yang diposisikan di tepi atas jendela dan konten halaman diperbesar cukup untuk menyesuaikan seluruh lebar halaman dalam jendela. Nilai null untuk atas menunjukkan bahwa nilai saat ini dari parameter tersebut akan dipertahankan tanpa perubahan.

```csharp
public sealed class FitHExplicitDestination : ExplicitDestination
```
