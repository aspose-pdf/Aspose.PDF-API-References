## Properti RunThreadCreateRequest.MaxPromptTokens

Mengambil atau mengatur jumlah maksimum token prompt yang dapat digunakan selama proses berjalan. Proses akan berusaha sebaik mungkin untuk hanya menggunakan jumlah token prompt yang ditentukan, di berbagai giliran proses. Jika proses melebihi jumlah token prompt yang ditentukan, proses akan berakhir dengan status tidak lengkap. Lihat incomplete_details untuk informasi lebih lanjut.
