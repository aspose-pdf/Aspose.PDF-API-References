## Properti CreateEmbeddingRequest.Input

Mendapatkan atau mengatur teks input untuk disematkan, yang dikodekan sebagai string atau array token. Untuk menyematkan beberapa input dalam satu permintaan, kirimkan array string atau array array token. Input tidak boleh melebihi token input maksimum untuk model (8192 token untuk text-embedding-ada-002), tidak boleh berupa string kosong, dan setiap array harus memiliki dimensi 2048 atau kurang.

```csharp
public string Input { get; set; }
```
