## Field StructureTypeStandard.TBody

(Grupp baris tubuh tabel; PDF 1.5) Sebuah grup baris yang merupakan bagian tubuh utama dari sebuah tabel. Jika tabel dibagi di beberapa halaman, area tubuh dapat terpisah pada batas baris. Sebuah tabel dapat memiliki beberapa elemen TBody untuk memungkinkan penggambaran batas atau latar belakang untuk satu set baris.

```csharp
public static readonly StructureTypeStandard TBody;
```
