## Indeks ElementList

Mendapatkan elemen pada indeks yang diberikan.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Parameter | Deskripsi |
| --- | --- |
| index | Indeks ke dalam daftar elemen. |

### Nilai Kembali

Elemen dengan indeks yang ditentukan dalam koleksi. Jika indeks lebih besar dari atau sama dengan jumlah elemen dalam daftar, ini mengembalikan null.
