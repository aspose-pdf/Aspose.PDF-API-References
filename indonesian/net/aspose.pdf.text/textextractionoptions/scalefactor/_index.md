## Properti TextExtractionOptions.ScaleFactor

Mendapatkan atau mengatur faktor yang akan diterapkan untuk mengubah ukuran font selama ekstraksi dalam mode murni. Penetapan nilai yang lebih rendah mengarah pada lebih banyak spasi dalam teks yang diekstrak. Nilai default adalah 1 - tanpa skala; Menetapkan nilai menjadi nol memungkinkan algoritma memilih skala secara otomatis.

```csharp
public double ScaleFactor { get; set; }
```
