## Metode FontRepository.LoadFonts

Memuat font yang diinstal di sistem dan font Pdf standar. Metode ini dirancang untuk mempercepat proses pemuatan font. Secara default, font dimuat pada permintaan pertama untuk font apa pun. Penggunaan metode ini memuat font sistem dan font Pdf standar segera sebelum dokumen Pdf dibuka.

public static void LoadFonts()
