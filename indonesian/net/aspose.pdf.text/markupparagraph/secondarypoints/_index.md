## Properti MarkupParagraph.SecondaryPoints

Titik-titik poligon sekunder menggambarkan kelanjutan paragraf. Ini tidak akan null jika paragraf dilanjutkan di kolom atau halaman berikutnya. Titik awal adalah sudut kiri bawah paragraf. Dan titik-titik berikutnya berada dalam urutan berlawanan arah jarum jam.

public List<Point[]> SecondaryPoints { get; }
