## Properti TextSearchOptions.UseFontEngineEncoding

Mendapatkan atau mengatur indikasi bahwa teks akan dicari menggunakan pengkodean mesin font. true - berarti bahwa pengkodean mesin font akan digunakan (coba ini jika pencarian teks gagal karena pengkodean yang tidak sempurna dalam dokumen) false - berarti bahwa pengkodean font dokumen akan digunakan (nilai default)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
