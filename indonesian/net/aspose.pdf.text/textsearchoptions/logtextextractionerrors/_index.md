## Properti TextSearchOptions.LogTextExtractionErrors

Mendapatkan atau mengatur indikasi bahwa kesalahan ekstraksi (dekode) teks akan dicatat di penyerap teks (fragmen). true - berarti bahwa kesalahan ekstraksi (dekode) teks akan dicatat. Ini dapat mengurangi kinerja. false (default) - tidak ada pencatatan kesalahan.

```csharp
public bool LogTextExtractionErrors { get; set; }
```
