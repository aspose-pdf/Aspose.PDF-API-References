## Enumerasi TextRenderingMode

Mode rendering teks, Tmode, menentukan apakah menampilkan teks akan menyebabkan garis besar glyph di-stroke, diisi, digunakan sebagai batas pemotongan, atau kombinasi dari ketiganya.

```csharp
public enum TextRenderingMode
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| FillText | `0` | Isi teks. |
| StrokeText | `1` | Garis teks. |
| FillThenStrokeText | `2` | Isi, lalu garis teks. |
| Invisible | `3` | Tidak mengisi atau menggambar teks (tidak terlihat). |
| FillTextAndAddPathToClipping | `4` | Isi teks dan tambahkan ke jalur untuk pemotongan (lihat 9.3.6, "Mode Rendering Teks,"). |
| StrokeTextAndAddPathToClipping | `5` | Garis teks dan tambahkan ke jalur untuk pemotongan. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Isi, lalu garis teks dan tambahkan ke jalur untuk pemotongan. |
| AddPathToClipping | `7` | Tambahkan teks ke jalur untuk pemotongan. |
