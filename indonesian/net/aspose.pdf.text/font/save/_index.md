## Metode Font.Save

Menyimpan font ke dalam aliran. Perhatikan bahwa font disimpan dalam format TTF sementara yang dimaksudkan untuk digunakan dalam salinan yang dikonversi dari dokumen asli saja. File font tidak dimaksudkan untuk digunakan di luar konteks dokumen asli.

```csharp
public void Save(Stream stream)
```

| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| stream | Stream | Aliran untuk menyimpan font. |
