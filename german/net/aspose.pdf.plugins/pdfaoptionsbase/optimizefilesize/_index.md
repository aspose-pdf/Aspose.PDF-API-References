## PdfAOptionsBase.OptimizeFileSize-Eigenschaft

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob versucht werden soll, die Dateigröße während des PDF/A-Konvertierungsprozesses zu reduzieren.

```csharp
public bool OptimizeFileSize { get; set; }
```

### Eigenschaftswert

`true`, wenn die Dateigröße so weit wie möglich reduziert werden soll; andernfalls `false`.

## Anmerkungen

Wenn auf `true` gesetzt, wird der Konvertierungsprozess versuchen, die resultierende Dateigröße zu minimieren. Dies könnte die Leistung des Konvertierungsprozesses beeinträchtigen.
