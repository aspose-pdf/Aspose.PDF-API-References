## PdfAOptionsBase.IsLowMemoryMode-Eigenschaft

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob der Energiesparmodus während des PDF/A-Konvertierungsprozesses aktiviert ist.

```csharp
public bool IsLowMemoryMode { get; set; }
```

### Eigenschaftswert

`true`, wenn der Energiesparmodus aktiviert ist; andernfalls `false`.
