## PdfAOptionsBase.SymbolicFontEncodingStrategy-Eigenschaft

Ruft die Strategie zum Kodieren symbolischer Schriftarten beim Konvertieren in das PDF/A-Format ab oder legt sie fest.

```csharp
public PdfASymbolicFontEncodingStrategy SymbolicFontEncodingStrategy { get; set; }
```

### Eigenschaftswert

Die Strategie zur Kodierung symbolischer Schriftarten.

## Anmerkungen

Diese Eigenschaft ermöglicht es Ihnen, zu steuern, welche CMap-Subtabelle in das Ergebnisdokument kopiert wird, wenn die ursprüngliche TrueType-symbole Schriftart im Quelldokument mehrere CMap-Subtabellen enthält.
