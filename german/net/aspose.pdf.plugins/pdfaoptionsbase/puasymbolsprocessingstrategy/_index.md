## PdfAOptionsBase.PuaSymbolsProcessingStrategy-Eigenschaft

Ruft die Strategie zum Verarbeiten von Symbolen im privaten Nutzungsbereich (PUA) im PDF-Dokument ab oder legt sie fest.

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### Eigenschaftswert

Die Strategie zum Verarbeiten von PUA-Symbolen. Der Standardwert ist None für die Level B-Konformitätsdokumente und SurroundPuaTextWithEmptyActualText für die Level A-Konformitätsdokumente.
