## PdfAOptionsBase.UnicodeProcessingRules-Eigenschaft

Ruft die Regeln für die Verarbeitung von ToUnicode CMap-Tabellen ab oder legt sie fest, die während des PDF/A-Konvertierungsprozesses nicht mit Unicode-Symbolen verknüpft sind.

```csharp
public ToUnicodeProcessingRules UnicodeProcessingRules { get; set; }
```

### Eigenschaftswert

Eine Instanz von [`ToUnicodeProcessingRules`](../../../aspose.pdf/tounicodeprocessingrules/), die die Regeln für den Umgang mit problematischen Fällen von Unicode-Zeichen definiert.
