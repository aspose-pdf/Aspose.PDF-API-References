## PdfToDocOptions.ConversionMode-Eigenschaft

Ermöglicht die Steuerung, wie ein PDF-Dokument in ein Textverarbeitungsdokument konvertiert wird.

```csharp
public ConversionMode ConversionMode { get; set; }
```

## Hinweise

Verwenden Sie den TextBox-Modus, wenn das resultierende Dokument nicht weiter stark bearbeitet werden soll. Textfelder sind einfach zu ändern, wenn nicht viel zu tun ist.

Verwenden Sie den Flow-Modus, wenn das Ausgabedokument weitere Bearbeitungen benötigt. Absätze und Textzeilen im Flow-Modus ermöglichen eine einfache Modifikation des Textes, aber nicht unterstützte Formatierungsobjekte sehen im Vergleich zum TextBox-Modus schlechter aus.
