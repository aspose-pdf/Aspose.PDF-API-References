## TextExtractorOptions.TextFormattingMode Aufzählung

Definiert verschiedene Modi, die beim Konvertieren eines PDF-Dokuments in Text verwendet werden können. Siehe [`TextExtractorOptions`](../textextractoroptions/) Klasse.

```csharp
public enum TextFormattingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Pure | `0` | Stellt den PDF-Textinhalt mit einigen Formatierungsroutinen dar. Dies umfasst die Berücksichtigung relativer Positionen und das Hinzufügen zusätzlicher Abstände, um den Text an die Breite der Seite auszurichten. |
| Raw | `1` | Stellt den PDF-Inhalt so dar, wie er ist, d.h. ohne Formatierung. |
| Plain | `2` | Stellt den PDF-Textinhalt unter Berücksichtigung der relativen Positionierung von Textfragmenten dar, jedoch ohne zusätzliche Abstände hinzuzufügen. |
