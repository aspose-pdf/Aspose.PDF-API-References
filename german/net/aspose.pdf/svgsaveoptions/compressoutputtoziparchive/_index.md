## SvgSaveOptions.CompressOutputToZipArchive-Feld

Gibt an, ob die Ausgabe als ein Zip-Archiv erstellt wird. Bitte beachten Sie den Kommentar zu den Optionen 'TreatTargetFileNameAsDirectory', um die Regeln zur Benennung der SVG-Dateien von Seiten für mehrseitige Quelldokumente zu sehen, die auch auf das gezippte Set von Ausgabedateien angewendet werden.

```csharp
public bool CompressOutputToZipArchive;
```
