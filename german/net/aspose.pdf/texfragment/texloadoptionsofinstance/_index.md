## TeXFragment.TeXLoadOptionsOfInstance property

Erhält oder setzt TeXLoadOptions, die zum Laden (und Rendern) von LaTeX in diese Instanz der Klasse verwendet werden. Bitte verwenden Sie es, wenn es notwendig ist, verwenden Sie spezifische Einstellungen für den Import von LaTeX für diese oder jene Instanz (z. B. wenn diese oder jene Instanz Verwenden Sie einen bestimmten BasePath für importiertes LaTeX oder sollten Sie einen bestimmten Loader für externe Ressourcen verwenden) Wenn der Parameter standardmäßig (null) ist, werden die LaTeX-Standardladeoptionen verwendet.
