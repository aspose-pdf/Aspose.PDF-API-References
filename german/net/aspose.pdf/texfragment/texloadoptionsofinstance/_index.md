## TeXFragment.TeXLoadOptionsOfInstance-Eigenschaft

Ruft die TeXLoadOptions ab oder legt sie fest, die zum Laden (und Rendern) von LaTeX in dieser Instanz der Klasse verwendet werden. Bitte verwenden Sie es, wenn es notwendig ist, spezifische Einstellungen für den Import von LaTeX für diese oder jene Instanz zu verwenden (z. B. wenn diese oder jene Instanz einen spezifischen BasePath für importiertes LaTeX verwenden oder einen spezifischen Loader für externe Ressourcen verwenden sollte). Wenn der Parameter standardmäßig (null) ist, werden die standardmäßigen LaTeX-Ladeoptionen verwendet.

```csharp
public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
```
