## OutlineCollection.VisibleCount-Eigenschaft

Count ist die Summe der Anzahl sichtbarer nachfolgender Gliederungselemente auf allen Ebenen. Hinweis: Bitte verwechseln Sie dies nicht mit Count, das die Anzahl der Elemente in der Sammlung ist.

public override int VisibleCount { get; }
