## FontSubsetStrategy-Enumeration

enumeriert Strategien für das Subsetting von Schriftarten

```csharp
public enum FontSubsetStrategy : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SubsetEmbeddedFontsOnly | `0` | Deklariert, dass nur Schriftarten, die bereits im Dokument eingebettet sind, subsettiert werden |
| SubsetAllFonts | `1` | Deklariert, dass alle Schriftarten, sowohl nicht eingebettete als auch eingebettete, subsettiert werden |
