## DocumentExtensions.SplitSharedImages-Methode

Für Bilder in Ressourcen überprüft sie, ob zwei Seiten gemeinsame XImages haben, und trennt sie in ähnlichen Fällen, indem sie doppelte XImages erstellt.

```csharp
public static void SplitSharedImages(this Document doc, Page page_1, Page page_2)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| doc | Document | Das Dokument, das beide Sammlungen enthält. |
| page_1 | Page | Erste Seite zum Vergleichen. |
| page_2 | Page | Zweite Seite zum Vergleichen. |
