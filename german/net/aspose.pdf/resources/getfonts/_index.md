## Resources.GetFonts-Methode

Gibt die Schriftartenkollektion zurück. Wenn die Ressourcen keinen Schriftarteintrag enthalten, wird dieser abhängig vom CreateIfAbsent-Flag erstellt.

```csharp
public FontCollection GetFonts(bool CreateIfAbsent)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| CreateIfAbsent | Boolean | Wenn dieses Flag wahr ist, werden Schriftarten erstellt, wenn dieser Eintrag fehlt. |

### Rückgabewert

Schriftartenkollektion.
