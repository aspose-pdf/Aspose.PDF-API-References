## HeadingRecognitionStrategy Enumeration

Stellt Typen von Header-Erkennungsstrategien dar.

```csharp
public enum HeadingRecognitionStrategy
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Outlines | `0` | Stellt die Header-Erkennungsstrategie durch Mittel der Gliederungen dar. |
| Heuristic | `1` | Stellt die Header-Erkennungsstrategie durch Mittel von heuristischen Regeln und Schriftgrößenstatistiken dar. |
| Auto | `2` | Bietet eine automatische Auswahl der Header-Erkennungsstrategie. Dies ist die Standardoption. Wenn das Dokument Lesezeichen enthält, wird die Outlines-Strategie ausgewählt, andernfalls Heuristic. |
| None | `3` | Erkenne keine Header. Diese Option kann in komplex formatierten Dokumenten nützlich sein. |
