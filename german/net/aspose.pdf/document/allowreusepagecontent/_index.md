## Document.AllowReusePageContent-Eigenschaft

Ermöglicht das Zusammenführen von Seiteninhalten zur Optimierung der Dokumentgröße. Wenn verwendet, können unterschiedliche, aber duplizierte Seiten auf dasselbe Inhaltsobjekt verweisen. Bitte beachten Sie, dass dieser Modus Nebenwirkungen verursachen kann, wie z.B. das Ändern des Seiteninhalts, wenn eine andere Seite geändert wird.

```csharp
public bool AllowReusePageContent { get; set; }
```
