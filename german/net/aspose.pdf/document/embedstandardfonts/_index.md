## Document.EmbedStandardFonts-Eigenschaft

Eigenschaft, die erklärt, dass das Dokument alle Standard-Type1-Schriften einbetten muss, die das Flag IsEmbedded auf true gesetzt haben. Alle PDF-Schriften können einfach durch Setzen des Flags IsEmbedded auf true in das Dokument eingebettet werden, aber PDF-Standard-Type1-Schriften sind eine Ausnahme von dieser Regel. Das Einbetten von Standard-Type1-Schriften erfordert viel Zeit, daher ist es notwendig, nicht nur das Flag IsEmbedded für die angegebene Schrift auf true zu setzen, sondern auch ein zusätzliches Flag auf Dokumentenebene - EmbedStandardFonts = true; Diese Eigenschaft kann nur einmal für alle Schriften gesetzt werden. Standardmäßig false.

```csharp
public bool EmbedStandardFonts { get; set; }
```
