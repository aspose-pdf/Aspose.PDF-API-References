## Document.HideWindowUI-Eigenschaft

Ruft das Flag ab oder legt es fest, das angibt, ob Benutzerschnittstellenelemente ausgeblendet werden sollen, wenn das Dokument aktiv ist.

```csharp
public bool HideWindowUI { get; set; }
```

Das Beispiel zeigt, wie man das HideWindowUI-Flag abruft:

