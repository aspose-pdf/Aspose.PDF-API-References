## Document.DisableFontLicenseVerifications property

Viele Operationen mit Schriftarten können nicht ausgeführt werden, wenn diese Operationen durch die Lizenz dieser Schriftart verboten sind. Einige Schriftarten können beispielsweise nicht in ein PDF-Dokument eingebettet werden, wenn die Lizenzregeln das Einbetten für diese Schriftart deaktivieren. Dieses Flag wird verwendet, um alle Lizenzbeschränkungen für alle Schriftarten im aktuellen PDF-Dokument zu deaktivieren. Seien Sie vorsichtig, wenn Sie dieses Flag verwenden. Wenn es gesetzt ist, bedeutet dies, dass die Person, die dieses Flag setzt, die gesamte Verantwortung für mögliche Lizenz-/Gesetzesverstöße auf sich trägt. Also übernimmt er es auf eigenes Risiko. Es wird dringend empfohlen, dieses Flag nur zu verwenden, wenn Sie absolut sicher sind, dass Sie nicht gegen das Urheberrechtsgesetz verstoßen. Standardmäßig falsch.

public bool DisableFontLicenseVerifications { get; set; }
