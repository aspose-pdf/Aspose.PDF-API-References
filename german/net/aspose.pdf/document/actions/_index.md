## Document.Actions property

Ruft Dokumentaktionen ab. Diese Eigenschaft ist eine Instanz der DocumentActions-Klasse, die es ermöglicht, BeforClosing-, BeforSaving- usw. Aktionen zu erhalten/einzustellen.

```csharp
public DocumentActionCollection Actions { get; }
```

Dieses Beispiel zeigt, wie man nach der Öffnungsaktion des Dokuments erhält:

```csharp
Aspose.Pdf.Document document = new Aspose.Pdf.Document("d:\\work\\aspose\\aspose.pdf.kit.net.new\\trunk\\testdata\\Aspose.Pdf\\PdfWithOpenAction.pdf");
Aspose.Pdf.Annotations.DocumentActionCollection actions = document.Actions;
Aspose.Pdf.Annotations.PdfAction afterSavingAction = actions.AfterSaving;
```
