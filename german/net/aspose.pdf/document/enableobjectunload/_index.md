## Document.EnableObjectUnload-Eigenschaft

Legt ein Flag fest, das es ermöglicht, das Dokument teilweise aus dem Speicher zu entladen. Dies ermöglicht eine Verringerung des Speicherverbrauchs, kann jedoch negative Auswirkungen auf die Leistung haben.

```csharp
public bool EnableObjectUnload { get; set; }
```
