## FloatingBox.IsNeedRepeating-Eigenschaft

Ruft einen booleschen Wert ab oder legt ihn fest, der angibt, ob der Absatz auf der nächsten Seite wiederholt werden muss. Der Standardwert ist false. Das Attribut ist nur gültig, wenn der Absatz selbst und das Objekt, auf das sich seine ReferenceParagraphID bezieht, beide in RepeatingRows enthalten sind.

```csharp
public bool IsNeedRepeating { get; set; }
```
