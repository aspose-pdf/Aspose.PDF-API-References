## HtmlLoadOptions.InputEncoding-Eigenschaft

Ruft das Attribut ab oder legt es fest, das die Kodierung angibt, die für dieses Dokument zum Zeitpunkt des Parsens verwendet wird. Wenn dieses Attribut null ist, wird die Kodierung aus dem Zeichensatzattribut des Dokuments bestimmt.

public string InputEncoding { get; set; }
