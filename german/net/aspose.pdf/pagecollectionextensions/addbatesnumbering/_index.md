## AddBatesNumbering(this PageCollection, Action&lt;BatesNArtifact&gt;) {#addbatesnumbering_1}

Fügt jeder Seite in der angegebenen Seitenkollektion Bates-Nummerierung hinzu, indem die angegebene Aktion verwendet wird, um das BatesNArtifact zu konfigurieren.

```csharp
public static void AddBatesNumbering(this PageCollection pageCollection, 
    Action<BatesNArtifact> action)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pageCollection | PageCollection | Die Sammlung von Seiten, zu der die Bates-Nummerierung hinzugefügt wird. |
| action | Action`1 | Eine Aktion zur Konfiguration des BatesNArtifact, bevor es jeder Seite hinzugefügt wird. |
