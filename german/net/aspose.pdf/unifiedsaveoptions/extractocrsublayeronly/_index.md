## UnifiedSaveOptions.ExtractOcrSublayerOnly-Eigenschaft

Dieses Attribut aktiviert die Funktion zum Extrahieren von Bildern oder Text für PDF-Dokumente mit OCR-Unterebene.

```csharp
public bool ExtractOcrSublayerOnly { get; set; }
```

### Eigenschaftswert

`true` Text wird im Ergebnisdokument extrahiert; andernfalls `false`.
