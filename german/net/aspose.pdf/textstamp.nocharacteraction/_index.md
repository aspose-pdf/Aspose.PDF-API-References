## TextStamp.NoCharacterAction Aufzählung

Aktion, die ausgeführt werden soll, wenn die Schriftart das erforderliche Zeichen nicht enthält.

```csharp
public enum NoCharacterAction
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ThrowException | `0` | Wenn die Schriftart nicht alle Zeichen der Zeichenfolge enthält, wird eine Ausnahme ausgelöst. |
| UseStandardFont | `1` | Ersetze die Schriftart durch eine Standardschriftart, die das erforderliche Zeichen enthält. |
| ReplaceAnyway | `2` | Ersetze den Text trotzdem ohne Schriftartsubstitution. |
| UseCustomReplacementFont | `3` | Ersetze die Schriftart durch die definierte Ersatzschriftart. |
