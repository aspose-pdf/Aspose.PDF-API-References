## HtmlPageLayoutOption enumeration

Gibt Flags an, die zusammen mit anderen Optionen die Größe und das Layout von Seiten bestimmen.

```csharp
public enum HtmlPageLayoutOption
```

### Werte

| Name | Wert | Beschreibung |

| None | `0` | Standardwert, der angibt, dass die HtmlPageLayoutOption die Größe und das Layout von Seiten nicht beeinflusst. |
