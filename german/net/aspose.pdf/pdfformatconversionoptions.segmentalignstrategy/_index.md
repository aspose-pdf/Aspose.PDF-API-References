## PdfFormatConversionOptions.SegmentAlignStrategy Aufzählung

Beschreibt Strategien zur Ausrichtung von Dokumenttextsegmenten. Derzeit wird nur die Strategie unterstützt, um Segmente auf die ursprünglichen Grenzen wiederherzustellen. In Zukunft könnten weitere Strategien hinzugefügt werden.

```csharp
public enum SegmentAlignStrategy : byte
```

### Werte

| Name | Wert | Beschreibung |

| RestoreSegmentBounds | `1` | Segmentgrenzen wiederherstellen. |
