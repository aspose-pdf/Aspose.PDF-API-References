## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled-Feld

Sollte von benutzerdefiniertem Code gesetzt werden, wenn es die URL, die im generierten HTML für die Referenzierung dieses CSS verwendet wird, nicht definieren kann oder sollte. Wenn es 'true' ist, wird die CSS-Datei auf standardmäßige Weise am standardmäßigen Ort gespeichert.

public bool CustomProcessingCancelled;
