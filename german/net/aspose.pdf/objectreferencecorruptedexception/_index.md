## Klasse ObjectReferenceCorruptedException

Stellt eine Ausnahme dar, die ausgelöst wird, wenn ein Objektverweis in einem PDF-Dokument als beschädigt erkannt wird. Das bedeutet, dass kein Objekt durch den Link angezeigt wird.

```csharp
public class ObjectReferenceCorruptedException : PdfException
```
