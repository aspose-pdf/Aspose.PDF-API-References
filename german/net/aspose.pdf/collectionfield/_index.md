## Eigenschaften

| Name | Beschreibung |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | Gibt ein Flag zurück, das angibt, ob der interaktive PDF-Prozessor Unterstützung für die Bearbeitung des Feldwerts bieten soll. Standardwert: false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | Gibt den Typ eines Feldwerts in einer Schemasammlung zurück. Dieses Feld beschreibt den Werttyp, der dem [`Subtype`](./subtype/) entspricht. |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | Gibt den textuellen Feldnamen zurück, der dem Benutzer vom interaktiven PDF-Prozessor angezeigt werden soll |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | Gibt die relative Reihenfolge des Feldnamens in der Benutzeroberfläche zurück. Felder sollen vom interaktiven PDF-Prozessor in aufsteigender Reihenfolge sortiert werden. |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | Gibt den Subtyp eines Feldwerts in einer Schemasammlung zurück. Der Subtyp des Sammlungsfelds oder des dateibezogenen Felds, den dieses Wörterbuch beschreibt. Dieser Eintrag identifiziert den Datentyp, der im Feld gespeichert werden soll. |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | Gibt die anfängliche Sichtbarkeit des Felds in der Benutzeroberfläche zurück. Standardwert: true. |
