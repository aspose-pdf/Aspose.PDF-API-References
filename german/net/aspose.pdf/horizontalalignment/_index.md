## HorizontalAlignment Aufzählung

Beschreibt die horizontale Ausrichtung.

```csharp
public enum HorizontalAlignment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine Ausrichtung. |
| Left | `1` | Nach links ausrichten. |
| Center | `2` | Zentrierte Ausrichtung. |
| Right | `3` | Nach rechts ausrichten. |
| Justify | `4` | Blocksatz. Der Text wird an beiden Rändern, links und rechts, ausgerichtet. |
| FullJustify | `5` | Ähnlich wie die 'Justify'-Ausrichtung, mit dem Unterschied, dass die letzte Zeile im 'Justify'-Modus nur linksbündig ist, während im 'FullJustify'-Modus alle Zeilen sowohl links- als auch rechtsbündig sind. |
