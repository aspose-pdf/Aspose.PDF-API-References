## BatesNArtifact.StartNumber-Eigenschaft

Ruft die Startnummer für die Bates-Nummerierung ab oder legt sie fest. Der Wert muss größer oder gleich 1 sein. Wenn ein Wert kleiner als 1 festgelegt wird, wird er auf 1 angepasst.

```csharp
public int StartNumber { get; set; }
```
