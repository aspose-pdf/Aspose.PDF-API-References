## BatesNArtifact.NumberOfDigits-Eigenschaft

Ruht die Anzahl der Ziffern für Bates-Nummerierung ab oder legt sie fest. Der Wert muss zwischen 3 und 15 liegen, einschließlich. Wenn ein Wert kleiner als 3 festgelegt wird, wird er auf 3 angepasst. Wenn ein Wert größer als 15 festgelegt wird, wird er auf 15 angepasst. Der Standardwert ist 6.

```csharp
public int NumberOfDigits { get; set; }
```
