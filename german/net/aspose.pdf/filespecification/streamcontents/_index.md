## FileSpecification.StreamContents-Eigenschaft

Gibt den Inhalt der Datei als Stream zurück. Der Inhalt wird nicht in den Speicher geladen, was die Speichernutzung verringert. Aber dieser Stream unterstützt keine Positionierung und die Length-Eigenschaft. Wenn Sie diese Funktionen benötigen, verwenden Sie bitte stattdessen die Contents-Eigenschaft.

```csharp
public Stream StreamContents { get; }
```
