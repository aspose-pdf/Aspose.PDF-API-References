## SaveOptions.WarningHandler-Eigenschaft

Callback zur Behandlung von Warnungen, die generiert werden. Der WarningHandler gibt ein ReturnAction-Enum-Element zurück, das entweder Continue oder Abort angibt. Continue ist die Standardaktion und der Speichervorgang wird fortgesetzt, jedoch kann der Benutzer auch Abort zurückgeben, in diesem Fall sollte der Speichervorgang eingestellt werden.

```csharp
public IWarningCallback WarningHandler { get; set; }
```
