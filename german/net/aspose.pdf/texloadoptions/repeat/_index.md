## TeXLoadOptions.Repeat-Eigenschaft

Ruft das Flag ab oder setzt es, das angibt, ob es notwendig ist, den TeX-Job zweimal auszuführen, falls beispielsweise Referenzen in den Eingabe-TeX-Datei(en) vorhanden sind. Im Allgemeinen ist dieses Verhalten nützlich, wenn die Engine während des Satzprozesses einige Daten sammelt und diese beim ersten Durchlauf in einer Hilfsdatei speichert. Und beim zweiten Durchlauf verwendet die Engine diese Daten irgendwie.

```csharp
public bool Repeat { get; set; }
```
