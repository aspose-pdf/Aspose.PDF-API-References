## TeXLoadOptions.RequiredInputDirectory-Eigenschaft

Holt/setzt das erforderliche Eingabeverzeichnis für TeX. Erforderliche Eingaben sind die Dateien, die irgendwie in die Haupt-.tex-Datei aufgenommen werden, z. B. Pakete, für die es keine integrierte Unterstützung gibt.

public ITeXInputDirectory RequiredInputDirectory { get; set; }
