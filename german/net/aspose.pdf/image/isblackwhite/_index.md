## Image.IsBlackWhite-Eigenschaft

Ruft einen booleschen Wert ab oder legt ihn fest, der angibt, ob das Bild auf Schwarzweiß gezwungen ist. Wenn ein TIFF-Bild des CCITT-Subformats verwendet wird, muss diese Eigenschaft auf true gesetzt werden.

```csharp
public bool IsBlackWhite { get; set; }
```
