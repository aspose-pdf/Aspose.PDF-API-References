## LoadOptions.ResourceLoadingResult.LoadingCancelled field

Manchmal sollte aus bestimmten Gründen das Laden von benutzerdefiniertem Code nicht erfolgen. In diesem Fall setzen Sie bitte dieses Flag auf True. In einem solchen Fall versucht der Konverter, den internen Ressourcenlader default zu verwenden, um dieses Ergebnis zu erhalten (wie er sich in Situationen verhält, in denen keine benutzerdefinierte Strategie angegeben ist).

public bool LoadingCancelled;
