## LoadOptions.ResourceLoadingResult.EncodingIfKnown field

Manchmal ist die Codierung der Ressource nach oder während des Ladens bekannt. In einem solchen Fall kann benutzerdefinierter Code den Konverter mit diesem Wissen über diesen Parameter versorgen. Sie können diesen Parameter null lassen, wenn die Kodierung unbekannt ist oder keine Rolle spielt.

public Encoding EncodingIfKnown;
