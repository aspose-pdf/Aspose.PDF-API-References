## LoadOptions.ResourceLoadingResult.EncodingIfKnown-Feld

Manchmal ist die Kodierung der Ressource nach oder während des Ladens bekannt. In diesem Fall kann benutzerdefinierter Code dem Konverter dieses Wissen über diesen Parameter bereitstellen. Sie können null in diesem Parameter lassen, wenn die Kodierung unbekannt ist oder keine Rolle spielt.

```csharp
public Encoding EncodingIfKnown;
```
