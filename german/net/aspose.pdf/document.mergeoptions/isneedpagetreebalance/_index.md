## Document.MergeOptions.IsNeedPageTreeBalance-Eigenschaft

Legt die Anforderung für die Balance des Seitenbaums fest. Der gesamte Seitenbaum im resultierenden Dokument wird neu ausbalanciert. Es erstellt einen ausgewogenen Seitenbaum, um den Zugriff auf Seiten zu beschleunigen.

public bool IsNeedPageTreeBalance { get; set; }
