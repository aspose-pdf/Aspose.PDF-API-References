## RenderingOptions.UseFontHinting-Eigenschaft

Die Verwendung dieses Flags aktiviert den Schriftart-Hinting-Mechanismus. Schriftart-Hinting ist die Verwendung von mathematischen Anweisungen zur Anpassung der Anzeige einer Outline-Schriftart. In einigen Fällen kann das Aktivieren dieses Flags Probleme mit der Lesbarkeit von Text lösen. Im aktuellen Moment kann die Verwendung dieses Flags nur für TTF-Schriftarten Wirkung zeigen, wenn diese Schriftarten im Quelldokument verwendet werden.

```csharp
public bool UseFontHinting { get; set; }
```
