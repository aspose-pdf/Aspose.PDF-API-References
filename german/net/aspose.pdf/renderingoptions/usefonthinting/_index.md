## RenderingOptions.UseFontHinting property

Die Verwendung dieses Flags schaltet den Font-Hinweismechanismus ein. Font Hinting ist die Verwendung von mathematischen Anweisungen, um die Anzeige einer Outline-Schriftart anzupassen. In einigen Fällen kann das Aktivieren dieses Flags Probleme mit der Textlesbarkeit lösen. Gegenwärtig kann die Verwendung dieses Flags nur für TTF-Schriftarten wirksam werden, wenn diese Schriftarten im Quelldokument verwendet werden.

```csharp
public bool UseFontHinting { get; set; }
```
