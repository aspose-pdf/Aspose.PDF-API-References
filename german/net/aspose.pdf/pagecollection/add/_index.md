## Add() {#add}

Fügt eine leere Seite hinzu. Wenn das Dokument bereits Seiten mit unterschiedlichen Größen enthält, wird die Größe der am häufigsten vorkommenden Seite ausgewählt. Falls es nur zwei unterschiedliche Seiten gibt, wird die Größe der ersten Seite verwendet.

```csharp
public Page Add()
```

### Rückgabewert

Hinzugefügte Seite.
