## ExtractImageMode Aufzählung

Definiert verschiedene Modi, die beim Extrahieren von Bildern aus Dokumenten verwendet werden können.

```csharp
public enum ExtractImageMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DefinedInResources | `0` | Definiert den Bildextraktionsmodus, in dem alle Bilder, die in den Ressourcen für eine bestimmte Seite definiert sind, extrahiert werden. |
| ActuallyUsed | `1` | Definiert den Bildextraktionsmodus, in dem nur die Bilder extrahiert werden, die tatsächlich auf einer Seite angezeigt werden. |
