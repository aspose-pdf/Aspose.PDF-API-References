## Teilmengenauflistung

Stellt die Teilmenge von Seiten dar, auf die ein Paginierungsartefakt angewendet werden kann.

```csharp
public enum Subset
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Alle | `0` | Gibt an, dass das Artefakt auf alle Seiten angewendet wird. |
| Gerade | `1` | Gibt an, dass das Artefakt nur auf gerade nummerierte Seiten angewendet wird. |
| Ungerade | `2` | Gibt an, dass das Artefakt nur auf ungerade nummerierte Seiten angewendet wird. |
