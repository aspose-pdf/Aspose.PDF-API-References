## ExcelSaveOptions.InsertBlankColumnAtFirst-Eigenschaft

Setzen Sie true, wenn Sie das Einfügen einer leeren Spalte als erste Spalte des Arbeitsblatts benötigen. Der Standardwert ist false; das bedeutet, dass keine leere Spalte eingefügt wird.

public bool InsertBlankColumnAtFirst { get; set; }
