## ExcelSaveOptions.MinimizeTheNumberOfWorksheets-Eigenschaft

Setzen Sie true, wenn Sie die Anzahl der Arbeitsblätter in der resultierenden Arbeitsmappe minimieren möchten. Der Standardwert ist false; das bedeutet, dass jede PDF-Seite als separates Arbeitsblatt gespeichert wird.

```csharp
public bool MinimizeTheNumberOfWorksheets { get; set; }
```
