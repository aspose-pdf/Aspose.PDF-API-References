## ExcelSaveOptions.MinimizeTheNumberOfWorksheets property

Legen Sie true fest, wenn Sie die Anzahl der Arbeitsblätter in der resultierenden Arbeitsmappe minimieren müssen. Der Standardwert ist false; es bedeutet, dass jede PDF-Seite als separates Arbeitsblatt gespeichert wird.
