## ExcelSaveOptions.UniformWorksheets-Eigenschaft

Setzen Sie true, um eine einheitliche Spaltenaufteilung im gesamten Dokument zu verwenden. Der Standardwert ist false; das bedeutet, dass die Spaltenaufteilung für jede Seite unabhängig ist.

```csharp
public bool UniformWorksheets { get; set; }
```
