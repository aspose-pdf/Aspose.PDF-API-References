## HtmlSaveOptions.HtmlMarkupGenerationModes Enumeration

Manchmal sind spezifische Anforderungen an das erstellte HTML vorhanden. Dieses Enum definiert HTML-Vorbereitungsmodi, die während der Konvertierung von PDF zu HTML verwendet werden können, um solchen spezifischen Anforderungen gerecht zu werden.

```csharp
public enum HtmlMarkupGenerationModes
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| WriteAllHtml | `0` | Standardmodus, wenn keine spezifischen Anforderungen vorhanden sind. Es wird eine Ausgabe generiert, die alle Teile von HTML ohne spezielle zusätzliche Verarbeitung enthält. |
| WriteOnlyBodyContent | `1` | Es wird sämtlicher HTML-Inhalt entfernt, der außerhalb des HTML-Körpers liegt, d.h. es bleibt nur der Inhalt, der sich innerhalb der \c \ \c \ Tags befindet |
