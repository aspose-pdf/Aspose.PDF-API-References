## OutputIntent-Klasse

Stellt eine Ausgabebestimmung dar, die die Farbeigenschaften eines PDF-Dokuments mit denen eines Zielausgabegeräts oder Produktionsumgebung abgleicht, in der das Dokument gedruckt wird.

```csharp
public sealed class OutputIntent
```

## Konstruktoren

| Name | Beschreibung |
| --- | --- |
| [OutputIntent](outputintent/)(string) | Initialisiert eine neue Instanz der `OutputIntent`-Klasse mit dem angegebenen Ausgabebedingungsbezeichner. |

## Eigenschaften

| Name | Beschreibung |
| --- | --- |
| [Info](../../aspose.pdf/outputintent/info/) { get; set; } | Ruft einen menschenlesbaren Text ab oder legt ihn fest, der zusätzliche Informationen oder Kommentare zum beabsichtigten Zielgerät oder zur Produktionsbedingung enthält. |
| [OutputCondition](../../aspose.pdf/outputintent/outputcondition/) { get; set; } | Ruft einen Text ab oder legt ihn fest, der das beabsichtigte Ausgabegerät oder die Produktionsbedingung in menschenlesbarer Form prägnant identifiziert. |
| [OutputConditionIdentifier](../../aspose.pdf/outputintent/outputconditionidentifier/) { get; set; } | Ruft einen Text ab oder legt ihn fest, der das beabsichtigte Ausgabegerät oder die Produktionsbedingung in menschen- oder maschinenlesbarer Form identifiziert. |
