## Matrix.Translate-Methode

Übersetzt eine Matrix um den angegebenen Betrag in x- und y-Richtung.

```csharp
public static Matrix Translate(double dx, double dy, Matrix source)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| dx | Double | Der Betrag, um den in x-Richtung übersetzt werden soll. |
| dy | Double | Der Betrag, um den in y-Richtung übersetzt werden soll. |
| source | Matrix | Die zu übersetzende Matrix. |
