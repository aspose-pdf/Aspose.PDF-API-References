## XImage.Name-Eigenschaft

Ruft den Bildnamen ab oder legt ihn fest. Bitte beachten Sie, dass das Dokument möglicherweise inkorrekt wird, wenn Sie den Namen des Bildes ändern, das in den Seiteninhalten referenziert wird. Bitte verwenden Sie in diesem Fall die Methode XImage.Rename.

```csharp
public string Name { get; set; }
```
