## XImage.ImageMask-Eigenschaft

Gibt ein Flag an, das angibt, ob das Bild als Bildmaske behandelt werden soll (siehe 8.9.6, "Maskierte Bilder"). Wenn dieses Flag wahr ist, muss der Wert von BitsPerComponent 1 sein und Mask und ColorSpace dürfen nicht angegeben werden; unmaskierte Bereiche werden mit der aktuellen nicht-strichfarbe bemalt. Standardwert: falsch.

```csharp
public bool ImageMask { get; }
```

### Eigenschaftswert

Wahr, wenn das Bild eine Bildmaske ist.
