## SaveOptions.ResourceSavingInfo.CustomProcessingCancelled-Feld

Dieses Flag muss im benutzerdefinierten Code auf "true" gesetzt werden, wenn aus irgendeinem Grund die vorgeschlagene Datei nicht mit benutzerdefiniertem Code, sondern mit dem Code des Konverters selbst auf die Standardweise des Konverters verarbeitet werden soll. Das Setzen auf true bedeutet, dass der benutzerdefinierte Code die referenzierte Datei nicht verarbeitet hat und der Konverter sie selbst (in beiden Sinne - zum Speichern irgendwo und zum Benennen in der referenzierenden Datei) behandeln muss.

```csharp
public bool CustomProcessingCancelled;
```
