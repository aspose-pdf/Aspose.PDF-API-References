## HtmlSaveOptions.HtmlPageMarkupSavingInfo.CustomProcessingCancelled-Feld

Sollte im benutzerdefinierten Code gesetzt werden, wenn nötig. Dieses Flag muss auf "true" im benutzerdefinierten Code gesetzt werden, wenn aus bestimmten Gründen der bereitgestellte HTML-Markup nicht mit benutzerdefiniertem Code, sondern mit dem Code des Konverters selbst auf die Standardweise des Konverters verarbeitet werden soll. Das Setzen dieses Flags im benutzerdefinierten Code bedeutet, dass der benutzerdefinierte Code die referenzierte Datei nicht verarbeitet hat und der Konverter dies selbst übernehmen muss.

```csharp
public bool CustomProcessingCancelled;
```
