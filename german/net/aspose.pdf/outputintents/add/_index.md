## OutputIntents.Add-Methode

Fügt der Sammlung eine Ausgabeabsicht hinzu.

```csharp
public void Add(OutputIntent item)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| item | OutputIntent | Die Ausgabeabsicht, die der Sammlung hinzugefügt werden soll. |

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentNullException | Das *item* ist null. |
| InvalidOperationException | Das Dokument, das die Sammlung enthält, hat kein Katalog, um auf die OutputIntents zuzugreifen. |
