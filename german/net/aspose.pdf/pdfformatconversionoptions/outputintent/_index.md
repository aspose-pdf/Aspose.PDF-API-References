## PdfFormatConversionOptions.OutputIntent-Eigenschaft

Ruft das [`OutputIntent`](../../outputintent/) für die PDF-Formatkonvertierung ab oder legt es fest.

```csharp
public OutputIntent OutputIntent { get; set; }
```

## Bemerkungen

Das `OutputIntent` gibt das beabsichtigte Ausgabegerät oder die Bedingung an, für die das PDF-Dokument vorbereitet wird. Es wird verwendet, um sicherzustellen, dass die Farben im Dokument auf dem Zielgerät korrekt wiedergegeben werden.
