## PdfSaveOptions.DefaultFontName-Eigenschaft

Schriftartname, der standardmäßig für Schriftarten verwendet wird, die auf dem Computer fehlen. Wenn das PDF-Dokument, das in PDF gespeichert wird, Schriftarten enthält, die im Dokument selbst und auf dem Gerät nicht verfügbar sind, ersetzt die API diese Schriftarten durch die Standardschriftart, wenn eine Schriftart mit `DefaultFontName` auf dem Gerät gefunden wird.

```csharp
public string DefaultFontName { get; set; }
```
