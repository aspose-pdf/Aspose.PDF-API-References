## TextStamp.Draw-Eigenschaft

Diese Eigenschaft bestimmt, wie der Stempel auf der Seite gezeichnet wird. Wenn Draw = true ist, wird der Stempel als grafische Operatoren gezeichnet, und wenn draw = false ist, wird der Stempel als Text gezeichnet.

```csharp
public bool Draw { get; set; }
```
