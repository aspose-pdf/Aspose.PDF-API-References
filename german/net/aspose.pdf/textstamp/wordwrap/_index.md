## TextStamp.WordWrap-Eigenschaft

Definiert den Zeilenumbruch. Wenn diese Eigenschaft auf true gesetzt und ein Breitenwert angegeben wird, wird der Text in mehrere Zeilen umgebrochen, um in die angegebene Breite zu passen. Standardwert: false.

```csharp
public bool WordWrap { get; set; }
```
