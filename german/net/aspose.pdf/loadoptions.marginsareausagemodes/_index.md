## LoadOptions.MarginsAreaUsageModes enumeration

Repräsentiert den Verwendungsmodus des Randbereichs während der Konvertierung (wie HTML, EPUB usw.), definiert die Behandlung von Anweisungen des importierten Formats in Bezug auf die Verwendung von Rändern.

```csharp
public enum MarginsAreaUsageModes
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PutContentOnMarginAreaIfNecessary | `0` | In diesem Modus gehorcht der Konverter dem Format des importierten Dokuments (z. B. CSS von importiertem HTML) bei der Verwendung des Randbereichs. |
| NeverPutContentOnMarginArea | `1` | Dieser Modus verbietet strikt die Verwendung des Randbereichs, daher wird der Konverter niemals den Randbereich zum Rendern verwenden, selbst wenn CSS oder das Format des Quelldokuments dies zulässt oder erfordert |
