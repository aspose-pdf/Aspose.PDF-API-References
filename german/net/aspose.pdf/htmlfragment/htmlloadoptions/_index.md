## HtmlFragment.HtmlLoadOptions property

Ruft HtmlLoadOptions ab oder setzt sie, die zum Laden (und Rendern) von HTML in diese Instanz der Klasse verwendet werden. Bitte verwenden Sie es, wenn es notwendig ist, spezifische Einstellungen für den Import von HTML für diese oder jene Instanz zu verwenden (z. B. wenn diese oder jene Instanz sollte Verwenden Sie einen bestimmten BasePath für importiertes HTML oder sollten Sie einen bestimmten Loader für externe Ressourcen verwenden) Wenn der Parameter standardmäßig (null) ist, werden Standard-HTML-Ladeoptionen verwendet.

public HtmlLoadOptions HtmlLoadOptions { get; set; }
