## Rectangle.ContainsLine-Methode

Bestimmt, ob das Rechteck eine Linie enthält, die durch zwei Punkte dargestellt wird.

```csharp
public bool ContainsLine(double x1, double y1, double x2, double y2)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x1 | Double | Die X-Koordinate des Startpunkts der Linie. |
| y1 | Double | Die Y-Koordinate des Startpunkts der Linie. |
| x2 | Double | Die X-Koordinate des Endpunkts der Linie. |
| y2 | Double | Die Y-Koordinate des Endpunkts der Linie. |

### Rückgabewert

`true`, wenn das Rechteck die Linie enthält; andernfalls `false`.
