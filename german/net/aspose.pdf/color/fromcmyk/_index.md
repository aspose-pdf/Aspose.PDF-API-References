## Color.FromCmyk-Methode

Erhält ein gültiges PDF-Farbobjekt aus CMYK-Farbkomponenten.

```csharp
public static Color FromCmyk(double c, double m, double y, double k)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | Double | Die Cyan-Farbkomponente (Wert 0 - 1). |
| m | Double | Die Magenta-Farbkomponente (Wert 0 - 1). |
| y | Double | Die Gelb-Farbkomponente (Wert 0 - 1). |
| k | Double | Die Schwarz-Farbkomponente (Wert 0 - 1). |

### Rückgabewert

Farbobjekt mit jedem Komponentenwert im [0..1]-Bereich.
