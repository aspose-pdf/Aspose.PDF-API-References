## MarkdownSaveOptions.UseImageHtmlTag-Eigenschaft

Ermöglicht das Setzen und Abrufen der Erlaubnis zur Verwendung eines img-Tags, um Bilder links und rechts vom Text einzufügen. In diesem Fall wird der Text im Markdown-Viewer um das Bild gewickelt.

```csharp
public bool UseImageHtmlTag { get; set; }
```
