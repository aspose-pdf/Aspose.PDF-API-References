## MarkdownSaveOptions.HeadingLevels-Eigenschaft

Definiert die erwarteten Überschriftsebenen, die in der FontSize-Erkennungsstrategie für Überschriften verwendet werden sollen. Wenn dieser Eigenschaftswert gesetzt ist, wird die Überschrifterkennung !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic Strategie ausgewählt, wenn die !:PdfToMarkdown.HeadingRecognitionStrategy.Auto Strategien gesetzt sind, selbst wenn das Dokument Lesezeichen enthält.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
