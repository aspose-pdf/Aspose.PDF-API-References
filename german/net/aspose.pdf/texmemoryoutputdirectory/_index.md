## TeXMemoryOutputDirectory-Klasse

Implementiert das Abrufen eines Ausgabestreams aus dem Speicher. Sie können es beispielsweise verwenden, wenn Sie nicht möchten, dass die begleitende Ausgabe (wie eine Protokolldatei) auf die Festplatte geschrieben wird, sondern Sie möchten sie anschließend aus dem Speicher lesen.
