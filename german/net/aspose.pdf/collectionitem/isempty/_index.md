## CollectionItem.IsEmpty-Eigenschaft

Gibt einen Wert zurück, der angibt, ob das Sammlungsobjekt leer ist.

```csharp
public bool IsEmpty { get; }
```

### Rückgabewert

true, wenn das Sammlungsobjekt leer ist; andernfalls false.

## Bemerkungen

Diese Eigenschaft gibt true zurück, wenn das Sammlungsobjekt keine Werte enthält, einschließlich Zeichenfolgenwerte, Double-Werte, Ganzzahlenwerte und Datumswerte. Wenn einer dieser Werttypen im Sammlungsobjekt vorhanden ist, gibt diese Eigenschaft false zurück.
