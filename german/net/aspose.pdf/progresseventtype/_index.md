## ProgressEventType-Enumeration

Dieses Enum beschreibt mögliche Fortschrittsereignistypen, die während der Konvertierung auftreten können

```csharp
public enum ProgressEventType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| TotalProgress | `0` | bedeutet, dass das aufgetretene Ereignis über den Gesamtfortschritt der Konvertierung informiert |
| SourcePageAnalysed | `1` | bedeutet, dass das aufgetretene Ereignis über das Ende der Analyse einer der Seiten vor der Konvertierung informiert |
| ResultPageCreated | `2` | bedeutet, dass das aufgetretene Ereignis über die Erstellung einer Ergebnisseite vor dem physischen Export informiert |
| ResultPageSaved | `3` | bedeutet, dass das aufgetretene Ereignis über das abgeschlossene Speichern einer Ergebnisseite informiert |
