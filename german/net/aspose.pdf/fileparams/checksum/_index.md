## FileParams.CheckSum-Eigenschaft

Ein 16-Byte-String, der die Prüfziffer der Bytes der unkomprimierten eingebetteten Datei ist. Die Prüfziffer wird berechnet, indem der Standard-MD5-Message-Digest-Algorithmus auf die Bytes des Streams der eingebetteten Datei angewendet wird.

public string CheckSum { get; }
