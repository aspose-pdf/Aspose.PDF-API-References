## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace-Eigenschaft

Einige Schriftarten bieten keine Informationen über Unicodes für einige Textsymbole. Dieser Mangel an Informationen führt zu einem Fehler "Text kann nicht in Unicode umgewandelt werden". Verwenden Sie dieses Flag, um nicht verlinkte Symbole auf den Unicode "Leerzeichen" (Code 32) abzubilden.

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
