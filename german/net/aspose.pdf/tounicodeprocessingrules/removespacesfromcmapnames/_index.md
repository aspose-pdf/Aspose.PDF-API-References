## ToUnicodeProcessingRules.RemoveSpacesFromCMapNames-Eigenschaft

Einige Schriftarten haben ToUnicode-Zeichencodierungszuordnungen mit Leerzeichen in den Namen. Diese Leerzeichen könnten Fehler bei der Unicode-Textzuordnung verursachen. Dieses Flag weist an, Leerzeichen aus den Namen der ToUnicode-Zeichencodierungszuordnungen zu entfernen. Standardmäßig falsch.

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
