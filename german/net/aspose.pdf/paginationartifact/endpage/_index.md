## PaginationArtifact.EndPage-Eigenschaft

Ruft die Endseitenzahl für das Artefakt ab oder legt sie fest. Der Wert muss größer oder gleich 0 sein. Wenn ein Wert kleiner als 0 festgelegt wird, wird er auf 0 angepasst. Der Standardwert von 0 bedeutet, dass es keine Endseitenbegrenzungen gibt.

```csharp
public int EndPage { get; set; }
```
