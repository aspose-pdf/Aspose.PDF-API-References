## PaginationArtifact.StartPage-Eigenschaft

Ruft die Startseitenzahl für das Artefakt ab oder legt sie fest. Der Wert muss größer oder gleich 1 sein. Wenn ein Wert kleiner als 1 festgelegt wird, wird er auf 1 angepasst.

```csharp
public int StartPage { get; set; }
```
