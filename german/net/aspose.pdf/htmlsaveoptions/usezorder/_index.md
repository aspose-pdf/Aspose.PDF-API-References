## HtmlSaveOptions.UseZOrder-Eigenschaft

Wenn das Attribut UseZOrder auf true gesetzt ist, werden Grafiken und Text entsprechend der Z-Reihenfolge im ursprünglichen PDF-Dokument zum resultierenden HTML-Dokument hinzugefügt. Wenn dieses Attribut false ist, werden alle Grafiken als einzelne Ebene eingefügt, was einige unnötige Effekte bei überlappenden Objekten verursachen kann.

```csharp
public bool UseZOrder { get; set; }
```
