## HtmlSaveOptions.UseZOrder property

Wenn das Attribut UseZORder auf true gesetzt ist, werden Grafiken und Text dem resultierenden HTML-Dokument entsprechend der Z-Reihenfolge im ursprünglichen PDF-Dokument hinzugefügt. Wenn dieses Attribut falsch ist, werden alle Grafiken als einzelne Ebene platziert, was einige unnötige Effekte für überlappende Objekte verursachen kann.

public bool UseZOrder { get; set; }
