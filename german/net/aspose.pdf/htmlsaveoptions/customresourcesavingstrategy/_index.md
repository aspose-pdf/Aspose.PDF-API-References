## HtmlSaveOptions.CustomResourceSavingStrategy-Feld

Dieses Feld kann eine Speicherstrategie enthalten, die verwendet werden muss (wenn vorhanden) während der Konvertierung für eine benutzerdefinierte Handhabung der erstellten referenzierten Ressourcendateien (wie Bilder und Schriftarten), die mit Knoten des gespeicherten HTML verbunden sind. Diese Strategie muss Ressourcen verarbeiten und einen String zurückgeben, der die wünschenswerte URL der gespeicherten Ressource im generierten HTML darstellt.

```csharp
public ResourceSavingStrategy CustomResourceSavingStrategy;
```
