## HtmlSaveOptions.DefaultFontName-Eigenschaft

Gibt den Namen einer installierten Schriftart an, die verwendet wird, um eine Dokumentenschriftart zu ersetzen, die nicht eingebettet und nicht im System installiert ist. Wenn null, wird die Standardschriftart für die Ersetzung verwendet.

```csharp
public string DefaultFontName { get; set; }
```

### Eigenschaftswert

Schriftartname
