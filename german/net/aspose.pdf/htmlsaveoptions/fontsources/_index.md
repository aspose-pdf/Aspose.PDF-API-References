## HtmlSaveOptions.FontSources-Eigenschaft

Schriftquellen von vorab gespeicherten Schriften.

```csharp
public FontSourceCollection FontSources { get; }
```

## Bemerkungen

Schriften können vorläufig zum Caching gespeichert und dann in den Html-Konvertierungsprozess übergeben werden. Zum Beispiel kann es in einem Szenario zur Dokumentenaufteilung nützlich sein und Dokumentseiten in mehreren Threads mit einem einzigen Satz von Schriften zu verarbeiten.
