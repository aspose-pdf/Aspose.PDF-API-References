## HtmlSaveOptions.SaveFullFont-Eigenschaft

Gibt an, dass die vollständige Schriftart gespeichert wird, unterstützt nur True Type Fonts. Standardmäßig ist SaveFullFont = false und der Konverter speichert die Teilmenge der ursprünglichen Schriftart, die benötigt wird, um den Text des Dokuments anzuzeigen.

```csharp
public bool SaveFullFont { get; set; }
```
