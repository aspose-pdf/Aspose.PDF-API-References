## HtmlSaveOptions.RenderTextAsImage-Eigenschaft

Wenn das Attribut RenderTextAsImage auf true gesetzt ist, wird der Text aus der Quelle zu einem Bild in HTML. Kann nützlich sein, um Text nicht auswählbar zu machen oder wenn HTML-Text nicht richtig gerendert wird.

```csharp
public bool RenderTextAsImage { get; set; }
```
