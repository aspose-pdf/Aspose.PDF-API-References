## HtmlSaveOptions.SaveTransparentTexts field

Pdf kann transparente Texte enthalten, die in der Zwischenablage ausgewählt werden können (normalerweise geschieht dies, wenn das Dokument Bilder und daraus extrahierte OCRed-Texte enthält). Diese Einstellung teilt dem Konverter mit, ob wir solche Texte als transparente auswählbare Texte im Ergebnis HTML speichern müssen

```csharp
public bool SaveTransparentTexts;
```
