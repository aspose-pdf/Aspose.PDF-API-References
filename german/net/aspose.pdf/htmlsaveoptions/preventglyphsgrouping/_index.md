## HtmlSaveOptions.PreventGlyphsGrouping-Eigenschaft

Dieses Attribut schaltet den Modus ein, in dem Textglyphen nicht in Wörter und Zeichenfolgen gruppiert werden. Dieser Modus ermöglicht es, maximale Präzision bei der Positionierung von Glyphen auf der Seite zu behalten, und kann für die Konvertierung von Dokumenten mit Musiknoten oder Glyphen verwendet werden, die separat platziert werden sollen. Dieser Parameter wird nur auf das Dokument angewendet, wenn der Wert des Attributs FixedLayout true ist.

```csharp
public bool PreventGlyphsGrouping { get; set; }
```
