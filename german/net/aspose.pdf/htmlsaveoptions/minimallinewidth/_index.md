## HtmlSaveOptions.MinimalLineWidth-Eigenschaft

Dieses Attribut legt die minimale Breite der grafischen Pfadlinie fest. Wenn die Dicke der Linie weniger als 1px beträgt, rundet Adobe Acrobat sie auf diesen Wert. Dieses Attribut kann verwendet werden, um dieses Verhalten für HTML-Browser zu emulieren.

```csharp
public float MinimalLineWidth { get; set; }
```
