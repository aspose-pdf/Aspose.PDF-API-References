## HtmlSaveOptions.MinimalLineWidth property

Dieses Attribut legt die Mindestbreite der Grafikpfadlinie fest. Wenn die Linienstärke weniger als 1 Pixel beträgt, rundet Adobe Acrobat sie auf diesen Wert. Daher kann dieses Attribut verwendet werden, um dieses Verhalten für HTML-Browser zu emulieren.

```csharp
public float MinimalLineWidth { get; set; }
```
