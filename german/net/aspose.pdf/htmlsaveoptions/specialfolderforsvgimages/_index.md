## HtmlSaveOptions.SpecialFolderForSvgImages-Feld

Erhält oder setzt den Pfad zum Verzeichnis, in dem nur SVG-Bilder gespeichert werden müssen, wenn sie beim Speichern des Dokuments als HTML gefunden werden. Wenn der Parameter leer oder null ist, werden SVG-Dateien (sofern vorhanden) zusammen mit anderen Bilddateien (in der Nähe der Ausgabedatei) oder in einem speziellen Ordner für Bilder (wenn dies in der Option SpecialImagesFolderIfAny angegeben ist) gespeichert. Es hat keinen Einfluss, wenn die Eigenschaft CustomImageSavingStrategy erfolgreich verwendet wurde, um die entsprechenden Bilddateien zu verarbeiten.

```csharp
public string SpecialFolderForSvgImages;
```
