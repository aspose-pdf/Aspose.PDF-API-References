## HtmlSaveOptions.ConvertMarkedContentToLayers-Eigenschaft

Wenn das Attribut ConvertMarkedContentToLayers auf true gesetzt ist, werden alle Elemente innerhalb eines PDF-markierten Inhalts (Schicht) in ein HTML-div mit dem Attribut "data-pdflayer" eingefügt, das einen Schichtnamen angibt. Dieser Schichtname wird aus den optionalen Eigenschaften des PDF-markierten Inhalts extrahiert. Wenn dieses Attribut false (standardmäßig) ist, werden keine Schichten aus dem PDF-markierten Inhalt erstellt.

```csharp
public bool ConvertMarkedContentToLayers { get; set; }
```
