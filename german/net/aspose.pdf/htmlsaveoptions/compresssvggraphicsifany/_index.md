## HtmlSaveOptions.CompressSvgGraphicsIfAny-Eigenschaft

Ruft das Flag ab oder legt es fest, das angibt, ob gefundene SVG-Grafiken (falls vorhanden) während des Speicherns in das SVGZ-Format komprimiert (gezipped) werden.

public bool CompressSvgGraphicsIfAny { get; set; }
