## HtmlSaveOptions.SimpleTextboxModeGrouping-Eigenschaft

Dieses Attribut gibt eine sequenzielle Gruppierung von Glyphen und Wörtern in Strings an. Zum Beispiel haben Tags und Wörter eine unterschiedliche Reihenfolge im konvertierten HTML und Sie möchten, dass sie übereinstimmen. Dieser Parameter wird nur auf das Dokument angewendet, wenn der Wert des Attributs FixedLayout true ist.

```csharp
public bool SimpleTextboxModeGrouping { get; set; }
```
