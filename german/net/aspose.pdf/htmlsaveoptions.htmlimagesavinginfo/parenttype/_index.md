## HtmlSaveOptions.HtmlImageSavingInfo.ParentType-Feld

Das gespeicherte Bild kann sich auf das HTML selbst beziehen oder aus SVG extrahiert werden, das in HTML eingebettet ist. Diese Eigenschaft kann benutzerdefiniertem Code mitteilen, welcher Typ von übergeordnetem Element das verarbeitete Bild hat. Es wird vom Konverter festgelegt und kann im benutzerdefinierten Code verwendet werden, um zu entscheiden, was mit diesem Bild geschehen soll (z. B. kann benutzerdefinierter Code entscheiden, wo das Bild gespeichert werden soll oder wie es im Inhalt des übergeordneten Elements referenziert werden muss).

```csharp
public ImageParentTypes ParentType;
```
