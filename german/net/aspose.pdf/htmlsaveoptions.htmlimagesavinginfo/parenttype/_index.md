## HtmlSaveOptions.HtmlImageSavingInfo.ParentType field

Das gespeicherte Bild kann zu HTML selbst gehören oder extrahiert werden. aus in HTML eingebettetem SVG. Diese Eigenschaft kann dem benutzerdefinierten Code mitteilen, welcher Elterntyp das verarbeitete Bild ist. Es wird vom Konverter festgelegt und kann im benutzerdefinierten Code verwendet werden, um zu entscheiden, was mit diesem Bild geschehen soll (z. B. kann benutzerdefinierter Code entscheiden, wo image gespeichert wird oder wie es im übergeordneten Inhalt referenziert werden muss).

public ImageParentTypes ParentType;
