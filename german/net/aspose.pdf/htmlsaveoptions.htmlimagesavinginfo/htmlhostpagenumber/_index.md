## HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber-Feld

Gibt dem benutzerdefinierten Code an, zu welcher Seite des generierten Satzes von HTML-Seiten-Dateien das gespeicherte Bild gehört. Wenn das Aufteilen in Seiten deaktiviert ist, enthält dieser Wert immer '1', da in diesem Fall nur eine HTML-Seite generiert wird.

```csharp
public int HtmlHostPageNumber;
```
