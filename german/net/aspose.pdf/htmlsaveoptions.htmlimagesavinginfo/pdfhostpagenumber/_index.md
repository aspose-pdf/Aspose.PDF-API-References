## HtmlSaveOptions.HtmlImageSavingInfo.PdfHostPageNumber-Feld

Gibt dem benutzerdefinierten Code an, zu welcher Seite des ursprünglichen PDF-Dokuments das gespeicherte Bild gehört. Da es möglich ist, dass nicht alle Seiten des ursprünglichen Dokuments gespeichert werden, gibt dieser Wert die Host-Seitennummer im ursprünglichen PDF an. Wenn die ursprüngliche Seitennummer aus irgendeinem Grund unbekannt ist, wird immer '1' zurückgegeben.

```csharp
public int PdfHostPageNumber;
```
