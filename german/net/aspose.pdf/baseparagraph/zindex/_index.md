## BaseParagraph.ZIndex-Eigenschaft

Ruft einen int-Wert ab oder legt ihn fest, der die Z-Reihenfolge des Graphen angibt. Ein Graph mit größerem ZIndex wird über dem Graphen mit kleinerem ZIndex platziert. ZIndex kann negativ sein. Graphen mit negativem ZIndex werden hinter dem Text auf der Seite platziert.

```csharp
public int ZIndex { get; set; }
```
