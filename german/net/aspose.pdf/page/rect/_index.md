## Page.Rect property

Holt oder setzt das Rechteck der Seite. Falls angegeben, wird Seitenzuschneidebox zurückgegeben, andernfalls wird Seitenmedienbox zurückgegeben. Bitte beachten Sie, dass diese Eigenschaft die Seitendrehung nicht berücksichtigt. Um ein Seitenrechteck unter Berücksichtigung der Drehung zu erhalten, verwenden Sie bitte ActualRect.

```csharp
public Rectangle Rect { get; set; }
```

### Beispiele

Das Beispiel zeigt, wie man ein Seitenrechteck erhält:

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
Rectangle pageRect = page.Rect;
```
