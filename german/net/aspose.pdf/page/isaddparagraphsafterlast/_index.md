## Page.IsAddParagraphsAfterLast-Eigenschaft

Ruft die Hinzufügung von Absätzen nach dem letzten Absatz der Seite ab oder legt sie fest

```csharp
public bool IsAddParagraphsAfterLast { get; set; }
```

### Eigenschaftswert

Der Wert gibt an, ob Absätze nach dem letzten Absatz der Seite hinzugefügt werden. Absätze werden nach dem letzten Absatz der Seite hinzugefügt, wenn der Wert true ist.
