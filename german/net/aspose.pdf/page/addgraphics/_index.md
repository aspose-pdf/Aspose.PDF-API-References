## Page.AddGraphics-Methode

Fügt Grafiken zur Seite hinzu. Funktioniert schneller als das Hinzufügen von Elementen einzeln mit der [`AddOnPage`](../../../aspose.pdf.vector/graphicelement/addonpage/) Methode.

```csharp
public void AddGraphics(GraphicElementCollection elements, Rectangle rectangle = null)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| elements | GraphicElementCollection | Grafiksammlung. |
| rectangle | Rectangle | Elemente werden zur Seite hinzugefügt, wenn sie sich im Rechteckbereich befinden. Wenn das Rechteck null ist, werden alle grafischen Elemente hinzugefügt |
