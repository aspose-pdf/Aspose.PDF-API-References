## ColumnInfo.ColumnSpacing property

Ruft eine Zeichenfolge ab oder legt sie fest, die den Abstand zwischen den Spalten enthält. Der Wert jedes Abstands sollte durch Leerzeichen getrennt werden. Die Standardeinheit ist Punkt, , aber cm und Zoll werden ebenfalls unterstützt. Beispiel: „120 2,5 cm 1,5 Zoll“.

```csharp
public string ColumnSpacing { get; set; }
```

### Bemerkungen

Wenn diese Eigenschaft nicht gesetzt ist, wird der Standardwert 0 für jeden Abstand verwendet.
