## ColumnInfo.ColumnSpacing-Eigenschaft

Ruht eine Zeichenfolge ab oder legt sie fest, die den Abstand zwischen den Spalten enthält. Der Wert jedes Abstands sollte durch ein Leerzeichen getrennt sein. Die Standard Einheit ist Punkt, aber cm und Zoll werden ebenfalls unterstützt. Zum Beispiel "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Bemerkungen

Wenn diese Eigenschaft nicht gesetzt ist, wird der Standardwert 0 für jeden Abstand verwendet.
