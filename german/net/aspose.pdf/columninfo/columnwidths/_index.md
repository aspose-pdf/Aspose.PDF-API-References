## ColumnInfo.ColumnWidths-Eigenschaft

Ruft eine Zeichenfolge ab oder legt sie fest, die die Breite der Spalten enthält. Der Wert jeder Spalte sollte durch ein Leerzeichen getrennt sein. Die Standard Einheit ist Punkt, aber cm, Zoll und Prozentsatz der verfügbaren Breite werden ebenfalls unterstützt. Zum Beispiel "120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
