## Stamp.Rotate-Eigenschaft

Setzt oder erhält die Rotation des Stamp-Inhalts gemäß [`Rotation`](../../rotation/) Werten. Hinweis. Diese Eigenschaft ist für festgelegte Winkel, die Vielfache von 90 Grad (0, 90, 180, 270 Grad) sind. Um einen beliebigen Winkel festzulegen, verwenden Sie die RotateAngle-Eigenschaft. Wenn der Winkel, der durch ArbitraryAngle festgelegt wird, kein Vielfaches von 90 ist, gibt die Rotate-Eigenschaft Rotation.None zurück.

```csharp
public Rotation Rotate { get; set; }
```
