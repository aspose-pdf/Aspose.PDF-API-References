## DocSaveOptions.MaxDistanceBetweenTextLines-Eigenschaft

Dieses Parameter wird verwendet, um Textzeilen in Absätze zu gruppieren. Bestimmt, wie weit zwei relative Textzeilen auseinanderliegen können. Angegeben in Hunderten von Prozent der Höhe der Textzeilen.

public float MaxDistanceBetweenTextLines { get; set; }
