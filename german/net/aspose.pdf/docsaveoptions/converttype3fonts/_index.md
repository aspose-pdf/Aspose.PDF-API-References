## DocSaveOptions.ConvertType3Fonts-Eigenschaft

Ruft die Konvertierung für Type3-Schriften ab oder legt sie fest. Bei Type 3-Schriften werden Glyphen durch Streams von Grafikoperatoren definiert. Das bedeutet, dass wir im DOC/DOCX-Ausgang Bilder anstelle von Text sehen. Setzen Sie dieses Flag auf true, um Type3-Schriften in TTF zu konvertieren und Text in der resultierenden Datei zu erhalten.

```csharp
public bool ConvertType3Fonts { get; set; }
```
