## DocSaveOptions.ReSaveFonts-Eigenschaft

Ruft das Verfahren zum erneuten Speichern von Schriftarten ab oder legt es fest. Wenn auf true gesetzt, laden wir Schriftarten auf jeder Seite neu, um den Einfluss vorheriger Schriftarteigenschaften zu vermeiden, und laden die neu erstellte Schriftart von Grund auf neu. Setzen Sie diese Option auf false, wenn Sie die Leistung verbessern möchten. Der Standardwert ist true;

```csharp
public bool ReSaveFonts { get; set; }
```
