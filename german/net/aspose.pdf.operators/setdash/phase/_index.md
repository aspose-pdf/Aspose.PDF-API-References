## SetDash.Phase-Eigenschaft

Dash-Phase. Bevor ein Pfad gestrichen wird, muss das Dash-Array durchlaufen werden, wobei die Längen der Striche und Lücken addiert werden. Wenn die akkumulierte Länge dem Wert entspricht, der durch die Dash-Phase angegeben ist, beginnt das Streichen des Pfades, und das Dash-Array wird ab diesem Punkt zyklisch verwendet.

```csharp
public int Phase { get; set; }
```
