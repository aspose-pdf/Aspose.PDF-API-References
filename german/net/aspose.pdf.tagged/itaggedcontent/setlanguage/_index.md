## ITaggedContent.SetLanguage-Methode

Setzt die natürliche Sprache für das PDF-Dokument.

Ein Sprachbezeichner, der die natürliche Sprache für allen Text im Dokument angeben soll, es sei denn, sie wird durch Sprachspezifikationen für Strukturelemente oder markierte Inhalte überschrieben.

```csharp
public void SetLanguage(string lang)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| lang | String | Ein Sprachbezeichner muss entweder die leere Textzeichenfolge sein, um anzuzeigen, dass die Sprache unbekannt ist, oder ein Language-Tag, wie in RFC 3066, Tags for the Identification of Languages, definiert. |
