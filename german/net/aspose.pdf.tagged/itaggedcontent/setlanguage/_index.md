## ITaggedContent.SetLanguage method

Legt die natürliche Sprache für das PDF-Dokument fest.

Eine Sprachkennung, die die natürliche Sprache für den gesamten Text im Dokument angibt, es sei denn, sie wird durch Sprachspezifikationen für Strukturelemente oder markierte Inhalte außer Kraft gesetzt.

```csharp
public void SetLanguage(string lang)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| lang | String | Eine Sprachkennung muss entweder die leere Textzeichenfolge sein, um anzuzeigen, dass die Sprache unbekannt ist, oder ein Sprach-Tag, wie in RFC 3066, Tags for the Identification of Languages, definiert. |
