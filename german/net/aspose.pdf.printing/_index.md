Der **Aspose.Pdf.Printing**-Namespace bietet Klassen und Funktionen zur Handhabung von PDF-Druckoperationen, einschließlich Druckereinstellungen, Seiteneinstellungen und Erweiterungen zur Verwaltung druckbezogener Konfigurationen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Bietet Daten für das [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/) Ereignis. |
| [PageSettings](./pagesettings/) | Gibt Einstellungen an, die für eine einzelne, gedruckte Seite gelten. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Stellt Erweiterungsmethoden für [`PageSettings`](../aspose.pdf.printing/pagesettings/) dar. |
| [PaperSize](./papersize/) | Gibt die Größe eines Blattes Papier an. |
| [PaperSizeExtensions](./papersizeextensions/) | Stellt Erweiterungsmethoden für [`PaperSize`](../aspose.pdf.printing/papersize/) dar. |
| [PaperSizes](./papersizes/) | Stellt die Standardpapiergrößen dar. |
| [PaperSource](./papersource/) | Gibt die Papierkassette an, aus der der Drucker Papier bezieht. |
| [PaperSourceExtensions](./papersourceextensions/) | Stellt Erweiterungsmethoden für [`PaperSource`](../aspose.pdf.printing/papersource/) dar. |
| [PaperSources](./papersources/) | Bietet eine Menge vordefinierter [`PaperSource`](../aspose.pdf.printing/papersource/) Instanzen, die gängige Papierquellen darstellen. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Bietet Daten für das [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/) Ereignis. |
| [PrinterResolution](./printerresolution/) | Stellt die von einem Drucker unterstützte Auflösung dar. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Stellt Erweiterungsmethoden für [`PrinterResolution`](../aspose.pdf.printing/printerresolution/) dar. |
| [PrinterSettings](./printersettings/) | Gibt Informationen darüber an, wie ein Dokument gedruckt wird, einschließlich des Druckers, der es druckt. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Stellt Erweiterungsmethoden für [`PrinterSettings`](../aspose.pdf.printing/printersettings/) dar. |
| [StartEndPageEventArgs](./startendpageeventargs/) | Bietet Daten für die [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) und [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) Ereignisse der [`PdfViewer`](../aspose.pdf.facades/pdfviewer/) Klasse. |
## Aufzählung
