## PrintRange-Enumeration

Gibt die Option an, die den Teil des Dokuments bezeichnet, der gedruckt werden soll.

```csharp
public enum PrintRange
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| AllPages | `0` | Alle Seiten werden gedruckt. |
| SomePages | `2` | Die Seiten zwischen [`FromPage`](../printersettings/frompage/) und [`ToPage`](../printersettings/topage/) werden gedruckt. |
| Selection | `1` | Die ausgewählten Seiten werden gedruckt. |
| CurrentPage | `4194304` | Die aktuelle Seite wird gedruckt. Bei Verwendung mit einigen Betriebssystemen können alle Seiten gedruckt werden. |
