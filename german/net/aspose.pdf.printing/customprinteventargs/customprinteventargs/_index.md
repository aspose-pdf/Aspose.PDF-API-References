## Konstruktor von CustomPrintEventArgs

Initialisiert [`CustomPrintEventArgs`](../) mit den angegebenen Drucker- und Seiteneinstellungen.

```csharp
public CustomPrintEventArgs(string fileName, PrinterSettings printerSettings, 
    PageSettings pageSettings)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fileName | String | Name der Datei, die gedruckt wird. |
| printerSettings | PrinterSettings | Informationen über den Drucker, auf dem das Dokument gedruckt werden soll. |
| pageSettings | PageSettings | Einstellungen, die auf jede Seite des Dokuments angewendet werden sollen. |
