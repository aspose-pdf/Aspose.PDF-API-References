## ValidationStatus Aufzählung

Stellt den Validierungsstatus einer Zertifikatsvalidierung dar.

```csharp
public enum ValidationStatus
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Gültig | `0` | Gibt an, dass das Zertifikat erfolgreich validiert wurde. |
| Ungültig | `1` | Gibt an, dass die Zertifikatsvalidierung fehlgeschlagen ist. |
| Unbestimmt | `2` | Gibt an, dass der Validierungsprozess nicht schlüssig oder nicht durchgeführt wurde. |

## Anmerkungen

Diese Aufzählung definiert die möglichen Ergebnisse der Zertifikatsvalidierung: - Gültig: Gibt an, dass das Zertifikat erfolgreich validiert wurde. - Ungültig: Gibt an, dass die Zertifikatsvalidierung fehlgeschlagen ist. - Unbestimmt: Gibt an, dass der Validierungsprozess nicht schlüssig oder nicht durchgeführt wurde.
