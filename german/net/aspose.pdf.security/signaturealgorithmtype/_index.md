## SignatureAlgorithmType Enumeration

Enumeriert die Typen von Signaturalgorithmen, die für digitale Signaturen verwendet werden.

```csharp
public enum SignatureAlgorithmType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Ecdsa | `0` | Der Elliptic Curve Digital Signature Algorithm (ECDSA), der für digitale Signaturen verwendet wird. |
| Rsa | `1` | Der Rivest–Shamir–Adleman (RSA) Algorithmus, der für digitale Signaturen verwendet wird. |
| Dsa | `2` | Der Digital Signature Algorithm (DSA), der für digitale Signaturen verwendet wird. |
| Timestamp | `3` | Der Timestamp, der als Signaturalgorithmus verwendet wird, um die digitalen Signaturen zu gewährleisten. |
| Unknown | `4` | Gibt an, dass der Typ des Signaturalgorithmus unbekannt ist. |
