## SignatureLengthMismatchException-Klasse

Stellt Fehler dar, die während der PDF-Signierung auftreten. Tritt auf, wenn [`SignHash`](../../aspose.pdf.forms/signhash/) verwendet wird, um ein Dokument zu signieren, und die tatsächliche Länge der Signatur größer ist als die in der [`DefaultSignatureLength`](../../aspose.pdf.forms/signature/defaultsignaturelength/) Option angegebene.

```csharp
public class SignatureLengthMismatchException : PdfException
```
