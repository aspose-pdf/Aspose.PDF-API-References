Der **Aspose.Pdf.Security** Namespace enthält Klassen, die für die Verschlüsselung und digitale Signatur verwendet werden.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [DsaAlgorithmInfo](./dsaalgorithminfo/) | Stellt eine Klasse für die Informationen über den DSA-Signaturalgorithmus dar. |
| [EcdsaAlgorithmInfo](./ecdsaalgorithminfo/) | Stellt eine Klasse für die Informationen über den ECDSA-Signaturalgorithmus dar. |
| [KeyedSignatureAlgorithmInfo](./keyedsignaturealgorithminfo/) | Stellt eine Klasse für Informationen über einen signierten Signaturalgorithmus dar. |
| [RsaAlgorithmInfo](./rsaalgorithminfo/) | Stellt eine Klasse für die Informationen über den RSA-Signaturalgorithmus dar. |
| [SignatureAlgorithmInfo](./signaturealgorithminfo/) | Stellt eine Klasse für Informationen über einen Signaturalgorithmus dar, einschließlich seines Typs, kryptografischen Standards und Hash-Algorithmus. |
| [SignatureLengthMismatchException](./signaturelengthmismatchexception/) | Stellt Fehler dar, die während der PDF-Signierung auftreten. Tritt auf, wenn [`SignHash`](../aspose.pdf.forms/signhash/) verwendet wird, um ein Dokument zu signieren, und die tatsächliche Länge der Signatur größer ist als die in der [`DefaultSignatureLength`](../aspose.pdf.forms/signature/defaultsignaturelength/) Option angegebene. |
| [TimestampAlgorithmInfo](./timestampalgorithminfo/) | Stellt eine Klasse für die Informationen über den Zeitstempel-Signaturalgorithmus dar. |
| [UnknownSignatureAlgorithmInfo](./unknownsignaturealgorithminfo/) | Stellt eine Klasse für die Informationen über den unbekannten Signaturalgorithmus dar. |
| [ValidationOptions](./validationoptions/) | Stellt Optionen zur Validierung einer digitalen Signatur in einem PDF-Dokument dar. |
| [ValidationResult](./validationresult/) | Stellt das Ergebnis eines Validierungsprozesses für ein Zertifikat dar. |
## Aufzählung
