## ValidationResult.Status-Eigenschaft

Gibt den Status des Validierungsprozesses für ein Zertifikat zurück.

```csharp
public ValidationStatus Status { get; }
```

## Bemerkungen

Die Status-Eigenschaft zeigt das Ergebnis der Zertifikatsvalidierung an. Mögliche Werte sind in der [`ValidationStatus`](../../validationstatus/) Enumeration definiert, wie Gültig, Ungültig oder Unbestimmt. Sie gibt einen Einblick, ob das Zertifikat die Validierungsprüfungen bestanden hat oder nicht.
