## ValidationOptions.RequestTimeout-Eigenschaft

Legt die Timeout-Dauer in Millisekunden für netzwerkbezogene Vorgänge während des Validierungsprozesses fest oder gibt sie zurück. Die RequestTimeout-Eigenschaft definiert die maximale Zeit, die das System warten sollte, um eine Netzwerkantwort zu erhalten, wenn auf Online-Ressourcen wie Widerrufsstatus oder OCSP-Server zugegriffen wird.

```csharp
public int RequestTimeout { get; set; }
```
