## Namespaces

| Namespace | Beschreibung |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | Der **Aspose.Pdf** ist ein Stamm-Namespace für alle Klassen der Aspose.Pdf-Bibliothek, die entweder direkt darin enthalten sind, wie **Document**, oder indirekt über mehrere Unternamespaces. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | Der **Aspose.Pdf.AI** Namespace bietet Klassen für KI-Funktionalitäten, einschließlich API-Clients und intelligenter Assistenten. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | Der **Aspose.Pdf.Annotations** Namespace bietet Klassen zum Arbeiten mit verschiedenen Arten von Aktionen, Zielen und anderen Funktionen von Dokumenten, die traditionell als interaktiv bezeichnet werden und es dem Benutzer ermöglichen, mit ihnen zu interagieren. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | Der **Aspose.Pdf.Comparison** Namespace bietet Klassen, die für den Dokumentenvergleich verwendet werden. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | Der **Aspose.Pdf.DataEditor** enthält Werkzeuge zum Bearbeiten von Daten innerhalb eines Dokuments. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | Der **Aspose.Pdf.Devices** Namespace bietet Klassen, die verwendet werden, um Dokumente als Bild(er) oder einfachen Text darzustellen. So kann das Dokument an textuelle oder verschiedene grafische Geräte gesendet werden, was bedeutet, dass wir eine textuelle oder grafische Darstellung erhalten möchten. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | Der **Aspose.Pdf.Drawing** Namespace bietet Klassen, die grafische Primitive wie Bogen, Linie, Kreis usw. beschreiben. Entsprechende Objekte können zur Erstellung neuer Dokumente verwendet werden. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | Der **Aspose.Pdf.Facades** Namespace bietet Klassen, die ursprünglich aus Aspose.Pdf.Kit stammen. Diese Klassen werden verwendet, um Dokumente zu manipulieren und Operationen wie Zusammenführen, Stempeln, Signieren, Annotieren usw. durchzuführen, jedoch auf hoher Ebene ohne Zugriff auf die innere Struktur eines Dokuments. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | Der **Aspose.Pdf.Forms** Namespace enthält Klassen, die Formulare (standard, statisch, dynamisch) und verschiedene Arten von Feldern wie Textfeld, Listenfeld, Optionsfeld usw. beschreiben. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | Der **Aspose.Pdf.Tagged.LogicalStructure** ist ein Namespace für PDF-Logische Strukturklassen. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | Der **Aspose.Pdf.Multithreading** Namespace bietet Klassen, die für die Verwendung in mehrthreadigen Szenarien gedacht sind. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | Der **Aspose.Pdf.Operators** ist ein Namespace für Operatorimplementierungen. Diese Klassen beschreiben Operatoren, die im Inhalt von PDF-Seiten verwendet werden. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | Der **Aspose.Pdf.Optimization** ist ein Namespace für Klassen zur Verwaltung des Dokumentenoptimierungsprozesses. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | Der **Aspose.Pdf.Plugins** ist ein Stamm-Namespace für alle Klassen der Aspose.Pdf.Plugins-Klassen, die entweder direkt darin enthalten sind, wie **PdfOrganizer**, oder indirekt über mehrere Unternamespaces. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | Der **Aspose.Pdf.Printing** Namespace bietet Klassen und Funktionen für die Handhabung von PDF-Druckoperationen, einschließlich Druckereinstellungen, Seiteneinstellungen und Erweiterungen zur Verwaltung druckbezogener Konfigurationen. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | Der **Aspose.Pdf.Sanitization** ist ein Namespace für Sanitäroperationen. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | Der **Aspose.Pdf.Security** Namespace enthält Klassen, die für die Verschlüsselung und digitale Signatur verwendet werden. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | Der **Aspose.Pdf.Structure** Namespace bietet Klassen, die helfen, mit der logischen Struktur eines Dokuments zu arbeiten, die die Organisation des Dokuments in Kapitel und Abschnitte oder die Identifizierung spezieller Elemente wie Abbildungen, Tabellen und Fußnoten darstellt. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | Der **Aspose.Pdf.Tagged** ist ein Namespace für Klassen zur Unterstützung von Tagged PDF-Dokumenten. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | Der **Aspose.Pdf.Text** Namespace bietet Klassen, die es ermöglichen, Text zu extrahieren, Text hinzuzufügen und bestehenden Text eines Dokuments zu manipulieren. Er enthält auch Klassen, die es ermöglichen, Schriftarten eines Dokuments zu extrahieren, zu ersetzen und zu substituieren. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | Der **Aspose.Pdf.Vector** ist ein Stamm-Namespace für Grafikoperationen. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | Der **Aspose.Pdf.XfaConverter** Namespace bietet Klassen, die es ermöglichen, XFA zu konvertieren. |
