## TiffSettings.SkipBlankPages-Eigenschaft

Ruht einen Wert ab oder legt einen Wert fest, der angibt, ob leere Seiten übersprungen werden sollen.

```csharp
public bool SkipBlankPages { get; set; }
```

### Eigenschaftswert

`true`, wenn leere Seiten übersprungen werden müssen; andernfalls `false`.

## Bemerkungen

Der Standardwert ist false
