### Bemerkungen

Die[`TextDevice`](../textdevice) Das Objekt wird im Wesentlichen verwendet, um Text aus einer PDF-Seite zu extrahieren.

### Beispiele

Das Beispiel zeigt, wie Text auf der ersten PDF-Dokumentseite extrahiert wird.

```csharp
Document doc = new Document(inFile);
string extractedText;

using (MemoryStream ms = new MemoryStream())
{
    // Textgerät erstellen
    TextDevice device = new TextDevice();

    // Konvertiere die Seite und speichere den Text im Stream
    device.Process(doc.Pages[1], ms);

    // den extrahierten Text verwenden
    ms.Close();
    extractedText = Encoding.Unicode.GetString(ms.ToArray());
}
```
