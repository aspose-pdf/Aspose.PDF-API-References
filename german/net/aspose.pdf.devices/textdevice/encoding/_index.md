## TextDevice.Encoding property

Ruft die Codierung von extrahiertem Text ab oder legt sie fest.

```csharp

```

### Beispiele

Das Beispiel zeigt, wie extrahierter Text in UTF-8-Codierung dargestellt wird.

```csharp
Document doc = new Document(inFile);
string extractedText;

// Textgerät erstellen
TextDevice device = new TextDevice(Encoding.UTF8);

// Konvertiere die Seite und speichere den Text im Stream
device.Process(doc.Pages[1], outFile);

// den extrahierten Text verwenden
extractedText = File.ReadAllText(outFile, Encoding.UTF8);
```
