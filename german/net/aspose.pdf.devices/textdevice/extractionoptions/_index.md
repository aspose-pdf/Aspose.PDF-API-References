### Beispiele

Das Beispiel zeigt, wie Text in Rohreihenfolge extrahiert wird.

```csharp
Document doc = new Document(inFile);
string extractedText;

// Textgerät erstellen
TextDevice device = new TextDevice(new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Raw));

// Konvertiere die Seite und speichere den Text im Stream
device.Process(doc.Pages[1], outFile);

// den extrahierten Text verwenden
extractedText = File.ReadAllText(outFile, Encoding.Unicode); 
```
