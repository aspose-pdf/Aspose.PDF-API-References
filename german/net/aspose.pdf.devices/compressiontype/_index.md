## CompressionType Aufzählung

Wird verwendet, um den Parameterwert an ein Tiff-Bildgerät zu übergeben.

```csharp
public enum CompressionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LZW | `0` | Gibt das LZW-Komprimierungsschema an. Kann als Parameter an den Tiff-Encoder übergeben werden, der zur Kategorie Kompression gehört. |
| CCITT4 | `1` | Gibt das CCITT4-Komprimierungsschema an. Kann als Parameter an den CCITT4-Encoder übergeben werden, der zur Kategorie Kompression gehört. |
| CCITT3 | `2` | Gibt das CCITT3-Komprimierungsschema an. Kann als Parameter an den CCITT3-Encoder übergeben werden, der zur Kategorie Kompression gehört. |
| RLE | `3` | Gibt das RLE-Komprimierungsschema an. Kann als Parameter an den RLE-Encoder übergeben werden, der zur Kategorie Kompression gehört. |
| None | `4` | Gibt an, dass keine Kompression erfolgt. Kann als Parameter an den Tiff-Encoder übergeben werden, der zur Kategorie Kompression gehört. |
