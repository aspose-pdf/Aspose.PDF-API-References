## CompressionType enumeration

Wird verwendet, um den Parameterwert anzugeben, der an ein TIFF-Bildgerät übergeben wird.

```csharp
public enum CompressionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| LZW | `0` | Gibt das LZW-Komprimierungsschema an. Kann als Parameter, der zur Kategorie Compression gehört, an den Tiff-Encoder übergeben werden. |
| CCITT4 | `1` | Gibt das CCITT4-Komprimierungsschema an. Kann an den CCITT4-Encoder als Parameter übergeben werden, der zur Kategorie Compression gehört. |
| CCITT3 | `2` | Gibt das CCITT3-Komprimierungsschema an. Kann an den CCITT3-Encoder als Parameter übergeben werden, der zur Kategorie Compression gehört. |
| RLE | `3` | Gibt das RLE-Komprimierungsschema an. Kann als Parameter an den RLE-Encoder übergeben werden, der zur Kategorie Compression gehört. |
| None | `4` | Gibt keine Komprimierung an. Kann dem Tiff-Encoder als Parameter übergeben werden, der zur Komprimierungskategorie gehört. |
