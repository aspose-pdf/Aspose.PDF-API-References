## FitBHExplicitDestination-Klasse

Stellt ein explizites Ziel dar, das die Seite mit der vertikalen Koordinate top an der oberen Kante des Fensters positioniert und den Inhalt der Seite gerade so vergrößert, dass die gesamte Breite des Begrenzungsrahmens innerhalb des Fensters passt. Ein Nullwert für top gibt an, dass der aktuelle Wert dieses Parameters unverändert beibehalten werden soll.

```csharp
public sealed class FitBHExplicitDestination : ExplicitDestination
```
