## MovieAnnotation.Poster-Eigenschaft

Ruft ein Flag oder einen Stream ab oder legt fest, ob und wie ein Posterbild, das den Film darstellt, angezeigt werden soll. Wenn true, wird das Posterbild aus der Filmdatei abgerufen; wenn false, wird kein Poster angezeigt.

public bool Poster { get; set; }
