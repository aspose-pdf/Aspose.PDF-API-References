## HighlightingMode-Enumeration

Enumeriert den Hervorhebungsmodus der Annotation, den visuellen Effekt, der verwendet wird, wenn die Maustaste innerhalb des aktiven Bereichs gedrückt oder gehalten wird.

```csharp
public enum HighlightingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine Hervorhebung. |
| Invert | `1` | Den Inhalt des Annotationsrechtecks umkehren. |
| Outline | `2` | Den Rand der Annotation umkehren. |
| Push | `3` | Das Aussehen der Annotation im gedrückten Zustand anzeigen, falls vorhanden. Wenn kein gedrücktes Aussehen definiert ist, den Inhalt des Annotationsrechtecks so verschieben, dass es aussieht, als würde es unter die Oberfläche der Seite gedrückt. |
| Toggle | `4` | Dasselbe wie Push (was bevorzugt wird). |
