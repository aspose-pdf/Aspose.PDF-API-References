## HighlightingMode enumeration

Zählt den Hervorhebungsmodus der Anmerkung auf, den visuellen Effekt, der verwendet werden soll, wenn die Maustaste in ihrem aktiven Bereich gedrückt oder gehalten wird.

```csharp
public enum HighlightingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine Hervorhebung. |
| Invert | `1` | Kehrt den Inhalt des Anmerkungsrechtecks um. |
| Outline | `2` | Kehren Sie den Rand der Anmerkung um. |
| Push | `3` | Zeigt die nach unten gerichtete Darstellung der Anmerkung an, falls vorhanden. Wenn kein Erscheinungsbild nach unten definiert ist, versetzen Sie den Inhalt des Anmerkungsrechtecks so, dass es so aussieht, als würde es unter die Oberfläche der Seite geschoben. |
| Toggle | `4` | Dasselbe wie Push (was bevorzugt wird). |
