## FreeTextAnnotation.TextRectangle-Eigenschaft

Rechteck, das die numerischen Unterschiede zwischen zwei Rechtecken beschreibt: dem Rect-Eintrag der Annotation und einem Rechteck, das innerhalb dieses Rechtecks enthalten ist. Das innere Rechteck ist der Ort, an dem der Text der Annotation angezeigt werden soll.

public Rectangle TextRectangle { get; set; }
