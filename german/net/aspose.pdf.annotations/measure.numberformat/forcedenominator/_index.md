## Measure.NumberFormat.ForceDenominator-Eigenschaft

Wenn FractionDisplayment als ShowAsFraction angezeigt wird, bestimmt dieser Wert, ob der Bruch reduziert werden kann oder nicht. Wenn der Wert true ist, kann der Bruch nicht reduziert werden.

public bool ForceDenominator { get; set; }
