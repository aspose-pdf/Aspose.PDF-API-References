## PrinterMarksKind Enumeration

Gibt die Arten von Druckmarken an, die zu einem Dokument hinzugefügt werden sollen.

```csharp
[Flags]
public enum PrinterMarksKind
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Gibt an, dass keine Druckmarken hinzugefügt werden sollen. |
| TrimMarks | `1` | Gibt an, dass Beschnittmarken hinzugefügt werden sollen. |
| BleedMarks | `2` | Gibt an, dass Anschnittmarken hinzugefügt werden sollen. |
| RegistrationMarks | `4` | Gibt an, dass Registrierungsmarken hinzugefügt werden sollen. |
| ColorBars | `8` | Gibt an, dass Farbtafeln hinzugefügt werden sollen. |
| PageInformation | `10` | Gibt an, dass Seiteninformationen hinzugefügt werden sollen. |
| All | `1F` | Gibt an, dass alle Arten von Druckmarken hinzugefügt werden sollen. |

## Anmerkungen

Diese Enumeration hat ein FlagsAttribute-Attribut, das eine bitweise Kombination ihrer Mitgliedswerte ermöglicht.
