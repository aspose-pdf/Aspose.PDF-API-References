## TextMarkupAnnotation.QuadPoints-Eigenschaft

Ruft ein Array von Punkten ab oder legt es fest, das die Koordinaten von n Vierecken angibt. Jedes Viereck umfasst ein Wort oder eine Gruppe von zusammenhängenden Wörtern im Text, der der Annotation zugrunde liegt.
