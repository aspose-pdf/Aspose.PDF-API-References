## LineEnding Enumeration

Enumeriert die Linienendstile, die beim Zeichnen der Linie verwendet werden sollen.

```csharp
public enum LineEnding
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Kein Linienende. |
| Square | `1` | Ein Quadrat, gefüllt mit der Innenfarbe der Annotation, falls vorhanden. |
| Circle | `2` | Ein Kreis, gefüllt mit der Innenfarbe der Annotation, falls vorhanden. |
| Diamond | `3` | Eine Raute, gefüllt mit der Innenfarbe der Annotation, falls vorhanden. |
| OpenArrow | `4` | Zwei kurze Linien, die sich in einem spitzen Winkel treffen, um eine offene Pfeilspitze zu bilden. |
| ClosedArrow | `5` | Zwei kurze Linien, die sich in einem spitzen Winkel wie im OpenArrow-Stil treffen und durch eine dritte Linie verbunden sind, um eine dreieckige geschlossene Pfeilspitze zu bilden, die mit der Innenfarbe der Annotation gefüllt ist, falls vorhanden. |
| Butt | `6` | Eine kurze Linie am Endpunkt, die senkrecht zur Linie selbst steht. |
| ROpenArrow | `7` | Zwei kurze Linien in die entgegengesetzte Richtung von OpenArrow. |
| RClosedArrow | `8` | Eine dreieckige geschlossene Pfeilspitze in die entgegengesetzte Richtung von ClosedArrow. |
| Slash | `9` | Eine kurze Linie am Endpunkt, die ungefähr 30 Grad im Uhrzeigersinn von der Senkrechten zur Linie selbst steht. |
