## Annotation.UpdateAppearanceOnConvert-Eigenschaft

Wenn wahr, wird das Erscheinungsbild der Annotation aktualisiert, bevor das PF-Dokument in ein Bild konvertiert wird. Dies ermöglicht eine korrekte Konvertierung der Felder, benötigt jedoch wahrscheinlich mehr Zeit.

```csharp
public static bool UpdateAppearanceOnConvert { get; set; }
```
