## Annotation.UseFontSubset-Eigenschaft

Wenn diese Eigenschaft auf true gesetzt ist, werden Schriftarten als Teilmengen zum Dokument hinzugefügt. Der Standardwert ist true.

```csharp
public static bool UseFontSubset { get; set; }
```
