## AnnotationState-Aufzählung

Die Aufzählung der Zustände, auf die die ursprüngliche Annotation gesetzt werden kann.

```csharp
public enum AnnotationState
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Undefined | `0` | Nicht definierter Zustand. |
| Marked | `1` | Die Annotation wurde vom Benutzer markiert. |
| Unmarked | `2` | Die Annotation wurde vom Benutzer nicht markiert. |
| Accepted | `3` | Der Benutzer stimmt der Änderung zu. |
| Rejected | `4` | Der Benutzer stimmt der Änderung nicht zu. |
| Cancelled | `5` | Die Änderung wurde storniert. |
| Completed | `6` | Die Änderung wurde abgeschlossen. |
| None | `7` | Der Benutzer hat nichts zur Änderung angegeben. |
