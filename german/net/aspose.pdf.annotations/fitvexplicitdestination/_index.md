## FitVExplicitDestination-Klasse

Stellt ein explizites Ziel dar, das die Seite mit der horizontalen Koordinate links am linken Rand des Fensters positioniert und den Inhalt der Seite so vergrößert, dass die gesamte Höhe der Seite im Fenster passt. Ein Nullwert für links gibt an, dass der aktuelle Wert dieses Parameters unverändert beibehalten werden soll.

```csharp
public sealed class FitVExplicitDestination : ExplicitDestination
```
