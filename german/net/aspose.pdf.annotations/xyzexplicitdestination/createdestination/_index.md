## XYZExplicitDestination.CreateDestination-Methode

Erstellen Sie ein Ziel an einem bestimmten Ort der Seite unter Berücksichtigung der Seitenrotation, falls erforderlich.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| page | Page | Zielseite. |
| left | Double | Linke Position auf der Seite. |
| top | Double | Obere Position auf der Seite. |
| zoom | Double | Zoomfaktor (0 für Standard). |
| considerRotation | Boolean | Wenn wahr, wird die Position gemäß der Seitenrotation neu berechnet. |
