## ReplyType Enumeration

Enumeriert die Arten der Beziehungen (der "Antworttyp") zwischen der Annotation und einer durch InReplyTo angegebenen.

```csharp
public enum ReplyType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Undefined | `0` | Unbestimmte Beziehung. |
| Reply | `1` | Die Annotation wird als Antwort auf die durch InReplyTo angegebene Annotation betrachtet. Viewer-Anwendungen sollten Antworten auf eine Annotation nicht einzeln anzeigen, sondern zusammen in Form von zusammenhängenden Kommentaren. |
| Group | `2` | Die Annotation ist mit der durch InReplyTo angegebenen Annotation gruppiert. |
