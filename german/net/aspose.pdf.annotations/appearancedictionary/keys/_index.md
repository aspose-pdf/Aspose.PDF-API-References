## AppearanceDictionary.Keys-Eigenschaft

Gibt die Schlüssel des Wörterbuchs zurück. Wenn das Erscheinungswörterbuch Unterwörterbücher hat, enthält `Keys` (N&#x7C;R&#x7C;D).state-Werte, wobei N - normales Erscheinungsbild, R - Hover-Erscheinungsbild, D - gedrücktes Erscheinungsbild und state - der Name des Zustands (z.B. Ein, Aus für Kontrollkästchen).

```csharp
public ICollection<string> Keys { get; }
```
