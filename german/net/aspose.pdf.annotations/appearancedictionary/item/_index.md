## AppearanceDictionary-Indexer

Stellt eine bequeme Form zum Abrufen von Erscheinungsströmen dar.

```csharp
public XForm this[string key] { get; set; }
```

| Parameter | Beschreibung |
| --- | --- |
| key | Stellt den Pfad zum Erscheinungsstrom dar. Wenn das Erscheinungswörterbuch Unterwörterbücher hat, muss der Pfad 2 Teile enthalten ([`Keys`](../keys/)), andernfalls hat der Pfad nur einen Teil. |

### Rückgabewert

XForm-Objekt (Erscheinungsstrom), das dem angegebenen Schlüssel entspricht.
