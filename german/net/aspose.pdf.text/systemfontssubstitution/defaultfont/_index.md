## SystemFontsSubstitution.DefaultFont-Eigenschaft

Ruht die Standardersatzschrift ab oder legt sie fest. Die Schriftart wird verwendet, wenn keine andere gültige Ersatzschrift gefunden wurde, aber die ursprüngliche Schriftart zur Zielersatzkategorie ([`FontCategories`](../fontcategories/)) gehört.

```csharp
public Font DefaultFont { get; set; }
```
