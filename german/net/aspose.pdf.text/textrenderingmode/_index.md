## TextRenderingMode Aufzählung

Der Textdarstellungsmodus, Tmode, bestimmt, ob das Anzeigen von Text dazu führen soll, dass Glyphenumrisse gestrichen, gefüllt, als Clipping-Grenze verwendet oder eine Kombination aus dreien verwendet wird.

```csharp
public enum TextRenderingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| FillText | `0` | Text füllen. |
| StrokeText | `1` | Text streichen. |
| FillThenStrokeText | `2` | Zuerst füllen, dann streichen. |
| Invisible | `3` | Weder füllen noch streichen (unsichtbar). |
| FillTextAndAddPathToClipping | `4` | Text füllen und zum Pfad für das Clipping hinzufügen (siehe 9.3.6, "Textdarstellungsmodus"). |
| StrokeTextAndAddPathToClipping | `5` | Text streichen und zum Pfad für das Clipping hinzufügen. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Zuerst füllen, dann streichen und zum Pfad für das Clipping hinzufügen. |
| AddPathToClipping | `7` | Text zum Pfad für das Clipping hinzufügen. |
