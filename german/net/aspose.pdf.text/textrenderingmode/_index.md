## TextRenderingMode enumeration

Der Textdarstellungsmodus, Tmode, legt fest, ob das Anzeigen von Text dazu führen soll, dass Glyphenumrisse gestrichen, gefüllt, als Begrenzungsbegrenzung verwendet werden oder eine Kombination der drei.

```csharp
public enum TextRenderingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| FillText | `0` | Fülltext. |
| StrokeText | `1` | Strichtext. |
| FillThenStrokeText | `2` | Text ausfüllen, dann streichen. |
| Invisible | `3` | Weder Füll- noch Strichtext (unsichtbar). |
| FillTextAndAddPathToClipping | `4` | Füllen Sie Text aus und fügen Sie ihn zum Beschneiden zum Pfad hinzu (siehe 9.3.6, "Textwiedergabemodus"). |
| StrokeTextAndAddPathToClipping | `5` | Text streichen und zum Ausschneiden zum Pfad hinzufügen. |
| FillThenStrokeTextAndAddPathToClipping | `6` | Füllen, dann Text streichen und zum Ausschneiden zum Pfad hinzufügen. |
| AddPathToClipping | `7` | Text zum Ausschneiden zum Pfad hinzufügen. |
