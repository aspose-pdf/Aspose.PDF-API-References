## TextEditOptions.ClippingPathsProcessingMode Aufzählung

Verarbeitungsmodi für Clipping-Pfade

```csharp
public enum ClippingPathsProcessingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| KeepIntact | `0` | Beibehaltung der Clipping-Pfade des ursprünglichen Seitenlayouts. (Standard) |
| Expand | `1` | Der ursprüngliche Clipping-Pfad wird erweitert, falls der bearbeitete Text mehr Platz benötigt. |
| Remove | `2` | Der ursprüngliche Clipping-Pfad wird entfernt, falls der bearbeitete Text mehr Platz benötigt. Vorsicht: Da Clipping-Pfade miteinander interagieren können, kann das Entfernen zu unerwarteten Ergebnissen im Seitenlayout führen. |
