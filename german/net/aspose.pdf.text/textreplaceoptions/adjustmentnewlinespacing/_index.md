## TextReplaceOptions.AdjustmentNewLineSpacing-Eigenschaft

Ruft den Wert des Zeilenabstands ab oder legt ihn fest, der verwendet wird, wenn die Ersetzungsanpassung gezwungen ist, eine neue Textzeile zu erstellen. Der erwartete Wert ist ein Vielfaches der Schriftgröße des ersetzten Textes. Standard ist 1.2.

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
