## TextFragmentAbsorber.Errors-Eigenschaft

Liste der [`TextExtractionError`](../../textextractionerror/) Objekte. Sie enthält Informationen über Fehler, die während der Textextraktion gefunden wurden. Die Suche nach Fehlern wird nur durchgeführt, wenn TextSearchOptions.LogTextExtractionErrors = true ist; und dies kann die Leistung beeinträchtigen.

```csharp
public List<TextExtractionError> Errors { get; }
```
