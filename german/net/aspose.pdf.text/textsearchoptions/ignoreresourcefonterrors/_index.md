## TextSearchOptions.IgnoreResourceFontErrors-Eigenschaft

Gibt an, dass Fehler im Zusammenhang mit dem Fehlen von Schriftarten vom Text (Fragment)-Absorber ignoriert werden. true - bedeutet, dass Fehler aufgrund des Fehlens von Schriftarten ignoriert werden. Textsegmente, die auf falsche Ressourcen verweisen, werden während der Verarbeitung übersprungen. false (Standard) - Fehler aufgrund des Fehlens von Schriftarten beenden die Verarbeitung, indem eine Ausnahme ausgelöst wird.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
