## TextSearchOptions.IgnoreShadowText property

Erhält oder setzt die Angabe, dass Textfragmente, die Schatten von normalem Text darstellen, während der Suche ignoriert werden. wahr – bedeutet, dass Schattentext nicht gefunden wird (versuchen Sie dies, wenn die Textsuche doppelte Fragmente an den nahen Positionen zurückgibt) falsch – bedeutet, dass Schatten Text wird ebenso gefunden wie normaler Text (Standardwert)
