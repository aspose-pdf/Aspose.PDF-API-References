## TextSearchOptions.IgnoreShadowText-Eigenschaft

Gibt an, dass Textfragmente, die den Schatten von normalem Text darstellen, während der Suche ignoriert werden. true - bedeutet, dass Schattentext nicht gefunden wird (versuchen Sie dies, wenn die Textsuche doppelte Fragmente an nahen Positionen zurückgibt) false - bedeutet, dass Schattentext sowie normaler Text gefunden wird (Standardwert)

```csharp
public bool IgnoreShadowText { get; set; }
```
