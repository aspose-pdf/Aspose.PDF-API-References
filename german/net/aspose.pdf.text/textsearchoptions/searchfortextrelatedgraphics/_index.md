## TextSearchOptions.SearchForTextRelatedGraphics-Eigenschaft

Ruft den Wert ab oder legt ihn fest, der das Suchen nach textbezogenen Grafiken (Unterstreichung, Hintergrund usw.) während der Textsuche ermöglicht. true - das Suchen nach textbezogenen Grafiken wird durchgeführt (Standardwert). false - grafische Elemente, die im Quelldokument vorhanden sein können, werden ignoriert. Setzen Sie dies im Falle von Leistungsproblemen oder wenn es nicht erforderlich ist, Unterstreichungen, Hintergründe oder Zuschnitte zu behandeln.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
