## TextSearchOptions.UseFontEngineEncoding-Eigenschaft

Gibt an, dass der Text mit der Schriftarten-Engine-Codierung durchsucht wird. true - bedeutet, dass die Schriftarten-Engine-Codierung verwendet wird (versuchen Sie dies, wenn die Textsuche aufgrund unvollständiger Codierung im Dokument fehlschlägt) false - bedeutet, dass die Dokumentschriftarten-Codierung verwendet wird (Standardwert)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
