## TextSearchOptions.UseFontEngineEncoding property

Erhält oder setzt die Angabe, dass Text mit der Zeichensatz-Engine-Codierung gesucht wird. true – bedeutet, dass die Zeichensatz-Engine-Codierung verwendet wird (versuchen Sie dies, wenn die Textsuche aufgrund einer unvollständigen Codierung im Dokument fehlschlägt) false – bedeutet, dass die Zeichensatz-Codierung des Dokuments verwendet wird verwendet werden (Standardwert)

public bool UseFontEngineEncoding { get; set; }
