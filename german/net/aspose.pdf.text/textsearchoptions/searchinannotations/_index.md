## TextSearchOptions.SearchInAnnotations-Eigenschaft

Erhält oder setzt den Wert, der das Suchen nach Text in Anmerkungen erlaubt. true - Text wird in Anmerkungen gesucht. false - Text in Anmerkungen wird nicht von TextFragmentAbsorber analysiert.

public bool SearchInAnnotations { get; set; }
