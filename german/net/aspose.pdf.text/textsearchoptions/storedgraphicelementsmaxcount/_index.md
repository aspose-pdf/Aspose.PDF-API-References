## TextSearchOptions.StoredGraphicElementsMaxCount-Eigenschaft

Ruft den Wert ab oder legt ihn fest, der die Suche nach textbezogenen Grafiken (Unterstreichungen, Hintergrund usw.) auf einer Seite für die angegebene Anzahl von Elementen einschränkt. Der Standardwert beträgt 250. Setzen Sie einen niedrigeren Wert im Falle von Leistungsproblemen, versuchen Sie einen höheren Wert, falls einige grafische Elemente nicht gefunden wurden.

```csharp
public int StoredGraphicElementsMaxCount { get; set; }
```
