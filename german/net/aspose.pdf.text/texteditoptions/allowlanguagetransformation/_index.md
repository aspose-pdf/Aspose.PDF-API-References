## TextEditOptions.AllowLanguageTransformation-Eigenschaft

Ruft den Wert ab oder legt ihn fest, der die Verwendung von Sprachtransformationen beim Hinzufügen oder Bearbeiten von Text erlaubt. true - Sprachtransformation wird angewendet, wenn nötig (Standardwert). false - Sprachtransformation wird NICHT angewendet.

```csharp
public bool AllowLanguageTransformation { get; set; }
```
