## TextState.CoordinateOrigin-Eigenschaft

Ruft den Text CoordinateOrigin ab oder legt ihn fest. Wenn CoordinateOrigin Descender ist, entspricht die Y-Koordinate des Textes dem tiefsten Punkt der Schriftart. Wenn CoordinateOrigin BaseLine ist, entspricht die Y-Koordinate des Textes der Grundlinie der Schriftart. Der Standardwert ist Descender. Wenn der Abwärtswert der Schriftart zu groß ist, kann der Text höher als andere Schriftarten gerendert werden. In diesem Fall kann CoordinateOrigin BaseLine für eine bessere Textdarstellung ausgewählt werden.

```csharp
public virtual CoordinateOrigin CoordinateOrigin { get; set; }
```
