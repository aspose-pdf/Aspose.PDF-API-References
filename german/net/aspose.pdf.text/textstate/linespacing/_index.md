## TextState.LineSpacing-Eigenschaft

Ruft den Zeilenabstand des Textes ab oder legt ihn fest.

```csharp
public virtual float LineSpacing { get; set; }
```

## Bemerkungen

Beachten Sie, dass der Wert nicht als Textmerkmal innerhalb des Dokuments gespeichert wird. Der Getter der LineSpacing-Eigenschaft funktioniert für ein Objekt, falls es zuvor explizit mit dem Setter der LineSpacing-Eigenschaft für dieses Objekt festgelegt wurde. Die Eigenschaft wird zur Laufzeit im Kontext des aktuellen Generierungs-/Änderungsprozesses verwendet.
