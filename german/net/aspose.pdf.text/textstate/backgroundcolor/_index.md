## TextState.BackgroundColor-Eigenschaft

Legt die Hintergrundfarbe des Textes fest.

```csharp
public virtual Color BackgroundColor { get; set; }
```

## Bemerkungen

Beachten Sie, dass der Wert nicht als Texteigenschaft innerhalb des Dokuments gespeichert wird. Der BackgroundColor-Eigenschafts-Getter funktioniert für ein Objekt, falls es zuvor explizit mit dem BackgroundColor-Setter für dieses Objekt festgelegt wurde. Die Eigenschaft wird zur Laufzeit im Kontext des aktuellen Generierungs-/Modifizierungsprozesses verwendet.
