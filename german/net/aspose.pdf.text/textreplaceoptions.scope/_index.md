## TextReplaceOptions.Scope Aufzählung

Geltungsbereich, in dem die Textersetzungsoperation standardmäßig auf REPLACE_FIRST angewendet wird. Diese veraltete Option wurde aus Kompatibilitätsgründen beibehalten. Sie wirkt sich auf PdfContentEditor aus und hat keine Auswirkungen auf TextFragmentAbsorber.

```csharp
public enum Scope
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| REPLACE_FIRST | `0` | Ersetze nur das erste Vorkommen des Textes auf jeder der betroffenen Seiten |
| REPLACE_ALL | `1` | Ersetze alle Textvorkommen auf allen betroffenen Seiten |
