## CustomFontSubstitutionBase.TrySubstitute method

Ersetzt die ursprüngliche Schriftart durch eine andere Schriftart.

```csharp
public virtual bool TrySubstitute(OriginalFontSpecification originalFontSpecification, 
    out Font substitutionFont)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| originalFontSpecification | OriginalFontSpecification | Ursprüngliche Schriftartspezifikation. |
| substitutionFont | Font& | Substitutionsschrift. |

### Rückgabewert

True, falls die Substitution erfolgreich war.

### Bemerkungen

Die Klasse CustomFontSubstitutionBase sollte geerbt werden, um eine benutzerdefinierte Schriftersetzungslogik zu implementieren. Die TrySubstitute-Methode sollte ordnungsgemäß überschrieben werden: Muss wahr zurückgeben, falls eine Ersetzung erforderlich ist. substitutionFont muss auf ein gültiges Font-Objekt gesetzt werden. Muss false zurückgeben, falls keine Ersetzung erforderlich ist. substitutionFont kann auf null gesetzt werden.
