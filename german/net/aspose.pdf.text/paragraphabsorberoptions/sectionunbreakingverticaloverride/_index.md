## ParagraphAbsorberOptions.SectionUnbreakingVerticalOverride-Eigenschaft

Ruft die maximale Länge der Zone mit 'null Füllstand' ab oder legt sie fest, die nicht als tatsächlicher vertikaler Abschnittswechsel behandelt wird (als Bruchteil der Breite / Höhe einer Seite). Der Standardwert ist 0.005.

```csharp
public double SectionUnbreakingVerticalOverride { get; set; }
```
