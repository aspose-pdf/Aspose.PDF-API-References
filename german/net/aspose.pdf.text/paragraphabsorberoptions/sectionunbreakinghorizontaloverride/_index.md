## ParagraphAbsorberOptions.SectionUnbreakingHorizontalOverride-Eigenschaft

Ruft die maximale Länge der Zone mit 'Nullfüllstand' ab oder legt sie fest, die nicht als tatsächlicher horizontaler Abschnittswechsel behandelt wird (als Bruchteil der Breite / Höhe einer Seite). Der Standardwert ist 0.005.

```csharp
public double SectionUnbreakingHorizontalOverride { get; set; }
```
