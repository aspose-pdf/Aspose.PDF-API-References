## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable property

Ruft einen Wert ab, der angibt, dass die Ersetzung unvermeidbar ist.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

### Bemerkungen

Gibt „true“ zurück, falls eine Ersetzung wegen Fehlens der Originalschriftart angefordert wurde oder falls die Originalschriftart im Zusammenhang mit einer Aufgabe nicht verwendet werden kann. Falls der Benutzer das Flag ignoriert und die Schrift nicht ersetzt, wird das Verfahren zum Ersetzen der Standardschriftart durchgeführt . Aber es bietet dem Benutzer die Möglichkeit, das Standardverfahren zum Ersetzen von Schriftarten zu ändern und eine bessere Schriftart für das System festzulegen. Gibt falsch zurück, falls die ursprüngliche Schriftart vorhanden ist, gültig, aber es ist dem Benutzer erlaubt, sie zu ersetzen.
