## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable-Eigenschaft

Gibt einen Wert zurück, der angibt, dass die Substitution unvermeidlich ist.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

## Bemerkungen

Gibt true zurück, falls eine Substitution aufgrund der Abwesenheit der ursprünglichen Schriftart angefordert wurde oder falls die ursprüngliche Schriftart im Kontext einer bestimmten Aufgabe nicht verwendet werden kann. Falls der Benutzer das Flag ignoriert und die Schriftart nicht substituiert - wird das Standardverfahren zur Schriftartsubstitution durchgeführt. Es bietet jedoch die Möglichkeit für den Benutzer, das Standardverfahren zur Schriftartsubstitution zu ändern und eine bessere Schriftart für das System festzulegen. Gibt false zurück, falls die ursprüngliche Schriftart vorhanden, gültig ist, es dem Benutzer jedoch erlaubt ist, sie zu substituieren.
