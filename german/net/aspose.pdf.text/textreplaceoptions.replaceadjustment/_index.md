## TextReplaceOptions.ReplaceAdjustment enumeration

Legt die Aktion fest, die nach dem Ersetzen des Textfragments zu einem kürzeren ausgeführt wird. Keine - keine Aktion, ersetzter Text kann den Rest der Zeile überlappen; AdjustSpaceWidth - versucht, Leerzeichen zwischen Wörtern anzupassen, um die Zeilenlänge beizubehalten; WholeWordsHyphenation - versucht, Wörter zu verteilen zwischen den Absatzzeilen, um das rechte Feld des Absatzes beizubehalten; ShiftRestOfLine - verschiebt den Rest der Zeile entsprechend der sich ändernden Textlänge, die Länge der Zeile kann geändert werden; Der Standardwert ist ShiftRestOfLine.

```csharp
[Flags]
public enum ReplaceAdjustment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine Aktion, ersetzter Text überlappt möglicherweise den Rest der Zeile |
| AdjustSpaceWidth | `1` | Versucht, Leerzeichen zwischen Wörtern anzupassen, um die Zeilenlänge beizubehalten |
| WholeWordsHyphenation | `2` | Versucht, Wörter zwischen Absatzzeilen zu verteilen, um das rechte Feld des Absatzes beizubehalten |
| ShiftRestOfLine | `4` | (Standard) Verschiebt den Rest der Zeile entsprechend der sich ändernden Textlänge, die Länge der Zeile kann geändert werden |
