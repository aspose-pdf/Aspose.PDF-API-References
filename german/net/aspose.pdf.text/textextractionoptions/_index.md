## Konstruktoren

| --- | --- |

## Eigenschaften

| Name | Beschreibung |
| --- | --- |
| [FormattingMode](../../aspose.pdf.text/textextractionoptions/formattingmode/) { get; set; } | Gibt den Formatierungsmodus zurück. |
| [ScaleFactor](../../aspose.pdf.text/textextractionoptions/scalefactor/) { get; set; } | Gibt den Faktor zurück oder legt ihn fest, der während der Extraktion im reinen Modus auf die Schriftgröße angewendet wird. Eine Einstellung mit einem geringeren Wert führt zu mehr Leerzeichen im extrahierten Text. Der Standardwert ist 1 - keine Skalierung; Das Festlegen des Wertes auf null ermöglicht es dem Algorithmus, die Skalierung automatisch auszuwählen. |
