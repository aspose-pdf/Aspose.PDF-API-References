## TextExtractionOptions.ScaleFactor-Eigenschaft

Ruft den Faktor ab oder legt ihn fest, der während der Extraktion im reinen Modus auf die Schriftgröße angewendet wird. Eine Einstellung mit einem niedrigeren Wert führt zu mehr Leerzeichen im extrahierten Text. Der Standardwert ist 1 - keine Skalierung; Das Setzen des Wertes auf null ermöglicht es dem Algorithmus, die Skalierung automatisch auszuwählen.

```csharp
public double ScaleFactor { get; set; }
```
