## TextAbsorber.HasErrors-Eigenschaft

Der Wert gibt an, ob während der Textextraktion Fehler gefunden wurden. Die Suche nach Fehlern wird nur durchgeführt, wenn TextSearchOptions.LogTextExtractionErrors = true; und dies kann die Leistung beeinträchtigen.

```csharp
public bool HasErrors { get; }
```
