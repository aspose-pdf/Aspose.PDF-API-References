## AppendLine(string, float) {#appendline_6}

Fügt eine Textzeile hinzu.

```csharp
public void AppendLine(string line, float lineSpacing)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| line | String | Der Text der neuen Zeile. |
| lineSpacing | Single | Zusätzlicher Abstand (0.0 ist Standard und entspricht der Standardhöhe der Textzeile). Der Abstandswert wird zum Standardzeilenabstand für die jeweilige Zeile hinzugefügt, sodass Sie 12.0 angeben können, um eine leere Zeile NACH einer mit 12pt Schriftart gezeichneten Textzeile zu erhalten. |
