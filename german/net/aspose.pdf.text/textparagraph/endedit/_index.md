
## Bemerkungen

Verbessert die Leistung der TextParagraph-Befüllung. Jede Layoutberechnung wird ausgesetzt, bis die EndEdit-Methode aufgerufen wird. Beachten Sie, dass der Methodenaufruf nicht geschachtelt werden kann.
