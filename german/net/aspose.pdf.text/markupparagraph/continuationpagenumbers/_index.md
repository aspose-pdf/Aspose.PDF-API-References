## MarkupParagraph.ContinuationPageNumbers-Eigenschaft

Liste der Seitenzahlen, auf denen der Absatz fortgesetzt wird. Es wird mit der Seite übereinstimmen, auf der der Absatz begonnen hat, wenn er in der nächsten Spalte auf derselben Seite fortgesetzt wird.

```csharp
public List<int> ContinuationPageNumbers { get; }
```
