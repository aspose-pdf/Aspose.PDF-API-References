## ParagraphAbsorber.SectionsSearchDepth property

Erhält oder setzt einen Wert, der angibt, wie oft sequentielle Suchen nach feineren Strukturelementen durchgeführt werden. Die Standardsuchtiefe ist 3. Dies bedeutet drei Suchen nach horizontal geteilten Abschnitten (Überschriften, Absätze usw.) und drei Suchen nach vertikal geteilten Einsen (Spalten).

```csharp
public int SectionsSearchDepth { get; set; }
```

### Bemerkungen

Das Erhöhen dieses Werts kann zu einer geringfügigen Verringerung der Leistung ohne sichtbare Änderungen im Suchergebnis führen. Das Verringern dieses Werts kann zu einer falschen Bestimmung von Absätzen in Abschnitten führen. Wir empfehlen nicht, den Wert auf einen niedrigeren Wert als den Standardwert einzustellen, wenn dies nicht der Fall ist Wunsch, nur 'grobe' Elemente der Seitenstruktur zu erhalten.
