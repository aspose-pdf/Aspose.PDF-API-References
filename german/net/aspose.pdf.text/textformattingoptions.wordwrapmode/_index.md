## TextFormattingOptions.WordWrapMode Aufzählung

Definiert Strategien für den Zeilenumbruch

```csharp
public enum WordWrapMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NoWrap | `0` | Es wird kein Umbruch durchgeführt |
| DiscretionaryHyphenation | `1` | Es wird ein optionaler Trennstrich durchgeführt. Erlaubt das Brechen von Wörtern in der Mitte. |
| ByWords | `2` | Der Zeilenumbruch erfolgt nur bei vollständigen Wörtern. Wenn das vollständige Wort nicht umgebrochen werden kann, wird versucht, einen optionalen Trennstrich zu verwenden |
| Undefined | `3` | WordWrapMode nicht gesetzt. Die Umbruchstrategien der übergeordneten Struktur (Tabellenzelle, Absatz usw.) werden verwendet. |
