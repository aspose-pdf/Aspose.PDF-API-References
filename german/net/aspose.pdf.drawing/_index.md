Der **Aspose.Pdf.Drawing**-Namespace bietet Klassen, die grafische Primitive wie Bogen, Linie, Kreis usw. beschreiben. Entsprechende Objekte können zur Erstellung neuer Dokumente verwendet werden.

| [Ellipse](./ellipse/) | Stellt eine Ellipse dar. |
| [GradientAxialShading](./gradientaxialshading/) | Stellt die Klasse für gradient axiale Schattierung dar. |
| [GradientRadialShading](./gradientradialshading/) | Stellt den Typ für gradient radiale Schattierung dar. |
| [Graph](./graph/) | Stellt einen Graphen - Grafikgenerator-Absatz dar. |
| [Line](./line/) | Stellt eine Linie dar. |
| [Path](./path/) | Stellt einen Bogen dar. |
| [PatternColorSpace](./patterncolorspace/) | Stellt die Basismusterklasse dar. |
| [Rectangle](./rectangle/) | Stellt ein Rechteck dar. |
| [Shape](./shape/) | Stellt eine Form dar - das Basisgrafikobjekt. |
