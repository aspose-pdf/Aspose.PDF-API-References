## LineBreakStyle-Enumeration

Stellt die möglichen Zeilenumbruchstile für eine Datei dar.

```csharp
public enum LineBreakStyle
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Windows | `0` | Wählt den Windows-Zeilenumbruchstil (CRLF) aus. |
| Unix | `1` | Wählt den Unix-Zeilenumbruchstil (LF) aus. Wird auch für OSX verwendet. |
| Auto | `2` | Wählt automatisch den Typ des Zeilenumbruchs basierend auf dem Betriebssystem aus. |
