## XfaParserOptions.Signed-Eigenschaft

Wenn diese Eigenschaft wahr ist, wird das Dokument unter Verwendung des XFA-Formularstreams konvertiert (sofern vorhanden). Wenn sie falsch ist, wird der XFA-Formularstream ignoriert. Diese Eigenschaft wurde eingeführt, weil unklar ist, wie die Prüfziffer berechnet wird, die zur Überprüfung der Signatur verwendet wird.

```csharp
public bool Signed { get; set; }
```
