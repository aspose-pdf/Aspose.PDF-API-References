## XfaParserOptions.EmulateRequierdGroups-Eigenschaft

Wenn diese Eigenschaft wahr ist, werden zusätzliche rote Rechtecke für erforderliche Xfa "ausgeschlossene Gruppen" gezeichnet. Diese Eigenschaft wurde eingeführt, weil es während der Konvertierung der Xfa-Darstellung von Formularen in den Standard an Analogien der ausgeschlossenen Gruppen mangelte. Standardmäßig ist sie falsch.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
