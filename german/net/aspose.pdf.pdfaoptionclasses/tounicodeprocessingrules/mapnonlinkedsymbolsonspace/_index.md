## ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace property

Einige Schriftarten bieten keine Informationen über Unicodes für einige Textsymbole. Dieser Mangel an Informationen ruft einen Fehler „Text kann nicht Unicode zugeordnet werden“ hervor. Verwenden Sie dieses Flag, um nicht verknüpfte Symbole auf Unicode „Leerzeichen“ (Code 32) abzubilden.
