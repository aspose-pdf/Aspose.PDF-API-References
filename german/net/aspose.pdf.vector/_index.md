Der **Aspose.Pdf.Vector** ist ein Stamm-Namespace für Grafikoperationen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [GraphicElement](./graphicelement/) | Stellt die Basisklasse für Grafikobjekte auf der Seite dar. |
| [GraphicElementCollection](./graphicelementcollection/) | Stellt eine [`GraphicElement`](../aspose.pdf.vector/graphicelement/) Sammlung dar. |
| [GraphicsAbsorber](./graphicsabsorber/) | Stellt ein Absorberobjekt für Grafikelemente dar. Führt eine Grafiksuche durch und bietet Zugriff auf die Suchergebnisse über die [`Elements`](../aspose.pdf.vector/graphicsabsorber/elements/) Sammlung. |
| [GraphicState](./graphicstate/) | Stellt den Grafikzustand des aktuellen [`GraphicElement`](../aspose.pdf.vector/graphicelement/) dar. |
| [SubPath](./subpath/) | Stellt ein Vektorgrafikobjekt auf der Seite dar. Grundsätzlich werden Vektorgrafikobjekte durch zwei Gruppen von SubPaths dargestellt. Eine von ihnen wird durch eine Menge von Linien und Kurven dargestellt. Die anderen werden als Rechtecke dargestellt und können manchmal verwechselt werden. Üblicherweise handelt es sich um einen rechteckigen Bereich, der eine Farbe hat, aber sehr oft wird dieses Rechteck am Anfang der Seite platziert und definiert den gesamten Raum der Seite in Weiß. So erhalten Sie den SubPath, aber visuell sehen Sie nur den Text auf der Seite. |
| [SubPathGroup](./subpathgroup/) | Stellt eine Klasse für eine Gruppe von Containern für Grafikelemente dar. Klassenobjekte haben ein Begrenzungsrahmen, um die Gruppengröße zu berücksichtigen. |
| [SvgExtractionOptions](./svgextractionoptions/) | Stellt eine Optionsklasse zum Extrahieren von Vektorgrafiken aus der PDF-Dokumentseite dar. |
| [SvgExtractor](./svgextractor/) | Stellt eine Klasse zur Extraktion von SVG-Bildern von der Seite dar. |
| [XFormPlacement](./xformplacement/) | Stellt die Platzierung von XForm dar. Wenn das XForm mehr als einmal auf der Seite angezeigt wird, haben alle XformPlacements, die mit diesem XForm verbunden sind, gemeinsame grafische Elemente, aber unterschiedliche grafische Zustände. |
