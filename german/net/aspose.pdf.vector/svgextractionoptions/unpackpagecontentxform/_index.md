## SvgExtractionOptions.UnpackPageContentXForm-Eigenschaft

Ruft ein Flag ab und legt es fest, das bestimmt, ob XFrom, die auf Seiten gefunden werden, entpackt werden sollen oder nicht. XFrom-Elemente können in verschiedenen SVG-Dateien enden. Nur XForms, die durch Do-Anweisungen aus dem Seiteninhalt gerendert werden, werden entpackt. Verschachtelte XForms werden nicht entpackt.

```csharp
public bool UnpackPageContentXForm { get; set; }
```
