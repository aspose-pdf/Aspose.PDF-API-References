## SvgExtractionOptions.GroupStrength-Eigenschaft

Ruht eine Option ab und setzt sie. Die Stärke der Gruppierung von Unterpfaden in Bilder. Ermöglicht es Ihnen, den Grad der Gruppierung von Unterpfaden zu konfigurieren. Der Wertebereich reicht von 0 bis 1. Ein Wert von 0 entspricht der aktivierten [`ExtractEverySubPathToSvg`](../extracteverysubpathtosvg/) Option. Ein Wert von 1 erstellt ein einzelnes Bild für alle Vektorpfade auf der Seite. Die Option hat eine Wirkung, wenn [`AutoGrouping`](../autogrouping/) falsch ist. Der Standardwert ist `0.8`.

public double GroupStrength { get; set; }
