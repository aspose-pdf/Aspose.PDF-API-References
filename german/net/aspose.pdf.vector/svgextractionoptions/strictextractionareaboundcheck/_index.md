## SvgExtractionOptions.StrictExtractionAreaBoundCheck-Eigenschaft

Legt eine Option fest, um streng zu überprüfen, ob Unterpfade innerhalb des angegebenen Rechtecks in [`ExtractionAreaBound`](../extractionareabound/) liegen. Wenn auf false gesetzt, werden Unterpfade, die nicht vollständig in [`ExtractionAreaBound`](../extractionareabound/) enthalten sind, extrahiert. Der Standardwert ist `True`.

```csharp
public bool StrictExtractionAreaBoundCheck { get; set; }
```
