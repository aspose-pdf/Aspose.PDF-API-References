## Field.IsSharedField-Eigenschaft

Eigenschaft für Generatorunterstützung. Wird verwendet, wenn das Feld in den Kopf- oder Fußbereich eingefügt wird. Wenn wahr, wird dieses Feld einmal erstellt und sein Erscheinungsbild ist auf allen Seiten des Dokuments sichtbar. Wenn falsch, wird für jede Dokumentseite ein separates Feld erstellt.

```csharp
public bool IsSharedField { get; set; }
```
