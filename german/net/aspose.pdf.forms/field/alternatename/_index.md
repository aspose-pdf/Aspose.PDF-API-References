## Field.AlternateName-Eigenschaft

Ruft den alternativen Namen des Feldes ab oder legt ihn fest (Ein alternativer Feldname, der anstelle des tatsächlichen Feldnamens verwendet werden soll, wo immer das Feld in der Benutzeroberfläche identifiziert wird). Der alternative Name wird als Feldtooltip in Adobe Acrobat verwendet.

```csharp
public string AlternateName { get; set; }
```
