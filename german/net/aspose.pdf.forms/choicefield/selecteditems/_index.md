## ChoiceField.SelectedItems-Eigenschaft

Ruft das Array der ausgewählten Elemente ab oder legt es fest. Für eine Mehrfachauswahl-Liste enthält das Array mehr als ein Element. Für eine Einzelauswahl-Liste enthält es ein einzelnes Element.
