## Form.FlattenSettings.UpdateAppearances-Eigenschaft

Wenn gesetzt, werden alle Felddarstellungen vor dem Flatten neu generiert. Diese Option kann helfen, wenn das Feld falsch geflattet ist. Diese Option kann die Leistung verringern. Standardmäßig auf false gesetzt.

public bool UpdateAppearances { get; set; }
