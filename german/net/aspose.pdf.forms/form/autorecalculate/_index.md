## Form.AutoRecalculate-Eigenschaft

Wenn gesetzt, werden alle Formularfelder neu berechnet, wenn ein Feld geändert wird. Der Standardwert ist true. Setzen Sie ihn auf false, um die Leistung beim Ausfüllen von Formularen mit einer großen Anzahl von berechneten Feldern zu erhöhen.

```csharp
public bool AutoRecalculate { get; set; }
```
