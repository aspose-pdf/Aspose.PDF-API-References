## Form.RemoveFieldAppearance-Methode

Entfernt das Erscheinungsbild des Feldes am angegebenen Index. Wenn nur ein untergeordnetes Erscheinungsbild übrig bleibt, bettet die Methode es in das Feld ein.

```csharp
public void RemoveFieldAppearance(Field field, int appearanceIndex)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| field | Field | Feld mit Erscheinungsbildern. |
| appearanceIndex | Int32 | Index der Erscheinungsbilder. |
