## Form.RemovePermission-Eigenschaft

Wenn diese Eigenschaft wahr ist, wird das "Perms"-Wörterbuch nach der Konvertierung dynamischer Dokumente in Standard aus dem PDF-Dokument entfernt. Das "Perms"-Wörterbuch kann Regeln enthalten, die die Anzeige der Auswahl von Pflichtfeldern im Adobe Acrobat Reader stören. Es ist standardmäßig falsch.

```csharp
public bool RemovePermission { get; set; }
```
