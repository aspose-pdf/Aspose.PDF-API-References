## Form.AddFieldAppearance-Methode

Fügt das zusätzliche Erscheinungsbild des Feldes auf der angegebenen Seite des Dokuments an der angegebenen Stelle hinzu.

```csharp
public void AddFieldAppearance(Field field, int pageNumber, Rectangle rect)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| field | Field | Feld, dessen Erscheinungsbild im Formular hinzugefügt werden soll. |
| pageNumber | Int32 | Nummer der Seite, auf der das Feld platziert werden muss. |
| rect | Rectangle | Rechteck, in dem das Feld platziert wird. |
