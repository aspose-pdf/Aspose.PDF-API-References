## Add(Field, string, int) {#add}

Fügt ein neues Feld zum Formular hinzu; Wenn dieses Feld bereits auf einem anderen oder diesem Formular platziert ist, wird eine Kopie des Feldes erstellt.

```csharp
public Field Add(Field field, string partialName, int pageNumber)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| field | Field | Feldname. |
| partialName | String | Name des Feldes im Formular. |
| pageNumber | Int32 | Seitennummer, auf der das Feld hinzugefügt wird. |

### Rückgabewert

Das hinzugefügte Feld wird zurückgegeben. Wenn eine Kopie des Feldes erstellt wurde, wird diese zurückgegeben.
