## Add(Field, string, int) {#add}

Fügt dem Formular ein neues Feld hinzu; Wenn dieses Feld bereits auf einem anderen oder diesem Formular platziert ist, wird die Kopie des Felds erstellt.

```csharp
public Field Add(Field field, string partialName, int pageNumber)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| field | Field | Feldname. |
| partialName | String | Name des Felds im Formular. |
| pageNumber | Int32 | Seitenzahl, wo das Feld hinzugefügt wird. |

### Rückgabewert

Hinzugefügtes Feld zurückgegeben. Wenn eine Kopie des Felds erstellt wurde, wird sie zurückgegeben.
