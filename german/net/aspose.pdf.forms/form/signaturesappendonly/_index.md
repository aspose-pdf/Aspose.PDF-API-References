## Form.SignaturesAppendOnly-Eigenschaft

Wenn gesetzt, enthält das Dokument Signaturen, die ungültig werden können, wenn die Datei auf eine Weise gespeichert (geschrieben) wird, die ihren vorherigen Inhalt ändert, im Gegensatz zu einem inkrementellen Update.

```csharp
public bool SignaturesAppendOnly { get; set; }
```
