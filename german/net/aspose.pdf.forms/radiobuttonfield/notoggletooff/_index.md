## RadioButtonField.NoToggleToOff-Eigenschaft

Legt das Flag fest, das es dem Radiobutton ermöglicht, keinen ausgewählten Wert zu haben. Wenn `true`, muss jederzeit genau ein Radiobutton ausgewählt sein; das Auswählen des derzeit ausgewählten Buttons hat keine Auswirkungen. Wenn `false`, hebt das Klicken auf den ausgewählten Button die Auswahl auf, sodass kein Button ausgewählt ist.

```csharp
public bool NoToggleToOff { get; set; }
```

## Anmerkungen

Einige PDF-Reader (einschließlich Adobe Acrobat) ignorieren möglicherweise den `false`-Zustand des Flags.
