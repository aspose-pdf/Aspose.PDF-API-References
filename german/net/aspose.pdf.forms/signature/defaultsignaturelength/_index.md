## Signature.DefaultSignatureLength-Eigenschaft

Ruft die Standardlänge der Signaturdaten in Bytes ab oder legt sie fest.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Bemerkungen

Dies ist eine Schätzung der Länge der Signatur in Bytes. Wird für die Signatur über [`CustomSignHash`](../customsignhash/) verwendet, wenn der Parameter [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) gesetzt ist. Der Standardwert beträgt 3000.
