## BarcodeField.XSymHeight-Eigenschaft

Gibt den vertikalen Abstand zwischen zwei Barcode-Modulen in Pixeln an. Das Verhältnis XSymHeight/XSymWidth muss ein ganzzahliger Wert sein. Für PDF417 liegt der akzeptable Verhältnisbereich zwischen 1 und 4. Für QRCode und DataMatrix muss dieses Verhältnis immer 1 sein.

public int XSymHeight { get; }
