## BarcodeField.ECC-Eigenschaft

Gibt einen ganzzahligen Wert zurück, der den Fehlerkorrekturkoeffizienten darstellt. Für PDF417 sollte er von 0 bis 8 reichen. Für QRCode sollte er von 0 bis 3 reichen (0 für 'L', 1 für 'M', 2 für 'Q' und 3 für 'H').

public int ECC { get; }
