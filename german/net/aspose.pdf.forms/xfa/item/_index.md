## XFA-Indexer

Holt oder setzt den Wert des Datenknotens gemäß *Pfad*.

```csharp
public string this[string path] { get; set; }
```

| Parameter | Beschreibung |
| --- | --- |
| path | Datenknotenpfad, z.B. form1[0].Subform1[0].Subform2[0].Subform3[0].TextField[0]. Stellen Sie sicher, dass Sie Indizes einfügen, auch wenn die Daten nur einzelne Vorkommen jedes Knotens enthalten, d.h. schreiben Sie node1[0].node2[0]... anstelle von node1.node2... |

### Rückgabewert

Wert des Datenknotens.
