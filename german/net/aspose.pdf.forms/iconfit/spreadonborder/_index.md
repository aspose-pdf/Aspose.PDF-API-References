## IconFit.SpreadOnBorder-Eigenschaft

Wenn wahr, zeigt an, dass das Erscheinungsbild des Buttons so skaliert werden soll, dass es vollständig innerhalb der Grenzen der Annotation passt, ohne die Linienbreite des Rahmens zu berücksichtigen.

```csharp
public bool SpreadOnBorder { get; set; }
```
