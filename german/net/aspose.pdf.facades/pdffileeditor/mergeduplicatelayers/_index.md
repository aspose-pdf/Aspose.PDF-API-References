## PdfFileEditor.MergeDuplicateLayers-Eigenschaft

Optionale Inhalte von zusammengefügten Dokumenten mit gleichen Namen werden in einer Schicht im resultierenden Dokument zusammengeführt, wenn diese Eigenschaft wahr ist. Andernfalls werden Schichten mit gleichen Namen als verschiedene Schichten im resultierenden Dokument gespeichert.

```csharp
public bool MergeDuplicateLayers { get; set; }
```
