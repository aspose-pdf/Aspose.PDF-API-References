## PdfFileEditor.KeepFieldsUnique-Eigenschaft

Wenn wahr, werden die Feldnamen eindeutig gemacht, wenn Formulare zusammengefügt werden. Suffixe werden zu den Feldnamen hinzugefügt, das Suffix-Template kann in der Eigenschaft UniqueSuffix angegeben werden.

```csharp
public bool KeepFieldsUnique { get; set; }
```
