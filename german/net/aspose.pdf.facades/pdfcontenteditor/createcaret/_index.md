## PdfContentEditor.CreateCaret method

Erstellt Caret-Annotation.

```csharp
public void CreateCaret(int page, Rectangle annotRect, Rectangle caretRect, string symbol, 
    string annotContents, Color color)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| page | Int32 | Die Nummer der Originalseite, auf der die Anmerkung erstellt wird. |
| annotRect | Rectangle | Das Anmerkungsrechteck, das die Position der Anmerkung auf der Seite definiert. |
| caretRect | Rectangle | Die tatsächlichen Grenzen des zugrunde liegenden Caretzeichens. |
| symbol | String | Dem Caretzeichen wird ein Symbol zugeordnet. Mögliche Werte: "P" (Absatz), "Keine". |
| annotContents | String | Der Inhalt der Anmerkung. |
| color | Color | Die Farbe der Anmerkung. |

### Beispiele

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateCaret(1,
    new System.Drawing.Rectangle(50, 50, 100, 100),
    new System.Drawing.Rectangle(60, 60, 70, 70),
    "None", "Welcome to Aspose", System.Drawing.Color.Red);
editor.Save("example_out.pdf");
```
