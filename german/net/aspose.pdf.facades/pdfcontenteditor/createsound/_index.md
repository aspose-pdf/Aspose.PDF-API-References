## PdfContentEditor.CreateSound-Methode

Erstellt Sound-Anmerkungen.

```csharp
public void CreateSound(Rectangle rect, string filePath, string name, int page, string rate)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | Rechteck | Das Anmerkungsrechteck, das den Standort der Anmerkung auf der Seite definiert. |
| filePath | Zeichenfolge | Der Dateipfad der Audiodatei. |
| name | Zeichenfolge | Der Name eines Symbols, das zur Anzeige der Anmerkung verwendet werden soll, einschließlich: Lautsprecher und Mikrofon. |
| page | Int32 | Die Seite, auf der die Sound-Anmerkung erstellt wird. |
| rate | Zeichenfolge | Die Abtastrate in Abtastungen pro Sekunde. |
