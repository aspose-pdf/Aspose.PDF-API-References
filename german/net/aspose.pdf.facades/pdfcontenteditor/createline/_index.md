## PdfContentEditor.CreateLine method

Erstellt Linienbeschriftung.

```csharp
public void CreateLine(Rectangle rect, string contents, float x1, float y1, float x2, float y2, 
    int page, int border, Color clr, string borderStyle, int[] dashArray, string[] LEArray)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | Rectangle | Das Anmerkungsrechteck, das die Position der Anmerkung auf der Seite definiert. |
| contents | String | Der Inhalt der Anmerkung. |
| x1 | Single | Die horizontale Startkoordinate der Linie. |
| y1 | Single | Die vertikale Startkoordinate der Linie. |
| x2 | Single | Die horizontale Endkoordinate der Linie. |
| y2 | Single | Die vertikale Endkoordinate der Linie. |
| page | Int32 | Die Nummer der Originalseite, auf der die Anmerkung erstellt wird. |
| border | Int32 | Die Rahmenbreite in Punkten. Wenn dieser Wert 0 ist, wird kein Rahmen gezeichnet. Der Standardwert ist 1. |
| clr | Color | Die Farbe der Linie. |
| borderStyle | String | Der Rahmenstil, der die Breite und das Strichmuster angibt, das beim Zeichnen der Linie verwendet werden soll. Dieser Wert kann sein: "S" (Vollständig), "D" (Gestrichelt), "B" (Abgeschrägt), "I" (Inset) , "U" (Unterstrichen). |
| dashArray | Int32[] | Ein Dash-Array, das ein Muster aus Strichen und Lücken definiert, das zum Zeichnen einer gestrichelten Umrandung verwendet wird. Wenn es verwendet wird, muss borderSyle entsprechend auf "D" gesetzt werden. |
| LEArray | String[] | Ein Array aus zwei Werten, die jeweils den Anfangs- und Endstil der Zeichenlinie angeben. Die Werte können sein: "Square", "Circle", "Diamond", "OpenArrow", "ClosedArrow", "None", "Butt" , "ROpenArrow", "RClosedArrow", "Slash". |

### Beispiele

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateLine(new System.Drawing.Rectangle(0, 0, 100, 100), "Welcome to Aspose", 0, 0, 100, 100,
    1, 1, System.Drawing.Color.Red, "D", new int[] {2, 3}, new string[] {"OpenArrow", "ClosedArrow"});
editor.Save("example_out.pdf");
```
