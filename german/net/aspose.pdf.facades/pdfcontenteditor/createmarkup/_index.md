## PdfContentEditor.CreateMarkup method

Erstellt Markup-Anmerkungen im PDF-Dokument.

```csharp
public void CreateMarkup(Rectangle rect, string contents, int type, int page, Color clr)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | Rectangle | Das Rechteck, das die Position der Anmerkung auf der Seite definiert. |
| contents | String | Der Inhalt der Anmerkung. |
| type | Int32 | Der Typ der Markup-Anmerkung. Kann 0 (Hervorheben), 1 (Unterstreichen), 2 (Durchstreichen), 3 (Schnörkel) sein. |
| page | Int32 | Die Nummer der Originalseite, auf der die Anmerkung erstellt wird. |
| clr | Color | Die Farbe des Markups. |

### Beispiele

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateMarkup(new System.Drawing.Rectangle(0, 0, 100, 100),
    "Welcome to Aspose", 0, 1, System.Drawing.Color.Red);
editor.Save("example_out.pdf");
```
