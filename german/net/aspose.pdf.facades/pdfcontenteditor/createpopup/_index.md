## PdfContentEditor.CreatePopup method

Erstellt eine Popup-Anmerkung im PDF-Dokument.

```csharp
public void CreatePopup(Rectangle rect, string contents, bool open, int page)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | Rectangle | Das Anmerkungsrechteck, das die Position der Anmerkung auf der Seite definiert. |
| contents | String | Der Inhalt der Anmerkung. |
| open | Boolean | Ein Flag, das angibt, ob die Popup-Anmerkung anfänglich geöffnet angezeigt werden soll. |
| page | Int32 | Die Nummer der Originalseite, auf der die Anmerkung erstellt wird. |

### Beispiele

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreatePopup(new System.Drawing.Rectangle(0, 0, 100, 100), "Welcome to Aspose", true, 1);
editor.Save("example_out.pdf");
```
