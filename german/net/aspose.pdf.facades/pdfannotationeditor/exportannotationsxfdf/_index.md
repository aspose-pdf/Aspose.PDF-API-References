Exportiert den Inhalt der angegebenen Anmerkungstypen in XFDF

```csharp
public void ExportAnnotationsXfdf(Stream xmlOutputStream, int start, int end, string[] annotTypes)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| xmlOutputStream | Stream | Der Ausgabe-XFDF-Stream. |
| start | Int32 | Startseite, von der die Anmerkungen des Dokuments exportiert werden. |
| end | Int32 | Endseite, auf die die Anmerkungen des Dokuments exportiert werden. |
| annotTypes | String[] | Das Array der Anmerkungstypen muss exportiert werden. |

### Beispiele

```csharp

editor.BindPdf("example.pdf");
string[] annotTypes = new string[] {"Text", "Highlight"};
using (Stream stream = File.Create("example.xfdf"))
{
    editor.ExportAnnotationsXfdf(stream, 1, 2, annotTypes);
}
```
