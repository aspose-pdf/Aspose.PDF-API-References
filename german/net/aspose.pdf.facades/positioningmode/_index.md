## PositioningMode Enumeration

Definiert den Positionierungsmodus. Mögliche Werte umfassen Legacy (Rückwärtskompatibilität) und Current (aktualisierte Methode zur Berechnung der Textposition)

```csharp
public enum PositioningMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Legacy | `0` | Legacy-Textpositionierung |
| ModernLineSpacing | `1` | Aktualisierte Zeilenabstände, die vertikale Positionsberechnung erfolgt nach den alten Regeln (d.h. der Text wird relativ zur unteren linken Ecke des angegebenen Rechtecks positioniert) |
| Current | `2` | Aktualisierte Zeilenabstände und die vertikale Positionsberechnung erfolgt basierend auf der oberen linken Ecke anstelle der unteren linken. |
