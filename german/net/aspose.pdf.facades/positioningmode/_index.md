## PositioningMode enumeration

Definiert den Positionierungsmodus. Mögliche Werte sind Legacy (Abwärtskompatibilität) und Current (aktualisierte Methode zur Berechnung der Textposition)

```csharp
public enum PositioningMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Legacy | `0` | Alte Textpositionierung |
| ModernLineSpacing | `1` | Aktualisierter Zeilenabstand, vertikale Positionsberechnung erfolgt nach den alten Regeln (dh Text wird relativ zur unteren linken Ecke des angegebenen Rechtecks positioniert) |
