## PdfFileSecurity.TryEncryptFile method

Verschlüsselt die PDF-Datei mit Benutzerkennwort und Eigentümerkennwort und legt die Zugriffsrechte für das Dokument fest. Das Benutzerkennwort und das Eigentümerkennwort können null oder leer sein. Das Eigentümerpasswort wird durch eine zufällige Zeichenfolge ersetzt, wenn das eingegebene Eigentümerpasswort null oder leer ist. Löst keine Ausnahme aus, wenn der Vorgang fehlschlägt.

```csharp
public bool TryEncryptFile(string userPassword, string ownerPassword, DocumentPrivilege privilege, 
    KeySize keySize)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| userPassword | String | Benutzer-Passwort. |
| ownerPassword | String | Besitzer-Passwort. |
| privilege | DocumentPrivilege | Privileg setzen. |
| keySize | KeySize | KeySize.x40 für 40-Bit-Verschlüsselung, KeySize.x128 für 128-Bit-Verschlüsselung und KeySize.x256 für 256-Bit-Verschlüsselung. |

### Rückgabewert

Wahr für Erfolg oder falsch.

### Beispiele

```csharp
[C#]
string inFile = "D:\\input.pdf"; //Der TestPath kann neu zugewiesen werden.
string outFile = "D:\\output.pdf"; //Der TestPath kann neu zugewiesen werden.

bool result = fileSecurity.TryEncryptFile("userpass", "ownerpass", DocumentPrivilege.Print, KeySize.x256);
```
