## PdfFileInfo.HasEditPassword-Eigenschaft

Gibt true zurück, wenn ein Passwort benötigt wird, um Berechtigungen oder die Dokumentensicherheitseigenschaft zu ändern. Beachten Sie, dass diese Eigenschaft nur gelesen werden kann, wenn ein gültiges Passwort im [`PdfFileInfo`](../) Konstruktor bereitgestellt wurde. Falls der PasswordType Inaccessible ist (was bedeutet, dass ein ungültiges Passwort bereitgestellt wurde), schlägt das Lesen dieser Eigenschaft mit [`InvalidPasswordException`](../../../aspose.pdf/invalidpasswordexception/) fehl.

```csharp
public bool HasEditPassword { get; }
```
