## PdfFileInfo.PasswordType-Eigenschaft

Gibt den Typ des Passworts zurück, das zur Erstellung der PdfFileInfo-Instanz übergeben wurde. Siehe mögliche Werte in `PasswordType`. Beachten Sie, dass das PDF-Dokument sowohl mit dem Benutzer- (oder Öffnungs-) Passwort als auch mit dem Eigentümer- (oder Berechtigungen, Bearbeiten) Passwort geöffnet werden kann.

```csharp
public PasswordType PasswordType { get; }
```
