## PdfViewer.EndPage-Ereignis

Tritt auf, wenn das Drucken einer Seite im PdfViewer endet.

```csharp
public event EventHandler<StartEndPageEventArgs> EndPage;
```

## Bemerkungen

Dieses Ereignis wird ausgelöst, nachdem eine Seite im PdfViewer verarbeitet wurde. Es kann verwendet werden, um Aktionen oder Aktualisierungen nach dem Drucken einer Seite durchzuführen.
