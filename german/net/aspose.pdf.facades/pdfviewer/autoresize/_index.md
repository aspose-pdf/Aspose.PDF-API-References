## PdfViewer.AutoResize-Eigenschaft

Ruft einen booleschen Wert ab oder legt ihn fest, der angibt, ob die Datei mit optimierter Größe gedruckt werden soll. Wenn false, drucken Sie die Seite ohne Seitenanpassung. Wenn true, drucken Sie die Seite mit Anpassung an den druckbaren Bereich.

public bool AutoResize { get; set; }
