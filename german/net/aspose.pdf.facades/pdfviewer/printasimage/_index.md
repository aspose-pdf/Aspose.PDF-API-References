## PdfViewer.PrintAsImage-Eigenschaft

Legt einen Modus fest oder ruft ihn ab, um PdfViewer als Bild zu drucken.

```csharp
public bool PrintAsImage { get; set; }
```

## Hinweise

Wenn wahr, wird immer als Bild gedruckt (generiert ein Bild, das gedruckt wird). Wenn falsch, wird direkt auf das Gerät gedruckt, sofern alle Funktionen unterstützt werden. Falls das Dokument nicht unterstützte Funktionen enthält, kann das System automatisch entscheiden, als Bild zu drucken. Der Standardwert ist falsch.
