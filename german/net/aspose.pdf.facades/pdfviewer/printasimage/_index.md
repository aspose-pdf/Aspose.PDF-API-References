## PdfViewer.PrintAsImage property

Setzt oder erhält einen Modus für PdfViewer zum Drucken als Bild.

```csharp
public bool PrintAsImage { get; set; }
```

### Bemerkungen

Wenn wahr, wird immer als Bild gedruckt (erzeugt ein gedrucktes Bild) Wenn falsch, wird es direkt auf das Gerät gedruckt, wenn alle Funktionen unterstützt werden. Falls das Dokument nicht unterstützte Funktionen enthält, kann das System automatisch entscheiden, als Bild zu drucken. Der Standardwert ist falsch.
