## PdfViewer.StartPage-Ereignis

Tritt auf, bevor eine Seite gedruckt wird.

```csharp
public event EventHandler<StartEndPageEventArgs> StartPage;
```

## Bemerkungen

Dieses Ereignis wird für jede Seite ausgelöst, die gedruckt werden soll. Es kann verwendet werden, um benutzerdefinierte Aktionen auszuführen, bevor der Druck der Seite beginnt.
