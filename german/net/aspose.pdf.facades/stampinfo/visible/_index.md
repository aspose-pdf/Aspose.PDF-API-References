## StampInfo.Visible-Eigenschaft

Gibt die Sichtbarkeit des Stempels zurück. Wenn false, ist der Stempel verborgen (mit HideStampById). Der verborgene Stempel kann mit ShowStampById wiederhergestellt werden.

public bool Visible { get; }
