## FormDataConverter.Table property

Erhält oder setzt den mittleren Datencontainer, eine DataTable. Er muss definiert werden, bevor Daten von einem Format in ein anderes Format konvertiert werden. Die Spalten und der Tabellenname der DataTable sollten definiert werden. Der Tabellenname ist der Name der Tabelle in der database. ColumnName jeder Spalte ist der qualifizierte Feldname des PDF. Caption jeder Spalte ist der Spaltenname der Tabelle in der Datenbank. Wenn der Feldname derselbe wie der Name der Tabellenspalte ist, muss die Beschriftung nicht angegeben werden.
