## PdfExtractor.ExtractImageMode-Eigenschaft

Legt den Modus für den Extraktionsprozess von Bildern fest.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## Hinweise

Der Standardwert ist ExtractImageMode.DefinedInResources, der alle in den Ressourcen definierten Bilder extrahiert. Um tatsächlich angezeigte Bilder zu extrahieren, sollte der Modus ExtractImageMode.ActuallyUsed verwendet werden.
