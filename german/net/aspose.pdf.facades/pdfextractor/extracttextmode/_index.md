## PdfExtractor.ExtractTextMode property

Legt den Modus für das Ergebnis des Extrahierens von Text fest.

```csharp
public int ExtractTextMode { get; set; }
```

### Eigentumswert

0 ist reiner Textmodus und 1 ist Rohordnungsmodus. Standard ist 0.

### Beispiele

Das Beispiel demonstriert die`ExtractTextMode` Eigenschaftsverwendung im Textextraktionsszenario.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractTextMode = 1;
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```
