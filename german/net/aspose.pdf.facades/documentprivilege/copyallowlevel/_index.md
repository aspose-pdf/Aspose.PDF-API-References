## DocumentPrivilege.CopyAllowLevel-Eigenschaft

Ruft das Kopierlevel der Dokumentenberechtigung ab und setzt es. Genau wie die Berechtigungseinstellungen von Adobe Professionals. 0: Keine. 1: Aktivieren Sie den Textzugriff für Bildschirmlesegeräte für sehbehinderte Personen. 2: Aktivieren Sie das Kopieren von Text, Bildern und anderen Inhalten.

```csharp
public int CopyAllowLevel { get; set; }
```

## Anmerkungen

Wenn die Eigenschaft den Wert -1 hat, ist das Level undefiniert.
