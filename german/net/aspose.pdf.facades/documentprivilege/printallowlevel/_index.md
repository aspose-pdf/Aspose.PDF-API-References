## DocumentPrivilege.PrintAllowLevel-Eigenschaft

Ruft das Druckniveau der Dokumentenberechtigung ab und setzt es. Genau wie die Druckerlaubteinstellungen von Adobe Professional. 0: Keine. 1: Niedrige Auflösung (150 dpi). 2: Hohe Auflösung.

```csharp
public int PrintAllowLevel { get; set; }
```

## Bemerkungen

Wenn die Eigenschaft den Wert -1 hat, ist das Niveau undefiniert.
