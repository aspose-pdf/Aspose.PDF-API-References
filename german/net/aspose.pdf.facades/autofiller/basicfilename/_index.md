## AutoFiller.BasicFileName-Eigenschaft

Ruft den Basisdateinamen ab oder legt ihn fest, wenn viele kleine Dateien generiert werden. Die generierte Datei wird wie "BasicFileName0","BasicFileName1",... sein. Es funktioniert mit einer anderen Eigenschaft [`GeneratingPath`](../generatingpath/)GeneratingPath.

public string BasicFileName { get; set; }
