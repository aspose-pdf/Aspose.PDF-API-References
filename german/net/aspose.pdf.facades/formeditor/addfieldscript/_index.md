## FormEditor.AddFieldScript-Methode

Fügen Sie JavaScript für ein PushButton-Feld hinzu. Wenn ein altes Ereignis vorhanden ist, wird das neue Ereignis danach hinzugefügt.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fieldName | String | Der vollqualifizierte Feldname. |
| script | String | Das Java-Skript, das in ein Push-Button-Feld eingefügt werden soll. |

### Rückgabewert

True, wenn das Skript erfolgreich hinzugefügt wurde.
