## FormEditor.SetFieldCombNumber method

Legt die Anzahl der Kämme für ein reguläres einzeiliges Textfeld fest (das Feld wird automatisch in so viele gleich beabstandete Positionen oder Kämme unterteilt, wie der Wert des Parameters combNumber).

```csharp
public bool SetFieldCombNumber(string fieldName, int combNumber)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fieldName | String | Der qualifizierte Feldname. |
| combNumber | Int32 | Die Anzahl der Kämme, in die das Feld unterteilt werden soll. |

### Rückgabewert

Bei Erfolg true zurückgeben, sonst false.

### Beispiele

```csharp
FormEditor formEditor = new FormEditor("PdfWithAcroForm.pdf", "FormEditor_SetFieldComb.pdf"));
formEditor.SetFieldCombNumber("textCombField", 5);
```
