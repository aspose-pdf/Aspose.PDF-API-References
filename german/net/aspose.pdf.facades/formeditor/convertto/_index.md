## FormEditor.ConvertTo-Eigenschaft

Legt das PDF-Dateiformat fest. Die Ergebnisdatei wird im angegebenen Dateiformat gespeichert. Wenn diese Eigenschaft nicht angegeben ist, wird die Datei im standardmäßigen PDF-Format ohne Konvertierung gespeichert.

```csharp
public PdfFormat ConvertTo { set; }
```
