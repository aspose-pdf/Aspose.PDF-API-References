## FormEditor.SetFieldScript-Methode

Setzen Sie JavaScript für ein PushButton-Feld. Wenn altes JavaScript vorhanden war, wird es durch das neue ersetzt.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fieldName | String | Der vollqualifizierte Feldname. |
| script | String | Das Java-Skript, das in ein Push-Button-Feld hinzugefügt/platziert werden soll. |

### Rückgabewert

true, wenn das Feldskript erfolgreich gesetzt wurde.
