## Bookmark.Action-Eigenschaft

Ruft die mit dem Lesezeichen verbundene Aktion ab oder legt sie fest. Wenn die Seitenzahl angegeben ist, kann die Aktion nicht spezifiziert werden. Der Aktionstyp umfasst: "GoTo", "GoToR", "Launch", "Named".

```csharp
public string Action { get; set; }
```
