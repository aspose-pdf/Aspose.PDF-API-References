## AnnotationElement.AlternateDescriptions-Eigenschaft

Ruft die alternativen Beschreibungen für die Annotation ab oder legt sie fest. Text, der für die Annotation angezeigt werden soll, oder, wenn dieser Typ von Annotation keinen Text anzeigt, eine alternative Beschreibung des Inhalts der Annotation in menschenlesbarer Form.

public string AlternateDescriptions { get; set; }
