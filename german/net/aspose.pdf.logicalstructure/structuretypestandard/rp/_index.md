## StructureTypeStandard.RP-Feld

(Ruby-Zeichensetzung) Zeichensetzung, die den Ruby-Anmerkungstext umgibt. Sie wird nur verwendet, wenn eine Ruby-Anmerkung nicht ordnungsgemäß im Ruby-Stil formatiert werden kann und stattdessen als normaler Kommentar formatiert wird oder wenn sie als Warichu formatiert wird. Sie enthält Text (normalerweise eine einzelne LINKEN oder RECHTEN KLAMMER oder ein ähnliches Klammerzeichen).

```csharp
public static readonly StructureTypeStandard RP;
```
