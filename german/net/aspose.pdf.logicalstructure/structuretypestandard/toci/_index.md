## StructureTypeStandard.TOCI-Feld

(Inhaltsverzeichnis-Element) Ein einzelnes Mitglied eines Inhaltsverzeichnisses. Die Kinder dieses Eintrags können einer der folgenden Strukturtypen sein:

**Lbl**

Ein Label

**Reference**

Ein Verweis auf den Titel und die Seitenzahl

**NonStruct**

Nicht-Strukturelemente zum Umhüllen eines Führungsartefakts

**P**

Beschreibender Text

**TOC**

Inhaltsverzeichniselemente für hierarchische Inhaltsverzeichnisse, wie für den TOC-Eintrag beschrieben

```csharp
public static readonly StructureTypeStandard TOCI;
```
