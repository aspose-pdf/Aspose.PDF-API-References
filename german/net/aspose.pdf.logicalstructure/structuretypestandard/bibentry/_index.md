## StructureTypeStandard.BibEntry-Feld

(Bibliographieeintrag) Eine Referenz, die die externe Quelle eines zitierten Inhalts identifiziert. Es kann ein Label (Strukturtyp Lbl) als Kind enthalten.

Obwohl ein Bibliographieeintrag wahrscheinlich Komponenten enthält, die den Autor, das Werk, den Verlag usw. des zitierten Inhalts identifizieren, sind auf dieser Detailstufe keine standardisierten Strukturtypen definiert.

```csharp
public static readonly StructureTypeStandard BibEntry;
```
