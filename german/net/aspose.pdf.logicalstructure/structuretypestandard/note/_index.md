## StructureTypeStandard.Note-Feld

(Hinweis) Ein Element erklärenden Textes, wie eine Fußnote oder eine Endnote, auf das im Textkörper des Dokuments verwiesen wird. Es kann ein Label (Strukturtyp Lbl) als Kind haben. Die Notiz kann als Kind des Strukturelements im Fließtext, der darauf verweist, enthalten sein oder an anderer Stelle (wie in einem Abschnitt für Endnoten) enthalten sein und über eine Referenz (Strukturtyp Reference) darauf zugegriffen werden.

Tagged PDF schreibt nicht vor, wo Fußnoten in der Reihenfolge des Seiteninhalts platziert werden. Sie können entweder inline oder am Ende der Seite stehen, nach Ermessen des konformen Schreibers.

```csharp
public static readonly StructureTypeStandard Note;
```
