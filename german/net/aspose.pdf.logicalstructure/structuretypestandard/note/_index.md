## StructureTypeStandard.Note field

(Anmerkung) Ein erläuternder Text, wie z. B. eine Fußnote oder eine Endnote, auf die im Hauptteil des Dokuments verwiesen wird. Es kann als Kind ein Label (Strukturtyp Lbl) haben. Die Notiz kann als untergeordnetes Element des Strukturelements in den Fließtext eingefügt werden, der darauf verweist, oder sie kann an anderer Stelle (z. B. in einem Endnotenabschnitt) eingefügt und über eine Referenz (Strukturtyp Referenz) aufgerufen werden.

Tagged PDF schreibt die Platzierung von Fußnoten in der Reihenfolge der Seiteninhalte nicht vor. Sie können nach Ermessen des konformen Autors entweder inline oder am Ende der Seite stehen.

```csharp
public static readonly StructureTypeStandard Note;
```
