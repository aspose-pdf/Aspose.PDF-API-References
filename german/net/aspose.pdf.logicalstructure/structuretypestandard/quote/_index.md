## StructureTypeStandard.Quote-Feld

(Zitat) Ein Inline-Textabschnitt, der einer anderen Person als dem Autor des umgebenden Textes zugeschrieben wird.

Der zitierte Text sollte inline innerhalb eines einzelnen Absatzes enthalten sein. Dies unterscheidet sich vom Blockelement BlockQuote, das aus einem oder mehreren vollständigen Absätzen (oder anderen Elementen, die so präsentiert werden, als wären sie vollständige Absätze) besteht.
