## StructureTypeStandard.TBody field

(Tabellenkörper-Zeilengruppe; PDF 1.5) Eine Gruppe von Zeilen, die den Hauptteil einer Tabelle bilden. Wenn die Tabelle auf mehrere Seiten aufgeteilt ist, kann der Körperbereich an einer Zeilengrenze auseinander gebrochen werden. Eine Tabelle kann mehrere TBody-Elemente haben, um das Zeichnen eines Rahmens oder Hintergrunds für eine Reihe von Zeilen zu ermöglichen.
