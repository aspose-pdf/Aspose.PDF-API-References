## StructureTypeStandard.TBody-Feld

(Tabelle Körperzeilengruppe; PDF 1.5) Eine Gruppe von Zeilen, die den Hauptkörperteil einer Tabelle bilden. Wenn die Tabelle über mehrere Seiten verteilt ist, kann der Körperbereich an einer Zeilenbegrenzung getrennt werden. Eine Tabelle kann mehrere TBody-Elemente haben, um das Zeichnen eines Rahmens oder Hintergrunds für eine Gruppe von Zeilen zu ermöglichen.

```csharp
public static readonly StructureTypeStandard TBody;
```
