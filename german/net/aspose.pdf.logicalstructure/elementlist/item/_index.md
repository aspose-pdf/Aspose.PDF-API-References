## ElementList.Item-Methode

Ruft ein Element am angegebenen Index ab.

```csharp
public abstract Element Item(int index)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| index | Int32 | Der Index in der Liste der Elemente. |

### Rückgabewert

Das Element mit dem angegebenen Index in der Sammlung. Wenn *index* größer oder gleich der Anzahl der Elemente in der Liste ist, wird null zurückgegeben.
