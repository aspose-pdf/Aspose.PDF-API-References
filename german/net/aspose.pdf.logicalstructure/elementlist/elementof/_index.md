## ElementList-Indexer

Gibt ein Element am angegebenen Index zurück.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Parameter | Beschreibung |
| --- | --- |
| index | Der Index in der Liste der Elemente. |

### Rückgabewert

Das Element mit dem angegebenen Index in der Sammlung. Wenn der Index größer oder gleich der Anzahl der Elemente in der Liste ist, wird null zurückgegeben.
