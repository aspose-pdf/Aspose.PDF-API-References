## TableTRElement.IsInNewPage-Eigenschaft

Gibt an, ob die feste Zeile auf einer neuen Seite ist - die Seite mit dieser Eigenschaft sollte auf die nächste Seite gedruckt werden. Standardmäßig falsch.

public bool IsInNewPage { get; set; }
