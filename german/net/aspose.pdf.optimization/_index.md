Die **Aspose.Pdf.Optimierung** ist ein Namensraum für Klassen zur Verwaltung des Dokumentoptimierungsprozesses.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [ImageCompressionOptions](./imagecompressionoptions/) | Klasse enthält eine Reihe von Optionen für die Bildkompression. |
| [OptimizationOptions](./optimizationoptions/) | Klasse, die den Algorithmus zur Dokumentoptimierung beschreibt. Eine Instanz dieser Klasse kann als Parameter der Methode OptimizeResources() verwendet werden. |
