## OptimizationOptions.LinkDuplcateStreams-Eigenschaft

Wenn dieses Flag auf true gesetzt ist, werden Ressourcenströme analysiert. Wenn doppelte Ströme gefunden werden (d.h. wenn der Inhalt der Ströme gleich ist), werden diese Ströme als ein Objekt gespeichert. Dies ermöglicht es, die Dokumentgröße in einigen Fällen zu verringern (z.B. wenn dasselbe Dokument mehrfach zusammengefügt wurde).

```csharp
public bool LinkDuplcateStreams { get; set; }
```
