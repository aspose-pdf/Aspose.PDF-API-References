## OptimizationOptions.RemoveUnusedStreams-Eigenschaft

Wenn dieses Flag auf true gesetzt ist, wird jede Ressource auf ihre Verwendung überprüft. Wenn eine Ressource niemals verwendet wird, wird sie entfernt. Dies kann die Dokumentgröße verringern, zum Beispiel wenn Seiten aus dem Dokument extrahiert wurden.

```csharp
public bool RemoveUnusedStreams { get; set; }
```
