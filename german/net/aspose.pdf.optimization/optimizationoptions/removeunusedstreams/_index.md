## OptimizationOptions.RemoveUnusedStreams property

Wenn dieses Flag auf true gesetzt ist, wird jede Ressource auf ihre Nutzung überprüft. Wenn die Ressource nie verwendet wird, werden die Ressourcen entfernt. Dies kann die Dokumentgröße verringern, beispielsweise wenn Seiten aus dem Dokument extrahiert wurden.

public bool RemoveUnusedStreams { get; set; }
