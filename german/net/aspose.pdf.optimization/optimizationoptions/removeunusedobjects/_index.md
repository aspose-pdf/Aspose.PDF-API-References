## OptimizationOptions.RemoveUnusedObjects-Eigenschaft

Wenn dieses Flag auf true gesetzt ist, werden alle Dokumentobjekte überprüft und ungenutzte Objekte (d.h. Objekte, die keine Referenz haben) werden aus dem Dokument entfernt.

```csharp
public bool RemoveUnusedObjects { get; set; }
```
