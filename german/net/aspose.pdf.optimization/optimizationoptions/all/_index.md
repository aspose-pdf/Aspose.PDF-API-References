## OptimizationOptions.All-Methode

Erstellt eine Optimierungsstrategie mit allen aktivierten Optionen. Bitte beachten Sie, dass nur Optionen aktiviert werden, die keine Funktionalität des Dokuments ändern. D.h. Bildkomprimierung und das Entbetten von Schriftarten werden nicht aktiviert (und können manuell eingebettet werden).

```csharp
public static OptimizationOptions All()
```
