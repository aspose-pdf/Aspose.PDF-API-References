## ComparisonOptions.ExtractionArea-Eigenschaft

Legen Sie das rechteckige Gebiet fest, in dem der Text der Seiten verglichen wird. Diese Option kann nicht zusammen mit den Optionen [`ExcludeTables`](../excludetables/), [`ExcludeAreas1`](../excludeareas1/) und [`ExcludeAreas2`](../excludeareas2/) festgelegt werden.

public Rectangle ExtractionArea { get; set; }
