## ComparisonOptions.ExcludeTables-Eigenschaft

Holen Sie sich die Option und setzen Sie sie, die bestimmt, ob Tabellen von dem Vergleich ausgeschlossen sind. Diese Option kann nicht zusammen mit der [`ExtractionArea`](../extractionarea/) Option gesetzt werden. Der Standardwert ist `false`.

```csharp
public bool ExcludeTables { get; set; }
```
