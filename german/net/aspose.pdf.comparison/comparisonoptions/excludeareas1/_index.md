## ComparisonOptions.ExcludeAreas1-Eigenschaft

Holen Sie sich die auszuschließenden Bereiche und setzen Sie diese. Wird für die erste Seite oder das Dokument in der Vergleichsmethode verwendet. Diese Option kann zusammen mit [`ExcludeTables`](../excludetables/) gesetzt werden. Diese Option kann nicht zusammen mit der [`ExtractionArea`](../extractionarea/)-Option gesetzt werden.

public Rectangle[] ExcludeAreas1 { get; set; }
