## GraphicalPdfComparer.GetDifference-Methode

Erhält Unterschiede zwischen Seitenbildern. Das Ergebnis enthält ein Bild der ersten verglichenen Seite und ein Array von Unterschieden.

```csharp
public ImagesDifference GetDifference(Page page1, Page page2)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| page1 | Seite | Die erste Seite. |
| page2 | Seite | Die zweite Seite. |

### Rückgabewert

Die [`ImagesDifference`](../../imagesdifference/) Instanz.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | Wenn die verglichenen Seiten unterschiedliche Größen haben. |
