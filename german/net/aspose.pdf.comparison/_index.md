| Klasse | Beschreibung |

| [DocumentComparisonStatistics](./documentcomparisonstatistics/) | Stellt eine Klasse für die Statistiken zum Dokumentenvergleich dar. |
| [GraphicalPdfComparer](./graphicalpdfcomparer/) | Stellt eine Klasse zum grafischen Vergleichen von PDF-Dokumenten dar. Sollte verwendet werden, um kleine Änderungen, hauptsächlich grafischer Natur, zu suchen. Um Änderungen im Textinhalt zu vergleichen, verwenden Sie andere PDF-Vergleichsklassen. |

| [MarkdownDiffOutputGenerator](./markdowndiffoutputgenerator/) | Stellt eine Klasse zum Generieren der Markdown-Darstellung von Textunterschieden dar. Aufgrund der Markdown-Syntax ist es nicht möglich, Änderungen an Leerzeichen anzuzeigen. Die Auswahl von Änderungen führt dazu, dass Leerzeichen um die Formatierung hinzugefügt werden, andernfalls wird der Markdown-Viewer den Text nicht korrekt anzeigen. Gelöschte Zeilenumbrüche werden durch - Absatzmarken angezeigt. |
