## SideBySideComparisonOptions.ExcludeTables-Eigenschaft

Legen Sie die Option fest, die bestimmt, ob Tabellen von der Vergleich ausgeschlossen sind. Diese Option kann nicht zusammen mit [`ComparisonArea1`](../comparisonarea1/) und [`ComparisonArea2`](../comparisonarea2/) festgelegt werden. Der Standardwert ist `false`.

```csharp
public bool ExcludeTables { get; set; }
```
