## SideBySideComparisonOptions.ExcludeAreas2-Eigenschaft

Holen Sie sich die auszuschließenden Bereiche und setzen Sie sie. Wird für die zweite Seite oder das Dokument in der Vergleichsmethode verwendet. Diese Option kann zusammen mit [`ExcludeTables`](../excludetables/) gesetzt werden. Diese Option kann nicht zusammen mit der [`ComparisonArea2`](../comparisonarea2/)-Option gesetzt werden.

```csharp
public Rectangle[] ExcludeAreas2 { get; set; }
```
