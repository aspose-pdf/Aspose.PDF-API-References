## SideBySideComparisonOptions.AdditionalChangeMarks-Eigenschaft

Holen Sie sich die Eigenschaft und setzen Sie sie, die bestimmt, ob zusätzliche Änderungsmarkierungen angezeigt werden. Wenn gesetzt, werden Änderungsmarkierungen angezeigt, die sich nicht auf der aktuellen Seite befinden, sondern auf einer anderen Seite vorhanden sind. Wenn sich die Änderung zwischen Wörtern befindet, kann die Markierung möglicherweise nicht genau relativ zum Leerzeichen positioniert werden. Der Standardwert ist `false`.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
