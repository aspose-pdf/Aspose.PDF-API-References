## SideBySideComparisonOptions.ExcludeAreas1-Eigenschaft

Ausschließen von Bereichen abrufen und festlegen. Wird für die erste Seite oder das Dokument in der Vergleichsmethode verwendet. Diese Option kann zusammen mit [`ExcludeTables`](../excludetables/) festgelegt werden. Diese Option kann nicht zusammen mit der [`ComparisonArea1`](../comparisonarea1/)-Option festgelegt werden.

```csharp
public Rectangle[] ExcludeAreas1 { get; set; }
```
