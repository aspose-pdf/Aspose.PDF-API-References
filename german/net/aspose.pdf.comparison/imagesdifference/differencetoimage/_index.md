## ImagesDifference.DifferenceToImage-Methode

Konvertiert das Differenzarray in ein Bitmap-Bild unter Verwendung der angegebenen Farben.

```csharp
public Bitmap DifferenceToImage(Color color, Color backgroundColor)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| color | Color | Die Farbe für nicht-null Unterschiede. |
| backgroundColor | Color | Die Hintergrundfarbe für null Unterschiede. |

### Rückgabewert

Ein Bitmap-Bild, das das Differenzarray darstellt.
