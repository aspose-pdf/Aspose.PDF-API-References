## CompromiseCheckResult-Klasse

Stellt eine Klasse zum Überprüfen digitaler Signaturen von Dokumenten auf Kompromittierung dar.

```csharp
public sealed class CompromiseCheckResult
```

## Eigenschaften

| Name | Beschreibung |
| --- | --- |
| [HasCompromisedSignatures](../../aspose.pdf.signatures/compromisecheckresult/hascompromisedsignatures/) { get; } | Gibt an, ob es im Dokument kompromittierte digitale Signaturen gibt. Gibt true zurück, wenn mindestens eine Signatur kompromittiert ist; andernfalls false. |
| [SignaturesCoverage](../../aspose.pdf.signatures/compromisecheckresult/signaturescoverage/) { get; } | Gibt den Abdeckungszustand digitaler Signaturen in einem Dokument zurück. Wenn er gleich Undefined ist, dann ist eine der Signaturen kompromittiert. |
