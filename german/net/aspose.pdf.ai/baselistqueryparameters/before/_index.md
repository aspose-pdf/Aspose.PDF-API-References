## BaseListQueryParameters.Before-Eigenschaft

Ruft einen Cursor ab oder legt diesen fest, der für die Paginierung verwendet wird. before ist eine Objekt-ID, die Ihren Platz in der Liste definiert. Wenn Sie beispielsweise eine Listenanfrage stellen und 100 Objekte erhalten, die mit obj_foo enden, kann Ihr nachfolgender Aufruf before=obj_foo enthalten, um die vorherige Seite der Liste abzurufen.

```csharp
public string Before { get; set; }
```
