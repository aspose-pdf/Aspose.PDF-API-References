## AssistantCreateRequest.ToolResources-Eigenschaft

Ruft Ressourcen ab oder legt sie fest, die von den Werkzeugen des Assistenten verwendet werden. Die Ressourcen sind spezifisch für den Typ des Werkzeugs. Zum Beispiel erfordert das code_interpreter-Werkzeug eine Liste von Datei-IDs, während das file_search-Werkzeug eine Liste von Vektor-Store-IDs.

```csharp
public ToolResources ToolResources { get; set; }
```
