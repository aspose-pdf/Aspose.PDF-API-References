## AssistantCreateRequest.Tools-Eigenschaft

Ruft eine Liste von aktivierten Tools im Assistenten ab oder legt diese fest. Es können maximal 128 Tools pro Assistenten vorhanden sein. Die Tools können von den Typen code_interpreter, file_search oder function sein.
