## AssistantResponse.Temperature-Eigenschaft

Legt fest, welche Abtasttemperatur zwischen 0 und 2 verwendet werden soll. Höhere Werte wie 0.8 machen die Ausgabe zufälliger, während niedrigere Werte wie 0.2 sie fokussierter und deterministisch machen.

```csharp
public double? Temperature { get; set; }
```
