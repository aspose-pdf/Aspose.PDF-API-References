## MessageContentRequest.CreateImageFileContent-Methode

Erstellt einen Bilddateiinhalt für eine Nachricht.

```csharp
public static MessageContentRequest CreateImageFileContent(string fileId, string detail = "auto")
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| fileId | String | Die ID der Bilddatei. |
| detail | String | Die Details der Bilddatei. Gibt die Detailstufe des Bildes an, wenn vom Benutzer angegeben. "low" verwendet weniger Tokens, Sie können sich für eine hohe Auflösung mit "high" entscheiden. Standard ist "auto". |
