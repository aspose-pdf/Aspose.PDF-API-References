## MessageContentRequest.CreateImageUrlContent-Methode

Erstellt einen Bild-URL-Inhalt für eine Nachricht.

```csharp
public static MessageContentRequest CreateImageUrlContent(string imageUrl, string detail = "auto")
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| imageUrl | String | Die URL des Bildes. |
| detail | String | Die Detailstufe der Bild-URL. Gibt die Detailstufe des Bildes an, wenn vom Benutzer angegeben. "low" verwendet weniger Tokens, Sie können sich für eine hohe Auflösung mit "high" entscheiden. Standard ist "auto". |

### Rückgabewert

Eine neue Instanz von MessageContentRequest mit ImageUrl-Inhalt.
