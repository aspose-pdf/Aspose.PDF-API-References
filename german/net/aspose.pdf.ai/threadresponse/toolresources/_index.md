## ThreadResponse.ToolResources-Eigenschaft

Ruft eine Menge von Ressourcen ab oder legt sie fest, die den Werkzeugen des Assistenten in diesem Thread zur Verfügung stehen. Die Ressourcen sind spezifisch für den Typ des Werkzeugs. Zum Beispiel benötigt das code_interpreter-Werkzeug eine Liste von Datei-IDs, während das file_search-Werkzeug eine Liste von Vektor-Store-IDs benötigt.

```csharp
public ToolResources ToolResources { get; set; }
```
