## CreateChatCompletionChunkResponse.Choices-Eigenschaft

Ruft eine Liste von Chat-Vervollständigungsoptionen ab oder legt sie fest. Kann mehr als ein Element enthalten, wenn n größer als 1 ist. Kann auch für den letzten Chunk leer sein, wenn Sie stream_options: {"include_usage": true} setzen.

public List<Choice> Choices { get; set; }
