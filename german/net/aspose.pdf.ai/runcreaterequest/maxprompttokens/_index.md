## RunCreateRequest.MaxPromptTokens-Eigenschaft

Ruft die maximale Anzahl von Prompt-Token ab oder legt sie fest, die im Verlauf des Laufs verwendet werden dürfen. Der Lauf wird sich bemühen, nur die angegebene Anzahl von Prompt-Token über mehrere Runden des Laufs zu verwenden. Wenn der Lauf die angegebene Anzahl von Prompt-Token überschreitet, endet der Lauf mit dem Status unvollständig. Siehe incomplete_details für weitere Informationen.

public int? MaxPromptTokens { get; set; }
