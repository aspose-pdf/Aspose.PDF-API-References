## RunCreateRequest.Temperature-Eigenschaft

Legt fest, welche Abtasttemperatur verwendet werden soll, zwischen 0 und 2. Höhere Werte wie 0.8 machen die Ausgabe zufälliger, während niedrigere Werte wie 0.2 sie fokussierter und deterministisch machen.

```csharp
public double? Temperature { get; set; }
```
