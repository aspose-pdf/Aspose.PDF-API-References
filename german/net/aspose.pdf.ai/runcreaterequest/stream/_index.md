## RunCreateRequest.Stream-Eigenschaft

Gibt an, ob Streaming verwendet werden soll. Wenn wahr, gibt es einen Stream von Ereignissen zurück, die während des Runs als servergesendete Ereignisse auftreten und enden, wenn der Run einen terminalen Zustand mit einer Daten: [DONE]-Nachricht erreicht.

public bool? Stream { get; set; }
