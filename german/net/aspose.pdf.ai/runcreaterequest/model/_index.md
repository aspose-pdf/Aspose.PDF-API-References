## RunCreateRequest.Model-Eigenschaft

Ruft die ID des Modells ab oder legt sie fest, das verwendet werden soll, um diesen Lauf auszuführen. Wenn hier ein Wert angegeben wird, überschreibt er das mit dem Assistenten verbundene Modell. Andernfalls wird das mit dem Assistenten verbundene Modell verwendet.

```csharp
public string Model { get; set; }
```
