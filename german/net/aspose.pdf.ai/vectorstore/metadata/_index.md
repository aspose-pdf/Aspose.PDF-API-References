## VectorStore.Metadata-Eigenschaft

Ruft ein Set von 16 Schlüssel-Wert-Paaren ab oder legt es fest, die an einen Vektorstore angehängt werden können. Dies kann nützlich sein, um zusätzliche Informationen über den Vektorstore in einem strukturierten Format zu speichern. Schlüssel können maximal 64 Zeichen lang sein und Werte maximal 512 Zeichen lang sein.

```csharp
public Dictionary<string, string> Metadata { get; set; }
```
