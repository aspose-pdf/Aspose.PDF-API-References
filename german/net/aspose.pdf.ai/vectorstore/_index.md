## VectorStore-Klasse

Ein Hilfsprogramm zum Erstellen eines Vektor-Speichers mit file_ids und zum Anhängen an diesen Thread. Es kann maximal 1 Vektor-Speicher an den Thread angehängt werden.

```csharp
public class VectorStore
```

| Name | Beschreibung |
| --- | --- |
| [VectorStore](vectorstore/)() | Der Standardkonstruktor. |

## Eigenschaften

| Name | Beschreibung |
| --- | --- |
| [FileIds](../../aspose.pdf.ai/vectorstore/fileids/) { get; set; } | Ruft eine Liste von Datei-IDs ab oder legt sie fest, die dem Vektor-Speicher hinzugefügt werden sollen. Es kann maximal 10000 Dateien in einem Vektor-Speicher geben. |
| [Metadata](../../aspose.pdf.ai/vectorstore/metadata/) { get; set; } | Ruft ein Set von 16 Schlüssel-Wert-Paaren ab oder legt es fest, die an einen Vektor-Speicher angehängt werden können. Dies kann nützlich sein, um zusätzliche Informationen über den Vektor-Speicher in einem strukturierten Format zu speichern. Schlüssel können maximal 64 Zeichen lang sein und Werte können maximal 512 Zeichen lang sein. |
