## CompletionCreateRequest.NumberOfChoices-Eigenschaft

Legt fest, wie viele Chat-Vervollständigungsoptionen für jede Eingabemeldung generiert werden sollen. Beachten Sie, dass Ihnen basierend auf der Anzahl der generierten Tokens über alle Optionen hinweg Kosten berechnet werden. Halten Sie n auf 1, um die Kosten zu minimieren.

```csharp
public int? NumberOfChoices { get; set; }
```
