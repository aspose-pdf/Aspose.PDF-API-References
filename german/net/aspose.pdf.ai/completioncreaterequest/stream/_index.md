## CompletionCreateRequest.Stream-Eigenschaft

Legt fest, ob Streaming verwendet werden soll. Wenn gesetzt, werden partielle Nachrichten-Deltas gesendet, wie in ChatGPT. Tokens werden als Daten-only server-sent events gesendet, sobald sie verfügbar sind, wobei der Stream durch eine Daten: [DONE]-Nachricht beendet wird.

```csharp
public bool? Stream { get; set; }
```
