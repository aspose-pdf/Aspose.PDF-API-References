## CompletionCreateRequest.ResponseFormat-Eigenschaft

Ruft ein Objekt ab oder legt es fest, das das Format angibt, das das Modell ausgeben muss. Kompatibel mit GPT-4 Turbo und allen GPT-3.5 Turbo-Modellen, die neuer sind als gpt-3.5-turbo-1106. Das Setzen auf { "type": "json_object" } aktiviert den JSON-Modus, der garantiert, dass die vom Modell generierte Nachricht gültiges JSON ist.

public ResponseFormat ResponseFormat { get; set; }
