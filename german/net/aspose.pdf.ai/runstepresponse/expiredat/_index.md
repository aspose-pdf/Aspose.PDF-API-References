## RunStepResponse.ExpiredAt-Eigenschaft

Ruft den Unix-Zeitstempel (in Sekunden) ab oder legt ihn fest, wann der Ausführungsschritt abgelaufen ist. Ein Schritt wird als abgelaufen betrachtet, wenn der übergeordnete Lauf abgelaufen ist.

```csharp
public long? ExpiredAt { get; set; }
```
