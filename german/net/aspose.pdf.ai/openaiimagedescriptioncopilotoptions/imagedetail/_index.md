## OpenAIImageDescriptionCopilotOptions.ImageDetail-Eigenschaft

Legt das Detailniveau des Bildes fest, wenn es vom Benutzer angegeben wird. "low" verwendet weniger Tokens, Sie können sich für eine hohe Auflösung mit "high" entscheiden. Wenn nicht festgelegt, wird standardmäßig "auto" verwendet.

```csharp
public string ImageDetail { get; set; }
```
