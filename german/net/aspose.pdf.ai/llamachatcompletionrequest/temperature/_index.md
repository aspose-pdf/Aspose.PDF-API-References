## LlamaChatCompletionRequest.Temperature-Eigenschaft

Legt die Sampling-Temperatur fest oder ruft sie ab, die zwischen 0 und 2 verwendet werden soll. Höhere Werte wie 0.8 machen die Ausgabe zufälliger, während niedrigere Werte wie 0.2 sie fokussierter und deterministisch machen. Der Standardwert ist 1.

```csharp
public double? Temperature { get; set; }
```
