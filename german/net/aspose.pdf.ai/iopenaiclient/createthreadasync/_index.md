## IOpenAIClient.CreateThreadAsync-Methode

Erstellt einen neuen Thread asynchron.

```csharp
public Task<ThreadResponse> CreateThreadAsync(ThreadCreateRequest threadCreateRequest, 
    CancellationToken? cancellationToken = default)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| threadCreateRequest | ThreadCreateRequest | Das Anforderungsobjekt, das Details zum Erstellen des Threads enthält. |
| cancellationToken | Nullable`1 | Ein Token zum Abbrechen der Operation. |

### Rückgabewert

Eine Aufgabe, die die asynchrone Operation darstellt. Das Ergebnis der Aufgabe enthält die Antwort von der Thread-Erstellung.
