## VectorStoreResponse.Status-Eigenschaft

Ruft den Status des Vektorspeichers ab oder legt ihn fest, der entweder abgelaufen, in Bearbeitung oder abgeschlossen sein kann. Ein Status von abgeschlossen zeigt an, dass der Vektorspeicher bereit zur Verwendung ist.

```csharp
public string Status { get; set; }
```
