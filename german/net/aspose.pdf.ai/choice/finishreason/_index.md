## Choice.FinishReason-Eigenschaft

Ruht den Grund ab oder setzt ihn, warum das Modell aufgehört hat, Tokens zu generieren. Dies wird gestoppt, wenn das Modell einen natürlichen Stoppunkt erreicht oder eine bereitgestellte Stoppsequenzlänge erreicht wurde, wenn die maximale Anzahl von Tokens, die in der Anfrage angegeben wurde, erreicht wurde.

public string FinishReason { get; set; }
