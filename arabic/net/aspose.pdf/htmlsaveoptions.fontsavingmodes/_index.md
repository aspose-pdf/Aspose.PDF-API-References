## HtmlSaveOptions.FontSavingModes enumeration

يعدد الأوضاع التي يمكن استخدامها لحفظ الخطوط المشار إليها في PDF المحفوظ.

```csharp
public enum FontSavingModes
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| AlwaysSaveAsWOFF | `0` | سيتم حفظ جميع الخطوط المشار إليها والإشارة إليها كخطوط WOFF. |
| AlwaysSaveAsTTF | `1` | سيتم حفظ جميع الخطوط المشار إليها والإشارة إليها كخطوط TTF. |
| AlwaysSaveAsEOT | `2` | سيتم حفظ جميع الخطوط المشار إليها والإشارة إليها كخطوط EOT. |
| SaveInAllFormats | `3` | سيتم حفظ جميع الخطوط المشار إليها (والإشارة إليها في CSS) كملفات مستقلة 3: EOT، TTH، WOFF. يزيد من حجم البيانات الناتجة ولكنه يجعل الناتج مناسبًا للأغلبية الساحقة من متصفحات الويب. |
| DontSave | `4` | لن يتم حفظ جميع الخطوط المشار إليها. |
