## خاصية Document.AllowReusePageContent

يسمح بدمج محتويات الصفحات لتحسين حجم الوثيقة. إذا تم استخدامه، فقد تشير الصفحات المختلفة ولكن المكررة إلى نفس كائن المحتوى. يرجى ملاحظة أن هذا الوضع قد يسبب آثار جانبية مثل تغيير محتوى الصفحة عند تغيير صفحة أخرى.

public bool AllowReusePageContent { get; set; }
