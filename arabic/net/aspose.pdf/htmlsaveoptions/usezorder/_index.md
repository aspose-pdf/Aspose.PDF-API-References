## خاصية HtmlSaveOptions.UseZOrder

إذا تم تعيين السمة UseZOrder إلى true، يتم إضافة الرسومات والنصوص إلى مستند HTML الناتج وفقًا لترتيب Z في مستند PDF الأصلي. إذا كانت هذه السمة false، يتم وضع جميع الرسومات كطبقة واحدة مما قد يتسبب في بعض التأثيرات غير الضرورية للأشياء المتداخلة.

```csharp
public bool UseZOrder { get; set; }
```
