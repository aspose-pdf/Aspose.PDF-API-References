## خاصية HtmlSaveOptions.SplitCssIntoPages

عند اختيار وضع الصفحات المتعددة (أي أن 'SplitIntoPages' هو 'true')، فإن هذه السمة تحدد ما إذا كان يجب إنشاء ملف CSS منفصل لكل صفحة HTML الناتجة. بشكل افتراضي، تكون هذه السمة خاطئة، لذا سيتم إنشاء ملف CSS كبير مشترك لجميع الصفحات التي تم إنشاؤها. الحجم الإجمالي لجميع ملفات CSS التي تم إنشاؤها في هذا الوضع (ملف CSS لكل صفحة) عادة ما يكون أكبر بكثير من حجم ملف CSS الكبير الواحد، لأن في الحالة السابقة تكون فئات CSS مكررة في عدة ملفات CSS لكل صفحة. لذا، فإن هذا الإعداد يُفضل استخدامه فقط عندما تكون مهتمًا بمعالجة كل صفحة HTML بشكل مستقل، وبالتالي فإن حجم CSS لكل صفحة على حدة هو القضية الأكثر أهمية.

```csharp
public bool SplitCssIntoPages { get; set; }
```
