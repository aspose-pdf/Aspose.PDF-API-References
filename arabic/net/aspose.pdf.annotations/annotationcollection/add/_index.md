## Add(Annotation, bool) {#add_1}

```csharp
public void Add(Annotation annotation, bool considerRotation)
```

| Parameter | Type | Description |
| --- | --- | --- |
| annotation | Annotation | التعليق الذي سيتم إضافته. |
| considerRotation | Boolean | إذا كان صحيحًا وإذا تم تدوير الصفحة، فسيتم إعادة حساب موضع التعليق وفقًا لتدوير الصفحة. |
