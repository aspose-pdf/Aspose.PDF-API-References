## خيارات البحث عن النصوص.بحث عن الرسوميات المتعلقة بالنص

تحصل أو تعين قيمة تسمح بالبحث عن الرسوميات المتعلقة بالنص (التسطير، الخلفية وما إلى ذلك) أثناء البحث عن النص. true - سيتم إجراء البحث عن الرسوميات المتعلقة بالنص (القيمة الافتراضية). false - سيتم تجاهل العناصر الرسومية التي قد تكون موجودة في الوثيقة المصدر. قم بتعيين هذا في حالة وجود مشاكل في الأداء أو عدم الحاجة للتعامل مع التسطير أو الخلفية أو القص.

public bool SearchForTextRelatedGraphics { get; set; }
