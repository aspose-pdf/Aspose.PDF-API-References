## خاصية TextSearchOptions.IgnoreResourceFontErrors

تحصل أو تعين الإشارة إلى أن الأخطاء المتعلقة بغياب الخط سيتم تجاهلها بواسطة ماص النص (الجزء). true - تعني أن أخطاء غياب الخط سيتم تجاهلها. سيتم تخطي مقاطع النص التي تشير إلى موارد غير صحيحة أثناء المعالجة. false (الافتراضي) - ستؤدي خطأ غياب الخط إلى إنهاء المعالجة عن طريق إلقاء استثناء.

```csharp
public bool IgnoreResourceFontErrors { get; set; }
```
