## خاصية TextSearchOptions.LogTextExtractionErrors

تحصل أو تعين إشارة بأن أخطاء استخراج النص (فك التشفير) سيتم تسجيلها في ممتص النص (الجزء). true - تعني أن أخطاء استخراج النص (فك التشفير) سيتم تسجيلها. قد يقلل من الأداء. false (افتراضي) - لا يوجد تسجيل للأخطاء.

```csharp
public bool LogTextExtractionErrors { get; set; }
```
