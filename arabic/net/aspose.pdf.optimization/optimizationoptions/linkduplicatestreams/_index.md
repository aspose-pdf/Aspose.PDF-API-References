## خاصية OptimizationOptions.LinkDuplicateStreams

إذا تم تعيين هذه العلامة إلى true، سيتم تحليل تدفقات الموارد. إذا تم العثور على تدفقات مكررة (أي إذا كانت محتويات التدفق متساوية)، فسيتم تخزين هذه التدفقات ككائن واحد. هذا يسمح بتقليل حجم الوثيقة في بعض الحالات (على سبيل المثال، عندما تم دمج نفس الوثيقة عدة مرات).

```csharp
public bool LinkDuplicateStreams { get; set; }
```
