## LineCap enumeration

يجب أن يحدد نمط نهاية الخط الشكل الذي سيتم استخدامه في نهايات المسارات الفرعية المفتوحة (والشرطات، إن وجدت) عند رسمها.

```csharp
public enum LineCap
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| ButtCap | `0` | نهاية مسطحة. يجب أن يكون الخط مربعًا عند نقطة نهاية المسار. لا يجب أن يكون هناك أي بروز يتجاوز نهاية المسار. |
| RoundCap | `1` | نهاية دائرية. يجب رسم قوس نصف دائري بقطر يساوي عرض الخط حول نقطة النهاية ويجب ملؤه. |
| SquareCap | `2` | نهاية مربعة بارزة. يجب أن يستمر الخط بعد نقطة نهاية المسار لمسافة تساوي نصف عرض الخط ويجب أن يكون مربعًا. |
