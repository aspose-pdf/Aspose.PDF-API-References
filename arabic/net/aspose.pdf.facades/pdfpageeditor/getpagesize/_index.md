### قيمة الإرجاع

النتيجة هي مثيل PageSize. استخدم خصائص العرض والارتفاع للكائن الذي تم إرجاعه للحصول على عرض الصفحة وارتفاعها.

### أمثلة

يوضح المثال التالي استخدام أسلوب GetPageSize:

```csharp
PdfPageEditor editor = new PdfPageEditor();
editor.BindPdf("sample.pdf");
PageSize size = editor.GetPageSize(1);
Console.WriteLine("Size of 1st page : " + size.Width + " x " + size.Height);
```
