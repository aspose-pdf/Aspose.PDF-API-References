## PdfContentEditor.CreateText method

إنشاء تعليق توضيحي نصي في مستند PDF

```csharp
public void CreateText(Rectangle rect, string title, string contents, bool open, string icon, 
    int page)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| rect | Rectangle | مستطيل التعليق التوضيحي الذي يحدد موقع التعليق التوضيحي على الصفحة. |
| title | String | عنوان التعليق التوضيحي. |
| contents | String | محتويات الشرح. |
| open | Boolean | علامة تحدد ما إذا كان يجب عرض التعليق التوضيحي مفتوحًا في البداية. |
| icon | String | سيتم استخدام اسم الرمز في عرض التعليق التوضيحي. يمكن أن تكون هذه القيمة: "تعليق" ، "مفتاح" ، "ملاحظة" ، "مساعدة" ، "NewParagraph" ، "فقرة" ، "إدراج" |
| page | Int32 | رقم الصفحة الأصلية حيث سيتم إنشاء التعليق التوضيحي النصي. |

### أمثلة

```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf("example.pdf");
editor.CreateText(new System.Drawing.Rectangle(0, 0, 100, 100),
    "Welcome to Aspose", "You are welcome to Aspose!", true, "Key", 1);
editor.Save("example_out.pdf");
```
