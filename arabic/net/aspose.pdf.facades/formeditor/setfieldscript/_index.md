## طريقة FormEditor.SetFieldScript

تعيين JavaScript لحقل PushButton. إذا كان هناك JavaScript قديم، فسيتم استبداله بالجديد.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| المعامل | النوع | الوصف |
| --- | --- | --- |
| fieldName | String | اسم الحقل المؤهل بالكامل. |
| script | String | Java script الذي سيتم إضافته/وضعه في حقل زر الدفع. |

### قيمة الإرجاع

true إذا تم تعيين نص الحقل بنجاح.
