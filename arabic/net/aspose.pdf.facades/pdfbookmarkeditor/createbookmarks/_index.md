## CreateBookmarks(Bookmark) {#createbookmarks_1}

لتكوين الإشارة المرجعية المحددة في المستند. يمكن استخدام الطريقة لتشكيل التسلسل الهرمي للإشارات المرجعية.

```csharp
public void CreateBookmarks(Bookmark bookmark)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| bookmark | Bookmark | ستتم إضافة الإشارة المرجعية إلى المستند. |

### أمثلة

```csharp
PdfBookmarkEditor editor = new PdfBookmarkEditor();
editor.BindPdf("example.pdf");
Bookmark bm1=new Bookmark();
bm1.PageNumber=1;
bm1.Title="First child";
Bookmark bm2=new Bookmark();
bm2.PageNumber=2;
bm2.Title="Second child";
Bookmark bm=new Bookmark();
bm.Action="GoTo";
bm.PageNumber=1;
bm.Title="Parent";
Bookmarks bms=new Bookmarks();
bms.Add(bm1);
bms.Add(bm2);
bm.ChildItem=bms;
editor.CreateBookmarks(bm);
editor.Save("example_out.pdf");
```
