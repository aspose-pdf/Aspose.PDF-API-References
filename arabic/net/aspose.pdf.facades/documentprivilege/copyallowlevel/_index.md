## خاصية DocumentPrivilege.CopyAllowLevel

تحصل على وتضبط مستوى النسخ لامتياز الوثيقة. تمامًا مثل إعدادات أذونات Adobe Professional. 0: لا شيء. 1: تمكين الوصول إلى النص لأجهزة قراءة الشاشة للمكفوفين. 2: تمكين نسخ النصوص والصور ومحتويات أخرى.

```csharp
public int CopyAllowLevel { get; set; }
```

## ملاحظات

إذا كانت الخاصية لها قيمة -1، فإن المستوى غير محدد.
