## PdfConverter.MergeImages method

يدمج قائمة تدفقات الصور كتدفق صورة واحد. يتم دعم تنسيقات مخرجات Png / jpg / tiff ، في حالة استخدام دفق إخراج بتنسيق غير مدعوم مشفر بتنسيق Jpeg افتراضيًا.

```csharp
public static Stream MergeImages(List<Stream> inputImagesStreams, ImageFormat outputImageFormat, 
    ImageMergeMode mergeMode, int? horizontal, int? vertical)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| inputImagesStreams | List`1 | قائمة تدفقات الصور المراد دمجها. |
| outputImageFormat | ImageFormat | تنسيق إخراج الصورة للتيار المدمج. |
| mergeMode | ImageMergeMode | وضع الدمج. تستخدم لتنسيقات Png / Jpg. |
| horizontal | Nullable`1 | النسبة الأفقية لتعيين عرض قماش الرسم لدفق الصورة الناتج. يستخدم لتنسيقات Png / Jpg مع ImageMergeMode.Center فقط. |
| vertical | Nullable`1 | النسبة العمودية لتعيين ارتفاع اللوحة القماشية لدفق الصورة الناتج. يستخدم لتنسيقات Png / Jpg مع ImageMergeMode.Center فقط. |
