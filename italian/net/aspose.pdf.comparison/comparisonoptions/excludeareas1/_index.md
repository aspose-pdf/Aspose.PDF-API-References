## Proprietà ComparisonOptions.ExcludeAreas1

Ottieni e imposta le aree da escludere. Utilizzato per la prima pagina o documento nel metodo di confronto. Questa opzione può essere impostata insieme a [`ExcludeTables`](../excludetables/). Questa opzione non può essere impostata insieme all'opzione [`ExtractionArea`](../extractionarea/).

public Rectangle[] ExcludeAreas1 { get; set; }
