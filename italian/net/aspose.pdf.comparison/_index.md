## Classi

| Classe | Descrizione |
| --- | --- |
| [ComparisonOptions](./comparisonoptions/) | Rappresenta una classe di opzioni per il confronto di documenti PDF. |
| [DiffOperation](./diffoperation/) | Rappresenta una classe di operazione di differenza. |
| [DocumentComparisonStatistics](./documentcomparisonstatistics/) | Rappresenta una classe di statistiche per il confronto di documenti. |
| [GraphicalPdfComparer](./graphicalpdfcomparer/) | Rappresenta una classe per confrontare graficamente documenti PDF. Dovrebbe essere utilizzata per cercare piccole modifiche, principalmente di natura grafica. Per confrontare le modifiche al contenuto testuale, utilizzare altre classi di confronto PDF. |
| [HtmlDiffOutputGenerator](./htmldiffoutputgenerator/) | Rappresenta una classe per generare una rappresentazione html delle differenze tra i testi. Le interruzioni di riga eliminate sono indicate da un segno di paragrafo. |
| [ImagesDifference](./imagesdifference/) | Rappresenta la classe risultato del confronto di due pagine PDF. |
| [JsonDiffOutputGenerator](./jsondiffoutputgenerator/) | Rappresenta una classe per visualizzare i risultati del confronto di documenti o pagine PDF in formato JSON. |
| [MarkdownDiffOutputGenerator](./markdowndiffoutputgenerator/) | Rappresenta una classe per generare una rappresentazione markdown delle differenze tra i testi. A causa della sintassi markdown, non è possibile mostrare le modifiche ai caratteri di spaziatura. La selezione delle modifiche comporta l'aggiunta di caratteri di spaziatura attorno alla formattazione, altrimenti il visualizzatore markdown non mostrerà correttamente il testo. Le interruzioni di riga eliminate sono indicate da - segno di paragrafo. |
| [OutputTextStyle](./outputtextstyle/) | Rappresenta una classe di set di stili per contrassegnare le modifiche al testo. |
| [PdfOutputGenerator](./pdfoutputgenerator/) | Rappresenta una classe per generare una rappresentazione PDF delle differenze tra i testi. |
| [SideBySideComparisonOptions](./sidebysidecomparisonoptions/) | Rappresenta una classe di opzioni per confrontare documenti con output affiancato. |
| [SideBySidePdfComparer](./sidebysidepdfcomparer/) |  |
| [TextItemComparisonStatistics](./textitemcomparisonstatistics/) | Rappresenta una classe di statistiche per il confronto di testi. |
| [TextPdfComparer](./textpdfcomparer/) | Rappresenta una classe per confrontare due pagine PDF o documenti PDF. |
| [TextStyle](./textstyle/) | Rappresenta una classe di stile per il testo. |
