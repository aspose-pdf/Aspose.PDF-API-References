## Proprietà SideBySideComparisonOptions.AdditionalChangeMarks

Ottieni e imposta la proprietà che determina se vengono visualizzati marcatori di cambiamento aggiuntivi. Se impostato, visualizza i marcatori di cambiamento che non sono nella pagina corrente ma sono presenti in un'altra pagina. Se il cambiamento si trova tra le parole, il marcatore potrebbe non essere posizionato esattamente rispetto al carattere di spazio bianco. Il valore predefinito è `false`.

```csharp
public bool AdditionalChangeMarks { get; set; }
```
