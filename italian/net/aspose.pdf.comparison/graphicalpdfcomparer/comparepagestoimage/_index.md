## Metodo GraphicalPdfComparer.ComparePagesToImage

Confronta le pagine graficamente. Il risultato del confronto è posizionato in un'immagine.

```csharp
public void ComparePagesToImage(Page page1, Page page2, string resultImagePath)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| page1 | Page | La prima pagina da confrontare. |
| page2 | Page | La seconda pagina da confrontare. |
| resultImagePath | String | Il percorso del file immagine di destinazione. |

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ArgumentException | Se le pagine da confrontare sono di dimensioni diverse. Se resultImagePath è nullo o una stringa vuota. C'è un formato di immagine di salvataggio sconosciuto. |
