Il **namespace Aspose.Pdf.Structure** fornisce classi che aiutano a lavorare con una struttura logica di un documento che presenta l'organizzazione del documento in capitoli e sezioni o l'identificazione di elementi speciali come figure, tabelle e note a piè di pagina.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Element](./element/) | Classe che rappresenta l'elemento base della struttura logica. |
| [ElementCollection](./elementcollection/) | Collezione di elementi base della struttura logica. |
| [FigureElement](./figureelement/) | Classe che rappresenta una figura della struttura logica. |
| [RootElement](./rootelement/) | Elemento radice della struttura. |
| [StructElement](./structelement/) | Elemento generale della struttura. |
| [TextElement](./textelement/) | Elemento di testo generale della struttura logica del documento. |
