## Proprietà Element.ActualText

(Opzionale; PDF 1.4) Testo che è una sostituzione esatta per l'elemento di struttura e i suoi figli. Questo testo di sostituzione (che dovrebbe applicarsi a un pezzo di contenuto il più piccolo possibile) è utile quando si estraggono i contenuti del documento a supporto dell'accessibilità per gli utenti con disabilità o per altri scopi.

public virtual string ActualText { get; set; }
