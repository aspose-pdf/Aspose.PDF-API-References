Lo **spazio dei nomi Aspose.Pdf.Drawing** fornisce classi che descrivono primitive grafiche come arco, linea, cerchio, ecc. Gli oggetti corrispondenti possono essere utilizzati per creare nuovi documenti.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Arc](./arc/) | Rappresenta un arco. |
| [Circle](./circle/) | Rappresenta un cerchio. |
| [Curve](./curve/) | Rappresenta una curva bezier. |
| [Ellipse](./ellipse/) | Rappresenta un'ellisse. |
| [GradientAxialShading](./gradientaxialshading/) | Rappresenta la classe di ombreggiatura assiale gradiente. |
| [GradientRadialShading](./gradientradialshading/) | Rappresenta il tipo di ombreggiatura radiale gradiente. |
| [Graph](./graph/) | Rappresenta un grafico - generatore di grafica paragrafo. |
| [Line](./line/) | Rappresenta una linea. |
| [Path](./path/) | Rappresenta un arco. |
| [PatternColorSpace](./patterncolorspace/) | Rappresenta la classe base del pattern. |
| [Rectangle](./rectangle/) | Rappresenta un rettangolo. |
| [Shape](./shape/) | Rappresenta una forma - l'oggetto grafico di base. |
