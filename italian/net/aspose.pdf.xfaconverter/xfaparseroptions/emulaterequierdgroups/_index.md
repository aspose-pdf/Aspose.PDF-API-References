## Proprietà XfaParserOptions.EmulateRequierdGroups

Se questa proprietà è vera, verranno disegnati rettangoli rossi aggiuntivi per i "gruppi esclusi" richiesti di Xfa. Questa proprietà è stata introdotta a causa dell'assenza di analoghi dei gruppi esclusi durante la conversione della rappresentazione Xfa dei moduli in standard. È falsa per impostazione predefinita.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
