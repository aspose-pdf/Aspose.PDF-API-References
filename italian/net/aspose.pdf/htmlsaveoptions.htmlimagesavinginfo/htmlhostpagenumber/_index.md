## Campo HtmlSaveOptions.HtmlImageSavingInfo.HtmlHostPageNumber

Indica al codice personalizzato a quale pagina del set generato di file di pagina HTML appartiene l'immagine salvata. Se la suddivisione in pagine è disattivata, questo valore contiene sempre '1' poiché in tal caso viene generata solo una pagina HTML.

```csharp
public int HtmlHostPageNumber;
```
