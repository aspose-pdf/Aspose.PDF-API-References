## Campi

| Nome | Descrizione |

| [ContentStream](../../aspose.pdf/resourcesavinginfo/contentstream) | Impostato dal convertitore. Rappresenta il contenuto binario del file salvato. |

Quindi, l'impostazione impostata su true significa che il codice personalizzato non ha elaborato il file di riferimento e il convertitore deve gestirlo da solo (in entrambi i casi - per il salvataggio da qualche parte e per la denominazione nel file di riferimento). |

| [ImageType](../../aspose.pdf/htmlimagesavinginfo/imagetype) | Rappresenta il tipo di immagine salvata referenziata in HTML. Impostato dal convertitore e può essere utilizzato nel codice personalizzato per decidere cosa fare |

Questa proprietà può indicare al codice personalizzato qual è il tipo di genitore dell'immagine elaborata. Viene impostato dal convertitore e può essere utilizzato nel codice personalizzato per decidere cosa deve essere fatto con quell'immagine (ad esempio il codice personalizzato può decidere dove salvare l'immagine o come deve essere referenziata nel contenuto del genitore). |
| [PdfHostPageNumber](../../aspose.pdf/htmlimagesavinginfo/pdfhostpagenumber) | Indica al codice personalizzato a quale pagina del documento PDF originale appartiene l'immagine salvata Poiché è possibile che non tutte le pagine del documento originale vengano salvate, questo valore indica il numero di pagina host nel PDF originale. Se il numero di pagina originale per qualche motivo è sconosciuto, restituisce sempre '1' |
