## TeXFragment.TeXLoadOptionsOfInstance property

Ottiene o imposta TeXLoadOptions che verrà utilizzato per il caricamento (e il rendering) di LaTeX in questa istanza di classe. Si prega di usarlo quando è necessario utilizzare l'impostazione specifica per l'importazione di LaTeX per questa o quella istanza (ad esempio quando questa o quella istanza dovrebbe utilizzare BasePath specifico per LaTeX importato o utilizzare un caricatore specifico di risorse esterne) Se il parametro è predefinito (null), verranno utilizzate le opzioni di caricamento LaTeX standard.

public TeXLoadOptions TeXLoadOptionsOfInstance { get; set; }
