## Proprietà XImage.Name

Ottiene o imposta il nome dell'immagine. Si prega di notare che se si cambia il nome dell'immagine che ha riferimenti nel contenuto della pagina, il documento potrebbe diventare errato. Si prega di utilizzare il metodo XImage.Rename in questo caso.

```csharp
public string Name { get; set; }
```
