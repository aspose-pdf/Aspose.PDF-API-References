## Proprietà XImage.ImageMask

Ottiene un flag che indica se l'immagine deve essere trattata come una maschera immagine (vedere 8.9.6, "Immagini Mascherate"). Se questo flag è vero, il valore di BitsPerComponent deve essere 1 e Mask e ColorSpace non devono essere specificati; le aree non mascherate devono essere dipinte utilizzando il colore attuale non di tratteggio. Valore predefinito: falso.

```csharp
public bool ImageMask { get; }
```

### Valore della Proprietà

Vero se l'immagine è una maschera immagine.
