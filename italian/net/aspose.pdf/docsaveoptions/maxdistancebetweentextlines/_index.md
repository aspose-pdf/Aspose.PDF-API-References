## Proprietà DocSaveOptions.MaxDistanceBetweenTextLines

Questo parametro è utilizzato per raggruppare le righe di testo in paragrafi. Determina quanto possono essere distanti due righe di testo relative. Specificato in centinaia di percento dell'altezza delle righe di testo.

```csharp
public float MaxDistanceBetweenTextLines { get; set; }
```
