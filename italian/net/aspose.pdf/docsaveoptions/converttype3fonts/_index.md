## Proprietà DocSaveOptions.ConvertType3Fonts

Ottiene o imposta la conversione per i font Type3. Nei font di tipo 3, i glifi devono essere definiti da flussi di operatori grafici. Questo significa che nell'output DOC/DOCX vediamo immagini invece di testo. Imposta questo flag su true per convertire i font Type3 in TTF e ottenere testo nel file risultante.

```csharp
public bool ConvertType3Fonts { get; set; }
```
