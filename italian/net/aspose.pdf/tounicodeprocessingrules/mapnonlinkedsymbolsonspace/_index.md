## Proprietà ToUnicodeProcessingRules.MapNonLinkedSymbolsOnSpace

Alcuni caratteri non forniscono informazioni sugli unicodes per alcuni simboli di testo. Questa mancanza di informazioni genera un errore "Il testo non può essere mappato su Unicode". Usa questo flag per mappare simboli non collegati su unicode "space"(codice 32).

```csharp
public bool MapNonLinkedSymbolsOnSpace { get; set; }
```
