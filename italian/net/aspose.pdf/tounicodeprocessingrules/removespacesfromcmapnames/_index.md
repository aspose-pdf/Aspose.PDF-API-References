## Proprietà ToUnicodeProcessingRules.RemoveSpacesFromCMapNames

Alcuni caratteri hanno mappe di codice carattere ToUnicode con spazi nei nomi. Questi spazi potrebbero causare errori con la mappatura del testo unicode. Questo flag comanda di rimuovere gli spazi dai nomi delle mappe di codice carattere ToUnicode. Per impostazione predefinita falso.

```csharp
public bool RemoveSpacesFromCMapNames { get; set; }
```
