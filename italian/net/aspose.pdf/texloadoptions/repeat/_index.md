## Proprietà TeXLoadOptions.Repeat

Ottiene/imposta il flag che indica se è necessario eseguire il lavoro TeX due volte nel caso, ad esempio, ci siano riferimenti nei file TeX di input. In generale, questo comportamento è utile quando il motore raccoglie alcuni dati durante il processo di composizione e li memorizza in un file ausiliario, tutto al primo avvio. E al secondo avvio, il motore in qualche modo utilizza quei dati.

```csharp
public bool Repeat { get; set; }
```
