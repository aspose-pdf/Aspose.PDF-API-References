## Proprietà PaginationArtifact.EndPage

Ottiene o imposta il numero di pagina finale per l'artefatto. Il valore deve essere maggiore o uguale a 0. Se viene impostato un valore inferiore a 0, verrà regolato a 0. Il valore predefinito di 0 significa che non ci sono limiti di pagina finale.

public int EndPage { get; set; }
