## SvgSaveOptions.TreatTargetFileNameAsDirectory field

Questa opzione definisce se verrà creata la directory di destinazione (se assente) con lo stesso nome del file di output richiesto invece del file di output richiesto stesso. Quindi, quella directory conterrà tutte le immagini SVG di output delle pagine (come descritto di seguito) . Se no, i file di output delle pagine diverse dalla prima verranno creati esattamente nella directory richiesta come file di output principale, ma conterranno nel nome del file il suffisso _[2...n], that è definito dal numero di pagina, fe se Definisci il file di output "C:\AsposeTests\output.svg" e l'output conterrà diversi file svg di pagine, quindi i file di pagine verranno creati anche nella directory "C:\AsposeTests\" e avranno i nomi 'output. svg', 'output_2.svg', 'output_3.svg' ecc.
