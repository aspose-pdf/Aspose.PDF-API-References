## HtmlFragment.HtmlLoadOptions property

Ottiene o imposta HtmlLoadOptions che verrà utilizzato per il caricamento (e il rendering) di HTML in questa istanza di classe. Usalo quando è necessario usa l'impostazione specifica per l'importazione di HTML per questa o quella istanza (ad esempio quando questa o quella istanza dovrebbe utilizzare BasePath specifico per l'HTML importato o utilizzare un caricatore specifico di risorse esterne) Se il parametro è predefinito (null), verranno utilizzate le opzioni di caricamento HTML standard.

public HtmlLoadOptions HtmlLoadOptions { get; set; }
