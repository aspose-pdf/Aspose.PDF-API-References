## Proprietà SaveOptions.CacheGlyphs

Ottiene o imposta un valore booleano che indica se i glifi del font saranno memorizzati nella cache durante la preparazione delle pagine aps. Migliora le prestazioni della conversione da pdf ad altri formati, ma aumenta il consumo di memoria.

```csharp
public bool CacheGlyphs { get; set; }
```
