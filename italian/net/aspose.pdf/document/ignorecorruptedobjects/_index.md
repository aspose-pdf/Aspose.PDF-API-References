## Document.IgnoreCorruptedObjects property

Ottiene o imposta il flag di ignoranza degli errori nei file di origine. Quando le pagine dal documento di origine vengono copiate nel documento di destinazione, il processo di copia viene interrotto con l'eccezione se alcuni oggetti nei file di origine sono danneggiati quando questo flag è falso. esempio: dest.Pages.Add(src.Pages); Se questo flag è impostato su true, gli oggetti danneggiati verranno sostituiti con valori vuoti. Per impostazione predefinita: true.
