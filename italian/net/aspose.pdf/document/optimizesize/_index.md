## Document.OptimizeSize property

Ottiene o imposta il flag di ottimizzazione. Quando le pagine vengono aggiunte al documento, flussi di risorse uguali nel file risultante vengono uniti in un oggetto PDF se questo flag è impostato. Ciò consente di ridurre la dimensione del file risultante, ma potrebbe causare un'esecuzione più lenta e requisiti di memoria maggiori. Valore predefinito: false.

public bool OptimizeSize { get; set; }
