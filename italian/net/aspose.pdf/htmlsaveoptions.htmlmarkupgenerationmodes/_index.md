## HtmlSaveOptions.HtmlMarkupGenerationModes enumeration

A volte sono presenti requisiti specifici per l'HTML creato. Questo enum definisce le modalità di preparazione dell'HTML che possono essere utilizzate durante la conversione di PDF in HTML per soddisfare tali requisiti specifici.

```csharp
public enum HtmlMarkupGenerationModes
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| WriteAllHtml | `0` | Modalità predefinita eventuali requisiti specifici sono assenti. Verrà generato un output che conterrà tutte le parti di HTML senza alcuna elaborazione aggiuntiva speciale. |
| WriteOnlyBodyContent | `1` | verrà rimosso tutto il contenuto HTML che è al di fuori del corpo dell'HTML, cioè verrà lasciato solo il contenuto che è all'interno tag |
