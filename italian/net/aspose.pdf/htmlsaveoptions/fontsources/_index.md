## Proprietà HtmlSaveOptions.FontSources

Fonti dei font pre-salvati.

```csharp
public FontSourceCollection FontSources { get; }
```

## Osservazioni

I font possono essere salvati preliminarmente per scopi di cache e poi passati nel processo di conversione Html. Ad esempio, potrebbe essere utile in uno scenario di suddivisione del documento e nell'elaborazione delle pagine del documento in più thread con un unico set di font.
