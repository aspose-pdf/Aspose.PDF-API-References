## Campo HtmlSaveOptions.HtmlMarkupGenerationMode

A volte sono presenti requisiti specifici per la generazione di markup HTML. Questo parametro definisce le modalità di preparazione HTML che possono essere utilizzate durante la conversione da PDF a HTML per soddisfare tali requisiti specifici.

```csharp
public HtmlMarkupGenerationModes HtmlMarkupGenerationMode;
```
