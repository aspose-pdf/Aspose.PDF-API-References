## Proprietà HtmlSaveOptions.SaveFullFont

Indica che il font completo sarà salvato, supporta solo i font True Type. Per impostazione predefinita SaveFullFont = false e il convertitore salva il sottoinsieme del font iniziale necessario per visualizzare il testo del documento.

```csharp
public bool SaveFullFont { get; set; }
```
