## Proprietà HtmlSaveOptions.MinimalLineWidth

Questo attributo imposta la larghezza minima della linea del percorso grafico. Se lo spessore della linea è inferiore a 1px, Adobe Acrobat lo arrotonda a questo valore. Quindi, questo attributo può essere utilizzato per emulare questo comportamento nei browser HTML.

```csharp
public float MinimalLineWidth { get; set; }
```
