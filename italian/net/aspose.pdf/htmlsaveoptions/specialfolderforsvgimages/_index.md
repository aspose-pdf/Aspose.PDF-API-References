## Campo HtmlSaveOptions.SpecialFolderForSvgImages

Ottiene o imposta il percorso della directory in cui devono essere salvate solo le immagini SVG se vengono incontrate durante il salvataggio del documento come HTML. Se il parametro è vuoto o nullo, i file SVG (se presenti) verranno salvati insieme ad altri file immagine (vicino al file di output) o in una cartella speciale per le immagini (se specificata nell'opzione SpecialImagesFolderIfAny). Non influisce su nulla se la proprietà CustomImageSavingStrategy è stata utilizzata con successo per elaborare il file immagine pertinente.

```csharp
public string SpecialFolderForSvgImages;
```
