## HtmlSaveOptions.CssClassNamesPrefix field

Quando il convertitore PDFtoHTML genera risultati CSS, i nomi delle classi CSS (qualcosa come ".stl_01 {}" ... ".stl_NN {}) vengono generati e utilizzati nei risultati CSS. Questa proprietà consente di impostare in modo forzato il prefisso del nome della classe Ad esempio, se si vuoi che tutti i nomi delle classi inizino con 'my_prefix_' (cioè erano qualcosa come 'my_prefix_1' ... 'my_prefix_NNN' ), quindi basta assegnare 'my_prefix_' a questa proprietà prima della conversione. Se questa proprietà rimarrà intatta (cioè null verrà lasciato come valore), quindi il convertitore genererà i nomi delle classi (sarà qualcosa come ".stl_01 {}" ... ".stl_NN {}")

```csharp
public string CssClassNamesPrefix;
```
