## Metodo OptimizedMemoryStream.Write

Quando sovrascritto in una classe derivata, scrive una sequenza di byte nello stream corrente e avanza la posizione corrente all'interno di questo stream in base al numero di byte scritti.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| buffer | Byte[] | Un array di byte. Questo metodo copia *count* byte da *buffer* allo stream corrente. |
| offset | Int32 | L'offset in byte basato su zero in *buffer* da cui iniziare a copiare byte nello stream corrente. |
| count | Int32 | Il numero di byte da scrivere nello stream corrente. |
