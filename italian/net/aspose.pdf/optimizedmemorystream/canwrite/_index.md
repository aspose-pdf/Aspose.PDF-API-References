## Proprietà OptimizedMemoryStream.CanWrite

Quando sovrascritta in una classe derivata, ottiene un valore che indica se lo stream corrente supporta la scrittura.

```csharp
public override bool CanWrite { get; }
```

### Valore di Ritorno

true se lo stream supporta la scrittura; altrimenti, false.
