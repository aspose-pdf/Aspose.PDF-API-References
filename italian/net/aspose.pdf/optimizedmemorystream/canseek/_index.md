## Proprietà OptimizedMemoryStream.CanSeek

Quando sovrascritta in una classe derivata, ottiene un valore che indica se lo stream corrente supporta la ricerca.

```csharp
public override bool CanSeek { get; }
```

### Valore di Ritorno

true se lo stream supporta la ricerca; altrimenti, false.
