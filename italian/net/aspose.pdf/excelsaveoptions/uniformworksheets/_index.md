## Proprietà ExcelSaveOptions.UniformWorksheets

Imposta true per utilizzare una divisione uniforme delle colonne attraverso il documento. Il valore predefinito è false; significa che la divisione delle colonne sarà indipendente per ogni pagina.

public bool UniformWorksheets { get; set; }
