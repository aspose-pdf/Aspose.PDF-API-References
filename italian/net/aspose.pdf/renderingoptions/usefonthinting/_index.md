## RenderingOptions.UseFontHinting property

L'utilizzo di questo flag attiva il meccanismo di suggerimento dei caratteri. Il suggerimento sui caratteri è l'uso di istruzioni matematiche per regolare la visualizzazione di un carattere di contorno. In alcuni casi l'attivazione di questo flag può risolvere problemi di leggibilità del testo. Al momento l'utilizzo di questo flag potrebbe avere effetto solo per i font TTF, se questi font sono usati nel documento sorgente.

```csharp
public bool UseFontHinting { get; set; }
```
