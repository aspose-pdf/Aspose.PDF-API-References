## Metodo DestinationCollection.GetPageNumber

Restituisce il numero di pagina della destinazione in base al nome.

```csharp
public int GetPageNumber(string destinameName, bool useCache)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| destinameName | String | Il nome della destinazione. |
| useCache | Boolean | Determina se viene utilizzata o meno la versione memorizzata nella cache della collezione. |

### Valore di Ritorno

Il numero di pagina se la destinazione è stata trovata; altrimenti, -1.
