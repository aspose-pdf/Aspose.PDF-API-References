## HtmlSaveOptions.CssUrlRequestInfo.CustomProcessingCancelled field

Dovrebbe essere impostato da codice personalizzato se non può o non deve definire l'URL che sarà utilizzato nell'HTML generato per il riferimento di quel CSS. Se è 'vero', allora il file CSS sarà salvato in modo standard nel luogo standard.

public bool CustomProcessingCancelled;
