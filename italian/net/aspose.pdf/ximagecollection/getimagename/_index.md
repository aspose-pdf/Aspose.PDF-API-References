## Metodo XImageCollection.GetImageName

Restituisce il nome nell'elenco delle immagini che è la chiave dell'immagine data.

```csharp
public string GetImageName(XImage image)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| image | XImage | Immagine da cercare. |

### Valore di ritorno

Nome (chiave) dell'immagine trovata; null se l'immagine non è stata trovata.
