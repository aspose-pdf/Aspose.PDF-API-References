## Replace(int, Stream, int, bool) {#replace_2}

Sostituisci l'immagine nella collezione con un'altra immagine.

```csharp
public void Replace(int index, Stream stream, int quality, bool isBlackAndWhite)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | Int32 | Indice dell'elemento della collezione che sarà sostituito nell'intervallo [1..numero di immagini]. |
| stream | Stream | Stream contenente i dati dell'immagine (in formato JPEG). |
| quality | Int32 | Qualità della compressione JPEG, in percentuale (valori validi sono 0..100). |
| isBlackAndWhite | Boolean | Se vero, l'immagine è compressa con il metodo di compressione CCITT che fornisce una migliore compressione per immagini in bianco e nero. Può essere utilizzato solo per immagini in bianco e nero. |
