## HtmlSaveOptions.CssSavingStrategy delegate

È possibile assegnare a questa proprietà una strategia personalizzata che implementa l'elaborazione o/e il salvataggio di una parte di CSS creata durante la conversione di PDF in HTML . In tal caso l'elaborazione (come il salvataggio su stream o disco) deve essere eseguita in quella personalizzazione codice

```csharp
public delegate void CssSavingStrategy(CssSavingInfo partSavingInfo);
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| partSavingInfo | CssSavingInfo | rappresenta l'insieme di dati che possono essere utilizzati per il salvataggio della parte CSS fornita |
