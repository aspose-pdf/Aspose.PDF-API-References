## SaveOptions.ResourceSavingInfo.CustomProcessingCancelled field

questo flag deve essere impostato su "true" nel codice personalizzato se per alcuni motivi il file proposto deve essere elaborato non con codice personalizzato ma con il codice del convertitore stesso in modo standard per il convertitore. Quindi, l'impostazione impostata su true significa che il codice personalizzato non ha elaborato il file di riferimento e il convertitore deve gestirlo da solo (in entrambi i casi - per il salvataggio da qualche parte e per la denominazione nel file di riferimento).

public bool CustomProcessingCancelled;
