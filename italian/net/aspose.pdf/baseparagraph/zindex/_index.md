## Proprietà BaseParagraph.ZIndex

Ottiene o imposta un valore int che indica l'ordine Z del grafico. Un grafico con un ZIndex maggiore sarà posizionato sopra il grafico con un ZIndex minore. ZIndex può essere negativo. I grafici con ZIndex negativo saranno posizionati dietro il testo nella pagina.

```csharp
public int ZIndex { get; set; }
```
