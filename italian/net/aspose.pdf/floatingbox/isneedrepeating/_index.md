## Proprietà FloatingBox.IsNeedRepeating

Ottiene o imposta un valore bool che indica se il paragrafo deve essere ripetuto nella pagina successiva. Il valore predefinito è false. L'attributo è valido solo quando il paragrafo stesso e l'oggetto a cui si riferisce il suo ReferenceParagraphID sono entrambi inclusi in RepeatingRows.

```csharp
public bool IsNeedRepeating { get; set; }
```
