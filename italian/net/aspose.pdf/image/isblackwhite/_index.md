## Proprietà Image.IsBlackWhite

Ottiene o imposta un valore bool che indica se l'immagine è forzata a essere in bianco e nero. Se viene utilizzata un'immagine TIFF del sottotipo CCITT, questa proprietà deve essere impostata su true.

```csharp
public bool IsBlackWhite { get; set; }
```
