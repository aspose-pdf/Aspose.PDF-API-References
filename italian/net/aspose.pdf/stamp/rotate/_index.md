## Proprietà Stamp.Rotate

Imposta o ottiene la rotazione del contenuto del timbro secondo i valori di [`Rotation`](../../rotation/). Nota. Questa proprietà è per angoli impostati che sono multipli di 90 gradi (0, 90, 180, 270 gradi). Per impostare un angolo arbitrario utilizzare la proprietà RotateAngle. Se l'angolo impostato da ArbitraryAngle non è un multiplo di 90, allora la proprietà Rotate restituisce Rotation.None.

```csharp
public Rotation Rotate { get; set; }
```
