## Proprietà Stamp.Zoom

Fattore di zoom del timbro. Consente di scalare il timbro. Si prega di notare che la coppia di proprietà ZoomX e ZoomY consente di impostare il fattore di zoom per ogni asse separatamente. L'impostazione di questa proprietà modifica sia le proprietà ZoomX che ZoomY. Se ZoomX e ZoomY sono diversi, allora la proprietà Zoom restituisce il valore di ZoomX.

```csharp
public double Zoom { get; set; }
```
