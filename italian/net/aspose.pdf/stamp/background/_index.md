## Proprietà Stamp.Background

Imposta o ottiene un valore bool che indica se il contenuto è timbrato come sfondo. Se il valore è true, il contenuto del timbro è posizionato in basso. Per impostazione predefinita, il valore è false, il contenuto del timbro è posizionato in alto.

```csharp
public bool Background { get; set; }
```
