## Proprietà MarkdownSaveOptions.UseImageHtmlTag

Ottiene e imposta il permesso di utilizzare un tag img per inserire immagini a sinistra e a destra del testo. In questo caso, nel visualizzatore markdown, il testo si adatterà attorno all'immagine.

```csharp
public bool UseImageHtmlTag { get; set; }
```
