## Proprietà MarkdownSaveOptions.HeadingLevels

Definisce i livelli di intestazione attesi da utilizzare nella strategia di riconoscimento delle intestazioni FontSize. Se il valore di questa proprietà è impostato, allora la strategia di riconoscimento delle intestazioni !:PdfToMarkdown.HeadingRecognitionStrategy.Heuristic sarà selezionata quando sono impostate le strategie !:PdfToMarkdown.HeadingRecognitionStrategy.Auto anche se il documento contiene segnalibri.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```
