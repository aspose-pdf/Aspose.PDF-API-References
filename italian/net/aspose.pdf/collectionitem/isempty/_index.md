## Proprietà CollectionItem.IsEmpty

Ottiene un valore che indica se l'elemento della collezione è vuoto.

```csharp
public bool IsEmpty { get; }
```

### Valore di Ritorno

true se l'elemento della collezione è vuoto; altrimenti, false.

## Osservazioni

Questa proprietà restituisce true se l'elemento della collezione non contiene alcun valore, inclusi valori di stringa, valori doppi, valori interi e valori di data. Se uno di questi tipi di valore è presente nell'elemento della collezione, questa proprietà restituisce false.
