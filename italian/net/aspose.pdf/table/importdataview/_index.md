```csharp
public void ImportDataView(DataView sourceDataView, bool isColumnNamesImported, int firstFilledRow, 
    int firstFilledColumn, int maxRows, int maxColumns)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| sourceDataView | DataView | IlDataView oggetto da importare. |
| isColumnNamesImported | Boolean | Indica se i nomi delle colonne verranno importati come prima riga. |
| firstFilledRow | Int32 | Il numero di riga in base zero della prima cella nella tabella di destinazione da cui inizierà l'importazione. Se la tabella di destinazione non contiene quella riga, verrà creata (e tutte le precedenti se necessario) |
