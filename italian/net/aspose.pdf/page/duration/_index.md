## Page.Duration property

Ottiene la durata di visualizzazione della pagina impostata. Questo è il tempo in secondi che la pagina deve essere visualizzata durante la presentazione. Returs -1 se la durata non è definita.

```csharp
public double Duration { get; set; }
```

### Esempi

L'esempio mostra come ottenere la durata della pagina

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
int pageRect = page.Duration;
```
