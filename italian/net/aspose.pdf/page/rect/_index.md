## Page.Rect property

Ottiene o imposta il rettangolo della pagina. La casella di ritaglio della pagina viene restituita se specificata, altrimenti viene restituita la casella di ritaglio della pagina. Si prega di notare che questa proprietà non considera la rotazione della pagina. Per ottenere il rettangolo della pagina considerando la rotazione, utilizzare ActualRect.

```csharp
public Rectangle Rect { get; set; }
```

### Esempi

Esempio mostra come ottenere il rettangolo di pagina:

```csharp
Document document = new Document("sample.pdf");
Page page = document.Pages[1];
Rectangle pageRect = page.Rect;
```
