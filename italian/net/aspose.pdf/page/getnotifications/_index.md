## Metodo Page.GetNotifications

Restituisce notifiche sulle operazioni interne con il contenuto della pagina. (Attualmente sono supportate solo le notifiche sugli eventi dei paragrafi negli scenari di aggiunta di testo.)

```csharp
public string GetNotifications()
```

### Valore di ritorno

Stringa che rappresenta le notifiche sulle operazioni interne con il contenuto della pagina.
