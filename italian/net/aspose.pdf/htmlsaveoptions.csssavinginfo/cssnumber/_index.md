## HtmlSaveOptions.CssSavingInfo.CssNumber field

Impostato dal convertitore. Durante la conversione vengono creati diversi file CSS. Questa proprietà mostra l'ordinal del file CSS salvato durante la conversione. Può essere utilizzato nella logica del codice personalizzato per decidere come elaborare o dove salvare il contenuto CSS

public int CssNumber;
