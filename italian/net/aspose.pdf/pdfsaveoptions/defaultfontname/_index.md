## PdfSaveOptions.DefaultFontName property

Nome font utilizzato di default per font assenti sul computer. Quando il documento PDF salvato in PDF contiene font non disponibili nel documento stesso e sul dispositivo, l'API sostituisce questi font con il font predefinito (se il carattere con`DefaultFontName` si trova sul dispositivo)
