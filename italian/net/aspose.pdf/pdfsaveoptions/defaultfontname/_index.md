## Proprietà PdfSaveOptions.DefaultFontName

Nome del carattere utilizzato per impostazione predefinita per i caratteri assenti sul computer. Quando il documento PDF salvato in PDF contiene caratteri che non sono disponibili nel documento stesso e sul dispositivo, l'API sostituisce questi caratteri con il carattere predefinito (se il carattere con `DefaultFontName` è trovato sul dispositivo)

```csharp
public string DefaultFontName { get; set; }
```
