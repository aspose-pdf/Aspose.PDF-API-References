## Proprietà ColumnInfo.ColumnWidths

Ottiene o imposta una stringa che contiene la larghezza delle colonne. Il valore di ogni colonna deve essere separato da uno spazio. L'unità predefinita è il punto, ma sono supportati anche cm, pollice e percentuale della larghezza disponibile. Ad esempio, "120 2.5cm 1.5inch"

```csharp
public string ColumnWidths { get; set; }
```
