## Proprietà ColumnInfo.ColumnSpacing

Ottiene o imposta una stringa che contiene lo spazio tra le colonne. Il valore di ogni spazio deve essere separato da uno spazio vuoto. L'unità predefinita è il punto, ma sono supportati anche cm e pollici. Ad esempio, "120 2.5cm 1.5inch".

```csharp
public string ColumnSpacing { get; set; }
```

## Osservazioni

Se questa proprietà non è impostata, verrà utilizzato il valore predefinito 0 per ogni spazio.
