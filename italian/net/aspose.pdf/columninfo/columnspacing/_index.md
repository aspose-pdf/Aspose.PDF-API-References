## ColumnInfo.ColumnSpacing property

Ottiene o imposta una stringa che contiene la spaziatura tra le colonne. Il valore di ciascuna spaziatura deve essere separato da uno spazio vuoto. L'unità predefinita è punto, ma sono supportati anche cm e pollici. Ad esempio, "120 2,5 cm 1,5 pollici".

```csharp
public string ColumnSpacing { get; set; }
```

### Osservazioni

Se questa proprietà non è impostata, verrà utilizzato il valore predefinito 0 per ogni spaziatura.
