## Classe CollectionField

Rappresenta una classe di campo dello schema di raccolta del documento.

```csharp
public class CollectionField
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [E](../../aspose.pdf/collectionfield/e/) { get; } | Ottiene un flag che indica se il processore PDF interattivo deve fornire supporto per la modifica del valore del campo. Valore predefinito: false |
| [FiledType](../../aspose.pdf/collectionfield/filedtype/) { get; } | Ottiene il tipo di un valore di campo in una raccolta di schema. Questo campo descrive il tipo di valore corrispondente a [`Subtype`](./subtype/). |
| [N](../../aspose.pdf/collectionfield/n/) { get; } | Ottiene il nome del campo testuale che deve essere presentato all'utente dal processore PDF interattivo |
| [O](../../aspose.pdf/collectionfield/o/) { get; } | Ottiene l'ordine relativo del nome del campo nell'interfaccia utente. I campi devono essere ordinati dal processore PDF interattivo in ordine crescente. |
| [Subtype](../../aspose.pdf/collectionfield/subtype/) { get; } | Ottiene il sottotipo di un valore di campo in una raccolta di schema. Il sottotipo del campo di raccolta o del campo relativo al file che questo dizionario descrive. Questa voce identifica il tipo di dati che devono essere memorizzati nel campo. |
| [V](../../aspose.pdf/collectionfield/v/) { get; } | Ottiene la visibilità iniziale del campo nell'interfaccia utente. Valore predefinito: true. |
