## Proprietà HtmlLoadOptions.InputEncoding

Ottiene o imposta l'attributo che specifica la codifica utilizzata per questo documento al momento dell'analisi. Se questo attributo è nullo, la codifica sarà determinata dall'attributo del set di caratteri del documento.

```csharp
public string InputEncoding { get; set; }
```
