## Add(Annotation, bool) {#add_1}

Aggiunge un'annotazione alla collezione. Se la pagina è ruotata, il rettangolo dell'annotazione verrà ricalcolato di conseguenza.

```csharp
public void Add(Annotation annotation, bool considerRotation)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| annotation | Annotation | Annotazione che deve essere aggiunta. |
| considerRotation | Boolean | Se vero e se la pagina è ruotata, la posizione dell'annotazione verrà ricalcolata in base alla rotazione della pagina. |
