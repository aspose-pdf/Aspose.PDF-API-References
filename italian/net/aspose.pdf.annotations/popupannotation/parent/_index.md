## Proprietà PopupAnnotation.Parent

Ottiene o imposta l'annotazione padre con cui questa annotazione pop-up sarà associata. Se questa voce è presente, le voci Contents, M, C e T dell'annotazione padre sovrascriveranno quelle dell'annotazione pop-up stessa.

public Annotation Parent { get; set; }
