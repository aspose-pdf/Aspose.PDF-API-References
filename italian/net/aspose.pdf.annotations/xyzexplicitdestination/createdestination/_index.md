## Metodo XYZExplicitDestination.CreateDestination

Crea una destinazione nella posizione specificata della pagina considerando la rotazione della pagina se necessario.

```csharp
public static XYZExplicitDestination CreateDestination(Page page, double left, double top, 
    double zoom, bool considerRotation)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| page | Page | Pagina di destinazione. |
| left | Double | Posizione a sinistra sulla pagina. |
| top | Double | Posizione in alto sulla pagina. |
| zoom | Double | Fattore di zoom (0 per predefinito). |
| considerRotation | Boolean | Se vero, la posizione sarà ricalcolata secondo la rotazione della pagina. |

### Valore di ritorno

Oggetto di destinazione.
