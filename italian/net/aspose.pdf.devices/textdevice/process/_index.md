## TextDevice.Process method

Converti la pagina e salvala come flusso di testo.

```csharp
public override void Process(Page page, Stream output)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| page | Page | La pagina da convertire. |
| output | Stream | Flusso di risultati. |

### Esempi

L'esempio mostra come estrarre il testo sulla prima pagina del documento PDF.

```csharp
Document doc = new Document(inFile);
string extractedText;

using (MemoryStream ms = new MemoryStream())
{
    // converte la pagina e salva il testo nello stream
    TextDevice device = new TextDevice();

    // crea un dispositivo di testo
    device.Process(doc.Pages[1], ms);

    // converte la pagina e salva il testo nello stream
    ms.Close();
    extractedText = Encoding.Unicode.GetString(ms.ToArray());
}
```
