Il **Aspose.Pdf.Devices**namespace fornisce classi che vengono utilizzate per rappresentare i documenti come immagini o testo normale. Quindi il documento può essere inviato a dispositivi testuali o grafici vari, il che significa che vogliamo ottenere una rappresentazione testuale o grafica.

| Classe | Descrizione |

| [Device](./device) | Classe astratta per tutti i tipi di dispositivi. Il dispositivo viene utilizzato per rappresentare un documento pdf in un formato. Ad esempio, la pagina del documento può essere rappresentata come immagine o testo. |
| [DocumentDevice](./documentdevice) | Classe astratta per tutti i dispositivi che viene utilizzata per elaborare l'intero documento pdf. |
| [EmfDevice](./emfdevice) | Rappresenta un dispositivo di immagine che aiuta a salvare le pagine del documento pdf in emf. |
| [GifDevice](./gifdevice) | Rappresenta un dispositivo di immagine che aiuta a salvare le pagine del documento pdf in gif. |
| [ImageDevice](./imagedevice) | Una classe astratta per dispositivi immagine. |
| [JpegDevice](./jpegdevice) | Rappresenta un dispositivo di immagine che aiuta a salvare le pagine di documenti pdf in jpeg. |
| [Margins](./margins) | Questa classe rappresenta i margini di un'immagine. |

| [TiffDevice](./tiffdevice) | Questa classe aiuta a salvare il documento pdf pagina per pagina in un'unica immagine tiff. |
| [TiffSettings](./tiffsettings) | Questa classe rappresenta le impostazioni per l'importazione di pdf in Tiff. |
