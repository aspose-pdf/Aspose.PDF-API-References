Lo **spazio dei nomi Aspose.Pdf.Devices** fornisce classi utilizzate per rappresentare documenti come immagine(i) o testo semplice. Quindi il documento può essere inviato a dispositivi testuali o grafici vari, il che significa che vogliamo ottenere la sua rappresentazione testuale o grafica.

## Classi

| Classe | Descrizione |
| --- | --- |
| [BmpDevice](./bmpdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in bmp. |
| [Device](./device/) | Classe astratta per tutti i tipi di dispositivi. Il dispositivo è utilizzato per rappresentare il documento pdf in un certo formato. Ad esempio, la pagina del documento può essere rappresentata come immagine o testo. |
| [DicomDevice](./dicomdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in formato Dicom. |
| [DocumentDevice](./documentdevice/) | Classe astratta per tutti i dispositivi utilizzati per elaborare l'intero documento pdf. |
| [EmfDevice](./emfdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in emf. |
| [GifDevice](./gifdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in gif. |
| [ImageDevice](./imagedevice/) | Una classe astratta per i dispositivi immagine. |
| [JpegDevice](./jpegdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in jpeg. |
| [Margins](./margins/) | Questa classe rappresenta i margini di un'immagine. |
| [PageDevice](./pagedevice/) | Classe astratta per tutti i dispositivi utilizzati per elaborare una certa pagina del documento pdf. |
| [PngDevice](./pngdevice/) | Rappresenta un dispositivo immagine che aiuta a salvare le pagine del documento pdf in png. |
| [Resolution](./resolution/) | Rappresenta una classe per contenere la risoluzione dell'immagine. |
| [TextDevice](./textdevice/) | Rappresenta una classe per convertire le pagine del documento pdf in testo. |
| [ThumbnailDevice](./thumbnaildevice/) | Rappresenta un dispositivo immagine che salva le pagine del documento pdf in un'immagine in miniatura. |
| [TiffDevice](./tiffdevice/) | Questa classe aiuta a salvare il documento pdf pagina per pagina in un'unica immagine tiff. |
| [TiffSettings](./tiffsettings/) | Questa classe rappresenta le impostazioni per importare pdf in Tiff. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [ColorDepth](./colordepth/) | Utilizzato per specificare il valore del parametro passato a un dispositivo immagine Tiff. |
| [CompressionType](./compressiontype/) | Utilizzato per specificare il valore del parametro passato a un dispositivo immagine Tiff. |
| [FormPresentationMode](./formpresentationmode/) | Utilizzato per specificare la modalità di presentazione del modulo durante la stampa o la conversione in immagini dei documenti pdf. |
| [ShapeType](./shapetype/) | Questa enum rappresenta il tipo di forma per le immagini estratte. |
