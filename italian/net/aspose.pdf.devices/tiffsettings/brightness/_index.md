## TiffSettings.Brightness property

Ottiene o imposta un limite di valore della trasformazione dei colori in bianco e nero. Questo parametro può essere applicato con EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle o ColorDepth.Format1bpp == 1

```csharp
public float Brightness { get; set; }
```

### Valore della proprietà

Il valore della luminosità deve essere compreso tra 0 e 1. Per impostazione predefinita, il valore è uguale a 0,33f
