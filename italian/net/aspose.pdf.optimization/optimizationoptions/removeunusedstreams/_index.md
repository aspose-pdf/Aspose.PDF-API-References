## OptimizationOptions.RemoveUnusedStreams property

Se questo flag è impostato su true, ogni risorsa viene controllata in base al suo utilizzo. Se la risorsa non viene mai utilizzata, le risorse vengono rimosse. Ciò potrebbe ridurre le dimensioni del documento, ad esempio quando le pagine sono state estratte dal documento.

public bool RemoveUnusedStreams { get; set; }
