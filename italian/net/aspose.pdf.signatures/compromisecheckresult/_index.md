## Classe CompromiseCheckResult

Rappresenta una classe per controllare le firme digitali dei documenti per compromissione.

```csharp
public sealed class CompromiseCheckResult
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [HasCompromisedSignatures](../../aspose.pdf.signatures/compromisecheckresult/hascompromisedsignatures/) { get; } | Indica se ci sono firme digitali compromesse nel documento. Restituisce true se almeno una firma è compromessa; altrimenti, false. |
| [SignaturesCoverage](../../aspose.pdf.signatures/compromisecheckresult/signaturescoverage/) { get; } | Ottiene lo stato di copertura delle firme digitali in un documento. Se è uguale a Undefined, allora una delle firme è compromessa. |

## Campi

| Nome | Descrizione |
| --- | --- |
| readonly [CompromisedSignatures](../../aspose.pdf.signatures/compromisecheckresult/compromisedsignatures/) | Ottiene una collezione di firme digitali che sono state identificate come compromesse. Questa proprietà contiene l'elenco di tutte le firme compromesse rilevate nel documento. |
