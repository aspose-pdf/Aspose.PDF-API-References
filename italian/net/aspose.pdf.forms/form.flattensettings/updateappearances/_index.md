## Proprietà Form.FlattenSettings.UpdateAppearances

Se impostato, tutte le apparizioni dei campi verranno rigenerate prima della compressione. Questa opzione può aiutare se il campo è stato compresso in modo errato. Questa opzione può ridurre le prestazioni. Per impostazione predefinita è impostato su false.

```csharp
public bool UpdateAppearances { get; set; }
```
