## Proprietà Form.RemovePermission

Se questa proprietà è vera, il dizionario "Perms" verrà rimosso dal documento pdf dopo la conversione di documenti dinamici in standard. Il dizionario "Perms" può contenere regole che disturbano la visualizzazione della selezione dei campi obbligatori in Adobe Acrobat reader. È falso per impostazione predefinita.

```csharp
public bool RemovePermission { get; set; }
```
