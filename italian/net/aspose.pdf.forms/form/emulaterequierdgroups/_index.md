## Proprietà Form.EmulateRequierdGroups

Se questa proprietà è vera, verranno disegnati ulteriori rettangoli di confine rossi per i contenitori degli elementi Xfa exclGroup richiesti. Questa proprietà è stata introdotta a causa dell'assenza di analoghi per l'exclGroup durante la conversione della rappresentazione Xfa dei moduli in standard. È falsa per impostazione predefinita.

```csharp
public bool EmulateRequierdGroups { get; set; }
```
