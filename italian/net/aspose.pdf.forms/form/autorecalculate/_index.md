## Proprietà Form.AutoRecalculate

Se impostato, tutti i campi del modulo verranno ricalcolati quando un campo viene modificato. Il valore predefinito è true. Impostare su false per aumentare le prestazioni quando si compila un modulo con un grande numero di campi calcolati.

```csharp
public bool AutoRecalculate { get; set; }
```
