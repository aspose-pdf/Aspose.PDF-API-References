## Proprietà RadioButtonField.NoToggleToOff

Ottiene o imposta il flag che consente al pulsante di opzione di non avere un valore selezionato. Se `true`, esattamente un pulsante di opzione deve essere selezionato in ogni momento; selezionare il pulsante attualmente selezionato non ha effetto. Se `false`, fare clic sul pulsante selezionato lo deseleziona, lasciando nessun pulsante selezionato.

```csharp
public bool NoToggleToOff { get; set; }
```

## Osservazioni

Alcuni lettori PDF (incluso Adobe Acrobat) potrebbero ignorare lo stato `false` del flag.
