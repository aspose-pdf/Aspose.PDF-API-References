## Proprietà Field.IsSharedField

Proprietà per il supporto del Generatore. Utilizzato quando il campo viene aggiunto all'intestazione o al piè di pagina. Se vero, questo campo verrà creato una sola volta e la sua apparizione sarà visibile su tutte le pagine del documento. Se falso, verrà creato un campo separato per ogni pagina del documento.

```csharp
public bool IsSharedField { get; set; }
```
