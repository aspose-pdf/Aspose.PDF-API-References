## Proprietà Field.AlternateName

Ottiene o imposta il nome alternativo del campo (Un nome alternativo del campo che deve essere utilizzato al posto del nome effettivo del campo ogni volta che il campo deve essere identificato nell'interfaccia utente). Il nome alternativo è utilizzato come tooltip del campo in Adobe Acrobat.

public string AlternateName { get; set; }
