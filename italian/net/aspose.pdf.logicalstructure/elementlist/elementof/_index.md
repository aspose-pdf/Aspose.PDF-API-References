## Indicizzatore ElementList

Ottiene un elemento all'indice specificato.

```csharp
public virtual Element ElementOf[int index] { get; }
```

| Parametro | Descrizione |
| --- | --- |
| index | L'indice nella lista degli elementi. |

### Valore di Ritorno

L'Elemento con l'indice specificato nella collezione. Se l'indice è maggiore o uguale al numero di elementi nella lista, restituisce null.
