## Campo StructureTypeStandard.RP

(Punteggiatura ruby) Punteggiatura che circonda il testo dell'annotazione ruby. Viene utilizzata solo quando un'annotazione ruby non può essere formattata correttamente in uno stile ruby e invece è formattata come un commento normale, o quando è formattata come un warichu. Contiene testo (di solito un singolo PARENTESE SINISTRA o DESTRA o un carattere di delimitazione simile).

```csharp
public static readonly StructureTypeStandard RP;
```
