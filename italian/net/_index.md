## Namespace

| Namespace | Descrizione |
| --- | --- |
| [Aspose.Pdf](./aspose.pdf/) | Il **Aspose.Pdf** è un namespace radice per tutte le classi della libreria Aspose.Pdf che sono direttamente in esso come **Document** o indirettamente attraverso diversi sottospazi dei nomi. |
| [Aspose.Pdf.AI](./aspose.pdf.ai/) | Il namespace **Aspose.Pdf.AI** fornisce classi per funzionalità AI, inclusi client API e assistenti intelligenti. |
| [Aspose.Pdf.Annotations](./aspose.pdf.annotations/) | Il namespace **Aspose.Pdf.Annotations** fornisce classi per lavorare con vari tipi di azioni, destinazioni e altre caratteristiche del documento che tradizionalmente vengono chiamate interattive, fornendo mezzi affinché l'utente possa interagire con esso. |
| [Aspose.Pdf.Comparison](./aspose.pdf.comparison/) | Il namespace **Aspose.Pdf.Comparison** fornisce classi utilizzate per il confronto dei documenti. |
| [Aspose.Pdf.DataEditor](./aspose.pdf.dataeditor/) | Il **Aspose.Pdf.DataEditor** contiene strumenti per modificare i dati all'interno di un documento. |
| [Aspose.Pdf.Devices](./aspose.pdf.devices/) | Il namespace **Aspose.Pdf.Devices** fornisce classi utilizzate per rappresentare un documento come immagine(i) o testo semplice. Quindi il documento può essere inviato a dispositivi testuali o grafici vari, il che significa che vogliamo ottenerne una rappresentazione testuale o grafica. |
| [Aspose.Pdf.Drawing](./aspose.pdf.drawing/) | Il namespace **Aspose.Pdf.Drawing** fornisce classi che descrivono primitive grafiche come arco, linea, cerchio, ecc. Gli oggetti corrispondenti possono essere utilizzati per creare nuovi documenti. |
| [Aspose.Pdf.Facades](./aspose.pdf.facades/) | Il namespace **Aspose.Pdf.Facades** fornisce classi originariamente provenienti da Aspose.Pdf.Kit. Queste classi sono utilizzate per manipolare documenti eseguendo operazioni come concatenazione, timbratura, firma, annotazione, ecc. ma a un livello elevato senza accesso alla struttura interna di un documento. |
| [Aspose.Pdf.Forms](./aspose.pdf.forms/) | Il namespace **Aspose.Pdf.Forms** ha classi che descrivono moduli (standard, statici, dinamici) e vari tipi di campi come casella di testo, casella di elenco, pulsante di opzione, ecc. |
| [Aspose.Pdf.LogicalStructure](./aspose.pdf.logicalstructure/) | Il **Aspose.Pdf.Tagged.LogicalStructure** è un namespace per le classi di Struttura Logica PDF. |
| [Aspose.Pdf.Multithreading](./aspose.pdf.multithreading/) | Il namespace **Aspose.Pdf.Multithreading** fornisce classi destinate ad essere utilizzate in scenari multi-thread. |
| [Aspose.Pdf.Operators](./aspose.pdf.operators/) | Il **Aspose.Pdf.Operators** è un namespace per le implementazioni degli Operatori. Queste classi descrivono gli operatori utilizzati nei contenuti delle pagine PDF. |
| [Aspose.Pdf.Optimization](./aspose.pdf.optimization/) | Il **Aspose.Pdf.Optimization** è un namespace per classi per la gestione del processo di ottimizzazione dei documenti. |
| [Aspose.Pdf.Plugins](./aspose.pdf.plugins/) | Il **Aspose.Pdf.Plugins** è un namespace radice per tutte le classi delle classi Aspose.Pdf.Plugins che sono direttamente in esso come **PdfOrganizer** o indirettamente attraverso diversi sottospazi dei nomi. |
| [Aspose.Pdf.Printing](./aspose.pdf.printing/) | Il namespace **Aspose.Pdf.Printing** fornisce classi e funzionalità per gestire le operazioni di stampa PDF, inclusi impostazioni della stampante, impostazioni della pagina e estensioni per gestire configurazioni relative alla stampa. |
| [Aspose.Pdf.Sanitization](./aspose.pdf.sanitization/) | Il **Aspose.Pdf.Sanitization** è un namespace per le operazioni di sanificazione. |
| [Aspose.Pdf.Security](./aspose.pdf.security/) | Il namespace **Aspose.Pdf.Security** contiene classi utilizzate per la crittografia e la firma digitale. |
| [Aspose.Pdf.Structure](./aspose.pdf.structure/) | Il namespace **Aspose.Pdf.Structure** fornisce classi che aiutano a lavorare con una struttura logica di un documento che presenta l'organizzazione del documento in capitoli e sezioni o l'identificazione di elementi speciali come figure, tabelle e note a piè di pagina. |
| [Aspose.Pdf.Tagged](./aspose.pdf.tagged/) | Il **Aspose.Pdf.Tagged** è un namespace per classi per il supporto di documenti PDF Tagged. |
| [Aspose.Pdf.Text](./aspose.pdf.text/) | Il namespace **Aspose.Pdf.Text** fornisce classi che consentono di estrarre testo, aggiungere testo, manipolare testo esistente di un documento. Contiene anche classi che consentono di estrarre, sostituire, sostituire i caratteri di un documento. |
| [Aspose.Pdf.Vector](./aspose.pdf.vector/) | Il **Aspose.Pdf.Vector** è un namespace radice per le operazioni grafiche. |
| [Aspose.Pdf.XfaConverter](./aspose.pdf.xfaconverter/) | Il namespace **Aspose.Pdf.XfaConverter** fornisce classi che consentono di convertire XFA. |
