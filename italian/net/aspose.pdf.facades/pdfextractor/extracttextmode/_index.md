## PdfExtractor.ExtractTextMode property

Imposta la modalità per estrarre il risultato del testo.

```csharp
public int ExtractTextMode { get; set; }
```

### Valore della proprietà

0 è la modalità di testo puro e 1 è la modalità di ordinamento grezzo. Il valore predefinito è 0.

### Esempi

L'esempio dimostra il`ExtractTextMode` utilizzo della proprietà nello scenario di estrazione del testo.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractTextMode = 1;
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```
