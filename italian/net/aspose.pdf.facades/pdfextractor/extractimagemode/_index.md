## Proprietà PdfExtractor.ExtractImageMode

Imposta la modalità per il processo di estrazione delle immagini.

```csharp
public ExtractImageMode ExtractImageMode { get; set; }
```

## Osservazioni

Il valore predefinito è ExtractImageMode.DefinedInResources che estrae tutte le immagini definite nelle risorse. Per estrarre le immagini effettivamente mostrate, dovrebbe essere utilizzata la modalità ExtractImageMode.ActuallyUsed.
