## Proprietà PdfExtractor.Resolution

Imposta o ottiene la risoluzione per le immagini estratte. Il valore predefinito è 150. Le immagini che hanno un valore di risoluzione maggiore sono più chiare. Tuttavia, l'aumento del valore di risoluzione comporta un aumento del tempo e della memoria necessari per estrarre le immagini. Di solito, per ottenere un'immagine chiara è sufficiente impostare la risoluzione a 150 o 300.

public int Resolution { get; set; }
