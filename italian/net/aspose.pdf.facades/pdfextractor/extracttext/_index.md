## ExtractText() {#extracttext}

Estrae il testo da un documento Pdf utilizzando la codifica Unicode.

```csharp
public void ExtractText()
```

### Esempi

Il primo esempio mostra come estrarre tutto il testo dal file PDF. Il secondo esempio mostra come estrarre il testo di ogni pagina in un file txt.

```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(@"D:\Text\text.pdf");
extractor.ExtractText();
extractor.GetText(@"D:\Text\text.txt");
```


```csharp
PdfExtractor extractor = new PdfExtractor();
extractor.BindPdf(TestPath + @"Aspose.Pdf.Kit.Pdf");
extractor.ExtractText();

String suffix = ".txt";
int pageCount = 1;
```
