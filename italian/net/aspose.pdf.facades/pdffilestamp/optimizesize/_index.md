## Proprietà PdfFileStamp.OptimizeSize

Ottiene o imposta il flag di ottimizzazione. I flussi di risorse uguali nel file risultante vengono uniti in un oggetto PDF se questo flag è impostato. Questo consente di ridurre la dimensione del file risultante, ma può causare un'esecuzione più lenta e maggiori requisiti di memoria. Valore predefinito: false.

```csharp
public bool OptimizeSize { get; set; }
```
