## PdfFileStamp.StartingNumber property

Ottiene o imposta il numero iniziale per la prima pagina nel file di input. Le pagine successive verranno numerate a partire da questo valore. Ad esempio, se StartingNumber è impostato su 100, le pagine del documento avranno i numeri 100, 101, 102...

```csharp
public int StartingNumber { get; set; }
```

### Esempi

```csharp
PdfFileStamp fileStamp = new PdfFileStamp("input.pdf", "output.pdf");
fileStamp.StartingNumber = 100;
fileStamp.AddPageNumber("Page #");
fileStamp.Close();
```
