## Metodo PdfContentEditor.CreateSound

Crea annotazioni sonore.

```csharp
public void CreateSound(Rectangle rect, string filePath, string name, int page, string rate)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| rect | Rectangle | Il rettangolo di annotazione che definisce la posizione dell'annotazione sulla pagina. |
| filePath | String | Il percorso del file audio. |
| name | String | Il nome di un'icona da utilizzare per visualizzare l'annotazione, includere: Speaker e Mic. |
| page | Int32 | La pagina in cui viene creata l'annotazione sonora. |
| rate | String | La frequenza di campionamento, in campioni al secondo. |
