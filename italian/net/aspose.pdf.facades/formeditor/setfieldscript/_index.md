## Metodo FormEditor.SetFieldScript

Imposta JavaScript per un campo PushButton. Se esisteva un vecchio JavaScript, verrà sostituito da quello nuovo.

```csharp
public bool SetFieldScript(string fieldName, string script)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fieldName | String | Il nome del campo completamente qualificato. |
| script | String | Il Java script da aggiungere/posizionare in un campo pulsante. |

### Valore di ritorno

true se lo script del campo è stato impostato con successo.
