## Metodo FormEditor.AddFieldScript

Aggiungi JavaScript per un campo PushButton. Se esiste un vecchio evento, il nuovo evento viene aggiunto dopo di esso.

```csharp
public bool AddFieldScript(string fieldName, string script)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fieldName | String | Il nome del campo completamente qualificato. |
| script | String | Il Java script da aggiungere/posizionare in un campo pulsante. |

### Valore di ritorno

True nel caso in cui lo script sia stato aggiunto con successo.
