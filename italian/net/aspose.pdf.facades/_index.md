# Aspose.Pdf.Facades  
## Riferimento API Aspose.PDF for .NET  

Questo documento riferisce all'API Aspose.Pdf.Facades, che offre funzionalità avanzate per la manipolazione di documenti PDF.  



## Enumerazioni  

### Enumerazioni  
- **Algorithm** - Algoritmi per la protezione del documento PDF  
- **AutoRotateMode** - Modalità di rotazione durante la stampa del documento PDF  
- **BlendingColorSpace** - Spazi di colorazione per il blend  
- **DataType** - Tipi di dati per i campi del documento PDF  
- **DefaultMetadataProperties** - Proprietà metadati standard XMP  
- **EncodingType** - Tipi di codifica del testo nel documento PDF  
- **FieldType** - Tipi di campi nel documento PDF  
- **FontStyle** - Stili di caratteri per il documento PDF  
- **ImageMergeMode** - Modalità di fusione delle immagini nel documento PDF  
- **KeySize** - Dimensioni delle chiavi per la protezione del documento PDF  
- **PositioningMode** - Modalità di posizionamento delle immagini nel documento PDF  
- **PropertyFlag** - Bandiere delle proprietà nel documento PDF  
- **StampType** - Tipi di stampature nel documento PDF  
- **SubmitFormFlag** - Bandiere per il submit dei form nel documento PDF  
- **WordWrapMode** - Modalità di avvolgimento delle parole nel documento PDF
