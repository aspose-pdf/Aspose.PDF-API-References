## PdfFileEditor.OptimizeSize property

Ottiene o imposta il flag di ottimizzazione. Flussi di risorse uguali nel file risultante vengono uniti in un oggetto PDF se questo flag è impostato. Ciò consente di ridurre la dimensione del file risultante ma può causare un'esecuzione più lenta e maggiori requisiti di memoria. Valore predefinito: false.
