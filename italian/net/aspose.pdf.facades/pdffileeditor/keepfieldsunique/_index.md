## Proprietà PdfFileEditor.KeepFieldsUnique

Se vero, i nomi dei campi saranno resi unici quando i moduli vengono concatenati. I suffissi saranno aggiunti ai nomi dei campi, il modello di suffisso può essere specificato nella proprietà UniqueSuffix.

```csharp
public bool KeepFieldsUnique { get; set; }
```
