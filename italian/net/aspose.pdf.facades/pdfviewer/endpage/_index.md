## Evento PdfViewer.EndPage

Si verifica quando la stampa di una pagina termina nel PdfViewer.

```csharp
public event EventHandler<StartEndPageEventArgs> EndPage;
```

## Osservazioni

Questo evento viene attivato dopo che una pagina è stata elaborata nel PdfViewer. Può essere utilizzato per eseguire azioni o aggiornamenti dopo la stampa di una pagina.
