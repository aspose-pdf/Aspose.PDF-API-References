## PdfViewer.AutoResize property

Ottiene o imposta un valore bool che indica se il file deve essere stampato con dimensioni ottimizzate.  Se falsa stampa la pagina senza ridimensionamento pagina. Se true stampa la pagina con ridimensionamento per adattarla all'area stampabile.
