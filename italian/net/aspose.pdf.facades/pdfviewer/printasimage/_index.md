## Proprietà PdfViewer.PrintAsImage

Imposta o ottiene una modalità per PdfViewer per stampare come immagine.

```csharp
public bool PrintAsImage { get; set; }
```

## Osservazioni

Se true stampa sempre come immagine (genera un'immagine che viene stampata) Se false stampa direttamente sul dispositivo se tutte le funzionalità sono supportate. Nel caso in cui il documento contenga funzionalità non supportate, il sistema può decidere automaticamente di stampare come immagine. Il valore predefinito è false.
