## Metodo PdfFileSignature.GetAccessPermissions

Restituisce il valore delle autorizzazioni di accesso del documento certificato dal tipo di firma MDP.

```csharp
public DocMDPAccessPermissions GetAccessPermissions()
```

### Valore di Ritorno

Se il documento è in fase di certificazione, restituisce il valore delle autorizzazioni di accesso; altrimenti, viene generata un'eccezione.
