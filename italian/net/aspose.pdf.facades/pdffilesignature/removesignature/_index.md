### Esempi

```csharp
[C#]
string inFile = TestPath + "example1.pdf";
PdfFileSignature pdfSign = new PdfFileSignature();
pdfSign.BindPdf(inFile); 
IList names = pdfSign.GetSignNames();
for(int i = 0; i < names.Count; i++)
