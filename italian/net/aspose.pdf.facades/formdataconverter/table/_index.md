## Proprietà FormDataConverter.Table

Ottiene o imposta il contenitore di dati intermedi, un DataTable. Deve essere definito prima di convertire i dati da un formato a un altro formato. Le colonne e il TableName del DataTable devono essere definiti. Il TableName è il nome della Tabella nel database. Ogni ColumnName della colonna è il nome del campo qualificato del pdf. Ogni Caption della colonna è il nome della colonna della tabella nel database. Se il nome del campo è lo stesso del nome della colonna della tabella, il Caption non deve essere specificato.

```csharp
public DataTable Table { get; set; }
```
