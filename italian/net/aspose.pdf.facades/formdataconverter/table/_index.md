## FormDataConverter.Table property

Ottiene o imposta il contenitore di dati intermedio, un DataTable. Deve essere definito prima di convertire i dati da un formato a un altro formato. È necessario definire le colonne e TableName di DataTable. Il TableName è il nome della tabella nel database. ColumnName di ogni colonna è il nome del campo qualificato del pdf. Caption di ogni colonna è il nome della colonna della tabella nel database. Se il nome del campo è uguale al nome della colonna della tabella, non è necessario specificare la didascalia.

public DataTable Table { get; set; }
