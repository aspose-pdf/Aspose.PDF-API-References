```csharp
public void ExportAnnotationsXfdf(Stream xmlOutputStream, int start, int end, string[] annotTypes)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| xmlOutputStream | Stream | Il flusso XFDF di output. |
| start | Int32 | Pagina iniziale da cui verranno esportate le annotazioni del documento. |
| end | Int32 | Pagina finale in cui verranno esportate le annotazioni del documento. |
| annotTypes | String[] | L'array di tipi di annotazione deve essere esportato. |

### Esempi

```csharp

editor.BindPdf("example.pdf");
string[] annotTypes = new string[] {"Text", "Highlight"};
using (Stream stream = File.Create("example.xfdf"))
{
    editor.ExportAnnotationsXfdf(stream, 1, 2, annotTypes);
}
```
