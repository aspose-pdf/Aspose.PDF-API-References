Lo **spazio dei nomi Aspose.Pdf.Printing** fornisce classi e funzionalità per gestire le operazioni di stampa PDF, comprese le impostazioni della stampante, le impostazioni della pagina e le estensioni per gestire le configurazioni relative alla stampa.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CustomPrintEventArgs](./customprinteventargs/) | Fornisce dati per l'evento [`CustomPrint`](../aspose.pdf.facades/pdfviewer/customprint/). |
| [PageSettings](./pagesettings/) | Specifica le impostazioni che si applicano a una singola pagina stampata. |
| [PageSettingsExtensions](./pagesettingsextensions/) | Rappresenta metodi di estensione per [`PageSettings`](../aspose.pdf.printing/pagesettings/). |
| [PaperSize](./papersize/) | Specifica la dimensione di un foglio di carta. |
| [PaperSizeExtensions](./papersizeextensions/) | Rappresenta metodi di estensione per [`PaperSize`](../aspose.pdf.printing/papersize/). |
| [PaperSizes](./papersizes/) | Rappresenta le dimensioni standard della carta. |
| [PaperSource](./papersource/) | Specifica il vassoio della carta da cui la stampante preleva la carta. |
| [PaperSourceExtensions](./papersourceextensions/) | Rappresenta metodi di estensione per [`PaperSource`](../aspose.pdf.printing/papersource/). |
| [PaperSources](./papersources/) | Fornisce un insieme di istanze predefinite di [`PaperSource`](../aspose.pdf.printing/papersource/) che rappresentano fonti di carta comuni. |
| [PdfQueryPageSettingsEventArgs](./pdfquerypagesettingseventargs/) | Fornisce dati per l'evento [`PdfQueryPageSettings`](../aspose.pdf.facades/pdfviewer/pdfquerypagesettings/). |
| [PrinterResolution](./printerresolution/) | Rappresenta la risoluzione supportata da una stampante. |
| [PrinterResolutionExtensions](./printerresolutionextensions/) | Rappresenta metodi di estensione per [`PrinterResolution`](../aspose.pdf.printing/printerresolution/). |
| [PrinterSettings](./printersettings/) | Specifica informazioni su come viene stampato un documento, comprese le stampanti che lo stampano. |
| [PrinterSettingsExtensions](./printersettingsextensions/) | Rappresenta metodi di estensione per [`PrinterSettings`](../aspose.pdf.printing/printersettings/). |
| [StartEndPageEventArgs](./startendpageeventargs/) | Fornisce dati per gli eventi [`StartPage`](../aspose.pdf.facades/pdfviewer/startpage/) e [`EndPage`](../aspose.pdf.facades/pdfviewer/endpage/) della classe [`PdfViewer`](../aspose.pdf.facades/pdfviewer/). |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [Duplex](./duplex/) | Specifica l'impostazione duplex della stampante. |

| [PaperSourceKind](./papersourcekind/) | Fonti di carta standard. |
| [PrinterResolutionKind](./printerresolutionkind/) | Specifica una risoluzione della stampante. |
| [PrintRange](./printrange/) | Specifica l'opzione che designa la parte del documento da stampare. |
