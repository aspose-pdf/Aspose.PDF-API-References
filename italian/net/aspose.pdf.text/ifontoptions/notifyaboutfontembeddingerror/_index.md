## IFontOptions.NotifyAboutFontEmbeddingError property

A volte non è possibile incorporare il carattere desiderato nel documento. Ci sono molte ragioni, per esempio restrizioni di licenza o quando il font desiderato non è stato trovato sul computer di destinazione. Quando si verifica questa situazione non è semplicemente da rilevare, perché il font desiderato è incorporato tramite il set di flag di proprietà Font.IsEmbedded = true; Ovviamente è possibile leggere questa proprietà subito dopo che è stata impostata ma non è un approccio conveniente. Flag NotifyAboutFontEmbeddingError applica il meccanismo di eccezione per i casi in cui il tentativo di incorporare il carattere non è riuscito. Se questo flag è impostato un'eccezione di tipo [`FontEmbeddingException`](../../../aspose.pdf/fontembeddingexception) sarà lanciato. Per impostazione predefinita false.

public bool NotifyAboutFontEmbeddingError { get; set; }
