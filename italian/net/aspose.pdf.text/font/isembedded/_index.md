## Font.IsEmbedded property

Ottiene o imposta un valore che indica se il carattere è incorporato.

```csharp
public bool IsEmbedded { get; set; }
```

### Esempi

L'esempio seguente mostra come trovare un carattere, contrassegnarlo come incorporato, cercare testo nella pagina del documento e sostituire il carattere del testo.

```csharp
// Apri documento
Font font = FontRepository.FindFont("Arial");
font.IsEmbedded = true;

// Crea un oggetto TextFragmentAbsorber per trovare tutte le occorrenze di testo "hello world".
Document doc = new Document(@"D:\Tests\input.pdf");

// Accetta l'assorbitore per la prima pagina
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");
// Visualizza il nome del carattere della prima occorrenza del testo
doc.Pages[1].Accept(absorber);

// Crea il carattere e contrassegnalo da incorporare
absorber.TextFragments[1].TextState.Font = font;

// apri il documento
doc.Save(@"D:\Tests\output.pdf"); 
```
