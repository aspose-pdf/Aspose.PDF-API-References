## Font.IsSubset property

Ottiene o imposta un valore che indica se il carattere è un sottoinsieme.

```csharp
public bool IsSubset { get; set; }
```

### Esempi

L'esempio mostra come cercare il testo sulla prima pagina e ottenere il valore che indica se il carattere è un sottoinsieme.

```csharp
// crea un oggetto TextFragmentAbsorber per trovare tutte le occorrenze di testo "hello world".
Document doc = new Document(@"D:\Tests\input.pdf");

// accetta l'assorbitore per la prima pagina
TextFragmentAbsorber absorber = new TextFragmentAbsorber("hello world");

// cambia il carattere per la prima occorrenza del testo
doc.Pages[1].Accept(absorber);

// salva il documento
if(absorber.TextFragments[1].TextState.Font.IsSubset)
   Console.Out.WriteLine("the font is a subset");
```
