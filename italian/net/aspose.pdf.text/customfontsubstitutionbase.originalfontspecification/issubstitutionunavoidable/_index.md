## CustomFontSubstitutionBase.OriginalFontSpecification.IsSubstitutionUnavoidable property

Ottiene un valore che indica che la sostituzione è inevitabile.

```csharp
public bool IsSubstitutionUnavoidable { get; }
```

### Osservazioni

Restituisce true nel caso in cui sia stata richiesta la sostituzione a causa dell'assenza del font originale o nel caso in cui il font originale non possa essere utilizzato nel contesto di qualche attività. Nel caso in cui l'utente ignori il flag e non sostituisca il font - viene eseguita la procedura di sostituzione del font predefinita . Ma offre l'opportunità all'utente di alternare la procedura standard di sostituzione dei caratteri e impostare caratteri migliori per il sistema. Restituisce false nel caso sia presente il carattere originale, valido, ma è consentito all'utente sostituirlo.
