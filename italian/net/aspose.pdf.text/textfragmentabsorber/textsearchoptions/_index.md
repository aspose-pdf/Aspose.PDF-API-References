## TextFragmentAbsorber.TextSearchOptions property

Ottiene o imposta le opzioni di ricerca. Le opzioni abilitano la ricerca usando le espressioni regolari.

```csharp
public TextSearchOptions TextSearchOptions { get; set; }
```

### Esempi

L'esempio mostra come eseguire la ricerca di testo utilizzando un'espressione regolare.

```csharp
// Apri documento
Document doc = new Document(@"D:\Tests\input.pdf");

// Crea un oggetto TextFragmentAbsorber per trovare tutte le occorrenze di testo "ciao".
TextFragmentAbsorber absorber = new TextFragmentAbsorber();

// cerca un'altra parola e sostituiscila
absorber.Phrase = @"h\w*?o";
absorber.TextSearchOptions = new TextSearchOptions(true);

// Apri documento
doc.Pages[1].Accept(absorber);
absorber.TextFragments[1].Text = "Hi"; 

// Apri documento
doc.Save(@"D:\Tests\output.pdf"); 
```
