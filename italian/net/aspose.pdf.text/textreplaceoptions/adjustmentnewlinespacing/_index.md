## Proprietà TextReplaceOptions.AdjustmentNewLineSpacing

Ottiene o imposta il valore dell'interlinea utilizzato se l'aggiustamento di sostituzione è forzato a creare una nuova riga di testo. Il valore atteso è un moltiplicatore della dimensione del carattere del testo sostituito. Il valore predefinito è 1.2.

```csharp
public double AdjustmentNewLineSpacing { get; set; }
```
