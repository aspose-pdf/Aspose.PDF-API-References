Estrae il testo nella pagina specificata

```csharp
public virtual void Visit(Page page)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| page | Page | Oggetto pagina documento PDF. |

### Esempi

L'esempio mostra come estrarre il testo sulla prima pagina del documento PDF.

```csharp
// apri il documento
Document doc = new Document(inFile);

// crea un oggetto TextAbsorber per estrarre il testo
TextAbsorber absorber = new TextAbsorber();

// accetta l'assorbitore per tutte le pagine del documento
absorber.Visit(doc.Pages[1]);

// ottieni il testo estratto
string extractedText = absorber.Text;
```
