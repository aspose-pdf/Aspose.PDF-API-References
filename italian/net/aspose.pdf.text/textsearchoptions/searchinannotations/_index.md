## Proprietà TextSearchOptions.SearchInAnnotations

Ottiene o imposta un valore che consente di cercare testo nelle Annotazioni. true - il testo sarà cercato nelle Annotazioni. false - il testo nelle Annotazioni non sarà analizzato da TextFragmentAbsorber.

```csharp
public bool SearchInAnnotations { get; set; }
```
