## TextSearchOptions.SearchForTextRelatedGraphics property

Ottiene o imposta un valore che consente la ricerca di elementi grafici relativi al testo (sottolineature, sfondo, ecc.) durante la ricerca di testo. true - verrà eseguita la ricerca di elementi grafici relativi al testo (valore predefinito). false - elementi grafici che potrebbero essere presenti nel documento di origine sarà ignorato. Impostalo in caso di problemi di prestazioni o non è necessario gestire la sottolineatura, lo sfondo o il ritaglio.

public bool SearchForTextRelatedGraphics { get; set; }
