## Proprietà TextSearchOptions.SearchForTextRelatedGraphics

Ottiene o imposta un valore che consente di cercare grafica correlata al testo (sottolineature, sfondo, ecc.) durante la ricerca di testo. true - verrà eseguita la ricerca di grafica correlata al testo (valore predefinito). false - gli elementi grafici che possono essere presenti nel documento sorgente verranno ignorati. Imposta questo in caso di problemi di prestazioni o se non è necessario gestire sottolineature, sfondo o ritagli.

```csharp
public bool SearchForTextRelatedGraphics { get; set; }
```
