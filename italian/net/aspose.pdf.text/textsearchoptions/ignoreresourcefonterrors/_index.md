## TextSearchOptions.IgnoreResourceFontErrors property

Ottiene o imposta l'indicazione che gli errori relativi all'assenza di carattere verranno ignorati dall'assorbitore di testo (frammento). true - significa che gli errori di assenza di carattere verranno ignorati. I segmenti di testo che fanno riferimento a risorse errate verranno ignorati durante l'elaborazione. false (predefinito) - l'assenza di errori di carattere interromperà l'elaborazione generando un'eccezione.

public bool IgnoreResourceFontErrors { get; set; }
