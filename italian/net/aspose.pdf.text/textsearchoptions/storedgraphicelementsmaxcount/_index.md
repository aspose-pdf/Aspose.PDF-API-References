## Proprietà TextSearchOptions.StoredGraphicElementsMaxCount

Ottiene o imposta un valore che limita la ricerca di grafica relativa al testo (sottolineatura, sfondo, ecc.) su una pagina per il numero specificato di elementi. Il valore predefinito è 250. Imposta un valore inferiore in caso di problemi di prestazioni, prova un valore maggiore nel caso in cui alcuni elementi grafici non siano stati trovati.

```csharp
public int StoredGraphicElementsMaxCount { get; set; }
```
