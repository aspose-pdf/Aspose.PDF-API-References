## TextSearchOptions.StoredGraphicElementsMaxCount property

Ottiene o imposta un valore che limita la ricerca di elementi grafici relativi al testo (sottolineature, sfondo, ecc.) in una pagina per il numero specificato di elementi. Il valore predefinito è 250. Impostare un valore minore in caso di problemi di prestazioni, provare un valore maggiore nel caso alcuni elementi grafici non sono stati trovati.

public int StoredGraphicElementsMaxCount { get; set; }
