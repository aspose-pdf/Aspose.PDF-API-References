## Proprietà TextSearchOptions.UseFontEngineEncoding

Ottiene o imposta l'indicazione che il testo sarà cercato utilizzando la codifica del motore dei caratteri. true - significa che verrà utilizzata la codifica del motore dei caratteri (prova questo se la ricerca del testo fallisce a causa di una codifica imperfetta nel documento) false - significa che verrà utilizzata la codifica del carattere del documento (valore predefinito)

```csharp
public bool UseFontEngineEncoding { get; set; }
```
