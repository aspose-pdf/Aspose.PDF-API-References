## TextState.LineSpacing property

Ottiene o imposta l'interlinea del testo.

```csharp
public virtual float LineSpacing { get; set; }
```

### Osservazioni

Si noti che il valore non viene conservato come caratteristica del testo all'interno del documento. Il getter della proprietà LineSpacing funziona per un oggetto nel caso in cui sia stato impostato esplicitamente in precedenza con il setter LineSpacing per quegli oggetti. La proprietà viene utilizzata dal runtime nel contesto di processo di generazione/modifica corrente.
