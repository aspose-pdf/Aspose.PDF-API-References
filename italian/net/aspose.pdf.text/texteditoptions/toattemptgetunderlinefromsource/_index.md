## Proprietà TextEditOptions.ToAttemptGetUnderlineFromSource

Ottiene o imposta un valore che consente di cercare la sottolineatura del testo nella pagina del documento sorgente. (Obsoleto) Si prega di utilizzare TextSearchOptions.SearchForTextRelatedGraphics invece di questo.

public bool ToAttemptGetUnderlineFromSource { get; set; }
