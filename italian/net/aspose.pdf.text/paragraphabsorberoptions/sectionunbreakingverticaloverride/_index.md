## Proprietà ParagraphAbsorberOptions.SectionUnbreakingVerticalOverride

Ottiene o imposta la lunghezza massima della zona con 'livello di riempimento zero' che non sarà trattata come un vero e proprio interruzione di sezione verticale (come frazione della larghezza / altezza di una pagina). Il valore predefinito è 0.005.

```csharp
public double SectionUnbreakingVerticalOverride { get; set; }
```
