## TextReplaceOptions.ReplaceAdjustment enumeration

Determina l'azione che verrà eseguita dopo la sostituzione del frammento di testo a un valore più breve. Nessuna: nessuna azione, il testo sostituito potrebbe sovrapporsi al resto della riga; AdjustSpaceWidth: prova a regolare gli spazi tra le parole per mantenere la lunghezza della riga; WholeWordsHyphenation: prova a distribuire le parole tra le righe del paragrafo per mantenere il campo a destra del paragrafo; ShiftRestOfLine - sposta il resto della riga in base alla modifica della lunghezza del testo, la lunghezza della riga può essere modificata; Il valore predefinito è ShiftRestOfLine.

```csharp
[Flags]
public enum ReplaceAdjustment
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Nessuna azione, il testo sostituito potrebbe sovrapporsi al resto della riga |
| AdjustSpaceWidth | `1` | Prova a regolare gli spazi tra le parole per mantenere la lunghezza della riga |
| WholeWordsHyphenation | `2` | Prova a distribuire le parole tra le righe di paragrafo per mantenere il campo destro del paragrafo |
| ShiftRestOfLine | `4` | (Predefinito) Sposta il resto della riga in base alla modifica della lunghezza del testo, la lunghezza della riga può essere modificata |
