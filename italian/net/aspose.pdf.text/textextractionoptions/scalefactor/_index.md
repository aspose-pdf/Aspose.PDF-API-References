## Proprietà TextExtractionOptions.ScaleFactor

Ottiene o imposta il fattore che verrà applicato per scalare la dimensione del carattere durante l'estrazione in modalità pura. Un valore inferiore porta a più spazi nel testo estratto. Il valore predefinito è 1 - nessuna scalatura; Impostare il valore a zero consente all'algoritmo di scegliere automaticamente la scalatura.

```csharp
public double ScaleFactor { get; set; }
```
