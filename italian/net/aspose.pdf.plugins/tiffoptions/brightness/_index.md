## Proprietà TiffOptions.Brightness

Ottiene o imposta un valore limite della trasformazione dei colori in bianco e nero. Questo parametro può essere applicato con EncoderValue.CompressionCCITT4, EncoderValue.CompressionCCITT3, EncoderValue.CompressionRle o ColorDepth.Format1bpp == 1

```csharp
public float Brightness { get; set; }
```

### Valore della Proprietà

Il valore della luminosità dovrebbe essere compreso nell'intervallo da 0 a 1. Per impostazione predefinita, il valore è uguale a 0.33f
