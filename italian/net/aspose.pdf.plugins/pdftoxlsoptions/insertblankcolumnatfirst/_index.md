## PdfToXlsOptions.InsertBlankColumnAtFirst property

Imposta true se hai bisogno di inserire una colonna vuota come prima colonna del foglio di lavoro. Il valore predefinito è false; significa che la colonna vuota non verrà inserita.

public bool InsertBlankColumnAtFirst { get; set; }
