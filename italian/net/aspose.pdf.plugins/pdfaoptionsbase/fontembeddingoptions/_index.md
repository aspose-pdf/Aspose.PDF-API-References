## Proprietà PdfAOptionsBase.FontEmbeddingOptions

Ottiene le opzioni per elaborare i caratteri che non possono essere incorporati nel documento.

```csharp
public FontEmbeddingOptions FontEmbeddingOptions { get; }
```

### Valore della Proprietà

Le opzioni di incorporamento dei caratteri.

## Osservazioni

Lo standard PDF/A richiede che tutti i caratteri devono essere incorporati nel documento. Questa proprietà fornisce opzioni per gestire i casi in cui non è possibile incorporare alcuni caratteri perché sono assenti nel PC di destinazione.
