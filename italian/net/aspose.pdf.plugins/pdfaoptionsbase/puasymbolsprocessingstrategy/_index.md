## Proprietà PdfAOptionsBase.PuaSymbolsProcessingStrategy

Ottiene o imposta la strategia per l'elaborazione dei simboli dell'Area di Uso Privato (PUA) nel documento PDF.

```csharp
public PuaProcessingStrategy PuaSymbolsProcessingStrategy { get; set; }
```

### Valore della Proprietà

La strategia per l'elaborazione dei simboli PUA. Il valore predefinito è None per i documenti di conformità di Livello B e SurroundPuaTextWithEmptyActualText per quelli di conformità di Livello A.
