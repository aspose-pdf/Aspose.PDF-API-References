## Metodo MessageContentRequest.CreateImageFileContent

Crea un contenuto di file immagine per un messaggio.

```csharp
public static MessageContentRequest CreateImageFileContent(string fileId, string detail = "auto")
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fileId | String | L'ID del file immagine. |
| detail | String | Il dettaglio del file immagine. Specifica il livello di dettaglio dell'immagine se specificato dall'utente. "low" utilizza meno token, puoi optare per un'alta risoluzione usando "high". Il valore predefinito è "auto". |

### Valore di ritorno

Una nuova istanza di MessageContentRequest con contenuto ImageFile.
