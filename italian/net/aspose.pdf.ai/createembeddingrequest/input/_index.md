## Proprietà CreateEmbeddingRequest.Input

Ottiene o imposta il testo di input da incorporare, codificato come stringa o array di token. Per incorporare più input in una singola richiesta, passare un array di stringhe o un array di array di token. L'input non deve superare il numero massimo di token di input per il modello (8192 token per text-embedding-ada-002), non può essere una stringa vuota e qualsiasi array deve avere dimensioni di 2048 o meno.

```csharp
public string Input { get; set; }
```
