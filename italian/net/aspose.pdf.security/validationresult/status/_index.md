## Proprietà ValidationResult.Status

Ottiene lo stato del processo di validazione per un certificato.

```csharp
public ValidationStatus Status { get; }
```

## Osservazioni

La proprietà Status indica l'esito della validazione del certificato. I valori possibili sono definiti nell'enumerazione [`ValidationStatus`](../../validationstatus/), come Valido, Non valido o Indefinito. Fornisce un'idea se il certificato ha superato i controlli di validazione o meno.
