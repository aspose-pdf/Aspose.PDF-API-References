## Aspose.PDF per .NET

{{% alert color="primary" %}} 

![Aspose.PDF per il logo del prodotto .NET](../aspose_pdf-for-net.png)

Aspose.PDF è un componente .NET creato per consentire agli sviluppatori di creare documenti PDF, semplici o complessi, al volo in modo programmatico. Aspose.PDF per .NET consente agli sviluppatori di inserire tabelle, grafici, immagini, collegamenti ipertestuali, caratteri personalizzati e altro ancora nei documenti PDF. Inoltre, è anche possibile comprimere documenti PDF. Aspose.PDF per .NET offre eccellenti funzionalità di sicurezza per sviluppare documenti PDF protetti. E la caratteristica più distintiva di Aspose.PDF per .NET è che supporta la creazione di documenti PDF sia tramite un'API che da modelli XML.

## Aspose.PDF per C++

{{% alert color="primary" %}} 

![Aspose.PDF per il logo del prodotto C++](../aspose_pdf-for-cpp.png)

Aspose.PDF per C++ è una libreria C++ nativa che consente agli sviluppatori di creare, leggere e manipolare documenti PDF senza utilizzare Adobe Acrobat. Aspose.PDF per C++ consente agli sviluppatori di creare moduli, aggiungere/modificare testo, manipolare pagine PDF, aggiungere annotazioni, gestire caratteri personalizzati e molto altro. Nella sezione della documentazione di Aspose.PDF per C++, puoi esplorare tutte le funzionalità dell'API insieme agli esempi di base per dimostrare l'implementazione di tali funzionalità nella tua applicazione C++.
